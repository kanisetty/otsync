Для изменения изображения профиля выполните следующие действия:

  1. Нажмите клавишу BlackBerry Меню и выберите Настройки.

  2. Коснитесь строки Фотография пользователя и выполните одно из следующих действий:

    • Щелкните Изменить фотографию, чтобы настроить изображение профиля, используя изображение на BlackBerry.

    • Щелкните Настроить по умолчанию, чтобы настроить изображение профиля, используя изображение профиля OpenText Tempo по умолчанию.

  3. Нажмите клавишу BlackBerry Выход, чтобы вернуться к файлам.
