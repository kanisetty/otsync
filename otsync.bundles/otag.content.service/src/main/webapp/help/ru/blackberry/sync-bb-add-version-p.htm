Чтобы добавить новую версию с помощью файла на устройстве BlackBerry, выполните следующие действия.

  1. Щелкните и перейдите к целевому файлу. Если целевой файл отмечен как избранный, щелкните .

  2. Выделите целевой файл, к которому необходимо добавить новую версию.

     

    Замечание:
    Не удастся добавить версию к файлу в общей папке, к которой имеется разрешение только на чтение, определенное .

     

  3. Нажмите клавишу BlackBerry Меню и выберите Передать новую версию.

  4. Перейдите на устройстве BlackBerry к новой версии файла.

  5. Выделите файл.

  6. Нажмите клавишу BlackBerry Меню и выберите Передать. Файл будет передан автоматически.

Чтобы передать файл, сохраненный на устройстве BlackBerry, выполните следующие действия:

  1. Щелкните и перейдите к целевой папке, в которую необходимо отправить файл.

  2. Нажмите клавишу BlackBerry Меню и выберите Передать на эту страницу.

  3. Перейдите к целевому документу на смартфоне BlackBerry.

     

    Замечание:
    Не удастся отправить файл в общую папку, к которой имеется разрешение только на чтение, определенное .

     

  4. Нажмите клавишу BlackBerry Меню и выберите Передать.

Для импорта файла из приложения BlackBerry выполните следующие действия.

  1. Запустите приложение BlackBerry.

  2. Нажмите клавишу BlackBerry Меню и выберите Добавить в OpenText Tempo.

  3. По выбору При отображении соответствующего запроса войдите в OpenText Tempo. Подробную информацию см. в Вход и управление профилем.

  4. Перейдите в целевое местоположение, в которое необходимо поместить файл.

     

    Замечание:
    Не удастся импортировать файл в общую папку, к которой имеется разрешение только на чтение, определенное .

     

  5. Нажмите клавишу BlackBerry Меню и выберите Передать на эту страницу.
