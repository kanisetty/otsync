Создание профиля

  1. При первом запуске приложения OpenText Tempo нажмите Создать профиль.

    Если при нахождении в системе OpenText Tempo требуется создать дополнительный профиль, выполните следующие действия.

    1. Нажмите клавишу BlackBerry Меню.

    2. Выберите Переключить профили.

    3. Щелкните Создать профиль.

  2. В поле Имя профиля введите уникальное на устройстве BlackBerry имя.

  3. В поле Имя пользователя введите имя пользователя. При нахождении вне компании введите свой адрес электронной почты.

     

    Замечание:
    Пользователи, которые находятся за пределами компании, до создания профиля должны сначала создать учетную запись OpenText Tempo.

    Для создания учетной записи пользователь в компании должен открыть общий доступ для внешнего пользователя. При этом внешнему пользователю будет отправлено по электронной почте приглашение с описанием шагов, которые необходимо выполнить для создания учетной записи.

     

  4. В поле Пароль введите пароль.

    Если пользователь находится вне компании, введите пароль, выбранный при создании учетной записи OpenText Tempo. Если вы забыли пароль, воспользуйтесь сообщением электронной почты, которое было отправлено вам при создании учетной записи OpenText Tempo.

     

    Замечание:
    Пароль чувствителен к регистру. Убедитесь, что пароль введен должным образом.

     

  5. В поле Сервер введите URL-адрес сервера OpenText Tempo.

    Чтобы узнать URL-адрес сервера OpenText Tempo, свяжитесь с администратором.

    Если пользователь находится за пределами компании, URL-адрес указывается в сообщении, которое было отправлено по электронной почте при создании учетной записи OpenText Tempo.

  6. По выбору Установите флажок Не отключать для мгновенного доступа к приложению при его следующем запуске. Единственные исключения: выход из OpenText Tempo и истечение времени ожидания сеанса. Информацию о данной функции см. в разделе Caution (Внимание).

     

    Замечание:
    Флажок недоступен, если BlackBerry находится в автономном режиме.

     

  7. Нажмите Сохранить и выполнить вход.

    К данным профиля относятся учетные данные учетной записи, URL-адрес OpenText Tempo и список избранных файлов. Профили связаны с устройством. Необходимо создать профили и пометить с помощью тегов файлы как избранные на всех смартфонах и планшетных ПК, на которых запущено приложение OpenText Tempo.

 

Предостережение

  • Разрешение OpenText Tempo не отключать сеанс пользователя может поставить безопасность под угрозу в случае использования BlackBerry другим пользователем.

  • Если на устройстве BlackBerry настроено несколько профилей, другие профили могут получить доступ к учетной записи OpenText Tempo при установке флажка Не отключать и вашем нахождении в системе. Чтобы выйти, нажмите клавишу BlackBerry Меню, а затем выберите Выход.
