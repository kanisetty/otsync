Открытие общего доступа к папке для внешнего пользователя

  1. Нажмите Файлы и перейдите к целевой папке.

  2. Выделите папку.

  3. Нажмите клавишу BlackBerry Меню и выберите Включить общий доступ к папке.

     

    Замечание:
    Если параметр Включить общий доступ к папке недоступен, разрешения на предоставление общего доступа к папке отсутствуют. Список папок, к которым нельзя открыть общий доступ см. в Исключения при открытии общего доступа для внешних пользователей.

     

  4. Введите адрес электронной почты внешнего пользователя, которому необходимо предоставить общий доступ к папке.

    Внешними пользователями называются лица, которые не являются сотрудниками компании. После ввода существующего адреса электронной почты в поле автозаполнения отобразится адрес электронной почты внешнего пользователя.

     

    Замечание:
    Если адрес электронной почты пропущен и отображается сообщение “Результаты не найдены”, разрешения на предоставление внешним пользователям общего доступа к папке отсутствуют. Для получения этой привилегии свяжитесь с администратором.

     

  5. Выделите адрес электронной почты и нажмите кнопку BlackBerry Сенсорная панель.

  6. Нажмите Да, чтобы подтвердить отправку этого приглашения. Папка теперь помечена как общая значком .

Исключения при открытии общего доступа для внешних пользователей

В папке OpenText Tempo можно открыть общий доступ ко всем папкам, за исключением следующих.

  • Нельзя открыть общий доступ к папке, которая является дочерней или родительской для общей папки.

  • Нельзя открыть общий доступ к папке, к которой вам предоставлен общий доступ.

  • Нельзя открыть общий доступ к корню общей папки, если вы не являетесь владельцем папки.

  • Нельзя открыть общий доступ к папке OpenText Tempo.

  • Пользователи, которые не являются сотрудниками компании, не могут открывать общий доступ к папкам.

 

Предостережение
При открытии общего доступа для внешнего пользователя этот пользователь получает полный доступ к содержимому папки. После внесения исправлений в общую папку изменения автоматически синхронизируются со всеми пользователями, которым предоставлен общий доступ к папке, включая владельца папки.
