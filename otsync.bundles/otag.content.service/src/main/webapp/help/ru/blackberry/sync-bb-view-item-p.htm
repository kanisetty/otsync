Просмотр элемента

  1. Нажмите Файлы и перейдите к целевому файлу. Если файл помечен тегом как избранное, нажмите Избранное.

  2. Выделите необходимый для просмотра элемент.

  3. Нажмите клавишу BlackBerry Сенсорная панель для просмотра элемента.

  4. Нажмите клавишу BlackBerry Выход, чтобы вернуться к файлам.
