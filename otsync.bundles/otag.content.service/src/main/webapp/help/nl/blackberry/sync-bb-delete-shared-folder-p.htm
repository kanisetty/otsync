Een gedeelde map verwijderen waarvan u de eigenaar bent:

  1. Klik op en navigeer naar de juiste gedeelde map (aangeduid met ).

  2. Markeer de gedeelde map.

  3. Druk op de BlackBerry-toets Menu en selecteer Map verwijderen.

     

    Let op
    Hierdoor wordt de map voor u en alle medewerkers verwijderd. U kunt ook medewerkers verwijderen. Zie Een medewerker verwijderen voor meer informatie.

     
     

    Opmerking:
    Als Map verwijderen niet beschikbaar is, hebt u een map geselecteerd die met u wordt gedeeld. Zie voor meer informatie onderstaand gedeelte Een map verwijderen die met u wordt gedeeld.

     

  4. Klik op OK.

Een map verwijderen die met u wordt gedeeld:

  1. Klik op .

  2. Markeer de gedeelde map.

  3. Druk op de BlackBerry-toets Menu en selecteer Gedeeld item verwijderen. Hierdoor wordt de verbinding met de gedeelde map verbroken en wordt de map uit uw OpenText Tempo-map verwijderd. Hierdoor wordt de map niet verwijderd voor de eigenaar en andere medewerkers.

     

    Let op
    Als Gedeeld item verwijderen niet beschikbaar is, hebt u een gedeelde map geselecteerd waarvan u de eigenaar bent. Raadpleeg voor meer informatie Een gedeelde map verwijderen waarvan u de eigenaar bent boven aan deze pagina.

     

  4. Klik op OK.
