To edit your profile:

  1. Press the BlackBerry Menu key and select Switch Profile.

  2. On the Profile screen, highlight your profile.

  3. Press the BlackBerry Menu key and select Edit Profile.

  4. Change the user name or server as required.

  5. Click Save Profile.

To update your password:

  1. Press the BlackBerry Menu key and select Log Out.

  2. On the Profile screen, highlight your profile and press the BlackBerry Trackpad key.

  3. Type your new password. Your password is case-sensitive. Ensure you are typing the password correctly.

  4. Press the BlackBerry Enter key.
