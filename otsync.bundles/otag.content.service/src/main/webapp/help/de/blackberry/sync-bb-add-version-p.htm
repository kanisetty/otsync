So fügen Sie eine neue Version mithilfe einer Datei auf Ihrem BlackBerry hinzu:

  1. Klicken Sie auf , und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, klicken Sie auf .

  2. Markieren Sie die Zieldatei, zu der Sie eine neue Version hinzufügen möchten.

     

    Hinweis:
    Einer Datei in einem freigegebenen Ordner, für den Sie nur Leserechte besitzen (durch gekennzeichnet), können Sie keine Version hinzufügen.

     

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Neue Version hochladen aus.

  4. Navigieren Sie zur neuen Version der Datei auf Ihrem BlackBerry.

  5. Markieren Sie die Datei.

  6. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hochladen aus. Die Datei wird automatisch hochgeladen.

So laden Sie eine auf Ihrem BlackBerry gespeicherte Datei hoch:

  1. Klicken Sie auf , und navigieren Sie zum Zielordner, wo Sie eine Datei hochladen möchten.

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hier hochladen aus.

  3. Navigieren Sie auf Ihrem BlackBerry-Smartphone zum Zieldokument.

     

    Hinweis:
    Eine Datei kann nicht in einen freigegebenen Ordner hochgeladen werden, für den Sie nur Leserechte besitzen (durch gekennzeichnet).

     

  4. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hochladen aus.

So importieren Sie eine Datei aus einer BlackBerry-Anwendung:

  1. Starten Sie eine BlackBerry-Anwendung.

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hinzufügen zu OpenText Tempo aus.

  3. Optional Wenn Sie dazu aufgefordert werden, melden Sie sich bei OpenText Tempo an. Ausführliche Informationen finden Sie in Anmelden und Profil verwalten.

  4. Navigieren Sie zum Zielspeicherort, an dem Sie die Datei platzieren möchten.

     

    Hinweis:
    Sie können eine Datei nicht in einen freigegebenen Ordner importieren, für den Sie nur Leserechte besitzen (durch gekennzeichnet).

     

  5. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hier hochladen aus.
