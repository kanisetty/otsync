So entfernen Sie einen Favoriten:

  1. Klicken Sie auf Favoriten, oder klicken Sie auf Dateien, und navigieren Sie zum Zielelement.

  2. Markieren Sie die Datei, die Sie aus der Favoritenliste entfernen möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Aus Favoriten entfernen aus. Das Element wird nicht mehr in der Liste der Favoriten angezeigt. Dadurch wird das Element nicht aus Ihrem OpenText Tempo-Ordner gelöscht.
