So erstellen Sie ein Profil:

  1. Wenn Sie erstmalig die Anwendung OpenText Tempo starten, klicken Sie auf Neues Profil.

    Wenn Sie ein weiteres Profil erstellen möchten und bereits bei OpenText Tempo angemeldet sind, gehen Sie wie folgt vor:

    1. Drücken Sie die BlackBerry Menü-Taste.

    2. Wählen Sie Profil wechseln.

    3. Klicken Sie auf Neues Profil.

  2. Geben Sie unter Profilname einen Profilnamen ein, der nur für Ihr BlackBerry gilt.

  3. Geben Sie im Feld Benutzername Ihren Benutzernamen ein. Wenn Sie sich außerhalb des Unternehmens befinden, geben Sie die E-Mail-Adresse ein.

     

    Hinweis:
    Ein Benutzer, der sich außerhalb des Unternehmens befindet, muss zunächst vor dem Erstellen eines Profils ein OpenText Tempo-Konto anlegen.

    Um ein Konto zu erstellen, muss ein Benutzer innerhalb des Unternehmens einen Ordner für den externen Benutzer freigeben. Dadurch wird an den externen Benutzer eine E-Mail-Einladung gesendet, in der die Schritte zum Erstellen eines Kontos beschrieben werden.

     

  4. Geben Sie Ihr Kennwort im Feld Kennwort ein.

    Wenn Sie sich außerhalb des Unternehmens befinden, geben Sie das Kennwort ein, das Sie beim Erstellen Ihres OpenText Tempo-Kontos ausgewählt haben. Falls Sie sich nicht an Ihr Kennwort erinnern können, finden Sie diese Informationen in der E-Mail, die Sie beim Erstellen Ihres OpenText Tempo-Kontos erhalten haben.

     

    Hinweis:
    Für das Kennwort muss Groß-/Kleinschreibung beachtet werden. Stellen Sie sicher, dass Sie das Kennwort korrekt eingeben.

     

  5. Geben Sie im Feld Server den URL Ihres OpenText Tempo-Servers ein.

    Den URL des OpenText Tempo-Servers erhalten Sie von Ihrem Administrator.

    Wenn Sie sich außerhalb des Unternehmens befinden, finden Sie die OpenText Tempo-URL in der E-Mail, die Sie nach dem Erstellen des OpenText Tempo-Kontos erhalten haben.

  6. Optional Aktivieren Sie das Kontrollkästchen Angemeldet bleiben, um beim nächsten Start der Anwendung einen sofortigen Zugriff zuzulassen. Dies gilt nicht, wenn Sie sich von OpenText Tempo abmelden oder die Sitzung abläuft. Lesen Sie die Informationen für diese Option unter Achtung.

     

    Hinweis:
    Dieses Kontrollkästchen ist nicht verfügbar, wenn das BlackBerry offline ist.

     

  7. Klicken Sie auf Speichern und anmelden.

    Zu den Profildaten gehören Ihre Kontoanmeldedaten, der OpenText Tempo-URL und eine Liste Ihrer Favoritendateien. Profile gelten speziell für Ihr Gerät. Sie müssen auf jedem Ihrer Smartphones und Tablets mit der OpenText Tempo-Anwendung ein Profil erstellen und Dateien als Favoriten markieren.

 

Achtung

  • Wenn Sie zulassen, dass Sie bei OpenText Tempo angemeldet bleiben, kann dies die Sicherheit beeinträchtigen, sollte Ihr BlackBerry von einer anderen Person verwendet werden.

  • Wenn Sie mehrere Profile auf Ihrem BlackBerry konfiguriert haben, können andere Profile auf Ihr OpenText Tempo-Konto zugreifen, wenn das Kontrollkästchen Angemeldet bleiben ausgewählt ist und Sie sich nicht abgemeldet haben. Um sich abzumelden, drücken Sie die BlackBerry Menü-Taste und wählen Sie Abmelden.
