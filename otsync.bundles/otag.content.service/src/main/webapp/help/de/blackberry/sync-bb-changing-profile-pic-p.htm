So ändern Sie Ihr Profilbild:

  1. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Einstellungen aus.

  2. Führen Sie in der Zeile Benutzerfoto einen der folgenden Schritte aus:

    • Klicken Sie auf Foto ändern, um Ihr Profilbild mit einem Bild auf Ihrem BlackBerry festzulegen.

    • Klicken Sie auf Als Standard festlegen, um als Profilbild das standardmäßige OpenText Tempo-Profilbild zu verwenden.

  3. Drücken Sie die BlackBerry Esc-Taste, um zu Ihren Dateien zurückzukehren.
