Pour effectuer une recherche dans votre dossier OpenText Tempo :

  1. Cliquez sur .

  2. Saisissez un terme de recherche.

  3. Appuyez sur la touche BlackBerry Entrée. Les résultats de la recherche s'affichent.

  4. Mettez en surbrillance le fichier ou dossier que vous souhaitez visualiser.

  5. Appuyez sur la touche BlackBerry Menu pour afficher les options disponibles.

Pour ouvrir l'emplacement d'un élément à partir des résultats d'une recherche :

  1. Recherchez dans votre dossier OpenText Tempo

  2. Mettez en surbrillance le fichier que vous souhaitez ouvrir.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Ouvrir l'emplacement.
