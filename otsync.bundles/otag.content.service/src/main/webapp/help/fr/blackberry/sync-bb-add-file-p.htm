Pour charger un fichier stocké sur votre BlackBerry :

  1. Cliquez sur Fichier, puis accédez au dossier cible dans lequel charger le fichier.

  2. Appuyez sur la touche BlackBerry Menu et sélectionnez Charger ici.

  3. Accédez au document cible sur votre smartphone BlackBerry.

  4. Appuyez sur la touche BlackBerry Menu et sélectionnez Téléchargement en amont.

Pour importer un fichier à partir d'une application BlackBerry :

  1. Lancez une application BlackBerry. Par exemple, l'application Appareil photo.

  2. Appuyez sur la touche BlackBerry Menu et sélectionnez Ajouter àOpenText Tempo.

  3. Facultatif Si vous y êtes invité, connectez-vous à OpenText Tempo. Pour plus d'informations, voir Connexion.

  4. Accédez à l'emplacement dans lequel vous souhaitez importer le fichier.

  5. Appuyez sur la touche BlackBerry Menu et sélectionnez Charger ici.
