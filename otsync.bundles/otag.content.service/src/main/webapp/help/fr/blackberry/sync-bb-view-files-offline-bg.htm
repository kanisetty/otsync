Pour visualiser des fichiers tout en étant hors ligne :

  1. Pour faire en sorte qu'un fichier puisse être visualisé hors ligne, marquez le fichier souhaité comme favori et synchronisez ou téléchargez le fichier quand vous êtes en ligne. Pour plus d'informations, voir Ajout, visualisation, suppression et synchronisation de favoris.

    Les fichiers marqués comme favoris sont indiqués par le recouvrement . Par exemple : .

    Si le recouvrement apparaît, le fichier marqué comme favori n'est pas téléchargé. Par exemple : . Pour télécharger le fichier, ouvrez-le ou synchronisez vos favoris. Pour plus d'informations, voir Ajout, visualisation, suppression et synchronisation de favoris.

  2. Cliquez sur .

  3. Mettez en surbrillance le fichier à visualiser.

  4. Appuyez sur la touche BlackBerry Trackpad pour visualiser le fichier.

    Quand vous êtes hors ligne, les fichiers que vous ouvrez peuvent ne pas correspondre aux versions les plus récentes puisque les modifications apportées via un autre client ne peuvent pas être synchonisées lorsque vous êtes hors ligne.

  5. Pour revenir à vos fichiers, appuyez sur la touche BlackBerry Echap.
