Para excluir uma pasta compartilhada que você possui:

  1. Clique em Arquivos e navegue até a pasta de destino.

  2. Destaque a pasta compartilhada.

  3. Pressione a tecla Menu do BlackBerry e selecione Excluir item.

     

    Nota:
    Essa ação excluirá a sua pasta e de todos os colaboradores. Como alternativa, você pode remover todos os colaboradores. Para obter detalhes, consulte Para remover colaboradores.

    Se Excluir pasta não estiver disponível, você selecionou uma pasta que é compartilhada com você. Antes de prosseguir, consulte Excluindo uma pasta compartilhada.

     

  4. Clique em OK.

Para excluir uma pasta compartilhada com você:

  1. Clique em Arquivos.

  2. Destaque a pasta compartilhada na qual você deseja romper uma conexão.

  3. Pressione a tecla Menu do BlackBerry e selecione Remover compartilhamento.

     

    Nota:
    Se Remover compartilhamento não estiver disponível, você selecionou uma pasta sua. Antes de prosseguir, consulte Excluindo uma pasta compartilhada.

     

  4. Clique em OK.

Excluindo uma pasta compartilhada

As pastas compartilhadas são indicadas por dois avatares verdes no canto inferior direito da pasta . Há dois tipos de pastas compartilhadas:

  • Uma pasta que você possui e compartilha com os colegas. A exclusão de uma pasta compartilhada que você possui excluirá a pasta para você e todos os colaboradores que possuem acesso de compartilhamento a ela. Como alternativa, você pode remover um ou todos os colaboradores. Para obter mais informações, consulte Para remover colaboradores.

  • Uma pasta que você não possui e é compartilhada com você. A exclusão de uma pasta que é compartilhada com você rompe a conexão com a pasta compartilhada e a remove da sua pasta do OpenText Tempo. Essa ação não exclui a pasta para o proprietário ou outros colaboradores.
