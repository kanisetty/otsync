  1. Para tornar um arquivo disponível para exibição off-line, marque-o como favorito, fazendo sua sincronização ou download quando ainda estiver on-line. Para obter detalhes, consulte Adicionando, exibindo, removendo e sincronizando favoritos.

  3. Realce o arquivo que deseja exibir.

  4. Pressione a tecla Trackpad do BlackBerry para exibir o arquivo.

    Os arquivos abertos quando você estava off-line talvez não sejam as versões mais recentes, uma vez que as edições feitas com o uso de outro cliente não podem ser sincronizadas quando você está off-line.
