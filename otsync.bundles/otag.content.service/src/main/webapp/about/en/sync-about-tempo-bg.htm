OpenText Tempo

OpenText Tempo offers secure file sharing and synchronization for the Enterprise. OpenText Tempo lets you manage and collaborate on documents in your OpenText Tempo folder regardless of your location. The content in your OpenText Tempo folder is stored online and is accessible using desktop clients, smartphones and tablets, or a Web browser. When the content is altered from any of these clients, the changes are automatically and instantly synchronized with all other connected OpenText Tempo clients.

Desktop Client

You can access your OpenText Tempo folder by installing the OpenText Tempo client on your Windows desktop computer. This creates a folder on your local file system of the content in your OpenText Tempo folder. Any time you add, delete, or save changes to the content in your OpenText Tempo folder on your desktop computer, the changes automatically synchronize with the Web client and all smartphones and tablets you have running the OpenText Tempo application.

Mobile Client

You can access your OpenText Tempo folder by installing the OpenText Tempo application on an Apple iPhone, Apple iPad, Android smartphone or tablet, as well as a BlackBerry smartphone. Like the desktop client, any changes you make to your content using the OpenText Tempo application are synced with all other clients.

Web Client

You can access your OpenText Tempo folder from any Web browser by entering the OpenText Tempo URL into the browser address region. Using the Web client, you can manage your content and share folders with other users. As with the desktop and mobile clients, any changes that you make to your OpenText Tempo content using the Web browser are synced with all other clients.

Sharing Content

With OpenText Tempo, you can also share content with other users for easy collaboration. Any changes made to the content in a shared folder are automatically synced to all members of the shared folder, regardless of who made the changes.
