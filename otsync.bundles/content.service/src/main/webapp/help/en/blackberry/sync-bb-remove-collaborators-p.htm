Removing a Collaborator

You can remove a collaborator from a shared folder that you own using the OpenText Tempo Box Web user interface. This severs the connection to the shared folder and deletes the folder from the collaboratorâ€™s Tempo Box folder.

You can access the Web user interface by entering the OpenText Tempo Box URL in the address of a Web browser and signing in to OpenText Tempo Box. For more information on removing collaborators, refer to the online help in the OpenText Tempo Box Web user interface.
