所有している共有フォルダーを削除するには、次の操作を行います。

  1. をクリックし、ターゲットの共有フォルダーに移動します。共有フォルダーには が表示されています。

  2. 共有フォルダーをハイライト表示します。

  3. BlackBerry [メニュー] キーを押し、[フォルダーの削除] を選択します。

     

    注意
    これで、自分自身またはすべての共同作業者のフォルダーが削除されます。また、共有作業者を削除することもできます。詳細については共同作業者の削除を参照してください。

     
     

    注意:
    [フォルダーの削除] が使用できない場合、共有されているフォルダーを選択しています。詳細については、下記の「共有されているフォルダーを削除するには」を参照してください。

     

  4. [OK] をクリックします。

共有されているフォルダーを削除するには、次の操作を行います。

  1. をクリックします。

  2. 共有フォルダーをハイライト表示します。

  3. BlackBerry [メニュー] キーを押し、[共有の削除] を選択します。これによって共有フォルダーへの接続が切断され、ユーザーの OpenText Tempo フォルダーから削除されます。これによって、所有者または他の共同作業者のフォルダーが削除されることはありません。

     

    注意
    [共有の削除] が使用できない場合、自身で共有しているフォルダーを選択しています。詳細については、このページの最初にある「所有している共有フォルダーを削除するには」を参照してください。

     

  4. [OK] をクリックします。
