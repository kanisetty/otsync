ファイルを編集するには、次の操作を実行します。

  1. [ファイル] をクリックして、ターゲット ファイルに移動します。ファイルにお気に入りタグが付いている場合、[お気に入り] をクリックします。

  2. 編集するファイルをハイライト表示します。

  3. BlackBerry [トラックパッド] キーを押し、ファイルを表示します。

  4. サード パーティーのアプリケーションでファイルが開きます。アプリケーションが編集をサポートする場合、ファイルを編集して変更を保存します。

    ファイルへの変更は、OpenText Tempo フォルダー内のファイルに保存されません。変更済みのファイルをサード パーティ アプリケーションまたは BlackBerry キャッシュ フォルダーからアップロードする必要があります。

  5. OpenText Tempo で、ファイルのバージョンを追加して、編集したファイルをアップロードします。詳細についてはファイルの新しいバージョンを追加するにはを参照してください。
