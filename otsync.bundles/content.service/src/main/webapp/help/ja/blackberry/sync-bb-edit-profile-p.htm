プロファイルを編集するには、次の操作を行います。

  1. BlackBerry [メニュー] キーを押し、[スイッチ プロファイル] を選択します。

  2. [プロファイル] 画面で、プロファイルをハイライト表示します。

  3. BlackBerry [メニュー] キーを押し、[プロフィールを編集] を選択します。

  4. 必要に応じて、ユーザー名またはサーバーを変更します。

  5. [プロファイルの保存] をクリックします。

パスワードを更新するには、次の操作を行います。

  1. BlackBerry [メニュー] キーを押し、[ログアウト] を選択します。

  2. [プロファイル] 画面で、プロファイルをハイライト表示し、BlackBerry [トラックパッド] キーを押します。

  3. 新しいパスワードを入力します。

     

    注意:
    パスワードでは大文字と小文字が区別されます。パスワードを正しく入力していることを確認してください。

     

  4. BlackBerry [Enter] キーを押します。
