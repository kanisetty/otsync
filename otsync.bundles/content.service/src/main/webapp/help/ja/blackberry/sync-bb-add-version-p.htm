BlackBerry にあるファイルに新しいバージョンを追加するには、次の操作を行います。

  1. をクリックし、ターゲット ファイルに移動します。ターゲット ファイルにお気に入りタグが付いている場合、 をクリックします。

  2. 新しいバージョンを追加するターゲット ファイルをハイライト表示します。

     

    注意:
    が表示されている、読み取り専用権限を持つ共有フォルダーのファイルにバージョンは追加できません。

     

  3. BlackBerry [メニュー] キーを押し、[新しいバージョンをアップロード] を選択します。

  4. BlackBerry を操作してファイルの新しいバージョンに移動します。

  5. ファイルをハイライト表示します。

  6. BlackBerry [メニュー] キーを押し、[アップロード] を選択します。ファイルが自動的にアップロードされます。

BlackBerry に保存されているファイルをアップロードするには、次の操作を行います。

  1. をクリックして、ファイルをアップロードするターゲットのフォルダーに移動します。

  2. BlackBerry [メニュー] キーを押し、[ここにアップロード] を選択します。

  3. BlackBerry スマートフォン上のターゲット ドキュメントに移動します。

     

    注意:
    が表示されている、読み取り専用権限を持つ共有フォルダーにファイルはアップロードできません。

     

  4. BlackBerry [メニュー] キーを押し、[アップロード] を選択します。

BlackBerry アプリケーションからファイルをインポートするには、次の操作を行います。

  1. BlackBerry アプリケーションを起動します。

  2. BlackBerry [メニュー] キーを押し、[OpenText Tempo への追加] を選択します。

  3. オプション 求められた場合は、OpenText Tempo にサインインします。詳細についてはサインインとプロファイルの管理を参照してください。

  4. ファイルを配置するターゲットの場所に移動します。

     

    注意:
    が表示されている、読み取り専用権限を持つ共有フォルダーにファイルはインポートできません。

     

  5. BlackBerry [メニュー] キーを押し、[ここにアップロード] を選択します。
