要编辑配置文件：

  1. 按 BlackBerry [菜单]键，然后选择[切换配置文件]。

  2. 在[配置文件]屏幕上，突出显示您的配置文件。

  3. 按 BlackBerry [菜单]键，然后选择[编辑配置文件]。

  4. 根据需要更改用户名或服务器。

  5. 单击[保存配置文件]。

要更新密码：

  1. 按 BlackBerry [菜单]键，然后选择[注销]。

  2. 在[配置文件]屏幕上，突出显示您的配置文件，然后按 BlackBerry [触控板]键。

  3. 键入您的新密码。

     

    注意:
    密码区分大小写。请确保您键入的密码正确。

     

  4. 按 BlackBerry [Enter] 键。
