Para compartir una carpeta con un usuario externo:

  1. Pulse Archivos y desplácese hasta la carpeta destino.

  2. Resalte la carpeta.

  3. Pulse la tecla BlackBerry Menú del dispositivo y seleccione Compartir carpeta.

     

    Nota:
    Si la opción Compartir carpeta no está disponible, es porque no tiene permiso para compartir esta carpeta. Para ver una lista de las carpetas que no pueden compartirse, consulte Excepciones para compartir una carpeta con un usuario externo.

     

  4. Escriba la dirección de correo electrónico del usuario externo con el que desea compartir esta carpeta.

    Los usuarios externos son usuarios que no son empleados de la compañía. Después de escribir una dirección de correo electrónico válida, la función 'autocompletar' mostrará la dirección de correo electrónico del usuario externo.

     

    Nota:
    Si la dirección de correo electrónico se ignora y se muestra “No se han encontrado resultados”, no tiene permiso para compartir carpetas con usuarios externos. Póngase en contacto con el administrador para solicitar este privilegio.

     

  5. Resalte la dirección de correo electrónico y pulse el BlackBerry panel táctil.

  6. Pulse Sí para confirmar que desea enviar esta invitación. La carpeta aparecerá ahora marcada como carpeta compartida, .

Excepciones para compartir una carpeta con un usuario externo

Puede compartir cualquier carpeta de su carpeta OpenText Tempo, pero se aplican las excepciones siguientes:

  • No puede compartir una carpeta que sea hija o padre de una carpeta compartida.

  • No puede compartir una carpeta que se comparta con usted.

  • No puede compartir la raíz de una carpeta compartida, a menos que sea el propietario de la carpeta.

  • El usuario no puede compartir su carpeta OpenText Tempo.

  • Si no es un empleado de la compañía, no puede compartir carpetas.

 

Atención
Al compartir una carpeta con un usuario externo, este obtiene el control completo del contenido de la carpeta. Tan pronto como se hagan revisiones a una carpeta compartida, los cambios se sincronizan automáticamente con todos los colaboradores que tienen acceso compartido a dicha carpeta, incluido el propietario de la misma.
