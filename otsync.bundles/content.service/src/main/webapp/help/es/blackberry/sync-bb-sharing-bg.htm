Para compartir una carpeta:

  1. Haga clic en y desplácese hasta la carpeta destino. Cuando se comparte una carpeta, esta puede estar situada en cualquier lugar de la carpeta OpenText Tempo. Para añadir una nueva carpeta para compartir, en el dispositivo BlackBerry pulse la tecla Menú y seleccione Añadir carpeta.

  2. Resalte la carpeta.

  3. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Compartir carpeta.

     

    Nota:
    Si Compartir carpeta no está disponible, es porque no tiene permiso para compartir esta carpeta. Para ver una lista de las carpetas que no pueden compartirse, consulte Excepciones para compartir carpetas abajo.

     

  4. Opcional Cuando se comparte una carpeta, puede otorgar a un colaborador permiso de lectura/escritura o de solo lectura para la carpeta. De forma predeterminada, los colaboradores reciben permiso de lectura/escritura, que se identifican mediante .

    Para conceder permiso de solo lectura a un colaborador, realice las siguientes acciones:

    1. Resalte .

    2. En el dispositivo BlackBerry, pulse el panel táctil.

    3. El icono cambia a .

    Con permisos de lectura/escritura, todos los colaboradores, incluidos los usuarios externos, poseen control absoluto sobre el contenido de la carpeta compartida. Pueden añadir, editar, copiar, mover, cambiar el nombre y suprimir archivos y carpetas. Una vez que se hayan hecho revisiones a la carpeta compartida, los cambios se sincronizan automáticamente con todos los colaboradores que tienen acceso compartido a dicha carpeta, incluido el propietario.

    Con permiso de solo lectura, el colaborador solo puede leer el contenido de la carpeta compartida.

  5. Escriba el nombre de la persona con la que desea compartir esta carpeta.

    Para compartir una carpeta con un usuario externo, escriba su dirección de correo electrónico. Los usuarios externos no son empleados de su empresa. Después de escribir una dirección de correo electrónico válida, la función de autocompletar mostrará la dirección de correo electrónico del usuario externo.

     

    Nota:
    Si la dirección de correo electrónico se ignora y se muestra “No se han encontrado resultados”, no tiene permiso para compartir carpetas con usuarios externos. Póngase en contacto con el administrador para solicitar este privilegio.

     

  6. Resalte el nombre o la dirección de correo electrónico del usuario y, en el dispositivo BlackBerry, pulse el panel táctil. Se le enviará una invitación para compartir la carpeta. Los colaboradores deben aceptar la invitación de uso para poder acceder a los contenidos de la carpeta.

    La carpeta aparecerá ahora marcada como carpeta compartida. Independientemente de si concede permisos de lectura/escritura o de solo lectura a los colaboradores, la carpeta compartida se mostrará como para usted.

Excepciones para compartir carpetas

Puede compartir cualquier carpeta de su carpeta OpenText Tempo, pero se aplican las excepciones siguientes:

  • Una carpeta que sea hija o padre de una carpeta compartida.

  • Una carpeta que se comparta con usted.

  • Su carpeta OpenText Tempo.

  • Los usuarios externos no pueden compartir carpetas.

Para obtener información sobre la administración de las carpetas compartidas, consulte Uso compartido de una carpeta y administración de carpetas compartidas.
