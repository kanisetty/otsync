Para adicionar uma foto usando a câmera BlackBerry:

  1. Clique em e navegue até a pasta de destino onde você deseja adicionar uma foto.

     

    Nota:
    Não é possível adicionar uma foto a uma pasta compartilhada para a qual você tem permissão somente leitura, identificada por .

     

  2. Pressione a tecla Menu do BlackBerry e selecione Adicionar foto.

  3. Pressione a tecla Trackpad do BlackBerry para tirar a foto.

  4. Clique em Adicionar aOpenText Tempo. Fotos salvas como arquivos .jpg. Não é possível editar o nome da foto antes de carregá-la.

Para adicionar uma foto de um aplicativo do BlackBerry:

  1. Inicie um aplicativo de câmera do BlackBerry.

  2. Escolha a foto que deseja importar.

  3. Pressione a tecla Menu do BlackBerry e selecione Adicionar ao OpenText Tempo.

  4. Opcional Se solicitado, faça login no OpenText Tempo. Para obter detalhes, consulte Conectando-se e gerenciando seu perfil.

  5. Navegue até o local de destino onde deseja colocar o arquivo.

     

    Nota:
    Não é possível adicionar uma foto a uma pasta compartilhada para a qual você tem permissão somente leitura, identificada por .

     

  6. Pressione a tecla Menu do BlackBerry e selecione Carregar aqui.

Para adicionar uma pasta:

  1. Clique em e navegue até o local de destino da nova pasta.

     

    Nota:
    Não é possível adicionar uma pasta a uma pasta compartilhada para a qual você tem permissão somente leitura, identificada por .

     

  2. Pressione a tecla Menu do BlackBerry e selecione Adicionar pasta.

  3. Na caixa Nome da pasta, digite um nome para essa pasta.

  4. Pressione a tecla Enter do BlackBerry.
