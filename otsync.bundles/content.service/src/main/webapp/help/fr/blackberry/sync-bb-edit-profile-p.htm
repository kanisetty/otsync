Pour modifier votre profil :

  1. Appuyez sur la touche Menu BlackBerry et sélectionnez Changer de profil.

  2. Dans l'écran Profil, mettez votre profil en surbrillance.

  3. Appuyez sur la touche BlackBerry Menu, puis sélectionnez Modifier le profil.

  4. Modifiez le nom de l'utilisateur ou le serveur.

  5. Cliquez sur Enregistrer le profil.

Pour mettre à jour votre mot de passe :

  1. Appuyez sur la touche Menu BlackBerry, puis sélectionnez Déconnexion.

  2. Dans l'écran Profil, mettez votre profil en surbrillance et appuyez sur la touche BlackBerry Trackpad.

  3. Saisissez votre nouveau mot de passe.

     

    Note :
    Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

     

  4. Appuyez sur la touche BlackBerry Entrée.
