Просмотр файла

  1. Щелкните и перейдите к целевому файлу, выделив папку и нажав клавишу BlackBerry Сенсорная панель. В строке заголовка OpenText Tempo отображается имя папки, просматриваемой в настоящий момент. Если целевой файл отмечен как избранный, щелкните .

  2. По выбору Чтобы убедиться, что просматривается самое новое содержимое, следует периодически обновлять содержимое папки. Нажмите клавишу BlackBerry Меню и выберите Обновить. При этом будет обновлен экран, но не будут загружены документы. Чтобы загрузить документ, откройте его.

  3. Выделите необходимый файл для просмотра.

  4. Нажмите клавишу BlackBerry Сенсорная панель для просмотра файла.

    При просмотре документов в общей папке, к которым имеются разрешения на чтение и запись, документы не резервируются. Вы можете открыть и редактировать документ, в который в данный момент другой пользователь вносит изменения. Общие папки с разрешением для чтения и записи определяются .

    При просмотре документов в общей папке, к которой имеется разрешение только на чтение, документы не резервируются. Вы можете просматривать, но не редактировать документы в данных папках. Общие папки с разрешением только для чтения определяются .При переходе к папке строка заголовка отображается оранжевым цветом.

  5. Нажмите клавишу BlackBerry Выход, чтобы вернуться к файлам.

Изменение файла

  1. Щелкните и перейдите к целевому файлу. Если целевой файл отмечен как избранный, щелкните .

  2. Выделите необходимый для изменения профиль.

     

    Замечание:
    Не удастся изменить файл в общей папке, к которой имеется разрешение только на чтение, определенное .

     

  3. Нажмите клавишу BlackBerry Сенсорная панель для просмотра файла.

  4. Файл будет открыт в стороннем приложении. Если приложение поддерживает редактирование, измените файл и сохраните изменения.

    Вносимые в файл изменения не сохраняются в папку OpenText Tempo. Необходимо передать измененный файл из стороннего приложения или из папки кэш-памяти BlackBerry.

  5. Передайте измененный файл в приложение OpenText Tempo с помощью добавления версии файла. Подробную информацию см. в Добавление файла.

Дополнительные сведения об управлении файлами см. в разделе Viewing, Editing, and Working With Your Files (Просмотр, редактирование и работа с файлами).
