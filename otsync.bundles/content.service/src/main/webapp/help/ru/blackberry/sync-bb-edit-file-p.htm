Изменение файла

  1. Нажмите Файлы и перейдите к целевому файлу. Если файл помечен тегом как избранное, нажмите Избранное.

  2. Выделите необходимый для изменения профиль.

  3. Нажмите клавишу BlackBerry Сенсорная панель для просмотра файла.

  4. Файл будет открыт в стороннем приложении. Если приложение поддерживает редактирование, измените файл и сохраните изменения.

    Вносимые в файл изменения не сохраняются в папку OpenText Tempo. Необходимо передать измененный файл из стороннего приложения или из папки кэш-памяти BlackBerry.

  5. Передайте измененный файл в приложение OpenText Tempo с помощью добавления версии файла. Подробную информацию см. в Добавление новой версии файла.
