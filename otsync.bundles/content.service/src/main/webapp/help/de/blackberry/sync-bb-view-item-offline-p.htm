So zeigen Sie Elemente offline an:

  1. Um ein Element zur Offline-Anzeige verfügbar zu machen, markieren Sie die Datei als Favorit, wenn Sie online sind. Ausführliche Informationen finden Sie in So fügen Sie einen Favoriten hinzu. Dateien, die als Favoriten markiert sind, werden durch die Überlagerung gekennzeichnet. Beispiel: .

  2. Klicken Sie auf Favoriten.

  3. Markieren Sie das Element, das Sie anzeigen möchten.

  4. Drücken Sie die BlackBerry Trackpad-Taste, um das Element anzuzeigen. Die Dateien, die Sie öffnen, wenn Sie offline sind, sind möglicherweise nicht die aktuellsten Versionen in Ihrem OpenText Tempo-Ordner, da Bearbeitungen der Dateien nicht synchronisiert werden können, wenn Sie offline sind.

  5. Drücken Sie die BlackBerry Esc-Taste, um zu Ihren Dateien zurückzukehren.
