So laden Sie eine Datei herunter:

  1. Klicken Sie auf , und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, klicken Sie auf .

  2. Markieren Sie die Datei, die Sie herunterladen möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Herunterladen aus.

  4. Navigieren Sie auf Ihrem BlackBerry-Smartphone zum Zielordner.

  5. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hier herunterladen aus.
