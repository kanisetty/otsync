So ändern Sie die OpenText Tempo-Einstellungen:

  1. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Einstellungen aus.

  2. Optional Markieren Sie die Speicher, drücken Sie die BlackBerry Trackpad-Taste, und stellen Sie den Speicherplatz ein.

  3. Optional Markieren Sie die Liste Favoriten synchronisieren, drücken Sie die BlackBerry Trackpad-Taste, und wählen Sie Manuell oder Automatisch aus.

    Bei automatischer Synchronisierung geschieht Folgendes:

    • Die Datei wird für die Offlineanzeige heruntergeladen, wenn Sie die Datei als Favorit markieren.

    • Neue Versionen von Favoritendateien werden im Hintergrund heruntergeladen.

    Bei manueller Synchronisierung geschieht Folgendes:

    • Sie müssen das Dokument öffnen oder Ihre Favoriten synchronisieren, um die Datei für die Offlineanzeige herunterzuladen.

      So synchronisieren Sie Ihre Favoriten manuell:

      1. Klicken Sie auf .

      2. Drücken Sie die BlackBerry Menü-Taste.

      3. Wählen Sie Favoriten synchronisieren.

    • Neue Versionen von Favoritendateien werden nicht automatisch heruntergeladen.

  4. Optional Markieren Sie die Liste Standardsortierreihenfolge, drücken Sie die BlackBerry Trackpad-Taste, und wählen Sie Datum oder Name aus.

  5. Drücken Sie die BlackBerry Esc-Taste, und wählen Sie Speichern aus.
