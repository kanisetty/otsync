Para subir un archivo guardado en su BlackBerry:

  1. Pulse Archivos y desplácese hasta la carpeta destino a la que quiere subir el archivo.

  2. Pulse la tecla BlackBerry Menú y seleccione Subir aquí.

  3. Desplácese hasta el documento destino de su smartphone BlackBerry.

  4. Pulse la tecla BlackBerry Menú y seleccione Subir.

Para importar un archivo desde una aplicación BlackBerry:

  1. Abra la aplicaciónBlackBerry. Por ejemplo, la aplicación Cámara.

  2. Pulse la tecla BlackBerry Menú y seleccione Añadir a OpenText Tempo.

  3. Opcional Si se le solicita, conéctese a OpenText Tempo. Para obtener más información, consulte Para conectarse.

  4. Desplácese hasta la ubicación destino en la que desea colocar el archivo.

  5. Pulse la tecla BlackBerry Menú y seleccione Subir aquí.
