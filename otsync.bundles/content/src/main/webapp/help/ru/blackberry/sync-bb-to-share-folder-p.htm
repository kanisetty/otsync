Открытие общего доступа для папки

  1. Нажмите Файлы и перейдите к целевой папке.

  2. Выделите папку.

  3. Нажмите клавишу BlackBerry Меню и выберите Включить общий доступ к папке.

     

    Замечание:
    Если параметр Включить общий доступ к папке недоступен, разрешения на предоставление общего доступа к папке отсутствуют. Список папок, к которым нельзя открыть общий доступ см. в Исключения при открытии общего доступа к папкам.

     

  4. Введите имя пользователя, которому необходимо предоставить общий доступ к папке.

  5. Выделите имя пользователя и нажмите кнопку BlackBerry Сенсорная панель. Будет отправлено приглашение на открытие общего доступа.

    Папка теперь помечена как общая значком .

Исключения при открытии общего доступа к папкам

В папке OpenText Tempo можно открыть общий доступ ко всем папкам, за исключением следующих.

  • Нельзя открыть общий доступ к папке, которая является дочерней или родительской для общей папки.

  • Нельзя открыть общий доступ к папке, к которой вам предоставлен общий доступ.

  • Нельзя открыть общий доступ к корню общей папки, если вы не являетесь владельцем папки.

  • Нельзя открыть общий доступ к папке OpenText Tempo.

  • Внешние пользователи, которые не являются сотрудниками компании, не могут открывать общий доступ ни к одной из папок.

Открытие доступа к папке для внешних пользователей

Администратор может предоставить привилегию для предоставления общего доступа пользователям за пределами компании. Для получения этой привилегии свяжитесь с администратором. Дополнительные сведения об открытии общего доступа к папке для внешних пользователей см. в Открытие общего доступа к папке для внешнего пользователя.

 

Предостережение
При открытии общего доступа для другого пользователя для совместной работы этот пользователь получает полный доступ к содержимому папки. После внесения исправлений в общую папку изменения автоматически синхронизируются со всеми пользователями, которым предоставлен общий доступ к папке, включая владельца папки.
