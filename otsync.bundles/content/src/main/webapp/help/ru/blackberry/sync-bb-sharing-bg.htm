Открытие общего доступа для папки

  1. Щелкните и перейдите к целевой папке. При открытии общего доступа к папке она может находится в любом местоположении в папке OpenText Tempo. Чтобы добавить новую папку для открытия общего доступа, нажмите клавишу BlackBerry Меню и выберите Add Folder (Добавить папку).

  2. Выделите папку.

  3. Нажмите клавишу BlackBerry Меню и выберите Включить общий доступ к папке.

     

    Замечание:
    Если параметр Включить общий доступ к папке недоступен, разрешения на предоставление общего доступа к папке отсутствуют. Список папок, к которым нельзя открыть общий доступ, см. ниже в разделе Исключения при открытии общего доступа к папкам.

     

  4. По выбору При открытии общего доступа к папке пользователю с общим доступом можно предоставить разрешение на чтение/запись или только на чтение папки. По умолчанию пользователям предоставляется разрешение на чтение/запись, которое определяется .

    Чтобы предоставить пользователю разрешение только на чтение, выполните следующие действия.

    1. Выделите .

    2. Нажмите клавишу BlackBerry Сенсорная панель.

    3. Значок изменится на .

    С разрешениями на чтение/запись все пользователи с общим доступом, включая внешних пользователей, получают полный доступ к содержимому папки. Они могут добавлять, изменять, копировать, переименовывать и удалять файлы и папки. После внесения исправлений в общую папку изменения автоматически синхронизируются со всеми пользователями, которым предоставлен общий доступ к папке, включая владельца папки.

    С разрешением только на чтение пользователь с общим доступом может только читать содержимое общей папки.

  5. Введите имя пользователя, которому необходимо предоставить общий доступ к папке.

    Чтобы открыть общий доступ к папке для внешнего пользователя, введите его адрес электронной почты. Внешние пользователи не являются сотрудниками компании. После ввода существующего адреса электронной почты в поле автозаполнения отобразится адрес электронной почты внешнего пользователя.

     

    Замечание:
    Если адрес электронной почты пропущен и отображается сообщение “Результаты не найдены”, разрешения на предоставление внешним пользователям общего доступа к папке отсутствуют. Для получения этой привилегии свяжитесь с администратором.

     

  6. Выделите имя пользователя или адрес электронной почты, и нажмите кнопку BlackBerry Сенсорная панель. Будет отправлено приглашение на открытие общего доступа. Пользователи с общим доступом должны принять эти приглашения для получения доступа к содержимому папки.

    Теперь папка помечена как общая. Несмотря на разрешения, предоставляемые для чтения/записи или только для чтения, общая папка отобразится как .

Исключения при открытии общего доступа к папкам

В папке OpenText Tempo можно открыть общий доступ ко всем папкам, за исключением следующих.

  • Папка, которая является дочерней или родительской для общей папки.

  • Папка, к которой открыт общий доступ.

  • Папка OpenText Tempo.

  • Внешние пользователи не могут открывать общий доступ ни к одной из папок.

Дополнительные сведения об управлении общими папками см. в разделе Sharing a Folder and Managing Shared Folders (Открытие доступа к папке и управление общими папками).
