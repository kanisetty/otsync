Добавление фотографии с камеры BlackBerry

  1. Щелкните и перейдите к целевой папке, в которую необходимо добавить фотографию.

     

    Замечание:
    Не удастся добавить фотографию в общую папку, к которой имеется разрешение только на чтение, определенное .

     

  2. Нажмите клавишу BlackBerry Меню и выберите Добавить фотографию.

  3. Нажмите клавишу BlackBerry Сенсорная панель, чтобы сделать фотографию.

  4. Щелкните Add toOpenText Tempo (Добавить к). Фотографии сохраняются в формате JPG. Невозможно изменить имя фотографии до загрузки.

Чтобы добавить фотографию из приложения BlackBerry, выполните следующие действия.

  1. Запустите приложение камеры BlackBerry.

  2. Перейдите к фотографии, которую необходимо импортировать.

  3. Нажмите клавишу BlackBerry Меню и выберите Добавить в OpenText Tempo.

  4. По выбору При отображении соответствующего запроса войдите в OpenText Tempo. Подробную информацию см. в Вход и управление профилем.

  5. Перейдите в целевое местоположение, в которое необходимо поместить файл.

     

    Замечание:
    Не удастся добавить фотографию в общую папку, к которой имеется разрешение только на чтение, определенное .

     

  6. Нажмите клавишу BlackBerry Меню и выберите Передать на эту страницу.

Добавление папки

  1. Щелкните и перейдите к целевому местоположению новой папки.

     

    Замечание:
    Не удастся добавить папку в общую папку, к которой имеется разрешение только на чтение, определенное .

     

  2. Нажмите клавишу BlackBerry Меню и выберите Добавить папку.

  3. В поле Имя папки введите имя для этой папки.

  4. Нажмите клавишу BlackBerry Ввод.
