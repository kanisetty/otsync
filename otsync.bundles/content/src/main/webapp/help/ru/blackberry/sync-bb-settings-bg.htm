Изменение параметров OpenText Tempo.

  1. Нажмите клавишу BlackBerry Меню и выберите Настройки.

  2. По выбору Выделите список Хранение, нажмите клавишу BlackBerry Сенсорная панель и настройте пространство хранения.

  3. По выбору Выделите список Синхронизировать элементы избранного, нажмите клавишу BlackBerry Сенсорная панель и выберите Вручную или Автоматически.

    В режиме автоматической синхронизации происходит следующее:

    • файл загружается для просмотра в автономном режиме при пометке файла тегом как избранного;

    • новые версии избранных файлов загружаются в фоновом режиме.

    В режиме синхронизации вручную происходит следующее:

    • для загрузки файла в целях просмотра в автономном режиме необходимо открыть документ или вручную синхронизировать элементы избранного.

      Чтобы синхронизовать элементы избранного вручную, выполните следующие действия.

      1. Щелкните .

      2. Нажмите клавишу BlackBerry Меню.

      3. Выберите Синхронизировать элементы избранного.

    • Новые версии избранных файлов не загружаются автоматически.

  4. По выбору Выделите список Порядок сортировки по умолчанию, нажмите клавишу BlackBerry Сенсорная панель и выберите Дата или Имя.

  5. Нажмите клавишу BlackBerry Выход и выберите Сохранить.
