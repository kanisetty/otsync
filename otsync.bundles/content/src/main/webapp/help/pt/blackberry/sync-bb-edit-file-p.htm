Para editar um arquivo:

  1. Clique em Arquivos e navegue até o item de destino. Se o arquivo estiver marcado como favorito, clique em Favoritos.

  2. Destaque o arquivo que deseja editar.

  3. Pressione a tecla Trackpad do BlackBerry para exibir o arquivo.

  4. O arquivo é aberto em um aplicativo de terceiros. Se o aplicativo suporta edição, edite o arquivo e salve as alterações.

    As alterações feitas no arquivo não são salvas no arquivo na pasta do OpenText Tempo. É necessário fazer o upload do arquivo alterado do aplicativo de terceiros ou da pasta de cache do BlackBerry.

  5. No OpenText Tempo, carregue o arquivo editado adicionando uma versão do arquivo. Para obter detalhes, consulte Para adicionar uma nova versão de um arquivo.
