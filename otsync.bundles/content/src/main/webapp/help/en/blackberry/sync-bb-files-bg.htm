To view a file:

  1. Click and navigate to the target file by highlighting a folder and pressing the BlackBerry Trackpad key. The title bar displays the name of the folder you are currently viewing.

    If the target file is a tagged favorite, click .

  2. Optional To ensure you are viewing the most up-to-date content, you should periodically refresh the folder content. Press the BlackBerry Menu key and select Refresh. This refreshes the display, but does not download the files. To download a file, open it.

  3. Highlight the file you want to view.

  4. Press the BlackBerry Trackpad key to view the file.

    When viewing files in a shared folder to which you have read/write access, the files are not reserved. You can open and edit a file at the same time a collaborator is making changes to it. Read/write shared folders are identified by the overlay. For example, or .

    When viewing files in a shared folder to which you have read-only access, the files are reserved. You can view, but not edit files in these folders. Read-only shared folders are identified by the overlay. For example, or . When navigating in a read-only shared folder, the title bar displays an orange background and the files are marked with the overlay. For example, .

  5. Press the BlackBerry Escape key to return to your files.

To edit a file:

  1. Click and navigate to the target file.

    You cannot save edits to a file in a shared folder to which you have read-only access. Files in a read-only shared folder are marked with the overlay. For example, .

  2. Highlight the file you want to edit.

  3. Press the BlackBerry Trackpad key to view the file.

  4. The file opens in a third party app. If the app supports editing, edit the file and save your changes.

  5. The changes you make to the file are not saved in your Tempo Box folder. You need to upload the changed file from the third party app or from the BlackBerry cache folder.

    In OpenText Tempo Box, upload the edited file by adding a version of the file. For details, see Adding a File.

For information on managing your files, refer to Viewing, Editing, and Working With Your Files.
