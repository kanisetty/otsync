Per visualizzare elementi mentre ci si trova offline:

  1. Per rendere un elemento disponibile per la visualizzazione offline, contrassegnare il file come preferito mentre ci si trova online. Per ulteriori dettagli, vedere la sezione Aggiunta di un preferito. I file contrassegnati come preferiti sono indicati dall'overlay . Ad esempio, .

  2. Fare clic su Preferiti.

  3. Evidenziare l'elemento che si desidera visualizzare.

  4. Premere il tasto Trackpad BlackBerry per visualizzare l'elemento. I file aperti quando si è offline potrebbero non essere le versioni più recenti dei file nella cartella OpenText Tempo, poiché le modifiche apportate ai file non possono essere sincronizzate quando si è offline.

  5. Premere il tasto Esc BlackBerry per tornare ai file.
