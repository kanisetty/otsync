Per modificare le impostazioni di OpenText Tempo:

  1. Premere sul BlackBerry il tasto Menu e selezionare Impostazioni.

  2. Opzionale Evidenziare l'elenco Memoria, premere sul BlackBerry il tasto Trackpad e regolare lo spazio di archiviazione.

  3. Opzionale Evidenziare l'elenco Sincronizza Preferiti, premere sul BlackBerry il tasto Trackpad e selezionare Manuale o Auto.

    Se si utilizza la sincronizzazione automatica:

    • Quando il file viene contrassegnato come preferito, viene scaricato per la visualizzazione offline.

    • Nuove versioni dei file preferiti vengono scaricate in background.

  4. Opzionale Evidenziare l'elenco Ordinamento predefinito, premere sul BlackBerry il tasto Trackpad e selezionare Data o Nome.

  5. Premere sul BlackBerry il tasto Esc e selezionare Salva.
