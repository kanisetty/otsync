Per caricare un file memorizzato sul dispositivo BlackBerry:

  1. Fare clic su File e posizionarsi nella cartella di destinazione in cui si desidera caricare un file.

  2. Premere il tasto Menu BlackBerry e selezionare Carica qui.

  3. Spostarsi nella posizione del file desiderato sul proprio smartphone BlackBerry.

  4. Premere il tasto Menu BlackBerry e selezionare Carica.

Per importare un file da un'applicazione BlackBerry:

  1. Avviare un'applicazione BlackBerry. Ad esempio, l'applicazione Fotocamera.

  2. Premere il tasto Menu BlackBerry e selezionare Aggiungi a OpenText Tempo.

  3. Opzionale Se richiesto, effettuare l'accesso a OpenText Tempo. Per ulteriori dettagli, vedere la sezione Effettuare l'accesso.

  4. Spostarsi alla posizione dove si desidera posizionare file.

  5. Premere il tasto Menu BlackBerry e selezionare Carica qui.
