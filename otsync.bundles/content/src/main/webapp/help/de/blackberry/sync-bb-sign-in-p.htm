So melden Sie sich an:

  1. Starten Sie die Anwendung OpenText Tempo.

  2. Markieren Sie Ihr Profil, und drücken Sie die Taste BlackBerry Trackpad. Wenn Sie kein Profil erstellt haben, finden Sie weitere Informationen unter So erstellen Sie ein Profil.

  3. Wenn Sie dazu aufgefordert werden, führen Sie einen der folgenden Schritte durch:

    • Geben Sie Ihr Kennwort ein. Wenn Sie sich nicht an Ihr Kennwort erinnern können, wenden Sie sich an Ihren Administrator.

    • Wenn Sie sich außerhalb des Unternehmens befinden, geben Sie das Kennwort ein, das Sie beim Erstellen Ihres OpenText Tempo-Kontos ausgewählt haben. Falls Sie sich nicht an Ihr Kennwort erinnern können, finden Sie diese Informationen in der E-Mail, die Sie beim Erstellen Ihres OpenText Tempo-Kontos erhalten haben.

     

    Hinweis:
    Für das Kennwort muss Groß-/Kleinschreibung beachtet werden. Stellen Sie sicher, dass Sie das Kennwort korrekt eingeben.

     

  4. Optional Aktivieren Sie das Kontrollkästchen Angemeldet bleiben, falls dieses verfügbar ist.

     

    Hinweis:
    Dieses Kontrollkästchen ist nicht verfügbar, wenn das BlackBerry offline ist.

     

  5. Drücken Sie die BlackBerry Eingabe-Taste.

 

Achtung

  • Wenn Sie zulassen, dass Sie bei OpenText Tempo angemeldet bleiben, kann dies die Sicherheit beeinträchtigen, sollte Ihr BlackBerry von einer anderen Person verwendet werden.

  • Wenn Sie mehrere Profile auf Ihrem BlackBerry konfiguriert haben, können andere Profile auf Ihr OpenText Tempo-Konto zugreifen, wenn das Kontrollkästchen Angemeldet bleiben ausgewählt ist und Sie sich nicht abgemeldet haben. Weitere Informationen zum Abmelden von OpenText Tempo finden Sie unter So melden Sie sich ab.
