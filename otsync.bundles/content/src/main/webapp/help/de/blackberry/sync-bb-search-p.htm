So durchsuchen Sie Ihren OpenText Tempo-Ordner:

  1. Klicken Sie auf .

  2. Geben Sie den Suchbegriff ein.

  3. Drücken Sie die BlackBerry Eingabe-Taste. Ihre Suchergebnisse werden angezeigt.

  4. Markieren Sie die Datei oder den Ordner, die bzw. den Sie anzeigen möchten.

  5. Drücken Sie die BlackBerry Menü-Taste, um die verfügbaren Optionen anzuzeigen.

So öffnen Sie den Speicherort eines Elements aus den Suchergebnissen:

  1. Durchsuchen Sie den OpenText Tempo-Ordner.

  2. Markieren Sie die Datei, die Sie öffnen möchten, am Dateispeicherort.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Speicherort öffnen aus.
