Pour supprimer un dossier partagé qui vous appartient :

  1. Cliquez sur Fichiers, puis accédez au dossier cible.

  2. Mettez en surbrillance le dossier partagé.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Supprimer un dossier.

     

    Note :
    Cette action supprime le dossier pour vous et pour l'ensemble des collaborateurs. Il vous est possible de supprimer des collaborateurs. Pour plus d'informations, voir Suppression de collaborateurs.

    Si Supprimer un dossier n'est pas disponible, vous avez sélectionné un dossier qu'un utilisateur partage avec vous. Avant de poursuivre, voir Suppression d'un dossier partagé.

     

  4. Cliquez sur OK.

Pour supprimer un dossier qu'un utilisateur partage avec vous :

  1. Cliquez sur Fichiers.

  2. Mettez en surbrillance le dossier partagé duquel vous souhaitez vous dissocier.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Supprimer le partage.

     

    Note :
    Si Supprimer le partage n'est pas disponible, vous avez sélectionné un dossier qui vous appartient. Avant de poursuivre, voir Suppression d'un dossier partagé.

     

  4. Cliquez sur OK.

Suppression d'un dossier partagé

Les dossiers partagés sont signalés par deux avatars verts dans le coin inférieur droit du dossier, . Il existe deux types de dossiers partagés :

  • Un dossier qui vous appartient et que vous partagez avec vos collègues. Supprimer un dossier partagé qui vous appartient supprime le dossier pour vous comme pour tous les collaborateurs qui bénéficient d'un accès partagé au dossier. Vous pouvez également supprimer un ou l'ensemble des collaborateurs. Pour plus d'informations, voir Suppression de collaborateurs.

  • Un dossier qui ne vous appartient pas, mais qu'un utilisateur partage avec vous. Supprimer un dossier qu'un utilisateur partage avec vous vous dissocie du dossier partagé et le supprime de votre dossier OpenText Tempo. Cela ne supprime le dossier ni pour le propriétaire ni pour les autres collaborateurs.
