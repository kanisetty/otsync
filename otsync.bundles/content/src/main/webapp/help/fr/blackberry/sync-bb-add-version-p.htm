Pour ajouter une nouvelle version à l'aide d'un fichier sur votre BlackBerry :

  1. Cliquez sur , puis accédez au fichier cible. Si le fichier cible est marqué comme favori, cliquez sur .

  2. Mettez en surbrillance le fichier cible auquel vous souhaitez ajouter une nouvelle version.

     

    Note :
    Vous ne pouvez pas ajouter une version à un fichier dans un dossier partagé sur lequel vous disposez de l'autorisation en lecture seule, identifié par .

     

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Charger une nouvelle version.

  4. Accédez à la nouvelle version du fichier sur votre BlackBerry.

  5. Mettez le fichier en surbrillance.

  6. Appuyez sur la touche BlackBerry Menu et sélectionnez Téléchargement en amont. Le fichier est chargé automatiquement.

Pour charger un fichier stocké sur votre BlackBerry :

  1. Cliquez sur , puis accédez au dossier cible dans lequel vous souhaitez charger un fichier.

  2. Appuyez sur la touche BlackBerry Menu et sélectionnez Charger ici.

  3. Accédez au document cible sur votre smartphone BlackBerry.

     

    Note :
    Vous ne pouvez pas charger un fichier dans un dossier partagé sur lequel vous disposez de l'autorisation en lecture seule, identifié par .

     

  4. Appuyez sur la touche BlackBerry Menu et sélectionnez Téléchargement en amont.

Pour importer un fichier à partir d'une application BlackBerry :

  1. Lancez une application BlackBerry.

  2. Appuyez sur la touche BlackBerry Menu et sélectionnez Ajouter àOpenText Tempo.

  3. Facultatif Si vous y êtes invité, connectez-vous à OpenText Tempo. Pour plus d'informations, voir Connexion et gestion de votre profil.

  4. Accédez à l'emplacement dans lequel vous souhaitez importer le fichier.

     

    Note :
    Vous ne pouvez pas importer un fichier dans un dossier partagé sur lequel vous disposez de l'autorisation en lecture seule, identifié par .

     

  5. Appuyez sur la touche BlackBerry Menu et sélectionnez Charger ici.
