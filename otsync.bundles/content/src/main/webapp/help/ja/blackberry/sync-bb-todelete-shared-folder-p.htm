所有している共有フォルダーを削除するには、次の操作を行います。

  1. [ファイル] をクリックし、ターゲット フォルダーに移動します。

  2. 共有フォルダーをハイライト表示します。

  3. BlackBerry [メニュー] キーを押し、[フォルダーの削除] を選択します。

     

    注意:
    これで、自分自身またはすべての共同作業者のフォルダーが削除されます。また、共有作業者を削除することもできます。詳細については共同作業者を削除するにはを参照してください。

    [フォルダーの削除] が使用できない場合、共有されているフォルダーを選択しています。先に進む前に、共有フォルダーの削除を参照してください。

     

  4. [OK] をクリックします。

共有しているフォルダーを削除するには、次の操作を行います。

  1. [ファイル] をクリックします。

  2. 接続を切断する共有フォルダーをハイライト表示します。

  3. BlackBerry [メニュー] キーを押し、[共有の削除] を選択します。

     

    注意:
    [共有の削除] が使用できない場合、所有しているフォルダーを選択しています。先に進む前に、共有フォルダーの削除を参照してください。

     

  4. [OK] をクリックします。

共有フォルダーの削除

共有フォルダーは、フォルダーの右下の 2 つの緑色のアバター ( ) で示されます。共有フォルダーには次の 2 つの種類があります。

  • 2 種類の共有フォルダーとは、自分で所有しているフォルダーと、同僚と共有するフォルダーです。所有している共有フォルダーを削除すると、所有者自身およびそのフォルダーへの共有アクセス権を持っているすべての共同作業者に対してそのフォルダーが削除されます。また、1 人またはすべての共有作業者を削除することもできます。詳細については、共同作業者を削除するにはを参照してください。

  • 自分で所有していない共有フォルダー。共有しているフォルダーを削除すると、共有フォルダーへの接続が解除され、OpenText Tempo フォルダーからこの共有フォルダーが削除されます。これによって、所有者または他の共同作業者のフォルダーが削除されることはありません。
