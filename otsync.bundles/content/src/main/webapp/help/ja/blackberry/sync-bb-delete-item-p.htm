ファイルまたはフォルダーを削除するには、次の操作を行います。

  1. をクリックし、ターゲット ファイルに移動します。ターゲット ファイルにお気に入りタグが付いている場合、 をクリックします。

     

    注意:
    操作を続ける前に、下記の注意情報を参照してください。

    が表示されている、読み取り専用権限を持つ共有フォルダーのファイルは削除できません。

     

  2. ターゲットのファイルをハイライト表示します。

  3. BlackBerry [メニュー] キーを押し、[アイテムの削除] を選択します。

  4. [OK] をクリックして、操作を確認します。

フォルダーを削除するには、次の操作を行います。

  1. をクリックし、ターゲット フォルダ－に移動します。

     

    注意:
    操作を続ける前に、下記の注意情報を参照してください。

    ターゲット フォルダーが共有フォルダーの場合は、操作を続ける前に共有フォルダーの削除を参照してください。共有フォルダーには、 または が表示されます。

     

  2. フォルダーをハイライト表示します。

  3. BlackBerry [メニュー] キーを押し、[フォルダーの削除] を選択します。

  4. [OK] をクリックして、操作を確認します。

 

注意

  • ファイルまたはフォルダーを削除すると、お気に入りとしてタグを付けられている場合でも、OpenText Tempo フォルダーから削除されます。つまり、ファイルまたはフォルダーは、インストールした他のすべての OpenText Tempo クライアント (デスクトップ クライアント)、その他のスマートフォン、タブレット、および OpenText Tempo Web ユーザー インターフェイスから削除されます。

  • が表示されている、読み取り/書き込み権限を持つ共有フォルダーからファイルまたはフォルダーを削除した場合、共有フォルダーの所有者を含めて、フォルダーへの共有アクセス権を持っているすべての共同作業者に対して、そのファイルまたはフォルダーが自動的に削除されます。
