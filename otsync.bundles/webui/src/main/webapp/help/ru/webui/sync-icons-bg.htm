OpenText Tempo Box: Пользовательская справка по веб-клиентам

   

Глава 1 – Начало работы с веб-клиентом

Веб-клиент OpenText Tempo Box позволяет получить доступ к содержимому своей Tempo Box папки с использованием браузера. Можно использовать веб-клиент для управления элементами и папками в своей папке Tempo Box, а также для открытия общего доступа к папкам и ответа на приглашения об открытии общего доступа от других пользователей.

Для просмотра папки Tempo Box щелкните имя папки, чтобы перейти по структуре каталогов. Щелкните ссылку в полосе иерархической навигации, чтобы вернуться назад, или щелкните Все файлы на левой панели для перехода в корень папки Tempo Box.

После выбора папки имя папки, просматриваемой в настоящее время, наряду с владельцем папки, появляется на правой панели. При выборе общей папки на правой панели появляется также список пользователей с общим доступом. В следующем примере можно видеть именованную общую папку с именем Шаблоны где имя владельца папки – Иван Куратов. Эта папка имеет трех пользователей с общим к ней доступом. Дополнительные сведения об общих папках см. в разделе Общий доступ к папкам и ответ на приглашение к совместному использованию.

Папки с несколькими пользователями с общим доступом имеют один из следующих значков.
