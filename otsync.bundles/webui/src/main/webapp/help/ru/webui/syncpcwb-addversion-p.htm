Добавление версии элемента.

  1. Перейдите к местоположению элемента.

    Вы не можете добавить версию к элементу в папке, к которой имеется доступ только для чтения. Папки, допускающие только чтение, обозначаются изображением замка. Например, или .

  2. Направьте курсор мыши на меню функций элемента и щелкните Add Version.

    Нельзя добавить версию в файл, зарезервированный другим пользователем. Зарезервированные файлы помечены наложением . Например, . Чтобы определить, кто зарезервировал файл, можно просмотреть журнал файла.

     

    Важно:
    При добавлении версии элемента к файлу в общей папке, к которой имеется доступ для чтения-записи, файл не резервируется автоматически. Может оказаться, что вы загружаете и редактируете файл в то же время, когда пользователь с общим доступом вносит в него изменения. Папки для чтения-записи обозначаются или . Чтобы это предотвратить, можно зарезервировать файл .

     
     

    Предостережение
    При редактировании содержимого в папке, к которой имеется доступ для чтения-записи, изменения синхронизируются для всех пользователей, имеющих доступ к папке.

    После редактирования содержимого в папке Enterprise изменения применяются к папке в рабочей области предприятия. Папки Enterprise можно опознать по значку .

     

  3. Выполните одно из следующих действий.

    • Если отображается кнопка Browse, щелкните Обзор, выберите элемент, затем щелкните Добавить версию.

    • Если отображается диалоговое окно File Upload, выберите элемент и щелкните Open.
