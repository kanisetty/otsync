Включение уведомлений по электронной почте

Включение использования уведомлений по электронной почте:

  1. На панели вверху справа щелкните свое имя, щелкните Tempo Box, затем щелкните Настройки.

  2. Под элементом Отправить себе письмо выберите один или оба из следующих флажков:

    When I receive a Sharing Request (При получении запроса на предоставление общего доступа)

    Получать электронную почту, если другой пользователь приглашает открыть общий доступ к папке с ним.

    При внесении другим пользователем изменений в любую общую папку

    Получение уведомлений по эл. почте при добавлении, изменении или удалении другим пользователем элемента в общей папке, участником которой вы являетесь. Сообщения эл. почты являются периодическими сводками, которые отправляются в соответствии с установленным администратором расписанием. Данная настройка применяется глобально для всех ваших общих папок. Если эта настройка не включена, можно выбрать получение уведомлений по эл. почте для отдельных общих папок. Дополнительные сведения см. в разделе Включение уведомлений электронной почты для общей папки.

    Для доступности этого варианта администратор должен включить использование уведомлений по эл. почте.

  3. Щелкните Save Settings. Сообщения эл. почты будут отправляться на адрес, указанный в поле E-mail в профиле Content Server. Если вы не получаете уведомления, обратитесь к своему администратору. Для внешнего пользователя электронная почта отправляется по адресу электронной почты, используемому при авторизации в Tempo Box.
