Пользовательский интерфейс

Этот формат предназначен для элементов графического интерфейса пользователя (GUI), таких как кнопки, названия значков, элементы меню и поля.

Filenames, commands и пример данных

Этот формат используется для имен файлов, путей, URL-адресов и команд в командных строках. Он также предназначен для примера данных, текста, который необходимо ввести в текстовое поле, и для других литералов.

 

Замечание:
При копировании примеров командной строки из PDF необходимо помнить, что в PDF может содержаться скрытый текст. OpenText рекомендует выполнять копирование из HTML-версии документа при ее наличии.

 
Комбинации клавиш

Комбинации клавиш отображаются в ALL CAPS, например:
Нажмите CTRL+V.

<Имя переменной>

Угловые скобки < > используются для обозначения переменной или метки-заполнителя. Пользователь заменяет скобки и описание соответствующим значением. Например, <server_name> заменяется на serv01.

Внутренние перекрестные ссылки

Нажмите перекрестную ссылку, чтобы перейти к конечному объекту ссылки текущего документа.

Внешние перекрестные ссылки

Внешними перекрестными ссылками, как правило, являются текстовые ссылки на другие документы. Однако, если имеется документ в формате HTML, например, в Knowledge Center, в качестве внешних перекрестных ссылок могут выступать активные ссылки на определенный раздел указанного документа.

Замечание:
