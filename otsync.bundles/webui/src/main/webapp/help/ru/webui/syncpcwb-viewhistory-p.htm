Просмотр журнала элементов или папок

Просмотр журнала элемента или папки:

  1. Направьте курсор мыши на меню функций элемента и щелкните Журнал.

  2. При просмотре журнала элемента щелкните одну из следующих вкладок:

    Журнал аудита

    Отображается каждое действие, выполненное над элементом.

    История версий

    Отображается список всех предыдущих версий элемента. Для просмотра предыдущей версии щелкните Document Name. Папки не поддерживают версий и лишь отображают Журнал аудита.

     

    Замечание:
    Содержимое вкладки Журнал аудита зависит от типов событий, для которых администратор включил поддержку аудита.
