Глава 1 – Начало работы с веб-клиентом

Веб-клиент позволяет получить доступ к содержимому сетевой папки OpenText Tempo на сервере OpenText Tempo. Веб-клиент можно использовать для управления элементами и папками в папке OpenText Tempo, а также для предоставления общего доступа к папкам и ответа на приглашения на открытие общего доступа от других пользователей.

Доступ к веб-клиенту можно получить одним из следующих способов:

  • Выбрав команду в контекстном меню OpenText Tempo > Включить режим общего доступа к этой папке на клиенте для настольных ПК OpenText Tempo.

  • Щелкнув правой кнопкой мыши OpenText Tempo значок клиента для настольных ПК в области уведомлений и выбрав Запустить веб-интерфейс.

Ссылки в верхней части веб-клиента можно использовать для выполнения следующих задач:

  • Обновление фотографии, которая отображается для других пользователей, просматривающих папки, для которых у них открыт общий доступ с вами.

  • Переключение на Content Server начальную страницу по умолчанию, если на OpenText Tempo настроена поддержка этой функции. Эта функция недоступна для внешних пользователей.

  • Если вы являетесь внешним пользователем, измените пароль, указанный при первой регистрации учетной записи OpenText Tempo.

  • Просмотр сведений о версии.

  • Выход из веб-клиента.
