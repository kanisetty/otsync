Включение уведомлений электронной почты для общей папки

Чтобы включить уведомления по эл. почте для общей папки, выполните следующие действия.

  1. Нажмите общую папку, владельцем которой вы не являетесь.

  2. На панели справа установите флажок Получать уведомления папки.

 

Примечания:

  • Уведомления по электронной почте отправляются по адресу, указанному в поле Электронная почта вашего профиля Content Server. Если вы не получаете уведомления, обратитесь к своему администратору. Для внешних пользователей электронная почта отправляется по адресу электронной почты, который используется при авторизации в своей учетной записи Tempo Box.

  • Этот параметр доступен, только если администратор включил уведомления по электронной почте, а пользователь отключил глобальный параметр получения уведомлений по электронной почте. Дополнительные сведения об этом глобальном параметре см. в разделе Включение уведомлений по электронной почте.
