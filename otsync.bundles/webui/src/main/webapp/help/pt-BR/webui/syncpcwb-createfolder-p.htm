Criando uma pasta

Para criar uma pasta:

  1. Navegue até o local onde você deseja criar uma pasta.

    Não é possível criar uma pasta em uma pasta para a qual você tem acesso somente leitura. Pastas somente leitura são identificadas pela camada de sobreposição de bloqueio. Por exemplo, ou .

  2. No painel central, clique em Nova pasta.

  3. Digite o nome de uma pasta na caixa e depois clique em Criar.
