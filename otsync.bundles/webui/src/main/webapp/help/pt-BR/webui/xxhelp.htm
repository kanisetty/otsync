Trabalho com a documentação

Utilize os seguintes elementos de navegação para encontrar as informações que você necessita:

  • Sumário
  • Pesquisar
  • Índice
  • Área de navegação
  • Teclado
  • Opções Copiar atalho ou Copiar localização do link
 

Nota:
Não é necessário JavaScript para acessar a documentação online. Entretanto, OpenText recomenda o uso do JavaScript para acessar os recursos de navegação avançados da documentação online. Se estiver usando uma versão mais antiga do navegador da Web, os recursos destacados e de reconfiguração podem não estar disponíveis no sumário.

 

Sumário

É possível definir quantos níveis de navegação você deseja visualizar no sumário. Clique em uma das guias para ajustar os níveis exibidos. Por exemplo, o exibe somente as seções principais e o exibe todas as seções até o quinto nível. O máximo é de cinco níveis. O sumário se ajusta automaticamente ao nível do tópico selecionado.

Pesquisar

Por padrão, a guia permite executar uma pesquisa do texto integral no documento inteiro ou na página atual selecionando somente a caixa de diálogo próxima ao botão Pesquisar. Os resultados são listados abaixo do botão Pesquisar.

Índice

A guia exibe o índice, se houver algum disponível.

Área de navegação

Cada página contém duas áreas de navegação: uma na parte superior e a outra na parte inferior. É possível navegar pela página anterior e pela próxima página do documento clicando em ou em na área superior ou inferior. Estas setas permitem que você navegue página a página do documento como se fosse um livro. A área de navegação superior disponibiliza as ferramentas de navegação adicionais:

  • Na parte superior, é possível clicar no título do documento para retornar à primeira seção do documento.

  • Abaixo do título, é possível visualizar o caminho de navegação para a página atual do documento. Clique em um título do caminho de navegação para ir àquela seção.

  • Á esquerda do caminho de navegação, os botões e navegam para a seção anterior ou para a próxima seção do documento. Clique no botão para ir imediatamente para o sumário da seção. Aponte para um desse botões sem clicar neles para visualizar o título da seção vinculada ao botão.

Teclado

É possível usar o teclado para navegar pelo documento no nível do documento ou da página:

  • Ao abrir um documento, você está no nível do documento. É possível usar as teclas Page up/down ou as setas Para baixo/cima para ir para a próxima página ou voltar para a anterior. Também é possível usar as teclar numéricas (1,2,3 e assim por diante) para alterar para outro nível do sumário ou a tecla i para exibir a página de índice.

  • Ao clicar em uma página, você geralmente está no nível da página. É possível usar as setas Para cima/baixo para rolar para cima ou para baixo da página exibida. Pressione a tecla ESC para retornar ao nível do documento, se desejar navegar pelas páginas do documento.

Opções Copiar atalho ou Copiar localização do link

Para criar uma referência para uma página específica no documento, clique com o botão direito no sumário e clique em Copiar atalho ou Copiar localização do link. Cole a URL no documento, email ou navegador da Web. Se o JavaScript estiver habilitado no navegador da Web, ele abrirá automaticamente a página dentro da estrutura do manual completo.
