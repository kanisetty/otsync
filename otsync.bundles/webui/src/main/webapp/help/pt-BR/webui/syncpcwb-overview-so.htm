Capítulo 1 – Introdução ao cliente da Web

O cliente da Web permite acessar o conteúdo da pasta on-line do OpenText Tempo no servidor do OpenText Tempo. Você pode usar o cliente da Web para gerenciar os itens e as pastas na pasta do OpenText Tempo, bem como compartilhar pastas e responder a convites de compartilhamento de outros usuários.

Você pode acessar o cliente da Web de uma destas formas:

  • Clicando no comando do menu contextual OpenText Tempo > Compartilhar esta pasta em uma pasta no cliente de desktop na pasta do OpenText Tempo.

  • Clicando com o botão direito do mouse no ícone do cliente de desktop do OpenText Tempo na área de notificação e clicando em Iniciar interface da Web.

É possível usar os links que aparecem na parte superior do cliente da Web para seguir o procedimento:

  • Atualize a foto que exibida para os outros usuários quando as pastas visualizadas são compartilhadas com você.

  • Troque para a página inicial padrão do Content Server, se o OpenText Tempo estiver configurado para suporte. Esta opção não está disponível para usuário externos.

  • Se você for um usuário externo, altera a senha que você especificou quando registrou sua conta do OpenText Tempo pela primeira vez.

  • Visualizar informações da versão.

  • Fazer logout do cliente Web.
