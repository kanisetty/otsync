第4章. 設定の指定

右上の画面で自分の名前をクリックすると、以下の操作を行うためのメニューが表示されます。

  • 自分に対して共有が設定されているフォルダーを表示するときに他のユーザーが見る写真を更新します。

  • 共有フォルダーの招待を受けたときや、共有フォルダーが更新されたときの OpenText Tempo Box の通知を設定します。電子メールの通知を受け取るには、管理者にその機能を有効にしてもらう必要があります。

  • バージョン情報を表示します。

  • Web クライアントからサインアウトします。
