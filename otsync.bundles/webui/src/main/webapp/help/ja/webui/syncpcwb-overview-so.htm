1. Web クライアントについて

第1章. Web クライアントについて

Web クライアントを使用すると、OpenText Tempo サーバーの [OpenText Tempo] フォルダーのコンテンツにアクセスできます。Web クライアントを使用して、[OpenText Tempo] フォルダー内のアイテムおよびフォルダーを管理したり、フォルダーを共有したり、他のユーザーからの共有の招待に応答したりできます。

次のいずれかの方法で Web クライアントにアクセスできます。

  • デスクトップ クライアントの OpenText Tempo フォルダー内のフォルダーで [OpenText Tempo > このフォルダーを共有] コンテキスト メニュー コマンドをクリックする。

  • 通知領域で OpenText Tempo デスクトップ クライアント アイコン を右クリックし、[Web インタフェースの起動] をクリックする。

Web クライアントの一番上に表示される [ようこそ] メニューのコマンドを使用することで、次の操作を実行できます。

  • 共有しているフォルダーを表示するときに他のユーザーが見る写真の更新。

  • Content Server のデフォルトのスタート ページへの切り替え (OpenText Tempo がサポートすることにより設定されている場合)。外部ユーザーは、このオプションを利用できません。

  • 外部ユーザーの場合、OpenText Tempo アカウントを最初に登録したときに指定したパスワードを変更します。

  • バージョン情報を表示します。

  • Web クライアントからサインアウトします。
