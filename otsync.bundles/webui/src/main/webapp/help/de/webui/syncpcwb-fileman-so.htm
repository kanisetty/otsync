OpenText Tempo Box: Benutzerhilfe für Web-Client

   

Dateien und Ordner verwalten

Mit Hilfe des Web-Client können Sie Objekte und Ordner hinzufügen, kopieren und an andere Speicherorte in Ihrem Tempo Box-Ordner verschieben. Außerdem können Sie innerhalb Ihres Ordners den Versions- und Protokollverlauf anzeigen bzw. Objekte und Ordner umbenennen oder löschen. Sämtliche Änderungen, die Sie vornehmen, werden automatisch mit den Tempo Box-Ordnern auf allen anderen Clients synchronisiert, die online sind.

 

Hinweise:

  • Sie können den Web-Client jedoch nicht zum Bearbeiten von Dateien in Ihrem Tempo Box-Ordner verwenden. Zum Aktualisieren einer Datei mithilfe des Web-Clients müssen Sie sie auf Ihren Desktop herunterladen, dort bearbeiten und dann als neue Version wieder hinzufügen .

  • Die Ergebnisse beim Hinzufügen, Umbenennen oder Löschen des freigegebenen Ordners variieren je nachdem, ob Sie dessen Eigentümer sind. Weitere Informationen finden Sie unter Freigegebene Ordner verschieben, umbenennen und löschen .

 
 

Wichtig:
Wenn Sie Dateien in einem freigegebenen Ordner anzeigen, auf den Sie Schreib-/Lesezugriff haben, werden die Dateien nicht automatisch reserviert. Sie können eine Datei herunterladen und bearbeiten, während ein Teilnehmer Änderungen daran vornimmt. Um dies zu verhindern, können Sie die Datei reservieren. Ordner mit Lese- und Schreibzugriff sind durch oder .

 

Ihr Administrator kann für Ihren Tempo Box-Ordner eine Größenbeschränkung festlegen. Wenn die Größe aller Objekte in Ihrem Ordner, außer Objekte in Ordnern, die für Sie freigegeben wurden, die Grenze überschreitet, müssen Sie Objekte löschen oder aus Ihrem Tempo Box-Ordner verschieben, bis Sie sich wieder unter dem Maximum befinden. Weitere Informationen finden Sie unter So überprüfen Sie die Größe Ihres Tempo Box-Ordners:.

Auf den Fotos und Bildern in Ihrem Tempo Box-Ordner ist eine Miniaturansicht des Bilds zu sehen. Um eine eine größere Darstellung zu sehen, können Sie auf die Miniaturansicht wechseln oder das Bild auswählen.

Sie können auch Objekte und Ordner in Content Server veröffentlichen. Diese Option ist für externe Benutzer nicht verfügbar. Wenn Sie einen Unternehmensordner abonniert haben und auf diesen Lese- und Schreibzugriff haben, können Sie direkt mit den Objekten in diesem Ordner interagieren. Weitere Informationen zum Hinzufügen eines Unternehmensordner zu Ihrem Tempo Box-Ordner finden Sie unter OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).
