OpenText Tempo Box: Benutzerhilfe für Web-Client

   

Erste Schritte mit dem Web-Client

Mit dem OpenText Tempo Box Web-Client können Sie über einen Browser auf die Inhalte Ihres Tempo Box-Ordners zugreifen. Mit dem Web-Client können Sie die Objekte und Objekte in Ihrem Tempo Box-Ordner verwaltenund Ordner freigeben und auf Freigabeeinladungen von anderen Benutzern antworten.

Um Ihren Tempo Box-Ordner zu durchsuchen, klicken Sie auf einen Ordnernamen, um durch die Struktur zu navigieren. Klicken Sie auf einen Link im "Breadcrumb-Trail", um wieder zurückzukehren, oder klicken Sie im linken Fensterbereich auf Alle Dateien, um in das Stammverzeichnis Ihres Tempo Box-Ordners zurückzukehren.

Wenn Sie einen Ordner auswählen, werden der Name des gerade angezeigten Ordners und sein Eigentümer im rechten Bereich angezeigt. Wenn Sie einen freigegebenen Ordner auswählen, erscheint im rechten Bereich außerdem eine Liste der Teilnehmer. Im folgenden Beispiel sehen Sie einen Ordner mit dem Namen Vorlagen , dessen Eigentümer Peter Day heißt. Für diesen Ordner gibt es drei Teilnehmer. Weitere Informationen zu freigegebenen Ordnern finden Sie unter Ordner freigeben und auf eine Freigabeeinladung antworten.

Ordner mit mehreren Teilnehmern haben eines der folgenden Symbole.

Symbole und Überlagerungen von freigegebenen Ordnern in OpenText Tempo Box

Symbol Beschreibung
Dieses Ordnersymbol deutet auf einen freigegebenen Ordner hin, für den Sie Lese-/Schreibzugriff haben. Ist der Ordnername um einen Benutzernamen ergänzt, wurde er für Sie freigegeben. Beispiel: Marketingmaterialien (jsmith). Weitere Informationen finden Sie unter Ordner freigeben und auf eine Freigabeeinladung antworten.
Eine Schlossüberlagerung steht für einen freigegebenen Ordner, für den Sie nur Lesezugriff haben. Weitere Informationen finden Sie unter Ordner freigeben und auf eine Freigabeeinladung antworten.
Dieses Ordnersymbol deutet auf einen Unternehmensordner hin, für den Sie Lese-/Schreibzugriff haben. Sie müssen sich beim Unternehmensordner in Content Server anmelden, oder eine Person mit der Berechtigung Berechtigungen bearbeiten für den Ordner muss Ihnen eine Freigabeeinladung senden, damit Sie auf den Inhalt aus Ihrem Tempo Box-Ordner zugreifen können. Weitere Informationen finden Sie unter Ordner freigeben und auf eine Freigabeeinladung antworten.
Eine Schlossüberlagerung auf diesem Ordnersymbol repräsentiert einen Unternehmensordner, für den Sie nur Lesezugriff haben. Sie müssen sich beim Unternehmensordner in Content Server anmelden, oder eine Person mit der Berechtigung Berechtigungen bearbeiten für den Ordner muss Ihnen eine Freigabeeinladung senden, damit Sie auf den Inhalt aus Ihrem Tempo Box-Ordner zugreifen können. Weitere Informationen finden Sie unter Ordner freigeben und auf eine Freigabeeinladung antworten.
Dieses Ordnersymbol steht für eine Dateibibliothek in einer Tempo Social-Community, für die Sie Lese-/Schreibzugriff haben. Sie müssen Mitglied einer Tempo Social-Community sein, um auf die Dateibibliotheken aus Ihrem Tempo Box-Ordner zugreifen zu können. Weitere Informationen zu Tempo Social-Communitys finden Sie unter OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social ist ein separates Produkt von OpenText Tempo Box. Wenn Tempo Social installiert ist und Sie die Dateibibliotheken in Ihrem Tempo Box-Ordner nicht sehen, kontaktieren Sie Ihren Administrator.
Eine Schlossüberlagerung auf diesem Ordnersymbol steht für eine Dateibibliothek in einer Tempo Social-Community, für die Sie nur Lesezugriff haben. Weitere Informationen hierzu finden Sie unter OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
