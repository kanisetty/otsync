E-Mail-Benachrichtigungen für einen freigegebenen Ordner aktivieren

So aktivieren Sie E-Mail-Benachrichtigungen für einen freigegebenen Ordner:

  1. Klicken Sie auf den freigegebenen Ordner eines anderen Teilnehmers.

  2. Wählen Sie im rechten Fensterbereich die Option Ordnerbenachrichtigungen erhalten.

 

Hinweise:

  • Die E-Mail-Benachrichtigungen werden an die im Feld E-Mail genannten Adressen in Ihrem Content Server-Profil gesendet. Wenn Sie keine Benachrichtigungen erhalten, wenden Sie sich an Ihren Administrator. Bei externen Benutzern werden die E-Mails an die Adresse gesendet, die bei der Anmeldung bei Ihrem Tempo Box-Konto eingegeben wurde.

  • Diese Option ist nur verfügbar, wenn Ihr Systemverwalter E-Mail-Benachrichtigungen aktiviert hat und Sie die globale Option zum Empfang von E-Mail-Benachrichtigungen deaktiviert haben. Weitere Informationen zur globalen Option finden Sie unter E-Mail-Benachrichtigungen aktivieren.
