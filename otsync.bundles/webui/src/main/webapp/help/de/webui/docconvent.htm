i – Dokumentationskonventionen

Benutzeroberfläche

Dieses Format wird für Elemente der grafischen Benutzeroberfläche (Graphical User Interface, GUI) wie z. B. Schaltflächen, Namen von Symbolen, Menüelemente, Namen von Dialogfeldern und Felder verwendet.

Filenames, commands und Beispieldaten

Dieses Format wird für Dateinamen, Pfade, URLs und Befehle in der Befehlszeile verwendet. Darüber hinaus wird es für Beispieldaten, in Textfeldern einzugebenden Text und sonstige Literale verwendet.

 

Hinweis:
Wenn Sie Beispiele für die Kommandozeile aus einer PDF kopieren, müssen Sie bedenken, dass PDFs verborgene Zeichen enthalten können. OpenText empfiehlt daher, diese aus einer HTML-Version des Dokuments zu kopieren (falls vorhanden).

 
TASTENNAMEN

Tastennamen werden in GROSSBUCHSTABEN dargestellt. Beispiel:
Drücken Sie STRG + V.

<Variablenname>

Spitze Klammern < > werden zur Kennzeichnung von Variablen oder Platzhaltern verwendet. Der Benutzer muss anschließend die Klammern und den beschreibenden Inhalt durch den entsprechenden Wert ersetzen. Beispiel: <server_name> wird serv01.

Hyperlink oder Weblink (http://www.opentext.com)

Diese Formate werden für Links verwendet. Über diese Links können Sie direkt auf andere Seiten in der Dokumentation (Hyperlink) oder im Internet (http://www.opentext.com) wechseln.

Interne Querverweise

Klicken Sie auf einen Querverweis, um direkt auf die entsprechende Seite im aktuellen Dokument zu wechseln.

Externe Querverweise

Unter externen Querverweisen versteht man im Allgemeinen Querverweise auf andere Dokumente. Liegt ein Dokument allerdings im HTML-Format vor (z. B. im Knowledge Center) können externe Querverweise auch aktive Verknüpfungen zu einem bestimmten Abschnitt in dem in der Referenz genannten Dokument darstellen.

Warnungen, Hinweise und Tipps
 

Achtung
Warnungen helfen Ihnen, dauerhafte Probleme zu vermeiden. Lesen Sie diese Informationen sorgfältig durch, und beachten Sie alle darin enthaltenen Anweisungen.

 
 

Wichtig:
Wichtige Hinweise helfen Ihnen, schwerwiegende Probleme zu vermeiden.

 
 

Hinweis:
Hinweise enthalten zusätzliche Informationen zu einer Tätigkeit.

 
 

Tipp:
Tipps bieten einen schnelleren oder leichteren Weg, eine Aufgabe auszuführen.
