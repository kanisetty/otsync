Dateien reservieren und Reservierungen aufheben

So reservieren Sie eine Datei:

  1. Zeigen Sie auf das Funktionsmenü des Objekts und klicken Sie dann auf Reservieren.

    Sie können eine Datei nicht in einem Ordner reservieren, für den Sie nur Lesezugriff besitzen. Ordner mit reinem Lesezugriff sind mit einer Schlossüberlagerung gekennzeichnet, zum Beispiel oder .

    Wenn Sie versuchen, eine Datei in einem Unternehmensordner zu reservieren, für den Sie nur Lese- und Schreibzugriff haben und die Option Reservieren nicht aktiviert ist, besteht für Sie keine Berechtigung zum Reservieren im Ordner in Content Server. Wenden Sie sich an Ihren Administrator, um diese Berechtigung anzufordern.

  2. Klicken Sie auf Reservieren. Die Datei ist mit der -Überlagerung gekennzeichnet, zum Beispiel .

So heben Sie die Reservierung einer Datei auf:

  1. Zeigen Sie auf das Funktionsmenü der Datei und klicken Sie dann auf Reservierung aufheben. Wenn Reservierung aufheben nicht verfügbar ist, ist die Datei von einem anderen Benutzer reserviert. Wer die Datei reserviert hat, können Sie anhand des Versionsverlaufs der Datei feststellen. Weitere Informationen finden Sie unter Objekt- oder Ordnerverlauf anzeigen.

  2. Klicken Sie auf Reservierung aufheben.
