OpenText Tempo Box: Benutzerhilfe für Web-Client

   

Einstellungen konfigurieren

Wenn Sie oben rechts auf Ihren Namen klicken, wird ein Menü angezeigt, in dem Sie folgende Aufgaben durchführen können:

  • Aktualisieren des Fotos, das andere Benutzer sehen, wenn Sie Ordner ansehen, die für Sie freigegeben sind.

  • Konfigurieren von OpenText Tempo Box, um per E-Mail benachrichtigt zu werden, sobald eine Freigabeeinladung eingegangen ist oder ein freigegebener Ordner aktualisiert wurde. Ihr Administrator muss E-Mail-Benachrichtigungen aktivieren, damit Sie diese erhalten.

  • Anzeigen der Versionsinformationen.

  • Abmelden vom Web-Client.
