OpenText Tempo Box: Benutzerhilfe für Web-Client

   

Ein Objekt hinzufügen

So fügen Sie ein Objekt hinzu:

  1. Navigieren Sie zu dem Speicherort, an dem Sie ein Objekt hinzufügen möchten.

    Sie können ein Objekt nicht zu einem freigegebenen Ordner hinzufügen, für den Sie nur Lesezugriff besitzen. Ordner mit reinem Lesezugriff sind mit einer Schlossüberlagerung gekennzeichnet, zum Beispiel oder .

    Sie können ein Objekt nicht über Tempo Box zu einem Ordner mit obligatorischen Kategorien und Attributen hinzufügen. Wie Sie bestimmen können, ob ein Ordner obligatorische Kategorien und Attribute hat, erfahren Sie unter Objekt- oder Ordnerverlauf anzeigen.

  2. Klicken Sie im mittleren Bereich auf Datei hinzufügen.

    Tempo Box kann nur Dateien bis zu 2 GB bearbeiten. Bei größeren Dateien schlägt der Vorgang fehl.

     

    Achtung
    Wenn Sie Inhalte in einem Ordner hinzufügen, für den Sie Lese- und Schreibzugriff besitzen, werden die Änderungen mit allen Benutzern synchronisiert, die Zugriff auf den Ordner haben. Ordner mit Lese- und Schreibzugriff sind gekennzeichnet durch oder .

    Wenn Sie Inhalte in einem Unternehmensordner bearbeiten, werden die Inhalte mit dem Ordner im Unternehmensarbeitsbereich synchronisiert. Unternehmensordner sind gekennzeichnet durch .

     

  3. Führen Sie einen der folgenden Schritte aus:

    • Wenn die Schaltfläche Durchsuchen angezeigt wird, klicken Sie auf Durchsuchen, wählen ein Objekt aus und klicken auf Hochladen.

    • Wenn das Dialogfeld Datei hochladen angezeigt wird, wählen Sie ein Objekt aus, und klicken Sie dann auf Öffnen.

 

Tipp:
Wenn Sie die neueste unterstützte Version von Chrome verwenden, können Sie eine Datei auch hinzufügen, indem Sie diese vom Desktop in das Browserfenster des Web-Clients ziehen. Sie können nur Dateien, nicht aber Ordner ziehen. Beim Ziehen einer Datei wird diese stets in den Web-Client kopiert, und zwar auch dann, wenn der Browser angibt, dass die Datei verschoben wird.
