Freigegebene Ordner verschieben, umbenennen und löschen

Die folgenden Bedingungen gelten für freigegebene Ordner, die Ihnen gehören:

  • Wenn Sie einen freigegebenen Ordner an einen anderen Speicherort in Ihrem Tempo Box-Ordner verschieben, bleibt er für alle anderen Benutzer freigegeben.

  • Wenn Sie einen Ordner an einen Speicherort außerhalb des Tempo Box-Ordners mit Ihrem Benutzernamen verschieben, ist er nicht mehr freigegeben und für andere Benutzer nicht mehr sichtbar. Der Ordner wird ein normaler Ordner, und der Inhalt wird aus Ihrem Tempo Box-Ordner gelöscht.

  • Wenn Sie einen freigegebenen Ordner umbenennen, wird dieser für alle anderen Teilnehmer ebenfalls umbenannt.

  • Wenn Sie einen freigegebenen Ordner löschen, wird dieser für alle anderen Teilnehmer ebenfalls gelöscht.

Die folgenden Bedingungen gelten für freigegebene Ordner, für die Sie Zugriffsberechtigung haben, die Ihnen aber nicht gehören:

  • Sie können die oberste Ebene eines freigegebenen Ordners, eines Unternehmensordners oder einer Community-Dateibibliotek nicht verschieben.

  • Wenn Sie den Ordner an einen Speicherort außerhalb des Ordners mit Ihrem Benutzernamen verschieben, ist er nicht mehr für Sie freigegeben. Der Ordner wird ein normaler Ordner, der nicht mehr synchronisiert wird. Wenn Sie den Ordner versehentlich an einen Speicherort außerhalb des Ordners mit Ihrem Benutzernamen verschieben, müssen Sie sich an den Eigentümers des Ordners wenden, damit dieser Ihnen eine neue Freigabeeinladung sendet.

  • Sie können die oberste Ebene eines freigegebenen Ordners nicht umbenennen.

  • Wenn Sie einen freigegebenen Ordner löschen, wird er nur aus dem Tempo Box-Ordner gelöscht. Andere Teilnehmer können weiter auf den Ordner zugreifen. Wenn Sie den Ordner versehentlich gelöscht haben, müssen Sie sich an den Eigentümers des Ordners wenden, damit dieser Ihnen eine neue Freigabeeinladung zusendet.

  • Wenn Sie einen Unternehmensordner löschen, wird er nur aus dem Tempo Box-Ordner gelöscht. Andere Teilnehmer können weiter auf den Ordner zugreifen. Wenn Sie den Ordner versehentlich gelöscht haben, können Sie ihn erneut aus dem Content Server-Unternehmensarbeitsbereich wieder einfügen. Weitere Informationen zum Hinzufügen Ihres Tempo Box-Ordners finden Sie unter OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).

  • Sie können die oberste Ebene eines Unternehmensordners nur dann umbenennen, wenn Sie für diesen Berechtigungen bearbeiten dürfen bzw. eine entsprechende Genehmigung haben.

  • Sie können die oberste Ebene einer Dateibibliothek in einer Tempo Social-Community nicht umbenennen.
