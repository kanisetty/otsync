1.2 – So wechseln Sie zur Standard-Startseite

So wechseln Sie zur Standard-Startseite:

  • Zeigen Sie im oberen Bereich auf Willkommen, und klicken Sie dann auf Meine Standard-Startseite.

 

Hinweise:

  • Abhängig von der OpenText Tempo-Konfiguration steht diese Option eventuell nicht zur Verfügung. Externe Benutzer können auf Content Server nicht zugreifen.

  • Die Standard-Startseite können Sie in Content Server auf der Seite Extras > Einstellungen > Meine Einstellungen - Allgemein auf der Registerkarte Allgemein festlegen.
