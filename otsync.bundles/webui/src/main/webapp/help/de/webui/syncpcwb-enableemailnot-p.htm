OpenText Tempo Box: Benutzerhilfe für Web-Client

   

E-Mail-Benachrichtigungen aktivieren

So aktivieren Sie E-Mail-Benachrichtigungen:

  1. Klicken Sie oben rechts auf Ihren Namen, klicken Sie auf Tempo Box, und klicken Sie dann auf Einstellungen.

  2. Wählen Sie unter E-Mail an mich eine der folgenden beiden Optionen:

    Wenn ich eine Freigabeeinladung erhalte

    Sobald Ihnen ein anderer Benutzer vorschlägt, einen Ordner mit ihm zusammen zu nutzen, erhalten Sie eine E-Mail.

    Wenn jemand einen freigegebenen Ordner ändert

    Sie empfangen eine E-Mail, wenn ein anderer Benutzer ein Element in einem freigegebenen Ordner, dessen Teilnehmer Sie sind, hinzufügt, bearbeitet oder löscht. Die E-Mails stellen regelmäßige Zusammenfassungen dar, die nach einem von Ihrem Administrator eingerichteten Zeitplan versendet werden. Diese Einstellung gilt global für alle Ihre freigegebenen Ordner. Auch wenn Sie diese Einstellung nicht aktivieren, können Sie sich immer noch für eine E-Mail-Benachrichtigung für einzelne freigegebene Ordner entscheiden. Weitere Informationen finden Sie unter E-Mail-Benachrichtigungen für einen freigegebenen Ordner aktivieren.

    Ihr Administrator muss die E-Mail-Benachrichtigungen aktivieren, damit diese Option verfügbar ist.

  3. Klicken Sie auf Einstellungen speichern. Die E-Mails werden an die im Feld E-Mail genannten Adressen in Ihrem Content Server-Profil gesendet. Wenn Sie keine Benachrichtigungen erhalten, kontaktieren Sie Ihren Administrator. Für externe Benutzer werden die E-Mails an die E-Mail-Adresse gesendet, die bei der Anmeldung bei Tempo Box verwendet wurde.
