Auf eine Freigabeeinladung antworten

So antworten Sie auf eine Freigabeeinladung:

  1. Zeigen Sie im oberen rechten Bereich auf das Symbol für die Freigabeeinladung .

  2. Klicken Sie auf die Freigabeeinladung, und führen Sie eine der folgenden Aktionen aus:

    • Klicken Sie auf , um die Einladung anzunehmen.

      Der freigegebene Ordner wird im Stammverzeichnis Ihres Tempo Box-Ordners platziert und als angezeigt, wenn Sie Lese-/Schreibzugriff haben bzw. als , wenn Sie nur Lesezugriff haben. Dem Ordnernamen wird der Benutzername des Ordnereigentümers angehängt. Beispiel: Marketingmaterialien (jsmith).

      Wenn Sie eine Freigabeeinladung für einen Unternehmensordner angenommen haben, wird der Ordner im Stammverzeichnis Ihres Tempo Box-Ordners platziert und als angezeigt, wenn Sie Lese-/Schreibzugriff haben, bzw. als , wenn Sie nur Lesezugriff haben. Für den Unternehmensordner wird evtl. der Name des übergeordneten Ordners an den Namen des Ordners angehängt, je nach Ihren Berechtigungen für den übergeordneten Ordner. Beispiel: Gehalt (Personal).

    • Klicken Sie auf und dann auf Ablehnen, um die Einladung abzulehnen.

    • Klicken Sie auf Alle akzeptieren, wenn Sie alle Freigabeeinladungen annehmen möchten.
