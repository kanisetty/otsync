OpenText Tempo Box: Web 客户端用户帮助

   

第1章. Web 客户端使用入门

使用 OpenText Tempo Box Web 客户端的浏览器，您可以访问 [Tempo Box] 文件夹的内容。您可以使用 Web 客户端在 [Tempo Box] 文件夹中管理项目和文件夹，也可以共享文件夹并响应来自其他用户的共享邀请 。

要浏览 [Tempo Box] 文件夹，请单击文件夹名以浏览结构。单击浏览线索中的链接返回，或单击左侧窗格中的[所有文件]返回 [Tempo Box] 文件夹的根目录。

选择文件夹时，当前正在查看的文件夹名与文件夹所有者会出现在右侧窗格中。如果选择共享文件夹，协作者列表也会出现在右侧窗格中。在下列示例中，可以看到名为模板 的共享文件夹，文件夹所有者名称为 Peter Day。此文件夹有三个协作者。有关共享文件夹的更多信息，请参阅 共享文件夹和响应共享邀请。

有多个协作者的文件夹具有以下图标之一。
