第3章. 共享文件夹和响应共享邀请

您的 Tempo Box 文件夹可能包含以下协作文件夹类型：共享文件夹、企业文件夹和社区文件库。

共享文件夹是您已经共享或与您共享的 Tempo Box 文件夹。如果文件夹名附加了用户名，则表明与您共享该文件夹。例如，营销材料 (jsmith)。
企业文件夹是同步到您的 Tempo Box 文件夹的企业工作区内的文件夹。具有该文件夹 编辑权限 的用户可以与您共享该文件夹，或者您可以导航到企业工作区中的该文件夹，并将该文件夹添加到您的 Tempo Box 文件夹。该文件夹必须允许 Tempo Box 订阅且您至少需要具有查看 权限才能添加该文件夹。有关将企业文件夹添加到您的 Tempo Box 文件夹的更多信息，请参阅 OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).
来自 Tempo Social 社区的文件库。如果您安装有 OpenText Tempo Social，则将仅显示社区文件库。请与管理员联系以获取更多信息。

您可以通过在 [Tempo Box] 文件夹内共享文件夹来与他人进行协作。共享文件夹时，您可以将其他用户添加为协作者，他们也可以访问您对该文件夹授予读/写或只读权限的文件夹。

读/写权限给予协作者对共享文件夹内容的完全控制权限，包括可以添加、编辑、复制、移动、重命名和删除共享文件夹中的项目与文件夹。

只读权限允许协作者可以查看并复制共享文件夹中的项目。只读协作者无法编辑、移动或删除项目。如果只读协作者添加项目，则该项目不会与其他协作者同步。协作者具有文件夹的只读权限时，会出现锁定覆盖。例如， 。

可以共享 Tempo Box 文件夹中的任何文件夹，但以下情况除外：

  • 根 [Tempo Box] 文件夹。

  • 共享文件夹的子文件夹或父文件夹。

  • 与您共享的文件夹。

  • 您具有只读权限的企业文件夹由 确定。如果您具有企业文件夹的读/写权限（其由 确定），则您可以共享该文件夹（如果您编辑权限 Content Server 中的文件夹的权限。）

  • 您是否具有读/写或只读权限的社区文件库。社区文件库由 和 标识。

  • 外部用户无法共享任何文件夹。

访问共享文件夹内容

作为协作者，您需要接受来自您必须接受才能查看文件夹的文件夹所有者的共享邀请。当您有待处理 共享邀请时，红色通知将出现在顶部窗格中。待定邀请的数目出现在通知中。接受之后，该共享文件夹就会位于 Tempo Box 文件夹的根目录中，文件夹名会附加文件夹所有者的用户名。例如， 营销材料 (jsmith)。该文件夹将显示为 您是否具有读/写权限或 您是否具有只读权限。

如果您具有企业文件夹的待定邀请，则通知也将出现。在这种情况下，该文件夹也放在您的 Tempo Box 文件夹的根中，将显示为 您是否具有读/写权限或 您是否具有该文件夹的只读权限。企业文件夹可能会附加到父文件夹名称，具体取决于您对父文件夹的权限。例如，薪资（人力资源）.

移动、重命名或删除共享文件夹的结果各不相同，具体取决于您是否为共享文件夹的所有者。有关详细信息，请参阅 移动、重命名和删除共享文件夹。

 

重要:
如果您打开某个项目以在您具有读/写权限的共享文件夹中编辑，则该项目不会自动保留。协作者可以在同一时刻对同一项目进行更改。要避免这种情况，您可以保留 该文件，然后再进行编辑。每次您或者协作者保存更改时，均会添加该项目的新版本。

 

共享企业文件夹

您可以共享企业文件夹，如果您具有编辑权限 Content Server 中的文件夹的权限。共享文件夹时，您可以授予该文件夹的读/写或只读权限，该操作则在 Content Server 中授予以下文件权限级别。

表 3-1: Tempo Box 中的文件夹权限和访问

Tempo Box 中的权限 Content Server 中的文件夹权限
读/写权限 修改文件夹内容的权限，其中包括所有权限，编辑权限 的所有子文件夹。
只读权限 查看 与查看内容 的所有子文件夹。

如果您尝试授予拥有高于查看 与查看内容权限的用户只读权限，则用户将会以当前访问级别添加到该共享。如果您从企业文件夹删除协作者，或者他们从 Tempo Box 文件夹删除该文件夹，则授予的访问级别仍然保持在 Content Server 中。

与外部用户共享

要与外部用户共享企业文件夹，您必须具有与没有 OpenText Tempo Box 用户帐户的用户共享文件夹的能力。请与管理员联系并请求此特权。您还必须具有编辑权限 Content Server 中的文件夹的权限。

如果同时满足这两个条件，则您必须在共享文件夹时提供用户的电子邮件地址。之后，外部用户会收到一封电子邮件共享邀请，邀请他们注册 OpenText Tempo Box 帐户。注册后，外部用户可以像常规用户一样安装客户端并访问 OpenText Tempo Box，但以下情况除外：

  • 外部用户可以接受来自其他用户的共享邀请，但他们不能与他人共享文件夹。

  • 外部用户不能发布项目或以其他方式访问 Content Server。
