1 – Web 客户端使用入门

第1章. Web 客户端使用入门

使用 Web 客户端，您可以访问 OpenText Tempo 服务器中联机 [OpenText Tempo] 文件夹的内容。您可以使用 Web 客户端来管理 [OpenText Tempo] 文件夹中的项目和文件夹，也可以共享文件夹并响应来自其他用户的共享邀请。

您可以通过以下方式之一来访问 Web 客户端：

  • 通过单击桌面客户端 OpenText Tempo 文件夹中某一文件夹上的 [OpenText Tempo > 共享此文件夹]上下文菜单命令。

  • 通过右键单击通知区域中的 OpenText Tempo 桌面客户端图标 ，然后单击[启动 Web 界面]。

您可以使用 Web 客户端顶部的[欢迎使用]菜单上显示的命令执行以下操作：

  • 查看他人与您共享的文件夹时更新其他用户看到的照片。

  • 切换到 Content Server 默认起始页（如果 OpenText Tempo 配置为支持该选项）。该选项对外部用户不可用。

  • 如果您是外部用户，则更改您首次注册 OpenText Tempo 帐户时指定的密码。

  • 查看版本信息。

  • 登出 Web 客户端。
