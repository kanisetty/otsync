移动、重命名和删除共享文件夹

以下条件适用于您拥有的共享文件夹：

  • 如果您将共享文件夹移至 Tempo Box 文件夹中的其他位置，则该文件夹仍然为所有其他协作者所共享。

  • 如果您将共享文件夹移至 Tempo Box 文件夹外部的某一位置，则该文件夹将不再被共享且其他协作者无法看到它。文件夹变为普通文件夹且内容会从 Tempo Box 文件夹删除。

  • 如果重命名共享文件夹，则会为所有协作者重命名该文件夹。

  • 如果删除共享文件夹，则会为所有协作者删除该文件夹。

以下条件适用于您有访问权限但并不拥有的共享文件夹：

  • 您无法移动共享文件夹、企业文件夹或社区文件库的根。

  • 如果将共享文件夹移出您的用户名文件夹，该文件夹就不再与您共享。文件夹变为普通文件夹，不再同步。如果您意外将文件夹移出您的用户名文件夹，则您必须联系文件夹所有者向您发送新的共享邀请。

  • 您不能重命名共享文件夹的根。

  • 如果您删除共享文件夹，则只从您的 Tempo Box 文件夹移除该文件夹。其他协作者可继续访问该文件夹。如果您意外删除该文件夹，则您必须联系文件夹所有者向您发送新的共享邀请。

  • 如果删除企业文件夹，则只从 Tempo Box 文件夹中移除该根文件夹。其他协作者可继续访问该文件夹。如果您意外删除该文件夹，则您可以从 Content Server 企业工作区再次添加。有关将企业文件夹添加到您的 Tempo Box 文件夹的更多信息，请参阅 OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD)。

  • 您无法重命名企业文件夹的根，除非您具有该文件夹编辑权限 的编辑权限。

  • 您无法重命名或移动 Tempo Social 社区中的文件库的根。
