OpenText Tempo Box: Web 客户端用户帮助

   

第2章. 管理文件和文件夹

您可以使用 Web 客户端将项目和文件夹添加、复制或移动到 [Tempo Box] 文件夹中的其他位置。您还可以查看版本并审核历史记录，并重命名或删除您的文件夹中的项目和文件夹。所做的任何更改均会自动同步到任何其他联机客户端上的 [Tempo Box] 文件夹。

 

注意:

  • 您无法使用 Web 客户端编辑[ Tempo Box] 文件夹中的文件。要使用 Web 客户端更新文件，您必须将其下载到您的桌面，在桌面进行编辑，然后将该项目作为新版本返回。 .

  • 移动、重命名或删除共享文件夹的结果各不相同，具体取决于您是否为共享文件夹的所有者。有关详细信息，请参阅 移动、重命名和删除共享文件夹。

 
 

重要:
当您在具有读/写权限的共享文件夹中查看文件时，不会自动保留这些文件。您可以在协作者正在对文件进行更改的同时下载并编辑文档。要避免这种情况，您可以 保留该文件. 读/写文件夹由 或 标识。

 

管理员可以设置 [Tempo Box] 文件夹的大小限制。如果文件夹中所有项目的合并大小（不包括与您共享的文件夹中的项目）超过该限制，就需要删除 [Tempo Box] 文件夹中的项目或移至他处，直至低于最大值。有关详细信息，请参阅 检查 Tempo Box 文件夹大小。

您的 Tempo Box 文件夹中的照片和图像显示图像的缩略图。要查看更大的图示，您可以切换至缩略图 或选择该图像。

您还可以将项目和文件夹发布到 Content Server. 该选项不提供给外部用户. 另外，如果您订阅了企业文件夹，则您具有该文件夹的读/写访问权限，您可以直接与该文件夹中的项目进行交互。有关将企业文件夹添加到您的 [Tempo Box] 文件夹的更多信息，请参阅 OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD)。
