添加项目

要添加项目：

  1. 导航到要添加项目的位置。

    您无法在具有只读权限的文件夹中添加项目。只读文件夹标识为锁定覆盖。例如， 或 .

    您无法通过 Tempo Box 添加具有强制类别和属性的文件夹中的项目。要确定文件夹是否具有强制类别和属性，请参阅 查看项目或文件夹历史记录。

  2. 在中间窗格中，单击[添加文件]。

    Tempo Box 仅处理最高 2 GB 的文件。大于该数值的文件将失败。

     

    小心
    如果添加具有读/写权限的文件夹中的内容，则这些更改会同步到拥有该文件夹访问权限的所有用户。读/写文件夹由 或 标识。

    如果您编辑企业文件夹中的内容，则这些更改会同步到企业工作区中的文件夹。企业文件夹标识为 。

     

  3. 执行以下操作之一：

    • 如果显示[浏览]按钮，单击[浏览]，选择项目，再单击[上载]。

    • 如果显示[文件上载]对话框，选择项目，然后单击[打开]。

 

提示:
如果您使用的是最新支持版本的 Chrome 浏览器，则还可以通过将文件从桌面拖动到 Web 客户端浏览器窗口来添加文件。只能拖动文件，而不能拖动文件夹。拖动文件总是会将其复制到 Web 客户端，即使浏览器指示该文件正在被移动。
