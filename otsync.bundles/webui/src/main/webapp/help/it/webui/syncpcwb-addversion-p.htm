Aggiunta della versione di un elemento

Per aggiungere una versione di un elemento:

  1. Spostarsi nella posizione in cui si trova l'elemento.

    Non è possibile aggiungere una versione a un elemento in una cartella per la quale si dispone dell'accesso in sola lettura. Le cartelle in sola lettura sono identificate dall'overlay di blocco. Ad esempio, o .

  2. Selezionare il menu funzionale dell'elemento e fare clic su Aggiungi versione.

    Se il file è già riservato da un altro utente, non è possibile aggiungere una versione al file. I file riservati sono contrassegnati con l'overlay . Ad esempio, . Per stabilire chi ha riservato il file, è possibile visualizzare la cronologia del file.

     

    Importante:
    Durante l'aggiunta di una versione elemento a un file in una cartella condivisa per cui si dispone dell'accesso in lettura/scrittura, il file non viene riservato automaticamente. È possibile scaricare e modificare un file anche mentre lo sta facendo un collaboratore. Le cartelle in lettura/scrittura sono identificate da o . Per evitare tale situazione, è possibile riservare il file. .

     
     

    Cautela
    Se si modifica il contenuto all'interno di una cartella per la quale si dispone dell'accesso in lettura/scrittura, le modifiche vengono sincronizzate per tutti gli utenti che hanno accesso alla cartella.

    Se si modifica il contenuto in una cartella Enterprise, le modifiche vengono sincronizzate per la cartella nell'Area di lavoro organizzazione. Le cartelle Enterprise sono identificate come .

     

  3. Eseguire una delle seguenti operazioni:

    • Se viene visualizzato un pulsante Sfoglia, fare clic su Sfoglia, selezionare un elemento, quindi fare clic su Aggiungi versione.

    • Se viene visualizzata la finestra di dialogo Carica file, selezionare un elemento, quindi fare clic su Apri.
