Aggiunta di un elemento

Per aggiungere un elemento:

  1. Posizionarsi nel percorso in cui si desidera aggiungere un elemento.

    Non è possibile aggiungere un elemento in una cartella per la quale si dispone dell'accesso in sola lettura. Le cartelle in sola lettura sono identificate dall'overlay di blocco. Ad esempio, o .

    Non è possibile aggiungere un elemento a una cartella che dispone di categorie e attributi obbligatori mediante Tempo Box. Per stabilire se una cartella ha categorie e attributi obbligatori, vedere Visualizzazione della cronologia di elementi e cartelle.

  2. Nel riquadro centrale, fare clic su Aggiungi file .

    Tempo Box gestisce solamente file fino a 2 GB. I file di dimensioni maggiori non vengono considerati.

     

    Cautela
    Se si aggiungono contenuti a una cartella per cui si dispone dell'accesso in lettura/scrittura, le modifiche vengono sincronizzate a tutti gli utenti che hanno accesso alla cartella. Le cartelle in lettura/scrittura sono identificate da o .

    Se si modifica il contenuto in una cartella Enterprise, le modifiche vengono sincronizzate per la cartella nell'Area di lavoro organizzazione. Le cartelle Enterprise sono identificate come .

     

  3. Eseguire una delle seguenti operazioni:

    • Se viene visualizzato il pulsante Sfoglia, fare clic su Sfoglia, quindi selezionare un elemento, infine fare clic su Carica.

    • Se viene visualizzata la finestra di dialogo Carica file, selezionare un elemento, quindi fare clic su Apri.

 

Suggerimento:
Se si utilizza la versione supportata più recente di Chrome, è possibile aggiungere un file trascinandolo dal desktop alla finestra del browser del client Web. È possibile trascinare file, ma non cartelle. Trascinare un file ne provoca sempre la copia sul client Web, anche se il browser indica che è in corso lo spostamento del file.
