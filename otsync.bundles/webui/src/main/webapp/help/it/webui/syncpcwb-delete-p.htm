Eliminazione di un file o di una cartella

Per eliminare un elemento o una cartella:

  1. Selezionare il menu funzionale dell'elemento e fare clic su Elimina.

    Non è possibile eliminare elementi in una cartella per la quale si dispone dell'accesso in sola lettura. Le cartelle in sola lettura sono identificate dall'overlay di blocco. Ad esempio, o . Se il comando Elimina non è disponibile, è stata selezionata una cartella che risulta in condivisione con l'utente o una cartella Enterprise. Vedere Eliminazione di una cartella condivisa prima di continuare.

    Non è possibile eliminare un file se risulta riservato. I file riservati sono contrassegnati con l'overlay . Ad esempio, . Per eliminare un file riservato, è innanzitutto necessario rilasciarlo.

    Non è possibile eliminare una cartella se contiene un file riservato. Per rilasciare il file, vedere Modalità per riservare e rilasciare un file.

     

    Cautela

    • Se una cartella o un elemento viene eliminato, viene rimosso dalla cartella Tempo Box. Ciò significa che l'elemento o la cartella viene eliminato da tutti gli altri client OpenText Tempo Box dell'utente, come client desktop e smartphone o tablet.

    • Se si elimina il contenuto in una cartella condivisa per la quale si dispone dell'accesso in lettura/scrittura, le modifiche vengono sincronizzate per tutti gli utenti che hanno accesso condiviso alla cartella, incluso il proprietario della cartella condivisa. Le cartelle condivise in lettura/scrittura sono identificate come o .

    • Se si elimina il contenuto in una cartella Enterprise per la quale si dispone dell'accesso in lettura/scrittura, le modifiche vengono sincronizzate per la cartella nell'Area di lavoro organizzazione. Le cartelle Enterprise sono identificate come .

     

  2. Fare clic su Elimina.

 

Suggerimento:
È possibile eliminare più elementi o cartelle all'interno di una cartella selezionando la casella di controllo a sinistra di ogni elemento o cartella, posizionando il puntatore del mouse su Altro nel riquadro centrale, quindi facendo clic su Elimina . Inoltre, è possibile selezionare la casella di controllo nell'intestazione della colonna per selezionare tutti gli elementi sulla pagina.
