Capitolo 1 – Guida introduttiva al client Web

Il client Web consente di accedere ai contenuti della cartella OpenText Tempo online sul server OpenText Tempo. È possibile utilizzare il client Web per gestire elementi e cartelle presenti nella cartella OpenText Tempo, oltre che per condividere cartelle e rispondere agli inviti di condivisione di altri utenti.

È possibile accedere al client Web in uno dei seguenti modi:

  • Facendo clic sul comando del menu contestuale OpenText Tempo > Condividi questa cartella di una cartella presente nella cartella OpenText Tempo del client desktop.

  • Facendo clic con il pulsante destro del mouse sull'icona OpenText Tempo del client desktop nell'area di notifica e scegliendo Avvia interfaccia Web.

I collegamenti visualizzati nella parte superiore del client Web consentono di effettuare le seguenti operazioni:

  • Aggiornare la foto mostrata agli altri utenti che visualizzano cartelle condivise con l'utente.

  • Passare alla pagina iniziale predefinita del Content Server, se OpenText Tempo è configurata per supportarlo. Questa opzione non è disponibile per gli utenti esterni.

  • Se l'utente è un utente esterno, modificare la password specificata durante la registrazione iniziale dell'account OpenText Tempo.

  • Visualizzare le informazioni sulla versione.

  • Scollegarsi dal client Web.
