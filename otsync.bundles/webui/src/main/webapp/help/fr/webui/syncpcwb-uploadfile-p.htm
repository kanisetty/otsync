Ajouter un élément

Pour ajouter un élément :

  1. Accédez à l'emplacement dans lequel vous souhaitez ajouter l'élément.

    Vous ne pouvez pas ajouter d'éléments dans un dossier pour lequel vous disposez d'un accès en lecture seule. Les dossiers en lecture seule sont identifiés par le cache de verrouillage. Par exemple, ou .

    Vous ne pouvez pas ajouter d'élément dans un dossier qui a des attributs et des catégories obligatoires via Tempo Box. Pour savoir si un dossier a des attributs et des catégories obligatoires, voir Afficher l'historique d'un élément ou dossier.

  2. Dans le volet central, cliquez sur Ajouter un fichier .

    Tempo Box gère uniquement les fichiers jusqu'à 2 Go. Une erreur sera générée pour les fichiers plus importants.

     

    Attention
    Si vous ajoutez du contenu dans un dossier pour lequel vous disposez d'un accès en lecture/écriture, les modifications sont synchronisées avec tous les utilisateurs qui ont accès au dossier partagé. Les dossiers en lecture/écriture sont identifiés par ou .

    Si vous modifiez le contenu d'un dossier Entreprise les modifications sont synchronisées avec le dossier dans l'Espace de travail Entreprise. Les dossiers Entreprise sont identifiés sous la forme .

     

  3. Procédez de l'une des façons suivantes :

    • Si le bouton Parcourir est disponible, cliquez sur Parcourir, sélectionnez un élément, puis cliquez sur Télécharger.

    • Si la boîte de dialogue Téléchargement du fichier s'ouvre, sélectionnez un élément, puis cliquez sur Ouvrir.

 

Astuce :
Si vous utilisez la dernière version prise en charge de Chrome, vous pouvez également ajouter un fichier en le faisant glisser de votre bureau vers la fenêtre de navigateur du client Web. Vous pouvez uniquement faire glisser les fichiers, pas les dossiers. Lorsque vous faites glisser un fichier vers le client Web, le fichier est toujours copié, même si le navigateur indique qu'il est déplacé.
