OpenText Tempo Box: Aide utilisateur du client Web

   

Configuration des paramètres

En haut, dans le volet de droite, un menu apparaît lorsque vous cliquez sur votre nom et vous permet d'exécuter les actions suivantes :

  • Actualisez la photo que les autres utilisateurs verront lorsqu'ils afficheront les dossiers partagés avec vous.

  • Configurer OpenText Tempo Box de sorte à vous avertir par e-mail lorsqu'une personne vous envoie une invitation de partage d'un dossier ou lorsqu'un dossier partagé est mis à jour. Votre administrateur doit activer les notifications par e-mail pour que vous les receviez.

  • Affichez les informations relatives à la version.

  • Déconnectez-vous du client Web.
