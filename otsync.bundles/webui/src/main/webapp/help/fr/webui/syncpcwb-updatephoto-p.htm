Actualiser votre photo

Pour actualiser votre photo :

  1. En haut, dans le volet de droite, cliquez sur votre nom, puis sur Profil.

  2. Procédez de l'une des façons suivantes :

    • Cliquez sur Parcourir pour sélectionner la photo que vous souhaitez utiliser.

      La photo doit être au format .png, .gif ou .jpg . Elle sera redimensionnée et rognée, si nécessaire, pour s'ajuster à l'espace d'affichage disponible.

    • Cliquez sur Utiliser par défaut pour utiliser la photo de profil générique.

  3. Cliquez sur Enregistrer les modifications.

 

Note :
Si vous utilisez OpenText Content Server Pulse, l'actualisation de votre photo OpenText Tempo Box met aussi à jour la photo de votre profil Pulse.
