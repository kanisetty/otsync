Chapitre 1 – Mise en route du client Web

Le client Web vous permet d'accéder au contenu de votre dossier OpenText Tempo en ligne dans le serveur OpenText Tempo. Vous pouvez utiliser le client Web pour gérer les éléments et dossiers de votre dossier OpenText Tempo et pour partager des dossiers et répondre à des invitations de partage d'autres utilisateurs.

Pour accéder au client Web, procédez de l'une des façons suivantes :

  • En cliquant sur la commande de menu contextuel OpenText Tempo > Partager ce dossier sur un dossier au sein du dossier OpenText Tempo du client de bureau.

  • En cliquant avec le bouton droit de la souris sur l'icône du client de bureau OpenText Tempo dans la zone de notification , puis en cliquant sur Lancer l'interface Web.

Vous pouvez utiliser les commandes qui apparaissent dans le menu Bienvenue, dans la partie supérieure du client Web, pour effectuer les opérations suivantes :

  • Actualisez la photo que les autres utilisateurs verront lorsqu'ils afficheront les dossiers partagés avec vous.

  • Basculez vers votre page de démarrage par défaut Content Server si OpenText Tempo est configuré pour prendre en charge cette fonction. Cette option n'est pas disponible pour les utilisateurs externes.

  • Si vous êtes un utilisateur externe, modifiez le mot de passe que vous avez indiqué lorsque vous avez enregistré pour la première fois votre compte OpenText Tempo.

  • Affichez les informations relatives à la version.

  • Déconnectez-vous du client Web.
