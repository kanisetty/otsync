Utilisation des fichiers d'aide

Bienvenue dans l'aide en ligne

Grâce aux éléments de navigation suivants, vous trouverez rapidement les informations dont vous avez besoin :

Table des matières

Vous pouvez définir le niveau de détail de la table des matières en sélectionnant la profondeur, en haut de la page :

  • affiche uniquement les sections principales.

  • inclut les sections subordonnées.

  • affiche toutes les sections jusqu'au troisième niveau.

  • etc.

Il suffit de cliquer sur l'onglet voulu.

Lorsqu'il y a plus de cinq niveaux de profondeur, le cinquième onglet indique tous les niveaux, de sorte que le nombre d'onglets ne dépasse pas cinq.

 

Notes :

  • Si vous passez à une section qui se situe en dessous de la table des matières actuellement affichée, la table des matières complète s'ouvre avec tous les niveaux de profondeur. Ceci est également vrai lorsque vous cliquez sur un onglet de profondeur insuffisante pour la section affichée.

  • Dans la table des matières, la page actuellement affichée est généralement surlignée, de même lorsque vous naviguez ou passez directement à une autre page. Dans certains cas (par exemple, lorsque vous revenez de la page d'index (voir ci-dessous) à une page de la table des matières), la surbrillance ne se fait pas. Vous pouvez alors resynchroniser la table des matières. Cliquez une ou deux fois sur l'entrée la plus basse dans la zone de navigation de la page d'aide.

 

Index

Lorsque vous cliquez sur l'onglet qui se trouve à côté des onglets de profondeur, l'Index s'affiche à la place de la table des matières (à condition qu'il existe un index pour le document). Vous obtenez alors la liste des principaux mots clés ainsi que des liens permettant d'accéder directement au passage du texte où se trouve un mot.

Navigation à l'aide du clavier

Dans les versions récentes des navigateurs (IE6, Mozilla 1.7, Firefox), vous pouvez utiliser le clavier pour parcourir un document. Il existe deux niveaux de navigation différents, le niveau document et le niveau page :

  • Les flèches haut/bas permettent de passer au niveau page et de faire défiler la page affichée vers le haut ou vers le bas.

  • La touche ESC permet de revenir au niveau page où vous pouvez utiliser les touches Page haut/bas ou les flèches gauche/droite pour passer à la page précédente ou à la page suivante.

    Au niveau document, vous pouvez également utiliser les touches numériques (1,2,3,...) pour changer la profondeur de la table des matières ou la touche i pour afficher la page d'index.

A l'ouverture d'un document, vous êtes au niveau document et vous pouvez en parcourir les pages à l'aide des touches de page. Mais dès lors que vous cliquez sur une page, vous passez au niveau page ; vous devez donc d'abord appuyer sur ESC pour parcourir à nouveau les pages du document.

Référencement d'une page spécifique

Vous voudrez peut-être renvoyer à des pages spécifiques d'un document HTML à partir de votre propre documentation ou dans un courrier électronique, surtout si vous mettez ce document à disposition sur un partage de fichiers central.

Pour ce faire, il suffit de définir l'URL de la page correspondante (que vous pouvez facilement copier dans le Presse-papiers en cliquant avec le bouton droit sur l'entrée voulue dans la table des matières et en sélectionnant « Copier le raccourci », « Copier le lien » ou une option équivalente). A condition que Javascript soit activé dans le navigateur Internet, celui-ci ouvrira alors automatiquement la page dans le cadre du manuel ; c.-à-d. qu'il ouvrira aussi la table des matières sur la gauche.

 

Notes :

  • Les fonctions automatiques de ce document en ligne sont disponibles uniquement si JavaScript est activé dans le navigateur Internet. Même dans ce cas, toutes les fonctions ne sont pas forcément prises en charge par tous les navigateurs ; ceci est vrai principalement pour les versions de navigateurs plus anciennes (Netscape 6 et antérieures, IE5 et antérieures). Les fonctions automatiques comprennent notamment la mise en surbrillance de la section actuelle de la table des matières, la navigation via le clavier comme décrite précédemment, ainsi que l'activation automatique de la table des matières lorsque vous ouvrez une page spécifique directement dans le navigateur.

    Cependant, JavaScript n'est pas indispensable pour lire un document ou le parcourir. JavaScript sert exclusivement à améliorer votre confort.
