Supprimer des collaborateurs d'un dossier partagé

Pour supprimer des collaborateurs d'un dossier partagé :

  1. Cliquez sur un dossier partagé dont vous êtes le propriétaire.

  2. Dans le volet de droite, sous COLLABORATEURS , cliquez sur en regard du nom du collaborateur, puis cliquez sur Annuler le partage .

    S'il y a plus de sept collaborateurs dans le dossier, cliquez sur Modifier les [ ] collaborateurs, puis sur en regard du nom du collaborateur, puis cliquez sur Enregistrer.

     

    Astuce :
    Pour annuler le partage du dossier pour tous les collaborateurs, cliquez sur Annuler tous.

     
 

Note :
Si vous ne souhaitez plus voir un dossier qu'un autre utilisateur partage avec vous, vous pouvez supprimer le dossier partagé. Vous devez uniquement supprimer le dossier partagé. Si vous supprimez du contenu de ce dossier partagé, ce contenu sera supprimé pour tous les collaborateurs. Pour plus d'informations sur la suppression d'un dossier partagé qu'un autre utilisateur a partagé avec vous, voir Suppression d'un dossier partagé.
