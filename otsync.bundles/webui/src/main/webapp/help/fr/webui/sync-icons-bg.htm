OpenText Tempo Box: Aide utilisateur du client Web

   

Mise en route du client Web

Le client Web OpenText Tempo Box vous permet d'accéder au contenu de votre dossier Tempo Box à l'aide d'un navigateur. Vous pouvez utiliser le client Web pour gérer les éléments et dossiers présents dans votre dossier Tempo Box., ainsi que pour partager des dossiers et répondre à des invitations de partage de la part d'autres utilisateurs.

Pour parcourir votre dossier Tempo Box, cliquez sur le nom d'un dossier pour avancer dans la structure. Cliquez sur un lien du chemin de navigation pour revenir en arrière ou cliquez sur Tous les fichiers dans le volet de gauche pour passer à la racine de votre dossier Tempo Box.

Lorsque vous sélectionnez un dossier, le nom du dossier en cours d'affichage, ainsi que le nom du propriétaire du dossier apparaissent dans le volet de droite. Si vous sélectionnez un dossier partagé, la liste des collaborateurs s'affiche également dans le volet de droite. L'exemple suivant présente un dossier partagé, nommé Modèles dont le propriétaire s'appelle Peter Day. Ce dossier compte trois collaborateurs. Pour plus d'informations sur les dossiers partagés, voir Partage de dossiers et réponse à une invitation de partage.

Les dossiers avec plusieurs collaborateurs ont l'une des icônes suivantes.

Icônes et superpositions de dossiers partagés d'OpenText Tempo Box

Icône Description
Cette icône de dossier représente un dossier partagé pour lequel vous disposez d'un accès en lecture/écriture. Si un nom d'utilisateur est ajouté au nom du dossier, cela signifie qu'un utilisateur partage ce dossier avec vous. Par exemple, Ressources marketing (rduval). Pour plus d'informations, consultez la section Partage de dossiers et réponse à une invitation de partage.
Le cache de verrouillage représente un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Pour plus d'informations, consultez la section Partage de dossiers et réponse à une invitation de partage.
Cette icône de dossier représente un dossier Entreprise pour lequel vous disposez d'un accès en lecture/écriture. Vous devez vous abonner à un dossier Entreprise dans Content Server ou connaître quelqu'un détenant l' Modifier les autorisations autorisation pour le dossier afin qu'il vous envoie une invitation de partage pour accéder au contenu depuis votre dossier Tempo Box. Pour plus d'informations, consultez la section Partage de dossiers et réponse à une invitation de partage.
Le cache de verrouillage sur cette icône de dossier représente un dossier Entreprise pour lequel vous disposez d'un accès en lecture seule. Vous devez vous abonner à un dossier Entreprise dans Content Server ou connaître quelqu'un détenant l' Modifier les autorisations autorisation pour le dossier afin qu'il vous envoie une invitation de partage pour accéder au contenu depuis votre dossier Tempo Box. Pour plus d'informations, consultez la section Partage de dossiers et réponse à une invitation de partage.
Cette icône de dossier représente une bibliothèque de fichier dans une communauté Tempo Social pour laquelle vous disposez d'un accès en lecture/écriture. Vous devez être membre d'une communauté Tempo Social pour accéder aux bibliothèques de fichier depuis votre dossier Tempo Box. Pour plus d'informations sur les communautés Tempo Social, voir OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social est un produit distinct d'OpenText Tempo Box. Si Tempo Social est installé et que vous ne voyez pas les bibliothèques de fichier dans votre dossierTempo Box, contactez votre administrateur.
Le cache de verrouillage sur cette icône de dossier représente une bibliothèque de fichier dans une communauté Tempo Social pour laquelle vous disposez d'un accès en lecture seule. Pour plus d'informations, reportez-vous au OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
