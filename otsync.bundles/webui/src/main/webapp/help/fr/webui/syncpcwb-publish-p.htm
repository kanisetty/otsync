Publier un élément ou un dossier

Pour publier un élément ou un dossier :

  1. Pointez sur le menu de fonction de l'élément , puis cliquez sur Publier.

    Contactez votre administrateur si cette option n'est pas disponible. Les utilisateurs externes ne peuvent pas publier d'éléments dans Content Server.

  2. Si vous publiez un élément, cliquez sur l'une des options suivantes :

    Publier une nouvelle version

    Permet d'ajouter l'élément en tant que nouvelle version d'un élément existant dans Content Server.

    Publier dans un dossier

    Permet d'ajouter l'élément en tant que nouvel élément à un emplacement Content Server tel que votre espace de travail personnel, un dossier ou un projet.

  3. Cliquez sur Parcourir Content Server, puis sélectionnez l'un des éléments suivants :

    • Un élément Content Server, si vous publiez une nouvelle version d'un élément.

    • Un dossier Content Server, si vous publiez un nouvel élément ou un dossier.

  4. Cliquez sur l'une des options suivantes :

    Conserver une copie de votre fichier

    Conserve l'élément ou le dossier existant à son emplacement actuel dans Tempo Box.

    Déplacer le fichier dans Content Server et supprimer l’original

    Supprime l'élément ou le dossier de votre dossier Tempo Box.

     

    Note :
    Si vous publiez un dossier partagé qui appartient à un autre utilisateur, vous devez en conserver une copie dans votre dossier Tempo Box.

     

  5. Cliquez sur Publier.
