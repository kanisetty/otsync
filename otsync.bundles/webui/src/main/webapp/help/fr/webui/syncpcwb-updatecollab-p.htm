Mettre à jour les collaborateurs d'un dossier partagé

Pour mettre à jour les collaborateurs d'un dossier partagé :

  1. Cliquez sur un dossier partagé dont vous êtes le propriétaire.

  2. Dans le volet droit, sous COLLABORATEURS, effectuez l'une des opérations suivantes :

    • Pour supprimer un collaborateur, cliquez sur en regard du nom du collaborateur, puis cliquez sur Annuler le partage .

    • Pour ajouter un collaborateur, effectuez les opérations suivantes :

      1. Cliquez sur , puis cliquez sur Lecture seule pour spécifier un collaborateur en lecture seule.

      2. Dans la zone de texte, saisissez le nom de la personne avec laquelle vous souhaitez partager ce dossier. Pour un utilisateur externe, saisissez son adresse e-mail.

         

        Note :
        Si l'adresse e-mail est ignorée et la saisie automatique n'affiche pas l'adresse e-mail de l'utilisateur externe, cela signifie que vous ne disposez pas des autorisations requises pour partager des dossiers avec les utilisateurs externes. Pour obtenir ces autorisations, contactez votre administrateur. Pour plus d'informations sur les utilisateurs externes, voir Partage avec des utilisateurs externes.

         

      3. Cliquez sur le nom ou l'adresse e-mail de l'utilisateur. Une invitation de partage est envoyée. Pour accéder au contenu d'un dossier, les collaborateurs doivent d'abord accepter l'invitation de partage.

 

Note :
S'il y a plus de sept collaborateurs pour un dossier, cliquez sur Modifier les [ ] collaborateurs pour mettre à jour les collaborateurs.
