Déplacement, changement de nom et suppression des dossiers partagés

Les conditions suivantes s'appliquent aux dossiers partagés dont vous êtes le propriétaire :

  • Si vous déplacez un dossier partagé vers un autre emplacement de votre dossier Tempo Box, il continue d'être partagé avec les autres collaborateurs.

  • Si vous déplacez un dossier partagé vers un autre emplacement hors de votre dossier Tempo Box, il n'est plus partagé et aucun autre utilisateur ne peut y accéder. Le dossier devient normal et le contenu est supprimé de votre dossier Tempo Box.

  • Si vous renommez un dossier partagé, il est renommé pour tous les autres collaborateurs.

  • Si vous supprimez un dossier partagé, il est supprimé pour tous les autres collaborateurs.

Les conditions suivantes s'appliquent aux dossiers partagés auxquels vous pouvez accéder, mais dont vous n'êtes pas le propriétaire :

  • Vous ne pouvez pas déplacer la racine d'un dossier partagé, d'un dossier Entreprise ou d'une bibliothèque de fichier de communauté.

  • Si vous déplacez un dossier partagé hors du dossier portant votre nom d'utilisateur, il n'est plus partagé avec vous. Le dossier devient normal et ne sera plus synchronisé. Si vous déplacez accidentellement le dossier hors du dossier portant votre nom d'utilisateur, vous devez contacter le propriétaire du dossier pour qu'il vous envoie une nouvelle invitation de partage.

  • Vous ne pouvez pas renommer la racine d'un dossier partagé.

  • Si vous supprimez un dossier partagé, cette suppression s'applique uniquement à votre dossier Tempo Box. Les autres collaborateurs peuvent toujours accéder au dossier. Si vous supprimez accidentellement le dossier, vous devez contacter le propriétaire du dossier pour qu'il vous envoie une nouvelle invitation de partage.

  • Si vous supprimez un dossier Entreprise, cette suppression s'applique uniquement à votre dossier Tempo Box. Les autres collaborateurs peuvent toujours accéder au dossier. Si vous avez accidentellement supprimé le dossier, vous pouvez l'ajouter une nouvelle fois depuis l'Espace de travail Entreprise Content Server. Pour plus d'informations sur l'ajout des dossiers Entreprise à votre dossier Tempo Box, voir OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).

  • Vous ne pouvez pas renommer la racine d'un dossier Entreprise sauf si vous disposez de Modifier les autorisations l'autorisation pour le dossier.

  • Vous ne pouvez pas renommer ou déplacer la racine d'une bibliothèque de fichier dans une communauté Tempo Social.
