OpenText Tempo Box: Ajuda per als usuaris del client web

   

Publicació d'un element o una carpeta

Per publicar un element o una carpeta:

  1. Seleccioneu el menú de funcions de l'element i, a continuació, feu clic a Publica.

    Si aquesta opció no està disponible, poseu-vos en contacte amb l'administrador. Els usuaris externs no poden publicar elements al Content Server.

  2. Si publiqueu un element, feu clic a una de les opcions següents:

    Publica una versió nova

    Afegir l'element com una versió nova d'un element existent del Content Server.

    Publica-ho en una carpeta

    Afegir l'element com un element nou en una ubicació del Content Server, com ara el vostre espai de treball personal, una carpeta o un projecte.

  3. Feu clic a Navega al Content Server i seleccioneu una de les opcions següents:

    • Un element del Content Server, per publicar una versió nova d'un element.

    • Una carpeta del Content Server, per publicar un element o una carpeta nous.

  4. Feu clic a una de les opcions següents:

    Mantén una còpia del teu fitxer

    Mantenir l'element o la carpeta existents a la ubicació actual de la carpeta Tempo Box.

    Desplaça el fitxer al Content Server i suprimeix l'original

    Suprimir l'element o la carpeta de la carpeta Tempo Box.

     

    Nota:
    Si publiqueu una carpeta compartida que pertany a un altre usuari, cal que en mantingueu una còpia a la vostra carpeta Tempo Box.

     

  5. Feu clic a Publica-ho.
