Capítulo 1 – Primeros pasos con el cliente web

El cliente web le permite acceder al contenido de la carpeta OpenText Tempo del servidor de OpenText Tempo. Puede utilizar el cliente web para gestionar los elementos y carpetas de la carpeta OpenText Tempo, así como para compartir carpetas y responder a las invitaciones de uso compartido de otros usuarios.

Puede acceder al cliente web de una de las formas siguientes:

  • Haciendo clic en el comando del menú contextual OpenText Tempo > Compartir esta carpeta de una carpeta dentro de la carpetaOpenText Tempo del cliente de escritorio.

  • Haciendo clic con el botón derecho del ratón en el icono del cliente de escritorio de OpenText Tempo en el área de notificación y, a continuación, haciendo clic en Iniciar interfaz web.

Puede usar los comandos que aparecen en el menú de bienvenida en la parte superior del cliente web para realizar las siguientes acciones:

  • Actualizar la foto que otros usuarios ven cuando visualizan carpetas que comparten con usted.

  • Pasar a su página de inicio predeterminada de Content Server, si OpenText Tempo se ha configurado para que sea compatible. Esta opción no está disponible para usuarios externos.

  • Si es un usuario externo, cambie la contraseña que especificó la primera vez que registró su cuenta de OpenText Tempo.

  • Ver información sobre la versión.

  • Desconectarse del cliente web.
