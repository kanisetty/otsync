So bearbeiten Sie Ihr Profil:

  1. Tippen Sie auf und dann auf Profile wechseln.

  2. Fügen Sie auf dem OpenText Tempo Box-Bildschirm Profil ein neues Profil mit dem neuen Benutzernamen oder der OpenText Tempo Box-URL hinzu, und wählen Sie einen eindeutigen Profilnamen aus. Ausführliche Informationen finden Sie in Profil erstellen.

  3. Melden Sie sich mit dem neuen Profil bei OpenText Tempo Box an. Sie können nun mit dem neuen Profil auf Ihren OpenText Tempo Box-Ordner zugreifen. Ihre Liste der Favoriten aus dem ursprünglichen Profil ist im neuen Profil jedoch nicht verfügbar.

    Um Ihre Favoritenliste beizubehalten, müssen Sie das ursprüngliche Profil wie folgt bearbeiten:

    1. Tippen Sie auf .

    2. Tippen Sie auf den Namen des Profils, das Sie bearbeiten möchten.

    3. Ändern Sie Benutzername oder Server bei Bedarf.

    4. Tippen Sie auf das Feld Kennwort, und geben Sie das Kennwort für diesen Benutzer ein.

    5. Tippen Sie auf Fertig.

    6. Tippen Sie auf Abmelden.

    7. Melden Sie sich auf dem Bildschirm OpenText Tempo Box Profil mit Ihrem ursprünglichen Profil bei OpenText Tempo Box an.

      Sie können nun das zweite Profil löschen, das Sie in Schritt 2 erstellt haben. Ausführliche Informationen finden Sie in Profil löschen.

So bearbeiten Sie Ihr Passwort:

  1. Tippen Sie auf .

  2. Tippen Sie auf Abmelden.

  3. Tippen Sie auf Ihr Profil.

  4. Tippen Sie auf das Feld Kennwort, und geben Sie das neue Kennwort ein.
