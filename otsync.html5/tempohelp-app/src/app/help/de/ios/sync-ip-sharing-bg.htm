Ordner freigeben und freigegebene Ordner verwalten

 

So geben Sie einen Ordner frei:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielordner. Ein Ordner, den Sie freigeben möchten, kann sich an einem beliebigen Speicherort in Ihrem Tempo Box-Ordner befinden. Weitere Informationen zum Hinzufügen eines neuen, freizugebenden Ordners finden Sie unter Fotos oder Ordner hinzufügen.

  2. Tippen Sie auf .

  3. Tippen Sie auf Freigeben. Wenn die Schaltfläche Freigeben nicht verfügbar ist, verfügen Sie nicht über die Berechtigung zum Freigeben dieses Ordners. Eine Liste von Ordnern, die nicht freigegeben werden können, finden Sie unter Ausnahmen für die Ordnerfreigabe und Unternehmensordner freigeben unten.

  4. Optional Wenn Sie einen Ordner freigeben, können Sie dem Teilnehmer Lese- und Schreibrechte oder nur Leserechte für den Ordner gewähren. Standardmäßig werden den Teilnehmern Lese- und Schreibrechte gewährt, gekennzeichnet durch .

    So gewähren Sie einem Teilnehmer nur Leserechte:

    1. Tippen Sie auf .

    2. Tippen Sie auf Schreibgeschützt. Das Symbol wechselt zu .

    3. Tippen Sie auf Fertig.

    Mit Lese- und Schreibrechten erhalten alle Teilnehmer, auch externe Benutzer, die vollständige Kontrolle über den Inhalt des freigegebenen Ordners. Sie können Dateien und Ordner hinzufügen, bearbeiten, kopieren, verschieben und löschen. Wenn am freigegebenen Ordner Änderungen vorgenommen wurden, werden diese Änderungen automatisch mit allen Personen synchronisiert, die Zugriff auf diesen Ordner haben, einschließlich des Eigentümers.

    Mit Leserecht kann ein Teilnehmer den Inhalt im freigegebenen Ordner nur lesen.

  5. Tippen Sie auf das Feld Name, und geben Sie den Namen der Person ein, für die Sie diesen Ordner freigeben möchten.

    Zum Freigeben eines Ordners für einen externen Benutzer geben Sie dessen E-Mail-Adresse ein. Externe Benutzer sind keine Mitarbeiter des Unternehmens. Nachdem Sie eine gültige E-Mail-Adresse eingegeben haben, wird die E-Mail-Adresse des externen Benutzers automatisch angezeigt.

     

    Hinweis:
    Wird die E-Mail-Adresse ignoriert und bleibt der Bereich Freigeben an leer, verfügen Sie nicht über die Berechtigung, Ordner für externe Benutzer freizugeben. Weitere Informationen finden Sie unter Unternehmensordner für externe Benutzer freigeben.

     

  6. Tippen Sie auf den Namen oder die E-Mail-Adresse des Benutzers. Eine Freigabeeinladung wird gesendet. Teilnehmer müssen die Freigabeeinladung akzeptieren, um auf die Inhalte des Ordners zugreifen zu können.

  7. Tippen Sie im daraufhin geöffneten Dialogfeld auf Ja, um diesen Ordner für eine andere Person freizugeben, und wiederholen Sie die Schritte 4 bis 6, oder tippen Sie auf Nein, um zur Seite Ordnerdetails zurückzukehren.

  8. Tippen Sie auf Zurück, um zu Ihren Dateien zurückzukehren.

    Der Ordner ist nun als freigegebener Ordner markiert. Unabhängig davon, ob Sie den Teilnehmern Lese- und Schreibrechte oder nur Leserechte gewähren, wird Ihnen der freigegebene Ordner als angezeigt.

Ausnahmen für die Ordnerfreigabe

Sie können jeden Ordner in Ihrem Tempo Box-Ordner freigeben. Dabei gelten jedoch folgende Ausnahmen:

  • Der Tempo Box-Ordner auf der untersten Ebene der Struktur.

  • Ordner, die einem freigegebenen Ordner untergeordnet oder übergeordnet sind.

  • Ordner, die für Sie freigegeben wurden.

  • Unternehmensordner bei nur Lesezugriff, die durch gekennzeichnet sind. Wenn Sie Lese-/Schreibzugriff für einen Unternehmensordner haben, der durch gekennzeichnet ist, können Sie den Ordner freigeben, wenn Sie die Berechtigung Berechtigungen bearbeiten für diesen Ordner in Content Server haben.

  • Community-Dateibibliotheken unabhängig vom Zugriff (Lese-/Schreibzugriff oder nur Lesezugriff). Community-Dateibibliotheken sind durch und gekennzeichnet.

  • Externe Benutzer können keine Ordner freigeben.

Unternehmensordner freigeben

Sie können einen Unternehmensordner freigeben, wenn Sie die Berechtigung Berechtigungen bearbeiten für diesen Ordner in Content Server haben. Bei der Freigabe eines Ordners können Sie Lese-/Schreibzugriff oder nur Lesezugriff auf den Ordner gewähren, wodurch folgende Ordnerberechtigungsebenen in Content Server gewährt werden.

Wenn Sie versuchen, einem Benutzer nur Lesezugriff zu gewähren, der umfassendere Berechtigungen als Anzeigen und Inhalt anzeigenhat, wird der Benutzer mit seiner aktuellen Zugriffsebene zur Freigabe hinzugefügt. Wenn Sie einen Teilnehmer aus einem Unternehmensordner entfernen oder wenn der Ordner aus dem Tempo Box-Ordner gelöscht wird, bleibt die gewährte Zugriffsebene in Content Server gleich.

Unternehmensordner für externe Benutzer freigeben

Um einen Unternehmensordner für externe Benutzer freizugeben, müssen Sie die Berechtigung haben, Ordner für Benutzer freizugeben, die keine OpenText Tempo Box-Benutzerkonten haben. Wenden Sie sich an Ihren Administrator, um diese Berechtigung anzufordern. Sie müssen ferner die Berechtigung Berechtigungen bearbeiten für den Order in Content Server haben.

Wenn beide Bedingungen erfüllt sind, müssen Sie bei der Freigabe eines Ordners die E-Mail-Adresse des Benutzers angeben. Der externe Benutzer erhält dann eine Freigabeeinladung, in der er zur Registrierung für ein OpenText Tempo Box-Konto eingeladen wird. Nach der Registrierung kann der externe Benutzer Clients installieren und wie ein regulärer Benutzer auf OpenText Tempo Box zugreifen, wobei folgende Ausnahmen bestehen:

  • Externe Benutzer können Freigabeeinladungen von anderen Benutzern annehmen, aber selbst keine Ordner für andere freigeben.

  • Externe Benutzer können keine Elemente in Content Server veröffentlichen oder anderweitig auf das Programm zugreifen.

Informationen zur Verwaltung Ihrer freigegebenen Ordner finden Sie in den anderen Themen unter Ordner freigeben und freigegebene Ordner verwalten.
