Dateien in einer anderen App öffnen

So öffnen Sie eine Datei in einer anderen App:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, tippen Sie auf Favoriten.

  2. Tippen Sie auf die Datei, die Sie exportieren möchten. Die Datei wird geöffnet.

  3. Tippen Sie auf . Eine Liste von Apps, die diese Datei öffnen können, wird angezeigt.

  4. Tippen Sie auf eine App.
