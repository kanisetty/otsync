Favoriten hinzufügen, anzeigen und entfernen

So fügen Sie einen Favoriten hinzu:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei. Sie können sowohl persönliche als auch freigegebene Dateien als Favoriten markieren.

  2. Tippen Sie auf die Datei, die Sie als Favorit markieren möchten. Die Datei wird geöffnet.

  3. Tippen Sie auf . Das Symbol wechselt zu und die Datei wird mit der Überlagerung in der Ordneransicht markiert. Zum Beispiel: . Die Markierung einer Datei als Favorit ermöglicht Ihnen, die Datei anzuzeigen, wenn Sie offline sind.

    Ein Datensatz Ihrer Favoritendateien wird in Ihrem Profil gespeichert, das spezifisch für Ihr iPad oder iPhone ist. Sie müssen auf jedem Ihrer Smartphones und Tablets mit der OpenText Tempo-App Dateien als Favoriten markieren.

  4. Tippen Sie auf Zurück, um zu Ihren Dateien zurückzukehren.

     

    Tipp:
    Sie können die Datei auch als Favorit markieren, indem Sie auf für die Datei und dann auf Zu Favoriten hinzufügen tippen.

     

So zeigen Sie einen Favoriten an:

  1. Tippen Sie auf Favoriten.

  2. Optional Tippen Sie auf ,um sicherzustellen, dass der aktuelle Inhalt angezeigt wird.

  3. Tippen Sie auf die Datei, die Sie anzeigen möchten.

  4. Tippen Sie auf Zurück, um zu Ihren Dateien zurückzukehren.

So entfernen Sie einen Favoriten:

  1. Tippen Sie auf Favoriten oder auf Tempo Box, und navigieren Sie dann zur Zieldatei.

  2. Tippen Sie auf die Datei, die Sie aus der Favoritenliste entfernen möchten. Die Datei wird geöffnet.

  3. Tippen Sie auf . Das Symbol wechselt dann zu . Die Datei wird nicht mehr in der Liste der Favoriten angezeigt.

    Dadurch wird die Datei nicht aus Ihrem Tempo Box-Ordner gelöscht.

  4. Tippen Sie auf Zurück, um zu Ihren Dateien zurückzukehren.

     

    Tipp:
    Sie können auch die Datei aus Ihrer Favoritenliste entfernen, indem Sie auf für die Datei und dann auf Favoriten entfernen tippen.
