Dateien hinzufügen

So fügen Sie eine Datei hinzu:

  1. Sie müssen die Datei aus einer anderen App importieren. Zeigen Sie die Datei an, die Sie importieren möchten, und tippen Sie auf und/oder Öffnen in.

  2. Tippen Sie auf In Tempo öffnen.

  3. Führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf Hochladen, um die Datei wird zum Stammverzeichnis Ihres Tempo Box-Ordners hinzuzufügen.

    • Um einen vom Stammverzeichnis Ihres Tempo Box-Ordners abweichenden Speicherort auszuwählen, tippen Sie auf Ziel: Persönlicher Arbeitsbereich.

      1. Tippen Sie auf OpenText Tempo Box. Das Stammverzeichnis Ihres Tempo Box-Ordners wird angezeigt.

      2. Navigieren Sie zum Zielordner.

        Sie können eine Datei nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

        Über Tempo Box können Sie keine Dateien zu Ordnern hinzufügen, die über obligatorische Kategorien oder Attribute verfügen. Zur Festlegung siehe Objektdetails anzeigen.

      3. Tippen Sie auf Auswählen, um den Zielordner auszuwählen.

        Um einen neuen Ordner hinzuzufügen, klicken Sie auf Neuer Ordner. Geben Sie den Namen des neuen Ordners ein, und tippen Sie auf Fertig. Tippen Sie auf Auswählen, um den neuen Ordner auszuwählen.

      4. Klicken Sie auf Hochladen.

  4. Optional Tippen Sie in OpenText Tempo auf die importierte Datei. Informationen zum Verschieben der Datei an einen anderen Speicherort finden Sie unter Dateien oder Ordner verschieben oder kopieren.
