So ändern Sie Ihr Profilbild:

  1. Tippen Sie auf .

  2. Tippen Sie auf Ihr Profil.

     

    Hinweis:
    Wenn mehrere Profile auf Ihrem Gerät vorhanden sind, können Sie nur das Profilbild des angemeldeten Profils ändern.

     

  3. Tippen Sie auf Ihr Profilbild, und führen Sie einen der folgenden Schritt aus:

    • Tippen Sie auf Von Kamera, um das Profilbild mithilfe der Apple-Kamera festzulegen.

      Nehmen Sie das Bild auf, und tippen Sie dann auf Verwenden, um das aktuelle Bild zu verwenden, oder tippen Sie auf Erneut aufnehmen, und nehmen Sie das Bild erneut auf.

    • Tippen Sie auf Aus Album, um Ihr Profilbild mit einem Bild aus dem Apple-Fotoalbum festzulegen.

      Navigieren Sie zu dem Bild, das Sie verwenden möchten, und tippen Sie dann auf das Bild.

    • Tippen Sie auf Als Standard festlegen, um als Profilbild das standardmäßige OpenText Tempo Box Profilbild zu verwenden.

  4. Tippen Sie auf Profil bearbeiten, um zum Bildschirm Profil bearbeiten zurückzukehren.

  5. Tippen Sie auf Fertig, um zum Bildschirm Einstellungen zurückzukehren.
