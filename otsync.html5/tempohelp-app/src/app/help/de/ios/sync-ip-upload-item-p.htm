Fotos oder Ordner hinzufügen

 

So fügen Sie ein Foto mit der Apple-Kamera hinzu:

  1. Navigieren Sie zum Zielordner, wo Sie ein Foto hinzufügen möchten.

    Sie können ein Foto nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

    Über Tempo Box können Sie keine Fotos zu Ordnern hinzufügen, die über obligatorische Kategorien oder Attribute verfügen. Zur Festlegung siehe Objektdetails anzeigen.

  2. Tippen Sie auf .

  3. Tippen Sie auf Von Kamera.

  4. Machen Sie ein Foto.

  5. Tippen Sie auf Foto verwenden, um das aktuelle Foto hinzuzufügen, oder tippen Sie auf Erneut aufnehmen, und wiederholen Sie Schritt 4.

  6. Optional Tippen Sie auf das Feld Dateiname, und bearbeiten Sie den Namen des Fotos. Alle mit der Gerätekamera aufgenommenen Fotos werden als JPEG-Datei gespeichert.

  7. Tippen Sie auf Hochladen.

So fügen Sie ein Foto aus dem Apple-Fotoalbum hinzu:

  1. Navigieren Sie zum Zielordner, wo Sie ein Foto hinzufügen möchten.

    Sie können ein Foto nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet.

    Über Tempo Box können Sie keine Fotos zu Ordnern hinzufügen, die über obligatorische Kategorien oder Attribute verfügen. Zur Festlegung siehe Objektdetails anzeigen.

  2. Tippen Sie auf .

  3. Tippen Sie auf Aus Album.

    Wenn Sie eine Fehlermeldung erhalten, stellen Sie sicher, dass in den Datenschutzeinstellungen auf Ihrem Gerät das Hochladen von Fotos mit OpenText Tempo zulässig ist. Tippen Sie auf der Gerätestartseite auf Einstellungen > Datenschutz > Fotos, und achten Sie darauf, dass der OpenText Tempo-Schalter auf EIN steht.

  4. Navigieren Sie durch die Alben auf Ihrem Gerät, und tippen Sie auf das Foto, das Sie hinzufügen möchten.

  5. Optional Tippen Sie auf das Feld Dateiname, und bearbeiten Sie den Namen des Fotos.

  6. Tippen Sie auf Hochladen.

So fügen Sie mehrere Fotos aus dem Apple-Fotoalbum hinzu:

  1. Navigieren Sie zum Zielordner, wo Sie ein Foto hinzufügen möchten.

    Sie können ein Foto nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet.

    Über Tempo Box können Sie keine Fotos zu Ordnern hinzufügen, die über obligatorische Kategorien oder Attribute verfügen. Zur Festlegung siehe Objektdetails anzeigen.

  2. Tippen Sie auf .

  3. Tippen Sie auf Aus Album.

    Wenn Sie eine Fehlermeldung erhalten, stellen Sie sicher, dass in den Datenschutzeinstellungen auf Ihrem Gerät das Hochladen von Fotos mit OpenText Tempo zulässig ist. Tippen Sie auf der Gerätestartseite auf Einstellungen > Datenschutz > Fotos, und achten Sie darauf, dass der OpenText Tempo-Schalter auf EIN steht.

  4. Navigieren Sie zu dem Album auf Ihrem Gerät, und tippen Sie auf die Fotos, die Sie hinzufügen möchten.

  5. Tippen Sie auf Fertig. Um die Aktion abzubrechen, klicken Sie auf das Symbol X.

  6. Optional Sie müssen eventuell auf tippen, um die importierten Bilder anzuzeigen.

So fügen Sie einen Ordner hinzu:

  1. Navigieren Sie zum Zielspeicherort für den neuen Ordner.

    Sie können einen Ordner nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet.

  2. Tippen Sie auf .

  3. Tippen Sie auf Neuen Ordner.

  4. Tippen Sie auf das Feld Ordnername, und geben Sie einen Namen für den neuen Ordner an.

  5. Tippen Sie auf Fertig.
