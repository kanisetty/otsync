So erstellen Sie ein Profil:

  1. Wenn Sie erstmalig die Anwendung OpenText Tempo Box starten, tippen Sie auf Profil hinzufügen.

    Wenn Sie ein weiteres Profil erstellen möchten und bereits bei OpenText Tempo Box angemeldet sind, gehen Sie wie folgt vor:

    1. Tippen Sie auf .

    2. Tippen Sie auf Profil wechseln.

    3. Tippen Sie auf Profil hinzufügen.

  2. Tippen Sie auf das Feld Benutzername, und geben Sie Ihren Benutzernamen ein. Wenn Sie sich außerhalb des Unternehmens befinden, geben Sie die E-Mail-Adresse ein.

     

    Hinweis:
    Ein Benutzer, der sich außerhalb des Unternehmens befindet, muss zunächst vor dem Erstellen eines Profils ein OpenText Tempo Box-Konto anlegen.

    Um ein Konto zu erstellen, muss ein Benutzer innerhalb des Unternehmens einen Ordner für den externen Benutzer freigeben. Dadurch wird an den externen Benutzer eine E-Mail-Einladung gesendet, in der die Schritte zum Erstellen eines Kontos beschrieben werden.

     

  3. Tippen Sie auf das Feld Kennwort, und geben Sie Ihr Kennwort ein.

    Wenn Sie sich außerhalb des Unternehmens befinden, geben Sie das Kennwort ein, das Sie beim Erstellen Ihres OpenText Tempo Box-Kontos ausgewählt haben. Falls Sie sich nicht an Ihr Kennwort erinnern können, finden Sie diese Informationen in der E-Mail, die Sie beim Erstellen Ihres OpenText Tempo Box-Kontos erhalten haben.

     

    Hinweis:
    Für das Kennwort muss Groß-/Kleinschreibung beachtet werden. Stellen Sie sicher, dass Sie Ihr Kennwort korrekt eingeben.

     

  4. Tippen Sie auf das Feld Server, und geben Sie den URL Ihres OpenText Tempo Box-Servers ein.

    Wenn Sie sich nicht an die OpenText Tempo Box-URL erinnern können, wenden Sie sich an Ihren Administrator.

    Wenn Sie sich außerhalb des Unternehmens befinden, finden Sie die OpenText Tempo Box-URL in der E-Mail, die Sie nach dem Erstellen des OpenText Tempo Box-Kontos erhalten haben.

  5. Tippen Sie auf das Feld Profilname, und geben Sie einen Profilnamen ein, der nur für Ihr Gerät gilt.

  6. Optional Tippen Sie auf Kennworterinnerung, um beim nächsten Start der Anwendung einen sofortigen Zugriff zuzulassen. Dies gilt nicht, wenn Sie sich von OpenText Tempo Box abmelden oder die Sitzung abläuft. Lesen Sie die Informationen unter Achtung weiter unten.

  7. Tippen Sie auf Fertig. Zu den Profildaten gehören Ihre Kontoanmeldedaten, der OpenText Tempo Box-URL und eine Liste Ihrer Favoritendateien. Profile gelten speziell für das iPhone und iPad. Sie müssen auf jedem Ihrer Smartphones und Tablets mit der OpenText Tempo Box-Anwendung ein Profil erstellen und Dateien als Favoriten markieren.

 

Achtung

  • Wenn Sie zulassen, dass sich OpenText Tempo Box an Ihr Kennwort erinnert, kann dies die Sicherheit beeinträchtigen, falls Ihr Gerät von einer anderen Person verwendet werden würde.

  • Wenn Sie mehrere Profile auf Ihrem Gerät konfiguriert haben, können andere Profile auf Ihr OpenText Tempo Box-Konto zugreifen, wenn Kennworterinnerung ausgewählt ist und Sie sich nicht abgemeldet haben. Zum Abmelden tippen Sie auf und dann auf Abmelden.
