Dateien in einer anderen App oder einem anderen Profil öffnen

 

So öffnen Sie eine Datei in einer anderen App:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, tippen Sie auf Favoriten.

  2. Berühren und halten Sie die Datei, die Sie exportieren möchten.

  3. Tippen Sie auf E-Mail / Exportieren. Wenn die letzte Version der Datei nicht auf Ihrem Gerät verfügbar ist, wird die Datei vor dem Exportieren heruntergeladen.

  4. Tippen Sie auf eine App, und befolgen Sie die Anweisungen auf dem Bildschirm.

So exportieren Sie eine Datei in ein anderes OpenText Tempo Box-Profil:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, tippen Sie auf Favoriten.

  2. Berühren und halten Sie die Datei, die Sie exportieren möchten.

  3. Tippen Sie auf E-Mail / Exportieren.

  4. Tippen Sie auf OpenText Tempo.

  5. Tippen Sie auf die Android Zurück-Taste, bis Sie zur Profil-Seite zurückgekehrt sind.

  6. Berühren und halten Sie Ihren Profilnamen, und tippen Sie dann auf Abmelden.

  7. Melden Sie sich bei dem Profil an, in das Sie die Datei importieren möchten.

  8. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielspeicherort, an dem Sie die Datei platzieren möchten. Sie können keine Datei in einen freigegebenen Ordner importieren, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

  9. Tippen Sie auf Einfügen. Nach Abschluss des Exports sind Sie weiterhin beim zweiten Profil angemeldet.

    Um zum ursprünglichen Profil zurückzukehren, melden Sie sich beim zweiten Profil ab, indem Sie auf die Taste Android Zurück tippen, um zur Profil-Seite zurückzukehren.

  10. Berühren und halten Sie den Namen des zweiten Profils, und tippen Sie dann auf Abmelden.

  11. Melden Sie sich auf der Seite Profil mit Ihrem ursprünglichen Profil an.
