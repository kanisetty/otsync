Favoriten hinzufügen, anzeigen, entfernen und synchronisieren

 

So fügen Sie einen Favoriten hinzu:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei. Sie können sowohl persönliche als auch freigegebene Dateien als Favoriten markieren.

  2. Berühren und halten Sie die Datei, die Sie als Favorit kennzeichnen möchten.

  3. Tippen Sie auf Zu Favoriten hinzufügen.

    Die Datei wird mit der Überlagerung gekennzeichnet. Zum Beispiel: .

  4. Optional Wenn die Überlagerung angezeigt wird, wurde die als Favorit markierte Datei nicht heruntergeladen. Zum Beispiel: . Um die Datei herunterzuladen, synchronisieren Sie Ihre Favoriten, oder öffnen Sie die Datei. Weitere Informationen finden Sie unter Manuelles Synchronisieren Ihrer Favoriten.

    Informationen zum automatischen Synchronisieren Ihrer Favoritendateien finden Sie unter OpenText Tempo-Einstellungen ändern.

    Eine Aufzeichnung Ihrer Favoritendateien wird in Ihrem Profil gespeichert. Profile gelten speziell für Ihr Gerät. Sie müssen auf jedem Ihrer Smartphones und Tablets mit der OpenText Tempo Box-App ein Profil erstellen und Dateien als Favoriten markieren.

So zeigen Sie einen Favoriten an:

  1. Tippen Sie auf Favoriten.

  2. Optional Um sicherzugehen, dass Sie den aktuellsten Inhalt anzeigen, sollten Sie die Anzeige regelmäßig aktualisieren. Weitere Informationen finden Sie unter Anzeige aktualisieren.

    Dabei werden die Dateien nicht zur Offline-Anzeige heruntergeladen. Um die Datei herunterzuladen, synchronisieren Sie Ihre Favoriten, oder öffnen Sie die Datei. Weitere Informationen finden Sie unter Manuelles Synchronisieren Ihrer Favoriten unten.

So öffnen Sie den Speicherort einer Favoritendatei:

  1. Tippen Sie auf Favoriten.

  2. Berühren und halten Sie die Datei, die Sie an dem Dateispeicherort öffnen möchten.

  3. Tippen Sie auf Speicherort öffnen.

So entfernen Sie einen Favoriten:

  1. Tippen Sie auf Favoriten oder Tempo Box, und navigieren Sie zur Zieldatei.

  2. Berühren und halten Sie die Datei, die Sie aus der Liste Ihrer Favoriten entfernen möchten.

  3. Tippen Sie auf Aus Favoriten entfernen. Die Datei wird nicht mehr in der Liste der Favoriten angezeigt. Dadurch wird die Datei nicht aus Ihrem Tempo Box-Ordner gelöscht.

So synchronisieren Sie Favoriten manuell:

  1. Tippen Sie auf Favoriten.

  2. Tippen Sie auf Menü > Favoriten aktualisieren. Wenn die App gerade die Synchronisierung der Favoriten ausführt oder Sie offline sind, ist Favoriten aktualisieren nicht verfügbar.

     

    Hinweis:
    Durch Tippen auf Aktualisieren wird die Anzeige der Seite Favoriten aktualisiert. Ihre Favoritendateien werden dabei nicht zur Offline-Anzeige auf Ihr Gerät heruntergeladen. Um die Datei herunterzuladen, müssen Sie Ihre Favoritendateien synchronisieren oder die Datei öffnen.

     

    Informationen zum automatischen Synchronisieren Ihrer Favoritendateien finden Sie unter OpenText Tempo-Einstellungen ändern.
