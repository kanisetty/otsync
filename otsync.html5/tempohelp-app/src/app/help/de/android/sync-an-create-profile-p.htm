So erstellen Sie ein Profil:

  1. Wenn Sie erstmalig die Anwendung OpenText Tempo Box starten, tippen Sie auf Profil hinzufügen.

    Wenn Sie ein weiteres Profil erstellen möchten und bereits bei OpenText Tempo Box angemeldet sind, gehen Sie wie folgt vor:

    1. Tippen Sie die Android Menü-Taste.

    2. Tippen Sie auf .

    3. Tippen Sie auf Profil hinzufügen.

  2. Tippen Sie auf das Feld Profilname, und geben Sie einen Profilnamen ein, der nur für Ihr Gerät gilt.

  3. Tippen Sie auf das Feld Benutzername, und geben Sie Ihren Benutzernamen ein. Wenn Sie sich außerhalb des Unternehmens befinden, geben Sie die E-Mail-Adresse ein.

     

    Hinweis:
    Ein Benutzer, der sich außerhalb des Unternehmens befindet, muss zunächst vor dem Erstellen eines Profils ein OpenText Tempo-Konto erstellen.

    Um ein Konto zu erstellen, muss ein Benutzer innerhalb des Unternehmens einen Ordner für den externen Benutzer freigeben. Dadurch wird an den externen Benutzer eine E-Mail-Einladung gesendet, in der die Schritte zum Erstellen eines Kontos beschrieben werden.

     

  4. Tippen Sie auf das Feld Server, und geben Sie den URL Ihres OpenText Tempo Box-Servers ein.

    Den URL des OpenText Tempo Box-Servers erhalten Sie von Ihrem Administrator.

    Wenn Sie sich außerhalb des Unternehmens befinden, finden Sie die OpenText Tempo Box-URL in der E-Mail, die Sie nach dem Erstellen des OpenText Tempo Box-Kontos erhalten haben.

  5. Tippen Sie auf Profil hinzufügen. Bei der Anmeldung bei OpenText Tempo Box werden Sie dann zur Eingabe Ihres Kennworts aufgefordert.

    Zu den Profildaten gehören Ihre Kontoanmeldedaten, der OpenText Tempo Box-URL und eine Liste Ihrer Favoritendateien. Profile gelten speziell für das mobile Gerät. Sie müssen auf jedem Ihrer Smartphones und Tablets mit der OpenText Tempo Box-Anwendung ein Profil erstellen und Dateien als Favoriten markieren.
