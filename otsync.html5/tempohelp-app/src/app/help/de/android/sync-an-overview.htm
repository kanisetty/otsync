Zusätzliche Symbole

Die folgenden Symbole werden auf der Hauptbenutzerschnittstelle angezeigt, wenn das Android-Betriebssystem Version 4.0 oder höher installiert ist. Bei früheren Versionen sind diese Funktionen im Menü Android verfügbar.

Symbol Beschreibung
Fügen Sie die zuletzt ausgeschnittene oder kopierte Datei ein. Dieses Symbol wird nur angezeigt, nachdem das Symbol zum Ausschneiden oder Kopieren ausgewählt wurde. Dieses Symbol wird auch angezeigt, wenn eine Datei aus einer anderen Anwendung importiert wird.
Durchsuchen Sie den aktuellen Ordner oder Unterordner.
Fügen Sie ein Foto oder Ordner von Ihrem Gerät hinzu, bzw. laden Sie eine Datei von Ihrem Android Gerät hoch.
Aktualisiert die Bildschirmanzeige.
Öffnen Sie den Bildschirm Einstellungen vom Bildschirm Profil aus.
