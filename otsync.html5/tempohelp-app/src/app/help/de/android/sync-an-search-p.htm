Tempo Box-Ordner durchsuchen

So durchsuchen Sie Ihren Tempo Box-Ordner:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielordner, von dem aus Sie die Suche starten möchten.

  2. Führen Sie je nach Android Betriebssystem einen der folgenden Schritte aus:

    • Tippen Sie auf die Android Menü-Taste und anschließend auf Suche.

    • Tippen Sie die Android Suchen-Taste.

  3. Geben Sie Ihre Suchzeichenfolge in das Suchfeld ein.

  4. Tippen Sie auf Los. Eine Liste der Suchergebnisse wird angezeigt.

  5. Führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf die Datei oder den Ordner, die bzw. den Sie anzeigen möchten.

    • Berühren und halten Sie die Datei oder den Ordner, um die verfügbaren Optionen anzuzeigen.

So öffnen Sie den Speicherort einer Datei aus den Suchergebnissen:

  1. Durchsuchen Sie den Tempo Box-Ordner.

  2. Berühren und halten Sie die Datei oder den Ordner, die/den Sie am Dateispeicherort öffnen möchten.

  3. Tippen Sie auf Speicherort öffnen.
