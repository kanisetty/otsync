Dateien oder Ordner löschen

So löschen Sie eine Datei:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei.

    Sie können eine Datei nicht aus einem freigegebenen Ordner löschen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

    Sie können eine Datei nicht löschen, solange sie reserviert ist. Reservierte Dateien sind durch die Überlagerung gekennzeichnet. Um eine reservierte Datei zu löschen, müssen Sie erst die Reservierung aufheben. Weitere Informationen finden Sie unter Dateien reservieren und Dateireservierungen aufheben.

     

    Hinweis:
    Bevor Sie fortfahren, sehen Sie sich unten die Warn- Hinweise an.

     

  2. Berühren und halten Sie die Datei, die Sie löschen möchten.

  3. Tippen Sie auf Löschen.

  4. Tippen Sie auf Ok, um diesen Vorgang zu bestätigen.

So löschen Sie einen Ordner:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielordner.

    Wenn es sich beim Zielordner um einen freigegebenen Ordner handelt, schauen Sie unter Freigegebenen Ordner löschen. Freigegebene Ordner sind durch die Überlagerung (Lese-/Schreibzugriff) oder (nur Lesezugriff) gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

    Sie können einen Ordner nicht löschen, wenn dieser eine reservierte Datei enthält. Weitere Informationen zum Aufheben der Reservierung von Dateien finden Sie unter Dateien reservieren und Dateireservierungen aufheben.

     

    Hinweis:
    Bevor Sie fortfahren, sehen Sie sich unten die Warn- Hinweise an.

     

  2. Berühren und halten Sie den Ordner, den Sie löschen möchten.

  3. Tippen Sie auf Löschen.

  4. Tippen Sie auf Ok, um diesen Vorgang zu bestätigen.

 

Achtung

  • Durch Löschen einer Datei oder eines Ordners wird diese(r) aus dem Tempo Box-Ordner entfernt, selbst wenn die Datei als Favorit markiert ist. Das heißt, die Datei oder der Ordner wird aus allen anderen installierten OpenText Tempo Box-Clients, z. B. dem Desktop-Client, oder von Smartphones oder Tablets sowie aus der OpenText Tempo Box-Web-Benutzeroberfläche gelöscht.

  • Wenn Sie eine Datei oder einen Ordner aus einem freigegebenen Ordner löschen, für den Sie Lese- und Schreibrechte besitzen, wird die Datei oder der Ordner automatisch für alle anderen Benutzer gelöscht, die Zugriff auf den freigegebenen Ordner haben, einschließlich des Eigentümers des freigegebenen Ordners. Freigegebene Ordner mit Lese-/Schreibzugriff sind durch die Überlagerung gekennzeichnet.

  • Wenn Sie eine Datei oder einen Ordner aus einem freigegebenen Ordner löschen, für den Sie Lese- und Schreibrechte besitzen, wird die Datei oder der Ordner automatisch aus dem Ordner im Unternehmensarbeitsbereich entfernt. Freigegebene Ordner mit Lese- und Schreibrechten sind gekennzeichnet durch .
