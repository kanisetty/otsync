So melden Sie sich an:

  1. Starten Sie die Anwendung OpenText Tempo Box. Um auf Ihren OpenText Tempo Box-Ordner zugreifen zu können, müssen Sie vor der Anmeldung zuerst ein Profil erstellen. Weitere Informationen hierzu finden Sie unter Profil erstellen.

  2. Tippen Sie auf Ihren OpenText Tempo Box-Profilnamen.

  3. Tippen Sie auf das Feld Kennwort, wenn Sie dazu aufgefordert werden, und geben Sie Ihr Kennwort ein.

    Wenn Sie sich außerhalb des Unternehmens befinden, geben Sie das Kennwort ein, das Sie beim Erstellen Ihres OpenText Tempo Box-Kontos ausgewählt haben. Falls Sie sich nicht an Ihr Kennwort erinnern können, finden Sie diese Informationen in der E-Mail, die Sie beim Erstellen Ihres OpenText Tempo Box-Kontos erhalten haben.

     

    Hinweis:
    Für das Kennwort muss Groß-/Kleinschreibung beachtet werden. Stellen Sie sicher, dass Sie das Kennwort korrekt eingeben.

     

  4. Optional Tippen Sie auf das Kontrollkästchen Kennwort speichern, um beim nächsten Start der Anwendung einen sofortigen Zugriff zuzulassen. Dies gilt nicht, wenn Sie sich von OpenText Tempo Box abmelden oder die Sitzung abläuft. Lesen Sie die Informationen unter Achtung weiter unten.

     

    Hinweis:
    Dieses Kontrollkästchen ist nicht verfügbar, wenn das Gerät offline ist.

     

  5. Tippen Sie auf Anmelden.

 

Achtung

  • Durch das Speichern Ihres Kennworts in OpenText Tempo Box kann Ihr Datenschutz gefährdet werden, wenn Ihr Gerät von einer anderen Person benutzt wird.

  • Wenn auf Ihrem Gerät mehrere Profile konfiguriert sind, können Benutzer anderer Profile auf Ihr OpenText Tempo Box-Konto zugreifen, wenn das Kontrollkästchen Kennwort speichern aktiviert wurde und Sie sich nicht abgemeldet haben. Tippen Sie auf die Android Menü-Taste und anschließend auf , um sich abzumelden.

 

Weitere Informationen zum Verwalten Ihres Profils finden Sie unter Anmelden und Profil verwalten.
