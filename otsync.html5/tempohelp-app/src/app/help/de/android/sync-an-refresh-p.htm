Anzeige aktualisieren

So aktualisieren Sie die Anzeige:

  1. Tippen Sie auf Tempo Box.

  2. Tippen Sie auf die Android Menü-Taste und anschließend auf Aktualisieren.

    Damit wird die Anzeige aktualisiert. Es werden jedoch weder die Dateien heruntergeladen, noch Ihre Favoriten synchronisiert. Um die neueste Version einer Datei herunterzuladen, öffnen Sie sie. Informationen zum Synchronisieren Ihrer Favoriten finden Sie unter Favoriten hinzufügen, anzeigen, entfernen und synchronisieren .
