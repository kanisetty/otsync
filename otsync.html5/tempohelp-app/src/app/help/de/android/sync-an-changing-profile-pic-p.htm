So ändern Sie Ihr Profilbild:

  1. Tippen Sie auf die Android Menü-Taste und anschließend auf .

  2. Tippen Sie auf die Zeile Profilfoto, und führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf Von Kamera, um das Profilbild mithilfe der Android-Kamera festzulegen. Nehmen Sie das Foto auf, und tippen Sie dann auf Speichern.

    • Tippen Sie auf Aus Galerie, um Ihr Profilbild mit einem Foto aus der Android Fotogalerie festzulegen. Navigieren Sie zu dem Bild, das Sie verwenden möchten, und tippen Sie dann auf das Bild.

    • Tippen Sie auf Als Standard festlegen, um als Profilbild das standardmäßige OpenText Tempo Box Profilbild zu verwenden.

  3. Tippen Sie auf die Android Zurück-Taste, um zum Bildschirm Einstellungen zurückzukehren.

  4. Tippen Sie auf die Android Zurück-Taste, um zu Ihren Dateien zurückzukehren.
