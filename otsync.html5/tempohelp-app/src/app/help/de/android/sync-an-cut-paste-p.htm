Dateien oder Ordner verschieben oder kopieren

 

So verschieben oder kopieren Sie eine Datei:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei.

    Sie können eine Datei nur aus einem freigegebenen Ordner kopieren, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

    Außerdem können Sie nur Dateien kopieren, die von einem anderen Teilnehmer in einem freigegebenen Ordner reserviert sind, für den Sie Schreib-/Leserechte haben. Freigegebene Ordner mit Lese-/Schreibzugriff sind durch die Überlagerung gekennzeichnet. Reservierte Dateien sind durch die Überlagerung gekennzeichnet. Anhand des Verlaufs der Datei können Sie feststellen, wer die Reservierung vorgenommen hat. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  2. Berühren und halten Sie die Datei, die Sie verschieben oder kopieren möchten.

  3. Tippen Sie auf Ausschneiden, um die Datei zu verschieben, oder auf Kopieren, um die Datei zu kopieren.

  4. Navigieren Sie zum Zielordner, in dem Sie den Inhalt der Zwischenablage platzieren möchten.

    Sie können eine Datei nicht in einen freigegebenen Ordner verschieben oder kopieren, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet.

  5. Tippen Sie auf die Android Menü-Taste und anschließend auf Einfügen.

So verschieben oder kopieren Sie einen Ordner:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielordner.

  2. Berühren und halten Sie den Ordner, den Sie verschieben oder kopieren möchten.

  3. Tippen Sie auf Ausschneiden, um den Ordner zu verschieben, oder auf Kopieren, um den Ordner zu kopieren.

     

    Hinweis:
    Bevor Sie fortfahren, sehen Sie sich unten die Hinweise unten.

     

  4. Navigieren Sie zum Zielordner, in dem Sie den Inhalt der Zwischenablage platzieren möchten.

    Sie können einen Ordner nicht in einen freigegebenen Ordner verschieben oder kopieren, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

  5. Tippen Sie auf die Android Menü-Taste und anschließend auf Einfügen.

 

Hinweise:

  • Sie können freigegebene Ordner, die Ihnen gehören, an eine beliebige Stelle im Tempo Box-Ordner verschieben, außer in einen anderen freigegebenen Ordner. Freigegebene Ordner, deren Eigentümer Sie sind, zeigen nur den Ordnernamen an und sind gekennzeichnet als .

  • Sie können nur das Stammverzeichnis eines Ordners kopieren, der für Sie freigegeben ist. Dabei wird eine private Kopie des Ordners an einem neuen Speicherort erstellt. Der ursprüngliche freigegebene Ordner bleibt jedoch auf der Stammverzeichnisebene Ihres Tempo Box-Ordners. An Ordner, die für Sie freigegeben sind, wird der Benutzername des Ordnereigentümers im Ordnernamen angehängt, und sie sind als (Lese-/Schreibzugriff) oder (nur Lesezugriff) gekennzeichnet.

  • Sie können nur das Stammverzeichnis eines Unternehmensordners kopieren, unabhängig von Ihrer Zugriffsebene. Sie können ferner nur Unterordner in einem Unternehmensordner kopieren, für den Sie nur Lesezugriff haben. Dabei wird eine private Kopie des Ordners an einem neuen Speicherort erstellt. Der ursprüngliche Unternehmensordner bleibt jedoch am ursprünglichen Speicherort in Ihrem Tempo Box-Ordner. Unternehmensordner werden als (Lese-/Schreibzugriff) oder (nur Lesezugriff) gekennzeichnet.

  • Sie können nur das Stammverzeichnis einer Dateibibliothek in einer Tempo Social-Community kopieren (gekennzeichnet als (Lese-/Schreibzugriff) oder (nur Lesezugriff)). Dabei wird eine private Kopie der Dateibibliothek an einem neuen Speicherort erstellt. Die ursprüngliche Dateibibliothek bleibt jedoch auf der Stammverzeichnisebene Ihres Tempo Box-Ordners.
