Freigegebenen Ordner löschen

So löschen Sie einen freigegebenen Ordner, dessen Eigentümer Sie sind:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum freigegebenen Zielordner.

    Freigegebene Ordner mit Lese-/Schreibzugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

    Wenn der Name des Zielordners mit einem Benutzernamen versehen wird, wird der Zielordner für Sie freigegeben. Weitere Informationen finden Sie unter So löschen Sie einen Ordner, der für Sie freigegeben wurde.

  2. Berühren und halten Sie den freigegebenen Ordner.

  3. Tippen Sie auf Löschen. Wenn Freigegebenen Ordner entfernen angezeigt wird, haben Sie einen Ordner, der für Sie freigegeben ist, einen Unternehmensordner oder eine Dateibibliothek aus einer Tempo Social-Community ausgewählt. Weitere Informationen finden Sie unter So löschen Sie einen Ordner, der für Sie freigegeben wurde, Unternehmensordner entfernen oder Eine Dateibibliothek aus einer Tempo Social-Community entfernen.

     

    Achtung
    Dadurch wird der Ordner für Sie und alle Teilnehmer gelöscht. Alternativ können Sie Teilnehmer entfernen. Weitere Informationen finden Sie unter Einen Teilnehmer entfernen.

     

  4. Tippen Sie auf Ok, um den Vorgang zu bestätigen.

So löschen Sie einen Ordner, der für Sie freigegeben wurde:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielordner.

    Ein Ordner, der für Sie freigegeben wurde, wird mit dem Benutzernamen des Ordnereigentümers im Ordnernamen angezeigt (beispielsweise Marketingmaterialien (jsmith))und ist als oder gekennzeichnet.

  2. Berühren und halten Sie den freigegebenen Ordner.

  3. Tippen Sie auf Freigegebenen Ordner entfernen.

    Hierdurch wird die Verbindung zum freigegebenen Ordner getrennt und der Ordner aus Ihrem Tempo Box-Ordner entfernt. Für den Eigentümer und andere Teilnehmer wird der Ordner nicht gelöscht.

    Wenn die Schaltfläche Löschen angezeigt wird, haben Sie einen freigegebenen Ordner ausgewählt, dessen Eigentümer Sie sind. Bevor Sie fortfahren, informieren Sie sich unter So löschen Sie einen freigegebenen Ordner, dessen Eigentümer Sie sind.

  4. Tippen Sie auf Ok, um den Vorgang zu bestätigen.

So entfernen Sie einen Unternehmensordner:

  1. Suchen Sie auf der untersten Ebene Ihrer Tempo Box-Ordnerstruktur nach dem Zielunternehmensordner. Unternehmensordner werden gekennzeichnet als oder .

  2. Berühren und halten Sie den Unternehmensordner.

  3. Tippen Sie auf Freigegebenen Ordner entfernen.

    Hierdurch wird die Verbindung zum Ordner getrennt und der Ordner aus Ihrem Tempo Box-Ordner entfernt. Der Ordner im Unternehmensarbeitsbereich wird nicht gelöscht.

     

    Achtung
    Sie müssen den tatsächlichen Ordner entfernen und dürfen den Inhalt im Unternehmensordner nicht löschen. Wenn Sie den Inhalt im Unternehmensordner löschen, wird der Inhalt für alle Teilnehmer, die sich für den Ordner angemeldet haben, sowie der Inhalt im Unternehmensarbeitsbereich gelöscht.

     

  4. Tippen Sie auf Ok, um den Vorgang zu bestätigen.

So entfernen Sie eine Dateibibliothek aus einer Tempo Social-Community:

  1. Suchen Sie auf der untersten Ebene Ihrer Tempo Box-Ordnerstruktur nach der Zieldateibibliothek. Dateibibliotheken werden gekennzeichnet als oder .

  2. Berühren und halten Sie die Dateibibliothek.

  3. Tippen Sie auf Freigegebenen Ordner entfernen.

    Hierdurch wird die Verbindung zur Dateibibliothek getrennt und der Ordner aus Ihrem Tempo Box-Ordner entfernt. Dadurch wird die Dateibibliothek nicht aus Tempo Social gelöscht. Wenn Sie die Dateibibliothek versehentlich gelöscht haben, navigieren Sie zur Community in Tempo Social, verlassen Sie die Community, und treten Sie der Community dann wieder bei. Weitere Informationen zu Tempo Social-Communitys finden Sie unter OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).

  4. Tippen Sie auf Ok, um den Vorgang zu bestätigen.
