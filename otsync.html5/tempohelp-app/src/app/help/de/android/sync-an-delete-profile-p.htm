Eigenes Profil löschen

 

So löschen Sie ein Profil:

  1. Berühren und halten Sie auf der Seite Profil das Profil, das Sie entfernen möchten.

     

    Tipp:
    Wenn Sie angemeldet sind, tippen Sie auf die Taste Android Zurück, um zur Seite Profil zurückzukehren.

     

  2. Tippen Sie auf Löschen.

  3. Tippen Sie auf Löschen, um diesen Vorgang zu bestätigen.
