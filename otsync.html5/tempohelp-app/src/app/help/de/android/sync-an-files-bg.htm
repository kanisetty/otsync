Dateien anzeigen, bearbeiten und verwenden

 

So zeigen Sie eine Datei an:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei. In der OpenText Tempo Box-Titelleiste wird der Name des aktuell angezeigten Ordners angezeigt.

  2. Optional Um sicherzugehen, dass Sie den aktuellsten Inhalt anzeigen, sollten Sie die Anzeige regelmäßig aktualisieren. Weitere Informationen finden Sie unter Anzeige aktualisieren.

  3. Führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf die Datei, die Sie anzeigen möchten. Eine Liste von Apps, die diese Datei öffnen können, wird angezeigt. Tippen Sie auf eine App.

    • Berühren und halten Sie die Datei, die Sie anzeigen möchten. Tippen Sie auf Öffnen und dann auf eine App.

     

    Hinweis:
    Um für einen bestimmten MIME-Typ eine Standard-App festzulegen, aktivieren Sie das Kontrollkästchen Standardmäßig für diese Aktion verwenden, bevor Sie eine App wählen.

     

  4. Tippen Sie auf die Android Zurück-Taste, um zu Ihren Dateien zurückzukehren.

So bearbeiten Sie eine Datei:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei.

     

    Wichtig:
    Bei der Anzeige von Dateien in einem freigegebenen Ordner, für den Sie nur Leserechte besitzen, werden diese Dateien nicht automatisch reserviert. Sie können eine Datei zur selben Zeit öffnen und bearbeiten, zu der ein anderer Teilnehmer Änderungen daran vornimmt. Um dies zu verhindern, können Sie die Datei reservieren. Weitere Informationen finden Sie unter Dateien reservieren und Dateireservierungen aufheben.

    Freigegebene Ordner mit Lese-/Schreibzugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

     

    Sie können keine Bearbeitungen in eine Datei speichern, solange diese von einem anderen Teilnehmer in einem freigegebenen Ordner reserviert ist. Reservierte Dateien sind durch die Überlagerung gekennzeichnet. Anhand des Verlaufs der Datei können Sie feststellen, wer die Reservierung vorgenommen hat. Weitere Informationen finden Sie unter Objektdetails anzeigen.

    In einer Datei in einem freigegebenen Ordner, für den Sie nur Leserechte besitzen, können Sie zudem keine Änderungen speichern. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet.

  2. Tippen Sie auf die zu bearbeitende Datei, und wählen Sie einen Editor eines Drittanbieters aus.

  3. Wenn die App das Bearbeiten unterstützt, bearbeiten Sie die Datei, und speichern Sie Ihre Änderungen.

    Die Änderungen, die Sie an der Datei vorgenommen haben, werden nicht in der Datei im Tempo Box-Ordner gespeichert. Sie müssen die geänderte Datei hochladen oder hinzufügen.

  4. Um die geänderte Datei in Ihren Tempo Box-Ordner hochzuladen oder einzufügen, führen sie eine der folgenden Aktionen aus:

    • Speichern Sie die Datei von der Drittanbieter-App aus auf Ihrem Android-Gerät. Fügen Sie in OpenText Tempo Box eine Version zur vorhandenen Datei hinzu. Weitere Informationen finden Sie unter Dateien hinzufügen.

    • Laden Sie die gespeicherte Datei von der Drittanbieter-App mithilfe der Option Freigeben oder Exportieren in Ihren Tempo Box-Ordner hoch. Auf diese Weise wird Ihrem Tempo Box-Ordner eine zweite Datei hinzugefügt. Weitere Informationen finden Sie unter Dateien hinzufügen.

Informationen zur Verwaltung Ihrer Dateien finden Sie in den anderen Themen unter Dateien anzeigen, bearbeiten und verwenden.
