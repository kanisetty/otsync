Dateien reservieren und Dateireservierungen aufheben

So reservieren Sie Dateien und heben Dateireservierungen auf:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei.

    Sie können eine Datei nicht in einem freigegebenen Ordner reservieren, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

  2. Berühren und halten Sie die Datei, die Sie reservieren oder deren Reservierung Sie aufheben möchten.

  3. Führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf Reservieren, um die Datei zu reservieren. Die Datei wird mit der Überlagerung gekennzeichnet.

      Wenn Sie versuchen, eine Datei in einem Unternehmensordner zu reservieren, für den Sie über die entsprechenden Schreib- und Leserechte verfügen und die Option Reservieren nicht angezeigt wird, haben Sie keine Berechtigung zum Reservieren für den Ordner in Content Server. Wenden Sie sich an Ihren Administrator, um diese Berechtigung anzufordern.

    • Tippen Sie auf Reservierung aufheben, um die Dateireservierung aufzuheben. Wenn Reservierung aufheben nicht verfügbar ist, ist die Datei von einem anderen Benutzer reserviert. Anhand des Verlaufs der Datei können Sie feststellen, wer die Reservierung vorgenommen hat. Weitere Informationen finden Sie unter Objektdetails anzeigen.
