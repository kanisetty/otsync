Objektdetails anzeigen

 

So zeigen Sie Objektdetails an:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei oder zum Zielordner.

  2. Berühren und halten Sie die Datei oder den Ordner.

  3. Tippen Sie auf Objektdetails.

  4. Führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf Eigenschaften, um Details zum Objekt anzuzeigen.

    • Tippen Sie auf Kategorien, um die Kategorien und Attribute anzuzeigen, die dem Objekt zugewiesen sind.

    • Tippen Sie auf Protokoll, um den Protokollverlauf des Objekts anzuzeigen.

  5. Tippen Sie auf die Android Zurück-Taste, um zu Ihren Dateien zurückzukehren.
