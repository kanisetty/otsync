Fotos oder Ordner hinzufügen

So fügen Sie ein Foto mit der Android-Kamera hinzu:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielordner, wo Sie ein Foto hinzufügen möchten.

    Sie können ein Foto nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

    Über Tempo Box können Sie keine Fotos zu Ordnern hinzufügen, die über obligatorische Kategorien oder Attribute verfügen. Zur Festlegung siehe Objektdetails anzeigen.

  2. Tippen Sie die Android Menü-Taste. Tippen Sie auf Hinzufügen > Von Kamera.

  3. Nehmen Sie ein Foto auf, und folgen Sie den Anweisungen auf dem Bildschirm. Fotos werden als JPG-Dateien gespeichert. Sie können den Namen des Fotos vor dem Hochladen nicht bearbeiten.

So importieren Sie ein Foto aus der Galerie auf dem Gerät:

  1. Starten Sie die Android-Galerie-App.

  2. Navigieren Sie zu dem Foto, das Sie importieren möchten.

  3. Tippen Sie auf die Android Menü-Taste und anschließend auf Freigeben.

  4. Tippen Sie auf Tempo Box.

  5. Navigieren Sie zum Zielordner, wo Sie ein Foto hinzufügen möchten.

    Sie können ein Foto nicht in einen freigegebenen Ordner importieren, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

    Über Tempo Box können Sie keine Fotos zu Ordnern hinzufügen, die über obligatorische Kategorien oder Attribute verfügen. Zur Festlegung siehe Objektdetails anzeigen.

  6. Tippen Sie auf die Android Menü-Taste und anschließend auf Einfügen.

So fügen Sie einen Ordner hinzu:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielspeicherort für den neuen Ordner.

    Sie können einen Ordner nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

  2. Tippen Sie die Android Menü-Taste. Tippen Sie auf Hinzufügen > Ordner.

  3. Tippen Sie auf das Feld Ordnername, und geben Sie einen Namen für diesen Ordner ein.

  4. Tippen Sie auf Ok.
