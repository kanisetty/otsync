Dateien in freigegebenen Ordnern anzeigen

So zeigen Sie Dateien in freigegebenen Ordnern an:

  1. Tippen Sie auf Benachrichtigungen. Ihre Benachrichtigungen können Einladungen zu freigegebenen Ordnern oder Unternehmensordnern umfassen. Weitere Informationen zu freigegebenen Ordnern und Unternehmensordnern finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

    Bevor Sie den Inhalt eines freigegebenen Ordners anzeigen können, muss der Ordnereigentümer den Ordner für Sie freigeben. Bei Unternehmensordnern kann ein Benutzer, der die Berechtigung Berechtigungen bearbeiten für einen Unternehmensordner hat, Ihnen eine Freigabeeinladung für den Ordner senden. Bei beiden Aktivitäten erhalten Sie eine Freigabeeinladung, die Sie annehmen müssen, bevor Sie den Inhalt im Ordner anzeigen können. Weitere Informationen zur Freigabe eines Ordners finden Sie unter Ordner freigeben und freigegebene Ordner verwalten.

     

    Tipp:
    Sie haben eine Freigabeeinladung erhalten, wenn eine Benachrichtigung angezeigt wird. Zum Beispiel: .

     

  2. Optional Um sicherzustellen, dass sämtliche Freigabeeinladungen angezeigt werden, aktualisieren Sie die Anzeige regelmäßig. Weitere Informationen finden Sie unter Anzeige aktualisieren.

  3. Tippen Sie auf eine Freigabeeinladung.

  4. Tippen Sie auf Annehmen.

    Der freigegebene Ordner wird im Stammverzeichnis Ihres Tempo Box-Ordners platziert und als (Lese-/Schreibzugriff) oder (Nur Lesezugriff) angezeigt.

    Mit Lese- und Schreibrechten können Sie den Inhalt in einem freigegebenen Ordner aufrufen und bearbeiten, so als ob es Ihr eigener wäre. Sämtliche Änderungen am Inhalt eines freigegebenen Ordners werden mit allen anderen Teilnehmern synchronisiert, die Zugriff auf diesen Ordner haben, einschließlich des Eigentümers.

    Mit Leserecht können Sie den Inhalt im freigegebenen Ordner nur lesen und kopieren.

    Dem Ordnernamen wird der Benutzername des Ordnereigentümers angehängt. Zum Beispiel: Marketingmaterialien (jsmith).

    Wenn Sie eine Freigabeeinladung für einen Unternehmensordner angenommen haben, wird der Ordner im Stammverzeichnis Ihres Tempo Box-Ordners platziert und als (Lese-/Schreibzugriff) oder (Nur Lesezugriff) angezeigt.

    Für den Unternehmensordner wird evtl. der Name des übergeordneten Ordners an den Namen des Ordners angehängt, je nach Ihren Berechtigungen für den übergeordneten Ordner. Zum Beispiel: Gehalt (Personal).
