Beschreibung der OpenText Tempo Box-Symbole

Die folgenden Symbole werden auf der Hauptbenutzerschnittstelle angezeigt, wenn das Android-Betriebssystem Version 4.0 oder höher installiert ist. Bei früheren Versionen sind diese Funktionen im Menü Android verfügbar.

Öffnen Sie die Seite Einstellungen von der Seite Profil aus.

Symbole für freigegebene Ordner

OpenText Tempo Box bietet die folgenden Typen von freigegebenen Ordnern.

Symbol Beschreibung
Dieses Ordnersymbol deutet auf einen freigegebenen Ordner hin, für den Sie Lese-/Schreibzugriff haben. Ist der Ordnername um einen Benutzernamen ergänzt, wurde er für Sie freigegeben. Zum Beispiel: Marketingmaterialien (jsmith). Weitere Informationen finden Sie unter Dateien in freigegebenen Ordnern anzeigen.
Eine Schlossüberlagerung steht für einen freigegebenen Ordner, für den Sie nur Lesezugriff haben. Weitere Informationen finden Sie unter Dateien in freigegebenen Ordnern anzeigen.
Dieses Symbol deutet auf einen Unternehmensordner hin, für den Sie Lese-/Schreibzugriff haben. Sie müssen sich beim Unternehmensordner in Content Server anmelden, oder eine Person mit der Berechtigung Berechtigungen bearbeiten für den Ordner muss Ihnen eine Freigabeeinladung senden, damit Sie auf den Inhalt aus Ihrem Tempo Box-Ordner zugreifen können. Weitere Informationen finden Sie unter Dateien in freigegebenen Ordnern anzeigen.
Eine Schlossüberlagerung auf diesem Ordnersymbol repräsentiert einen Unternehmensordner, für den Sie nur Lesezugriff haben. Sie müssen sich beim Unternehmensordner in Content Server anmelden, oder eine Person mit der Berechtigung Berechtigungen bearbeiten für den Ordner muss Ihnen eine Freigabeeinladung senden, damit Sie auf den Inhalt aus Ihrem Tempo Box-Ordner zugreifen können. Weitere Informationen finden Sie unter Dateien in freigegebenen Ordnern anzeigen .
Dieses Symbol repräsentiert eine Dateibibliothek in einer Tempo Social-Community, für die Sie Lese-/Schreibzugriff haben. Sie müssen Mitglied einer Tempo Social-Community sein, um auf die Dateibibliotheken aus Ihrem Tempo Box-Ordner zugreifen zu können. Weitere Informationen zu Tempo Social-Communitys finden Sie unter OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social ist ein separates Produkt von OpenText Tempo Box. Wenn Tempo Social installiert ist und Sie die Dateibibliotheken in Ihrem Tempo Box-Ordner nicht sehen, kontaktieren Sie Ihren Administrator.
Eine Schlossüberlagerung auf diesem Ordnersymbol repräsentiert eine Dateibibliothek in einer Tempo Social-Community, für die Sie nur Lesezugriff haben. Weitere Informationen hierzu finden Sie unter OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
Eine Schlossüberlagerung auf einer Datei repräsentiert eine Datei in einem freigegebenen Ordner nur mit Lesezugriff.
Eine Häkchenüberlagerung auf einer Datei repräsentiert eine Datei, die reserviert ist.
