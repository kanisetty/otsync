Dateien hinzufügen

 

So fügen Sie eine neue Version mithilfe einer Datei auf Ihrem Gerät hinzu:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zur Zieldatei.

     

    Wichtig:
    Beim Einfügen von Dateien in einen freigegebenen Ordner, für den Sie Lese- und Schreibrechte besitzen, wird die Datei nicht automatisch reserviert. Sie können eine eine neue Version der zur selben Zeit einfügen, zu der ein anderer Teilnehmer Änderungen daran vornimmt. Um dies zu verhindern, können Sie die Datei reservieren. Weitere Informationen finden Sie unter Dateien reservieren und Dateireservierungen aufheben.

    Freigegebene Ordner mit Lese-/Schreibzugriff sind durch die Überlagerung gekennzeichnet. Weitere Informationen finden Sie unter Beschreibung der OpenText Tempo Box-Symbole.

     

    Sie können keine neue Version zu einer Datei hinzufügen, solange diese von einem anderen Teilnehmer in einem freigegebenen Ordner reserviert ist. Reservierte Dateien sind durch die Überlagerung gekennzeichnet. Anhand des Verlaufs der Datei können Sie feststellen, wer die Reservierung vorgenommen hat. Weitere Informationen finden Sie unter Objektdetails anzeigen.

    In eine Datei in einem freigegebenen Ordner, für den Sie nur Leserechte besitzen, können Sie zudem keine neue Version einfügen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet.

  2. Berühren und halten Sie die Datei, der Sie eine neue Version hinzufügen möchten.

  3. Tippen Sie auf Version hinzufügen > Von Gerät.

  4. Navigieren Sie zur neuen Version der Datei auf Ihrem Gerät.

  5. Tippen Sie auf die neue Version der Datei.

So laden Sie eine auf Ihrem Gerät gespeicherte Datei hoch:

  1. Tippen Sie auf Tempo Box, und navigieren Sie zum Zielordner, wo Sie eine Datei hochladen möchten.

    Sie können eine Datei nicht in einen freigegebenen Ordner hochladen, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet.

    Über Tempo Box können Sie keine Dateien zu Ordnern hinzufügen, die über obligatorische Kategorien oder Attribute verfügen. Zur Festlegung siehe Objektdetails anzeigen.

  2. Berühren und halten Sie den Zielordner. Tippen Sie auf Hier hochladen > Von Gerät.

  3. Navigieren Sie zur Zieldatei auf Ihrem Android-Gerät.

  4. Tippen Sie auf die Datei.

So importieren Sie eine Datei aus einer anderen App:

  1. Zeigen Sie in einer anderen App eine Datei an.

  2. Tippen Sie auf Exportieren oder Freigeben.

  3. Tippen Sie auf Tempo Box.

  4. Optional Wenn Sie dazu aufgefordert werden, melden Sie sich bei OpenText Tempo Box an.

  5. Navigieren Sie zum Zielspeicherort, an dem Sie die Datei platzieren möchten.

    Sie können keine Datei in einen freigegebenen Ordner importieren, für den Sie nur Leserechte besitzen. Freigegebene Ordner mit reinem Lesezugriff sind durch die Überlagerung gekennzeichnet.

    Über Tempo Box können Sie keine Dateien zu Ordnern hinzufügen, die über obligatorische Kategorien oder Attribute verfügen. Zur Festlegung siehe Objektdetails anzeigen.

  6. Tippen Sie auf die Android Menü-Taste und anschließend auf Einfügen.
