OpenText Tempo-Einstellungen ändern

So ändern Sie die OpenText Tempo Box-Einstellungen:

  1. Tippen Sie auf die Android Menü-Taste und anschließend auf Einstellungen.

  2. Optional Tippen Sie auf die Zeile Standardsortierreihenfolge und dann auf Datum oder Name, um Ihre Dateien nach dem Dateinamen oder dem Bearbeitungsdatum der Datei zu sortieren.

    Sie müssen die Ordneranzeige aktualisieren, um diese Änderung sehen zu können. Weitere Informationen finden Sie unter Anzeige aktualisieren.

  3. Optional Tippen Sie auf die Zeile Speicher und anschließend auf ein neues Speicherlimit für Ihre Dateien im Zwischenspeicher. Wenn Sie das Speicherlimit festlegen, werden die ältesten zwischengespeicherten Dateien gelöscht, bis die Größe des Zwischenspeichers unterhalb des festgelegten Limits liegt. Die Standardgröße des Zwischenspeichers beträgt 1 GB.

     

    Hinweise:

    • In der Zeile Verwendeter Speicherplatz wird die Gesamtgröße Ihrer Favoritendateien, nicht die Ihrer Dateien im Zwischenspeicher, angezeigt.

    • In der Zeile Kontingentbedarf wird das Kontingent und die Größe Ihres Tempo Box-Ordners angezeigt. Davon ausgeschlossen sind freigegebene Ordner, deren Eigentümer Sie nicht sind.

     

  4. Optional Tippen Sie auf die Zeile Favoriten synchronisieren und anschließend auf Auto oder Manuell.

    Bei automatischer Synchronisierung geschieht Folgendes:

    • Neue Versionen von Favoritendateien werden im Hintergrund heruntergeladen.

    • Die Datei wird für die Offlineanzeige heruntergeladen, wenn Sie die Datei als Favorit markieren.

    Bei manueller Synchronisierung geschieht Folgendes:

    • Neue Versionen von Favoritendateien werden nicht automatisch heruntergeladen.

    • Sie müssen die Datei öffnen oder Ihre Favoriten manuell synchronisieren, um die Datei für die Offline-Anzeige herunterzuladen. So synchronisieren Sie Ihre Favoriten manuell:

      1. Tippen Sie auf Favoriten.

      2. Tippen Sie auf Menü > Favoriten aktualisieren.

  5. Tippen Sie auf die Android Zurück-Taste, um zu Ihren Dateien zurückzukehren.
