Удаление профиля

 

Удаление профиля:

  1. На странице Мой профиль коснитесь и удерживайте профиль, который требуется удалить.

     

    Подсказка:
    Если вход выполнен, нажмите кнопку Главная для возврата на целевую страницу, затем выберите Настройки > Выход для возращения на страницу Мой профиль.

     

  2. Нажмите Удалить.

  3. Нажмите Удалить, чтобы подтвердить это действие.
