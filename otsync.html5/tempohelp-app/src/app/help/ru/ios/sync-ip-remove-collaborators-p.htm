Удаление пользователя с общим доступом

Удаление пользователя с общим доступом:

  1. Нажмите Tempo Box и перейдите к целевой общей папке.

  2. Нажмите для папки, для которой необходимо удалить пользователя с общим доступом.

  3. Проведите пальцем по горизонтали по имени пользователя, для которого необходимо отменить общий доступ к папке. В строке папки отобразится кнопка Удалить.

    Если после того, как вы проведете пальцем по имени пользователя, ничего не произойдет, это означает, что вы не являетесь владельцем этой папки. Удалять пользователей с совместным доступом может только владелец папки, который указывается в области Общий доступ для.

  4. Коснитесь Удалить. При этом будет разорвано соединение с общей папкой и общая папка будет удалена из папки пользователя с общим доступом Tempo Box.

  5. Коснитесь Назад, чтобы вернуться к своим файлам.
