Создание профиля

  1. Если это первый запуск приложения OpenText Tempo Box коснитесь Добавить профиль.

    Если при нахождении в системе OpenText Tempo Box требуется создать дополнительный профиль, выполните следующие действия.

    1. Коснитесь .

    2. Коснитесь Переключить профили.

    3. Коснитесь надписи Добавить профиль.

  2. Коснитесь поля Имя пользователя и введите имя пользователя. При нахождении вне компании введите свой адрес электронной почты.

     

    Замечание:
    Пользователи, которые находятся за пределами компании, до создания профиля должны сначала создать учетную запись OpenText Tempo Box.

    Для создания учетной записи пользователь в компании должен открыть общий доступ для внешнего пользователя. При этом внешнему пользователю будет отправлено по электронной почте приглашение с описанием шагов, которые необходимо выполнить для создания учетной записи.

     

  3. Коснитесь поля Пароль и введите пароль.

    Если пользователь находится вне компании, введите пароль, выбранный при создании учетной записи OpenText Tempo Box. Если вы забыли пароль, воспользуйтесь сообщением электронной почты, которое было отправлено вам при создании учетной записи OpenText Tempo Box.

     

    Замечание:
    Пароль чувствителен к регистру. Убедитесь, что пароль правильно введен.

     

  4. Коснитесь поля Сервер и введите URL-адрес используемого сервера OpenText Tempo Box.

    Если вы забыли URL-адрес OpenText Tempo Box, обратитесь к администратору.

    Если пользователь находится за пределами компании, URL-адрес OpenText Tempo Box указывается в сообщении, которое было отправлено по электронной почте при создании учетной записи OpenText Tempo Box.

  5. Коснитесь поля Имя профиля и введите имя профиля, которое является уникальным на данном устройстве.

  6. По выбору Установите флажок Запомнить пароль для мгновенного доступа к приложению при его следующем запуске. Единственные исключения: выход из OpenText Tempo Box и истечение времени ожидания сеанса. См. информацию ниже в разделе Caution (Внимание).

  7. Коснитесь Готово. К данным профиля относятся учетные данные учетной записи, URL-адрес OpenText Tempo Box и список избранных файлов. Профили связаны с iPhone и iPad. Необходимо создать профили и пометить с помощью тегов файлы как избранные на всех смартфонах и планшетных ПК, на которых запущено приложение OpenText Tempo Box.

 

Предостережение

  • Настройка в OpenText Tempo Box запоминания пароля может привести к нарушению безопасности при использовании устройства другим пользователем.

  • Если на устройстве настроено несколько профилей, то для других профилей можно разрешить доступ к учетной записи OpenText Tempo Box, установив флажок Запомнить пароль не выполняя выход. Чтобы выйти, коснитесь , а затем Выход.
