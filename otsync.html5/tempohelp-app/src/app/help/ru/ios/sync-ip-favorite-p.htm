Глава 6 – Добавление, просмотр и удаление элементов избранного

Добавление в избранное:

  1. Нажмите Tempo Box и перейдите к целевому файлу. Как личные, так и общие файлы можно пометить тегами как избранные.

  2. Коснитесь файла, который следует отметить как элемент избранного. Файл откроется.

  3. Нажмите . Значок изменится на , и файл будет отмечен наложением в представлении папки. Например, . Пометка файла в качестве избранного позволяет просматривать файл в автономном режиме.

    Запись избранных файлов сохраняется в вашем профиле, относящемся к iPad или iPhone. Необходимо пометить файлы как избранные на всех смартфонах и планшетных ПК с установленным приложением OpenText Tempo.

  4. Коснитесь Назад, чтобы вернуться к своим файлам.

     

    Подсказка:
    Кроме того, можно пометить файл в качестве избранного, коснувшись для файла, а затем нажав Добавить в избранное.

     

Просмотр избранного файла:

  1. Нажмите Элементы избранного.

  2. По выбору Нажмите , чтобы убедиться, что просматривается самое новое содержимое.

  3. Щелкните файл, который хотите просмотреть.

  4. Коснитесь Назад, чтобы вернуться к своим файлам.

Удаление избранного файла:

  1. Нажмите Элементы избранного или Tempo Box, а затем перейдите к целевому файлу.

  2. Коснитесь файла, который следует удалить из списка избранных. Файл откроется.

  3. Нажмите . Значок изменится на . Файл больше не отображается в списке избранного.

    При этом файл не удаляется из папки Tempo Box.

  4. Коснитесь Назад, чтобы вернуться к своим файлам.

     

    Подсказка:
    Кроме того, можно удалить файл из списка избранного, коснувшись для файла, а затем нажав Удалить из элементов избранного.
