Глава 1 – Обзор значков OpenText Tempo Box

В приложении OpenText Tempo Box используются следующие основные значки.

Значок Описание
Осуществляет переход на целевую страницу из приложения.
Позволяет добавить новую папку или фотографию в папку Tempo Box.
Обновляет изображение экрана.
Переход к полноэкранному отображению при работе с iPad в альбомном режиме.
Отображает основные значки OpenText Tempo Box при работе с iPad в портретном режиме.
Позволяет просматривать сведения о папке или предоставлять общий доступ к папке.

Значки файла

При просмотре файла отображаются следующие значки.

Значок Описание
Вырезание открытого файла для перемещения.
Копирование открытого файла.
Вставка последнего вырезанного или скопированного файла. Значок отображается только после выбора значка копирования или вырезания.
Удаление открытого файла.
Файл не отмечен тегом как элемент избранного.
Файл отмечен тегом как элемент избранного. Наложение означает файл, отмеченный тегом как элемент избранного. Например, .
Отправка выбранного файла по электронной почте.
Открытие выбранного файла в другом приложении.

Значки общей папки

OpenText Tempo Box поддерживает следующие типы общих папок.

Значок Описание
Данный значок папки соответствует общей папке с разрешением на чтение и запись. Если имя папки дополнено именем пользователя, у вас есть общий доступ к папке. Например, “Маркетинговые материалы (икузнецов)”. Дополнительные сведения см. в разделе Просмотр файлов в общих папках.
Изображение замка обозначает общую папку с наличием доступа только для чтения. Дополнительные сведения см. в разделе Просмотр файлов в общих папках.
Данный значок папки соответствует папке Enterprise с разрешением на чтение и запись. Вы должны быть подписаны на папку Enterprise на сервере содержимого; альтернативно пользователь, у которого есть разрешение на Редактирование разрешений для этой папки, должен прислать вам приглашение к совместному использованию; тогда у вас будет доступ к содержимому папки Tempo Box. Дополнительные сведения см. в разделе Просмотр файлов в общих папках.
Изображение замка на этом значке папки соответствует папке Enterprise с разрешением только на чтение. Вы должны быть подписаны на папку Enterprise на сервере содержимого; альтернативно пользователь, у которого есть разрешение на Редактирование разрешений для этой папки, должен прислать вам приглашение к совместному использованию; тогда у вас будет доступ к содержимому папки Tempo Box. Чтобы иметь доступ к библиотеке файлов из папки Tempo Social, вы должны быть членом сообщества Tempo Box. Дополнительные сведения о сообществах Tempo Social см. в разделе OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social – это отдельный продукт от OpenText Tempo Box. Если у вас установлен продукт Tempo Social, но вы не видите библиотек файлов в папке Tempo Box, обратитесь к своему администратору.
