Перемещение или копирование файла или папки

Перемещение или копирование файла:

  1. Нажмите Tempo Box и перейдите к целевому файлу.

    Если файл находится в общей папке с разрешением только на чтение, его можно только копировать. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

    Файл, зарезервированный другим пользователем и находящийся в общей папке с разрешением на чтение и запись, можно только копировать. Общие папки с разрешением на чтение и запись обозначаются наложением . Зарезервированные файлы помечены наложением . Чтобы узнать, кем зарезервирован файл, загляните в журнал аудита файла. Подробнее см. в разделе Просмотр сведений об объекте.

  2. Коснитесь необходимого для перемещения или копирования файла.

  3. Нажмите для перемещения файла или для его копирования.

  4. Коснитесь Назад и перейдите к целевой папке, в которую необходимо поместить содержимое буфера.

    Переместить или скопировать файл в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению .

  5. Нажмите для вставки файла.

Перемещение или копирование папки:

  1. Нажмите Tempo Box и перейдите к целевой папке.

  2. Нажмите , чтобы просмотреть сведения о папке, которую необходимо переместить или скопировать.

  3. Нажмите для перемещения папки или для ее копирования.

     

    Замечание:
    Перед тем как продолжить, см. раздел Примечания.

     

  4. Коснитесь Назад и перейдите к целевой папке, в которую необходимо поместить содержимое буфера.

    Переместить или скопировать папку в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

  5. Нажмите для вставки папки.

 

Примечания:

  • Принадлежащие вам общие папки можно перемещать в любые местоположения папки Tempo Box, кроме другой общей папки. У принадлежащих вам общих папок показывается только имя папки; их можно опознать по значку .

  • Корневой каталог папки, к которой вам предоставлен общий доступ, можно только копировать. При этом в новом местоположении создается закрытая копия папки, а исходная общая папка остается на корневом уровне папки Tempo Box. У папок, к которым вам предоставлен общий доступ, к названию папки добавляется имя владельца; их можно опознать по значку (разрешение на чтение и запись) или (только для чтения).

  • Независимо от вашего уровня доступа, корневой каталог папки Enterprise можно только копировать. Если у вас есть разрешение только на чтение папки Enterprise, вложенные в нее папки можно только копировать. При этом в новом расположении будет создана закрытая копия папки, а исходная папка Enterprise останется в исходном месте в вашей папке Tempo Box. Папки Enterprise можно опознать по значку (разрешение на чтение и запись) или (только для чтения).

  • Можно только копировать корневой каталог библиотеки файлов в сообществе Tempo Social, который можно опознать по значку (разрешение на чтение и запись) или (только для чтения). При этом в новом местоположении создается закрытая копия библиотеки файлов, а исходная библиотека файлов остается на корневом уровне папки Tempo Box.
