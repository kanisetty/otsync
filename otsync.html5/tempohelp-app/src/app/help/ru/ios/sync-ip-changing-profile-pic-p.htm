Для изменения изображения профиля выполните следующие действия:

  1. Коснитесь .

  2. Коснитесь своего профиля.

     

    Замечание:
    Если профилей несколько, можно изменить изображение только того профиля, с помощью которого выполнен вход.

     

  3. Коснитесь изображения профиля и выполните одно из следующих действий:

    • Коснитесь С камеры, чтобы настроить изображение профиля с помощью камеры Apple.

      Сделайте снимок и коснитесь Использовать, чтобы использовать текущее изображение, или коснитесь Повторно создать и снова сделайте снимок.

    • Коснитесь Из альбома, чтобы настроить изображение профиля, используя фотографию из фотоальбома Apple.

      Перейдите к необходимому изображению и коснитесь его.

    • Коснитесь Настроить по умолчанию, чтобы настроить изображение профиля, используя изображение профиля OpenText Tempo Box по умолчанию.

  4. Коснитесь Изменить профиль, чтобы вернуться на экране Изменение профиля.

  5. Коснитесь Готово, чтобы вернуться к экрану Настройки.
