Чтобы изменить профиль, выполните следующие действия.

  1. Коснитесь , а затем Переключение профилей.

  2. На экране OpenText Tempo Box Профиль добавьте новый профиль, используя новое имя пользователя или URL-адрес OpenText Tempo Box. Выберите уникальное имя профиля. Подробную информацию см. в Создание профиля.

  3. Войдите в OpenText Tempo Box с помощью нового профиля. Теперь можно получить доступ к папке OpenText Tempo Box с помощью нового профиля. Однако список избранного из исходного профиля не будет доступен в новом профиле.

    Чтобы сохранить список избранного, необходимо изменить исходный профиль, выполнив следующие действия.

    1. Коснитесь .

    2. Коснитесь имени профиля, который необходимо изменить.

    3. Измените имя пользователя или сервера в соответствии с требованиями.

    4. Коснитесь поля Пароль и введите пароль для этого пользователя.

    5. Коснитесь Готово.

    6. Коснитесь Выход.

    7. На экране OpenText Tempo Box Профиль войдите в OpenText Tempo Box, используя исходный профиль.

      Теперь можно удалить второй профиль, созданный в шаг 2. Подробную информацию см. в Удаление профиля.

Изменение пароля

  1. Коснитесь .

  2. Коснитесь Выход.

  3. Коснитесь своего профиля.

  4. Коснитесь поля Пароль и введите новый пароль.
