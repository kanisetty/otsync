Добавление файла

Добавление файла:

  1. Следует импортировать файл из другого приложения. Найдите файл, который необходимо импортировать, и коснитесь и/или Открыть в.

  2. Коснитесь Открыть в Tempo.

  3. Выполните одно из следующих действий.

    • Для добавления файла в корень папки Tempo Box коснитесь Передать.

    • Для выбора местоположения, отличного от корня папки Tempo Box, коснитесь Назначение: Личная рабочая область

      1. Коснитесь OpenText Tempo Box. Отобразится корень папки Tempo Box.

      2. Перейдите к целевой папке.

        Добавить файл в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

        Нельзя добавить файл в папку с обязательными категориями и атрибутами при помощи Tempo Box. В этом случае см. раздел Просмотр сведений об объекте.

      3. Для выбора целевой папки коснитесь Выбрать.

        Для добавления новой папки нажмите Создать папку. Введите имя новой папки и коснитесь Готово. Для выбора новой папки коснитесь Выбрать.

      4. Нажмите Передать.

  4. По выбору В OpenText Tempo коснитесь импортированного файла. Для перемещения файла в другое местоположение см. раздел Перемещение или копирование файла или папки.
