Глава 4 – Открытие доступа к папке и управление общими папками

 

Открытие общего доступа к папке:

  1. Нажмите Tempo Box и перейдите к целевой папке. При открытии общего доступа к папке она может находиться в любом месте в папке Tempo Box. Подробнее о добавлении новой папки для общего доступа см. в разделе Добавление фотографий и папок.

  2. Нажмите .

  3. Коснитесь Открыть общий доступ. Если кнопка Открыть общий доступ недоступна, это указывает на отсутствие разрешения для открытия общего доступа к этой папке. Список папок, к которым нельзя открыть общий доступ, см. в разделе Исключения при открытии общего доступа к папкам и Общий доступ к папкам Enterprise ниже.

  4. По выбору При открытии общего доступа к папке пользователю можно предоставить разрешение либо на чтение и запись, либо только на чтение. По умолчанию другим пользователям предоставляется разрешение на чтение и запись, которое обозначается как .

    Чтобы предоставить пользователю разрешение только на чтение, выполните следующие действия.

    1. Нажмите .

    2. Коснитесь Только для чтения. Значок изменится на .

    3. Коснитесь Готово.

    Разрешение на чтение и запись дает всем пользователям с общим доступом, включая внешних пользователей, право на полный доступ к содержимому папки. Пользователи, имеющие такое разрешение, могут добавлять, изменять, копировать, перемещать и удалять файлы и папки. После внесения исправлений в общую папку изменения автоматически синхронизируются со всеми пользователями, которым предоставлен общий доступ к папке, включая владельца папки.

    Разрешение только для чтения дает пользователям с общим доступом право только на чтение и копирование содержимого общей папки.

  5. Нажмите поле Имя и введите имя пользователя, для которого необходимо открыть общий доступ к этой папке.

    Чтобы открыть общий доступ к папке для внешнего пользователя, введите его адрес электронной почты. Под внешними понимаются пользователи, которые не являются сотрудниками вашей компании. После ввода действительного адреса электронной почты в поле автозаполнения отобразится адрес электронной почты внешнего пользователя.

     

    Замечание:
    Если адрес электронной почты игнорируется, а область Общий доступ остается пустой, то это указывает на отсутствие у вас разрешения для открытия общего доступа к папкам для внешних пользователей. Дополнительные сведения см. в разделе Открытие доступа к папкам Enterprise для внешних пользователей.

     

  6. Нажмите имя пользователя или адрес электронной почты. Будет отправлено приглашение на открытие общего доступа. Чтобы получить доступ к содержимому папки, пользователь должен принять это приглашение.

  7. В открывшемся диалоговом окне коснитесь Да, чтобы открыть доступ к этой папке другому пользователю, и повторите шаги 4–6 или коснитесь Нет, чтобы вернуться на страницу Сведения о папке.

  8. Коснитесь Назад, чтобы вернуться к своим файлам.

    Теперь папка помечена как общая. Независимо от типа разрешения, предоставленного другим пользователям (либо чтение и запись, либо только чтение), общая папка будет отображаться у вас как .

Исключения при открытии общего доступа к папкам

В папке Tempo Box можно открыть общий доступ ко всем папкам, за исключением следующих:

  • корневая папка Tempo Box.

  • Папка, которая является дочерней или родительской для общей папки.

  • Папка, к которой у вас есть общий доступ.

  • Папки Enterprise только для чтения можно опознать по значку . Если у вас есть разрешение на чтение и запись для папки Enterprise, что можно понять по значку , то вы можете открыть общий доступ к папке, для которой у вас есть разрешение на Редактирование разрешений для этой папки в Content Server.

  • Библиотеки файлов сообщества (независимо от типа доступа: на чтение и запись или только на чтение). Библиотеки файлов сообщества обозначаются и .

  • Внешние пользователи не могут открывать общий доступ ни к одной из папок.

Общий доступ к папкам Enterprise

Вы можете открыть общий доступ к папке Enterprise, если у вас есть разрешение на Редактирование разрешений для этой папки в Content Server. Открыв общий доступ к папке, можно предоставить пользователю разрешение либо на чтение и запись, либо только на чтение. Это соответствует назначению следующих уровней разрешения на папку в Content Server.

Таблица 4-1: Разрешение для папки и доступ в Tempo Box

Доступ в Tempo Box Разрешение для папки в Content Server
Разрешение на чтение и запись Разрешение на изменение содержимого папки, включая все разрешения, за исключением разрешения на Редактирование разрешений .
Разрешение только на чтение Просмотр и Просмотр содержимого .

Если вы попытаетесь предоставить доступ только для чтения пользователю, у которого есть разрешения выше, чем Просмотр и Просмотр содержимого, то для этого пользователя будет открыт доступ к общей папке с существующим уровнем разрешения. Если вы закроете для пользователя общий доступ к папке Enterprise или если пользователь удалит папку Enterprise из своей папки Tempo Box, то в Content Server останется предоставленный уровень доступа.

Открытие доступа к папкам Enterprise для внешних пользователей

Чтобы открыть общий доступ к папке Enterprise для внешних пользователей, необходимо наличие разрешения на открытие общего доступа к папкам для пользователей, не имеющим учетной записи OpenText Tempo Box. Для получения этой привилегии свяжитесь с администратором. Вам также необходимо разрешение на Редактирование разрешений для папки на сервере содержимого (Content Server).

При выполнении обоих этих условий от вас требуется указать адрес электронной почты пользователя, которому вы хотите предоставить общий доступ к папке. Затем внешний пользователь получит по электронной почте приглашение зарегистрироваться для создания учетной записи OpenText Tempo Box. После регистрации внешний пользователь сможет установить клиентские программы и получит такой же доступ к OpenText Tempo Box, как и обычные пользователи, но со следующими исключениями.

  • Внешние пользователи могут принимать приглашения к совместному использованию от других пользователей, но не могут сами открывать общий доступ к папкам.

  • Внешние пользователи не могут публиковать файлы или иным способом получать доступ к Content Server.

Подробнее об управлении общими папками см. в других темах в разделе Открытие доступа к папке и управление общими папками.
