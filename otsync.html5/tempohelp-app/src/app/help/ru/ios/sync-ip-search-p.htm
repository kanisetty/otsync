Поиск папки Tempo Box

 

Поиск папки Tempo Box:

  1. Нажмите Tempo Box и перейдите к целевой папке, с которой следует начать поиск.

  2. В поле Выполнить поиск в этой папке введите строку поиска.

  3. Коснитесь Поиск. Отобразится список результатов поиска.

  4. Нажмите файл или папку, которую необходимо просмотреть.
