Добавление фотографий и папок

 

Добавление фотографии с камеры Apple:

  1. Перейдите к целевой папке, в которую необходимо добавить фотографию.

    Добавить фотографию в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

    Невозможно добавить фотографию в папку, которая имеет обязательные категории и атрибуты, с помощью Tempo Box. В этом случае см. раздел Просмотр сведений об объекте.

  2. Нажмите .

  3. Коснитесь С камеры.

  4. Сделайте снимок.

  5. Коснитесь Использовать фотографию, чтобы добавить текущую фотографию, или коснитесь Повторно создать и повторите шаг 4.

  6. По выбору Коснитесь поля Имя файла и измените имя фотографии. Все фотографии, снятые с использованием камеры, сохраняются как файлы в формате JPEG.

  7. Коснитесь Передать.

Добавление фотографии из фотоальбома устройства Apple:

  1. Перейдите к целевой папке, в которую необходимо добавить фотографию.

    Добавить фотографию в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению .

    Невозможно добавить фотографию в папку, которая имеет обязательные категории и атрибуты, с помощью Tempo Box. В этом случае см. раздел Просмотр сведений об объекте.

  2. Нажмите .

  3. Коснитесь Из альбома.

    Если отображается сообщение об ошибке, убедитесь, что параметры конфиденциальности для фотографий на вашем устройстве позволяют OpenText Tempo передавать фотографии. На домашней странице устройства найдите и коснитесь Настройки > Конфиденциальность > Фотографии и установите переключатель OpenText Tempo в положение ВКЛ..

  4. Перейдите к альбомам на устройстве и коснитесь фотографии, которую необходимо добавить.

  5. По выбору Коснитесь поля Имя файла и измените имя фотографии.

  6. Коснитесь Передать.

Чтобы добавить несколько фотографий из альбома на устройстве Apple, выполните следующие действия.

  1. Перейдите к целевой папке, в которую необходимо добавить фотографию.

    Добавить фотографию в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению .

    Невозможно добавить фотографию в папку, которая имеет обязательные категории и атрибуты, с помощью Tempo Box. В этом случае см. раздел Просмотр сведений об объекте.

  2. Нажмите .

  3. Коснитесь Из альбома.

    Если отображается сообщение об ошибке, убедитесь, что параметры конфиденциальности для фотографий на вашем устройстве позволяют OpenText Tempo передавать фотографии. На домашней странице устройства найдите и коснитесь Настройки > Конфиденциальность > Фотографии и установите переключатель OpenText Tempo в положение ВКЛ..

  4. Перейдите к альбомам на устройстве и выберите фотографии, которые необходимо добавить.

  5. Коснитесь Готово. Чтобы отменить операцию, щелкните значок X.

  6. По выбору Возможно, потребуется коснуться для отображения импортированных изображений.

Добавление папки:

  1. Перейдите к целевому местоположению новой папки.

    Добавить папку в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению .

  2. Нажмите .

  3. Коснитесь Создать папку.

  4. Коснитесь поля Имя папки и введите имя новой папки.

  5. Коснитесь Готово.
