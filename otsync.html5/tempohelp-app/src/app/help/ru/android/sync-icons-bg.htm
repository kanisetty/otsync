Значки общей папки

OpenText Tempo Box поддерживают следующие типы общих папок.

Значок Описание
Данный значок папки соответствует общей папке с разрешением на чтение и запись. Если имя папки дополнено именем пользователя, у вас есть общий доступ к папке. Например, “Маркетинговые материалы (икузнецов)”. Дополнительные сведения см. в разделе Просмотр файлов в общих папках.
Изображение замка обозначает общую папку с наличием доступа только для чтения. Дополнительные сведения см. в разделе Просмотр файлов в общих папках.
Данный значок соответствует папке Enterprise с разрешением на чтение и запись. Вы должны быть подписаны на папку Enterprise на сервере содержимого; альтернативно пользователь, у которого есть разрешение на Редактирование разрешений для этой папки, должен прислать вам приглашение к совместному использованию; тогда у вас будет доступ к содержимому папки Tempo Box. Дополнительные сведения см. в разделе Просмотр файлов в общих папках.
Изображение замка на этом значке папки соответствует папке Enterprise с разрешением только на чтение. Вы должны быть подписаны на папку Enterprise на сервере содержимого; как вариант, пользователь, у которого есть разрешение на Редактирование разрешений для этой папки, должен прислать вам приглашение к совместному использованию; тогда у вас будет доступ к содержимому папки Tempo Box. Дополнительные сведения см. в разделе Просмотр файлов в общих папках .
Данный значок соответствует библиотеке файлов в сообществе Tempo Social, для которой у вас разрешение на чтение и запись. Чтобы иметь доступ к библиотеке файлов из папки Tempo Social, вы должны быть членом сообщества Tempo Box. Дополнительные сведения о сообществах Tempo Social см. в разделе OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social— это не OpenText Tempo Box, а отдельный продукт. Если у вас установлен продукт Tempo Social, но вы не видите библиотек файлов в папке Tempo Box, обратитесь к своему администратору.
Изображение замка на этом значке папки обозначает библиотеку файлов в сообществе Tempo Social, к которой у вас есть доступ только для чтения. Дополнительные сведения см. в разделе OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
Изображение замка на файле обозначает файл в общей папке с разрешением только для чтения.
Наложение флажка на файл обозначает зарезервированный файл.
