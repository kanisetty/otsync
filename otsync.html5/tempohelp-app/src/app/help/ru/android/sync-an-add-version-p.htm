Добавление файла

Добавление новой версии с помощью файла на устройстве:

  1. Нажмите Tempo Box и перейдите к целевому файлу.

     

    Важно:
    Когда вы добавляете новую версию в файл, находящийся в общей папке с разрешением на чтение и запись, этот файл автоматически не резервируется. Вы можете добавить новую версию в файл одновременно с внесением в него изменений другим пользователем. Чтобы предотвратить это, можно зарезервировать файл. Подробнее см. в разделе Резервирование и отмена резервирования файла.

    Общие папки с разрешением на чтение и запись обозначаются наложением . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

     

    Нельзя добавить новую версию в файл, зарезервированный другим пользователем в общей папке. Зарезервированные файлы помечены наложением . Чтобы узнать, кем зарезервирован файл, загляните в журнал аудита файла. Подробнее см. в разделе Просмотр сведений об объекте.

    Также нельзя добавить новую версию в файл, находящийся в общей папке с разрешением только на чтение. Общие папки только для чтения можно опознать по наложению .

  2. Коснитесь файла, к которому нужно добавить новую версию, и удерживайте его.

  3. Нажмите Добавить версию > С устройства.

  4. Перейдите к новой версии файла на устройстве.

  5. Нажмите новую версию файла.

Отправка файла, сохраненного на устройстве:

  1. Нажмите Tempo Box и перейдите к целевой папке, в которую следует отправить файл.

    Нельзя отправить файл в общую папку с разрешением только на чтение. Общие папки только для чтения можно опознать по наложению .

    Нельзя добавить файл в папку с обязательными категориями и атрибутами при помощи Tempo Box. В этом случае см. раздел Просмотр сведений об объекте.

  2. Коснитесь целевой папки и удерживайте ее. Нажмите Передать сюда > С устройства.

  3. Перейдите к целевому файлу на устройстве Android.

  4. Выберите файл.

Импорт файла из другого приложения:

  1. Откройте файл для просмотра в другом приложении.

  2. Нажмите Экспорт или Предоставить общий доступ.

  3. Нажмите Tempo Box.

  4. По выбору При отображении соответствующего запроса войдите в OpenText Tempo Box.

  5. Перейдите в целевое местоположение, в которое следует поместить файл.

    Нельзя импортировать файл в общую папку с разрешением только на чтение. Общие папки только для чтения можно опознать по наложению .

    Нельзя добавить файл в папку с обязательными категориями и атрибутами при помощи Tempo Box. В этом случае см. раздел Просмотр сведений об объекте.

  6. Нажмите клавишу Android Меню, затем нажмите Вставить.
