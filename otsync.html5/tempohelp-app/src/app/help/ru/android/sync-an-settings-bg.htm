Глава 7 – Изменение настроек OpenText Tempo

 

Изменение настроек OpenText Tempo Box:

  1. Нажмите клавишу AndroidМеню, а затем выберите Настройки.

  2. По выбору Чтобы отсортировать файлы по имени или по дате изменения, нажмите строку Порядок сортировки по умолчанию, затем нажмите Дата или Имя.

    Чтобы отобразить это изменение, обновите отображение папки. Подробнее см. в разделе Обновление экрана.

  3. По выбору Нажмите строку Хранение и установите новое ограничение для хранения кэшированных файлов. В случае превышения заданного ограничения для хранения данных старые файлы кэша будут удаляться, пока размер кэша не будет ниже указанного ограничения. Размер кэша по умолчанию – 1 ГБ.

     

    Примечания:

    • В строке Использованное пространство отображается общий размер избранных, а не кэшированных файлов.

    • В строке Использование квоты отображается квота и размер папки Tempo Box. При отображении данных не учитывается размер чужих общих папок.

     

  4. По выбору Нажмите строку Синхронизировать элементы избранного и выберите Автоматически или Вручную.

    При автоматической синхронизации:

    • Новые версии избранных файлов загружаются в фоновом режиме.

    • Если пометить файл как избранный, то он будет загружен для просмотра в автономном режиме.

    При ручной синхронизации:

    • Новые версии избранных файлов не загружаются автоматически.

    • Чтобы загрузить файл для просмотра в автономном режиме, откройте этот файл или вручную синхронизируйте избранные файлы. Чтобы вручную синхронизовать избранные файлы, выполните следующие действия.

      1. Нажмите Элементы избранного.

      2. Выберите Меню > Обновить элементы избранного.

  5. Нажмите клавишу Android Назад, чтобы вернуться к файлам.
