Просмотр сведений об объекте

Просмотр сведений об объекте:

  1. Нажмите Tempo Box и перейдите к целевому файлу или папке.

  2. Коснитесь целевого файла (папки) и удерживайте его.

  3. Нажмите Сведения об объекте.

  4. Выполните одно из следующих действий.

    • Для просмотра сведений об объекте нажмите Свойства.

    • Для просмотра категорий и атрибутов, назначенных для объекта, нажмите Категории.

    • Для просмотра журнала аудита объекта нажмите Аудит.

  5. Нажмите клавишу Android Назад, чтобы вернуться к файлам.
