Глава 2 – Просмотр, редактирование и работа с файлами

 

Просмотр файла:

  1. Нажмите Tempo Box и перейдите к целевому файлу. В строке заголовка OpenText Tempo Box отображается имя папки, просматриваемой в настоящий момент.

  2. По выбору Чтобы перед вами всегда были самые свежие данные, периодически обновляйте экран. Подробнее см. в разделе Обновление экрана.

  3. Выполните одно из следующих действий.

    • Щелкните файл, который хотите просмотреть. Отобразится список приложений, в которых можно открыть файл. Нажмите приложение.

    • Коснитесь файла, который хотите просмотреть, и удерживайте его. Нажмите Открыть и выберите приложение.

     

    Замечание:
    Чтобы установить приложение по умолчанию для определенного типа MIME, перед выбором приложения установите флажок Использовать по умолчанию для этого действия.

     

  4. Нажмите клавишу Android Назад, чтобы вернуться к файлам.

Изменение файла:

  1. Нажмите Tempo Box и перейдите к целевому файлу.

     

    Важно:
    При просмотре файлов в общей папке с разрешением на чтение и запись файлы не резервируются автоматически. Можно открыть файл и вносить в него изменения одновременно с другим пользователем. Чтобы предотвратить это, можно зарезервировать файл. Подробнее см. в разделе Резервирование и отмена резервирования файла.

    Общие папки с разрешением на чтение и запись обозначаются наложением . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

     

    Невозможно сохранить изменения в файле, если он зарезервирован другим пользователем с общим доступом в общей папке. Зарезервированные файлы помечены наложением . Чтобы узнать, кем зарезервирован файл, загляните в журнал аудита файла. Подробнее см. в разделе Просмотр сведений об объекте.

    Сохранить изменения в файле, который находится в общей папке с разрешением только на чтение, невозможно. Общие папки только для чтения можно опознать по наложению .

  2. Нажмите файл, в который хотите внести изменения, и выберите стороннее приложение для редактирования.

  3. Если это приложение поддерживает редактирование, внесите в файл изменения, а затем сохраните его.

    Внесенные изменения не будут сохранены в файл, находящийся в вашей папке Tempo Box. Измененный файл необходимо отправить или добавить в соответствующую папку.

  4. Чтобы отправить или добавить измененный файл в папку Tempo Box, выполните одно из следующих действий.

    • При помощи стороннего приложения сохраните файл на устройство Android. В OpenText Tempo Box добавьте новую версию к существующему файлу. Подробнее см. в разделе Добавление файла.

    • Передайте сохраненный файл из стороннего приложения в папку Tempo Box с помощью команды Предоставить общий доступ или Экспорт. В результате в папку Tempo Box будет добавлен второй файл. Подробнее см. в разделе Добавление файла.

Подробнее об управлении файлами см. в других темах в разделе Просмотр, редактирование и работа с файлами.
