Создание профиля

 

Создание профиля:

  1. При первом запуске приложения OpenText Tempo Box нажмите Добавить профиль.

    Если, войдя в систему OpenText Tempo Box, вы решите создать дополнительный профиль, сделайте следующее.

    1. Нажмите клавишу Android Меню.

    2. Нажмите .

    3. Нажмите Добавить профиль.

  2. Нажмите поле Имя профиля и введите имя профиля, уникальное для данного устройства.

  3. Нажмите поле Имя пользователя и введите имя пользователя. Если вы находитесь вне компании, введите свой адрес электронной почты.

     

    Замечание:
    Пользователи, которые находятся за пределами компании, до создания профиля должны создать учетную запись OpenText Tempo.

    Для создания учетной записи пользователь в компании должен открыть общий доступ для внешнего пользователя. При этом внешнему пользователю будет отправлено по электронной почте приглашение с инструкциями для создания учетной записи.

     

  4. Нажмите поле Сервер и введите URL-адрес сервера OpenText Tempo Box.

    URL-адрес сервера OpenText Tempo Box вы можете узнать у своего администратора.

    Если вы находитесь за пределами компании, то можете найти URL-адрес OpenText Tempo Box в сообщении, отправленном вам по электронной почте при создании вами учетной записи OpenText Tempo Box.

  5. Нажмите Добавить профиль. При входе в систему OpenText Tempo Box вам будет предложено ввести пароль.

    К данным профиля относятся учетные данные учетной записи, URL-адрес OpenText Tempo Box и список избранных файлов. Профили связаны с мобильным устройством. Необходимо создать профили и пометить с помощью тегов файлы как избранные на всех смартфонах и планшетных компьютерах, на которых вы используете приложение OpenText Tempo Box.
