Изменение изображения профиля

Изменение изображения профиля:

  1. Нажмите клавишу AndroidМеню, а затем нажмите .

  2. Нажмите строку Фотография профиля и выполните одно из следующих действий.

    • Нажмите С камеры, чтобы задать изображение профиля, используя камеру Android. Сделайте снимок и нажмите Сохранить.

    • Нажмите Из галереи, чтобы задать изображение профиля, используя фотографию из галереи Android. Перейдите к необходимому изображению и коснитесь его.

    • Нажмите Задать по умолчанию, чтобы задать изображение профиля по умолчаниюOpenText Tempo Box.

  3. Нажмите клавишу Android Назад, чтобы вернуться к экрану Настройки.

  4. Нажмите клавишу Android Назад, чтобы вернуться к файлам.
