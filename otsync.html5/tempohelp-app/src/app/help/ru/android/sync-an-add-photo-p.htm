Добавление фотографии или папки

Добавление фотографии с камеры Android:

  1. Нажмите Tempo Box и перейдите к целевой папке, в которую следует добавить фотографию.

    Добавить фотографию в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

    Невозможно добавить фотографию в папку, которая имеет обязательные категории и атрибуты, с помощью Tempo Box. В этом случае см. раздел Просмотр сведений об объекте.

  2. Нажмите клавишу Android Меню. Нажмите Добавить > С камеры.

  3. Сделайте фотографию, а затем следуйте инструкциям на экране. Фотографии сохраняются в формате JPG. Нельзя изменить название файла фотографии до отправки.

Импорт фотографии из галереи устройства:

  1. Откройте приложение “Галерея Android”.

  2. Перейдите к фотографии, которую следует импортировать.

  3. Нажмите клавишу AndroidМеню, а затем выберите Предоставить общий доступ.

  4. Нажмите Tempo Box.

  5. Перейдите к целевой папке, в которую необходимо добавить фотографию.

    Нельзя импортировать фотографию в общую папку с разрешением только на чтение. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

    Невозможно добавить фотографию в папку, которая имеет обязательные категории и атрибуты, с помощью Tempo Box. В этом случае см. раздел Просмотр сведений об объекте.

  6. Нажмите клавишу Android Меню, затем нажмите Вставить.

Добавление папки:

  1. Нажмите Tempo Box и перейдите к целевому местоположению для новой папки.

    Добавить папку в общую папку с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

  2. Нажмите клавишу Android Меню. Выберите Добавить > Папка.

  3. Нажмите поле Имя папки и введите имя папки.

  4. Нажмите кнопку ОК.
