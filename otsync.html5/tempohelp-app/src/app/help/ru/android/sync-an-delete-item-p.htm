Удаление файла или папки

Удаление файла:

  1. Нажмите Tempo Box и перейдите к целевому файлу.

    Удалить файл из общей папки с разрешением только на чтение невозможно. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

    Невозможно удалить зарезервированный файл. Зарезервированные файлы помечены наложением . Для удаления зарезервированного файла необходимо сначала отменить его резервирование. Подробнее см. в разделе Резервирование и отмена резервирования файла.

     

    Замечание:
    Перед тем как продолжить, см. раздел Внимание! ниже.

     

  2. Коснитесь файла, который следует удалить, и удерживайте его.

  3. Нажмите Удалить.

  4. Нажмите кнопку ОК для подтверждения действия.

Удаление папки:

  1. Нажмите Tempo Box и перейдите к целевой папке.

    Если целевая папка является общей папкой, см. раздел Удаление общей папки. Общие папки обозначаются наложением (разрешение на чтение и запись) или (разрешение только на чтение). Подробнее см. в разделе Обзор значков OpenText Tempo Box.

    Нельзя удалить папку, содержащую зарезервированный файл. Подробнее об отмене резервирования файла см. в разделе Резервирование и отмена резервирования файла.

     

    Замечание:
    Перед тем как продолжить, см. раздел Внимание! ниже.

     

  2. Коснитесь папки, которую следует удалить, и удерживайте ее.

  3. Нажмите Удалить.

  4. Нажмите кнопку ОК для подтверждения действия.

 

Предостережение

  • При удалении файла или папки они удаляются из папки Tempo Box, даже если файл был помечен как избранный. Это означает, что файл был удален со всех других установленных клиентов OpenText Tempo Box: настольного клиента, других смартфонов или планшетов, а также из веб-интерфейса пользователя OpenText Tempo Box.

  • При удалении файла или папки из общей папки с разрешением на чтение и запись они автоматически удаляются у всех пользователей, имеющих доступ к этой общей папке, включая владельца общей папки. Общие папки с разрешением на чтение и запись обозначаются наложением .

  • При удалении файла или папки из папки Enterprise с разрешением на чтение и запись они автоматически удаляются из папки в корпоративной рабочей области. Папки Enterprise с разрешением на чтение и запись можно опознать по .
