Резервирование и отмена резервирования файла

 

Резервирование и отмена резервирования файла:

  1. Нажмите Tempo Box и перейдите к целевому файлу.

    Нельзя зарезервировать файл в общей папке с разрешением только на чтение. Общие папки только для чтения можно опознать по наложению . Подробнее см. в разделе Обзор значков OpenText Tempo Box.

  2. Коснитесь файла, который хотите зарезервировать или для которого хотите отменить резервирование, и удерживайте его.

  3. Выполните одно из следующих действий.

    • Нажмите Зарезервировать, чтобы зарезервировать файл. Файл помечается наложением .

      Если параметр Зарезервировать недоступен, когда вы пытаетесь зарезервировать файл, находящийся в папке Enterprise с разрешением на чтение и запись, то это означает, что у вас нет разрешения на Резервирование для папки в Content Server. Для получения этого разрешения обратитесь к своему администратору.

    • Нажмите Отменить резервирование, чтобы отменить резервирование файла. Если команда Отменить резервирование недоступна, то это означает, что файл зарезервирован другим пользователем. Чтобы узнать, кем зарезервирован файл, загляните в журнал аудита файла. Подробнее см. в разделе Просмотр сведений об объекте.
