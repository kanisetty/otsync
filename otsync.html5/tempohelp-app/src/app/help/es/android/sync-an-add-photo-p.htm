Adición de una fotografía o carpeta

 

Para añadir una fotografía desde la cámara de Android:

  1. Pulse Tempo Box y desplácese hasta la carpeta destino en la que desea añadir la fotografía.

    No puede añadir una foto a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

    No puede añadir una foto a una carpeta que posea categorías y atributos obligatorios mediante Tempo Box. Para averiguarlo, consulte Visualización de detalles de objetos.

  2. Pulse la tecla Menú de Android. Pulse Añadir > Desde cámara.

  3. Realice la fotografía y siga las instrucciones que se muestran en la pantalla. Las fotos se guardan como archivos .jpg. No puede editar el nombre de la foto antes de cargarla.

Para importar una fotografía desde la galería del dispositivo:

  1. Abra la aplicación de galería Android.

  2. Desplácese hasta la fotografía que desea importar.

  3. Pulse la tecla Menú de Android y, a continuación, pulse Compartir.

  4. Pulse Tempo Box.

  5. Desplácese hasta la carpeta destino en la que desea añadir una fotografía.

    No puede importar una foto a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

    No puede añadir una foto a una carpeta que posea categorías y atributos obligatorios mediante Tempo Box. Para averiguarlo, consulte Visualización de detalles de objetos.

  6. Pulse la tecla Menú de Android y, a continuación, pulse Pegar.

Para añadir una carpeta:

  1. Pulse Tempo Box y desplácese hasta la ubicación destino para la nueva carpeta.

    No puede añadir una carpeta a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

  2. Pulse la tecla Menú de Android. Pulse Añadir > Carpeta.

  3. Pulse el recuadro Nombre de carpeta y escriba un nombre para esta carpeta.

  4. Pulse Aceptar.
