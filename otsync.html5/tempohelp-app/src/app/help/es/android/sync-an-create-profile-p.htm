Para crear un perfil:

  1. Si es la primera vez que abre la aplicación OpenText Tempo Box, pulse Añadir perfil.

    Si desea crear un perfil adicional y ya se ha conectado con OpenText Tempo Box, realice las siguientes acciones:

    1. Pulse la tecla Menú de Android.

    2. Pulse .

    3. Pulse Añadir perfil.

  2. Pulse el recuadro Nombre de perfil y escriba un nombre de perfil que sea único en su dispositivo.

  3. Pulse el recuadro Nombre de usuario y escriba su nombre de usuario. Si no pertenece a la compañía, escriba su dirección de correo electrónico.

     

    Nota:
    Si el usuario no pertenece a la compañía, debe crear primero un cuenta de OpenText Tempo antes de crear un perfil.

    Para crear una cuenta, un usuario que se encuentre en la compañía debe compartir una carpeta con el usuario externo. Con esta acción, el usuario externo recibe una invitación por correo electrónico en la que se explican los pasos necesarios para crear una cuenta.

     

  4. Pulse el recuadro Servidor y escriba la URL del servidor de OpenText Tempo Box.

    Póngase en contacto con el administrador para obtener la URL del servidor de OpenText Tempo Box.

    Si no pertenece a la compañía, la URL de OpenText Tempo Box aparece en el mensaje de correo electrónico que recibió cuando creó la cuenta de OpenText Tempo Box.

  5. Pulse Añadir perfil. Al conectarse a OpenText Tempo Box, deberá introducir su contraseña.

    Los datos del perfil incluyen los credenciales de su cuenta, la URL de OpenText Tempo Box y una lista de sus archivos favoritos. Los perfiles son específicos del dispositivo móvil. Debe crear un perfil y marcar los archivos como favoritos en cada uno de los smartphones o tabletas que usen la aplicación OpenText Tempo Box.
