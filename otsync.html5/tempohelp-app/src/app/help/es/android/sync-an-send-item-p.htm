Apertura de un archivo en otra aplicación o perfil

 

Para abrir un archivo en otra aplicación:

  1. Pulse Tempo Box y desplácese hasta el archivo destino. Si el archivo de destino se ha marcado como favorito, pulse Favoritos.

  2. Pulse y mantenga pulsado el archivo que desea exportar.

  3. Pulse Enviar por correo electrónico/Exportar. Si en su dispositivo no está disponible la última versión del archivo, el archivo se baja antes de que se inicie la exportación.

  4. Pulse una aplicación y siga las instrucciones que se muestran en la pantalla.

Para exportar una archivo a otro perfil de OpenText Tempo Box:

  1. Pulse Tempo Box y desplácese hasta el archivo destino. Si el archivo de destino se ha marcado como favorito, pulse Favoritos.

  2. Pulse y mantenga pulsado el archivo que desea exportar.

  3. Pulse Enviar por correo electrónico/Exportar.

  4. Pulse OpenText Tempo.

  5. Pulse la tecla Volver de Android tantas veces como sea necesario para regresar a la página Perfil.

  6. Mantenga pulsado el nombre del perfil y, a continuación, pulse Desconectar.

  7. Conéctese con el perfil en el que desee importar el archivo.

  8. Pulse Tempo Box y desplácese hasta la ubicación destino en la que desea colocar el archivo. No puede importar un archivo a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

  9. Pulse Pegar. Cuando haya finalizado la exportación, seguirá conectado con el segundo perfil.

    Para volver al perfil original, desconecte el segundo perfil mediante la tecla Volver de Android para volver a la página Perfil.

  10. Mantenga pulsado el nombre del segundo perfil y, a continuación, pulse Desconectar.

  11. En la página Perfil, conéctese con el perfil original.
