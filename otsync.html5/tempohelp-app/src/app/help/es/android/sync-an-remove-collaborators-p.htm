Eliminación de un colaborador

Puede eliminar a un colaborador de una carpeta compartida de su propiedad mediante la interfaz de usuario web de OpenText Tempo Box. Así se interrumpirá la conexión a la carpeta compartida y se suprimirá la carpeta de la carpeta Tempo Box del colaborador.

Puede acceder a la interfaz de usuario web introduciendo la URL de OpenText Tempo Box en la barra de direcciones de un navegador y conectándose a OpenText Tempo Box. Para obtener más información sobre cómo eliminar a colaboradores, consulte la ayuda en línea en la interfaz de usuario web de OpenText Tempo Box.
