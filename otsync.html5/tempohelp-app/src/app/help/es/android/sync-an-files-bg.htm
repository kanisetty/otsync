Capítulo 2 – Visualización, edición y trabajo con archivos

Para ver un archivo:

  1. Pulse Tempo Box y desplácese hasta el archivo destino. La barra de título de OpenText Tempo Box muestra el nombre de la carpeta que está viendo en ese momento.

  2. Opcional Para asegurarse de que está viendo el contenido más actualizado, debe actualizar de forma periódica la pantalla. Para obtener más información, consulte Actualización de la pantalla.

  3. Realice una de las acciones siguientes:

    • Pulse el archivo que desea ver. Se muestra una lista de aplicaciones que pueden abrir el archivo. Pulse una aplicación.

    • Pulse y mantenga pulsado el archivo que desea ver. Pulse Abrir y, a continuación, pulse una aplicación.

     

    Nota:
    Para definir una aplicación determinada para un tipo MIME concreto, pulse el recuadro de selección Usar como predeterminada para esta acción antes de seleccionar una aplicación.

     

  4. Pulse la tecla Volver de Android para regresar a los archivos.

Para editar un archivo:

  1. Pulse Tempo Box y desplácese hasta el archivo destino.

     

    Importante:
    Cuando visualice archivos en una carpeta compartida para la que tenga acceso de lectura/escritura, dichos archivos no se reservarán de forma automática. Puede abrir y editar un archivo mientras un colaborador realiza cambios en él. Para evitarlo, puede reservar el archivo. Para obtener más información, consulte Reserva y cancelación de la reserva de un archivo.

    Las carpetas compartidas de lectura/escritura se identifican mediante el icono . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

     

    No podrá guardar las modificaciones realizadas en un archivo cuando este se encuentre reservado por otro colaborador en una carpeta compartida. Los archivos reservados están marcados con la superposición . Para averiguar quién ha reservado el archivo, puede comprobar su historial de auditorías. Para obtener más información, consulte Visualización de detalles de objetos.

    Asimismo, no puede guardar cambios realizados en un archivo de una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición .

  2. Pulse el archivo que desea editar y seleccione un editor de terceros.

  3. Si la aplicación admite la edición, edite el archivo y guarde los cambios.

    Los cambios que haga en el documento no se guardan en el archivo de la carpeta Tempo Box. Deberá subir o añadir el archivo modificado.

  4. Para cargar o añadir el archivo modificado a la carpeta de Tempo Box, lleve a cabo una de las acciones siguientes:

    • Mediante la aplicación de terceros, guarde el archivo en su dispositivo Android. Desde OpenText Tempo Box, añada una versión nueva al archivo existente. Para obtener más información, consulte Adición de un archivo.

    • Mediante la aplicación de terceros, suba el archivo guardado a la carpeta de Tempo Box mediante las opciones Compartir o Exportar. Esto añadirá un segundo archivo a la carpeta Tempo Box. Para obtener más información, consulte Adición de un archivo.

Para obtener información sobre la gestión de los archivos, consulte las demás secciones de Visualización, edición y trabajo con archivos.
