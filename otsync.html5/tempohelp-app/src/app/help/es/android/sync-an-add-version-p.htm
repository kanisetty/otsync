Adición de un archivo

 

Para añadir una nueva versión mediante un archivo en el dispositivo:

  1. Pulse Tempo Box y desplácese hasta el archivo destino.

     

    Importante:
    Cuando añada una versión nueva al archivo de una carpeta compartida para la que tenga acceso de lectura/escritura, dicho archivo no se reservará de forma automática. Puede añadir una versión nueva al archivo mientras un colaborador realiza cambios en él. Para evitarlo, puede reservar el archivo. Para obtener más información, consulte Reserva y cancelación de la reserva de un archivo.

    Las carpetas compartidas de lectura/escritura se identifican mediante el icono . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

     

    No podrá añadir una versión nueva a un archivo cuando este se encuentre reservado por otro colaborador en una carpeta compartida. Los archivos reservados están marcados con la superposición . Para averiguar quién ha reservado el archivo, puede comprobar su historial de auditorías. Para obtener más información, consulte Visualización de detalles de objetos.

    Asimismo, no puede añadir una versión nueva al archivo de una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición .

  2. Pulse y mantenga pulsado el archivo al que desea añadir una versión nueva.

  3. Pulse Añadir versión > Desde dispositivo.

  4. Desplácese hasta la nueva versión del archivo de su dispositivo.

  5. Pulse la nueva versión del archivo.

Para subir un archivo guardado en su dispositivo:

  1. Pulse Tempo Box y desplácese hasta la carpeta destino a la que desea subir el archivo.

    No puede subir un archivo a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición .

    No puede añadir un archivo a una carpeta que posea categorías y atributos obligatorios mediante Tempo Box. Para averiguarlo, consulte Visualización de detalles de objetos.

  2. Pulse y mantenga pulsada la carpeta destino. Pulse Subir aquí > Desde dispositivo.

  3. Desplácese hasta el archivo destino en su dispositivo Android.

  4. Pulse el archivo.

Para importar un archivo desde otra aplicación:

  1. Visualización de un archivo desde otra aplicación.

  2. Pulse Exportar o Compartir.

  3. Pulse Tempo Box.

  4. Opcional Si se le solicita, conéctese a OpenText Tempo Box.

  5. Desplácese hasta la ubicación destino en la que desea colocar el archivo.

    No puede importar un archivo a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición .

    No puede añadir un archivo a una carpeta que posea categorías y atributos obligatorios mediante Tempo Box. Para averiguarlo, consulte Visualización de detalles de objetos.

  6. Pulse la tecla Menú de Android y, a continuación, pulse Pegar.
