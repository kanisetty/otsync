Supresión de un archivo o carpeta

 

Para suprimir un archivo:

  1. Pulse Tempo Box y desplácese hasta el archivo destino.

    No puede suprimir un archivo de una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

    No puede suprimir un archivo que está reservado. Los archivos reservados están marcados con la superposición . Para suprimir un archivo reservado, primero tendrá que cancelar la reserva. Para obtener más información, consulte Reserva y cancelación de la reserva de un archivo.

     

    Nota:
    Antes de continuar, consulte la información de precaución que se indica más abajo.

     

  2. Pulse y mantenga pulsado el archivo que desea suprimir.

  3. Pulse Suprimir.

  4. Pulse Aceptar para confirmar la acción.

Para suprimir una carpeta:

  1. Pulse Tempo Box y desplácese hasta la carpeta destino.

    Si la carpeta de destino es una carpeta compartida, consulte Eliminación de una carpeta compartida. Las carpetas compartidas se identifican mediante la superposición (acceso de lectura/escritura) o la superposición (acceso de solo lectura). Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

    No puede suprimir una carpeta si esta contiene un archivo reservado. Para cancelar la reserva de un archivo, consulte Reserva y cancelación de la reserva de un archivo.

     

    Nota:
    Antes de continuar, consulte la información de precaución que se indica más abajo.

     

  2. Pulse y mantenga pulsada la carpeta que desea suprimir.

  3. Pulse Suprimir.

  4. Pulse Aceptar para confirmar la acción.

 

Atención

  • Al suprimir un archivo o carpeta, se elimina de la carpeta Tempo Box, incluso si el archivo se marcó como favorito. Esto significa que el archivo o carpeta se elimina del resto de clientes de OpenText Tempo Box instalados, como el cliente de escritorio, otros smartphones o tabletas, así como la interfaz de usuario web de OpenText Tempo Box.

  • Si se suprime un archivo o una carpeta de una carpeta compartida para la que tiene acceso de lectura/escritura, dicho archivo o dicha carpeta se suprime automáticamente para todos los usuarios que tienen acceso a esa carpeta compartida, incluido su propietario. Las carpetas compartidas de lectura/escritura se identifican mediante el icono .

  • Si suprime un archivo o una carpeta de una carpeta de empresa para la que tiene acceso de lectura/escritura, el archivo o la carpeta se suprime automáticamente de la carpeta en la Zona de la empresa. Las carpetas de empresa de lectura/escritura se identifican con .
