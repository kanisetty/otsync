Visualización de detalles de objetos

 

Para ver los detalles de los objetos:

  1. Pulse Tempo Box y desplácese hasta el archivo o la carpeta de destino.

  2. Mantenga pulsado el archivo o la carpeta.

  3. Pulse Detalles del objeto.

  4. Realice una de las acciones siguientes:

    • Pulse Propiedades para ver los detalles del objeto.

    • Pulse Categorías para ver las categorías y los atributos asignados al objeto.

    • Pulse Auditoría para ver el historial de auditorías del objeto.

  5. Pulse la tecla Volver de Android para regresar a los archivos.
