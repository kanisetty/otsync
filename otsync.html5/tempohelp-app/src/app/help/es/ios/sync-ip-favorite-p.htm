Capítulo 6 – Adición, visualización y eliminación de favoritos

Para añadir un favorito:

  1. Pulse Tempo Box y desplácese hasta el archivo destino. Puede marcar archivos personales y archivos compartidos como favoritos.

  2. Pulse el archivo que desea marcar como favorito. El archivo se abre.

  3. Pulse . El icono cambia a y el archivo se marca con la superposición en la vista de carpeta. Por ejemplo, . Si marca un archivo como favorito podrá visualizarlo cuando se encuentre fuera de línea.

    En su perfil se guarda un registro de los archivos favoritos, que es específico de su iPad o iPhone. Debe marcar los archivos como favoritos en cada uno de los smartphones y tabletas que usen la aplicación OpenText Tempo.

  4. Pulse Volver para regresar a los archivos.

     

    Sugerencia:
    También puede marcar el archivo como favorito pulsando para el archivo y, a continuación, pulsando Añadir a favoritos.

     

Para ver un favorito:

  1. Pulse Favoritos.

  2. Opcional Pulse para asegurarse de que está viendo el contenido más actualizado.

  3. Pulse el archivo que desea ver.

  4. Pulse Volver para regresar a los archivos.

Para eliminar un favorito:

  1. Pulse Favoritos o Tempo Box y desplácese hasta el archivo de destino.

  2. Pulse el archivo que desea eliminar de la lista de favoritos. El archivo se abre.

  3. Pulse . El icono cambia a . El archivo ya no aparece en la lista de favoritos.

    Esto no suprime el elemento de la carpeta Tempo Box.

  4. Pulse Volver para regresar a los archivos.

     

    Sugerencia:
    También puede eliminar el archivo de la lista de favoritos pulsando para el archivo y, a continuación, pulsando Eliminar favorito.
