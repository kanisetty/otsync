Supresión de un perfil

Para suprimir un perfil:

  1. En la página Perfil, mantenga pulsado el perfil que desea eliminar.

     

    Sugerencia:
    Si está conectado, pulse el botón Inicio para volver a la página Aterrizaje y, a continuación, pulse Configuración > Desconectarse para volver a la página Perfil.

     

  2. Pulse Suprimir.

  3. Pulse Suprimir para confirmar la acción.
