En la aplicación OpenText Tempo Box aparecen los iconos principales siguientes.

Icono Descripción
Le lleva a la página de aterrizaje de una aplicación.
Permite añadir una carpeta nueva o una fotografía a su carpeta Tempo Box.
Actualiza la pantalla.
Muestra la pantalla completa cuando iPad está en modo apaisado.
Muestra los iconos principales de OpenText Tempo Box cuando iPad está en modo retrato.
Le permite ver detalles de una carpeta y compartirla.

Icono Descripción
Cortar el archivo abierto para moverlo.
Copiar el archivo abierto.
Pegar el archivo cortado o copiado más recientemente Este icono aparece después de haber seleccionado el icono de cortar o pegar.
Suprimir el archivo abierto.
El archivo no está marcado como favorito.
El archivo está marcado como favorito. Los comandos del menú identifica un archivo que está marcado como favorito. Por ejemplo, .
Enviar el archivo seleccionado por correo electrónico.

Iconos de carpeta compartida

OpenText Tempo Box incluye los siguientes tipos de carpetas compartidas.

Icono Descripción
Este icono de carpeta representa una carpeta compartida a la que tiene acceso de lectura/escritura. Si el nombre de carpeta lleva añadido un nombre de usuario, indica que usted comparte dicha carpeta. Por ejemplo, Materiales de marketing (jbenito). Para obtener más información, consulte Visualización de archivos en carpetas compartidas.
Una superposición de bloqueo significa que se trata de una carpeta compartida a la que tiene acceso de solo lectura. Para obtener más información, consulte Visualización de archivos en carpetas compartidas.
Este icono de carpeta representa una carpeta de empresa a la que tiene acceso de lectura/escritura. Deberá suscribirse a la carpeta de empresa en Content Server o hacer que un usuario con el permiso Editar permisos en esa carpeta le envíe una invitación para compartir y poder así acceder al contenido desde su carpeta de Tempo Box. Para obtener más información, consulte Visualización de archivos en carpetas compartidas.
Una superposición de bloqueo en este icono de carpeta representa una carpeta de empresa a la que tiene acceso de solo lectura. Deberá suscribirse a la carpeta de empresa en Content Server o hacer que un usuario con el permiso Editar permisos en esa carpeta le envíe una invitación para compartir y poder así acceder al contenido desde su carpeta de Tempo Box. Para obtener más información, consulte Visualización de archivos en carpetas compartidas .
Este icono de carpeta representa una biblioteca de archivos en una comunidad de Tempo Social a la que tiene acceso de lectura/escritura. Debe ser miembro de una comunidad de Tempo Social para poder acceder a las bibliotecas de archivos desde la carpeta de Tempo Box. Para obtener más información acerca de las comunidades de Tempo Social, consulte OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social es un producto independiente de OpenText Tempo Box. Si tiene instalado Tempo Social y no puede ver las bibliotecas de archivos en la carpeta de Tempo Box, póngase en contacto con el administrador.
Una superposición de bloqueo en este icono de carpeta representa una biblioteca de archivos en una comunidad de Tempo Social a la que tiene acceso de solo lectura. Para obtener más información, consulte OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
Una superposición de bloqueo en un archivo representa un archivo en una carpeta compartida en modo de solo lectura.
Una superposición de marca de verificación en un archivo representa un archivo reservado.
