Capítulo 4 – Uso compartido de una carpeta y administración de carpetas compartidas

Para compartir una carpeta:

  1. Pulse Tempo Box y desplácese hasta la carpeta destino. Cuando se comparte una carpeta, esta puede estar situada en cualquier lugar de la carpeta Tempo Box. Para añadir una nueva carpeta para compartir, consulte Adición de fotografías y carpetas.

  2. Pulse .

  3. Pulse Compartir. Si el botón Compartir no está disponible, es porque no tiene permiso para compartir esta carpeta. Para ver una lista de las carpetas que no pueden compartirse, consulte Excepciones para compartir carpetas y Uso compartido de carpetas de empresa que se muestra más abajo.

  4. Opcional Cuando se comparte una carpeta, puede otorgar a un colaborador acceso de lectura/escritura o de solo lectura para la carpeta. De forma predeterminada, los colaboradores reciben acceso de lectura/escritura, que se identifica mediante .

    Para conceder acceso de solo lectura a un colaborador, realice las siguientes acciones:

    1. Pulse .

    2. Pulse Solo lectura. El icono cambia a .

    3. Toque Hecho.

    Con acceso de lectura/escritura, todos los colaboradores, incluidos los usuarios externos, poseen control absoluto sobre el contenido de la carpeta compartida. Pueden añadir, editar, copiar, mover y suprimir archivos y carpetas. Una vez que se hayan hecho revisiones a la carpeta compartida, los cambios se sincronizan automáticamente con todos los colaboradores que tienen acceso compartido a dicha carpeta, incluido el propietario.

    Con acceso de solo lectura, el colaborador solo puede leer y copiar el contenido de la carpeta compartida.

  5. Pulse el recuadro Nombre y escriba el nombre de la persona con la que desea compartir esta carpeta.

    Para compartir una carpeta con un usuario externo, escriba su dirección de correo electrónico. Los usuarios externos no son empleados de su empresa. Después de escribir una dirección de correo electrónico válida, la función de autocompletar mostrará la dirección de correo electrónico del usuario externo.

     

    Nota:
    Si se ignora la dirección de correo electrónico y el área Compartir con está vacía, no tiene permiso para compartir carpetas con usuarios externos. Para obtener más información, consulte Uso compartido de carpetas de empresa con usuarios externos.

     

  6. Pulse el nombre o la dirección de correo electrónico del usuario. Se le enviará una invitación para compartir la carpeta. Los colaboradores deben aceptar la invitación de uso para poder acceder a los contenidos de la carpeta.

  7. En el cuadro de diálogo que se abre, pulse Sí para compartir esta carpeta con otra persona y repita los pasos de 4 a 6, o pulse No para regresar a la página Detalles de carpeta.

  8. Pulse Volver para regresar a los archivos.

    La carpeta aparecerá ahora marcada como carpeta compartida. Independientemente de si concede acceso de lectura/escritura o de solo lectura a los colaboradores, la carpeta compartida tendrá el aspecto para usted.

Excepciones para compartir carpetas

Puede compartir cualquier carpeta de su carpeta Tempo Box, pero se aplican las excepciones siguientes:

  • La carpeta raíz de Tempo Box.

  • Una carpeta que sea hija o padre de una carpeta compartida.

  • Una carpeta que se comparta con usted.

  • Carpetas de empresa si tiene acceso de solo lectura, que se identifica mediante . Si tiene acceso de lectura/escritura a una carpeta de empresa, que se identifica mediante , podrá compartir la carpeta si posee el permiso Editar permisos para dicha carpeta en Content Server.

  • Bibliotecas de archivos de la comunidad, independientemente de que tenga acceso de lectura/escritura o solo lectura. Las bibliotecas de archivos de la comunidad se identifican mediante y .

  • Los usuarios externos no pueden compartir carpetas.

Uso compartido de carpetas de empresa

Puede compartir una carpeta de empresa si posee el permiso Editar permisos para dicha carpeta en Content Server. Al compartir una carpeta, puede conceder acceso de lectura/escritura o solo lectura a la carpeta, que a su vez concede los niveles de permisos de carpeta siguientes en Content Server.

Tabla 4-1: Permiso de carpetas y acceso en Tempo Box

Acceso en Tempo Box Permiso de carpetas en Content Server
Acceso de lectura/escritura Permiso para modificar el contenido de la carpeta, que incluye todos los permisos excepto el permiso Editar permisos .
Acceso de solo lectura Consulte la sección de y Ver contenido .

Si trata de conceder acceso de solo lectura a un usuario que posee permisos superiores a Consulte la sección de y Ver contenido, el usuario se añadirá al uso compartido con su nivel de acceso actual. Si elimina a un colaborador de una carpeta de empresa o suprime la carpeta de su carpeta de Tempo Box, el nivel de acceso que se había concedido sigue vigente en Content Server.

Uso compartido de carpetas de empresa con usuarios externos

Para compartir una carpeta de empresa con usuarios externos, deberá poseer la capacidad de compartir carpetas con usuarios que no dispongan de cuentas de usuario de OpenText Tempo Box. Póngase en contacto con el administrador para solicitar este privilegio. También debe poseer un permiso Editar permisos para la carpeta en Content Server.

Si cumple ambos requisitos, debe facilitar la dirección de correo electrónico del usuario para compartir una carpeta. El usuario externo recibe entonces una invitación para compartir mediante un mensaje de correo electrónico que le invita a registrarse para obtener una cuenta de OpenText Tempo Box. Después de registrarse, el usuario externo puede instalar clientes y acceder a OpenText Tempo Box del mismo modo que los usuarios normales, con las excepciones siguientes:

  • Los usuarios externos pueden aceptar invitaciones para compartir de otros usuarios, pero no pueden compartir carpetas con los demás.

  • Los usuarios externos no pueden publicar elementos en Content Server ni acceder al mismo.

Para obtener información sobre la gestión de las carpetas compartidas, consulte las demás secciones de Uso compartido de una carpeta y administración de carpetas compartidas.
