Adición de fotografías y carpetas

 

Para añadir una fotografía desde la cámara de Apple:

  1. Desplácese hasta la carpeta destino en la que desea añadir una fotografía.

    No puede añadir una foto a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

    No puede añadir una foto a una carpeta que posea categorías y atributos obligatorios mediante Tempo Box. Para averiguarlo, consulte Visualización de detalles de objetos.

  2. Pulse .

  3. Pulse Desde cámara.

  4. Tome la fotografía.

  5. Pulse Usar foto para añadir la fotografía actual, o pulse Volver a tomar y repita el paso 4.

  6. Opcional Pulse el recuadro Nombre de archivo y escriba el nombre de la fotografía. Todas las fotografías tomadas con la cámara del dispositivo se guardan como archivos .jpeg.

  7. Pulse Subir.

Para añadir una fotografía desde el álbum de fotos de Apple:

  1. Desplácese hasta la carpeta destino en la que desea añadir una fotografía.

    No puede añadir una foto a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición .

    No puede añadir una foto a una carpeta que posea categorías y atributos obligatorios mediante Tempo Box. Para averiguarlo, consulte Visualización de detalles de objetos.

  2. Pulse .

  3. Pulse Desde álbum.

    Si recibe un mensaje de error, compruebe si en la configuración de privacidad para fotografías del dispositivo se permite a OpenText Tempo subir fotografías. En la página de inicio del dispositivo, busque y pulse Configuración > Privacidad > Fotos y cerciórese de que el conmutador de OpenText Tempo se encuentra Activado.

  4. Desplácese a los álbumes del dispositivo y pulse la fotografía que desea añadir.

  5. Opcional Pulse el recuadro Nombre de archivo y escriba el nombre de la fotografía.

  6. Pulse Subir.

Para añadir varias fotografías desde el álbum de fotos de Apple:

  1. Desplácese hasta la carpeta destino en la que desea añadir una fotografía.

    No puede añadir una foto a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición .

    No puede añadir una foto a una carpeta que posea categorías y atributos obligatorios mediante Tempo Box. Para averiguarlo, consulte Visualización de detalles de objetos.

  2. Pulse .

  3. Pulse Desde álbum.

    Si recibe un mensaje de error, compruebe si en la configuración de privacidad para fotografías del dispositivo se permite a OpenText Tempo subir fotografías. En la página de inicio del dispositivo, busque y pulse Configuración > Privacidad > Fotos y cerciórese de que el conmutador de OpenText Tempo se encuentra Activado.

  4. Desplácese hasta el álbum del dispositivo y pulse las fotografías que desea añadir.

  5. Toque Hecho. Para cancelar la acción, haga clic en el icono X.

  6. Opcional Puede que tenga que pulsar para visualizar las imágenes importadas.

Para añadir una carpeta:

  1. Desplácese hasta la ubicación de destino para la nueva carpeta.

    No puede añadir una carpeta a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición .

  2. Pulse .

  3. Pulse Nueva carpeta.

  4. Pulse el cuadro Nombre de carpeta y escriba un nombre de la carpeta nueva.

  5. Toque Hecho.
