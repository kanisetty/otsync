Para editar el perfil:

  1. Pulse y, a continuación, pulse Cambiar perfiles.

  2. En la pantalla Perfil de OpenText Tempo Box añada un perfil nuevo con el nombre de usuario nuevo o la URL de OpenText Tempo Box y elija un nombre de perfil único. Para más información, consulte Creación de un perfil.

  3. Conéctese a OpenText Tempo Box utilizando el perfil nuevo. Ya puede acceder a su carpeta OpenText Tempo Box mediante el nuevo perfil. No obstante, su lista de favoritos del archivo original no está disponible en el nuevo perfil.

    Para conservar la lista de favoritos, debe editar el perfil original con los siguientes pasos:

    1. Pulse .

    2. Pulse el nombre del perfil que desea editar.

    3. Cambie el nombre de usuario o el servidor como sea necesario.

    4. Pulse el recuadro Contraseña y escriba la contraseña para este usuario.

    5. Toque Hecho.

    6. Pulse Desconectarse.

    7. En la pantalla Perfil de OpenText Tempo Box, conéctese a OpenText Tempo Box utilizando su perfil original.

      Ahora, puede suprimir el segundo perfil que creó en paso 2. Para más información, consulte Supresión de su perfil.

Para editar su contraseña:

  1. Pulse .

  2. Pulse Desconectarse.

  3. Pulse su perfil.

  4. Pulse el recuadro Contraseña y escriba la contraseña nueva.
