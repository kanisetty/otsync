Apertura de un archivo en otra aplicación

Para abrir un archivo en otra aplicación:

  1. Pulse Tempo Box y desplácese hasta el archivo destino. Si el archivo de destino se ha marcado como favorito, pulse Favoritos.

  2. Pulse el archivo que desea exportar. El archivo se abre.

  3. Pulse . Se muestra una lista de aplicaciones que pueden abrir el archivo.

  4. Pulse una aplicación.
