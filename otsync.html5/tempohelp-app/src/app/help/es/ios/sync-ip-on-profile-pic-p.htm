Cambio de su imagen de perfil

Para cambiar su imagen de perfil:

  1. En la página Aterrizaje, pulse Configuración.

  2. Pulse Hacer clic para cambiar la foto.

  3. Realice una de las acciones siguientes:

    • Pulse Desde cámara para establecer su imagen de perfil con la cámara de Apple.

      Haga la foto y pulse Usar para usar la foto actual, o pulse Volver a tomar y vuelva a hacer la foto.

    • Pulse Desde álbum para establecer su imagen de perfil con una foto del álbum de fotos de Apple.

      Desplácese hasta la imagen que desea usar y, a continuación, púlsela.

    • Pulse Establecer por defecto para definir su imagen de perfil con la imagen de perfil predeterminada.

  4. Pulse Configuración para volver a la página Configuración.
