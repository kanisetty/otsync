Para cambiar su imagen de perfil:

  1. Pulse .

  2. Pulse su perfil.

     

    Nota:
    Si hay varios perfiles en su dispositivo, puede cambiar únicamente la imagen de perfil o el perfil con el que está conectado.

     

  3. Pulse la foto del perfil y realice una de las siguientes acciones:

    • Pulse Desde cámara para establecer su imagen de perfil con la cámara de Apple.

      Haga la foto y pulse Usar para usar la foto actual, o pulse Volver a tomar y vuelva a hacer la foto.

    • Pulse Desde álbum para establecer su imagen de perfil con una imagen del álbum de fotos de Apple.

      Desplácese hasta la imagen que desea usar y, a continuación, púlsela.

    • Pulse Definir como predeterminada para establecer su imagen de perfil con la imagen de perfil predeterminada de OpenText Tempo Box.

  4. Pulse Editar perfil para volver a la pantalla Editar perfil.

  5. Pulse Hecho para volver a la pantalla Configuración.
