Adición de un archivo

 

Para añadir un archivo:

  1. Debe importar el archivo de otra aplicación. Visualice el archivo que desea importar y pulse o Abrir en.

  2. Pulse Abrir en Tempo.

  3. Realice una de las acciones siguientes:

    • Para añadir el archivo a la raíz de la carpeta de Tempo Box, pulse Subir.

    • Para seleccionar una ubicación distinta de la raíz de la carpeta de Tempo Box, pulse Destino: Zona personal

      1. Pulse OpenText Tempo Box. Aparece la raíz de la carpeta de Tempo Box.

      2. Vaya a la carpeta de destino.

        No puede añadir un archivo a una carpeta compartida para la que tenga acceso de solo lectura. Las carpetas compartidas de solo lectura se identifican mediante la superposición . Para obtener más información, consulte Interpretación de los iconos de OpenText Tempo Box.

        No puede añadir un archivo a una carpeta que posea categorías y atributos obligatorios mediante Tempo Box. Para averiguarlo, consulte Visualización de detalles de objetos.

      3. Para seleccionar la carpeta de destino, pulse Elegir.

        Para añadir una carpeta nueva, haga clic en Nueva carpeta. Introduzca el nombre de la carpeta nueva y pulse Hecho. Pulse Elegir para seleccionar la carpeta nueva.

      4. Haga clic en Subir.

  4. Opcional En OpenText Tempo, pulse el archivo importado. Para mover el archivo a otra ubicación, consulte Desplazamiento o copia de un archivo o carpeta.
