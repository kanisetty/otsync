Per crear un perfil:

  1. Si és la primera vegada que obriu l'aplicació OpenText Tempo Box, polseu Afegeix perfil.

    Per crear un altre perfil després d'iniciar sessió, feu el següent:

    1. Polseu la tecla Menú d'Android.

    2. Polseu .

    3. Polseu Afegeix perfil.

  2. Polseu el quadre Nom del perfil i escriviu un nom de perfil únic.

  3. Polseu el quadre Nom de l'usuari i escriviu el vostre nom d'usuari. Si sou extern a l'empresa, escriviu la vostra adreça de correu electrònic.

     

    Nota:
    Per poder crear un perfil, els usuaris externs a l'empresa primer han de crear un compte d'OpenText Tempo Box.

    Per crear un compte, cal que un usuari de l'empresa comparteixi una carpeta amb l'usuari extern. L'usuari extern rebrà una invitació per correu electrònic amb informació sobre què cal fer per crear un compte.

     

  4. Polseu el quadre Servidor i escriviu l'adreça URL del vostre servidor d'OpenText Tempo Box.

    Per obtenir l'adreça URL del servidor d'OpenText Tempo Box, poseu-vos en contacte amb l'administrador.

    Si sou extern a l'empresa, podeu consultar l'adreça URL d'OpenText Tempo Box al missatge de correu electrònic que vàreu rebre en crear el vostre compte d'OpenText Tempo Box.

  5. Polseu Afegeix perfil. Quan inicieu sessió a l'OpenText Tempo Box, se us demanarà la contrasenya.

    Posat que els perfils són específics de cada dispositiu, cal que en creeu un per a cada telèfon intel·ligent o tauleta tàctil on utilitzeu l'aplicació OpenText Tempo Box.
