Per canviar la contrasenya:

  1. Polseu .

  2. Polseu Tanca la sessió.

  3. Polseu el vostre perfil.

  4. Polseu el quadre Contrasenya i escriviu la nova contrasenya.

Per editar el perfil:

  1. Polseu > Canvia els perfils.

  2. Per canviar el nom d'usuari o l'adreça URL d'OpenText Tempo Box del perfil original, cal que en creeu un de nou. A la pantalla Perfil d'OpenText Tempo Box, afegiu un perfil nou. Escriviu el nom d'usuari o l'adreça URL d'OpenText Tempo Box nous i, tot seguit, trieu un nom únic per al perfil. Per obtenir més informació, consulteu Crear un perfil.

  3. Inicieu sessió a l'OpenText Tempo Box utilitzant el nou perfil. Per obtenir més informació, consulteu Iniciar sessió i gestionar el perfil.

    Podeu continuar accedint a la vostra carpeta de Tempo Box amb el nou perfil, però haureu d'actualitzar la vostra imatge de perfil. Si preferiu accedir a la vostra carpeta de Tempo Box utilitzant el perfil original, ara podreu canviar el nom d'usuari o l'adreça URL d'OpenText Tempo Box. En ambdós casos, es perdrà la llista de preferits del perfil original.

    Per editar el perfil original:

    1. Polseu .

    2. Polseu el nom del perfil que vulgueu editar.

    3. Si cal, canvieu el nom d'usuari i el servidor.

    4. Polseu el quadre Contrasenya i escriviu la contrasenya d'aquest usuari.

    5. Polseu Fet.

    6. Polseu Tanca la sessió.

    7. A la pantalla Perfil d'OpenText Tempo Box, inicieu sessió a l'OpenText Tempo Box utilitzant el perfil original.

      Ara podeu suprimir el segon perfil que vàreu crear al pas 2. Per obtenir més informació, consulteu Suprimir el perfil.
