Présentation des icônes d'OpenText Tempo Box

 

Les icônes suivantes apparaissent dans l'interface principale du système d'exploitation Android 4.0 (ou version ultérieure). Pour les versions antérieures, ces fonctions sont disponibles dans le menu Android.

Icône Description
Permet d'ouvrir la page Paramètres à partir de la page Profil.

Icônes des dossiers partagés

OpenText Tempo Box propose les types de dossiers partagés suivants.

Icône Description
Cette icône de dossier représente un dossier partagé pour lequel vous disposez d'un accès en lecture/écriture. Si un nom d'utilisateur est ajouté au nom du dossier, cela signifie qu'un utilisateur partage ce dossier avec vous. Par exemple, Ressources marketing (rduval). Pour plus d'informations, consultez la section Visualisation des fichiers d'un dossier partagé.
Le cache de verrouillage représente un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Pour plus d'informations, consultez la section Visualisation des fichiers d'un dossier partagé.
Cette icône représente un dossier Entreprise pour lequel vous disposez d'un accès en lecture/écriture. Vous devez vous abonner à un dossier Entreprise dans Content Server ou connaître quelqu'un détenant l' Modifier les autorisations autorisation pour le dossier afin qu'il vous envoie une invitation de partage pour accéder au contenu depuis votre dossier Tempo Box. Pour plus d'informations, consultez la section Visualisation des fichiers d'un dossier partagé.
Le cache de verrouillage sur cette icône de dossier représente un dossier Entreprise pour lequel vous disposez d'un accès en lecture seule. Vous devez vous abonner au dossier Entreprise dans Content Server ou connaître quelqu'un détenant l' Modifier les autorisations autorisation pour le dossier afin qu'il vous envoie une invitation de partage pour accéder au contenu depuis votre dossier Tempo Box. Pour plus d'informations, consultez la section Visualisation des fichiers d'un dossier partagé .
Cette icône représente une bibliothèque de fichier dans une communauté Tempo Social pour laquelle vous disposez d'un accès en lecture/écriture. Vous devez être membre d'une communauté Tempo Social pour accéder aux bibliothèques de fichier depuis votre dossier Tempo Box. Pour plus d'informations sur les communautés Tempo Social, voir OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social est un produit distinct d'OpenText Tempo Box. Si Tempo Social est installé et que vous ne voyez pas les bibliothèques de fichier dans votre dossierTempo Box, contactez votre administrateur.
Le cache de verrouillage sur cette icône de dossier représente une bibliothèque de fichier dans une communauté Tempo Social pour laquelle vous disposez d'un accès en lecture seule. Pour plus d'informations, reportez-vous au OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
Le cache de verrouillage sur un fichier représente un fichier dans un dossier partagé en lecture seule.
Une coche superposée sur un fichier représente un fichier réservé.
