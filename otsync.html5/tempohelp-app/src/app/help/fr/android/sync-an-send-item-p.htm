Ouverture d'un fichier dans une autre application ou un autre profil

 

Pour ouvrir un fichier dans une autre application :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible. Si le fichier cible est marqué comme favori, appuyez sur Favoris.

  2. Maintenez votre doigt sur le fichier que vous souhaitez exporter.

  3. Appuyez sur Envoyer par e-mail / Exporter. Si la version la plus récente du fichier n'est pas disponible sur votre périphérique, le fichier est téléchargé avant d'être exporté.

  4. Appuyez sur une application et suivez les instructions à l'écran.

Pour exporter un fichier vers un autre profil OpenText Tempo Box :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible. Si le fichier cible est marqué comme favori, appuyez sur Favoris.

  2. Maintenez votre doigt sur le fichier que vous souhaitez exporter.

  3. Appuyez sur Envoyer par e-mail / Exporter.

  4. Appuyez sur OpenText Tempo.

  5. Appuyez sur la touche Retour Android jusqu'au retour sur la page Profil.

  6. Maintenez votre doigt sur votre nom de profil, puis appuyez sur Déconnexion.

  7. Connectez-vous au profil dans lequel vous voulez importer le fichier.

  8. Appuyez sur Tempo Box, puis accédez à l'emplacement cible dans lequel placer le fichier. Vous ne pouvez pas importer de fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

  9. Appuyez sur Coller. Une fois l'exportation terminée, vous êtes toujours connecté au second profil.

    Pour revenir au profil d'origine, déconnectez-vous du second profil en appuyant sur la touche Android Retour pour revenir à la page Profil.

  10. Maintenez le doigt sur le nom du second profil, puis appuyez sur Déconnexion.

  11. Sur la page Profil, connectez-vous au profil d'origine.
