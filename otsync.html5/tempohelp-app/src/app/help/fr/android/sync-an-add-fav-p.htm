Ajout, visualisation, suppression et synchronisation de favoris

 

Pour ajouter un favori :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible. Vous pouvez marquer comme favoris aussi bien des fichiers personnels que partagés.

  2. Maintenez votre doigt sur le fichier que vous souhaitez marquer comme favori.

  3. Appuyez sur Ajouter aux favoris.

    Le fichier est marqué à l'aide du recouvrement. Par exemple, .

  4. Facultatif Si l'icône recouvrement apparaît, le fichier marqué comme favori n'est pas téléchargé. Par exemple, . Pour télécharger le fichier, synchronisez vos favoris ou ouvrez le fichier. Pour plus de détails, voir Synchroniser manuellement vos favoris.

    Pour synchroniser vos fichiers favoris automatiquement, voir Modification des paramètres d'OpenText Tempo.

    Un enregistrement de vos fichiers favoris est stocké dans votre profil. Les profils sont spécifiques à votre périphérique. Vous devez créer un profil et marquer les fichiers comme favoris sur chacun de vos smartphones et tablettes qui utilisent l'application OpenText Tempo Box.

Pour visualiser un favori :

  1. Appuyez sur Favoris.

  2. Facultatif Pour être sûr de visualiser le contenu le plus récent, actualisez régulièrement l'affichage. Pour plus de détails, voir Actualisation de l'affichage.

    Les fichiers ne sont pas téléchargés pour être visualisés hors ligne. Pour télécharger le fichier, synchronisez vos favoris ou ouvrez-le. Pour plus de détails, voir Synchroniser manuellement vos favoris ci-dessous.

Pour ouvrir l'emplacement d'un fichier favori :

  1. Appuyez sur Favoris.

  2. Maintenez votre doigt sur le fichier que vous souhaitez ouvrir.

  3. Appuyez sur Ouvrir l'emplacement.

Pour supprimer un favori :

  1. Appuyez sur Favoris, ou sur Tempo Box et accédez au fichier cible.

  2. Maintenez votre doigt sur le fichier que vous souhaitez supprimer de votre liste de fichiers favoris.

  3. Appuyez sur Supprimer des favoris. Le fichier n'apparaît plus dans votre liste de favoris. Il n'est toutefois pas supprimé de votre dossier Tempo Box.

Pour synchroniser manuellement des favoris :

  1. Appuyez sur Favoris.

  2. Appuyez sur Menu > Actualiser les favoris. Si le processus de synchronisation de vos favoris de l'application est en cours ou si vous êtes hors ligne, Actualiser les favoris n'est pas disponible.

     

    Note :
    Appuyez sur Actualiser pour mettre à jour l'affichage de la page Favoris. Toutefois, vos fichiers favoris ne sont pas téléchargés sur votre appareil pour une visualisation hors ligne. Pour télécharger le fichier, vous devez synchroniser vos fichiers favoris ou ouvrir le fichier.

     

    Pour synchroniser vos fichiers favoris automatiquement, voir Modification des paramètres d'OpenText Tempo.
