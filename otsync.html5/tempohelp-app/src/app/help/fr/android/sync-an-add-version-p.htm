Ajout d'un fichier

 

Pour ajouter une nouvelle version à l'aide d'un fichier sur votre périphérique :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible.

     

    Important :
    Quand vous ajoutez une nouvelle version au fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture/écriture, le fichier n'est pas automatiquement réservé. Vous pouvez ajouter une nouvelle version au fichier alors même qu'un de vos collaborateurs y apporte des modifications. Pour éviter cela, vous pouvez réserver le fichier. Pour plus de détails, voir Réserver un fichier et annuler sa réservation.

    Les dossiers partagés en lecture/écriture sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

     

    Vous ne pouvez pas ajouter une nouvelle version à un fichier lorsqu'il est réservé par un autre collaborateur dans un dossier partagé. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vous pouvez vérifier l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

    De même, vous ne pouvez pas ajouter une nouvelle version à un fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement.

  2. Maintenez votre doigt sur le fichier auquel vous souhaitez ajouter une nouvelle version.

  3. Appuyez sur Ajouter une version > A partir du périphérique.

  4. Accédez à la nouvelle version du fichier sur votre périphérique.

  5. Appuyez sur la nouvelle version du fichier.

Pour charger un fichier stocké sur votre périphérique :

  1. Appuyez sur Tempo Box, puis accédez au dossier cible dans lequel vous souhaitez charger un fichier.

    Vous ne pouvez pas charger de fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement.

    Vous ne pouvez pas ajouter de fichier à un dossier qui a des attributs et des catégories obligatoires via Tempo Box. Pour déterminer cela, voir Afficher les détails de l'objet.

  2. Maintenez votre doigt sur le dossier cible. Appuyez sur Charger ici > A partir du périphérique.

  3. Accédez au fichier cible sur votre périphérique Android.

  4. Appuyez sur le fichier.

Pour importer un fichier à partir d'une autre application :

  1. Dans une autre application, visualisez un fichier.

  2. Appuyez sur Exporter ou sur Partager.

  3. Appuyez sur Tempo Box.

  4. Facultatif Si vous y êtes invité, connectez-vous à OpenText Tempo Box.

  5. Accédez à l'emplacement dans lequel vous souhaitez importer le fichier.

    Vous ne pouvez pas importer de fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement.

    Vous ne pouvez pas ajouter de fichier à un dossier qui a des attributs et des catégories obligatoires via Tempo Box. Pour déterminer cela, voir Afficher les détails de l'objet.

  6. Appuyez sur la touche Menu Android puis sur Coller.
