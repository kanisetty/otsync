Afficher les détails de l'objet

Pour afficher les détails de l'objet :

  1. Appuyez sur Tempo Box, puis accédez au fichier ou dossier cible.

  2. Maintenez votre doigt sur le fichier ou le dossier.

  3. Appuyez sur Détails de l'objet.

  4. Procédez de l'une des façons suivantes :

    • Appuyez sur Propriétés pour consulter les détails concernant l'objet.

    • Appuyez sur Catégories pour consulter les catégories et les attributs assignés à l'objet.

    • Appuyez sur Audit pour consulter l'historique des audits de l'objet.

  5. Appuyez sur la touche Android Retour pour revenir à vos fichiers.
