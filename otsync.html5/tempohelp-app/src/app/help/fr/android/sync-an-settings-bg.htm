Modification des paramètres d'OpenText Tempo

 

Pour modifier les paramètres d'OpenText Tempo Box :

  1. Appuyez sur la touche Menu Android, puis sur Paramètres.

  2. Facultatif Appuyez sur la rangée Ordre de tri par défaut, puis sur Date ou Nom pour trier vos fichiers par nom ou modifier la date du fichier.

    Pour faire apparaître ces modifications, vous devez actualiser l'affichage du dossier. Pour plus de détails, voir Actualisation de l'affichage.

  3. Facultatif Appuyez sur la rangée Stockage du périphérique puis indiquez une nouvelle limite de stockage pour vos fichiers en cache. Si vous avez dépassé la limite de stockage définie, les fichiers les plus anciens de la mémoire cache sont supprimés de sorte que la taille du cache n'excède pas la limite spécifiée. La taille du cache par défaut est de 1 Go.

     

    Notes :

    • La rangée Espace utilisé indique le volume total de vos fichiers favoris, et non la taille du cache.

    • La rangée Utilisation du quota indique le quota et la taille de votre dossier Tempo Box. Ceci exclut les dossiers partagés dont vous n'êtes pas le propriétaire.

     

  4. Facultatif Appuyez sur la ligne Synchroniser les favoris, puis sur Automatique ou Manuel.

    Avec la synchronisation automatique,

    • Les nouvelles versions des fichiers favoris sont téléchargées en arrière-plan.

    • Le fichier est téléchargé pour pouvoir être consulté hors ligne dès que vous le marquez comme favori.

    Avec la synchronisation manuelle,

    • Les nouvelles versions des fichiers favoris ne sont pas téléchargées automatiquement.

    • Vous devez ouvrir le fichier ou synchroniser manuellement vos favoris pour télécharger le fichier afin de pouvoir le consulter hors ligne. Pour synchroniser vos favoris manuellement, procédez comme suit :

      1. Appuyez sur Favoris.

      2. Appuyez sur Menu > Actualiser les favoris.

  5. Appuyez sur la touche Android Retour pour revenir à vos fichiers.

 

Astuce :
Si vous avez installé la version 4.0 ou une version ultérieure du système d'exploitation Android, vous pouvez accéder à la page Paramètres depuis la page Profil en appuyant sur .
