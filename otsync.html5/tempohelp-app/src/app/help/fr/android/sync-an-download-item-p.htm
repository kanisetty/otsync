Télécharger un fichier vers votre appareil

 

Pour télécharger un fichier :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible. Si le fichier cible est marqué comme favori, appuyez sur Favoris.

  2. Maintenez votre doigt sur le fichier que vous souhaitez télécharger.

  3. Appuyez sur Télécharger.

  4. Accédez au dossier cible sur votre périphérique Android.

  5. Appuyez sur la touche Android Menu, puis sur Télécharger.
