Pour créer un profil :

  1. Si vous lancez l'application OpenText Tempo Box pour la première fois, appuyez sur Ajouer un profil.

    Si vous voulez créer un autre profil et que vous êtes déjà connecté à OpenText Tempo Box, procédez comme suit :

    1. Appuyez sur la touche Android Menu.

    2. Appuyez sur .

    3. Appuyez sur Ajouter un profil.

  2. Appuyez sur la case Nom du profil et saisissez un nom de profil unique sur votre périphérique.

  3. Appuyez sur le champ Nom de l'utilisateur et saisissez votre nom d'utilisateur. Si vous n'appartenez pas à l'entreprise, saisissez votre adresse e-mail.

     

    Note :
    Un utilisateur qui n'appartient pas à l'entreprise doit d'abord créer un compte OpenText Tempo pour pouvoir créer un profil.

    Pour créer un compte, un utilisateur au sein de l'entreprise doit partager un dossier avec l'utilisateur externe. Cela permet à l'utilisateur externe de recevoir un e-mail d'invitation détaillant la procédure à suivre pour créer un compte.

     

  4. Appuyez sur la case Serveur et saisissez l'URL de votre serveur OpenText Tempo Box.

    Pour connaître l'URL du serveur OpenText Tempo Box, contactez votre administrateur.

    Si vous n'appartenez pas à l'entreprise, l'URL d'OpenText Tempo Box apparaît dans l'e-mail que vous avez reçu lors de la création de votre compte OpenText Tempo Box.

  5. Appuyez sur Ajouter un profil. Lorsque vous vous connecterez à OpenText Tempo Box, vous serez invité à saisir votre mot de passe.

    Les données du profil incluent les informations d'identification de votre compte, l'URL d'OpenText Tempo Box et la liste de vos fichiers favoris. Les profils sont spécifiques au périphérique mobile. Vous devez créer un profil et marquer les fichiers comme favoris sur chacun de vos smartphones et tablettes qui utilisent l'application OpenText Tempo Box.
