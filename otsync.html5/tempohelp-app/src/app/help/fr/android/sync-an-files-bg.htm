Visualisation, modification et utilisation de vos fichiers

 

Pour visualiser un fichier :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible. La barre de titre d'OpenText Tempo Box affiche le nom du dossier visualisé.

  2. Facultatif Pour être sûr de visualiser le contenu le plus récent, actualisez régulièrement l'affichage. Pour plus de détails, voir Actualisation de l'affichage.

  3. Procédez de l'une des façons suivantes :

    • Appuyez sur le fichier à visualiser. Une liste d'applications capables d'ouvrir le fichier apparaît. Appuyez sur une application.

    • Maintenez votre doigt sur le fichier que vous souhaitez visualiser. Appuyez sur Ouvrir, puis sur une application.

     

    Note :
    Pour définir une application par défaut pour un type MIME spécifique, activez la case Utiliser par défaut pour cette action avant de sélectionner l'application.

     

  4. Appuyez sur la touche Android Retour pour revenir à vos fichiers.

Pour modifier un fichier :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible.

     

    Important :
    Quand vous visualisez des fichiers dans un dossier partagé pour lequel vous disposez d'un accès en lecture/écriture, les fichiers ne sont pas automatiquement réservés. Vous pouvez ouvrir et modifier un document alors même qu'un de vos collaborateurs y apporte des modifications. Pour éviter cela, vous pouvez réserver le fichier. Pour plus de détails, voir Réserver un fichier et annuler sa réservation.

    Les dossiers partagés en lecture/écriture sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

     

    Vous ne pouvez pas enregistrer de modifications apportées à un fichier lorsqu'il est réservé par un autre collaborateur dans un dossier partagé. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vous pouvez vérifier l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

    De même, vous ne pouvez pas enregistrer de modifications apportées à un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement.

  2. Appuyez sur le fichier que vous souhaitez modifier et sélectionnez un éditeur tiers.

  3. Si l'application prend en charge les modifications, modifiez le fichier et enregistrez vos modifications.

    Les modifications apportées ne sont pas enregistrées dans le fichier qui se trouve dans votre dossier Tempo Box. Vous devez charger ou ajouter le fichier modifié.

  4. Pour charger ou ajouter le fichier modifié à votre dossier Tempo Box, procédez de l'une des façons suivantes :

    • Dans l'application tierce, enregistrez le fichier sur votre périphérique Android. Dans OpenText Tempo Box, ajoutez une nouvelle version au fichier existant. Pour plus de détails, voir Ajout d'un fichier.

    • Dans l'application tierce, chargez le fichier enregistré vers le dossier Tempo Box en utilisant les options Partager ou Exporter. Un deuxième fichier sera ajouté à votre dossier Tempo Box. Pour plus de détails, voir Ajout d'un fichier.

Pour plus d'informations sur la gestion des fichiers, reportez-vous aux autres rubriques dans Visualisation, modification et utilisation de vos fichiers.
