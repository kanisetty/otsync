Pour se connecter :

  1. Lancez l'application OpenText Tempo Box. Pour accéder à votre dossier OpenText Tempo Box, vous devez d'abord créer un profil avant de vous connecter. Pour plus d'informations, voir Création d'un profil.

  2. Appuyez sur votre nom de profil OpenText Tempo Box.

  3. Si vous y êtes invité, appuyez sur le champ Mot de passe, puis saisissez votre mot de passe.

    Si vous n'appartenez pas à l'entreprise, saisissez le mot de passe que vous avez choisi lorsque vous avez créé votre compte OpenText Tempo Box. Si vous avez oublié votre mot de passe, reportez-vous à l'e-mail que vous avez reçu lorsque vous avec créé votre compte OpenText Tempo Box.

     

    Note :
    Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

     

  4. Facultatif Appuyez sur la case Enregistrer le mot de passe pour un accès immédiat lors du prochain lancement de l'application. Les exceptions sont les suivantes : si vous vous déconnectez d'OpenText Tempo Box ou si la session arrive à expiration. Reportez-vous à la section Attention ci-dessous.

     

    Note :
    Cette case à cocher n'est pas disponible si le périphérique est hors ligne.

     

  5. Appuyez sur Connexion.

 

Attention

  • En autorisant OpenText Tempo Box à enregistrer votre mot de passe, vous risquez de compromettre la sécurité de votre périphérique s'il est utilisé par un tiers.

  • Lorsque plusieurs profils sont configurés sur votre périphérique, d'autres profils pourront accéder à votre compte OpenText Tempo Box si la case Enregistrer le mot de passe est sélectionnée et si vous n'êtes pas déconnecté. Pour vous déconnecter, appuyez sur la touche Menu Android, puis sur .

 

Pour plus d'informations sur la gestion de votre profil, voir Connexion et gestion de votre profil.
