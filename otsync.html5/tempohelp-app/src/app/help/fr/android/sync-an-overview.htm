Icônes supplémentaires

Les icônes suivantes apparaissent dans l'interface principale du système d'exploitation Android 4.0 (ou version ultérieure). Pour les versions antérieures, ces fonctions sont disponibles dans le menu Android.

Icône Description
Permet de coller le dernier fichier coupé ou copié. Cette icône apparaît uniquement après avoir sélectionné l'icône Couper ou Copier. Elle apparaît également lors de l'importation d'un fichier d'une autre application.
Permet d'effectuer une recherche dans le dossier en cours et dans ses sous-dossiers.
Permet d'ajouter une photo, un dossier ou de charger un fichier à partir de votre périphérique Android.
Permet d'actualiser l'affichage.
Permet d'ouvrir l'écran Paramètres à partir de l'écran Profil.
