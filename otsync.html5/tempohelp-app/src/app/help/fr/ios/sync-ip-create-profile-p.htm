Pour créer un profil :

  1. Si vous lancez l'application OpenText Tempo Box pour la première fois, appuyez sur Ajouer un profil.

    Si vous voulez créer un autre profil et que vous êtes déjà connecté à OpenText Tempo Box, procédez comme suit :

    1. Appuyez sur .

    2. Appuyez sur Changer de profil.

    3. Appuyez sur Ajouter un profil.

  2. Appuyez sur le champ Nom de l'utilisateur et saisissez votre nom d'utilisateur. Si vous n'appartenez pas à l'entreprise, saisissez votre adresse e-mail.

     

    Note :
    Un utilisateur qui n'appartient pas à l'entreprise doit d'abord créer un compte OpenText Tempo Box pour pouvoir créer un profil.

    Pour créer un compte, un utilisateur au sein de l'entreprise doit partager un dossier avec l'utilisateur externe. Cela permet à l'utilisateur externe de recevoir un e-mail d'invitation détaillant la procédure à suivre pour créer un compte.

     

  3. Appuyez sur le champ Mot de passe, puis saisissez votre mot de passe.

    Si vous n'appartenez pas à l'entreprise, saisissez le mot de passe que vous avez choisi lorsque vous avez créé votre compte OpenText Tempo Box. Si vous avez oublié votre mot de passe, reportez-vous à l'e-mail que vous avez reçu lorsque vous avec créé votre compte OpenText Tempo Box.

     

    Note :
    Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

     

  4. Appuyez sur la case Serveur et saisissez l'URL de votre serveur OpenText Tempo Box.

    Si vous avez oublié l'URL OpenText Tempo Box, contactez votre administrateur.

    Si vous n'appartenez pas à l'entreprise, l'URL d'OpenText Tempo Box apparaît dans l'e-mail que vous avez reçu lors de la création de votre compte OpenText Tempo Box.

  5. Appuyez sur la case Nom du profil et saisissez un nom de profil unique sur votre périphérique.

  6. Facultatif Appuyez sur Retenir le mot de passe pour un accès immédiat lors du prochain lancement de l'application. Les exceptions sont les suivantes : si vous vous déconnectez d'OpenText Tempo Box ou si la session arrive à expiration. Reportez-vous à la section Attention ci-dessous.

  7. Appuyez sur Terminé. Les données du profil incluent les informations d'identification de votre compte, l'URL d'OpenText Tempo Box et la liste de vos fichiers favoris. Les profils sont spécifiques à iPhone et à iPad. Vous devez créer un profil et marquer les fichiers comme favoris sur chacun de vos smartphones et tablettes qui utilisent l'application OpenText Tempo Box.

 

Attention

  • En autorisant OpenText Tempo Box à retenir votre mot de passe, vous risquez de compromettre la sécurité de votre périphérique s'il est utilisé par un tiers.

  • Lorsque plusieurs profils sont configurés sur votre périphérique, d'autres profils pourront accéder à votre compte OpenText Tempo Box si vous avez sélectionné Retenir le mot de passe et si vous n'êtes pas déconnecté. Pour vous déconnecter, appuyez sur , puis sur Déconnexion.
