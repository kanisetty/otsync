Réserver un fichier et annuler sa réservation

Pour réserver un fichier ou annuler sa réservation :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible.

    Vous ne pouvez pas réserver de fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

  2. Appuyez sur pour afficher les détails du fichier que vous souhaitez réserver ou pour lequel vous souhaitez annuler la réservation.

  3. Procédez de l'une des façons suivantes :

    • Appuyez sur Réserver pour réserver le fichier. Le fichier est marqué à l'aide du recouvrement.

      Si vous essayez de réserver un fichier dans un dossier Entreprise pour lequel vous disposez d'un accès en lecture/écriture et que l'option Réserver n'est pas disponible vous ne disposez pas de Réserver l'autorisation pour le dossier dans Content Server. Pour obtenir cette autorisation, contactez votre administrateur.

    • Appuyez sur Annuler la réservation pour annuler la réservation du fichier. Si Annuler la réservation n'est pas disponible, le fichier est réservé par un autre utilisateur. Pour savoir qui a réservé le fichier, vous pouvez vérifier l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

  4. Appuyez sur Retour.
