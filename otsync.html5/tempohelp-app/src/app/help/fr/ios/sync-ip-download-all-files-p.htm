Téléchargement de tous les fichiers dans un dossier

Pour télécharger tous les fichiers dans un dossier :

  1. Appuyez sur Tempo Box, puis accédez au dossier cible.

  2. Appuyez sur .

  3. Appuyez sur Télécharger le contenu du dossier. Le contenu du dossier ne doit pas dépasser la taille maximum du cache, soit 2 Go. Le téléchargement ne démarre pas si votre appareil ne dispose pas d'assez d'espace pour télécharger le contenu.

  4. Pour revenir à vos fichiers, appuyez sur Retour.
