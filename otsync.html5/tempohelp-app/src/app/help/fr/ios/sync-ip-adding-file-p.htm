Ajout d'un fichier

Pour ajouter un fichier :

  1. Vous devez importer le fichier depuis une autre application Affichez le fichier que vous voulez importer et appuyez sur et/ou Ouvrir dans.

  2. Appuyez sur Ouvrir dans Tempo.

  3. Procédez de l'une des façons suivantes :

    • Pour ajouter le fichier à la racine de votre dossier Tempo Box, appuyez sur Télécharger.

    • Pour choisir un emplacement autre que la racine de votre dossier Tempo Box, appuyez sur Destination : Espace de travail personnel

      1. Appuyez sur OpenText Tempo Box. La racine de votre dossier Tempo Box apparaît.

      2. Accédez au dossier cible.

        Vous ne pouvez pas ajouter de fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

        Vous ne pouvez pas ajouter de fichier à un dossier qui a des attributs et des catégories obligatoires via Tempo Box. Pour déterminer cela, voir Afficher les détails de l'objet.

      3. Pour sélectionner le dossier cible, appuyez sur Choisir .

        Pour ajouter un nouveau dossier, cliquez sur Nouveau dossier. Saisissez le nom du nouveau dossier, puis appuyez sur Terminé. Appuyez sur Choisir pour sélectionner le nouveau dossier.

      4. Cliquez sur Charger.

  4. Facultatif Dans OpenText Tempo, appuyez sur le fichier importé. Pour déplacer le fichier à un autre emplacement, voir Déplacement ou copie d'un fichier ou dossier.
