Ouverture d'un fichier dans une autre application

Pour ouvrir un fichier dans une autre application :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible. Si le fichier cible est marqué comme favori, appuyez sur Favoris.

  2. Appuyez sur le fichier à exporter. Le fichier s'ouvre.

  3. Appuyez sur . Une liste d'applications capables d'ouvrir le fichier apparaît.

  4. Appuyez sur une application.
