Présentation des icônes d'OpenText Tempo Box

Les principales icônes suivantes s'affichent dans toute l'application OpenText Tempo Box.

Icône Description
Vous permet d'accéder à la page d'accueil depuis une application.
Vous permet d'ajouter un nouveau dossier ou une photo à votre dossier Tempo Box.
Actualise l'affichage.
Affiche l'écran complet lorsque le iPad est en mode paysage.
Affiche les principales icônes d'OpenText Tempo Box quand l'iPad est en mode portrait.
Permet de visualiser les détails d'un dossier et de partager un dossier.

Icônes de fichiers

Quand vous visualisez le fichier, les icônes suivantes sont affichées.

Icône Description
Permet de couper le fichier ouvert pour le déplacer.
Permet de copier le fichier ouvert.
Permet de coller le dernier fichier coupé ou copié. Cette icône apparaît après avoir sélectionné l'icône Couper ou Copier.
Permet de supprimer le fichier ouvert.
Fichier non marqué comme favori.
Fichier marqué comme favori. Le recouvrement identifie un fichier marqué comme favori. Par exemple, .
Permet d'envoyer par e-mail le fichier sélectionné.
Permet d'ouvrir le fichier sélectionné dans une autre application.

Icônes des dossiers partagés

OpenText Tempo Box propose les types de dossiers partagés suivants.

Icône Description
Cette icône de dossier représente un dossier partagé pour lequel vous disposez d'un accès en lecture/écriture. Si un nom d'utilisateur est ajouté au nom du dossier, cela signifie qu'un utilisateur partage ce dossier avec vous. Par exemple, Ressources marketing (rduval). Pour plus d'informations, consultez la section Visualisation des fichiers d'un dossier partagé.
Le cache de verrouillage représente un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Pour plus d'informations, consultez la section Visualisation des fichiers d'un dossier partagé.
Cette icône de dossier représente un dossier Entreprise pour lequel vous disposez d'un accès en lecture/écriture. Vous devez vous abonner à un dossier Entreprise dans Content Server ou connaître quelqu'un détenant l' Modifier les autorisations autorisation pour le dossier afin qu'il vous envoie une invitation de partage pour accéder au contenu depuis votre dossier Tempo Box. Pour plus d'informations, consultez la section Visualisation des fichiers d'un dossier partagé.
Le cache de verrouillage sur cette icône de dossier représente un dossier Entreprise pour lequel vous disposez d'un accès en lecture seule. Vous devez vous abonner à un dossier Entreprise dans Content Server ou connaître quelqu'un détenant l' Modifier les autorisations autorisation pour le dossier afin qu'il vous envoie une invitation de partage pour accéder au contenu depuis votre dossier Tempo Box. Pour plus d'informations, consultez la section Visualisation des fichiers d'un dossier partagé .
Cette icône de dossier représente une bibliothèque de fichier dans une communautéTempo Social pour laquelle vous disposez d'un accès en lecture/écriture. Vous devez être membre d'une communauté Tempo Social pour accéder aux bibliothèques de fichier depuis votre dossier Tempo Box. Pour plus d'informations sur les communautés Tempo Social, voir OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social est un produit distinct d'OpenText Tempo Box. Si Tempo Social est installé et que vous ne voyez pas les bibliothèques de fichier dans votre dossierTempo Box, contactez votre administrateur.
Le cache de verrouillage sur cette icône de dossier représente une bibliothèque de fichier dans une communauté Tempo Social pour laquelle vous disposez d'un accès en lecture seule. Pour plus d'informations, reportez-vous au OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
Le cache de verrouillage sur un fichier représente un fichier dans un dossier partagé en lecture seule.
Une coche superposée sur un fichier représente un fichier réservé.
