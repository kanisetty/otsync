Pour modifier votre profil :

  1. Appuyez sur , puis sur Changer de profil.

  2. Sur l'écran OpenText Tempo Box Profil, ajoutez un nouveau profil en utilisant le nouveau nom d'utilisateur ou la nouvelle adresse URL OpenText Tempo Box et choisissez un nom de profil unique. Pour plus d'informations, voir Création d'un profil.

  3. Connexion à OpenText Tempo Box à l'aide du nouveau profil. Vous pouvez à présent accéder à votre dossier OpenText Tempo Box en utilisant le nouveau profil. Toutefois, la liste des favoris du profil d'origine ne sera pas disponible dans le nouveau profil.

    Pour préserver votre liste de favoris, vous devez modifier le profil d'origine en procédant comme suit :

    1. Appuyez sur .

    2. Appuyez sur le nom du profil à modifier.

    3. Modifiez le nom de l'utilisateur ou le serveur.

    4. Appuyez sur le champ Mot de passe, puis saisissez le mot de passe de cet utilisateur.

    5. Appuyez sur Terminé.

    6. Appuyez sur Déconnexion.

    7. Dans l'écran Profil OpenText Tempo Box, connectez-vous à OpenText Tempo Box avec votre profil d'origine.

      Vous pouvez à présent supprimer le deuxième profil que vous aviez créé dans étape 2. Pour plus d'informations, voir Suppression de votre profil.

Pour modifier votre mot de passe :

  1. Appuyez sur .

  2. Appuyez sur Déconnexion.

  3. Appuyez sur votre profil.

  4. Appuyez sur le champ Mot de passe, puis saisissez votre nouveau mot de passe.
