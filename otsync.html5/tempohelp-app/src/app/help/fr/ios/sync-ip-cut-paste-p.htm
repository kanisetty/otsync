Déplacement ou copie d'un fichier ou dossier

 

Pour déplacer ou copier un fichier :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible.

    Vous pouvez uniquement copier un fichier d'un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

    De même, vous pouvez uniquement copier un fichier lorsqu'il est réservé par un autre collaborateur dans un dossier partagé pour lequel vous disposez d'un accès en lecture/écriture. Les dossiers partagés en lecture/écriture sont identifiés par le recouvrement. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vous pouvez vérifier l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

  2. Appuyez sur le fichier que vous souhaitez déplacer ou copier.

  3. Appuyez sur pour déplacer le fichier ou pour copier le fichier.

  4. Appuyez sur Retour, puis accédez au dossier cible dans lequel vous souhaitez placer le contenu du presse-papiers.

    Vous ne pouvez pas déplacer ou copier un fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement.

  5. Appuyez sur pour coller le fichier.

Pour déplacer ou copier un dossier :

  1. Appuyez sur Tempo Box, puis accédez au dossier cible.

  2. Appuyez sur pour afficher les détails du dossier que vous souhaitez déplacer ou copier.

  3. Appuyez sur pour déplacer le dossier ou pour copier le dossier.

     

    Note :
    Avant de poursuivre, reportez-vous aux Notes.

     

  4. Appuyez sur Retour, puis accédez au dossier cible dans lequel vous souhaitez placer le contenu du presse-papiers.

    Vous ne pouvez pas déplacer ou copier un dossier dans un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

  5. Appuyez sur pour coller le dossier.

 

Notes :

  • Vous pouvez déplacer des dossiers partagés qui vous appartiennent à l'emplacement de votre choix dans votre dossier Tempo Box, excepté dans un autre dossier partagé. Les dossiers partagés qui vous appartiennent affichent uniquement le nom du dossier et sont identifiés sous la forme .

  • Vous pouvez uniquement copier la racine d'un dossier partagé avec vous. Dans ce cas, une copie privée du dossier est créée au nouvel emplacement, mais le dossier partagé d'origine reste à la racine de votre dossier Tempo Box. Le nom d'utilisateur du propriétaire d'un dossier est ajouté au nom du dossier lorsqu'il est partagé avec vous et est identifié sous la forme (lecture/écriture) ou (lecture seule).

  • Vous pouvez uniquement copier la racine d'un dossier Entreprise, quel que soit votre niveau d'accès. De même, vous pouvez uniquement copier les sous-dossiers au sein d'un dossier Entreprise pour lequel vous disposez d'un accès en lecture seule. Dans ce cas, une copie privée du dossier est créée au nouvel emplacement, mais le dossier Entreprise d'origine reste à l'emplacement d'origine de votre dossier Tempo Box. Les dossiers Entreprise sont identifiés sous la forme (lecture/écriture) ou (lecture seule).

  • Vous pouvez uniquement copier la racine d'une bibliothèque de fichier dans une communauté Tempo Social, qui est identifiée sous la forme (lecture/écriture) ou (lecture seule). Dans ce cas, une copie privée de la bibliothèque de fichier est créée au nouvel emplacement, mais la bibliothèque de fichier d'origine reste à la racine de votre dossier Tempo Box.
