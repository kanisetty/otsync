Modifier et gérer votre profil

 

Pour modifier votre profil :

  1. Sur la page Profil, maintenez le doigt sur votre profil, puis appuyez sur Modifier.

     

    Astuce :
    Si vous êtes connecté, appuyez sur le bouton Accueil pour revenir à la page d'accueil, puis appuyez sur Paramètres > Déconnexion pour revenir à la page Profil.

     

  2. Modifiez le nom de l'utilisateur ou le serveur.

  3. Appuyez sur Terminé.

Pour modifier votre mot de passe :

  1. Sur la page Profil, appuyez sur votre profil.

     

    Astuce :
    Si vous êtes connecté, appuyez sur le bouton Accueil pour revenir à la page d'accueil, puis appuyez sur Paramètres > Déconnexion pour revenir à la page Profil.

     

  2. Appuyez sur le champ Mot de passe, puis saisissez votre nouveau mot de passe.

  3. Appuyez sur Terminé.

Pour plus d'informations sur la gestion de votre profil, reportez-vous aux autres rubriques dans Modifier et gérer votre profil .
