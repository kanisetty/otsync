Suppression d'un profil

Pour supprimer un profil :

  1. Sur la page Profil, maintenez votre doigt sur le profil à supprimer.

     

    Astuce :
    Si vous êtes connecté, appuyez sur le bouton Accueil pour revenir à la page d'accueil, puis appuyez sur Paramètres > Déconnexion pour revenir à la page Profil.

     

  2. Appuyez sur Supprimer.

  3. Pour confirmer cette opération, appuyez sur Supprimer.
