Ajout, visualisation et suppression de favoris

 

Pour ajouter un favori :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible. Vous pouvez marquer comme favoris aussi bien des fichiers personnels que partagés.

  2. Appuyez sur le fichier que vous souhaitez marquer comme favori. Le fichier s'ouvre.

  3. Appuyez sur . L'icône prend l'apparence suivante et le fichier est marqué à l'aide du recouvrement dans l'affichage du dossier. Par exemple, . Marquer un fichier comme favori vous permet d'afficher le fichier lorsque vous êtes hors ligne.

    Un enregistrement de vos fichiers favoris est stocké dans votre profil, ce qui est spécifique à votre iPad ou iPhone. Vous devez marquer les fichiers comme favoris sur chacun de vos smartphones et tablettes qui utilisent l'application OpenText Tempo.

  4. Pour revenir à vos fichiers, appuyez sur Retour.

     

    Astuce :
    Vous pouvez également marquer les fichiers comme favoris en appuyant sur pour le fichier, puis cliquez sur Ajouter aux favoris.

     

Pour visualiser un favori :

  1. Appuyez sur Favoris.

  2. Facultatif Appuyez sur pour être sûr de visualiser le contenu le plus récent.

  3. Appuyez sur le fichier à visualiser.

  4. Pour revenir à vos fichiers, appuyez sur Retour.

Pour supprimer un favori :

  1. Appuyez sur Favoris ou sur Tempo Box, puis accédez au fichier cible.

  2. Appuyez sur le fichier que vous souhaitez supprimer de votre liste de fichiers favoris. Le fichier s'ouvre.

  3. Appuyez sur . L'icône prend alors l'apparence suivante . Le fichier n'apparaît plus dans votre liste de favoris.

    Il n'est toutefois pas supprimé de votre dossier Tempo Box.

  4. Pour revenir à vos fichiers, appuyez sur Retour.

     

    Astuce :
    Vous pouvez également supprimer le fichier de votre liste de favoris en appuyant sur pour le fichier, puis sur Enlever le favori.
