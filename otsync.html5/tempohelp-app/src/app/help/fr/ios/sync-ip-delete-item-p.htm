Suppression d'un fichier ou d'un dossier

 

Pour supprimer un fichier :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible. Si le fichier cible est marqué comme favori, appuyez sur Favoris.

    Vous ne pouvez pas supprimer un fichier d'un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

    Vous ne pouvez pas supprimer un fichier réservé. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour supprimer un fichier réservé, vous devez d'abord annuler sa réservation. Pour plus de détails, voir Réserver un fichier et annuler sa réservation.

  2. Faites glisser votre doigt horizontalement sur le nom du fichier. Un bouton Supprimer apparaît en regard du fichier.

     

    Note :
    Avant de poursuivre, reportez-vous aux Attention informations ci-dessous.

     

  3. Appuyez sur Supprimer.

  4. Pour confirmer cette opération, appuyez sur Supprimer cet élément.

Pour supprimer un fichier en cours de visualisation :

  1. Appuyez sur Tempo Box, puis accédez au fichier cible.

    Vous ne pouvez pas supprimer un fichier d'un dossier partagé pour lequel vous disposez d'un accès en lecture seule. Les dossiers partagés en lecture seule sont identifiés par le recouvrement. Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

    Vous ne pouvez pas supprimer un fichier réservé. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour supprimer un fichier réservé, vous devez d'abord annuler sa réservation. Pour plus de détails, voir Réserver un fichier et annuler sa réservation.

  2. Appuyez sur le fichier à supprimer. Le fichier s'ouvre.

     

    Note :
    Avant de poursuivre, reportez-vous aux Attention informations ci-dessous.

  4. Pour confirmer cette opération, appuyez sur Supprimer cet élément.

Pour supprimer un dossier :

  1. Appuyez sur Tempo Box, puis accédez au dossier cible.

    Si le dossier cible est un dossier partagé, voir Suppression d'un dossier partagé avant de poursuivre. Les dossiers partagés sont identifiés par le recouvrement (accès en lecture/écriture) ou recouvrement (accès en lecture seule). Pour plus de détails, voir Présentation des icônes d'OpenText Tempo Box.

    Vous ne pouvez pas supprimer un dossier s'il contient un fichier réservé. Pour annuler la réservation du fichier, voir Réserver un fichier et annuler sa réservation.

  2. Faites glisser votre doigt horizontalement sur le nom du dossier. Un bouton Supprimer apparaît en regard du dossier.

     

    Note :
    Avant de poursuivre, reportez-vous aux Attention informations ci-dessous.

     

  3. Appuyez sur Supprimer.

  4. Pour confirmer cette opération, appuyez sur Supprimer cet élément.

 

Attention

  • Supprimer un fichier ou un dossier entraîne sa suppression du dossier Tempo Box, même si le fichier est marqué comme favori. Cela signifie que le fichier ou dossier est supprimé de tous les autres clients OpenText Tempo Box que vous avez installés, tels que le client de bureau, d'autres smartphones ou tablettes, ainsi que l'interface utilisateur Web OpenText Tempo Box.

  • Si un fichier ou dossier est supprimé d'un dossier partagé pour lequel vous disposez d'un accès en lecture/écriture, le fichier ou dossier est automatiquement supprimé pour tous les utilisateurs qui bénéficient d'un accès au dossier partagé, y compris pour le propriétaire du dossier partagé. Les dossiers partagés en lecture/écriture sont identifiés par le recouvrement.

  • Si vous supprimez un fichier ou un dossier d'un dossier Entreprise pour lequel vous disposez d'un accès en lecture/écriture, le fichier ou dossier est automatiquement supprimé du dossier dans l'Espace de travail Entreprise.
