Het aanpassen van uw profiel Picture

 

Om uw profielfoto te veranderen:

  1. Op de Landing pagina, tikt Instellingen.

  2. Tik op Profiel Foto rij en voer een van de volgende:

    • Tik Van camera om uw profielfoto instellen met behulp van de Android camera. Maak de foto en tik vervolgens op Opslaan.

    • Tik Van Galerij om uw profielfoto instellen met behulp van een foto van de Android fotogalerij. Navigeer naar de afbeelding die u wilt gebruiken en tik op de foto.

    • Tap Set To Default to set your profile picture using the default profile picture.

  3. Tap the Android Back key to return to the Settings page.

  4. Tap the Android Back key to return to the Landing page.
