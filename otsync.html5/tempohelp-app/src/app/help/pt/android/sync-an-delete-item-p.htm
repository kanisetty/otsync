Excluindo um arquivo ou pasta

Para excluir um arquivo:

  1. Toque em Tempo Box e navegue até o arquivo de destino.

    Não é possível excluir um arquivo de uma pasta compartilhada à qual você tem acesso somente leitura. As pastas compartilhadas somente leitura são identificadas pela camada . Para obter detalhes, consulte Entendendo os ícones do OpenText Tempo Box.

    Não é possível excluir um arquivo enquanto ele estiver reservado. Os arquivos reservados foram marcados com a camada . Para excluir um arquivo reservado, você primeiramente deve cancelar a reserva do arquivo. Para obter detalhes, consulte Reservando e cancelando a reserva de um arquivo.

     

    Nota:
    Antes de continuar, leia atentamente as informações de Cuidado abaixo.

     

  2. Mantenha o toque no arquivo que você deseja excluir.

  3. Toque em Excluir.

  4. Toque em Ok para confirmar essa ação.

Para excluir uma pasta:

  1. Toque em Tempo Box e navegue até a pasta de destino.

    Se a pasta de destino for uma pasta compartilhada, consulte Excluindo uma pasta compartilhada. As pastas compartilhadas são identificadas pela camada (acesso de leitura/gravação) ou (acesso somente leitura). Para obter detalhes, consulte Entendendo os ícones do OpenText Tempo Box.

    Não é possível excluir uma pasta se ela contiver um arquivo reservado. Para cancelar a reserva do arquivo, consulte Reservando e cancelando a reserva de um arquivo.

     

    Nota:
    Antes de continuar, leia atentamente as informações de Cuidado abaixo.

     

  2. Mantenha o toque na pasta que você deseja excluir.

  3. Toque em Excluir.

  4. Toque em Ok para confirmar essa ação.

 

Cautela

  • A exclusão de um arquivo ou pasta o remove da pasta do Tempo Box, mesmo que ele tenha sido marcado como favorito. Isso significa que o arquivo ou pasta será excluído de todos os outros clientes OpenText Tempo Box instalados, como o cliente de desktop, outros smartphones ou tablets, além da interface de usuário da Web do OpenText Tempo Box.

  • Se você excluir um arquivo ou pasta de uma pasta compartilhada à qual você tem acesso de leitura/gravação, o arquivo ou pasta será automaticamente excluído para todos os usuários que têm acesso à pasta compartilhada, incluindo o proprietário dela. As pastas compartilhadas para leitura/gravação são identificadas pela camada .
