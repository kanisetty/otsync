Capítulo 1 – Entendendo os ícones do OpenText Tempo Box

 

Os ícones a seguir são exibidos na interface principal no sistema operacional Android versão 4.0 ou posterior. Para obter versões anteriores, estes recursos estão disponíveis no menu Android.

Ícone Descrição
Inicie a página Configurações na página Perfil.

Ícones de pasta compartilhada

O OpenText Tempo Box apresenta dois tipos de pastas compartilhadas.

Ícone Descrição
Esse ícone de pasta representa uma pasta compartilhada para a qual você tem acesso de leitura/gravação. Se o nome da pasta aparecer junto ao nome de usuário, é porque a pasta está compartilhada com você. Por exemplo, Materiais de marketing (jsmith). Para obter mais informações, consulte Exibindo arquivos em pastas compartilhadas.
Uma camada com cadeado representa uma pasta compartilhada à qual você tem acesso somente leitura. Para obter mais informações, consulte Exibindo arquivos em pastas compartilhadas.
Esse ícone representa uma pasta corporativa à qual você tem acesso de leitura/gravação. Você precisa se inscrever na pasta corporativa no Content Server ou que alguém com permissão Permissões de Edição para a pasta envie a você um convite de compartilhamento para acessar o conteúdo de sua pasta do Tempo Box. Para obter mais informações, consulte Exibindo arquivos em pastas compartilhadas.
Uma camada com cadeado nesse ícone de pasta representa uma pasta corporativa à qual você tem acesso somente leitura. Você precisa se inscrever na pasta corporativa no Content Server ou pedir que alguém com a permissão Permissões de Edição para a pasta envie a você um convite de compartilhamento para acessar o conteúdo de sua pasta do Tempo Box. Para obter mais informações, consulte Exibindo arquivos em pastas compartilhadas .
Esse ícone representa uma biblioteca de arquivos em uma comunidade do Tempo Social à qual você tem acesso de leitura/gravação. É necessário ser membro de uma comunidade do Tempo Social para acessar as bibliotecas de arquivos da sua pasta do Tempo Box. Para obter mais informações sobre as comunidades do Tempo Social, consulte OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). O OpenText Tempo Social é um produto separado do OpenText Tempo Box. Se você tem o Tempo Social instalado e não consegue ver as bibliotecas de arquivos na sua pasta do Tempo Box, entre em contato com seu administrador.
Uma camada com cadeado nesse ícone de pasta representa uma biblioteca de arquivos em uma comunidade do Tempo Social à qual você tem acesso somente leitura. Para obter mais informações, consulte OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
Uma camada com cadeado em um arquivo representa um arquivo em uma pasta compartilhada com acesso somente leitura.
Uma camada com marca de seleção em um arquivo representa um arquivo que está reservado.
