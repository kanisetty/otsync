Removendo um colaborador

Você pode remover um colaborador de uma pasta compartilhada que lhe pertence usando a interface de usuário do OpenText Tempo Box na Web. Essa ação rompe a conexão com a pasta compartilhada e a exclui da pasta do Tempo Box de colaboradores.

Você pode acessar a interface de usuário da Web digitando o URL do OpenText Tempo Box no endereço de um navegador da Web e fazendo login no OpenText Tempo Box. Para obter mais informações sobre a remoção de colaboradores, consulte a ajuda on-line na interface de usuário do OpenText Tempo Box na Web.
