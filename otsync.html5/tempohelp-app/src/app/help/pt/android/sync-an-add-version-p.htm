Adicionando um arquivo

Para adicionar uma nova versão de um arquivo em seu dispositivo:

  1. Toque em Tempo Box e navegue até o arquivo de destino.

     

    Importante:
    Ao adicionar uma nova versão ao arquivo em uma pasta compartilhada para a qual você tem acesso de leitura/gravação, o arquivo não reservado automaticamente. É possível adicionar uma nova versão ao arquivo ao mesmo tempo em que um colaborador faz alterações nele. Para evitar isso, você pode reservar o arquivo. Para obter detalhes, consulte Reservando e cancelando a reserva de um arquivo.

    As pastas compartilhadas para leitura/gravação são identificadas pela camada . Para obter detalhes, consulte Entendendo os ícones do OpenText Tempo Box.

     

    Você não pode adicionar uma nova versão a um arquivo quando esse estiver reservado por outro colaborador em uma pasta compartilhada. Os arquivos reservados foram marcados com a camada . Para descobrir quem reservou o arquivo, você pode verificar o histórico de auditorias do arquivo. Para obter detalhes, consulte Visualizando detalhes de objetos.

    Além disso, não é possível adicionar uma nova versão a um arquivo em uma pasta compartilhada à qual você tem acesso somente leitura. As pastas compartilhadas somente leitura são identificadas pela camada .

  2. Mantenha o toque no arquivo ao qual deseja adicionar uma nova versão.

  3. Toque em Adicionar versão > Do dispositivo.

  4. Navegue até a nova versão do arquivo no seu dispositivo.

  5. Toque na nova versão do arquivo.

Para carregar um arquivo armazenado no seu dispositivo:

  1. Toque em Tempo Box e navegue até a pasta de destino onde deseja fazer o upload do arquivo.

    Não é possível fazer upload de um arquivo em uma pasta compartilhada à qual você tem acesso somente leitura. As pastas compartilhadas somente leitura são identificadas pela camada .

    Não é possível adicionar um arquivo a uma pasta que possui categorias e atributos obrigatórios através do Tempo Box. Para determinar isso, consulte Visualizando detalhes de objetos.

  2. Mantenha o toque na pasta de destino. Toque em Fazer upload aqui > Do dispositivo.

  3. Navegue até o arquivo de destino no dispositivo Android.

  4. Toque no arquivo.

Para importar um arquivo de outro aplicativo:

  1. No outro aplicativo, exiba um arquivo.

  2. Toque em Exportar ou Compartilhar.

  3. Toque em Tempo Box.

  4. Opcional Se solicitado, faça login no OpenText Tempo Box.

  5. Navegue até o local de destino onde deseja colocar o arquivo.

    Não é possível importar um arquivo para uma pasta compartilhada à qual você tem acesso somente leitura. As pastas compartilhadas somente leitura são identificadas pela camada .

    Não é possível adicionar um arquivo a uma pasta que possui categorias e atributos obrigatórios através do Tempo Box. Para determinar isso, consulte Visualizando detalhes de objetos.

  6. Toque na tecla Menu do Android e toque em Colar.
