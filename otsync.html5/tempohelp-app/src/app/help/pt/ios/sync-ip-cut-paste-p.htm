Movendo ou copiando um arquivo ou pasta

Para mover ou copiar um arquivo:

  1. Toque em Tempo Box e navegue até o arquivo de destino.

    É possível somente copiar arquivos de uma pasta compartilhada para a qual você tem acesso somente leitura. As pastas compartilhadas somente leitura são identificadas pela camada . Para obter detalhes, consulte Entendendo os ícones do OpenText Tempo Box.

    Além disso, só é possível copiar arquivos quando eles estiverem reservados por outro colaborador em uma pasta compartilhada, à qual você tem acesso de leitura/gravação. As pastas compartilhadas para leitura/gravação são identificadas pela camada . Os arquivos reservados foram marcados com a camada . Para descobrir quem reservou o arquivo, você pode verificar o histórico de auditorias do arquivo. Para obter detalhes, consulte Visualizando detalhes de objetos.

  2. Toque no arquivo que você deseja mover ou copiar.

  3. Toque em para mover o arquivo ou para copiar o arquivo.

  4. Toque em Voltar e navegue até a pasta de destino onde você deseja colocar o conteúdo da área de transferência.

    Não é possível mover nem copiar um arquivo para uma pasta compartilhada para a qual você tem acesso somente leitura. As pastas compartilhadas somente leitura são identificadas pela camada .

  5. Toque em para colar o arquivo.

Para mover ou copiar uma pasta:

  1. Toque no Tempo Box e navegue até a pasta de destino.

  2. Toque em para visualizar os detalhes da pasta que você deseja mover ou copiar.

  3. Toque em para mover a pasta ou para copiar a pasta.

     

    Nota:
    Antes de continuar, leia atentamente as Observações.

     

  4. Toque em Voltar e navegue até a pasta de destino onde você deseja colocar o conteúdo da área de transferência.

    Não é possível mover nem copiar uma pasta para uma pasta compartilhada para a qual você tem acesso somente leitura. As pastas compartilhadas somente leitura são identificadas pela camada . Para obter detalhes, consulte Entendendo os ícones do OpenText Tempo Box.

  5. Toque em para colar a pasta.

 

Notas:

  • Você pode mover suas pastas compartilhadas para qualquer lugar de sua pasta do Tempo Box, exceto para outra pasta compartilhada. As pastas compartilhadas que você possui exibem somente o nome da pasta e são identificadas como .

  • Só é possível copiar a raiz de uma pasta compartilhada com você. Ao fazer isso, uma cópia particular será criada no novo local, mas a pasta compartilhada original permanecerá no nível da raiz da pasta do Tempo Box. As pastas compartilhadas com você recebem o nome de usuário do proprietário da pasta no nome da pasta e são identificadas como (leitura/gravação) ou (somente leitura).

  • Só é possível copiar a raiz de uma pasta corporativa, independentemente do seu nível de acesso. Além disso, só é possível copiar subpastas em uma pasta corporativa à qual você tem acesso somente leitura. Ao fazer isso, uma cópia particular será criada no novo local, mas a pasta corporativa original permanecerá no local original em sua pasta do Tempo Box. As pastas corporativas são identificadas como (leitura/gravação) ou (somente leitura).

  • Só é possível copiar a raiz de uma biblioteca de arquivos em uma comunidade de Tempo Social, que é identificada como (leitura/gravação) ou (somente leitura). Ao fazer isso, uma cópia particular da biblioteca de arquivos será criada no novo local, mas a biblioteca de arquivos original permanecerá no nível da raiz da pasta do Tempo Box.
