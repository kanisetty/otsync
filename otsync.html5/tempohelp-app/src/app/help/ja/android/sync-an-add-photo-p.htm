写真やフォルダーの追加

Android カメラを使用して写真を追加するには:

  1. [Tempo Box] をタップし、写真を追加する対象のフォルダーに移動します。

    読み取り専用アクセス権を持つ共有フォルダーに写真を追加することはできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

    Tempo Box では、必須のカテゴリと属性のあるフォルダーに写真を追加することはできません。その点を確認する方法については、次を参照してください。 オブジェクトの詳細の表示.

  2. Android の [メニュー] キーをタップします。[追加] > [カメラから] をタップします。

  3. 写真を撮り、画面に表示される指示に従います。写真は .jpg ファイルとして保存されます。アップロードする前に写真の名前を編集することはできません。

デバイス ギャラリーから写真をインポートするには:

  1. Android ギャラリー アプリを起動します。

  2. インポートする写真がある場所へ移動します。

  3. Android の [メニュー] キーをタップして、[共有] をタップします。

  4. [Tempo Box] をタップします。

  5. 写真を追加するターゲット フォルダーに移動します。

    読み取り専用アクセス権を持つ共有フォルダーに写真をインポートすることはできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

    Tempo Box では、必須のカテゴリと属性のあるフォルダーに写真を追加することはできません。その点を確認する方法については、次を参照してください。 オブジェクトの詳細の表示.

  6. Android の [メニュー] キーをタップして、[貼り付け] をタップします。

フォルダーを追加するには:

  1. [Tempo Box] をタップし、新しいフォルダーの追加場所に移動します。

    読み取り専用アクセス権を持つ共有フォルダーにフォルダーを追加することはできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

  2. Android の [メニュー] キーをタップします。[追加] > [フォルダー] をタップします。

  3. [フォルダー名] ボックスをタップし、このフォルダーの名前を入力します。

  4. [OK] をタップします。
