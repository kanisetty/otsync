プロファイルの写真の変更

 

プロファイルの写真を編集するには、次の操作を行います。

  1. Android の [メニュー] キーをタップして、 をタップします。

  2. [プロファイルの写真] 行をタップして、次のいずれかの操作を実行します。

    • [カメラから] をタップして、Android のカメラを使用して、プロファイルの写真を設定します。写真を撮影して、[保存] をタップします。

    • [ギャラリーから] をタップして、Android の写真ギャラリーにある写真を選択して、プロファイルの写真を設定します。使用する写真に移動して、写真をタップしてください。

    • [データファイルに設定] をタップして、OpenText Tempo Box のデフォルトのプロファイル写真を使用して、プロファイルの写真を設定します。

  3. Android の [戻る] キーをタップして、[設定] 画面に戻ります。

  4. Android の [戻る] キーをタップして、ファイルに戻ります。
