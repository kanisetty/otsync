プロファイルの作成

 

プロファイルを作成するには、次の操作を行います。

  1. 初めて OpenText Tempo Box アプリケーションを起動した場合は、[プロファイルの追加] をタップします。

    OpenText Tempo Box に既にサインインしており、追加のプロファイルを作成する場合は、次の操作を行います。

    1. Android の [メニュー] キーをタップします。

    2. をタップします。

    3. [プロファイルの追加] をタップします。

  2. [プロファイル名] ボックスをタップして、お使いのデバイスに固有のプロファイル名を入力してください。

  3. [ユーザー名] ボックスをタップし、ユーザー名を入力します。社外のユーザーの場合、E メール アドレスを入力します。

     

    注意:
    社外ユーザーはプロファイルを作成する前に OpenText Tempo アカウントを作成する必要があります。

    アカウントを作成するには、社内ユーザーが外部ユーザーとフォルダーを共有する必要があります。これによって、外部ユーザーに、アカウントを作成するために必要な手順を説明した招待が E メールで送信されます。

     

  4. [サーバー] ボックスをタップし、OpenText Tempo Box サーバーの URL を入力します。

    OpenText Tempo Box サーバーの URL については、管理者にお問い合わせください。

    社外ユーザーの場合は、OpenText Tempo Box アカウントの作成時に送信された E メールに OpenText Tempo Box URL がリストされています。

  5. [プロファイルの追加] をタップします。OpenText Tempo Box にサインインすると、パスワードを入力するように求められます。

    プロファイル データにはアカウントの証明書、OpenText Tempo Box URL、お気に入りのファイルのリストが含まれます。プロファイルは、モバイル デバイスに固有です。OpenText Tempo Box アプリケーションを実行するスマートフォンやタブレットごとにプロファイルを作成し、ファイルにお気に入りタグを付ける必要があります。
