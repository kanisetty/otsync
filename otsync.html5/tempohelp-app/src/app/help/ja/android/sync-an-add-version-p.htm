ファイルの追加

 

デバイスにあるファイルに新しいバージョンを追加するには:

  1. [Tempo Box] をタップし、対象のファイルに移動します。

     

    重要項目:
    読み取り/書き込みアクセス権を持つ共有フォルダーのファイルに新しいバージョンを追加するときに、そのファイルが自動的に予約されることはありません。ファイルに新しいバージョンを追加しているとき、同時に共同作業者が変更を加えている可能性があります。そうならないようにするため、そのファイルを予約できます。詳細については、次を参照してください。 ファイルの予約と予約解除.

    読み取り/書き込みが可能な共有フォルダーには、 のオーバーレイが表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

     

    共有フォルダー内で他の共同作業者によって予約されているファイルに新しいバージョンを追加することはできません。予約されているファイルには、 のオーバーレイが表示されます。そのファイルを予約したユーザーを確認するには、そのファイルの監査履歴を調べます。詳細については、次を参照してください。 オブジェクトの詳細の表示.

    読み取り専用アクセス権を持つ共有フォルダーのファイルに新しいバージョンを追加することもできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。

  2. 新しいバージョンを追加するファイルをタッチしてホールドします。

  3. [バージョンの追加] > [デバイスから] をタップします。

  4. デバイスにあるファイルの新しいバージョンに移動します。

  5. ファイルの新しいバージョンをタップします。

デバイスに保存されているファイルをアップロードするには:

  1. [Tempo Box] をタップして、ファイルをアップロードする対象のフォルダーに移動します。

    読み取り専用アクセス権を持つ共有フォルダーにファイルをアップロードすることはできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。

    Tempo Box では、必須のカテゴリと属性のあるフォルダーにファイルを追加することはできません。その点を確認する方法については、次を参照してください。 オブジェクトの詳細の表示.

  2. 対象のフォルダーをタッチしてホールドします。[ここにアップロード] をタップして、[デバイスから] をタップします。

  3. Android デバイス上の対象のファイルに移動します。

  4. ファイルをタップします。

別のアプリからファイルをインポートするには:

  1. 別のアプリから、ファイルを表示します。

  2. [エクスポート] または [共有] をタップします。

  3. [Tempo Box] をタップします。

  4. オプション サインインが求められた場合は、OpenText Tempo Box にサインインします。

  5. ファイルを配置する対象の場所に移動します。

    読み取り専用アクセス権を持つ共有フォルダーにファイルをインポートすることはできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。

    Tempo Box では、必須のカテゴリと属性のあるフォルダーにファイルを追加することはできません。その点を確認する方法については、次を参照してください。 オブジェクトの詳細の表示.

  6. Android の [メニュー] キーをタップして、[貼り付け] をタップします。
