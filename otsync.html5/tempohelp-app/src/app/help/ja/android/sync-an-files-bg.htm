第2章. ファイルの表示、編集、作業

 

ファイルを表示するには:

  1. [Tempo Box] をタップし、対象のファイルに移動します。OpenText Tempo Box タイトル バーに、現在表示しているフォルダー名が表示されます。

  2. オプション 最新のコンテンツが表示されるようにするには、表示を定期的に更新する必要があります。詳細については、次を参照してください。 表示の更新.

  3. 以下のいずれかの操作を行います。

    • 表示するファイルをタップします。ファイルを開くことができるアプリのリストが表示されます。アプリをタップします。

    • 表示するファイルをタッチしてホールドします。[開く] をタップして、アプリをタップします。

     

    注意:
    特定の MIME タイプのデフォルトのアプリを設定するには、アプリを選択する前に、[この操作にデフォルトで使用する] チェック ボックスをタップします。

     

  4. Android の [戻る] キーをタップして、ファイルに戻ります。

ファイルを編集するには:

  1. [Tempo Box] をタップし、対象のファイルに移動します。

     

    重要項目:
    読み取り/書き込みアクセス権を持つ共有フォルダーのファイルを表示したとき、それらのファイルが自動的に予約されることはありません。ファイルを開いて編集しているとき、同時に共同作業者が変更を加えている可能性があります。そうならないようにするため、そのファイルを予約できます。詳細については、次を参照してください。 ファイルの予約と予約解除.

    読み取り/書き込みが可能な共有フォルダーには、 のオーバーレイが表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

     

    共有フォルダー内で他の共同作業者によって予約されているファイルの編集内容を保存することはできません。予約されているファイルには、 のオーバーレイが表示されます。そのファイルを予約したユーザーを確認するには、そのファイルの監査履歴を調べます。詳細については、次を参照してください。 オブジェクトの詳細の表示.

    読み取り専用アクセス権を持つ共有フォルダー内のファイルの編集内容を保存することもできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。

  2. 編集するファイルをタップして、サードパーティのエディターを選択します。

  3. アプリが編集をサポートする場合、ファイルを編集して変更を保存します。

    ファイルへの変更は、Tempo Box フォルダー内のファイルに保存されません。変更したファイルをアップロードするか、追加する必要があります。

  4. 変更したファイルを Tempo Box フォルダーにアップロードまたは追加するには、以下のいずれかの操作を行います。

    • サードパーティのアプリからファイルを Android デバイスに保存します。OpenText Tempo Box 内から既存のファイルに新しいバージョンを追加します。詳細については、次を参照してください。 ファイルの追加.

    • サードパーティのアプリから [共有] オプションまたは [エクスポート] オプションを使用して、保存したファイルを Tempo Box フォルダーにアップロードします。第 2 のファイルが Tempo Box フォルダーに追加されます。詳細については、次を参照してください。 ファイルの追加.

ファイルの管理については、以下の項目にある他のトピックを参照してください。ファイルの表示、編集、作業.
