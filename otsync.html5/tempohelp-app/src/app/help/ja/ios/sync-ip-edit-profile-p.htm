プロファイルを編集するには、次の操作を行います。

  1. をタップして [スイッチ プロファイル] をタップします。

  2. OpenText Tempo Box の [プロファイル] 画面で、新しいユーザー名または OpenText Tempo Box の URL を使用して新しいプロファイルを追加し、固有のプロファイル名を選択します。詳細についてはプロファイルの作成を参照してください。

  3. 新しいプロファイルを使用して、OpenText Tempo Box にサインインします。新しいプロファイルを使用して OpenText Tempo Box にアクセスできるようになります。ただし、元のプロファイルのお気に入りのリストは、新しいプロファイルでは利用できません。

    お気に入りのリストを保持するには、元のプロファイルを編集し、次の操作を行う必要があります。

    1. をタップします。

    2. 編集するプロファイル名をタップします。

    3. 必要に応じて、ユーザー名またはサーバーを変更します。

    4. [パスワード] ボックスをタップし、このユーザーのパスワードを入力します。

    5. [完了] をタップします。

    6. [ログアウト] をタップします。

    7. OpenText Tempo Box [プロファイル] 画面で、元のプロファイルを使用して OpenText Tempo Box にサインインします。

      これで、ステップ 2 で作成した 2 番目のプロファイルを削除できるようになりました。詳細についてはプロファイルの削除を参照してください。

パスワードを編集するには、次の操作を行います。

  1. をタップします。

  2. [ログアウト] をタップします。

  3. プロファイルをタップします。

  4. [パスワード] ボックスをタップし、新しいパスワードを入力します。
