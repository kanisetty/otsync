ファイルの追加

 

ファイルを追加するには:

  1. 他のアプリからファイルをインポートする必要があります。インポートするファイルを表示して、 や [プログラムから開く] をタップします。

  2. [Tempo で開く] をタップします。

  3. 以下のいずれかの操作を行います。

    • Tempo Box フォルダーのルートにファイルを追加するには、[アップロード] をタップします。

    • Tempo Box フォルダーのルート以外の場所を選択するには、[指定先: パーソナル ワークスペース] をタップします。

      1. [OpenText Tempo Box] をタップします。Tempo Box フォルダーのルートが表示されます。

      2. 対象のフォルダーに移動します。

        読み取り専用アクセス権を持つ共有フォルダーにファイルを追加することはできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

        Tempo Box では、必須のカテゴリと属性のあるフォルダーにファイルを追加することはできません。その点を確認する方法については、次を参照してください。 オブジェクトの詳細の表示.

      3. ターゲット フォルダーを選択するには、[選択] をタップします。

        新しいフォルダーを追加するには、[新規フォルダー] をクリックします。新しいフォルダーの名前を入力して、[完了] をタップします。[選択] をタップして、新しいフォルダーを選択します。

      4. [アップロード] をクリックします。

  4. オプション OpenText Tempo で、インポートしたファイルをタップします。別の場所にファイルを移動する方法については、次を参照してください。 ファイルまたはフォルダーの移動またはコピー.
