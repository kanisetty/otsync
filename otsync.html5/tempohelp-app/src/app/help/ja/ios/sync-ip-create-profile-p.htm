プロファイルを作成するには、次の操作を行います。

  1. 初めて OpenText Tempo Box アプリケーションを起動した場合は、[プロファイルの追加] をタップします。

    OpenText Tempo Box に既にサインインしており、追加のプロファイルを作成する場合は、次の操作を行います。

    1. をタップします。

    2. [スイッチ プロファイル] をタップします。

    3. [プロファイルの追加] をタップします。

  2. [ユーザー名] ボックスをタップし、ユーザー名を入力します。社外のユーザーの場合、E メール アドレスを入力します。

     

    注意:
    社外ユーザーはプロファイルを作成する前に、OpenText Tempo Box アカウントを作成する必要があります。

    アカウントを作成するには、社内ユーザーが外部ユーザーとフォルダーを共有する必要があります。これによって、外部ユーザーに、アカウントを作成するために必要な手順を説明した招待が E メールで送信されます。

     

  3. [パスワード] ボックスをタップし、パスワードを入力します。

    社外のユーザーの場合は、OpenText Tempo Box アカウントの作成時に選択したパスワードを入力します。パスワードを忘れた場合は、OpenText Tempo Box アカウントの作成時に送信された E メールを参照してください。

     

    注意:
    パスワードでは大文字と小文字が区別されます。パスワードを正しく入力していることを確認してください。

     

  4. [サーバー] ボックスをタップし、OpenText Tempo Box サーバーの URL を入力します。

    OpenText Tempo Box URL がわからない場合は、管理者にお問い合わせください。

    社外ユーザーの場合は、OpenText Tempo Box アカウントの作成時に送信された E メールに OpenText Tempo Box URL がリストされています。

  5. [プロファイル名] ボックスをタップして、お使いのデバイスに固有のプロファイル名を入力してください。

  6. オプション [パスワードの保存] をタップすると、次回アプリケーションを起動したときにすばやくアクセスできます。唯一の例外は、OpenText Tempo Box からログアウトしている場合やセッションがタイムアウトになっている場合です。下記の注意情報を参照してください。

  7. [完了] をタップします。プロファイル データにはアカウントの証明書、OpenText Tempo Box URL、お気に入りのファイルのリストが含まれます。プロファイルは、iPhone および iPad に固有です。OpenText Tempo Box アプリケーションを実行するスマートフォンやタブレットごとにプロファイルを作成し、ファイルにお気に入りタグを付ける必要があります。

 

注意

  • OpenText Tempo Box でパスワードを保存できるようにすると、他の人によってデバイスが使用されることによってセキュリティに問題が生じる可能性があります。

  • デバイスに複数のプロファイルを設定している場合、[パスワードを記録] がオンになっていてログアウトしていない場合に、他のプロファイルで OpenText Tempo Box アカウントにアクセスされる可能性があります。ログアウトするには、 をタップして、[ログアウト] をタップします。
