第6章. お気に入りの追加、表示、削除

お気に入りを追加するには:

  1. [Tempo Box] をタップし、対象のファイルに移動します。個人のファイルと共有ファイルの両方にお気に入りのタグを付けることができます。

  2. お気に入りのタグを付けるファイルをタップします。ファイルが開きます。

  3. 次のアイコンをタップします。. アイコンが に変わり、フォルダー表示のファイルに のオーバーレイが表示されます。例:. ファイルにお気に入りのタグを付けると、オフライン状態でそのファイルを表示できるようになります。

    お気に入りのファイルの記録がプロファイルに保管されます。そのプロファイルは、iPad や iPhone ごとに異なります。OpenText Tempo アプリを実行するスマートフォンやタブレットごとに、ファイルにお気に入りのタグを付ける必要があります。

  4. ファイルに戻るには、[戻る] をタップします。

     

    ヒント:
    ファイルにお気に入りのタグを付けるには、ファイルの をタップして、[お気に入りに追加] をタップするという方法もあります。

     

お気に入りを表示するには:

  1. [お気に入り] をタップします。

  2. オプション をタップすると、確実に最新のコンテンツが表示されます。

  3. 表示するファイルをタップします。

  4. ファイルに戻るには、[戻る] をタップします。

お気に入りを削除するには:

  1. [お気に入り] をタップするか、[Tempo Box] をタップして、ターゲット ファイルに移動します。

  2. お気に入りリストから削除するファイルをタップします。ファイルが開きます。

  3. 次のアイコンをタップします。. アイコンが次のように変化します。 . ファイルがお気に入りのリストに表示されなくなります。

    これによって Tempo Box フォルダーからファイルが削除されることはありません。

  4. ファイルに戻るには、[戻る] をタップします。

     

    ヒント:
    お気に入りからファイルを削除するには、ファイルの をタップして、[お気に入りから削除する] をタップするという方法もあります。
