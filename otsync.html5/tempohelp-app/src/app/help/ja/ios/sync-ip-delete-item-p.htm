ファイルまたはフォルダーの削除

ファイルまたはフォルダーを削除するには:

  1. [Tempo Box] をタップし、対象のファイルに移動します。対象のファイルにお気に入りのタグが付いている場合は、[お気に入り] をタップします。

    読み取り専用アクセス権を持つ共有フォルダーからファイルを削除することはできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

    予約されているファイルを削除することはできません。予約されているファイルには、 のオーバーレイが表示されます。予約されているファイルを削除するには、まずそのファイルの予約を解除する必要があります。詳細については、次を参照してください。 ファイルの予約と予約解除.

  2. ファイル名を指で水平になぞります。ファイル行に [削除] ボタンが表示されます。

     

    注意:
    作業を進める前に、下記の注意 を参照してください。

     

  3. [削除] をタップします。

  4. [このアイテムを削除] をタップして、この操作を確認します。

    読み取り専用アクセス権を持つ共有フォルダーからファイルを削除することはできません。読み取り専用の共有フォルダーには、 のオーバーレイが表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

    予約されているファイルを削除することはできません。予約されているファイルには、 のオーバーレイが表示されます。予約されているファイルを削除するには、まずそのファイルの予約を解除する必要があります。詳細については、次を参照してください。 ファイルの予約と予約解除.

  2. 削除するファイルをタップします。ファイルが開きます。

     

    注意:
    作業を進める前に、下記の注意 を参照してください。

     

  3. 次のアイコンをタップします。 .

フォルダーを削除するには

  1. [Tempo Box] をタップし、対象のフォルダ－に移動します。

    対象のフォルダーが共有フォルダーの場合は、作業を進める前に、共有フォルダーの削除 を参照してください。共有フォルダーには、 オーバーレイ (読み取り/書き込みアクセス権) または オーバーレイ (読み取り専用アクセス権) が表示されます。詳細については、次を参照してください。 OpenText Tempo Box アイコンについて.

    予約されているファイルが入っているフォルダーを削除することはできません。ファイルの予約を解除する方法については、次を参照してください。 ファイルの予約と予約解除.

  2. フォルダー名を指で水平になぞります。フォルダー行に [削除] ボタンが表示されます。

     

    注意:
    作業を進める前に、下記の注意 を参照してください。

     

  3. [削除] をタップします。

  4. [このアイテムを削除] をタップして、この操作を確認します。

 

注意

  • ファイルまたはフォルダーを削除すると、お気に入りとしてタグを付けられている場合でも、Tempo Box フォルダーから削除されます。つまり、ファイルまたはフォルダーは、インストールした他のすべての OpenText Tempo Box クライアント (デスクトップ クライアント)、その他のスマートフォン、タブレット、および OpenText Tempo Box Web ユーザー インターフェイスから削除されます。

  • 読み取り/書き込みアクセス権を持つ共有フォルダーからファイルまたはフォルダーを削除すると、共有フォルダーの所有者を含めて、共有フォルダーへのアクセス権限を持っているすべてのユーザーに対して、そのファイルまたはフォルダーが自動的に削除されます。読み取り/書き込みが可能な共有フォルダーには、 のオーバーレイが表示されます。

  • 読み取り/書き込みアクセス権を持つエンタープライズ フォルダーからファイルまたはフォルダーを削除すると、そのファイルまたはフォルダーがエンタープライズ ワークスペース内のフォルダーから自動的に削除されます。読み取り/書き込みが可能なエンタープライズ フォルダーには、次のアイコンが表示されます。 .
