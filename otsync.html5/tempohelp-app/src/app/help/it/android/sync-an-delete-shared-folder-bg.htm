Eliminazione di una cartella condivisa

Per eliminare una cartella condivisa di cui si è proprietari:

  1. Toccare Tempo Box e posizionarsi sulla cartella condivisa desiderata.

    Le cartelle condivise in lettura/scrittura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

    Se al nome della cartella desiderata viene aggiunto un nome utente, si dispone dell'accesso in condivisione della cartella desiderata. Per ulteriori informazioni, vedere Eliminazione di una cartella per la quale si dispone dell'accesso in condivisione.

  2. Tenere premuta la cartella condivisa.

  3. Toccare Elimina. Se viene visualizzato Rimuovi cartella condivisa, è stata selezionata una cartella condivisa con l'utente, una cartella Enterprise o una libreria di file da una comunità Tempo Social. Per ulteriori informazioni, vedere Eliminazione di una cartella per la quale si dispone dell'accesso in condivisione, Eliminazione di una cartella Enterprise o Rimozione di una libreria di file da una comunità Tempo Social.

     

    Cautela
    Questa operazione elimina la cartella per l'utente e tutti i collaboratori. In alternativa, è possibile rimuovere i collaboratori. Per ulteriori dettagli, vedere Rimozione di un collaboratore.

     

  4. Toccare OK per confermare l'azione.

Per eliminare una cartella condivisa:

  1. Toccare Tempo Box e individuare la cartella di destinazione.

    Una cartella condivisa mostra il nome utente del proprietario della cartella nel nome cartella, ad esempio, Materiale di marketing (jsmith), ed è identificata come o .

  2. Tenere premuta la cartella condivisa.

  3. Toccare Rimuovi cartella condivisa.

    Questa operazione interrompe la connessione alla cartella condivisa ed elimina la cartella dalla propria cartella di Tempo Box. Questa operazione non elimina la cartella per il proprietario o per gli altri collaboratori.

    Se viene visualizzato il pulsante Elimina, è stata selezionata una cartella condivisa di cui si è proprietari. Prima di continuare, vedere Eliminazione di una cartella condivisa di cui si è proprietari .

  4. Toccare OK per confermare l'azione.

Per rimuovere una cartella Enterprise:

  1. Nella radice della cartella di Tempo Box in uso, individuare la cartella aziendale di destinazione. Le cartelle Enterprise sono identificate come o .

  2. Tenere premuta la cartella Enterprise.

  3. Toccare Rimuovi cartella condivisa.

    Questa operazione interrompe la connessione alla cartella e la rimuove dalla cartella Tempo Box in uso. Non elimina la cartella nell'Area di lavoro organizzazione.

     

    Cautela
    È necessario rimuovere la cartella effettiva e non eliminare il contenuto all'interno della cartella Enterprise. L'eliminazione dei contenuti nella cartella Enterprise elimina i contenuti per tutti i collaboratori con sottoscrizione alla cartella, oltre a eliminare i contenuti nell'Area di lavoro organizzazione.

     

  4. Toccare OK per confermare l'azione.

Per rimuovere una libreria di file da una comunità Tempo Social:

  1. Nella radice della cartella di Tempo Box in uso, individuare la libreria di file di destinazione. Le librerie di file sono identificate come o .

  2. Tenere premuta la libreria di file.

  3. Toccare Rimuovi cartella condivisa.

    Questa operazione interrompe la connessione alla libreria di file e la rimuove dalla propria cartella Tempo Box. Questa operazione non elimina la libreria di file in Tempo Social. Se si elimina involontariamente la libreria di file, passare alla comunità in Tempo Social, lasciare la comunità e quindi unirsi nuovamente. Per ulteriori informazioni sulle comunità Tempo Social, vedere OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).

  4. Toccare OK per confermare l'azione.
