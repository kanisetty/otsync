Modifica e gestione del profilo

Per modificare il proprio profilo:

  1. Nella pagina Profilo, tenere premuto il profilo e toccare Modifica.

     

    Suggerimento:
    Se è stato eseguito l'accesso, toccare il tasto AndroidIndietro per tornare alla pagina Profilo.

     

  2. Modificare il nome utente o le informazioni sul server in base alle esigenze.

  3. Toccare Salva modifiche.

Per aggiornare la propria password:

  1. Nella pagina Profilo, toccare il profilo in uso.

     

    Suggerimento:
    Se è stato eseguito l'accesso, toccare il tasto AndroidIndietro per tornare alla pagina Profilo.

     

  2. Toccare la casella Password e digitare la nuova password. La password è sensibile alle maiuscole/minuscole. Assicurarsi di digitare la password correttamente.

  3. Toccare Accesso.

Per ulteriori informazioni sulla gestione del profilo, fare riferimento agli altri argomenti in Modifica e gestione del profilo .
