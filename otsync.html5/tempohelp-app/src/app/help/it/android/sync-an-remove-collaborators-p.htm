Rimozione di un collaboratore

È possibile rimuovere un collaboratore da una cartella condivisa di cui si è proprietari utilizzando l'interfaccia utente Web di OpenText Tempo Box. Questa operazione interrompe la connessione alla cartella condivisa ed elimina la cartella dalla cartella di Tempo Box del collaboratore.

È possibile accedere all'interfaccia utente Web digitando l'URL di OpenText Tempo Box nell'indirizzo di un browser Web ed effettuando l'accesso a OpenText Tempo Box. Per ulteriori informazioni sulla rimozione di collaboratori, fare riferimento alla guida online nell'interfaccia utente Web di OpenText Tempo Box.
