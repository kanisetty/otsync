Modalità per riservare e rilasciare un file

Per riservare o rilasciare un file:

  1. Toccare Tempo Box e posizionarsi sul file desiderato.

    Non è possibile riservare un file all'interno di una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

  2. Tenere premuto il file che si desidera riservare o rilasciare.

  3. Eseguire una delle seguenti operazioni:

    • Toccare Riserva per riservare il file. Il file è contrassegnato con l'overlay .

      Se si tenta di riservare un file in una cartella Enterprise per la quale si dispone dell'accesso in lettura/scrittura e l'opzione Riserva non è disponibile, ciò significa che non si dispone dell'autorizzazione Riserva per la cartella in Content Server. Contattare l'amministratore per richiedere l'autorizzazione.

    • Toccare Rilascia per rilasciare il file. Se l'opzione Rilascia non è disponibile, il file è riservato da un altro utente. Per stabilire chi ha riservato il file, è possibile verificare la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.
