Per creare un profilo:

  1. Se è la prima volta che si avvia l'applicazione OpenText Tempo Box, toccare Aggiungi profilo.

    Se si desidera creare un profilo aggiuntivo ed è già stato effettuato l'accesso a OpenText Tempo Box, eseguire le seguenti operazioni:

    1. Toccare il tasto Android Menu.

    2. Toccare .

    3. Toccare Aggiungi profilo.

  2. Toccare la casella Nome profilo e digitare un nome di profilo univoco sul dispositivo.

  3. Toccare la casella Nome utente e digitare il proprio nome utente. Se ci si trova fuori dall'azienda, digitare il proprio indirizzo e-mail.

     

    Nota:
    Un utente esterno all'azienda deve creare un account OpenText Tempo prima di poter creare un profilo.

    Per creare un account, un utente interno all'azienda deve condividere una cartella con l'utente esterno. Questa operazione genera un invito via e-mail per l'utente esterno, in cui vengono specificati i passaggi richiesti per creare un account.

     

  4. Toccare la casella Server e digitare l'URL del proprio server OpenText Tempo Box.

    Contattare l'amministratore per conoscere l'URL del server OpenText Tempo Box.

    Se ci si trova fuori dall'azienda, l' URL di OpenText Tempo Box è contenuto nel messaggio e-mail ricevuto dopo la creazione dell'account OpenText Tempo Box.

  5. Toccare Aggiungi profilo. Quando si effettua l'accesso a OpenText Tempo Box, verrà richiesto di immettere la password.

    I dati di profilo includono le credenziali dell'account, l'URL di OpenText Tempo Box e un elenco dei file preferiti. I profili sono specifici del dispositivo mobile. È necessario creare un profilo e contrassegnare file come preferiti su ogni smartphone e tablet in cui viene eseguita l'applicazione OpenText Tempo Box.
