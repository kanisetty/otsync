Spostamento o copia di un file o di una cartella

 

Per spostare o copiare un file:

  1. Toccare Tempo Box e posizionarsi sul file desiderato.

    Un file di una cartella condivisa può essere solo copiato se si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

    Inoltre, è possibile copiare un file solamente se è riservato da un altro collaboratore in una cartella condivisa per cui si dispone dell'accesso in lettura/scrittura. Le cartelle condivise in lettura/scrittura sono identificate dall'overlay . I file riservati sono contrassegnati con l'overlay . Per stabilire chi ha riservato il file, è possibile verificare la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

  2. Tenere premuto il file che si desidera spostare o copiare.

  3. Toccare Taglia per spostare il file, oppure Copia per copiarlo.

  4. Passare alla cartella di destinazione dove si desidera inserire il contenuto degli appunti.

    Non è possibile spostare o copiare un file in una cartella condivisa per la quale si dispone di accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay .

  5. Toccare il tasto AndroidMenu, quindi toccare Incolla.

Per spostare o copiare una cartella:

  1. Toccare Tempo Box e passare alla cartella desiderata.

  2. Tenere premuta la cartella che si desidera spostare o copiare.

  3. Toccare Taglia per spostare la cartella, oppure Copia per copiarla.

     

    Nota:
    Prima di continuare, fare riferimento alla sezione Note riportata di seguito.

     

  4. Passare alla cartella di destinazione dove si desidera inserire il contenuto degli appunti.

    Non è possibile spostare o copiare una cartella in una cartella condivisa per la quale si dispone di accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

  5. Toccare il tasto AndroidMenu, quindi toccare Incolla.

 

Note:

  • È possibile spostare cartelle condivise di cui si è proprietari in qualsiasi posizione nella cartella Tempo Box, ma non in un'altra cartella condivisa. Le cartelle condivise dell'utente visualizzano solamente il nome della cartella e sono identificate come .

  • È possibile copiare solo la radice di una cartella che viene condivisa con l'utente. Così facendo, viene creata una copia privata della cartella nella nuova posizione, ma la cartella condivisa originale rimane nella radice della propria cartella Tempo Box. Le cartelle che vengono condivise con l'utente aggiungono il nome utente del proprietario della cartella nel nome cartella e sono identificate come (lettura/scrittura) o (sola lettura).

  • La radice della cartella Enterprise può essere solo copiata, indipendentemente dal livello di accesso. È inoltre possibile copiare solamente sottocartelle all'interno di una cartella Enterprise per la quale si dispone dell'accesso in sola lettura. Durante tale operazione, viene creata una copia privata nella nuova posizione, ma la cartella Enterprise originale resta nella posizione originale nella cartella Tempo Box. Le cartelle Enterprise sono identificate come (lettura/scrittura) o (sola lettura).

  • Nella comunità Tempo Social è possibile copiare solo la radice di una libreria di file, identificata come (lettura/scrittura) o (sola lettura). In questo modo, viene creata una copia privata della libreria di file nella nuova posizione, ma la libreria di file originale rimane nella radice della propria cartella Tempo Box.
