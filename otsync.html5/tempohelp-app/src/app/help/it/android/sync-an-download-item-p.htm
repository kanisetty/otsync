Scaricamento di un file nel dispositivo

 

Per scaricare un file:

  1. Toccare Tempo Box e posizionarsi sul file desiderato. Se il file Ã¨ contrassegnato come preferito, toccare Preferiti.

  2. Tenere premuto il file che si desidera scaricare.

  3. Toccare Scarica.

  4. Spostarsi nella cartella di destinazione sul proprio dispositivo Android.

  5. Toccare il tasto Android Menu, quindi toccare Scarica.
