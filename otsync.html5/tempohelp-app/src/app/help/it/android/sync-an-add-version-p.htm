Aggiunta di un file

 

Per aggiungere una nuova versione utilizzando un file sul proprio dispositivo:

  1. Toccare Tempo Box e posizionarsi sul file desiderato.

     

    Importante:
    Durante l'aggiunta di una nuova versione al file in una cartella condivisa per cui si dispone di accesso in lettura/scrittura, il file non viene automaticamente riservato. È possibile aggiungere una nuova versione al file anche mentre un collaboratore sta apportando delle modifiche allo stesso. Per evitare tale situazione, è possibile riservare il file. Per ulteriori dettagli, vedere Modalità per riservare e rilasciare un file.

    Le cartelle condivise in lettura/scrittura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

     

    Non è possibile aggiungere una nuova versione a un file quando è riservato da un altro collaboratore in una cartella condivisa. I file riservati sono contrassegnati con l'overlay . Per stabilire chi ha riservato il file, è possibile verificare la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

    Inoltre, non è possibile aggiungere una nuova versione a un file in una cartella condivisa per la quale si dispone di accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay .

  2. Tenere premuto il file di cui si desidera aggiungere una nuova versione.

  3. Toccare Aggiungi versione > Da dispositivo.

  4. Passare alla nuova versione del file sul dispositivo.

  5. Toccare la nuova versione del file.

Per caricare un file memorizzato sul dispositivo:

  1. Toccare Tempo Box e posizionarsi nella cartella di destinazione in cui si desidera caricare un file.

    Non è possibile caricare un file in una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay .

    Non è possibile aggiungere un file a una cartella che dispone di categorie e attributi obbligatori tramite Tempo Box. Per determinare tali proprietà delle cartella, vedere Visualizzazione dei dettagli dell'oggetto.

  2. Tenere premuta la cartella di destinazione. Toccare Carica qui > Da dispositivo.

  3. Spostarsi nella posizione del file desiderato sul proprio dispositivo Android.

  4. Toccare il file.

Per importare un file da un'altra applicazione:

  1. Da un'altra applicazione, visualizzare un file.

  2. Toccare Esporta o Condividi.

  3. Toccare Tempo Box.

  4. Opzionale Se richiesto, effettuare l'accesso a OpenText Tempo Box.

  5. Spostarsi nella posizione in cui si desidera posizionare file.

    Non è possibile importare un file in una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay .

    Non è possibile aggiungere un file a una cartella che dispone di categorie e attributi obbligatori tramite Tempo Box. Per determinare tali proprietà della cartella, vedere Visualizzazione dei dettagli dell'oggetto.

  6. Toccare il tasto AndroidMenu, quindi toccare Incolla.
