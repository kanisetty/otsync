Apertura di un file in un'altra applicazione o profilo

Per aprire un file in un'altra applicazione:

  1. Toccare Tempo Box e posizionarsi sul file desiderato. Se il file è contrassegnato come preferito, toccare Preferiti.

  2. Tenere premuto il file che si desidera esportare.

  3. Toccare E-mail/Esporta. Se la versione più recente del file non è disponibile sul proprio dispositivo, il file viene scaricato prima dell'esportazione.

  4. Toccare un'applicazione e seguire le istruzioni sullo schermo.

Per esportare un file in un altro profilo OpenText Tempo Box:

  1. Toccare Tempo Box e posizionarsi sul file desiderato. Se il file è contrassegnato come preferito, toccare Preferiti.

  2. Tenere premuto il file che si desidera esportare.

  3. Toccare E-mail/Esporta.

  4. Toccare OpenText Tempo.

  5. Toccare il tasto AndroidIndietro finché non si ritorna alla pagina Profilo.

  6. Tenere premuto il nome profilo e toccare Uscita.

  7. Effettuare l'accesso al profilo nel quale si desidera importare il file.

  8. Toccare Tempo Box e spostarsi nella posizione dove si desidera posizionare file. Non è possibile importare un file in una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

  9. Toccare Incolla. Al termine dell'esportazione, l'accesso al secondo profilo resta attivo.

    Per tornare al profilo originale, disconnettersi dal secondo profilo toccando il tasto AndroidIndietro per tornare alla pagina Profilo.

  10. Tenere premuto il nome del secondo profilo e toccare Uscita.

  11. Nella pagina Profilo, effettuare l'accesso al profilo originale.
