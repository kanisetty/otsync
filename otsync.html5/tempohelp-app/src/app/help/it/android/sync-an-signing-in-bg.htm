Per effettuare l'accesso:

  1. Avviare l'applicazione OpenText Tempo Box. Per effettuare l'accesso alla propria cartella OpenText Tempo Box, è necessario prima creare un profilo. Per ulteriori informazioni, vedere la sezione Creazione di un profilo.

  2. Toccare il proprio nome di profilo OpenText Tempo Box.

  3. Se richiesto, toccare la casella Password e digitare la propria password.

    Se ci si trova fuori dall'azienda, digitare la password scelta quando è stato creato l'account OpenText Tempo Box. Se non si riesce a ricordare la password, fare riferimento al messaggio e-mail inviato dopo la creazione dell'account OpenText Tempo Box.

     

    Nota:
    La password è sensibile alle maiuscole/minuscole. Assicurarsi di digitare la password correttamente.

     

  4. Opzionale Toccare la casella di controllo Salva password per consentire l'accesso immediato all'avvio successivo dell'applicazione. Le uniche eccezioni a questa regola si presentano quando ci si scollega da OpenText Tempo Box o se la sessione scade. Vedere le informazioni contenute in Attenzione in basso.

     

    Nota:
    Questa casella di controllo non è disponibile se il dispositivo è offline.

     

  5. Toccare Accesso.

 

Cautela

  • Consentire a OpenText Tempo Box di salvare la propria password può rappresentare un rischio per la sicurezza, se il dispositivo dovesse essere utilizzato da un'altra persona.

  • Se sul proprio dispositivo sono presenti più profili, altri profili possono accedere al proprio account OpenText Tempo Box se la casella di controllo Salva password è selezionata e non ci si è scollegati. Per uscire, toccare il tasto Android Menu, quindi toccare .

 

Per ulteriori informazioni sulla gestione del proprio profilo, fare riferimento alla sezione Accesso e gestione del proprio profilo.
