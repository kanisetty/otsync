Modifica delle impostazioni di OpenText Tempo

 

Per modificare le impostazioni di OpenText Tempo Box:

  1. Toccare il tasto AndroidMenu, quindi toccare Impostazioni.

  2. Opzionale Toccare la riga Ordinamento predefinito e toccare Data o Nome per ordinare i file per nome file o data di modifica.

    È necessario aggiornare la visualizzazione della cartella per visualizzare la modifica. Per ulteriori dettagli, vedere Aggiornamento della visualizzazione.

  3. Opzionale Toccare la riga Memoria dispositivo e toccare un nuovo limite di memoria per i file memorizzati nella cache. Se è stato superato il limite impostato, i file meno recenti nella cache vengono eliminati finché la dimensione della cache non sarà inferiore al limite specificato. La dimensione di cache predefinita è di 1 GB.

     

    Note:

    • La riga Spazio utilizzato mostra la dimensione totale dei file preferiti, non dei file memorizzati nella cache.

    • La riga Utilizzo quote mostra la quota e le dimensioni della cartella Tempo Box. Questo esclude le cartelle condivise di cui non si è proprietario.

     

  4. Opzionale Toccare la riga Sincronizza Preferiti e toccare Auto o Manuale.

    Con la sincronizzazione automatica,

    • Nuove versioni dei file preferiti vengono scaricate in background.

    • Quando il file viene contrassegnato come preferito, viene scaricato per la visualizzazione offline.

    Con la sincronizzazione manuale,

    • Nuove versioni dei file preferiti non vengono scaricate automaticamente.

    • È necessario aprire il file o sincronizzare manualmente i preferiti per scaricare il file per la visualizzazione offline. Per sincronizzare i preferiti manualmente, eseguire le seguenti operazioni:

      1. Toccare Preferiti.

      2. Toccare Menu > Aggiorna preferiti.

  5. Toccare il tasto AndroidIndietro per tornare ai propri file.

 

Suggerimento:
Se è installata la versione 4.0 del sistema operativo Android, o una versione successiva, sarà possibile accedere alla pagina Impostazioni dalla pagina Profilo toccando .
