Comprensione delle icone di OpenText Tempo Box

 

Se è installato il sistema operativo Android versione 4.0 o successiva, nell'interfaccia principale vengono visualizzate le icone riportate di seguito. Per le versioni precedenti, tali funzionalità sono disponibili nel menu Android.

Icona Descrizione
Avviare la pagina Impostazioni dalla pagina Profilo.

Icone cartella condivisa

OpenText Tempo Box offre i seguenti tipi di cartelle condivise.

Icona Descrizione
Questa icona rappresenta una cartella condivisa per la quale si dispone dell'accesso in lettura/scrittura. Se al nome della cartella viene aggiunto un nome utente, l'utente dispone dell'accesso in condivisione alla cartella. Ad esempio, Materiale di marketing (jsmith). Per ulteriori informazioni, vedere Visualizzazione di file nelle cartelle condivise.
Un overlay di blocco rappresenta una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Per ulteriori informazioni, vedere Visualizzazione di file nelle cartelle condivise.
Questa icona rappresenta una cartella Enterprise per la quale si dispone dell'accesso in lettura/scrittura. È necessario effettuare la sottoscrizione alla cartella Enterprise in Content Server o disporre di un utente con autorizzazioni di modifica per la cartella che invia un invito alla condivisione in modo da permettere all'utente di accedere ai contenuti dalla propria cartella Tempo Box. Per ulteriori informazioni, vedere Visualizzazione di file nelle cartelle condivise.
Un'overlay di blocco su questa icona rappresenta una cartella Enterprise per la quale si dispone dell'accesso in sola lettura. È necessario effettuare la sottoscrizione alla cartella Enterprise in Content Server o disporre di un utente con autorizzazioni di modifica per la cartella che invia un invito alla condivisione in modo da permettere all'utente di accedere ai contenuti dalla propria cartella Tempo Box. Per ulteriori informazioni, vedere Visualizzazione di file nelle cartelle condivise .
Questa icona rappresenta una libreria di file in una comunità Tempo Social per la quale si dispone dell'accesso in lettura/scrittura. È necessario appartenere a una comunità Tempo Social per accedere alle librerie di file dalla propria cartella Tempo Box. Per ulteriori informazioni sulle comunità Tempo Social, vedere OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS). OpenText Tempo Social è un prodotto distinto da OpenText Tempo Box. Se Tempo Social è installato e non si visualizzano le librerie di file nella cartella Tempo Box, contattare l'amministratore.
Un overlay di blocco su questa icona rappresenta una libreria di file in una comunità Tempo Social per la quale si dispone dell'accesso in sola lettura. Per ulteriori informazioni, vedere OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).
Un overlay di blocco su un file rappresenta un file in una cartella condivisa di sola lettura.
Un overlay con segno di spunta su un file rappresenta un file riservato.
