Visualizzazione, modifica e utilizzo dei file

Per visualizzare un file:

  1. Toccare Tempo Box e posizionarsi sul file desiderato. Sulla barra del titolo di OpenText Tempo Box è visualizzato il nome della cartella visualizzata.

  2. Opzionale Per assicurarsi di consultare sempre i contenuti più recenti, aggiornare periodicamente la visualizzazione. Per ulteriori dettagli, vedere Aggiornamento della visualizzazione.

  3. Eseguire una delle seguenti operazioni:

    • Toccare il file che si desidera visualizzare. Viene visualizzato un elenco delle applicazioni che sono in grado di aprire il file. Toccare un'applicazione.

    • Tenere premuto il file che si desidera visualizzare. Toccare Apri, quindi toccare un'applicazione.

     

    Nota:
    Per impostare un'applicazione predefinita per un tipo MIME particolare, toccare la casella di controllo Usa come applicazione predefinita per questa azione prima di selezionare un'applicazione.

     

  4. Toccare il tasto Android Indietro per tornare ai propri file.

Per modificare un file:

  1. Toccare Tempo Box e posizionarsi sul file desiderato.

     

    Importante:
    Durante la visualizzazione di file in una cartella condivisa per cui si dispone di accesso in lettura/scrittura, i file non sono riservati automaticamente. È possibile aprire e modificare un file anche mentre lo sta facendo un collaboratore. Per evitare tale situazione, è possibile riservare il file. Per ulteriori dettagli, vedere Modalità per riservare e rilasciare un file.

    Le cartelle condivise in lettura/scrittura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

     

    Non è possibile salvare le modifiche apportate a un file quando è riservato da un altro collaboratore in una cartella condivisa. I file riservati sono contrassegnati con l'overlay . Per stabilire chi ha riservato il file, è possibile verificare la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

    Inoltre, non è possibile salvare le modifiche apportate a un file in una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay .

  2. Toccare il file che si desidera modificare e selezionare un editor di terze parti.

  3. Se l'applicazione supporta la modifica, modificare il file e salvare le modifiche.

    Le modifiche apportate al file non vengono salvate sul file nella cartella Tempo Box. È necessario caricare o aggiungere il file modificato.

  4. Per caricare o aggiungere il file modificato nella cartella di Tempo Box, eseguire una delle seguenti operazioni:

    • Dall'applicazione di terze parti salvare il file sul proprio dispositivo Android. Da OpenText Tempo Box, aggiungere una nuova versione al file esistente. Per ulteriori dettagli, vedere Aggiunta di un file.

    • Dall'applicazione di terze parti, caricare il file salvato nella cartella di Tempo Box utilizzando le opzioni Condividi o Esporta. Verrà aggiunto un secondo file alla cartella di Tempo Box. Per ulteriori dettagli, vedere Aggiunta di un file.

Per ulteriori informazioni sulla gestione dei file in uso, fare riferimento agli altri argomenti in Visualizzazione, modifica e utilizzo dei file.
