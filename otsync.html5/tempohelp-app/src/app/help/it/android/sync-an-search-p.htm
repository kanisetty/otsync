Ricerca della propria cartella di Tempo Box

Per cercare la propria cartella di Tempo Box:

  1. Toccare Tempo Box e posizionarsi nella cartella di destinazione dalla quale si desidera iniziare la ricerca.

  2. Eseguire una delle seguenti operazioni, in base al sistema operativo Android:

    • Toccare il tasto AndroidMenu, quindi toccare Cerca.

    • Toccare il tasto Android Cerca.

  3. Nella casella di ricerca, digitare la stringa di ricerca.

  4. Toccare Vai. Viene visualizzato un elenco di risultati di ricerca.

  5. Eseguire una delle seguenti operazioni:

    • Toccare il file o la cartella che si desidera visualizzare.

    • Tenere premuto il file o la cartella per visualizzare le opzioni disponibili.

Per aprire il percorso di un file dai risultati della ricerca:

  1. Cercare la propria cartella Tempo Box.

  2. Tenere premuto il file o la cartella che si desidera aprire nel percorso del file.

  3. Toccare Apri percorso.
