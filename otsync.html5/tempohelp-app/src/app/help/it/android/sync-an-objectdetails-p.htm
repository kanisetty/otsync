Visualizzazione dei dettagli dell'oggetto

Per visualizzare i dettagli di un oggetto:

  1. Toccare Tempo Box e passare al file o alla cartella di destinazione.

  2. Tenere premuto il file o la cartella.

  3. Toccare Dettagli oggetto.

  4. Eseguire una delle seguenti operazioni:

    • Toccare Propriet√† per visualizzare i dettagli relativi all'oggetto.

    • Toccare Categorie per visualizzare le categorie e gli attributi assegnati all'oggetto.

    • Toccare Eventi per visualizzare la cronologia eventi dell'oggetto.

  5. Toccare il tasto AndroidIndietro per tornare ai propri file.
