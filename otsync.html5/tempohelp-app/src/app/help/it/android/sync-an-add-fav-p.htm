Aggiunta, visualizzazione, rimozione e sincronizzazione dei preferiti

 

Per aggiungere un preferito:

  1. Toccare Tempo Box e posizionarsi sul file desiderato. È possibile contrassegnare come preferiti sia file personali che condivisi.

  2. Tenere premuto il file che si desidera contrassegnare come preferito.

  3. Toccare Aggiungi a Preferiti.

    Il file è contrassegnato con l'overlay . Ad esempio, .

  4. Opzionale Se viene visualizzato l'overlay il file preferito non è stato scaricato. Ad esempio, . Per scaricare il file, sincronizzare i preferiti o aprire il file. Per ulteriori dettagli, vedere Sincronizzazione manuale dei Preferiti.

    Per sincronizzare i file preferiti automaticamente, vedere Modifica delle impostazioni di OpenText Tempo.

    Nel proprio profilo è memorizzato l'elenco dei propri file preferiti. I profili sono specifici del dispositivo. È necessario creare un profilo e contrassegnare file come preferiti su ogni smartphone e tablet che esegue l'applicazione OpenText Tempo Box.

Per visualizzare un preferito:

  1. Toccare Preferiti.

  2. Opzionale Per assicurarsi di consultare sempre i contenuti più recenti, aggiornare periodicamente la visualizzazione. Per ulteriori dettagli, vedere Aggiornamento della visualizzazione.

    Questa operazione non consente di scaricare i file per la visualizzazione offline. Per scaricare il file, sincronizzare i preferiti o aprire il file. Per ulteriori dettagli, vedere Sincronizzazione manuale dei Preferiti di seguito.

Per aprire il percorso di un file preferito:

  1. Toccare Preferiti.

  2. Tenere premuto il file di cui si desidera aprire il percorso.

  3. Toccare Apri percorso.

Per rimuovere un preferito:

  1. Toccare Preferiti, o toccare Tempo Box e passare al file di destinazione.

  2. Tenere premuto il file che si desidera rimuovere dall'elenco dei preferiti.

  3. Toccare Rimuovi dai Preferiti. Il file non viene più visualizzato nell'elenco dei preferiti. Questa operazione non elimina il file dalla cartella Tempo Box.

Per sincronizzare manualmente i preferiti:

  1. Toccare Preferiti.

  2. Toccare Menu > Aggiorna preferiti. Se l'applicazione sta sincronizzando i preferiti, o ci si trova offline, l'opzione Aggiorna preferiti non è disponibile.

     

    Nota:
    Toccando Aggiorna viene aggiornata la visualizzazione della pagina Preferiti. Questa operazione non consente di scaricare file preferiti sul dispositivo in uso per la visualizzazione offline. Per scaricare il file, è necessario sincronizzare i file preferiti o aprire il file.

     

    Per sincronizzare i file preferiti automaticamente, vedere Modifica delle impostazioni di OpenText Tempo.
