Visualizzazione, modifica e lavoro sui file

 

Per visualizzare un file:

  1. Toccare Tempo Box e posizionarsi sul file desiderato. Assicurarsi di toccare il nome della cartella e non , che visualizza i dettagli del file o della cartella. Sulla barra del titolo della cartella è visualizzato il nome della cartella visualizzata.

  2. Opzionale Toccare per assicurarsi di visualizzare i contenuti più recenti. Questa operazione consente di aggiornare la visualizzazione, ma non di scaricare i file. Per scaricare un file, aprirlo.

  3. Toccare il nome del file che si desidera aprire.

  4. Toccare Indietro per tornare ai file.

Per modificare un file:

  1. Toccare Tempo Box e posizionarsi sul file desiderato.

     

    Importante:
    Durante la visualizzazione di file in una cartella condivisa per cui si dispone di accesso in lettura/scrittura, i file non sono riservati automaticamente. È possibile aprire e modificare un file anche mentre lo sta facendo un collaboratore. Per evitare tale situazione, è possibile riservare il file. Per ulteriori dettagli, vedere Modalità per riservare e rilasciare un file.

    Le cartelle condivise in lettura/scrittura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

     

    Non è possibile salvare le modifiche apportate a un file quando è riservato da un altro collaboratore in una cartella condivisa. I file riservati sono contrassegnati con l'overlay . Per stabilire chi ha riservato il file, è possibile verificare la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

    Inoltre, non è possibile salvare le modifiche apportate a un file in una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay .

  2. Toccare il file che si desidera modificare.

  3. Toccare e selezionare un editor.

  4. Modificare il file e salvare le modifiche.

  5. Nell'editor, toccare e selezionare Tempo Box.

  6. In OpenText Tempo Box toccare Carica nuova versione. Le modifiche vengono aggiunte al file originale.

     

    Nota:
    Se si utilizza la funzionalità di ricerca nell'applicazione OpenText Tempo Box per trovare il file da modificare, tutte le modifiche apportate al file in un editor verranno salvate in un nuovo file e inserite a livello di radice.

     

  7. Toccare per tornare ai file.

Per ulteriori informazioni sulla gestione dei file in uso, fare riferimento agli altri argomenti in Visualizzazione, modifica e lavoro sui file.
