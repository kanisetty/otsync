Eliminazione di un file o di una cartella

 

Per eliminare un file:

  1. Toccare Tempo Box e posizionarsi sul file desiderato. Se il file è contrassegnato come preferito, toccare Preferiti.

    Non è possibile eliminare un file da una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

    Non è possibile eliminare un file se risulta riservato. I file riservati sono contrassegnati con l'overlay . Per eliminare un file riservato, è innanzitutto necessario rilasciarlo. Per ulteriori dettagli, vedere Modalità per riservare e rilasciare un file.

  2. Scorrere orizzontalmente con il dito sul nome del file. Viene visualizzato un pulsante Elimina nella riga del file.

     

    Nota:
    Prima di continuare, fare riferimento alla sezione Attenzione riportata di seguito.

     

  3. Toccare Elimina.

  4. Toccare Elimina questo elemento per confermare questa azione.

Per eliminare un file mentre lo si sta visualizzando:

  1. Toccare Tempo Box e posizionarsi sul file desiderato.

    Non è possibile eliminare un file da una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

    Non è possibile eliminare un file se risulta riservato. I file riservati sono contrassegnati con l'overlay . Per eliminare un file riservato, è innanzitutto necessario rilasciarlo. Per ulteriori dettagli, vedere Modalità per riservare e rilasciare un file.

  2. Toccare il file che si desidera eliminare. Il file viene aperto.

     

    Nota:
    Prima di continuare, fare riferimento alla sezione Attenzione riportata di seguito.

     

  3. Toccare .

  4. Toccare Elimina questo elemento per confermare questa azione.

Per eliminare una cartella:

  1. Toccare Tempo Box e posizionarsi sulla cartella desiderata.

    Se la cartella di destinazione è una cartella condivisa, vedere Eliminazione di una cartella condivisa prima di continuare. Le cartelle condivise sono identificate dall'overlay (accesso in lettura/scrittura) o dall'overlay (accesso in sola lettura). Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

    Non è possibile eliminare una cartella se contiene un file riservato. Per rilasciare il file, vedere Modalità per riservare e rilasciare un file.

  2. Scorrere orizzontalmente con il dito sul nome della cartella. Viene visualizzato il pulsante Elimina nella riga della cartella.

     

    Nota:
    Prima di continuare, fare riferimento alla sezione Attenzione riportata di seguito.

     

  3. Toccare Elimina.

  4. Toccare Elimina questo elemento per confermare questa azione.

 

Cautela

  • Eliminando un file o una cartella, lo si rimuove dalla cartella Tempo Box, anche se il file era stato contrassegnato come preferito. Ciò significa che il file o la cartella è stato eliminato da tutti gli altri client OpenText Tempo Box installati come client desktop, dagli altri smartphone o tablet, nonché dall'interfaccia utente Web OpenText Tempo Box.

  • Se si elimina una file o una cartella da una cartella condivisa per la quale si dispone di accesso in lettura/scrittura, l'elemento in questione viene automaticamente eliminato per tutti gli utenti che hanno accesso alla cartella condivisa, incluso il proprietario della cartella condivisa. Le cartelle condivise in lettura/scrittura sono identificate dall'overlay .

  • Se si elimina un file o una cartella da una cartella Enterprise per la quale si dispone di accesso in lettura/scrittura, il file o la cartella vengono eliminati automaticamente dalla cartella nell'Area di lavoro organizzazione.
