Per modificare il proprio profilo:

  1. Toccare , quindi toccare Scambia profili.

  2. Nella schermata OpenText Tempo Box Profilo aggiungere un nuovo profilo utilizzando il nuovo nome utente o l'URL di OpenText Tempo Box e scegliendo un nome di profilo univoco. Per ulteriori dettagli, vedere la sezione Creazione di un profilo.

  3. Accedere a OpenText Tempo Box utilizzando il nuovo profilo. A questo punto è possibile accedere alla cartella OpenText Tempo Box utilizzando il nuovo profilo. Tuttavia, l'elenco dei preferiti del profilo originale non è disponibile nel nuovo profilo.

    Per mantenere tale elenco è necessario modificare il profilo originale come indicato di seguito:

    1. Toccare .

    2. Toccare il nome del profilo che si desidera modificare.

    3. Modificare il nome utente o il server come richiesto.

    4. Toccare la casella Password e digitare la password per questo utente.

    5. Toccare Fatto.

    6. Toccare Esci.

    7. Nella schermata Profilo di OpenText Tempo Box, accedere a OpenText Tempo Box utilizzando il proprio profilo originario.

      A questo punto è possibile eliminare il secondo profilo creato in passaggio 2. Per ulteriori dettagli, vedere la sezione Eliminazione del proprio profilo.

Per modificare la propria password:

  1. Toccare .

  2. Toccare Esci.

  3. Toccare il proprio profilo.

  4. Toccare la casella Password e digitare la nuova password.
