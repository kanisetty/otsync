Per creare un profilo:

  1. Se è la prima volta che si avvia l'applicazione OpenText Tempo Box, toccare Aggiungi profilo.

    Se si desidera creare un profilo aggiuntivo ed è già stato effettuato l'accesso a OpenText Tempo Box, eseguire le seguenti operazioni:

    1. Toccare .

    2. Toccare Scambia profilo.

    3. Toccare Aggiungi profilo.

  2. Toccare la casella Nome utente e digitare il proprio nome utente. Se ci si trova fuori dall'azienda, digitare il proprio indirizzo e-mail.

     

    Nota:
    Un utente esterno all'azienda deve creare un account OpenText Tempo Box prima di poter creare un profilo.

    Per creare un account, un utente interno all'azienda deve condividere una cartella con l'utente esterno. Questa operazione genera un invito via e-mail per l'utente esterno, in cui vengono specificati i passaggi richiesti per creare un account.

     

  3. Toccare la casella Password e digitare la propria password.

    Se ci si trova fuori dall'azienda, digitare la password scelta quando è stato creato l'account OpenText Tempo Box. Se non si riesce a ricordare la password, fare riferimento al messaggio e-mail inviato dopo la creazione dell'account OpenText Tempo Box.

     

    Nota:
    La password è sensibile alle maiuscole/minuscole. Assicurarsi di digitare la password correttamente.

     

  4. Toccare la casella Server e digitare l'URL del proprio server OpenText Tempo Box.

    Se non si riesce a ricordare l'URL di OpenText Tempo Box, contattare l'amministratore.

    Se ci si trova fuori dall'azienda, l' URL di OpenText Tempo Box è contenuto nel messaggio e-mail ricevuto dopo la creazione dell'account OpenText Tempo Box.

  5. Toccare la casella Nome profilo e digitare un nome di profilo univoco sul dispositivo.

  6. Opzionale Toccare Ricorda password per consentire l'accesso immediato all'avvio successivo dell'applicazione. Le uniche eccezioni a questa regola si presentano quando ci si scollega da OpenText Tempo Box o se la sessione scade. Vedere le informazioni contenute in Attenzione in basso.

  7. Toccare Fatto. I dati di profilo includono le credenziali dell'account, l'URL di OpenText Tempo Box e un elenco dei file preferiti. I profili sono specifici del dispositivo iPhone e iPad. È necessario creare un profilo e contrassegnare file come preferiti su ogni smartphone e tablet in cui viene eseguita l'applicazione OpenText Tempo Box.

 

Cautela

  • Consentire a OpenText Tempo Box di ricordare la propria password può rappresentare un rischio per la sicurezza, se il dispositivo dovesse essere utilizzato da un'altra persona.

  • Se sul dispositivo sono configurati più profili, quando la casella di controllo Ricorda password è selezionata e l'utente non si è scollegato, altri profili possono accedere all'account OpenText Tempo Box dell'utente. Per uscire, toccare , quindi toccare Esci.
