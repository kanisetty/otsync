Per effettuare l'accesso:

  1. Avviare l'applicazione OpenText Tempo Box.

  2. Per effettuare l'accesso alla propria cartella OpenText Tempo Box, è necessario prima creare un profilo. Per ulteriori informazioni, vedere la sezione Creazione di un profilo.

  3. Toccare il proprio nome di profilo OpenText Tempo Box.

  4. Se richiesto, toccare la casella Password e digitare la propria password.

    Se ci si trova fuori dall'azienda, digitare la password scelta quando è stato creato l'account OpenText Tempo Box. Se non si riesce a ricordare la password, fare riferimento al messaggio e-mail inviato dopo la creazione dell'account OpenText Tempo Box.

     

    Nota:
    La password è sensibile alle maiuscole/minuscole. Assicurarsi di digitare la password correttamente.

     

  5. Opzionale Toccare Ricorda password per consentire l'accesso immediato all'avvio successivo dell'applicazione. Le uniche eccezioni a questa regola si presentano quando ci si scollega da OpenText Tempo Box o se la sessione scade. Vedere le informazioni contenute in Attenzione in basso.

  6. Toccare Fatto.

 

Cautela

  • Consentire a OpenText Tempo Box di ricordare la propria password può rappresentare un rischio per la sicurezza, se il dispositivo dovesse essere utilizzato da un'altra persona.

  • Se sul dispositivo sono configurati più profili, quando la casella di controllo Ricorda password è selezionata e l'utente non si è scollegato, altri profili possono accedere all'account OpenText Tempo Box dell'utente. Per uscire, toccare , quindi toccare Esci.

 

Per ulteriori informazioni sulla gestione del proprio profilo, fare riferimento alla sezione Accesso e gestione del proprio profilo.
