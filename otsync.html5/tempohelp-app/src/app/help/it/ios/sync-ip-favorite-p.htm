Aggiunta, visualizzazione e rimozione di preferiti

 

Per aggiungere un preferito:

  1. Toccare Tempo Box e posizionarsi sul file desiderato. È possibile contrassegnare come preferiti sia file personali che condivisi.

  2. Toccare il file che si desidera contrassegnare come preferito. Il file viene aperto.

  3. Toccare . L'icona diventa e il file viene contrassegnato con l'immagine sovrapposta nella vista cartelle. Ad esempio, . Quando il file viene contrassegnato come preferito, è possibile visualizzarlo anche offline.

    Nel proprio profilo è memorizzato l'elenco dei file preferiti, specifico per iPad o iPhone. È necessario contrassegnare file come preferiti su ogni smartphone e tablet in cui viene eseguita l'applicazione OpenText Tempo.

  4. Toccare Indietro per tornare ai file.

     

    Suggerimento:
    È possibile contrassegnare i file come preferiti anche toccando per il file, quindi toccando Aggiungi a Preferiti.

     

Per visualizzare un preferito:

  1. Toccare Preferiti.

  2. Opzionale Toccare per assicurarsi di visualizzare i contenuti più recenti.

  3. Toccare il file che si desidera visualizzare.

  4. Toccare Indietro per tornare ai file.

Per rimuovere un preferito:

  1. Toccare Preferiti o Tempo Box e raggiungere il file desiderato.

  2. Toccare il file che si desidera rimuovere dall'elenco dei preferiti. Il file viene aperto.

  3. Toccare . L'icona diventa quindi . Il file non viene più visualizzato nell'elenco dei preferiti.

    Questa operazione non elimina il file dalla cartella Tempo Box.

  4. Toccare Indietro per tornare ai file.

     

    Suggerimento:
    È anche possibile rimuovere il file dall'elenco dei preferiti toccando per il file, quindi toccando Rimuovi Preferito.
