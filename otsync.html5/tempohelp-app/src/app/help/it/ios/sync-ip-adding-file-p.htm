Aggiunta di un file

Per aggiungere un file:

  1. È necessario importare il file da un'altra applicazione. Trovare il file da importare e toccarlo e/o Apri in.

  2. Toccare Apri in Tempo.

  3. Eseguire una delle seguenti operazioni:

    • Per aggiungere il file alla radice della cartella Tempo Box, toccare Carica.

    • Per scegliere una posizione diversa dalla radice della cartella Tempo Box, toccare Destinazione: Area di lavoro personale

      1. Toccare OpenText Tempo Box. Compare la radice della cartella Tempo Box.

      2. Raggiungere la cartella destinazione.

        Non è possibile aggiungere un file a una cartella condivisa per la quale si dispone dell'accesso in sola lettura. Le cartelle condivise in sola lettura sono identificate dall'overlay . Per ulteriori dettagli, vedere Comprensione delle icone di OpenText Tempo Box.

        Con Tempo Box, non si possono aggiungere file a una cartella con categorie e attributi obbligatori. Per determinare tali proprietà della cartella, vedere Visualizzazione dei dettagli dell'oggetto.

      3. Per selezionare la cartella di destinazione, toccare Scegli.

        Per aggiungere una nuova cartella, fare clic su Nuova cartella. Digitare il nome della nuova cartella, quindi selezionare Fatto. Per selezionare la nuova cartella, toccare Scegli.

      4. Fare clic su Carica.

  4. Opzionale In OpenText Tempo, toccare il file importato. Per spostare il file in un'altra posizione, vedere Spostamento o copia di un file o di una cartella.
