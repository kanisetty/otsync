Download di tutti i file in una cartella

 

Per scaricare tutti i file in una cartella:

  1. Toccare Tempo Box e posizionarsi sulla cartella desiderata.

  2. Toccare .

  3. Toccare Scarica contenuti cartella. I contenuti della cartella non devono superare la dimensione massima della cache di 2 GB. Se il dispositivo non possiede abbastanza spazio per scaricare i contenuti della cartella, il download non inizia.

  4. Toccare Indietro per tornare ai file.
