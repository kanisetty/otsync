要创建配置文件：

  1. 如果这是您第一次启动 OpenText Tempo Box 应用程序，点击[添加配置文件]。

    如果要创建其他配置文件，而且您已经登录到了 OpenText Tempo Box，请执行以下操作：

    1. 点击 。

    2. 点击[切换配置文件]。

    3. 点击[添加配置文件]。

  2. 点击[用户名]框，然后键入您的用户名。如果您是公司外部人员，请键入电子邮件地址。

     

    注意:
    公司外面的用户创建配置文件之前，必须先创建 OpenText Tempo Box 帐户。

    要创建帐户，公司内部的用户必须与外部用户共享文件夹。这会向外部用户发送电子邮件邀请，概述创建帐户所需的步骤。

     

  3. 点击[密码]框，然后键入您的密码。

    如果您是公司外部人员，请键入您在创建 OpenText Tempo Box 帐户时选择的密码。如果您不记得密码，请参阅创建 OpenText Tempo Box 帐户时发送给您的电子邮件。

     

    注意:
    密码区分大小写。请确保您键入的密码正确。

     

  4. 点击[服务器]框，然后键入您的 OpenText Tempo Box 服务器的 URL。

    如果您不记得 OpenText Tempo Box URL，请与管理员联系。

    如果您是公司外部人员，则 OpenText Tempo Box URL 列在您创建 OpenText Tempo Box 帐户时发送给您的电子邮件中。

  5. 点击[配置文件名称]框，然后键入此设备上的唯一配置文件名称。

  6. 可选 点击[记住密码]，以便在下次启动应用程序时立即进行访问。唯一的例外是如果您从 OpenText Tempo Box 注销或会话超时。请参阅下方的注意信息。

  7. 点击[完成]。配置文件数据包括帐户凭据、OpenText Tempo Box URL 和收藏文件列表。配置文件是 iPhone 和 iPad 所特有的。因此必须在运行 OpenText Tempo Box 应用程序的每个智能手机和平板电脑上创建配置文件并将文件标记为收藏。

 

小心

  • 允许 OpenText Tempo Box 记住密码可能会危及安全性，导致您的设备被别人使用。

  • 如果在设备上配置了多个配置文件，若已选中[记住密码]且未注销，则其他配置文件可以访问您的 OpenText Tempo Box 帐户。要注销，请点击 ，然后点击[注销]。
