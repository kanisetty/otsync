要编辑配置文件：

  1. 点击 ，然后点击[切换配置文件]。

  2. 在 OpenText Tempo Box [配置文件]屏幕上，使用新用户名或 OpenText Tempo Box URL 添加新配置文件，并选择唯一的配置文件名称。有关详细信息，请参阅创建配置文件。

  3. 使用新配置文件登录 OpenText Tempo Box。您现在可以使用新配置文件访问 OpenText Tempo Box 文件夹。但是，来自原始配置文件的收藏列表在新配置文件中不可用。

    要保留收藏列表，您必须按照以下步骤编辑原始配置文件：

    1. 点击 。

    2. 点击要编辑的配置文件名称。

    3. 根据需要更改用户名或服务器。

    4. 点击[密码]框，然后键入此用户的密码。

    5. 点击[完成]。

    6. 点击[注销]。

    7. 在 OpenText Tempo Box [配置文件]屏幕上，使用原始配置文件登录到 OpenText Tempo Box。

      现在，您可以删除 步骤 2 中创建的第二个配置文件。有关详细信息，请参阅删除配置文件。

要编辑密码：

  1. 点击 。

  2. 点击[注销]。

  3. 点击您的配置文件。

  4. 点击[密码]框，然后键入新密码。
