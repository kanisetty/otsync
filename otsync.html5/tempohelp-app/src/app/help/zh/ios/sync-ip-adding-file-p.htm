添加文件

要添加文件：

  1. 您必须从另一个应用程序导入该文件。查看您希望导入的文件，然后点击 和/或[打开方式]。

  2. 点击 [在 Tempo 中打开]。

  3. 执行以下操作之一：

    • 要将文件添加到您的 Tempo Box 文件夹的根目录，请点击[上载]。

    • 要选择您的 Tempo Box 文件夹之外的位置，请点击 [目标：个人工作区]

      1. 点击 [OpenText Tempo Box]。此时出现您的 Tempo Box 文件夹的根目录。

      2. 导航至目标文件夹。

        您无法在具有只读权限时，添加文件至共享文件夹中。只读共享文件夹由 覆盖表示。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

        您无法通过 Tempo Box 在具有强制类别和属性的文件夹中添加文件。要确定这一情况，请参阅 查看对象详细信息。

      3. 要选择目标文件夹，请点击[选择]。

        要添加新文件夹，请单击[新建文件夹]。键入新文件夹的名称，然后点击[完成]。点击[选择]以选择新文件夹。

      4. 单击[上载]。

  4. 可选 在 OpenText Tempo 中，点击已导入的文件。要将文件移至其他位置，请参阅 移动或复制文件或文件夹。
