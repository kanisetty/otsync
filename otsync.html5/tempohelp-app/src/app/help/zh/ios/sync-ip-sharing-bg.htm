第4章. 共享文件夹和管理共享文件夹

 

要共享文件夹：

  1. 点击 [Tempo Box]，然后导航到目标文件夹。共享文件夹时，文件夹可位于 Tempo Box 文件夹中的任何位置。要添加新文件夹以共享，请参阅 添加图片或文件夹。

  2. 点击 .

  3. 点击[共享]。如果[共享]按钮不可用，则您没有共享此文件夹的权限。有关不能共享的文件夹列表，请参阅文件夹共享例外 与共享企业文件夹 。

  4. 可选 共享文件夹时，您可以授予协作者读/写或只读文件夹的权限。默认情况下，将授予协作者读/写权限，由 标识。

    要授予协作者只读权限，请执行以下操作：

    1. 点击 .

    2. 点击[只读]。图标更改为 .

    3. 点击[完成]。

    具有读/写权限时，包括外部用户在内的所有协作者将拥有对共享文件夹内容的完全控制权限。他们可以添加、编辑、复制、移动和删除文件以及文件夹。一旦对共享文件夹进行修订，更改会自动与包括所有者在内对该文件夹拥有共享访问权限的所有协作者同步。

    具有只读权限时，协作者只能读取和复制共享文件夹中的内容。

  5. 点击[名称]框，然后键入要与之共享此文件夹的人的名称。

    要与外部用户共享文件夹，请键入外部用户的电子邮件地址。外部用户不是您公司的员工。键入有效的电子邮件地址后，自动完成会显示外部用户的电子邮件地址。

     

    注意:
    如果电子邮件地址被忽略且[共享人员]区域仍然为空，则您没有与外部用户共享文件夹的权限。有关详细信息，请参阅 与外部用户共享企业文件夹。

     

  6. 点击该用户的名称或电子邮件地址。共享邀请将会发出。协作者必须在访问文件夹内容之前接受共享邀请。

  7. 在打开的对话框中，点击[是]以与其他人共享此文件夹，然后重复步骤 4 - 6 或点击[否]以返回到[文件夹详细信息]页面。

  8. 点击[返回]返回到文件。

    文件夹现在被标记为共享文件夹。无论您授予协作者读/写权限还是只读权限，对您来说共享文件夹都将显示为 。

文件夹共享例外

可以共享 Tempo Box 文件夹中的任何文件夹，但以下情况除外：

  • 根 [Tempo Box] 文件夹。

  • 共享文件夹的子文件夹或父文件夹。

  • 与您共享的文件夹。

  • 您具有只读权限的企业文件夹由 确定。如果您具有企业文件夹的读/写权限（其由 确定），则您可以共享该文件夹（如果您编辑权限 Content Server 中的文件夹的权限。）

  • 您是否具有读/写或只读权限的社区文件库。社区文件库由 和 标识。

  • 外部用户无法共享任何文件夹。

共享企业文件夹

您可以共享企业文件夹，如果您具有编辑权限 Content Server 中的文件夹的权限。共享文件夹时，您可以授予该文件夹的读/写或只读权限，该操作则在 Content Server 中授予以下文件权限级别。

表 4-1: Tempo Box 中的文件夹权限和访问

Tempo Box 中的权限 Content Server 中的文件夹权限
读/写权限 修改文件夹内容的权限，其中包括所有权限，编辑权限 的所有子文件夹。
只读权限 查看 与查看内容 的所有子文件夹。

如果您尝试授予拥有高于查看 与查看内容权限的用户只读权限，则用户将会以当前访问级别添加到该共享。如果您从企业文件夹删除协作者，或者他们从 Tempo Box 文件夹删除该文件夹，则授予的访问级别仍然保持在 Content Server 中。

与外部用户共享企业文件夹

要与外部用户共享企业文件夹，您必须具有与没有 OpenText Tempo Box 用户帐户的用户共享文件夹的能力。请与管理员联系并请求此特权。您还必须具有编辑权限 Content Server 中的文件夹的权限。

如果同时满足这两个条件，则您必须在共享文件夹时提供用户的电子邮件地址。之后，外部用户会收到一封电子邮件共享邀请，邀请他们注册 OpenText Tempo Box 帐户。注册后，外部用户可以像常规用户一样安装客户端并访问 OpenText Tempo Box，但以下情况除外：

  • 外部用户可以接受来自其他用户的共享邀请，但他们不能与他人共享文件夹。

  • 外部用户不能发布项目或以其他方式访问 Content Server。

有关管理共享文件夹的信息，请参考其他主题：共享文件夹和管理共享文件夹.
