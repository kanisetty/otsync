第2章. 查看、编辑和处理文件

 

要查看文件：

  1. 点击 [Tempo Box]，然后导航到目标文件。OpenText Tempo Box 标题栏显示当前正在查看的文件夹名称。

  2. 可选 要确保正在查看的是最新内容，您应该定期刷新显示。有关详细信息，请参阅 刷新显示。

  3. 执行以下操作之一：

    • 点击要查看的文件。将会显示可以打开文件的应用程序列表。点击所需的应用程序。

    • 轻触并按住要查看的文件。点击[打开]，然后点击某个应用程序。

     

    注意:
    要将默认的应用程序设置成特定的 MIME 类型，先点击[默认情况下用于此操作]复选框，然后再选择应用程序。

     

  4. 点击 Android [返回]键返回到文件。

要编辑文件：

  1. 点击 [Tempo Box]，然后导航到目标文件。

     

    重要:
    当您在具有读/写权限的共享文件夹中查看文件时，不会自动保留这些文件。您可以在协作者正在对文件进行更改的同时打开并编辑文档。要避免这种情况，您可以保留该文件。有关详细信息，请参阅 保留和取消保留文件。

    读/写共享文件夹由 覆盖标记。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

     

    由另一个协作者在共享文件夹中进行保留时，您无法保存对文件的编辑。保留文件标有 覆盖标记。要确定谁保留了文件，您可以检查文件的审核历史记录。有关详细信息，请参阅 查看对象详细信息。

    此外，您无法在具有只读权限时，在共享文件夹中保存编辑文件。只读共享文件夹由 覆盖标记。

  2. 点击您要编辑的文件并选择第三方编辑器。

  3. 如果该应用程序支持编辑，则编辑文件并保存更改。

    对文件所做的更改不会保存到您的 Tempo Box 文件夹中的文件。您需要上载或添加更改的文件。

  4. 要将已更改的文件上载或添加到您 Tempo Box 文件夹，请执行以下操作之一：

    • 从第三方应用程序中将文件保存到 Android 设备。在 OpenText Tempo Box 中，将新版本添加到现有文件。有关详细信息，请参阅 添加文件。

    • 通过使用[共享]或[导出]选项，从第三方应用程序中将已保存的文件上载到 Tempo Box 中。此操作会将第二个文件添加到您的 Tempo Box 文件夹。有关详细信息，请参阅 添加文件。

有关管理文件的信息，请参考其他主题：查看、编辑和处理文件。
