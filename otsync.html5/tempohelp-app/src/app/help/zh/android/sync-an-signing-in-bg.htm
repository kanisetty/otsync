要登录：

  1. 启动 OpenText Tempo Box 应用程序。要访问 OpenText Tempo Box 文件夹，您必须首先在登录之前创建配置文件。有关详细信息，请参阅创建配置文件。

  2. 点击您的 OpenText Tempo Box 配置文件名称。

  3. 如果出现提示，点击[密码]框，然后键入您的密码。

    如果您是公司外部人员，请键入您在创建 OpenText Tempo Box 帐户时选择的密码。如果您不记得密码，请参阅创建 OpenText Tempo Box 帐户时发送给您的电子邮件。

     

    注意:
    密码区分大小写。请确保您键入的密码正确。

     

  4. 可选 点击[保存密码]复选框，以便在下次启动应用程序时立即进行访问。唯一的例外是如果您从 OpenText Tempo Box 注销或会话超时。请参阅下方的注意信息。

     

    注意:
    如果设备离线，则此复选框不可用。

     

  5. 点击[登录]。

 

小心

  • 允许 OpenText Tempo Box 保存密码可能会危及安全性，导致您的设备被别人使用。

  • 如果在设备上配置了多个配置文件，若已选中[保存密码]复选框且未注销，则其他配置文件可以访问 OpenText Tempo Box 帐户。要注销，请点击 Android [菜单]键，然后点击 。

 

有关管理配置文件的信息，请参阅登录和管理配置文件。
