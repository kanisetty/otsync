保留和取消保留文件

要保留或撤消保留文件：

  1. 点击 [Tempo Box]，然后导航到目标文件。

    您无法在具有只读权限的共享文件夹中保留文件。只读共享文件夹由 覆盖标记。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

  2. 轻触并按住要保留或撤消保留的文件。

  3. 执行以下操作之一：

    • 点击[保留]以保留该文件。文件使用 覆盖标记。

      如果您尝试在您具有读/写权限的企业文件夹中保留文件，则[保留]选项不可用，您没有保留 Content Server 中的文件夹的权限。请与管理员联系以请求此权限。

    • 点击[撤消保留]以撤消保留该文件。如果[取消保留]不可用，则文件由另一个用户保留。要确定谁保留了文件，您可以检查文件的审核历史记录。有关详细信息，请参阅 查看对象详细信息。
