要创建配置文件：

  1. 如果这是您第一次启动 OpenText Tempo Box 应用程序，点击[添加配置文件]。

    如果要创建其他配置文件，而且您已经登录到了 OpenText Tempo Box，请执行以下操作：

    1. 点击 Android [菜单]键。

    2. 点击 。

    3. 点击[添加配置文件]。

  2. 点击[配置文件名称]框，然后键入此设备上的唯一配置文件名称。

  3. 点击[用户名]框，然后键入您的用户名。如果您是公司外部人员，请键入电子邮件地址。

     

    注意:
    公司外面的用户创建配置文件之前，必须先创建 OpenText Tempo 帐户。

    要创建帐户，公司内部的用户必须与外部用户共享文件夹。这会向外部用户发送电子邮件邀请，概述创建帐户所需的步骤。

     

  4. 点击[服务器]框，然后键入您的 OpenText Tempo Box 服务器的 URL。

    请与管理员联系以获取 OpenText Tempo Box 服务器的 URL。

    如果您是公司外部人员，则 OpenText Tempo Box URL 列在您创建 OpenText Tempo Box 帐户时发送给您的电子邮件中。

  5. 点击[添加配置文件]。登录到 OpenText Tempo Box 时，将会提示您输入密码。

    配置文件数据包括帐户凭据、OpenText Tempo Box URL 和收藏文件列表。配置文件是移动设备所特有的。因此必须在运行 OpenText Tempo Box 应用程序的每个智能手机和平板电脑上创建配置文件并将文件标记为收藏。
