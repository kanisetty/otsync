删除文件或文件夹

要删除文件：

  1. 点击 [Tempo Box]，然后导航到目标文件。

    您无法在具有只读权限的共享文件夹中删除文件。只读共享文件夹由 覆盖标记。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

    文件进行保留时您无法删除。保留文件标有 覆盖标记。要删除已经保留的文件，您必须先取消保留该文件。有关详细信息，请参阅 保留和取消保留文件。

     

    注意:
    继续操作之前，请参阅下方的注意 信息。

     

  2. 轻触并按住要删除的文件。

  3. 点击[删除]。

  4. 点击[确定]以确认此操作。

要删除文件夹：

  1. 点击 [Tempo Box]，然后导航到目标文件夹。

    如果目标文件夹是共享文件夹，请参阅 删除共享文件夹。共享文件夹由 覆盖（读/写权限）或 覆盖（只读权限）标识。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

    如果文件夹包含保留的文件，则您无法删除。要取消保留文件，请参阅 保留和取消保留文件。

     

    注意:
    继续操作之前，请参阅下方的注意 信息。

     

  2. 轻触并按住要删除的文件夹。

  3. 点击[删除]。

  4. 点击[确定]以确认此操作。

 

小心

  • 即使文件标记为收藏，删除该文件或文件夹也会将其从 Tempo Box 文件夹移除。这意味着该文件或文件夹会从您安装的所有其他 OpenText Tempo Box 客户端删除，例如桌面客户端、其他智能手机或平板电脑以及 OpenText Tempo Box Web 用户界面。

  • 如果从您具有读/写权限的共享文件夹删除文件或文件夹，将自动对所有具有访问共享文件夹权限的用户（包括共享文件夹的所有者）删除文件或文件夹。读/写共享文件夹由 覆盖标记。

  • 如果从您具有读/写权限的企业文件夹删除文件或文件夹，则文件或文件夹将从企业工作区中的该文件夹自动删除。读/写共享文件夹由 标识。
