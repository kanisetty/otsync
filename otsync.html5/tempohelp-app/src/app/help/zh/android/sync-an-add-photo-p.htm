添加照片或文件夹

要使用 Android 照相机添加照片：

  1. 点击 [Tempo Box]，然后导航到要在其中添加照片的目标文件夹。

    您无法在具有只读权限时，添加照片至共享文件夹中。只读共享文件夹由 覆盖标记。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

    您无法通过 Tempo Box 向具有强制类别和属性的文件夹添加照片。要确定这一情况，请参阅 查看对象详细信息。

  2. 点击 Android [菜单]键。点击[添加] > [从相机]。

  3. 拍摄照片，然后遵循屏幕上的说明进行操作。照片保存为 .jpg 文件。上载照片之前不能编辑照片名称。

要从设备图库导入照片：

  1. 启动 Android 图库应用程序。

  2. 导航到您要导入的照片。

  3. 点击 Android [菜单]键，然后点击[共享]。

  4. 点击 [Tempo Box]。

  5. 导航到要在其中添加照片的目标文件夹。

    您无法在具有只读权限时，将照片导入到共享文件夹中。只读共享文件夹由 覆盖标记。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

    您无法通过 Tempo Box 向具有强制类别和属性的文件夹添加照片。要确定这一情况，请参阅 查看对象详细信息。

  6. 点击 Android [菜单]键，然后点击[粘贴]。

要添加文件夹：

  1. 点击 [Tempo Box]，然后导航到新文件夹的目标位置。

    您无法在具有只读权限时，添加文件夹至共享文件夹中。只读共享文件夹由 覆盖标记。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

  2. 点击 Android [菜单]键。点击[添加] > [文件夹]。

  3. 点击[文件夹名称]框，然后键入此文件夹的名称。

  4. 点击[确定]。
