添加文件

要使用设备上的文件添加新版本：

  1. 点击 [Tempo Box]，然后导航到目标文件。

     

    重要:
    当您在具有读/写权限的共享文件夹中向文件添加新版本时，不会自动保留这些文件。您可以在协作者正在对文件进行更改的同时向其添加新版本。要避免这种情况，您可以保留该文件。有关详细信息，请参阅 保留和取消保留文件。

    读/写共享文件夹由 覆盖标记。有关详细信息，请参阅 了解 OpenText Tempo Box 图标。

     

    由另一个协作者在共享文件夹中进行保留时，您无法保存向文件添加新版本。保留文件标有 覆盖标记。要确定谁保留了文件，您可以检查文件的审核历史记录。有关详细信息，请参阅 查看对象详细信息。

    此外，您无法在具有只读权限时，在共享文件夹中添加文件版本。只读共享文件夹由 覆盖标记。

  2. 轻触并按住要向其添加新版本的文件。

  3. 点击[添加版本] > [从设备]。

  4. 导航至您设备中文件的新版本。

  5. 点击文件的新版本。

要上载存储在设备中的文件：

  1. 点击 [Tempo Box]，然后导航到要上载文件的目标文件夹。

    您无法在具有只读权限时，上载文件至共享文件夹中。只读共享文件夹由 覆盖标记。

    您无法通过 Tempo Box 在具有强制类别和属性的文件夹中添加文件。要确定这一情况，请参阅。

  2. 轻触并按住目标文件夹。点击[在此上载] > [从设备]。

  3. 导航到 Android 设备上的目标文件。

  4. 点击文件。

要从其他应用程序导入文件：

  1. 在其他应用程序中，查看文档。

  2. 点击[导出]或[共享]。

  3. 点击 [Tempo Box]。

  4. 可选 如果出现提示，登录到 OpenText Tempo Box。

  5. 导航到要在其中放置文件的目标位置。

    您无法在具有只读权限时，导入文件至共享文件夹中。只读共享文件夹由 覆盖标记。

    您无法通过 Tempo Box 在具有强制类别和属性的文件夹中添加文件。要确定这一情况，请参阅 查看对象详细信息。

  6. 点击 Android [菜单]键，然后点击[粘贴]。
