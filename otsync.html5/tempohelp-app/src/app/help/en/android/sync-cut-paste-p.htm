Moving or Copying a File or Folder

To move or copy a file:

  1. Tap Tempo Box and navigate to the target file.

    You can only copy a file from a shared folder for which you have read-only access. Read-only shared folders are identified by the overlay. For details, see Understanding the Tempo App Icons.

    Also, you can only copy a file when it is reserved by another collaborator in a shared folder for which you have read/write access. Reserved files are marked with the overlay. To determine who reserved the file, review the audit history of the file. For details, see Viewing Object Details.

  2. Touch and hold the file that you want to move or copy. You can only copy a file if you have read-only access to it. Read-only files are identified by the lock overlay .

  3. Tap Cut to move the file or tap Copy to copy the file.

  4. Navigate to the target folder where you want to place the file.

    You cannot move or copy a file to a shared folder or Enterprise folder for which you have read-only access. Read-only shared folders are identified by the overlay.

    You cannot move or copy an item to a folder that has mandatory categories and attributes using the Tempo app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.

  5. Tap .

To move or copy a folder:

  1. Tap Tempo Box and navigate to the target folder.

    You can only copy a subfolder from a shared folder for which you have read-only access. Read-only shared folders are identified by the overlay. For details, see Understanding the Tempo App Icons.

  2. Touch and hold the folder that you want to move or copy. You can only copy an item if you have read-only access to the item. Read-only files and folders are identified by the lock overlay .

  3. Tap Cut to move the item or tap Copy to copy the item.

    You can only copy the root of an Enterprise folder, regardless of your access level. Also, you can only copy Enterprise folders for which you have read-only access. In doing this, a private copy of the folder is created at the new location, but the original Enterprise folder remains at the original location.

     

    Note:
    Before proceeding, refer to the Notes.

     

  4. Navigate to the target folder where you want to place the item.

    You cannot move or copy a file to a shared folder for which you have read-only access. Read-only shared folders are identified by the overlay.

    You cannot move or copy an item to a folder that has mandatory categories and attributes using the Tempo app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.

  5. Tap . You cannot paste a file or folder in an Enterprise folder for which you have read-only access. Read-only folders are identified by the lock overlay .

 

Notes:

  • You can move shared folders that you own to any location in your Tempo Box folder, except to another shared folder. Shared folders that you own display the folder name only and are identified as .

  • You can only copy the root of a folder that is shared with you. In doing this, a private copy of the folder is created at the new location, but the original shared folder remains at the root level of your Tempo Box folder. Folders that are shared with you append the user name of the folder owner in the folder name and are identified as (read/write) or (read-only).

  • You can only copy the root of an Enterprise folder, regardless of your access level. Also, you can only copy subfolders within an Enterprise folder for which you have read-only access. In doing this, a private copy of the folder is created at the new location, but the original Enterprise folder remains at the original location in your Tempo Box folder. Enterprise folders are identified as (read/write) or (read-only).

  • You can only copy the root of a file library in a Tempo Social community, which are identified as (read/write) or (read-only). In doing this, a private copy of the file library is created at the new location, but the original file library remains at the root level of your Tempo Box folder.
