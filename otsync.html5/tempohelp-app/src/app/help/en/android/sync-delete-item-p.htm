Deleting a File or Folder

To delete a file:

  1. Tap Tempo Box and navigate to the target file.

    You cannot delete a file from a shared folder for which you have read-only access. Read-only shared folders are identified by the overlay. For details, see Understanding the Tempo App Icons.

    You cannot delete a file while it is reserved. Reserved files are marked with the overlay. To delete a reserved file, you must first unreserve the file. For details, see Reserving and Unreserving a File.

  2. Touch and hold the name of the item you want to delete. You cannot delete a file or folder if you have read-only access to the item. Read-only files and folders are identified by the lock overlay .

     

    Note:
    Before proceeding, refer to the Caution information below.

     

  3. Tap Delete.

    If you are trying to delete a file in an Enterprise folder for which you have read/write access and the Delete option is not available, you do not have Delete permission to the folder in Content Server. Contact your administrator to request this permission.

  4. Tap OK to confirm this action.

To delete a folder:

  1. Tap Tempo Box and navigate to the target folder.

    If the target folder is a shared folder, see Deleting a Shared Folder before proceeding. Shared folders are identified by the icon (read/write access) or (read-only access). For details, see Understanding the Tempo App Icons.

    You cannot delete a folder if it contains a reserved file. To unreserve the file, see Reserving and Unreserving a File.

  2. Touch and hold the name of the folder you want to delete. You cannot delete a folder if you have read-only access to the folder. Read-only folders are identified by the lock overlay .

     

    Note:
    Before proceeding, refer to the Caution information below.

     

  3. Tap Delete.

  4. Tap OK to confirm this action.

 

Caution

  • Deleting a file or folder removes it from your Tempo Box folder, even if the file was tagged as a favorite. This means the file or folder is deleted from all other OpenText Tempo Box clients you have installed such as the desktop client, other smartphones or tablets, as well as the OpenText Tempo Box Web user interface.

  • If you delete a file or folder from a shared folder for which you have read/write access, the file or folder is automatically deleted for all users who have access to the shared folder, including the owner of the shared folder. Read/write shared folders are identified by the icon.
