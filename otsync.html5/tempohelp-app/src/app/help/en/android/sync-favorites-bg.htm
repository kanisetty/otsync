Adding, Syncing, and Working with Favorites

To add a favorite file or folder:

  1. Tap Tempo Box and navigate to the target file or folder.

  2. Touch and hold the file or folder that you want to tag as a favorite.

  3. Tap Add to Favorites.

    The file or folder is marked with the overlay. For example, . This gives you easy access to the file or folder from the Favorites app. For folders, this creates a shortcut to the folder. It does not favorite the files and subfolders in the folder.

To view a favorite:

  1. Tap Favorites.

  2. Optional Swipe down on your list of favorites to refresh the display. If the overlay appears on any of your files, you do not have the latest version of the file downloaded to your device. To download a file, sync your favorites or open the file. For details, see Syncing Your Favorites.

  3. Tap the file or folder that you want to view.

To remove a favorite:

  1. Tap Favorites, or tap Enterprise Workspace or Personal Workspace and navigate to the target file.

  2. Touch and hold the file that you want to remove from your favorites list.

  3. Tap Remove From Favorites. The file no longer appears in your list of favorites. This does not delete the file from your workspace.
