Deleting a Task

To delete a task:

  1. Tap Tempo Box and navigate to the target folder that contains a task that you created. Tasks appear in the folder properties. You can only delete the tasks you created.

  2. Tap .

     

    Tip:
    Folders with active tasks will show a task icon . For example, . The task icon disappears when the status of all of the tasks is set to Completed or Cancelled, or all of the tasks in the folder are deleted.

     

  3. Tap Tasks.

  4. Locate the task of interest and tap the X in the top right corner of the task.
