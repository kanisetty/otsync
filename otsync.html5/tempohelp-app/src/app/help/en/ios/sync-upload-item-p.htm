Tempo Box Help

 

Adding Photos or Folders

To add a photo using the device camera:

  1. Tap Tempo Box and navigate to the target folder where you want to add a photo.

    You cannot add a photo to a shared folder for which you have read-only access. Read-only shared folders are identified by the overlay. For details, see Understanding the Tempo App Icons.

    You cannot add a photo to a folder that has mandatory categories and attributes using the ECM Everywhere app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.

  2. Tap and then tap From Camera.

  3. Take the picture and follow the on-screen instructions. Photos are saved as .jpg files. You cannot edit the name of the photo before uploading it.

To add a folder:

  1. Tap Tempo Box and navigate to the target location for the new folder.

    You cannot add a folder to a shared folder for which you have read-only access. Read-only shared folders are identified by the overlay.

  2. Tap and then tap Folder.

  3. Tap the Folder Name box and type a name for this folder.

  4. Tap OK.
