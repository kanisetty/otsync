Viewing Object Details

To view object details:

  1. Tap Tempo Box and navigate to the target file or folder.

  2. Touch and hold the item.

  3. Tap Object Details.

  4. Do one of the following:

    • Tap Properties to see details about the object.

    • Tap Categories to see the categories and attributes assigned to the object.

    • Tap Audit to see the audit history of the object.

  5. Tap to return to the workspace.
