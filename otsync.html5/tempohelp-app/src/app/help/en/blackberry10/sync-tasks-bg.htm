Creating and Managing Tasks

To create a task:

  1. Tap Tempo Box and navigate to the target folder. Tasks must be created in a folder.

     

    Important:
    Users with read-only access to a Tempo Box shared folder or Enterprise folder cannot create tasks.

     

  2. Tap to view the folder properties.

  3. Tap Tasks.

  4. Tap the Task Description box, enter a description for your task, and then tap Done. Every task that you create must have a unique description.

    If you are assigning a task to a collaborator, it’s best to provide a thorough description of the task and state the document(s) you want the collaborator to review.

    You cannot update the description after the task is created. If you want to alter the description, you need to create a new task.

  5. Optional Tap Assignee, select the colleague’s name from the list that appears, and then tap Done.

    You can assign a task to Content Server users, Standard users, or External users as long as they have access to the folder. You can also assign a task to yourself.

     

    Important:
    You can only assign a task to individual users in a shared or Enterprise folder. Assigning tasks to groups is not supported at this time.

     

  6. Optional Tap Due Date, select a date, and then tap Done.

    The due date must be greater than the current date.

    You cannot assign or update the due date after the task is created. If you want to alter the due date of a task, you need to create a new task.

  7. Tap Create Task.

  8. Tap Back to return to your folder list.

 

Tip:
The folder will show a task icon when the folder has active tasks. For example, . When the status of all of the tasks is set to Completed or Cancelled, the task icon disappears.

 

For information on deleting and updating the status of tasks, refer to the other topics in Creating and Managing Tasks.
