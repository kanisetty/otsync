Tempo Box Help

 

Viewing, Editing, and Working With Your Files

To view a file:

  1. Tap Tempo Box and navigate to the target file. The title bar displays the name of the folder you are currently viewing.

  2. Optional To ensure you are viewing the most up-to-date content, you should periodically refresh the folder content. Swipe down on the folder list. This refreshes the display, but does not download the files. To download a file, open it.

  3. Tap the file you want to view.

  4. Tap to return to your files.

To edit a file:

  1. Tap Tempo Box and navigate to the target file. To edit a file, you need to edit the file in a third party app and then import the edited file into Tempo Box again.

     

    Important:
    When viewing files in a shared folder for which you have read/write access, the files are not automatically reserved. You can open and edit a file at the same time a collaborator is making changes to it. To prevent this, you can reserve the file. For details, see Reserving and Unreserving a File.

     

    You cannot save edits to a file when it is reserved by another collaborator in a shared folder. Reserved files are marked with the overlay. To determine who reserved the file, you can check the audit history of the file. For details, see Viewing Object Details.

    Also, you cannot save edits to a file in a shared folder for which you have read-only access. Read-only shared folders are identified by the overlay.

  2. Tap the file you want to edit. If you have an app on your device that allows editing of this file type, the third party app will open.

  3. In the third party app, edit the file and save your changes to the device.

  4. In OpenText Tempo Box, you can now add a new version to the original file. For details, see Adding a File.

For information on working with your files, refer to the other sections in Viewing, Editing, and Working With Your Files.
