Deleting a Shared Folder

To delete a shared folder that you own:

  1. Tap Tempo Box and navigate to the target shared folder.

    Read/write shared folders are identified by the overlay. For details, see Understanding the Tempo App Icons.

    If the folder name of the target folder is appended with a user name, the target folder is shared with you. For more information, see Deleting a Folder That is Shared With You below.

  2. Swipe your finger horizontally over the name of the shared folder. A Delete button appears in the folder row.

    If a Remove Share button appears, you selected a folder that is shared with you, an Enterprise folder, or a file library from a Tempo Social Community. For more information, see Deleting a Folder That is Shared With You, Removing an Enterprise Folder or Removing a File Library from a Tempo Social Community.

  3. Tap Delete.

     

    Caution
    This will delete the folder for you and all collaborators. As an alternative, you can remove collaborators. For details, see Removing a Collaborator.

     

  4. Tap Delete this item to confirm this action.

To delete a folder that is shared with you:

  1. In the root of your Tempo Box folder, locate the target folder.

    A folder that is shared with you shows the user name of the folder owner in the folder name, for example, Marketing Materials (jsmith), and is identified as or .

  2. Swipe your finger horizontally over the folder name. A Remove Share button appears in the folder row.

    If a Delete button appears, you selected a shared folder that you own. Before proceeding, see Deleting a Shared Folder That You Own.

  3. Tap Remove Share.

    This will sever the connection to the shared folder and remove it from your Tempo Box folder. This does not delete the folder for the owner or other collaborators.

  4. Tap Remove this share to confirm this action.

To remove an Enterprise folder:

  1. In the root of your Tempo Box folder, locate the target Enterprise folder. Enterprise folders are identified as or .

  2. Swipe your finger horizontally over the folder name. A Remove Share button appears in the folder row.

     

    Caution
    You must remove the actual folder and not delete the content inside the Enterprise folder. Deleting the content in the Enterprise folder will delete the content for all collaborators who have subscribed to the folder, as well as delete the content in the Enterprise Workspace.

     

  3. Tap Remove Share.

    This will sever the connection to the folder and remove it from your Tempo Box folder. This does not delete the folder in the Enterprise Workspace. If you accidentally deleted the Enterprise folder, Content Server users can navigate to the folder in the Enterprise Workspace and add the folder once again.

  4. Tap Remove this share to confirm this action.

To remove a file library from a Tempo Social community:

  1. In the root of your Tempo Box folder, locate the target file library. File libraries are identified as or .

  2. Swipe your finger horizontally over the folder name. A Remove Share button appears in the folder row.

    This will sever the connection to the file library and remove it from your Tempo Box folder. This does not delete the file library in Tempo Social. If you accidentally deleted the file library, navigate to the community in Tempo Social, leave and then rejoin the community. For more information about Tempo Social communities, see the OpenText Tempo Social - Social Workplace Site User Help (WCSOC-H-UWS).

  3. Tap Remove this share to confirm this action.
