Viewing Files in Shared Folders

To view files in shared folders:

  1. Tap Notifications. Your notifications can include invitations to share folders or Enterprise objects. For details on share and Enterprise objects, see Understanding the Tempo App Icons.

    Before you can view the content in a shared folder, the folder owner must share the folder with you. With Enterprise folders, a user who has the Edit Permissions permission to an Enterprise folder can send you a sharing invitation to the folder. Both of these activities will send you a share invitation that you must accept before you can see the content in the folder. For details on sharing a folder, see Sharing a Folder and Managing Shared Folders.

  2. Tap a share invitation.

  3. Tap Accept.

    The shared folder is placed in the root of your Tempo Box folder and will appear as (read/write access) or (read-only access).

    With read/write access, you can access and edit the content in a shared folder as if it was your own. Any changes made to the content are synced with all other collaborators who have share access to the folder, including the owner.

    With read-only access, you can only read and copy the content in the shared folder.

    The folder name is appended with the user name of the folder owner. For example, Marketing Materials (jsmith).

    If you accepted a share invitation for an Enterprise folder, the folder is placed in the root of your Tempo Box folder and will appear as (read/write access) or (read-only access).

    The Enterprise folder may append the parent folder name to the folder depending on your permissions to the parent folder. For example, Payroll (Human Resources).
