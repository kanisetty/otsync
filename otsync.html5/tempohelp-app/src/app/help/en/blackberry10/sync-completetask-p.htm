Completing a Task Assignment

To complete a task assignment:

  1. Tap Tempo Box and navigate to the target folder that contains the task. Tasks appear in the folder properties.

     

    Tip:
    Folders with active tasks will show a task icon . For example, . When the status of all of the tasks is set to Completed or Cancelled, the task icon disappears.

     

  2. Optional To view the task assignment, do the following:

    1. Tap for the folder that contains the task.

    2. Tap Tasks. Locate the task of interest for details on completing the task.

    3. Tap Back until you reach the folder list.

  3. Tap the folder that contains your task and complete the task as described.

     

    Tip:
    You can also open the folder that contains your task from the Tempo home page by tapping the notification icon .

     

  4. Optional To update the status of the task, do the following:

    1. Tap Back.

    2. Tap for the folder that contains the task.

    3. Tap Tasks.

    4. Locate the task assigned to you and change the status from Pending to one of the other settings and then tap Done.

      If a task is not assigned, you can update the task status. Updating the status of an unassigned task does not assign the task to you. You cannot update the status of the tasks assigned to other users.
