Tempo Box Help

 

Adding a File

To add a file:

  1. You must import the file from another app. View the file you want to import and tap and/or Open In.

  2. Tap Open in Tempo.

  3. Do one of the following:

    • To add the file to the root of your Tempo Box folder, tap Upload.

    • To choose a location other than the root of your Tempo Box folder, tap Destination: Personal Workspace

      1. Tap OpenText Tempo Box. The root of your Tempo Box folder appears.

      2. Navigate to the target folder.

        You cannot add a file to a shared folder for which you have read-only access. Read-only shared folders are identified by the overlay. For details, see Understanding the Tempo App Icons.

        If the target folder that has mandatory categories or attributes, you cannot add a photo to this folder using the Tempo app. To determine if the folder has mandatory categories or attribute, see Viewing Object Details.

      3. To select the target folder, tap Choose.

        To add a new folder, tap New Folder. Type the name of the new folder and tap Done. Tap Choose to select the new folder.

      4. Tap Upload.

  4. Optional In OpenText Tempo, tap the imported file. To move the file to another location, see Moving or Copying a File or Folder.
