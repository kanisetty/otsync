第4章. 添加、查看和移除收藏

 

要添加收藏文件或文件夹：

  1. 点击[企业工作区]或[个人工作区]，然后导航至目标文件或文件夹。

  2. 点击 。

  3. 点击[添加到收藏夹]。文件或文件夹在文件夹视图下使用 覆盖标记。这使您能够从[收藏]应用程序轻松访问文件或文件夹。对于文件夹，不会收藏文件夹中的文件和子文件夹。

    收藏文件的记录存储在配置文件中，其特定于您的 iPad 或 iPhone。因此必须在运行 OpenText ECM Everywhere 应用程序的每个智能手机和平板电脑上将文件标记为收藏。

  4. 点击[返回]返回到文件。

  2. 点击要标记为收藏的文件。将会打开文件。

  4. 点击[返回]。

要查看收藏：

  1. 点击[收藏]。

  2. 可选 点击 确保您正在查看的是最新的内容。

  3. 点击要查看的文件。

  4. 点击[返回]返回到文件。

要移除收藏：

  1. 点击[收藏]，或点击[企业工作区]或[个人工作区]，然后导航至目标文件或文件夹。

  2. 点击项目的 ，然后点击[移除收藏项]。该文件将不再显示在收藏列表中。此操作不会从您的工作区中删除文件或文件夹。

  3. 点击[返回]返回到文件。
