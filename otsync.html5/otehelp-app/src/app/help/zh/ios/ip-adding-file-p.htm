添加文件

 

要添加项目：

  1. 您必须通过从另一个应用程序上载来导入该项目（如果应用程序支持此功能）。从第三方应用程序，查看您希望导入的文件，然后采取必要步骤在 OpenText ECM Everywhere 中打开文档。例如，点击 , ，[共享]或[打开方式]。将会显示应用程序列表。

  2. 点击 [Everywhere 10]。

  3. 可选 如果出现提示，登录到 OpenText ECM Everywhere。

  4. 点击[目标：个人工作区]以导航至要添加此文件的位置，然后点击[选择]。

  5. 可选 点击[文件名]框，然后编辑文件名。点击[完成]。

  6. 点击[上载]。
