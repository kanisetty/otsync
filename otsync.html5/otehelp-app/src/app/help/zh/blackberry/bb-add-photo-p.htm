要使用 BlackBerry 照相机添加照片：

  1. 在[主页]页面上，单击[ Enterprise Workspace] 或 [个人工作区]，然后导航到要在其中添加照片的目标文件夹。您将照片添加到 [Enterprise Workspace] 中的文件夹的能力取决于您对选定文件夹的权限。

  2. 按 BlackBerry [菜单]键，然后选择[添加照片]。

  3. 按 BlackBerry [触控板]键以拍摄照片。

  4. 单击[添加到 OpenText Everywhere]。照片保存为 .jpg 文件。上载照片之前不能编辑照片名称。

要从 BlackBerry 应用程序添加照片：

  1. 启动 BlackBerry 相机应用程序。

  2. 导航到您要导入的照片。

  3. 按 BlackBerry [菜单]键，然后选择[添加到 OpenText Everywhere]。

  4. 可选 如果出现提示，登录到 ECM Everywhere。有关详细信息，请参阅登录和管理配置文件。

  5. 单击[Enterprise Workspace] 或 [个人工作区]。

  6. 导航到要添加照片的位置。您将照片添加到 [Enterprise Workspace] 中的文件夹的能力取决于您对选定文件夹的权限。

  7. 按 BlackBerry [菜单]键，然后选择[在此上载]。

要添加文件夹：

  1. 在 [Enterprise Workspace] 或 [个人工作区] 中，导航到新文件夹的目标位置。您在 [Enterprise Workspace] 中添加文件夹的能力取决于您对选定文件夹的权限。

  2. 按 BlackBerry [菜单]键，然后选择[添加文件夹]。

  3. 在[文件夹名称]框中，键入此文件夹的名称。

  4. 按 BlackBerry [Enter] 键。
