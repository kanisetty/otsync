要删除文件或文件夹：

  1. 在[主页]页面上，单击[ Enterprise Workspace] 或 [个人工作区]，然后导航到目标文件或文件夹。

     

    注意:
    继续操作之前，请参阅下方的注意 信息。

     

  2. 突出显示目标文件或文件夹。

  3. 按 BlackBerry [菜单]键，然后选择[删除项目]或[删除文件夹]。

  4. 单击[确定]。

 

小心
即使文件标记为收藏，删除该文件或文件夹也会将其从 [Enterprise Workspace] 或 [个人工作区] 中移除。这意味着文件或文件夹将从 Content Server 中删除。
