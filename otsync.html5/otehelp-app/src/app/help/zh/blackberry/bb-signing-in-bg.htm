要登录：

  1. 您必须先创建配置文件才能登录到该应用程序。如果您不具有配置文件，请执行以下操作：

    1. 在[配置文件]页面上，突出显示[新配置文件]，然后按 BlackBerry [触控板]键。您必须联机才能创建新的配置文件并登录。

    2. 点击[配置文件名]框，然后键入唯一的配置文件名。

    3. 在[用户名]框中，键入您的 Content Server 用户名。

    4. 在[密码]框中，键入您的 Content Server 密码。密码区分大小写。请确保您键入的密码正确。

    5. 在[服务器]框中，键入 ECM Everywhere (OpenText AppWorks Gateway) 服务器的 URL。

      请与管理员联系以获取 ECM Everywhere 服务器的 URL。

    6. 选择[保持登录]复选框，以便在下次启动应用程序时立即进行访问。唯一的例外是如果您从 ECM Everywhere 注销或会话超时。请参阅以下与该选项相关的注意 信息。

    7. 单击[保存并登录]。

  2. 如果您已经具有配置文件，请执行以下操作：

    1. 在[配置文件]页面上，突出显示您的配置文件，然后按 BlackBerry [触控板]键。

    2. 如果出现提示，请键入您的密码。密码区分大小写。请确保您键入的密码正确。

    3. 选择[保持登录]复选框，以便在下次启动应用程序时立即进行访问。唯一的例外是如果您从应用程序注销或会话超时。请参阅以下与该选项相关的注意 信息。

    4. 按 BlackBerry [Enter] 键。

 

小心
允许 ECM Everywhere 保持您的登陆状态可能危及安全。如您的 BlackBerry 设备在注销 ECM Everywhere 应用程序后被另一位用户使用，该用户可以进入您的 Content Server 帐户。要注销，请导航到[主页]页面，按 BlackBerry [菜单]键，并选择[注销]。

 

有关管理配置文件的信息，请参考其他章节：登录和管理配置文件 。
