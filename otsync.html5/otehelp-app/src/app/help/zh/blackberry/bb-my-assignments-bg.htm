要查看您的任务分配：

  1. 在[主页]页面上，单击[任务分配]，然后单击要查看的任务分配的 >。

    应用程序中所见的工作流任务分配不支持说明和注释区域中的 HTML 标记。如果未正确呈现说明，则从 Content Server 中的工作流程图中移除 HTML 标记。

  2. 编辑属性。

  3. 单击[转送]。

要编辑属性：

  1. 单击[任务分配]，然后单击要查看的工作流的 >。

  2. 单击属性行，然后键入必需信息或单击提供的选项列表并从中进行选择。

     

    注意:
    必需属性均标有星号。

     

  3. 单击[转送]。

要在用户字段属性中添加同事：

  1. 单击[任务分配]，然后单击要查看的工作流的 >。

  2. 单击用户字段属性。

  3. 单击文本框。

  4. 键入您同事的名字、姓氏或用户名。

  5. 在结果列表中选择您的同事。

有关处理任务分配的信息，请参考其他章节：查看、编辑和处理任务分配。
