要搜索 ECM Everywhere：

  1. 在[主页]页面上，单击[ Enterprise Workspace] 或 [个人工作区]，然后导航到要开始搜索的目标文件夹。

  2. 使用 BlackBerry [触控板]来突出显示并选择 。

  3. 键入搜索词。

  4. 按 BlackBerry [Enter] 键。搜索结果将会显示。

  5. 突出显示要查看的文件或文件夹。

  6. 按 BlackBerry [菜单]键以显示可用选项。

要从搜索结果中打开项目的位置：

  1. 搜索 [Enterprise Workspace] 或 [ 个人工作区]。

  2. 突出显示要在文件位置打开的文件。

  3. 按 BlackBerry [菜单]键，然后选择[打开位置]。

  4. 按 BlackBerry [退出] 键返回到搜索结果。
