要使用您 BlackBerry 上的文件添加新版本：

  1. 在[主页]页面上，单击[ Enterprise Workspace] 或 [个人工作区]，然后导航至目标文件。

  2. 突出显示要向其添加新版本的目标文件。

  3. 按 BlackBerry [菜单]键，然后选择[上载新版本]。

  4. 导航至您 BlackBerry 中的新版本文件。

  5. 突出显示该文件。

  6. 按 BlackBerry [菜单]键，然后选择[上载]。您将版本添加到 [Enterprise Workspace] 中的文件的能力取决于您对选定文件夹的权限。

要上载存储在 BlackBerry 中的文件：

  1. 在 [Enterprise Workspace] 或 [个人工作区] 中，导航到要上载文件的目标文件夹。

  2. 按 BlackBerry [菜单]键，然后选择[在此上载]。

  3. 导航至 BlackBerry 智能手机上的目标文件。您将文件上载到 [Enterprise Workspace] 中的文件夹的能力取决于您对选定文件夹的权限。

  4. 按 BlackBerry [菜单]键，然后选择[上载]。

要从 BlackBerry 应用程序导入文件：

  1. 启动 BlackBerry 应用程序，然后导航到您要导入的目标文件。

  2. 按 BlackBerry [菜单]键，然后选择[添加到 OpenText Everywhere]。

  3. 可选 如果出现提示，登录到 ECM Everywhere。有关详细信息，请参阅登录和管理配置文件。

  4. 单击[Enterprise Workspace] 或 [个人工作区]。

  5. 导航到要添加文件的位置。您将文件添加到 [Enterprise Workspace] 中的文件夹的能力取决于您对选定文件夹的权限。

  6. 按 BlackBerry [菜单]键，然后选择[在此上载]。
