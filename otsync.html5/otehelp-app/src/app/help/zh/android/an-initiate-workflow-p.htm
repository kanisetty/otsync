启动工作流

 

要启动工作流：

  1. 点击 [Enterprise Workspace] 或 [Personal Workspace]，然后导航到工作流程图位置。

    您在 Enterprise Workspace 中启动工作流的能力取决于您对选定文件夹的权限。

  2. 点击要启动的工作流程图名称。启动工作流之前，工作流程图摘要将显示您可在何处编辑属性、查看附件或添加注释。

    OpenText ECM Everywhere 中所见的工作流任务分配不支持说明和注释区域中的 HTML 标记。如果未正确呈现文本，则从 Content Server 中的工作流程图中移除 HTML 标记。

  3. 可选 编辑属性（如果可用）。必需属性均标有星号。有关详细信息，请参阅查看、编辑和处理任务分配。

  4. 可选 在[注释]框中为步骤受派者键入注释。

  5. 点击[启动]。

    填充必需属性后，[启动]按钮才可用。
