添加照片或文件夹

 

要使用 Android 照相机添加照片：

  1. 点击 [Enterprise Workspace] 或 [Personal Workspace]，然后导航到要添加照片的目标文件夹。

    您无法使用 OpenText ECM Everywhere 应用程序在具有强制类别和属性的文件夹中添加照片。要确定文件夹是否具有强制性类别或属性，请查看对象详细信息。有关详细信息，请参阅查看对象详细信息。

  2. 点击[添加]，然后点击[来自相机]。

  3. 拍摄照片，然后遵循屏幕上的说明进行操作。照片保存为 .jpg 文件。上载照片之前不能编辑照片名称。

要从设备图库上载照片：

  1. 启动 Android 图库应用程序，然后导航至要导入的照片。

  2. 点击[共享]或[打开方式]。将会显示应用程序列表。

  3. 点击 [Everywhere 10]。

  4. 可选 如果出现提示，登录到 OpenText ECM Everywhere。

  5. 点击 [Enterprise Workspace] 或 [Personal Workspace]，然后导航到要添加此照片的位置。

    您无法使用 OpenText ECM Everywhere 应用程序在具有强制类别和属性的文件夹中上载照片。要确定文件夹是否具有强制性类别或属性，请查看对象详细信息。有关详细信息，请参阅查看对象详细信息。

  6. 点击[粘贴]。

要添加文件夹：

  1. 点击 [Enterprise Workspace] 或 [Personal Workspace]，然后导航到新文件夹的目标位置。

  2. 点击[添加]，然后点击[文件夹]。

  3. 点击[文件夹名称]框，然后键入此文件夹的名称。

  4. 点击[确定]。
