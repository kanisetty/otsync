第1章. 查看、编辑和处理文件

要查看项目：

  1. 在 [Enterprise Workspace] 或 [Personal Workspace] 中，导航到目标项目。

  2. 可选 要确保正在查看的是最新内容，您应该定期刷新显示。点击[菜单]，然后点击[刷新]。

  3. 点击要查看的文件。

     

    提示:
    您也可轻触并按住要查看的文件，然后点击[打开]。将会显示可以打开文件的应用程序列表。点击所需的应用程序。

     

  4. 点击 Android [返回]键返回到文件。

要编辑项目：

  1. 点击 [Enterprise Workspace] 或 [Personal Workspace]，然后导航到目标项目。要编辑文件，您需要将文件导出至第三方应用程序，然后将编辑后的文件再次导入 OpenText ECM Everywhere 中。

    文件由另一名同事在 Enterprise Workspace 中进行保留时，您无法将编辑保存到其中。保留文件标有 覆盖标记。要确定谁保留了文件，请查看文件的审核历史记录。有关详细信息，请参阅查看对象详细信息。

  2. 轻触并按住项目。

  3. 点击[电子邮件/导出]。

  4. 点击第三方应用程序。

  5. 在第三方应用程序中，编辑文件并将您的更改保存到设备中。

  6. 此时您可在 OpenText ECM Everywhere 中将新版本添加到原始文件。有关详细信息，请参阅添加文件。

    如果您从第三方应用程序导入文件，这会在 OpenText ECM Everywhere 中添加一个新文件。

有关处理文件的信息，请参阅查看和处理文件。
