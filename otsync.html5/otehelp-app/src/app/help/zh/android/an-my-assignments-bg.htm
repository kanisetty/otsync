第4章. 查看、编辑和处理任务分配

 

要查看您的任务分配：

  1. 点击[任务分配]，然后点击要查看的工作流的 >。OpenText ECM Everywhere 中所见的工作流任务分配不支持说明和注释区域中的 HTML 标记。如果未正确呈现说明，则从 Content Server 中的工作流程图中移除 HTML 标记。

  2. 编辑属性。

  3. 点击[转送]。

要编辑属性：

  1. 点击[任务分配]，然后点击要查看的工作流的 >。

  2. 点击属性行，然后键入必需信息或点击提供的选项列表并从中进行选择。

    必需属性均标有星号。

  3. 点击[转送]。

要在用户字段属性中添加同事：

  1. 在[任务分配]中，点击您的任务分配列表中的工作流任务。

  2. 点击用户字段属性。

  3. 点击文本框。

  4. 键入您同事的姓名。

  5. 点击以在结果列表中选择您的同事。

有关处理任务分配的信息，请参阅查看、编辑和处理任务分配 。
