添加文件

要使用设备上的文件添加新版本：

  1. 点击 [Enterprise Workspace] 或 [Personal Workspace]，然后导航到目标文件。

  2. 轻触并按住要向其添加新版本的文件。

    文件由另一名同事在 Enterprise Workspace 中进行保留时，您无法将新版本添加到其中。保留文件标有 覆盖标记。要确定谁保留了文件，请查看文件的审核历史记录。有关详细信息，请参阅查看对象详细信息。

    您将版本添加到 Enterprise Workspace 中的文件的能力取决于您对选定文件夹的权限。

  3. 点击[添加版本]，然后点击[从设备]。

  4. 导航至您设备中文件的新版本。

  5. 点击文件。

要上载存储在设备中的文件：

  1. 点击 [Enterprise Workspace] 或 [Personal Workspace]，然后导航到要上载文件的目标文件夹。

  2. 轻触并按住目标文件夹。

    您无法使用 OpenText ECM Everywhere 应用程序在具有强制类别和属性的文件夹中上载文件。要确定文件夹是否具有强制性类别或属性，请查看对象详细信息。有关详细信息，请参阅查看对象详细信息。

    您将文件上载到 Enterprise Workspace 中的文件夹的能力取决于您对选定文件夹的权限。

  3. 点击[在此上载]，然后执行以下操作之一：

    • 点击[来自相机]以使用手机拍摄照片并将其上载到目标文件夹。照片保存为 .jpg 文件。上载照片之前不能编辑照片名称。

    • 点击[从设备]以上载设备上存储的文件。导航到 Android 设备上的目标文件，然后点击文件。

    • 点击[来自图库]以上载设备照片图库中的图片。浏览设备图库，然后点击图片。

要从其他应用程序导入文件：

  1. 从第三方应用程序查看要添加的文件。

  2. 点击[共享]或[打开方式]。将会显示应用程序列表。

  3. 点击 [Everywhere 10]。

  4. 可选 如果出现提示，登录到 OpenText ECM Everywhere。

  5. 点击 [Enterprise Workspace] 或 [Personal Workspace]，然后导航到要添加此文件的位置。您将文件导入到 Enterprise Workspace 中的文件夹的能力取决于您对选定文件夹的权限。

    您无法使用 OpenText ECM Everywhere 应用程序在具有强制类别和属性的文件夹中导入文件。要确定文件夹是否具有强制性类别或属性，请查看对象详细信息。有关详细信息，请参阅查看对象详细信息。

  6. 点击[粘贴]。
