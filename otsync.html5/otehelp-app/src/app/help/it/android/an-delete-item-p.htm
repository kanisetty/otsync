Eliminazione di un file o di una cartella

 

Per eliminare un elemento:

  1. Toccare Enterprise Workspace o Area di lavoro personale e spostarsi alla cartella o al file desiderato.

    Non è possibile eliminare un file se risulta riservato. I file riservati sono contrassegnati con l'overlay . Per eliminare un file riservato, è innanzitutto necessario rilasciarlo. Per ulteriori dettagli, vedere Modalità per riservare e rilasciare un file.

  2. Toccare e tenere premuto il nome dell'elemento che si desidera eliminare.

  3. Toccare Elimina. Se si elimina un file o una cartella, l'elemento viene rimosso da Enterprise Workspace o Area di lavoro personale anche se era stato contrassegnato come preferito.

  4. Toccare OK per confermare questa azione.
