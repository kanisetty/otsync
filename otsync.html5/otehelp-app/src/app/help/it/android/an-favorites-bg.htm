Visualizzazione di file offline

Per visualizzare file offline:

  1. Contrassegnare il file come preferito e sincronizzarlo o scaricarlo mentre si è online. Le cartelle non sono disponibili offline. Per ulteriori dettagli, vedere Aggiunta, visualizzazione, rimozione e sincronizzazione dei preferiti.

    I file contrassegnati come preferiti sono indicati dall'overlay . Ad esempio, .

    Se l'icona il file preferito non è stato scaricato. Ad esempio, . Per scaricare il file, aprirlo o sincronizzare i preferiti. Per ulteriori dettagli, vedere Aggiunta, visualizzazione, rimozione e sincronizzazione dei preferiti.

  2. Toccare Preferiti, quindi toccare il file che si desidera visualizzare.

    I file che si aprono quando si è offline potrebbero non essere le versioni più recenti, in quanto non è possibile sincronizzare con il dispositivo eventuali modifiche apportate ai file utilizzando un client che è online.

  3. Toccare il tasto Android Indietro per tornare ai propri file.
