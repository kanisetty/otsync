Visualizzazione, modifica e lavoro sui file

Per visualizzare un elemento:

  1. In Enterprise Workspace o Area di lavoro personale, spostarsi all'elemento desiderato.

  2. Opzionale Per assicurarsi di consultare sempre i contenuti più recenti, aggiornare periodicamente la visualizzazione. Toccare Menu, quindi Aggiorna.

  3. Toccare il file che si desidera visualizzare.

     

    Suggerimento:
    È anche possibile toccare e tenere premuto il file da visualizzare e quindi toccare Apri. Viene visualizzato un elenco delle applicazioni che sono in grado di aprire il file. Toccare un'applicazione.

     

  4. Toccare il tasto Android Indietro per tornare ai propri file.

Per modificare un elemento:

  1. Toccare Enterprise Workspace o Area di lavoro personale e spostarsi all'elemento desiderato. Per modificare un file, è necessario prima esportarlo in un'applicazione di terze parti e poi importare nuovamente il file in OpenText ECM Everywhere dopo averlo modificato.

    Impossibile salvare le modifiche a un file quando viene riservato da un altro collega in Enterprise Workspace. I file riservati sono contrassegnati con l'overlay . Per stabilire chi ha riservato il file, rivedere la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

  2. Toccare e tenere premuto l'elemento.

  3. Toccare E-mail/Esporta.

  4. Toccare un'applicazione di terze parti.

  5. Nell'applicazione di terze parti, modificare il file e salvare nel modifiche sul dispositivo.

  6. A questo punto, è possibile aggiungere in OpenText ECM Everywhere una nuova versione del file originale. Per ulteriori dettagli, vedere Aggiunta di un file.

    Se si importa il file da un'applicazione di terze parti, viene aggiunto un nuovo file in OpenText ECM Everywhere.

Per informazioni sull'utilizzo dei file, fare riferimento a Visualizzazione e lavoro sui file.
