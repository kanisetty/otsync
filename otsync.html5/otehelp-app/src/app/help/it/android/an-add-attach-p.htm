Aggiunta di un allegato

Per aggiungere un allegato:

  • Quando si aggiunge un aggiornamento di stato o si commenta un post, toccare Allega ed eseguire una delle operazioni seguenti:

    • Toccare Da macchina fotografica per allegare un'immagine utilizzando la macchina fotografica Android. Scattare la foto e seguire le istruzioni sullo schermo per salvare l'immagine.

    • Toccare Da file system per allegare un file memorizzato sul proprio dispositivo Android. Visualizzare il file e toccarlo.

    • Toccare Da galleria per allegare un'immagine dalla galleria di immagini Android. Visualizzare l'immagine e toccarla.
