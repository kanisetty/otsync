Visualizzazione dei commenti su un file o una cartella

 

Per visualizzare i commenti su un file o una cartella:

  1. Toccare Enterprise Workspace o Area di lavoro personale e spostarsi alla cartella desiderata.

  2. Toccare e tenere premuto l'elemento.

  3. Eseguire una delle seguenti operazioni:

    • Toccare Commenti per visualizzare tutti i post associati all'elemento selezionato.

    • Toccare Commenti da qui per visualizzare tutti i commenti aggiunti ad ogni file e cartella all'interno di una cartella.

  4. Toccare il tasto Indietro in Android per tornare all'area di lavoro.
