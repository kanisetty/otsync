Apertura di un file in un'altra applicazione

Per aprire un file in un'altra applicazione:

  1. Toccare Enterprise Workspace o Area di lavoro personale e spostarsi al documento o l'immagine desiderata.

  2. Toccare e tenere premuto l'elemento.

  3. Toccare E-mail/Esporta. Viene visualizzato un elenco delle applicazioni che sono in grado di aprire il file.

  4. Toccare un'applicazione e seguire le istruzioni sullo schermo.
