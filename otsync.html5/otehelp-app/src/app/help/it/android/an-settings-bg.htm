Modifica delle impostazioni di OpenText ECM Everywhere

 

Per modificare le impostazioni di OpenText ECM Everywhere:

  1. Toccare Impostazioni.

  2. Opzionale Toccare la riga Ordinamento predefinito e toccare Data o Nome per ordinare i file per nome file o data di modifica.

  3. Opzionale Toccare la riga Memoria dispositivo e toccare un nuovo limite di memoria per i file memorizzati nella cache. Se è stato superato il limite impostato, i file meno recenti nella cache vengono eliminati finché la dimensione della cache non sarà inferiore al limite specificato. La dimensione predefinita della cache è 1 GB.

     

    Nota:
    La riga Spazio utilizzato mostra la dimensione totale dei file preferiti, non dei file memorizzati nella cache.

     

  4. Toccare il tasto Indietro in Android per tornare alla schermata iniziale.

 

Suggerimento:
È possibile accedere alla schermata Impostazioni dalla schermata Profilo toccando .
