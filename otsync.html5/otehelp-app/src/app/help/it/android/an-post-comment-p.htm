Aggiunta di commenti

Per commentare un post in Feed:

  1. Toccare Feed.

  2. Toccare il post che si desidera commentare.

  3. Toccare Rispondi.

  4. Digitare un commento nella casella di testo.

  5. Opzionale Per segnalare il post a un collega in particolare, utilizzare il simbolo @ per inviargli una menzione. Per ulteriori informazioni sulle menzioni, vedere Menzione di un utente e visualizzazione delle menzioni.

    Per aggiungere un allegato, consultare Aggiunta di un allegato.

  6. Toccare Post.

 

Suggerimento:
Per commentare un post nella posizione originale dell'elemento, consultare Commento su un file o una cartella.
