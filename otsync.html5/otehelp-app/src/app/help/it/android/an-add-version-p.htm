Aggiunta di un file

 

Per aggiungere una nuova versione utilizzando un file sul proprio dispositivo:

  1. Toccare Enterprise Workspace o Area di lavoro personale e spostarsi al file desiderato.

  2. Tenere premuto il file di cui si desidera aggiungere una nuova versione.

    Impossibile aggiungere una nuova versione a un file quando viene riservato da un altro collega in Enterprise Workspace. I file riservati sono contrassegnati con l'overlay . Per stabilire chi ha riservato il file, rivedere la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

    La capacità di aggiungere una versione a un file in Enterprise Workspace dipende dalle autorizzazioni per la cartella selezionata.

  3. Toccare Aggiungi versione, quindi toccare Da dispositivo.

  4. Passare alla nuova versione del file sul dispositivo.

  5. Toccare il file.

Per caricare un file memorizzato sul dispositivo:

  1. Toccare Enterprise Workspace o Area di lavoro personale e spostarsi alla cartella in cui si desidera caricare un file.

  2. Tenere premuta la cartella di destinazione.

    Impossibile caricare un file in una cartella che dispone di categorie e attributi obbligatori utilizzando l'applicazione OpenText ECM Everywhere. Per determinare se la cartella dispone di categorie o attributi obbligatori, visualizzare i dettagli dell'oggetto. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

    La capacità di caricare un file in una cartella in Enterprise Workspace dipende dalle autorizzazioni per la cartella selezionata.

  3. Toccare Carica qui ed eseguire una delle operazioni seguenti:

    • Toccare Da fotocamera per scattare una foto con il telefono e caricarla nella cartella selezionata. Le foto vengono salvate come file .jpg. Non è possibile modificare il nome della foto prima di caricarla.

    • Toccare Da dispositivo per caricare un file memorizzato sul proprio dispositivo. Spostarsi nella posizione del dispositivo Android in cui è memorizzato il file e toccare il file.

    • Toccare Da galleria per caricare una foto dalla galleria di immagini del dispositivo. Spostarsi nella galleria di immagini del dispositivo e toccare un'immagine.

Per importare un file da un'altra applicazione:

  1. In un'applicazione di terze parti, spostarsi al file da aggiungere.

  2. Toccare Condividi o Apri in. Viene visualizzato un elenco di applicazioni.

  3. Toccare Everywhere 10.

  4. Opzionale Se richiesto, effettuare l'accesso a OpenText ECM Everywhere.

  5. Toccare Enterprise Workspace o Area di lavoro personale e spostarsi nella posizione in cui si desidera aggiungere il file. La capacità di importare un file in una cartella in Enterprise Workspace dipende dalle autorizzazioni per la cartella selezionata.

    Impossibile importare un file in una cartella che dispone di categorie e attributi obbligatori utilizzando l'applicazione OpenText ECM Everywhere. Per determinare se la cartella dispone di categorie o attributi obbligatori, visualizzare i dettagli dell'oggetto. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

  6. Toccare Incolla.
