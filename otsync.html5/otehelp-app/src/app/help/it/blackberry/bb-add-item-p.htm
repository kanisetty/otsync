Per aggiungere una nuova versione utilizzando un file sul BlackBerry:

  1. Nella pagina Home, fare clic su Enterprise Workspace o Area di lavoro personale e passare al file di destinazione.

  2. Evidenziare il file a cui si desidera aggiungere una nuova versione.

  3. Premere il tasto Menu del BlackBerry e selezionare Carica nuova versione.

  4. Posizionarsi sulla nuova versione del file sul BlackBerry.

  5. Evidenziare il file.

  6. Premere il tasto Menu del BlackBerry e selezionare Carica. La capacità di aggiungere una versione a un file in Enterprise Workspace dipende dalle autorizzazioni per la cartella selezionata.

Per caricare un file memorizzato sul BlackBerry:

  1. In Enterprise Workspace o Area di lavoro personale, passare alla cartella di destinazione in cui si desidera caricare un file.

  2. Premere il tasto Menu del BlackBerry e selezionare Carica qui.

  3. Posizionarsi sul file desiderato nel proprio smartphone BlackBerry. La capacità di caricare un file in una cartella in Enterprise Workspace dipende dalle autorizzazioni per la cartella selezionata.

  4. Premere il tasto Menu del BlackBerry e selezionare Carica.

Per importare un file da un'applicazione del BlackBerry:

  1. Avviare l'applicazione del BlackBerry e passare alla cartella di destinazione che si desidera importare.

  2. Premere il tasto Menu del BlackBerry e selezionare Aggiungi a OpenText Everywhere.

  3. Opzionale Se richiesto, effettuare l'accesso a ECM Everywhere. Per ulteriori dettagli, vedere Accesso e gestione del proprio profilo.

  4. Fare clic su Enterprise Workspace o Area di lavoro personale.

  5. Spostarsi nel percorso in cui si desidera aggiungere questo file. La capacità di aggiungere un file in una cartella in Enterprise Workspace dipende dalle autorizzazioni per la cartella selezionata.

  6. Premere il tasto Menu del BlackBerry e selezionare Carica qui.
