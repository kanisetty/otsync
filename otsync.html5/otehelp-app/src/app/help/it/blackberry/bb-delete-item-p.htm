Per eliminare un file o una cartella:

  1. Nella pagina Home, fare clic su Enterprise Workspace o Area di lavoro personale e spostarsi nel file o cartella di destinazione.

     

    Nota:
    Prima di continuare, fare riferimento alla sezione Attenzione riportata di seguito.

     

  2. Evidenziare il file o la cartella di destinazione.

  3. Premere il tasto Menu del BlackBerry e selezionare Elimina elemento o Elimina cartella.

  4. Fare clic su OK.

 

Cautela
Se si elimina un file o una cartella, il file viene rimosso da Enterprise Workspace o Area di lavoro personale anche se era stato contrassegnato come preferito. Questo significa che il file o la cartella verranno eliminati da Content Server.
