Per aggiungere un preferito:

  1. Nella pagina Home, fare clic su Enterprise Workspace o Area di lavoro personale e spostarsi nel file o cartella di destinazione.

  2. Evidenziare il file o la cartella.

  3. Premere il tasto Menu del BlackBerry e selezionare Aggiungi a preferiti.

    Il file o la cartella è contrassegnato con l'overlay . Ad esempio, . Grazie a questa impostazione, è possibile accedere rapidamente al file o alla cartella dall'applicazione Preferiti.

  4. Opzionale Se viene visualizzato l'overlay il file non è stato scaricato. Ad esempio, . Per scaricare il file, aprirlo o sincronizzare i preferiti. Per ulteriori dettagli, vedere Sincronizzazione manuale dei Preferiti.

    Contrassegnando una cartella come un preferito, i file e le sottocartelle in essa contenuti non vengono sincronizzati. Di conseguenza, l'overlay viene sempre visualizzato.

Per ulteriori informazioni sui preferiti, fare riferimento alle altre sezioni in Aggiunta, visualizzazione, rimozione e sincronizzazione dei preferiti .
