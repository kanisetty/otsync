Per modificare il proprio profilo:

  1. Nella schermata Profilo, evidenziare il profilo.

     

    Suggerimento:
    Se è già stato effettuato l'accesso, premere il tasto Menu sul BlackBerry e selezionare Scambia profilo per tornare alla schermata Profilo.

     

  2. Premere il tasto Menu del BlackBerry e selezionare Modifica profilo.

  3. Modificare il nome profilo, il nome utente o il server come richiesto.

  4. Fare clic su Salva profilo.

Per aggiornare la propria password:

  1. Premere il tasto Menu del BlackBerry e selezionare Uscita. Per cambiare la password è necessario essere online.

  2. Nella schermata Profilo, evidenziare il proprio profilo e premere il tasto Trackpad del BlackBerry.

  3. Digitare la nuova password. La password è sensibile alle maiuscole/minuscole. Assicurarsi di digitare la password correttamente.

  4. Premere il tasto Invio del BlackBerry.
