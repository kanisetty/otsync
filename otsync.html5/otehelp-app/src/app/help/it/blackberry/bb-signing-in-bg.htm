Per effettuare l'accesso:

  1. Per accedere all'applicazione è prima necessario creare un profilo. Se non si dispone di un profilo, attenersi alla seguente procedura:

    1. Nella pagina Profilo, evidenziare Nuovo profilo e premere il tasto Trackpad del BlackBerry. Per creare un nuovo profilo ed eseguire l'accesso è necessario essere online.

    2. Digitare un nome di profilo univoco nella casella Nome utente.

    3. Digitare il proprio nome utente Content Server nella casella Nome utente.

    4. Digitare la propria password Content Server nella casella Password. La password è sensibile alle maiuscole/minuscole. Assicurarsi di digitare la password correttamente.

    5. Digitare l'URL del server ECM Everywhere (OpenText AppWorks Gateway) nella casella Server.

      Contattare l'amministratore per conoscere l'URL del server ECM Everywhere.

    6. Selezionare la casella di controllo Mantieni connessione per consentire l'accesso immediato all'avvio successivo dell'applicazione. Le uniche eccezioni si presentano se l'utente esce da ECM Everywhere o se scade la sessione. Vedere la sezione Attenzione di seguito relativa a questa opzione.

    7. Fare clic su Salva e accedi.

  2. Se si dispone già di un profilo, attenersi alla seguente procedura:

    1. Nella pagine Profilo, evidenziare il proprio profilo e premere il tasto Trackpad del BlackBerry.

    2. Se richiesto, digitare la password. La password è sensibile alle maiuscole/minuscole. Assicurarsi di digitare la password correttamente.

    3. Selezionare la casella di controllo Mantieni connessione per consentire l'accesso immediato all'avvio successivo dell'applicazione. Le uniche eccezioni si presentano se l'utente esce dall'applicazione o se scade la sessione. Vedere la sezione Attenzione di seguito relativa a questa opzione.

    4. Premere il tasto Invio del BlackBerry.

 

Cautela
Consentire a ECM Everywhere di mantenere la connessione può rappresentare un rischio per la sicurezza. Se il dispositivo BlackBerry viene utilizzato da un altro utente senza aver effettuato l'uscita dall'applicazione ECM Everywhere, l'altro utente può accedere all'account Content Server dell'utente precedente. Per uscire, passare alla pagina Home, premere il tasto Menu del BlackBerry e selezionare Uscita.

 

Per ulteriori informazioni sulla gestione del proprio profilo, fare riferimento alle altre sezioni in Accesso e gestione del proprio profilo .
