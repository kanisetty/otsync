Per modificare le impostazioni dell'applicazione:

  1. Nella pagina Home, fare clic su Impostazioni.

  2. Opzionale Evidenziare l'elenco Memoria del dispositivo, premere il tasto Trackpad del BlackBerry e regolare lo spazio di archiviazione per i file nella cache. Se è stato superato il limite impostato, i file meno recenti nella cache vengono eliminati finché la dimensione della cache non sarà inferiore al limite specificato. La dimensione predefinita della cache è 10 MB.

    La riga Spazio utilizzato mostra la dimensione totale dei file preferiti, non dei file memorizzati nella cache.

  3. Opzionale Evidenziare l'elenco Ordinamento predefinito, premere il tasto Trackpad del BlackBerry e selezionare Data o Nome per ordinare il contenuto della cartella in base alla data di modifica o al nome degli elementi.

  4. Premere il tasto Esc del BlackBerry e selezionare Salva.
