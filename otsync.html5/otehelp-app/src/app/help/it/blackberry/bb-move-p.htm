Per spostare o copiare un file o una cartella:

  1. Nella pagina Home, fare clic su Enterprise Workspace o Area di lavoro personale e spostarsi nel file o cartella di destinazione.

  2. Evidenziare il file o la cartella che si desidera spostare o copiare.

  3. Premere il tasto Menu del BlackBerry e selezionare Taglia per spostare l'elemento o Copia per copiare l'elemento.

  4. Passare alla cartella di destinazione dove si desidera inserire il contenuto degli appunti.
