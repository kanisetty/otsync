Per visualizzare file offline:

  1. Per rendere un file disponibile per la visualizzazione offline, contrassegnarlo come preferito e sincronizzarlo o scaricarlo mentre ci si trova online. Grazie a questa impostazione, è possibile accedere rapidamente al file dall'applicazione Preferiti. Per ulteriori dettagli, vedere Aggiunta, visualizzazione, rimozione e sincronizzazione dei preferiti.

    Contrassegnando una cartella come un preferito, i file e le sottocartelle in essa contenuti non vengono sincronizzati. Di conseguenza, il contenuto all'interno delle cartelle non è disponibile in modalità offline.

    I file contrassegnati come preferiti sono indicati dall'overlay . Ad esempio, .

    Se viene visualizzato l'overlay quando si è online, ad esempio , è necessario aprire il file o sincronizzare i preferiti per rendere questo file disponibile per la visualizzazione offline. L'overlay viene sempre visualizzato su cartelle.

  2. Nella pagina Home, fare clic su Preferiti ed evidenziare il file desiderato.

  3. Premere il tasto Trackpad del BlackBerry per visualizzare il file.

    I file che si aprono quando si è offline potrebbero non essere le versioni più recenti, in quanto non è possibile sincronizzare con il dispositivo eventuali modifiche apportate ai file utilizzando un client che è online.

  4. Premere il tasto Esc del BlackBerry per tornare ai propri file.
