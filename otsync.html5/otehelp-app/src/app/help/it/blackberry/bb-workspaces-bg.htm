Per visualizzare un elemento:

  1. Nella pagina Home, fare clic su Enterprise Workspace o Area di lavoro personale e passare al file di destinazione evidenziando una cartella e premendo il tasto Trackpad del BlackBerry. Sulla barra del titolo della cartella è visualizzato il nome della cartella visualizzata.

  2. Opzionale Per assicurarsi di visualizzare sempre i contenuti più recenti, aggiornare periodicamente i contenuti della cartella. Premere il tasto Menu del BlackBerry e selezionare Aggiorna. Questa operazione consente di aggiornare la visualizzazione, ma non di scaricare i file. Per scaricare un file, aprirlo.

  3. Evidenziare il file che si desidera visualizzare.

  4. Premere il tasto Trackpad del BlackBerry per visualizzare il file.

  5. Premere il tasto Esc del BlackBerry per tornare ai propri file.

Per visualizzare un nome file lungo:

  1. In Enterprise Workspace o Area di lavoro personale, spostarsi nel percorso dell'elemento ed evidenziare l'elemento da visualizzare.

  2. Premere il tasto Menu del BlackBerry e selezionare Dettagli oggetto. Viene visualizzato il nome file completo.

Per modificare un file:

  1. Fare clic su Enterprise Workspace o Area di lavoro personale e passare al file di destinazione.

  2. Scaricare il file nel dispositivo BlackBerry. Per ulteriori dettagli, vedere Download di un file.

  3. Aprire il file in un'applicazione di terze parti che supporta la modifica. Modificare il file e salvare le modifiche.

  4. Le modifiche apportate al file in un'applicazione di terze parti non vengono salvate nell'area di lavoro. È necessario caricare il file modificato dall'applicazione di terze parti o dalla cartella della cache del BlackBerry.

    Nell'applicazione ECM Everywhere, caricare il file modificato aggiungendo una versione del file. Per ulteriori dettagli, vedere Aggiunta di un file.

Per ulteriori informazioni sull'utilizzo dei file, fare riferimento alle altre sezioni in Visualizzazione, modifica e lavoro sui file.
