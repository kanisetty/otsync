Aggiunta, visualizzazione e rimozione di preferiti

Per aggiungere un file o una cartella preferita:

  1. Toccare Area di lavoro Enterprise o Area di lavoro personale e spostarsi al file o alla cartella desiderata.

  2. Toccare .

  3. Toccare Aggiungi a Preferiti. Il file o la cartella è contrassegnato con l'overlay nella vista cartelle. Grazie a questa impostazione, è possibile accedere rapidamente al file o alla cartella dall'applicazione Preferiti. I file e le sottocartelle all'interno di una cartella non vengono contrassegnati come preferiti.

    Nel proprio profilo è memorizzato l'elenco dei file preferiti, specifico per iPad o iPhone. È necessario contrassegnare file come preferiti su ogni smartphone e tablet in cui viene eseguita l'applicazione OpenText ECM Everywhere.

  4. Toccare Indietro per tornare ai file.

Per aggiungere un preferito mentre lo si sta visualizzando:

  1. Toccare Area di lavoro Enterprise o Area di lavoro personale e spostarsi al file desiderato.

  2. Toccare il file che si desidera contrassegnare come preferito. Il file viene aperto.

  3. Toccare . L'icona diventa e il file viene contrassegnato con l'immagine sovrapposta nella vista cartelle. Ad esempio, . Tramite questa operazione il file viene scaricato sul dispositivo iPad o iPhone, in modo da poterlo visualizzare quando si è offline.

  4. Toccare Indietro.

Per visualizzare un preferito:

  1. Toccare Preferiti.

  2. Opzionale Toccare per assicurarsi di visualizzare i contenuti più recenti.

  3. Toccare il file che si desidera visualizzare.

  4. Toccare Indietro per tornare ai file.

Per rimuovere un preferito:

  1. Toccare Preferiti oppure Area di lavoro organizzazione o Area di lavoro personale e spostarsi nel file o nella cartella desiderata.

  2. Toccare corrispondente all'elemento, quindi toccare Rimuovi preferito. Il file non viene più visualizzato nell'elenco dei preferiti. Con questa operazione il file o la cartella non vengono eliminati dall'area di lavoro.

  3. Toccare Indietro per tornare ai file.
