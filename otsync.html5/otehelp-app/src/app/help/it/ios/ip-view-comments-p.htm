Visualizzazione dei commenti su un file o una cartella

 

Per visualizzare i commenti su un file o una cartella:

  1. Toccare Area di lavoro Enterprise o Area di lavoro personale e spostarsi al file o alla cartella desiderata.

  2. Toccare .

  3. Toccare Commenti da qui.

  4. Toccare per tornare alla schermata dei dettagli cartella.

  5. Toccare Indietro per tornare ai file.
