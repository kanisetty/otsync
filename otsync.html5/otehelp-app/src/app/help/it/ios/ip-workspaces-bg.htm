Visualizzazione, modifica e lavoro sui file

Per visualizzare un elemento:

  1. Toccare Area di lavoro Enterprise o Area di lavoro personale e spostarsi al file desiderato. Assicurarsi di toccare il nome della cartella e non , che visualizza i dettagli della cartella. Sulla barra del titolo della cartella è visualizzato il nome della cartella visualizzata.

  2. Opzionale Toccare per assicurarsi di visualizzare i contenuti più recenti. Con questa operazione viene aggiornata la visualizzazione, senza scaricare i file. Per scaricare un file, aprirlo.

  3. Toccare il nome dell'elemento che si desidera aprire. Assicurarsi di toccare il nome file e non , che visualizza le proprietà dell'elemento.

  4. Toccare Indietro per tornare ai file.

Per modificare un elemento:

  1. Toccare Area di lavoro Enterprise o Area di lavoro personale e spostarsi all'elemento desiderato.

    Impossibile salvare le modifiche a un file quando viene riservato da un altro collega nell'Area di lavoro organizzazione. I file riservati sono contrassegnati con l'overlay . Per stabilire chi ha riservato il file, è possibile verificare la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione dei dettagli dell'oggetto.

  2. Toccare il file che si desidera modificare. Il file viene aperto.

  3. Toccare e selezionare un editor.

  4. Modificare il file e salvare le modifiche.

  5. Nell'editor, toccare e selezionare Apri in Everywhere 10.

  6. In OpenText ECM Everywhere toccare Carica nuova versione. Le modifiche vengono aggiunte al file originale.

     

    Nota:
    Se si utilizza la funzionalità di ricerca nell'applicazione OpenText ECM Everywhere per trovare il file da modificare, tutte le modifiche apportate al file in un editor verranno salvate in un nuovo file e inserite a livello di radice.

     

Per ulteriori informazioni sull'utilizzo dei file, fare riferimento agli altri argomenti in Visualizzazione, modifica e lavoro sui file.
