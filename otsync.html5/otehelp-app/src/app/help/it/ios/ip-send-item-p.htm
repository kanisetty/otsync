Apertura di un file in un'altra applicazione

Per aprire un file in un'altra applicazione:

  1. Toccare Area di lavoro Enterprise o Area di lavoro personale e spostarsi al file desiderato.

  2. Toccare il file che si desidera esportare. Il file viene aperto.

  3. Toccare . Viene visualizzato un elenco delle applicazioni che sono in grado di aprire il file.

  4. Toccare un'applicazione.
