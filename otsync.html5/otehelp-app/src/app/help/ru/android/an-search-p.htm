Поиск OpenText ECM Everywhere

 

Поиск папки OpenText:

  1. Коснитесь Enterprise Workspace или Личная рабочая область и перейдите к целевой папке, с которой вы хотите начать поиск.

  2. Коснитесь Меню и выберите Поиск.

  3. В строке поиска введите данные для поиска.

  4. Нажмите Перейти. Отобразится список результатов поиска.

  5. Выполните одно из следующих действий.

    • Нажмите файл или папку, которую необходимо просмотреть.

    • Нажмите файл (папку) и удерживайте его, чтобы отобразить доступные параметры.

  6. Нажмите клавишу Android Назад, чтобы вернуться к результатам поиска.

Открытие местоположения файла в результатах поиска:

  1. Выполните поиск папки OpenText.

  2. Нажмите и удерживайте файл (папку), который следует открыть в местоположении файла.

  3. Нажмите Открыть местоположение.
