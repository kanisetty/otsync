Изменение изображения профиля

Изменение изображения профиля:

  1. На OpenText ECM Everywhere начальном экране коснитесь Настройки.

  2. Нажмите строку Фотография профиля и выполните одно из следующих действий.

    • Нажмите С камеры, чтобы задать изображение профиля, используя камеру Android. Сделайте фотографию, а затем следуйте инструкциям на экране, чтобы сохранить ее.

    • Нажмите Из галереи, чтобы задать изображение профиля, используя фотографию из галереи Android. Перейдите к нужному изображению и коснитесь его.

    • Нажмите Задать по умолчанию, чтобы задать изображение профиля по умолчанию.

  3. Нажмите клавишу Android Назад, чтобы вернуться к экрану Настройки.

  4. Коснитесь клавиши Android Назад, чтобы вернуться к начальному экрану.
