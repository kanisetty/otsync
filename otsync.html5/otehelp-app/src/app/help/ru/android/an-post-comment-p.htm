Добавление комментариев

 

Как добавить комментарий к сообщению в веб-каналах:

  1. Коснитесь Веб-каналы.

  2. Коснитесь сообщения, к которому необходимо добавить комментарий.

  3. Коснитесь Ответить.

  4. Введите в текстовое поле комментарий.

  5. По выбору Чтобы обратить внимание определенного сотрудника на ваше сообщение, используйте @ упоминание. Дополнительные сведения об упоминаниях см. в разделе Упоминание пользователя и просмотр упоминаний.

    Сведения о добавлении вложения см. в разделе Добавление вложения.

  6. Коснитесь Сообщение.

 

Подсказка:
Сведения о добавлении комментария к сообщению в местоположении элемента см. в разделе Добавление комментариев к файлу или папке.
