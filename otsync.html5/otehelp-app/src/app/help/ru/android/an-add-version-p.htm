Добавление файла

 

Добавление новой версии с помощью файла на устройстве:

  1. Коснитесь Enterprise Workspace или Личная рабочая область и перейдите к целевому файлу.

  2. Коснитесь файла, к которому нужно добавить новую версию, и удерживайте его.

    Нельзя добавить новую версию в файл, если он зарезервирован другим сотрудников в Enterprise Workspace. Зарезервированные файлы помечены наложением. Чтобы узнать, кем зарезервирован файл, загляните в журнал аудита файла. Подробнее см. в разделе Просмотр сведений об объекте.

    Возможность добавить версию в файл в Enterprise Workspace зависит от ваших разрешений для выбранной папки.

  3. Коснитесь Добавить версию, затем коснитесь Из галереи устройства.

  4. Перейдите к новой версии файла на устройстве.

  5. Выберите файл.

Отправка файла, сохраненного на устройстве:

  1. Коснитесь Enterprise Workspace или Личная рабочая область и перейдите к целевой папке, в которую необходимо передать файл.

  2. Коснитесь целевой папки и удерживайте ее.

    При помощи приложения OpenText ECM Everywhere нельзя передать файл в папку с обязательными категориями и атрибутами. Чтобы узнать, нет ли у папки обязательных категорий и атрибутов, взгляните на сведения об объекте. Подробнее см. в разделе Просмотр сведений об объекте.

    Возможность передать файл в папку в Enterprise Workspace зависит от ваших разрешений для выбранной папки.

  3. Коснитесь Загрузить и затем выполните одно из следующих действий.

    • Коснитесь С камеры, чтобы сделать фотографию при помощи телефона, и загрузите ее в целевую папку. Фотографии сохраняются в формате JPG. Нельзя изменить название файла фотографии до отправки.

    • Коснитесь С устройства, чтобы загрузить файл, сохраненный на устройстве. Перейдите к целевому файлу на устройстве Android и коснитесь его.

    • Коснитесь Из галереи, чтобы загрузить изображение из фотогалереи устройства. Перейдите к галерее устройства и коснитесь изображения.

Импорт файла из другого приложения:

  1. Просмотрите файл, который требуется добавить из стороннего приложения.

  2. Коснитесь Общий доступ или Открыть в. Откроется список приложений.

  3. Коснитесь Everywhere 10.

  4. По выбору При отображении соответствующего запроса войдите в OpenText ECM Everywhere.

  5. Коснитесь Enterprise Workspace или Личная рабочая область и перейдите к местоположению, в которое необходимо добавить этот файл. Возможность импортировать файл в папку в Enterprise Workspace зависит от ваших разрешений для выбранной папки.

    Нельзя импортировать файл в папку с обязательными категориями и атрибутами при помощи приложения OpenText ECM Everywhere. Чтобы узнать, нет ли у папки обязательных категорий и атрибутов, взгляните на сведения об объекте. Подробнее см. в разделе Просмотр сведений об объекте.

  6. Нажмите Вставить.
