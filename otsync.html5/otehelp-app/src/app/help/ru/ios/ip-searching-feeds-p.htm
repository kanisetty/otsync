Поиск по веб-каналам

Как выполнять поиск по веб-каналам:

  1. Коснитесь Веб-каналы. Поиск выполняется по активным веб-каналам. Чтобы определить активный веб-канал, просмотрите строки непосредственно под клавишами Поиск, Фильтр и Статус.

  2. По выбору Чтобы изменить веб-канал, по которому выполняется поиск, коснитесь Фильтр и выберите параметры фильтра. Подробнее см. в разделе Фильтрация, просмотр, комментирование и поиск обновлений в веб-каналах.

  3. Коснитесь Поиск.

  4. Коснитесь строки поиска и введите данные для поиска. Поиск выполняется по всем опубликованным обновлениям. Вы не можете выполнять поиск по пользователям или вложениям.

  5. Коснитесь Поиск.
