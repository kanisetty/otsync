Запуск рабочего процесса

Как запустить рабочий процесс:

  1. Коснитесь Корпоративная рабочая область или Личная рабочая область и перейдите к целевому элементу.

  2. Коснитесь имени карты рабочего процесса, который необходимо запустить. Отобразится сводка карты рабочего процесса, где можно изменять атрибуты, просматривать вложения и добавлять комментарии до запуска рабочего процесса.

    Требуемые атрибуты помечены звездочкой. Кнопка Запустить не доступна, пока требуемые атрибуты не заполнены.

    Назначения рабочего процесса, просматриваемые в OpenText ECM Everywhere, не поддерживают теги HTML в инструкциях или областях комментариев. Если текст не визуализируется должным образом, удалите теги HTML из карты рабочего процесса в Content Server.

  3. По выбору Если доступно, измените атрибуты. Дополнительные сведения см. в разделе Просмотр, редактирование и работа с назначениями.

  4. По выбору Введите комментарии для адресата назначения шага в поле Комментарии.

  5. Коснитесь Запустить.
