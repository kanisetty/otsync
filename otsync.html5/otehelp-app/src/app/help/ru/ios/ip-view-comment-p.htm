Просмотр комментариев

 

Как просмотреть сообщение с длинным комментарием:

  1. Коснитесь Веб-каналы.

  2. Коснитесь сообщения, чтобы просмотреть сведения о длинном комментарии.

Как просмотреть все комментарии к сообщению:

  1. Коснитесь Веб-каналы. Отобразятся сообщения с несколькими комментариями , а также количество добавленных комментариев.

  2. Коснитесь сообщения.

  3. Коснитесь ссылки Просмотреть темы для просмотра обновления с несколькими комментариями.

  4. Нажмите для возврата к сообщению.

 

Подсказка:
Сведения о просмотре комментариев в местоположении элемента см. в разделе Просмотр комментариев к файлу или папке.
