Добавление файла

 

Как добавить элемент:

  1. Необходимо импортировать элемент, загрузив его из другого приложения, если приложение поддерживает эту функцию. В стороннем приложении просмотрите файл, который хотите импортировать, и выполните необходимые действия, чтобы открыть документ в OpenText ECM Everywhere. Например, коснитесь , Общий доступ или Открыть в. Откроется список приложений.

  2. Коснитесь Everywhere 10.

  3. По выбору При отображении соответствующего запроса войдите в OpenText ECM Everywhere.

  4. Коснитесь Назначение: Личная рабочая область, чтобы перейти к местоположению, куда необходимо добавить этот файл, и коснитесь Выбрать.

  5. По выбору Коснитесь поля Имя файла и измените имя файла. Коснитесь Готово.

  6. Коснитесь Передать.
