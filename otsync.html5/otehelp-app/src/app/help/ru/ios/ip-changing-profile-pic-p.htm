Изменение изображения профиля

 

Изменение изображения профиля:

  1. На начальном экране коснитесь Настройки.

  2. Коснитесь Нажмите для изменения фотографии.

  3. Выполните одно из следующих действий.

    • Нажмите С камеры, чтобы задать изображение профиля, используя камеру Apple.

      Сделайте снимок и коснитесь Использовать фотографию, чтобы использовать текущее изображение, или коснитесь Повторно создать и снова сделайте снимок.

    • Коснитесь Из альбома, чтобы настроить изображение профиля, используя фотографию из фотоальбома Apple.

      Перейдите к необходимому изображению и коснитесь его.

    • Нажмите Задать по умолчанию, чтобы задать изображение профиля по умолчанию.

  4. Коснитесь Настройки.
