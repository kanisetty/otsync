Изменение настроек приложения:

  1. На странице Главная щелкните Настройки.

  2. По выбору Выделите список Хранение, нажмите клавишу BlackBerry Сенсорная панель и настройте параметры пространства для хранения кэшированных файлов. В случае превышения заданного ограничения для хранения данных старые файлы кэша будут удаляться, пока размер кэша не будет ниже указанного ограничения. По умолчанию размер кэша составляет 10 МБ.

    В строке Использованное пространство отображается общий размер избранных, а не кэшированных файлов.

  3. По выбору Выделите список Порядок сортировки по умолчанию, нажмите клавишу BlackBerry Сенсорная панель и выберите Дата или Имя для сортировки содержимого папки по дате изменения или именам элементов.

  4. Нажмите клавишу BlackBerry Выход и выберите Сохранить.
