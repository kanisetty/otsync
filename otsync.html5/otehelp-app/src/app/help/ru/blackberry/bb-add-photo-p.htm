Добавление фотографии с камеры BlackBerry:

  1. На странице Главная щелкните Enterprise Workspace или Личная рабочая область и перейдите к целевой папке, в которую необходимо добавить фотографию. Возможность добавить фотографию в папку в Enterprise Workspace зависит от ваших разрешений для выбранной папки.

  2. Нажмите клавишу BlackBerry Меню и выберите Добавить фотографию.

  3. Нажмите клавишу BlackBerry Сенсорная панель, чтобы сделать фотографию.

  4. Нажмите Добавить в OpenText Everywhere. Фотографии сохраняются в формате JPG. Нельзя изменить название файла фотографии до отправки.

Как добавить фотографию из приложения BlackBerry:

  1. Запустите приложение камеры BlackBerry.

  2. Перейдите к фотографии, которую следует импортировать.

  3. Нажмите клавишу BlackBerry Меню и выберите Добавить в OpenText Everywhere.

  4. По выбору При отображении соответствующего запроса войдите в ECM Everywhere. Подробнее см. в разделе Вход и управление профилем.

  5. Щелкните Enterprise Workspace или Личная рабочая область.

  6. Перейдите к местоположению, в которое необходимо добавить фотографию. Возможность добавить фотографию в папку в Enterprise Workspace зависит от ваших разрешений для выбранной папки.

  7. Нажмите клавишу BlackBerry Меню и выберите Передать.

Добавление папки:

  1. В Enterprise Workspace или Личная рабочая область перейдите к целевому местоположению для новой папки. Возможность добавить папку в Enterprise Workspace зависит от ваших разрешений для выбранной папки.

  2. Нажмите клавишу BlackBerry Меню и выберите Добавить папку.

  3. В поле Имя папки введите имя папки.

  4. Нажмите клавишу BlackBerry Ввод.
