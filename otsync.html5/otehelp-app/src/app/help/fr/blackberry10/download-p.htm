Télécharger un fichier vers votre appareil

 

Pour télécharger un élément :

  1. Appuyez sur Enterprise Workspace ou sur Espace de travail personnel, puis accédez au fichier cible. Si le fichier cible est marqué comme favori, appuyez sur Favoris.

  2. Maintenez votre doigt sur le fichier que vous souhaitez télécharger.

  3. Appuyez sur Télécharger.

  4. Accédez au dossier cible sur votre périphérique BlackBerry.

  5. Appuyez sur Télécharger.
