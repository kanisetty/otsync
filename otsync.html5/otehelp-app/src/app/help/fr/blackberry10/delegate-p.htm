Délégation d'une étape

Pour déléguer une étape :

  1. Appuyez sur Tâches assignées.

  2. Appuyez sur une tâche de workflow dans votre liste de tâches assignées.

  3. Appuyez sur Déléguer.

  4. Appuyez sur le champ Utilisateur avec une tâche assignée.

     

    Note :
    Le champ Utilisateur avec une tâche assignée doit être renseigné. Le bouton Envoyer pour vérification ne devient actif qu'une fois ce champ renseigné.

     

  5. Saisissez le nom de l'utilisateur Content Server auquel vous souhaitez envoyer cette étape de workflow.

  6. Appuyez sur le nom de votre collègue dans la liste des résultats pour le sélectionner.

  7. Appuyez sur Déléguer.
