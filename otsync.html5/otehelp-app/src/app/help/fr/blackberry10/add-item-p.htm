Ajout d'un fichier

Pour ajouter une nouvelle version à l'aide d'un fichier sur votre BlackBerry :

  1. Appuyez sur Enterprise Workspace ou sur Espace de travail personnel, puis accédez au fichier cible.

  2. Maintenez votre doigt sur le fichier auquel vous souhaitez ajouter une nouvelle version.

    Vous ne pouvez pas ajouter une nouvelle version à un fichier si celui-ci est réservé par un autre collègue dans Enterprise Workspace. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vérifiez l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

    Pour pouvoir ajouter une version à un fichier dans Enterprise Workspace, vous devez disposer des autorisations correspondantes pour le dossier sélectionné.

  3. Appuyez sur Ajouter une version.

  4. Accédez à la nouvelle version du fichier sur votre BlackBerry.

  5. Appuyez sur le fichier.

Pour charger un fichier stocké sur votre BlackBerry :

  1. Depuis Enterprise Workspace ou Espace de travail personnel, accédez au dossier cible dans lequel vous souhaitez charger un fichier.

  2. Maintenez votre doigt sur le dossier cible.

    Vous ne pouvez pas charger de fichier dans un dossier qui a des attributs et des catégories obligatoires via l'application ECM Everywhere. Pour déterminer si le dossier a des attributs ou des catégories obligatoires, affichez les détails de l'objet. Pour plus de détails, voir Afficher les détails de l'objet.

  3. Appuyez sur Charger ici, puis effectuez l'une des opérations suivantes :

    • Appuyez sur Depuis un appareil photo pour prendre une photo à l'aide de votre téléphone et la charger dans le dossier cible. Les photos sont enregistrées en tant que fichiers .jpg. Il n'est pas possible de modifier le nom de la photo avant de la télécharger.

    • Appuyez sur Depuis un appareil pour charger un fichier stocké sur votre appareil. Accédez au fichier cible sur votre appareil BlackBerry, puis appuyez sur le fichier.

    • Appuyez sur Depuis une galerie pour joindre une photo depuis la galerie de photos Android. Accédez à la galerie de photos, puis appuyez sur l'image.
