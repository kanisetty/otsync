Visualisation, modification et utilisation de vos fichiers

Pour afficher un objet :

  1. Dans Enterprise Workspace ou Espace de travail personnel, accédez à l'élément cible.

  2. Facultatif Pour être sûr de visualiser le contenu le plus récent, actualisez régulièrement le contenu du dossier. Balayer vers le bas sur la liste des dossiers. L'affichage est actualisé, mais les fichiers ne sont pas téléchargés. Pour télécharger un fichier, ouvrez-le.

  3. Appuyez sur le fichier à visualiser.

  4. Pour revenir à vos fichiers, appuyez sur Retour.

Pour modifier un fichier :

  1. Appuyez sur Enterprise Workspace ou sur Espace de travail personnel, puis accédez au fichier cible. Pour modifier un fichier, vous devez l'exporter dans une application tierce, puis importer le fichier modifié dans ECM Everywhere.

    Vous ne pouvez pas enregistrer les modifications que vous effectuez dans un fichier si celui-ci est réservé par un autre collègue dans Enterprise Workspace. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vérifiez l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

  2. Maintenez le doigt sur l'élément.

  3. Appuyez sur Envoyer par e-mail / Exporter.

  4. Appuyez sur une application tierce.

  5. Dans l'application tierce, modifiez le fichier et enregistrez les modifications sur l'appareil.

  6. ECM Everywhere vous permet désormais d'ajouter une nouvelle version au fichier d'origine. Pour plus de détails, voir Ajout d'un fichier.

Pour plus d'informations sur la gestion de vos fichiers, reportez-vous aux autres rubriques dans Visualisation, modification et utilisation de vos fichiers.
