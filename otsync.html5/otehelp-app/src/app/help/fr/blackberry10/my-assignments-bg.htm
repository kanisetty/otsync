Visualisation, modification et utilisation de vos tâches assignées

Pour afficher vos tâches assignées :

  1. Appuyez sur Tâches assignées, puis sur un workflow. Appuyez sur le fichier à visualiser. Les tâches assignées d'un workflow affichées dans l'application ne prennent pas en charge les balises HTML dans les zones d'instructions ou de commentaires. Si les instructions ne s'affichent pas correctement, supprimez les balises HTML du schéma de workflow dans Content Server.

  2. Modifiez les attributs (le cas échéant).

  3. Appuyez sur Soumettre.

Pour modifier des attributs :

  1. Appuyez sur Tâches assignées, puis sur un workflow. Appuyez sur le fichier à visualiser.

  2. Appuyez sur la ligne d'attribut, puis saisissez les informations requises ou appuyez sur une option de la liste fournie pour la sélectionner.

    Les attributs requis sont marqués d'un astérisque.

  3. Appuyez sur Soumettre.

Pour plus d'informations sur la gestion de vos tâches assignées, reportez-vous aux autres rubriques dans Visualisation, modification et utilisation de vos tâches assignées.
