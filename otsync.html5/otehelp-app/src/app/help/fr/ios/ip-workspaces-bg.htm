Visualisation, modification et utilisation de vos fichiers

Pour afficher un objet :

  1. Appuyez sur Espace de travail professionnel ou sur Espace de travail personnel, puis accédez au fichier cible. Veillez à appuyer sur le nom du dossier et non sur , qui affiche les détails du dossier. La barre de titre affiche le nom du dossier visualisé.

  2. Facultatif Appuyez sur pour être sûr de visualiser le contenu le plus récent. L'affichage est actualisé, mais aucun fichier n'est téléchargé. Pour télécharger un fichier, ouvrez-le.

  3. Appuyez sur le nom de l'élément que vous souhaitez ouvrir. Veillez à appuyer sur le nom du fichier et non sur , qui affiche les propriétés de l'élément.

  4. Pour revenir à vos fichiers, appuyez sur Retour.

Pour modifier un élément :

  1. Appuyez sur Espace de travail professionnel ou sur Espace de travail personnel, puis accédez à l'élément cible.

    Vous ne pouvez pas enregistrer les modifications que vous effectuez dans un fichier si celui-ci est réservé par un autre collègue dans l'espace de travail Entreprise. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vous pouvez vérifier l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

  2. Appuyez sur le fichier à modifier. Le fichier s'ouvre.

  3. Appuyez sur puis sélectionnez un éditeur.

  4. Modifiez le fichier, puis enregistrez les modifications.

  5. Dans l'éditeur, appuyez sur et sélectionnez Ouvrir dans Everywhere 10.

  6. Dans OpenText ECM Everywhere, appuyez sur Charger une nouvelle version . Les modifications sont ajoutées au fichier original.

     

    Note :
    Si vous utilisez la fonction de recherche dans l'application OpenText ECM Everywhere pour rechercher le fichier que vous souhaitez modifier, les modifications effectuées dans le fichier à l'aide d'un éditeur seront enregistrées dans un nouveau fichier, lequel sera placé au niveau racine.

     

Pour plus d'informations sur la gestion de vos fichiers, reportez-vous aux autres rubriques dans Visualisation, modification et utilisation de vos fichiers.
