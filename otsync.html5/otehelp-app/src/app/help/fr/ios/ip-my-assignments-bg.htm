Visualisation, modification et utilisation de vos tâches assignées

 

Pour modifier les attributs de workflow :

  1. Appuyez sur Tâches assignées, puis sur une tâche de workflow.

  2. Pour saisir des valeurs d'attribut, procédez comme suit :

    • Appuyez sur un attribut de date, sélectionnez une date, puis appuyez sur Terminé pour revenir à la tâche de workflow.

    • Appuyez sur un attribut contextuel, sélectionnez une option, puis appuyez sur Retour pour revenir à la tâche de workflow.

    • Appuyez sur un attribut de nombre entier ou réel, saisissez une valeur, puis appuyez sur Retour pour revenir à la tâche de workflow.

      Les attributs requis sont marqués d'un astérisque.

      Le bouton Faire suivre n'est disponible qu'une fois les attributs requis renseignés.

Pour ajouter un collègue à l'attribut Champ utilisateur :

  1. Appuyez sur Tâches assignées, puis sur une tâche de workflow.

  2. Appuyez sur l'attribut Champ utilisateur.

  3. Appuyez sur la zone de texte.

  4. Saisissez le nom de votre collègue.

  5. Appuyez sur le nom de votre collègue dans la liste des résultats pour le sélectionner.

Pour plus d'informations sur la gestion de vos tâches assignées, reportez-vous aux autres rubriques dans Visualisation, modification et utilisation de vos tâches assignées.
