Recherche dans les flux

 

Pour lancer une recherche dans les flux :

  1. Appuyez sur Flux. La recherche est lancée dans le flux actif. Pour savoir quel est le flux actif, consultez la ligne située directement sous les boutons Rechercher, Filtrer et Statut.

  2. Facultatif Pour modifier le flux sur lequel effectuer la recherche, appuyez sur Filtrer, puis sélectionnez l'une des options de filtrage. Pour plus de détails, voir Filtrage, affichage, ajout de commentaires et recherche de mises à jour dans les flux.

  3. Appuyez sur Rechercher.

  4. Appuyez sur le champ de recherche et saisissez la chaîne souhaitée. La recherche porte sur toutes les mises à jour publiées. Vous ne pouvez pas rechercher d'utilisateurs ou de pièces jointes.

  5. Appuyez sur Rechercher.
