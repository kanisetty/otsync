Activation des notifications

 

Pour activer les notifications :

  1. Appuyez sur Paramètres sur votre appareil mobile.

  2. Appuyez sur Centre de notifications.

  3. Appuyez sur Everywhere 10.

  4. Activez le type de notification que vous souhaitez recevoir.

  5. Appuyez sur le bouton ACTIVÉ de Afficher dans le centre de notifications.

 

Note :
Les notifications de flux dans l'application OpenText ECM Everywhere diffèrent des paramètres de notification Pulse dans Content Server. Vous devez donc les configurer de manière distincte sur chaque système.
