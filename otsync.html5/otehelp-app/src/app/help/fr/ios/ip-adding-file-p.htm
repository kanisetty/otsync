Ajout d'un fichier

 

Pour ajouter un élément :

  1. Vous devez importer l'élément en le chargeant depuis une autre application, si cette dernière prend en charge cette fonctionnalité. Depuis une application tierce, affichez le fichier à importer et effectuez les opérations nécessaires pour ouvrir le document dans OpenText ECM Everywhere. Par exemple, appuyez sur , Partager ou Ouvrir dans. Une liste d'applications apparaît.

  2. Appuyez sur Everywhere 10.

  3. Facultatif Si vous y êtes invité, connectez-vous à OpenText ECM Everywhere.

  4. Appuyez sur Destination : espace de travail personnel pour accéder à l'emplacement auquel vous souhaitez ajouter ce fichier, puis appuyez sur Choisir.

  5. Facultatif Appuyez sur le champ Nom du fichier, puis modifiez le nom du fichier. Appuyez sur Terminé.

  6. Appuyez sur Téléchargement en amont.
