Déplacement ou copie d'un élément

 

Pour déplacer ou copier un élément :

  1. Appuyez sur Espace de travail professionnel ou sur Espace de travail personnel, puis accédez à l'élément cible.

    Vous ne pouvez pas copier un élément si celui-ci est réservé par un autre collègue. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vous pouvez vérifier l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

  2. Appuyez sur le fichier que vous souhaitez déplacer ou copier. Le fichier s'ouvre.

  3. Appuyez sur pour déplacer le fichier ou pour copier le fichier.

  4. Appuyez sur Retour, puis accédez au dossier cible dans lequel vous souhaitez placer le contenu du presse-papiers.

  5. Appuyez sur pour coller le fichier.

Pour déplacer ou copier un dossier :

  1. Appuyez sur Espace de travail professionnel ou sur Espace de travail personnel, puis accédez au dossier cible.

  2. Appuyez sur pour afficher les détails du dossier que vous souhaitez déplacer ou copier.

  3. Appuyez sur pour déplacer le dossier ou pour copier le dossier.

    Vous pouvez uniquement copier la racine d'un dossier Entreprise, quel que soit votre niveau d'accès. De même, vous pouvez uniquement copier les sous-dossiers au sein d'un dossier Entreprise pour lequel vous disposez d'un accès en lecture seule. Dans ce cas, une copie privée du dossier est créée au nouvel emplacement, mais le dossier Entreprise d'origine reste à l'emplacement d'origine.

  4. Appuyez sur Retour, puis accédez au dossier cible dans lequel vous souhaitez placer le contenu du presse-papiers.

  5. Appuyez sur pour coller le dossier.
