Lancement d'un workflow

 

Pour lancer un workflow :

  1. Appuyez sur Espace de travail professionnel ou sur Espace de travail personnel, puis accédez à l'élément cible.

  2. Appuyez sur le nom du schéma de workflow que vous souhaitez lancer. Un récapitulatif du schéma de workflow s'affiche, vous permettant de modifier les attributs, d'afficher les pièces jointes ou encore d'ajouter des commentaires avant de lancer le workflow.

    Les attributs requis sont marqués d'un astérisque. Le bouton Lancer n'est pas disponible tant que les attributs requis n'ont pas été renseignés.

    Les tâches assignées d'un workflow affichées dans OpenText ECM Everywhere ne prennent pas en charge les balises HTML dans les zones d'instructions ou de commentaires. Si le texte ne s'affiche pas correctement, supprimez les balises HTML du schéma de workflow dans Content Server.

  3. Facultatif Modifiez les attributs (le cas échéant). Pour plus d'informations, consultez la section Visualisation, modification et utilisation de vos tâches assignées.

  4. Facultatif Saisissez les commentaires à l'intention du destinataire de la tâche dans le champ Commentaires.

  5. Appuyez sur Lancer.
