Pour afficher un objet :

  1. Sur la page Accueil, cliquez sur Enterprise Workspace ou Espace de travail personnel, puis accédez au fichier cible en mettant en surbrillance un dossier et en appuyant sur la touche BlackBerry Trackpad. La barre de titre affiche le nom du dossier visualisé.

  2. Facultatif Pour être sûr de visualiser le contenu le plus récent, actualisez régulièrement le contenu du dossier. Appuyez sur la touche BlackBerry Menu et sélectionnez Actualiser. L'affichage est actualisé, mais les fichiers ne sont pas téléchargés. Pour télécharger un fichier, ouvrez-le.

  3. Mettez en surbrillance le fichier à visualiser.

  4. Appuyez sur la touche BlackBerry Trackpad pour visualiser le fichier.

  5. Pour revenir à vos fichiers, appuyez sur la touche BlackBerry Echap.

Pour afficher un long nom de fichier :

  1. Dans Enterprise Workspace ou Espace de travail personnel, accédez à l'emplacement de l'élément et mettez ce dernier en surbrillance.

  2. Appuyez sur la touche BlackBerry Menu, puis sélectionnez Détails de l'objet. Le nom de fichier entier s'affiche.

Pour modifier un fichier :

  1. Cliquez sur Enterprise Workspace ou Espace de travail personnel, puis accédez au fichier cible.

  2. Téléchargez le fichier vers votre appareil BlackBerry. Pour plus de détails, voir Téléchargement d'un fichier.

  3. Ouvrez le fichier dans une application tierce prenant en charge les modifications. Modifiez le fichier, puis enregistrez les modifications.

  4. Les modifications apportées à un fichier dans une application tierce ne sont pas enregistrées dans votre espace de travail. Vous devez charger le fichier modifié dans l'application tiers ou le dossier contenu dans la mémoire cache de BlackBerry.

    Dans l'application ECM Everywhere, chargez le fichier modifié en ajoutant une version du fichier. Pour plus de détails, voir Ajout d'un fichier.

Pour plus d'informations sur la gestion de vos fichiers, reportez-vous aux autres rubriques dans Visualisation, modification et utilisation de vos fichiers.
