Pour visualiser des fichiers tout en étant hors ligne :

  1. Pour faire en sorte qu'un fichier puisse être visualisé hors ligne, marquez le fichier souhaité comme favori et synchronisez ou téléchargez le fichier quand vous êtes en ligne. Cela vous permet d'y accéder rapidement depuis l'application Favoris. Pour plus de détails, voir Ajout, visualisation, suppression et synchronisation de favoris.

    Lorsque vous marquez un dossier comme favori, les fichiers et sous-dossiers qu'il contient ne sont pas synchronisés. Par conséquent, vous ne pouvez pas accéder au contenu des dossiers lorsque vous êtes hors ligne.

    Les fichiers marqués comme favoris sont indiqués par le recouvrement. Par exemple, .

    Si l'icône recouvrement apparaît lorsque vous êtes en ligne, par exemple , vous devez ouvrir le fichier ou synchroniser vos favoris afin de pouvoir consulter le fichier hors ligne. Le recouvrement est systématiquement affiché sur les dossiers.

  2. Sur la page Accueil, cliquez sur Favoris, puis sélectionnez le fichier à visualiser.

  3. Appuyez sur la touche BlackBerry Trackpad pour visualiser le fichier.

    Les fichiers que vous ouvrez lorsque vous êtes hors ligne peuvent être d'une version antérieure. Les éventuelles modifications apportées au fichier ne peuvent être synchronisées sur votre appareil.

  4. Pour revenir à vos fichiers, appuyez sur la touche BlackBerry Echap.
