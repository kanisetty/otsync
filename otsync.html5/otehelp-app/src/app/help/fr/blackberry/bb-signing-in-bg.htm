Pour vous connecter :

  1. Vous devez créer un profil avant de pouvoir vous connecter à l'application. Si vous n'avez pas de profil, procédez comme suit :

    1. Sur la page Profil, mettez Nouveau profil en surbrillance, puis appuyez sur la touche BlackBerry Trackpad. Vous devez être en ligne pour pouvoir créer un profil et vous connecter.

    2. Dans le champ Nom de profil, entrez un nom de profil unique.

    3. Dans le champ Nom d'utilisateur, saisissez votre nom d'utilisateur Content Server.

    4. Dans le champ Mot de passe, saisissez votre mot de passe Content Server. Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

    5. Dans le champ Serveur, saisissez l'URL de votre serveur ECM Everywhere (OpenText AppWorks Gateway).

      Pour connaître l'URL du serveur ECM Everywhere, contactez votre administrateur.

    6. Cochez la case Rester connecté pour un accès immédiat lors du prochain lancement de l'application. Les exceptions sont les suivantes : si vous vous déconnectez d'ECM Everywhere ou si la session arrive à expiration. Voir les précautions relatives à cette option ci-dessous.

    7. Cliquez sur Enregistrer et se connecter.

  2. Si vous avez déjà un profil, procédez comme suit :

    1. Sur la page Profil, mettez votre profil en surbrillance et appuyez sur la touche BlackBerry Trackpad.

    2. Si vous y êtes invité, saisissez votre mot de passe. Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

    3. Cochez la case Rester connecté pour un accès immédiat lors du prochain lancement de l'application. Les exceptions sont les suivantes : si vous vous déconnectez de l'application ou si la session arrive à expiration. Voir les précautions relatives à cette option ci-dessous.

    4. Appuyez sur la touche BlackBerry Entrée.

 

Attention
En autorisant ECM Everywhere à maintenir votre connexion, vous risquez de compromettre la sécurité. Si votre appareil BlackBerry est utilisé par une autre personne alors que vous ne vous êtes pas déconnecté de l'application ECM Everywhere, l'autre utilisateur pourra avoir accès à votre compte Content Server. Pour vous déconnecter, accédez à la page Accueil, appuyez sur la touche BlackBerry Menu, puis sélectionnez Déconnexion.

 

Pour plus d'informations sur la gestion de votre profil, reportez-vous aux autres rubriques dans Connexion et gestion de votre profil .
