Pour modifier votre profil :

  1. Dans l'écran Profil, mettez votre profil en surbrillance.

     

    Astuce :
    Si vous êtes connecté, appuyez sur la touche BlackBerry Menu, puis sélectionnez Changer de profil pour revenir à l'écran Profil.

     

  2. Appuyez sur la touche BlackBerry Menu, puis sélectionnez Modifier le profil.

  3. Modifiez le nom de votre profil, votre nom d'utilisateur ou le serveur.

  4. Cliquez sur Enregistrer le profil.

Pour mettre à jour votre mot de passe :

  1. Appuyez sur la touche BlackBerryMenu, puis sélectionnez Déconnexion. Vous ne pouvez pas modifier votre mot de passe si vous n'êtes pas en ligne.

  2. Dans l'écran Profil, mettez votre profil en surbrillance et appuyez sur la touche BlackBerry Trackpad.

  3. Saisissez votre nouveau mot de passe. Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

  4. Appuyez sur la touche BlackBerry Entrée.
