Pour lancer un workflow :

  1. Sur la page Accueil, cliquez sur Enterprise Workspace ou Espace de travail personnel, puis accédez à l'emplacement du schéma de workflow.

    Pour pouvoir lancer un workflow dans Enterprise Workspace, vous devez disposer des autorisations correspondantes pour le dossier sélectionné.

  2. Cliquez sur le nom du schéma de workflow que vous souhaitez lancer. Un récapitulatif du schéma de workflow s'affiche, vous permettant de modifier les attributs, d'afficher les pièces jointes ou encore d'ajouter des commentaires avant de lancer le workflow.

  3. Facultatif Ajoutez une pièce jointe au workflow. Pour plus de détails, voir Ajout d'une pièce jointe.

  4. Facultatif Modifiez les attributs (le cas échéant). Pour plus d'informations, consultez la section Visualisation, modification et utilisation de vos tâches assignées. Les attributs requis sont marqués d'un astérisque.

  5. Facultatif Saisissez les commentaires à l'intention du destinataire de la tâche dans le champ Commentaires.

  6. Cliquez sur Lancer.

  7. Cliquez sur OK.
