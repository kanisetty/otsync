Pour afficher vos tâches assignées :

  1. Sur la page Accueil, cliquez sur Tâches assignées, puis sur > pour la tâche assignée à visualiser.

    Les tâches assignées d'un workflow affichées dans l'application ne prennent pas en charge les balises HTML dans les zones d'instructions ou de commentaires. Si les instructions ne s'affichent pas correctement, supprimez les balises HTML du schéma de workflow dans Content Server.

  2. Modifiez les attributs.

  3. Cliquez sur Faire suivre.

Pour modifier des attributs :

  1. Cliquez sur Tâches assignées, puis sur > pour le workflow à visualiser.

  2. Cliquez sur la ligne d'attribut, puis saisissez les informations requises ou cliquez sur une option de la liste fournie pour la sélectionner.

     

    Note :
    Les attributs requis sont marqués d'un astérisque.

     

  3. Cliquez sur Faire suivre.

Pour ajouter un collègue à l'attribut Champ utilisateur :

  1. Cliquez sur Tâches assignées, puis sur > pour le workflow à visualiser.

  2. Cliquez sur l'attribut Champ d'utilisateur.

  3. Cliquez sur la zone de texte.

  4. Saisissez le prénom, le nom ou le nom d'utilisateur de votre collègue.

  5. Sélectionnez votre collègue dans la liste des résultats.

Pour plus d'informations sur la gestion de vos tâches assignées, reportez-vous aux autres rubriques dans Visualisation, modification et utilisation de vos tâches assignées.
