Pour supprimer un profil :

  1. Dans l'écran Profil, mettez le profil à supprimer en surbrillance.

     

    Astuce :
    Si vous êtes connecté, appuyez sur la touche BlackBerry Menu, puis sélectionnez Changer de profil pour revenir à l'écran Profil.

     

  2. Appuyez sur la touche BlackBerry Menu et sélectionnez Supprimer le profil.

  3. Cliquez sur OK.
