Pour afficher les utilisateurs qui aiment une publication :

  1. Sur la page Accueil, cliquez sur Flux.

  2. Cliquez sur une publication pour laquelle l'icône J'aime est affichée, .

  3. Cliquez sur le lien Afficher les mentions J'aime.
