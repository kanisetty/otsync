Pour effectuer une recherche dans toutes les publications :

  1. Sur la page Accueil, cliquez sur Flux.

  2. Appuyez sur la touche BlackBerry Menu, puis sélectionnez Rechercher.

    Dans le champ Recherche dans les flux, saisissez la chaîne de recherche, puis appuyez sur la touche BlackBerry Entrée.

  3. Appuyez sur la touche BlackBerry Échap pour revenir au flux standard.
