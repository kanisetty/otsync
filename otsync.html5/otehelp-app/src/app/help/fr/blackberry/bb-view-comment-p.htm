Pour afficher tous les commentaires d'une publication dans les flux :

  1. Sur la page Accueil, cliquez sur Flux. Pour les publications avec au moins un commentaire, s'affiche avec un nombre correspondant au nombre de commentaires ajoutés.

  2. Cliquez sur une publication.

  3. Cliquez sur le lien Afficher le fil de discussion.

    La première entrée de la liste correspond au commentaire d'origine. Les entrées suivantes sont répertoriées dans l'ordre de leur saisie, la plus récente étant affichée en premier.

  4. Cliquez sur la touche BlackBerry Échap pour revenir au flux standard.

Pour afficher les publications longues :

  1. Cliquez sur Flux.

  2. Cliquez sur la publication pour afficher les détails d'un commentaire long.

  3. Cliquez sur la touche BlackBerry Échap pour revenir au flux standard.
