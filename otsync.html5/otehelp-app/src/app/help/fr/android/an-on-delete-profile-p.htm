Suppression d'un profil

Pour supprimer un profil :

  1. Dans l'écran Profil, maintenez votre doigt sur le profil à supprimer.

     

    Astuce :
    Si vous êtes connecté, appuyez sur la touche Android Retour pour revenir à l'écran Profil.

     

  2. Appuyez sur Supprimer.

  3. Pour confirmer cette opération, appuyez sur Supprimer.
