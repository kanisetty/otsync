Affichage de commentaires

 

Pour afficher tous les commentaires d'une publication dans les flux :

  1. Appuyez sur Flux. Pour les publications avec plusieurs commentaires, s'affiche avec le nombre de commentaires ajoutés.

  2. Appuyez sur une publication.

  3. Appuyez sur le lien Afficher les fils pour afficher une mise à jour comportant plusieurs commentaires.

  4. Appuyez sur la touche Android Retour pour revenir à l'application Flux.

 

Astuce :
Pour afficher tous les commentaires ajoutés à chacun des fichiers et dossiers situés dans un dossier à l'emplacement de l'élément, reportez-vous à la rubrique Affichage des commentaires relatifs à un fichier ou un dossier .

 

Pour afficher les publications longues :

  1. Appuyez sur Flux.

  2. Appuyez sur la publication pour afficher les détails d'un commentaire long.
