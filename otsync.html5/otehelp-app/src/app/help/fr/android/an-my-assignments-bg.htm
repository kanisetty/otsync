Visualisation, modification et utilisation de vos tâches assignées

Pour afficher vos tâches assignées :

  1. Appuyez sur Tâches assignées, puis sur > pour le workflow que vous souhaitez visualiser. Les tâches assignées d'un workflow affichées dans OpenText ECM Everywhere ne prennent pas en charge les balises HTML dans les zones d'instructions ou de commentaires. Si les instructions ne s'affichent pas correctement, supprimez les balises HTML du schéma de workflow dans Content Server.

  2. Modifiez les attributs.

  3. Appuyez sur Soumettre.

Pour modifier des attributs :

  1. Appuyez sur Tâches assignées, puis sur > pour le workflow que vous souhaitez visualiser.

  2. Appuyez sur la ligne d'attribut, puis saisissez les informations requises ou appuyez sur une option de la liste fournie pour la sélectionner.

    Les attributs requis sont marqués d'un astérisque.

  3. Appuyez sur Soumettre.

Pour ajouter un collègue à l'attribut Champ utilisateur :

  1. Dans Tâches assignées, appuyez sur une tâche de workflow dans votre liste de tâches assignées.

  2. Appuyez sur l'attribut Champ utilisateur.

  3. Appuyez sur la zone de texte.

  4. Saisissez le nom de votre collègue.

  5. Appuyez sur le nom de votre collègue dans la liste des résultats pour le sélectionner.

Pour plus d'informations sur la gestion de vos tâches assignées, reportez-vous à la rubrique Visualisation, modification et utilisation de vos tâches assignées .
