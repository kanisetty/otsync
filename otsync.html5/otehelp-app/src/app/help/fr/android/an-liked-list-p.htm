Affichage des utilisateurs qui aiment une publication

 

Pour afficher les utilisateurs qui aiment une publication :

  1. Appuyez sur Flux.

  2. Appuyez sur une publication pour laquelle l'icône J'aime est affichée, .

  3. Appuyez sur le lien Afficher les mentions J'aime.

  4. Appuyez sur la touche Android Retour pour revenir à la publication.
