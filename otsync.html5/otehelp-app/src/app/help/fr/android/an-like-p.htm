Ajout d'une mention « J'aime » à une publication

 

Pour ajouter une mention « J'aime » à une publication dans les flux :

  1. Appuyez sur Flux.

  2. Appuyez sur une publication.

  3. Appuyez sur J'aime. Pour annuler cette opération, cliquez sur Enlever.

  4. Appuyez sur la touche Android Retour pour revenir à l'application Flux.

Pour ajouter une mention « J'aime » à une publication à l'emplacement de l'élément :

  1. Dans Enterprise Workspace ou Personal Workspace, accédez à un élément cible.

  2. Maintenez votre doigt sur l'élément, puis appuyez sur Commentaires.

  3. Appuyez sur une publication, puis sur J'aime.

  4. Appuyez sur la touche Android Retour pour revenir à l'espace de travail.
