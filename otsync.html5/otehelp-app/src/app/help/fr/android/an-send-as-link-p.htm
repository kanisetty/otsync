Ouverture d'un fichier dans une autre application

Pour ouvrir un fichier dans une autre application :

  1. Appuyez sur Enterprise Workspace ou sur Personal Workspace, puis accédez au document ou à l'image cible.

  2. Maintenez le doigt sur l'élément.

  3. Appuyez sur Envoyer par e-mail / Exporter. Une liste d'applications capables d'ouvrir le fichier apparaît.

  4. Appuyez sur une application et suivez les instructions à l'écran.
