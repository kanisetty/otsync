Affichage de pièces jointes

Pour afficher des pièces jointes :

  1. Appuyez sur Tâches assignées, puis sur > pour le workflow que vous souhaitez visualiser.

  2. Appuyez sur Afficher/ajouter des pièces jointes.

  3. Pour afficher une pièce jointe, appuyez sur son nom. Pour afficher les propriétés de la pièce jointe, maintenez votre doigt sur la pièce jointe, puis appuyez sur Détails sur l'objet.

  4. Appuyez sur la touche Android Retour pour revenir à la tâche de workflow.
