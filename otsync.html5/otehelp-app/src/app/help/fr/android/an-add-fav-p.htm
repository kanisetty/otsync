Ajout, visualisation, suppression et synchronisation de favoris

 

Pour ajouter un fichier ou un dossier favori :

  1. Appuyez sur Enterprise Workspace ou sur Personal Workspace, puis accédez au fichier ou au dossier cible.

  2. Maintenez votre doigt sur le fichier ou le dossier que vous souhaitez marquer comme favori.

  3. Appuyez sur Ajouter aux favoris.

    Le fichier ou le dossier est marqué à l'aide du recouvrement. Par exemple, . Cela vous permet d'y accéder rapidement depuis l'application Favoris. Pour les dossiers, cette opération crée un raccourci vers le dossier. Les fichiers et sous-dossiers présents dans le dossier ne sont pas marqués comme favoris.

  4. Facultatif Si l'icône recouvrement apparaît, le fichier n'est pas téléchargé. Par exemple, . Pour télécharger le fichier, synchronisez vos favoris ou ouvrez le fichier. Pour plus de détails, voir Synchroniser manuellement vos favoris.

    Pour synchroniser vos fichiers favoris automatiquement, voir Modification des paramètres d'OpenText ECM Everywhere.

    Un enregistrement de vos fichiers favoris est stocké dans votre profil. Les profils sont spécifiques à votre périphérique. Vous devez créer un profil et marquer les fichiers comme favoris sur chacun de vos smartphones et tablettes qui utilisent l'application OpenText ECM Everywhere.

Pour visualiser un favori :

  1. Appuyez sur Favoris.

  2. Facultatif Pour être sûr de visualiser le contenu le plus récent, actualisez régulièrement l'affichage. Pour plus de détails, voir Actualisation de l'affichage.

    Les fichiers ne sont pas téléchargés pour être visualisés hors ligne. Pour télécharger le fichier, synchronisez vos favoris ou ouvrez le fichier. Pour plus de détails, voir Synchroniser manuellement vos favoris ci-dessous.

Pour ouvrir l'emplacement d'un fichier favori :

  1. Appuyez sur Favoris.

  2. Maintenez votre doigt sur le fichier que vous souhaitez ouvrir.

  3. Appuyez sur Ouvrir l'emplacement.

Pour supprimer un favori :

  1. Appuyez sur Favoris ou sur Enterprise Workspace ou Personal Workspace et accédez au fichier cible.

  2. Maintenez votre doigt sur le fichier que vous souhaitez supprimer de votre liste de fichiers favoris.

  3. Appuyez sur Supprimer des favoris. Le fichier n'apparaît plus dans votre liste de favoris. Il n'est toutefois pas supprimé de votre espace de travail.

Pour synchroniser manuellement des favoris :

  1. Appuyez sur Favoris.

  2. Appuyez sur Synchroniser. Si le processus de synchronisation de vos favoris de l'application est en cours ou si vous êtes hors ligne, l'option Synchroniser n'est pas disponible.

     

    Note :
    Appuyez sur Actualiser pour mettre à jour l'affichage de la page Favoris. Toutefois, vos fichiers favoris ne sont pas téléchargés sur votre appareil pour une visualisation hors ligne. Pour télécharger le fichier, vous devez synchroniser vos fichiers favoris ou ouvrir le fichier.

     

    Pour synchroniser vos fichiers favoris automatiquement, voir Modification des paramètres d'OpenText ECM Everywhere.
