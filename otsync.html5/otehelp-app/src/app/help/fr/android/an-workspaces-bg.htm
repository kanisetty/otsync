Visualisation, modification et utilisation de vos fichiers

Pour afficher un objet :

  1. Dans Enterprise Workspace ou Personal Workspace, accédez à l'élément cible.

  2. Facultatif Pour être sûr de visualiser le contenu le plus récent, actualisez régulièrement l'affichage. Appuyez sur Menu, puis sur Actualiser.

  3. Appuyez sur le fichier à visualiser.

     

    Astuce :
    Vous pouvez également maintenir le doigt sur le fichier à visualiser et appuyer sur Ouvrir. Une liste d'applications capables d'ouvrir le fichier apparaît. Appuyez sur une application.

     

  4. Appuyez sur la touche Android Retour pour revenir à vos fichiers.

Pour modifier un élément :

  1. Appuyez sur Enterprise Workspace ou sur Personal Workspace, puis accédez à l'élément cible. Pour modifier un fichier, vous devez l'exporter dans une application tierce, puis importer le fichier modifié dans OpenText ECM Everywhere.

    Vous ne pouvez pas enregistrer les modifications que vous effectuez dans un fichier si celui-ci est réservé par un autre collègue dans Enterprise Workspace. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vérifiez l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

  2. Maintenez le doigt sur l'élément.

  3. Appuyez sur Envoyer par e-mail / Exporter.

  4. Appuyez sur une application tierce.

  5. Dans l'application tierce, modifiez le fichier et enregistrez les modifications sur l'appareil.

  6. OpenText ECM Everywhere vous permet désormais d'ajouter une nouvelle version au fichier d'origine. Pour plus de détails, voir Ajout d'un fichier.

    Si vous importez le fichier depuis une application tierce, un nouveau fichier est ajouté dans OpenText ECM Everywhere.

Pour plus d'informations sur la gestion de vos fichiers, reportez-vous à la rubrique Visualisation et utilisation de vos fichiers.
