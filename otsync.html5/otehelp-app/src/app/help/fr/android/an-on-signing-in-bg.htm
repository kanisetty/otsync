Modification et gestion de votre profil

Pour modifier votre profil :

  1. Sur l'écran Profil, maintenez le doigt sur votre profil, puis appuyez sur Modifier.

     

    Astuce :
    Si vous êtes connecté, appuyez sur la touche Android Retour pour revenir à l'écran Profil.

     

  2. Modifiez le nom de l'utilisateur ou le serveur.

  3. Appuyez sur Enregistrer les modifications.

Pour mettre à jour votre mot de passe :

  1. Dans l'écran Profil, appuyez sur votre profil.

     

    Astuce :
    Si vous êtes connecté, appuyez sur la touche Android Retour pour revenir à l'écran Profil.

     

  2. Appuyez sur le champ Mot de passe, puis saisissez votre nouveau mot de passe. Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

  3. Appuyez sur Connexion.

Pour plus d'informations sur la gestion de votre profil, reportez-vous aux autres rubriques dans Modification et gestion de votre profil .
