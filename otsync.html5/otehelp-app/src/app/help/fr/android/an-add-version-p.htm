Ajout d'un fichier

Pour ajouter une nouvelle version à l'aide d'un fichier sur votre périphérique :

  1. Appuyez sur Enterprise Workspace ou sur Personal Workspace, puis accédez au fichier cible.

  2. Maintenez votre doigt sur le fichier auquel vous souhaitez ajouter une nouvelle version.

    Vous ne pouvez pas ajouter une nouvelle version à un fichier si celui-ci est réservé par un autre collègue dans Enterprise Workspace. Les fichiers réservés sont marqués à l'aide du recouvrement. Pour savoir qui a réservé le fichier, vérifiez l'historique des audits du fichier. Pour plus de détails, voir Afficher les détails de l'objet.

    Pour pouvoir ajouter une version à un fichier dans Enterprise Workspace, vous devez disposer des autorisations correspondantes pour le dossier sélectionné.

  3. Appuyez sur Ajouter une version, puis sur Depuis un périphérique.

  4. Accédez à la nouvelle version du fichier sur votre périphérique.

  5. Appuyez sur le fichier.

Pour charger un fichier stocké sur votre périphérique :

  1. Appuyez sur Enterprise Workspace ou sur Personal Workspace, puis accédez au dossier cible dans lequel vous souhaitez charger un fichier.

  2. Maintenez votre doigt sur le dossier cible.

    Vous ne pouvez pas charger de fichier dans un dossier qui a des attributs et des catégories obligatoires via l'application OpenText ECM Everywhere. Pour déterminer si le dossier a des attributs ou des catégories obligatoires, affichez les détails de l'objet. Pour plus de détails, voir Afficher les détails de l'objet.

    Pour pouvoir charger un fichier dans un dossier de Enterprise Workspace, vous devez disposer des autorisations correspondantes pour le dossier sélectionné.

  3. Appuyez sur Charger ici, puis effectuez l'une des opérations suivantes :

    • Appuyez sur Depuis un appareil photo pour prendre une photo à l'aide de votre téléphone et la charger dans le dossier cible. Les photos sont enregistrées en tant que fichiers .jpg. Il n'est pas possible de modifier le nom de la photo avant de la télécharger.

    • Appuyez sur Depuis un appareil pour charger un fichier stocké sur votre appareil. Accédez au fichier cible sur votre appareil Android, puis appuyez dessus.

    • Appuyez sur Depuis une galerie pour charger une image depuis la galerie de photos de l'appareil. Accédez à la galerie de l'appareil, puis appuyez sur l'image.

Pour importer un fichier à partir d'une autre application :

  1. Depuis une application tierce, affichez le fichier que vous souhaitez ajouter.

  2. Appuyez sur Partager ou sur Ouvrir dans. Une liste d'applications apparaît.

  3. Appuyez sur Everywhere 10.

  4. Facultatif Si vous y êtes invité, connectez-vous à OpenText ECM Everywhere.

  5. Appuyez sur Enterprise Workspace ou sur Personal Workspace, puis accédez à l'emplacement auquel vous souhaitez ajouter ce fichier. Pour pouvoir importer un fichier dans un dossier de Enterprise Workspace, vous devez disposer des autorisations correspondantes pour le dossier sélectionné.

    Vous ne pouvez pas importer de fichier dans un dossier qui a des attributs et des catégories obligatoires via l'application OpenText ECM Everywhere. Pour déterminer si le dossier a des attributs ou des catégories obligatoires, affichez les détails de l'objet. Pour plus de détails, voir Afficher les détails de l'objet.

  6. Appuyez sur Coller.
