Visualización de comentarios en un archivo o carpeta

Para visualizar los comentarios de un archivo o carpeta:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta la carpeta destino.

  2. Pulse y mantenga pulsado el elemento.

  3. Realice una de las acciones siguientes:

    • Pulse Comentarios para ver todas las publicaciones asociadas con el elemento seleccionado.

    • Pulse Comentarios desde aquí para ver todos los comentarios añadidos a cada archivo y carpeta dentro de una carpeta.

  4. Pulse la tecla Volver de Android para regresar a la zona de trabajo.
