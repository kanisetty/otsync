Adición de un archivo adjunto

 

Para añadir un archivo adjunto:

  • Cuando añada una actualización de estado o realice un comentario de una publicación, pulse Adjuntar y realice una de las siguientes acciones:

    • Pulse Desde cámara para adjuntar una imagen con la cámara de Android. Tome la fotografía y luego siga las instrucciones que se muestran en la pantalla para guardar la imagen.

    • Pulse Desde sistema de archivos para adjuntar un archivo que esté almacenado en su dispositivo Android. Navegue hasta el archivo y luego púlselo.

    • Pulse Desde galería para adjuntar una imagen desde la galería de fotos de Android. Navegue hasta la imagen y luego púlsela.
