Desplazamiento o copia de un archivo o carpeta

 

Para mover o copiar un archivo o carpeta:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta el elemento destino.

    Solo puede copiar un elemento cuando está reservado por otro compañero en Enterprise Workspace. Los archivos reservados están marcados con la superposición . Para averiguar quién reservó el archivo, revise el historial de auditorías del archivo. Para obtener más información, consulte Visualización de detalles de objetos.

  2. Pulse y mantenga pulsado el elemento que desea mover o copiar.

  3. Pulse Cortar para mover el elemento o pulse Copiar para copiarlo.

  4. Desplácese hasta la carpeta destino en la que desea colocar el contenido del portapapeles.

    No puede mover ni copiar un elemento a una carpeta que tenga categorías y atributos obligatorios mediante el uso de la aplicación OpenText ECM Everywhere. Para averiguar si la carpeta tiene categorías o atributos obligatorios, vea los detalles de los objetos. Para obtener más información, consulte Visualización de detalles de objetos.

  5. Pulse Pegar.
