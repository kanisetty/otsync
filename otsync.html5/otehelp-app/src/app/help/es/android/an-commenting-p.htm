Realización de comentarios sobre un archivo o carpeta

Para comentar acerca de un archivo o carpeta:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta el elemento del que desea comentar.

  2. Pulse y mantenga pulsado el elemento.

  3. Pulse Comentarios.

  4. Pulse el cuadro de texto y escriba su comentario.

    Para llamar la atención de un compañero concreto sobre la publicación, utilice una mención @. Para obtener más información sobre las menciones, consulte Mención de un usuario y visualización de menciones.

    Para añadir un archivo adjunto, consulte Adición de un archivo adjunto.

  5. Pulse Publicar.

  6. Pulse la tecla Volver de Android para regresar a la zona de trabajo.

 

Sugerencia:
Para obtener más información sobre cómo añadir actualizaciones, consulte Cómo filtrar, ver, comentar y realizar búsquedas en publicaciones en Fuentes.
