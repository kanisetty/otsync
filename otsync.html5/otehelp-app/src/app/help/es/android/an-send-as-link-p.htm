Apertura de un archivo en otra aplicación

 

Para abrir un archivo en otra aplicación:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta el documento o imagen destino.

  2. Pulse y mantenga pulsado el elemento.

  3. Pulse Enviar por correo electrónico/Exportar. Se muestra una lista de aplicaciones que pueden abrir el archivo.

  4. Pulse una aplicación y siga las instrucciones que se muestran en la pantalla.
