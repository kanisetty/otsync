Capítulo 3 – Adición, visualización, eliminación y sincronización de favoritos

 

Para añadir un archivo o carpeta favorito:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta el archivo o la carpeta destino.

  2. Mantenga pulsado el archivo o la carpeta que desea marcar como favorito.

  3. Pulse Añadir a favoritos.

    El archivo o la carpeta se marcan con la superposición . Por ejemplo, . Esto le proporciona un fácil acceso al archivo o carpeta desde la aplicación Favoritos Para las carpetas, esto genera un método abreviado. No marca como favoritos los archivos y las subcarpetas dentro de la carpeta original.

  4. Opcional Si aparece la superposición , significa que el archivo no se ha bajado. Por ejemplo, . Para bajar el archivo, sincronice sus favoritos o abra el archivo. Para obtener más información, consulte Sincronización manual de favoritos.

    Para sincronizar los archivos favoritos automáticamente, consulte Cambio de OpenText ECM Everywhere configuración.

    En su dispositivo se guarda un registro de sus archivos favoritos. Los perfiles son específicos de su dispositivo. Debe crear un perfil y marcar los archivos como favoritos en cada uno de los smartphones y tabletas que usen la aplicación OpenText ECM Everywhere.

Para ver un favorito:

  1. Pulse Favoritos.

  2. Opcional Para asegurarse de que está viendo el contenido más actualizado, debe actualizar de forma periódica la pantalla. Para obtener más información, consulte Actualización de la pantalla.

    Esta operación no baja los archivos para su visualización fuera de línea. Para bajar el archivo, sincronice sus favoritos o abra el archivo. Para obtener más información, consulte Sincronización manual de favoritos que se muestra más abajo.

Para abrir la ubicación de un archivo favorito:

  1. Pulse Favoritos.

  2. Pulse y mantenga pulsado el archivo para el que desea abrir la ubicación de archivos.

  3. Pulse Abrir ubicación.

Para eliminar un favorito:

  1. Pulse Favoritos o pulse Enterprise Workspace o Zona personal y desplácese hasta el archivo destino.

  2. Pulse y mantenga pulsado el archivo que desea eliminar de la lista de favoritos.

  3. Pulse Eliminar de favoritos. El archivo ya no aparece en la lista de favoritos. Esto no suprime el elemento de su zona de trabajo.

Para sincronizar los favoritos manualmente:

  1. Pulse Favoritos.

  2. Pulse Sincronizar. Si la aplicación se encuentra en el proceso de sincronización de los favoritos, o si usted está fuera de línea, la opciónSincronizar no estará disponible.

     

    Nota:
    Al pulsar Actualizar, se actualiza la pantalla de la página Favoritos. No baja los documentos favoritos a su dispositivo para su visualización fuera de línea. Para bajar el archivo, debe sincronizar los archivos favoritos o abrir el archivo en cuestión.

     

    Para sincronizar los archivos favoritos automáticamente, consulte Cambio de OpenText ECM Everywhere configuración.
