Adición de una fotografía o carpeta

Para añadir una fotografía desde la cámara de Android:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta la carpeta destino en la que desea añadir una fotografía.

    No puede añadir una fotografía a una carpeta que tenga categorías y atributos obligatorios mediante el uso de la aplicación OpenText ECM Everywhere. Para averiguar si la carpeta tiene categorías o atributos obligatorios, vea los detalles de los objetos. Para obtener más información, consulte Visualización de detalles de objetos.

  2. Pulse Añadir y luego pulse Desde cámara.

  3. Realice la fotografía y siga las instrucciones que se muestran en la pantalla. Las fotos se guardan como archivos .jpg. No puede editar el nombre de la foto antes de cargarla.

Para subir una fotografía desde la galería del dispositivo:

  1. Inicie la aplicación de galería Android y desplácese hasta la fotografía que desea importar.

  2. Pulse Compartir o Abrir en . Se muestra una lista de aplicaciones.

  3. Pulse Everywhere 10.

  4. Opcional Si se le solicita, conéctese a OpenText ECM Everywhere.

  5. Pulse Enterprise Workspace o Zona personal y desplácese hasta la ubicación en la que desea añadir esta fotografía.

    No puede subir una fotografía a una carpeta que tenga categorías y atributos obligatorios mediante el uso de la aplicación OpenText ECM Everywhere. Para averiguar si la carpeta tiene categorías o atributos obligatorios, vea los detalles de los objetos. Para obtener más información, consulte Visualización de detalles de objetos.

  6. Pulse Pegar.

Para añadir una carpeta:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta la ubicación destino para la nueva carpeta.

  2. Pulse Añadir y luego pulse Carpeta.

  3. Pulse el recuadro Nombre de carpeta y escriba un nombre para esta carpeta.

  4. Pulse Aceptar.
