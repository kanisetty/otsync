Capítulo 1 – Visualización, edición y trabajo con archivos

Para ver un elemento:

  1. En Enterprise Workspace o Zona personal , desplácese hasta el elemento destino.

  2. Opcional Para asegurarse de que está viendo el contenido más actualizado, debe actualizar de forma periódica la pantalla. Pulse Menú y luego pulse Actualizar.

  3. Pulse el archivo que desea ver.

     

    Sugerencia:
    También puede pulsar y mantener pulsado el archivo que quiere ver y, a continuación, pulsar Abrir. Se muestra una lista de aplicaciones que pueden abrir el archivo. Pulse una aplicación.

     

  4. Pulse la tecla Volver de Android para regresar a los archivos.

Para editar un elemento:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta el elemento destino. Para editar un archivo, tiene que exportar el archivo a una aplicación de terceros y luego importar el archivo editado nuevamente a OpenText ECM Everywhere.

    No puede guardar modificaciones en un archivo cuando está reservado por otro compañero en Enterprise Workspace. Los archivos reservados están marcados con la superposición . Para averiguar quién reservó el archivo, revise el historial de auditorías del archivo. Para obtener más información, consulte Visualización de detalles de objetos.

  2. Pulse y mantenga pulsado el elemento.

  3. Pulse Enviar por correo electrónico/Exportar.

  4. Pulse una aplicación de terceros.

  5. En la aplicación de terceros, modifique el archivo y guarde sus cambios en el dispositivo.

  6. En OpenText ECM Everywhere, podrá ahora añadir una nueva versión al archivo original. Para obtener más información, consulte Adición de un archivo.

    Si importa el archivo desde la aplicación de terceros, se añade un nuevo archivo en OpenText ECM Everywhere.

Para obtener más información sobre cómo trabajar con sus archivos, consulte Visualización y trabajo con archivos.
