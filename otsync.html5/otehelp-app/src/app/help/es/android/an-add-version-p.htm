Adición de un archivo

Para añadir una nueva versión mediante un archivo en el dispositivo:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta el archivo destino.

  2. Pulse y mantenga pulsado el archivo al que desea añadir una versión nueva.

    No puede añadir una nueva versión a un archivo cuando está reservado por otro compañero en Enterprise Workspace. Los archivos reservados están marcados con la superposición . Para averiguar quién reservó el archivo, revise el historial de auditorías del archivo. Para obtener más información, consulte Visualización de detalles de objetos.

    Su capacidad para añadir una versión a un archivo en Enterprise Workspace depende de sus permisos para la carpeta seleccionada.

  3. Pulse Añadir versión y, a continuación, pulse Desde dispositivo.

  4. Desplácese hasta la nueva versión del archivo de su dispositivo.

  5. Pulse el archivo.

Para subir un archivo guardado en su dispositivo:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta la carpeta destino a la que desea subir un archivo.

  2. Pulse y mantenga pulsada la carpeta destino.

    No puede subir un archivo a una carpeta que tenga categorías y atributos obligatorios mediante el uso de la aplicación OpenText ECM Everywhere. Para averiguar si la carpeta tiene categorías o atributos obligatorios, vea los detalles de los objetos. Para obtener más información, consulte Visualización de detalles de objetos.

    Su capacidad para subir un archivo a una carpeta en Enterprise Workspace depende de sus permisos para la carpeta seleccionada.

  3. Pulse Subir aquí y realice una de las siguientes acciones:

    • Pulse Desde cámara para hacer una fotografía usando su teléfono y subirla a la carpeta destino. Las fotos se guardan como archivos .jpg. No puede editar el nombre de la foto antes de cargarla.

    • Pulse Desde dispositivo para subir un archivo que esté almacenado en su dispositivo. Desplácese hasta el archivo destino en su dispositivo Android y pulse el archivo.

    • Pulse Desde galería para subir una fotografía desde la galería de fotos del dispositivo. Desplácese por la galería del dispositivo y pulse la fotografía.

Para importar un archivo desde otra aplicación:

  1. Desde una aplicación de terceros, visualice el archivo que desea añadir.

  2. Pulse Compartir o Abrir en . Se muestra una lista de aplicaciones.

  3. Pulse Everywhere 10.

  4. Opcional Si se le solicita, conéctese a OpenText ECM Everywhere.

  5. Pulse Enterprise Workspace o Zona personal y desplácese hasta la ubicación en la que desea añadir este archivo. Su capacidad para importar un archivo a una carpeta en Enterprise Workspace depende de sus permisos para la carpeta seleccionada.

    No puede importar un archivo a una carpeta que tenga categorías y atributos obligatorios mediante el uso de la aplicación OpenText ECM Everywhere. Para averiguar si la carpeta tiene categorías o atributos obligatorios, vea los detalles de los objetos. Para obtener más información, consulte Visualización de detalles de objetos.

  6. Pulse Pegar.
