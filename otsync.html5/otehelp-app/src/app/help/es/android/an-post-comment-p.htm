Adición de comentarios

 

Para comentar en una publicación en Fuentes:

  1. Pulse Fuentes.

  2. Pulse la publicación que desea comentar.

  3. Pulse Réplica.

  4. Escriba su comentario en el cuadro de texto.

  5. Opcional Para llamar la atención de un compañero concreto sobre la publicación, utilice una mención @. Para obtener más información sobre las menciones, consulte Mención de un usuario y visualización de menciones.

    Para añadir un archivo adjunto, consulte Adición de un archivo adjunto.

  6. Pulse Publicar.

 

Sugerencia:
Para comentar una publicación en la ubicación del elemento, consulte Realización de comentarios sobre un archivo o carpeta.
