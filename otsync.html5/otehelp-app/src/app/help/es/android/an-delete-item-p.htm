Supresión de un archivo o carpeta

Para eliminar un elemento:

  1. Pulse Enterprise Workspace o Zona personal y desplácese hasta el archivo o la carpeta destino.

    No puede suprimir un archivo que está reservado. Los archivos reservados están marcados con la superposición . Para suprimir un archivo reservado, primero tendrá que cancelar la reserva. Para obtener más información, consulte Reserva y cancelación de la reserva de un archivo.

  2. Pulse y mantenga pulsado el nombre del elemento que desea suprimir.

  3. Pulse Suprimir. Al suprimir un archivo o carpeta, se elimina de Enterprise Workspace o Zona personal, incluso si el elemento se marcó como favorito.

  4. Pulse Aceptar para confirmar la acción.
