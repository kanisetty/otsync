Ver comentarios

Para ver todos los comentarios de una publicación en Fuentes:

  1. Pulse Fuentes. Las publicaciones con diversos comentarios se muestran junto con la cantidad de comentarios añadidos.

  2. Pulse una publicación.

  3. Pulse el enlace Vista de proceso para ver una actualización con varios comentarios.

  4. Pulse la tecla Volver de Android para regresar a la aplicación Fuentes.

 

Sugerencia:
Para ver todos los comentarios añadidos a cada archivo y carpeta dentro de una carpeta en la ubicación del elemento, consulte Visualización de comentarios en un archivo o carpeta .

 

Para ver publicaciones largas:

  1. Pulse Fuentes.

  2. Pulse la publicación para ver los detalles completos de un comentario largo.
