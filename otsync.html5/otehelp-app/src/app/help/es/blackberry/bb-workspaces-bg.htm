Para ver un elemento:

  1. En la página de Inicio, haga clic en Enterprise Workspace o en Zona personal y desplácese hasta el archivo de destino resaltando una carpeta y pulsando la tecla Panel táctil del dispositivo BlackBerry. La barra de título muestra el nombre de la carpeta que está viendo en ese momento.

  2. Opcional Para asegurar que está viendo el contenido más actualizado, debe actualizar de forma periódica el contenido de la carpeta. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Actualizar. Esto actualiza la pantalla, pero no baja los archivos. Para bajar un archivo, ábralo.

  3. Resalte el archivo que desea ver.

  4. Pulse el panel táctil del dispositivo BlackBerry para ver el archivo.

  5. Pulse la tecla Escape del dispositivo BlackBerry para volver a sus archivos.

Para ver un nombre de archivo largo:

  1. En Enterprise Workspace o Zona personal, desplácese hasta la ubicación del elemento y resalte el elemento que desea ver.

  2. Pulse la tecla Menú del dispositivo BlackBerry y luego seleccione Detalles del objeto. Se muestra el nombre completo del archivo.

Para editar un archivo:

  1. Haga clic en Enterprise Workspace o en Zona personal y desplácese hasta el archivo de destino.

  2. Descargue el archivo en su dispositivo BlackBerry. Para obtener más información, consulte Bajada de un archivo.

  3. Abra el archivo en una aplicación de terceros que admita la edición. Edite el archivo y guarde los cambios.

  4. Los cambios que le realice al archivo en una aplicación de terceros, no se guardan en su zona de trabajo. Debe subir el archivo modificado desde la aplicación de terceros o desde la carpeta de caché de BlackBerry.

    En la aplicación ECM Everywhere, suba el archivo editado añadiendo una versión del archivo. Para obtener más información, consulte Adición de un archivo.

Para obtener más información sobre cómo trabajar con sus archivos, consulte las demás secciones de Visualización, edición y trabajo con archivos.
