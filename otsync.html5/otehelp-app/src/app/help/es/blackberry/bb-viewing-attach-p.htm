Para añadir un archivo adjunto a una asignación:

  1. En la página de Inicio, haga clic en Asignaciones y en > para la asignación que desea ver.

  2. Haga clic en Ver o añadir archivos adjuntos.

  3. Pulse la tecla Menú del dispositivo BlackBerry y realice una de las siguientes acciones:

    • Seleccione Subir nueva versión para añadir una nueva versión del archivo adjunto. Para obtener más información, consulte Adición de un archivo.

    • Seleccione Subir aquí para añadir un nuevo archivo adjunto que resida en su BlackBerry. Para obtener más información, consulte Adición de un archivo.

    • Seleccione Añadir foto para añadir un nuevo archivo adjunto mediante el uso de una imagen en su BlackBerry. Para obtener más información, consulte Adición de una fotografía o carpeta.

  4. Pulse la tecla Escape del dispositivo BlackBerry para volver a la asignación.
