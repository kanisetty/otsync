Para mover o copiar un archivo o carpeta:

  1. En la página de Inicio, haga clic en Enterprise Workspace o en Zona personal y desplácese hasta el archivo o la carpeta de destino.

  2. Resalte el archivo o la carpeta que desea mover o copiar.

  3. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Cortar para mover el elemento o Copiar para copiar el elemento.

  4. Desplácese hasta la carpeta destino en la que desea colocar el contenido del portapapeles.

    Su capacidad para mover o copiar un elemento a una carpeta en Enterprise Workspace depende de sus permisos para la carpeta seleccionada.

  5. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Pegar.
