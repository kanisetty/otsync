Adición de un archivo

 

Para agregar un elemento:

  1. Debe importar el elemento subiéndolo desde otra aplicación, si la aplicación soporta esta función. Desde una aplicación de terceros, vea el archivo que desea importar y lleve a cabo los pasos necesarios para abrir el documento en OpenText ECM Everywhere. Por ejemplo, pulse , Compartir o Abrir en. Se muestra una lista de aplicaciones.

  2. Pulse Everywhere 10.

  3. Opcional Si se le solicita, conéctese a OpenText ECM Everywhere.

  4. Pulse Destino: Zona de la empresa para desplazarse hasta la ubicación en la que desea añadir este archivo y, a continuación, pulse Seleccionar.

  5. Opcional Pulse el recuadro Nombre de archivo y escriba el nombre del archivo. Toque Hecho.

  6. Pulse Subir.
