Supresión de un perfil

 

Para suprimir un perfil:

  1. En la pantalla Perfil, pulse y mantenga pulsado el perfil que desea eliminar.

     

    Sugerencia:
    Si está conectado, pulse para volver a la pantalla Página de llegada y luego pulse Configuración > Desconectarse para volver a la pantalla Perfil.

     

  2. Pulse Suprimir.

  3. Pulse Suprimir para confirmar la acción.
