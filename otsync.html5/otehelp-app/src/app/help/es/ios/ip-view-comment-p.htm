Ver comentarios

Para ver una publicación con un comentario largo:

  1. Pulse Fuentes.

  2. Pulse la publicación para ver los detalles completos de un comentario largo.

Para ver todos los comentarios de una publicación:

  1. Pulse Fuentes. Las publicaciones con diversos comentarios se muestran junto con la cantidad de comentarios añadidos.

  2. Pulse una publicación.

  3. Pulse el enlace Vista de procesos para ver una actualización con varios comentarios.

  4. Pulse para volver a la publicación.

 

Sugerencia:
Para ver los comentarios en la ubicación del elemento, consulte Visualización de comentarios en un archivo o carpeta.
