Movimiento o copia de un elemento

 

Para mover o copiar un elemento:

  1. Pulse Zona de la empresa o Zona personal y, a continuación, desplácese hasta el elemento destino.

    Solo puede copiar un elemento cuando está reservado por otro compañero. Los archivos reservados están marcados con la superposición . Para averiguar quién ha reservado el archivo, puede comprobar su historial de auditorías. Para obtener más información, consulte Visualización de detalles de objetos.

  2. Pulse el archivo que desea mover o copiar. El archivo se abre.

  3. Pulse para mover el archivo o para copiar el archivo.

  4. Pulse Volver y desplácese hasta la carpeta destino en la que desea colocar el contenido del portapapeles.

  5. Pulse para pegar el archivo.

Para mover o copiar una carpeta:

  1. Pulse Zona de la empresa o Zona personal y desplácese hasta la carpeta destino.

  2. Pulse para ver los detalles de la carpeta que desea mover o copiar.

  3. Pulse para mover la carpeta o para copiar la carpeta.

    Independientemente del nivel de acceso que posea, solo podrá copiar la raíz de una carpeta de empresa. Asimismo, solo podrá copiar subcarpetas de una carpeta de empresa para las que tenga acceso de solo lectura. Al hacerlo, se crea una copia privada de la carpeta en la ubicación nueva, pero la carpeta de empresa original permanece en la ubicación original.

  4. Pulse Volver y desplácese hasta la carpeta destino en la que desea colocar el contenido del portapapeles.

  5. Pulse para pegar la carpeta.
