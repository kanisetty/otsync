Inicio de un workflow

Para iniciar un workflow:

  1. Pulse Zona de la empresa o Zona personal y, a continuación, desplácese hasta el elemento destino.

  2. Pulse el nombre del esquema de workflow que desea iniciar. Se muestra un resumen del esquema de workflow en el que puede editar atributos, ver archivos adjuntos o añadir comentarios antes de iniciar el workflow.

    Los atributos necesarios están marcados con un asterisco. El botón Iniciar no está disponible hasta que los atributos necesarios se llenan.

    Las asignaciones de workflow que se vieron en OpenText ECM Everywhere no soportan las etiquetas HTML en los apartados de instrucciones o comentarios. Si el texto no se entrega adecuadamente, quite las etiquetas HTML del Esquema de workflow en Content Server.

  3. Opcional Editar los atributos si están disponibles. Para obtener más información, consulte Visualización, edición y trabajo con sus asignaciones.

  4. Opcional Escriba comentarios para la persona asignada al paso, en el recuadro de Comentarios.

  5. Pulse Iniciar.
