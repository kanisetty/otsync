Visualización de comentarios en un archivo o carpeta

Para visualizar los comentarios de un archivo o carpeta:

  1. Pulse Zona de la empresa o Zona personal y, a continuación, desplácese hasta el archivo o carpeta destino.

  2. Pulse .

  3. Pulse Comentarios desde aquí.

  4. Pulse para volver a la pantalla de detalles de carpeta.

  5. Pulse Volver para regresar a los archivos.
