Reserva y cancelación de la reserva de un archivo

Para reservar o cancelar la reserva de un archivo:

  1. Pulse Zona de la empresa o Zona personal y, a continuación, desplácese hasta el archivo destino.

  2. Pulse para ver los detalles del archivo que desea reservar o cuya reserva desea cancelar.

  3. Realice una de las acciones siguientes:

    • Pulse Reservar para reservar el archivo. El archivo se marca con la superposición .

      Si intenta reservar un archivo en una carpeta de empresa para la que tiene acceso de lectura/escritura y la opción Reservar no está disponible, es que no tiene el permiso Reservar para la carpeta en Content Server. Póngase en contacto con el administrador para solicitar este permiso.

    • Pulse Cancelar la reserva para cancelar la reserva del archivo. Si no está disponible Cancelar la reserva, significa que otro usuario ha reservado el archivo. Para averiguar quién ha reservado el archivo, puede comprobar su historial de auditorías. Para obtener más información, consulte Visualización de detalles de objetos.

  4. Pulse Volver.
