Adición de una imagen

Para añadir una imagen:

  1. Cuando añada una actualización de estado o realice un comentario de una publicación, pulse Adjuntar y realice una de las siguientes acciones:

    • Pulse Desde cámara para adjuntar una imagen con la Apple cámara.

      Tome la imagen y luego pulse Usar fotografía para usar la imagen actual o pulse Volver a tomar y vuelva a tomar la imagen.

    • Pulse Desde galería para adjuntar una imagen desde el álbum de fotos de Apple.

      Desplácese hasta la imagen que desea adjuntar y luego púlsela.

    Si cambia de parecer, puede pulsar Eliminar para eliminar el archivo adjunto.

  2. Escriba su comentario en el cuadro de texto y pulse Publicar.
