Zuweisungen anzeigen, bearbeiten und verwenden

 

So bearbeiten Sie Workflow-Attribute:

  1. Tippen Sie auf Zuweisungen und dann auf eine Workflow-Aufgabe.

  2. Gehen Sie wie folgt vor, um Attributwerte einzugeben:

    • Tippen Sie auf ein Datumsattribut, wählen Sie ein Datum aus, und tippen Sie anschließend auf Fertig, um zu der Workflow-Aufgabe zurückzukehren.

    • Tippen Sie auf ein Popup-Attribut, wählen Sie eine Option, und tippen Sie auf Zurück, um zur Workflow-Aufgabe zurückzukehren.

    • Tippen Sie auf eine Ganzzahl oder ein reales Attribut, geben Sie einen Wert ein, und tippen Sie anschließend auf Zurück, um zur Workflow-Aufgabe zurückzukehren.

      Erforderliche Attribute sind mit einem Sternchen markiert.

      Die Schaltfläche Weiterleiten steht erst zur Verfügung, wenn die erforderlichen Attribute angegeben wurden.

So fügen Sie im Attribut Benutzerfeld einen Kollegen hinzu:

  1. Tippen Sie auf Zuweisungen und dann auf eine Workflow-Aufgabe.

  2. Tippen Sie auf das Benutzerfeld-Attribut.

  3. Tippen Sie auf das Textfeld.

  4. Geben Sie den Namen Ihres Kollegen ein.

  5. Wählen Sie Ihren Kollegen in der Ergebnisliste per Fingertipp aus.

Informationen zur Verwendung Ihrer Zuordnungen finden Sie in den anderen Themen unter Zuweisungen anzeigen, bearbeiten und verwenden.
