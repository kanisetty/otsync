Profilbild ändern

 

So ändern Sie Ihr Profilbild:

  1. Tippen Sie im Landing-Bildschirm auf Einstellungen.

  2. Tippen Sie auf Zum Ändern des Fotos hier klicken.

  3. Führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf Von Kamera, um das Profilbild mithilfe der Apple-Kamera festzulegen.

      Nehmen Sie das Bild auf, und tippen Sie dann auf Foto verwenden, um das aktuelle Bild zu verwenden, oder tippen Sie auf Erneut aufnehmen, und nehmen Sie das Bild erneut auf.

    • Tippen Sie auf Aus Album, um Ihr Profilbild mit einem Foto aus der Apple-Fotogalerie festzulegen.

      Navigieren Sie zu dem Bild, das Sie verwenden möchten, und tippen Sie dann auf das Bild.

    • Tippen Sie auf Als Standard festlegen, um als Profilbild das standardmäßige Profilbild zu verwenden.

  4. Tippen Sie auf Einstellungen.
