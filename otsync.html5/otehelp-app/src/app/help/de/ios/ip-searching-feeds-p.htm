Feeds durchsuchen

So durchsuchen Sie Feeds:

  1. Tippen Sie auf Feeds. Die Suche wird im aktiven Feed durchgeführt. Um den aktiven Feed zu ermitteln, prüfen Sie die Zeile direkt unter den Schaltflächen Suchen, Filter und Status.

  2. Optional Um den zu durchsuchenden Feed zu ändern, tippen Sie auf Filter, und wählen Sie eine der Filteroptionen. Weitere Informationen finden Sie unter Aktualisierungen in Feeds filtern, anzeigen, kommentieren und durchsuchen.

  3. Tippen Sie auf Suchen.

  4. Tippen Sie auf das Suchfeld, und geben Sie eine Suchzeichenfolge ein. Die Suche wird für alle geposteten Aktualisierungen durchgeführt. Sie können nicht nach Benutzern oder Anlagen suchen.

  5. Tippen Sie auf Suchen.
