Kommentare hinzufügen

 

So kommentieren Sie einen Beitrag in Feeds:

  1. Tippen Sie auf Feeds.

  2. Tippen Sie auf den Beitrag, den Sie kommentieren möchten.

  3. Tippen Sie auf Antworten.

  4. Geben Sie Ihren Kommentar in das Textfeld ein.

  5. Optional Um Ihren Beitrag einem bestimmten Kollegen mitzuteilen, verwenden Sie eine @Nennung. Weitere Informationen zu Nennungen finden Sie unter Benutzer erwähnen und Erwähnungen anzeigen.

    Informationen zum Hinzufügen einer Anlage finden Sie unter Bild hinzufügen.

  6. Tippen Sie auf Posten.
