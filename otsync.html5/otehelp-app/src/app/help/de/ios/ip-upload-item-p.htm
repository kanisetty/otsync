Fotos oder Ordner hinzufügen

 

So fügen Sie ein Foto mit der Apple-Kamera hinzu:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zum Zielordner, in dem Sie ein Foto hinzufügen möchten.

    Wenn der Zielordner obligatorische Kategorien oder Attribute enthält, können Sie diesem Ordner mithilfe der OpenText ECM Everywhere-App kein Foto hinzufügen. Wie Sie bestimmen können, ob der Ordner obligatorische Kategorien oder Attribute hat, erfahren Sie unter Objektdetails anzeigen.

  2. Tippen Sie auf .

  3. Tippen Sie auf Von Kamera.

  4. Machen Sie ein Foto.

  5. Tippen Sie auf Foto verwenden, um das aktuelle Foto hinzuzufügen, oder tippen Sie auf Erneut aufnehmen, und wiederholen Sie Schritt 4.

  6. Optional Tippen Sie auf das Feld Dateiname, und bearbeiten Sie den Namen des Fotos. Tippen Sie auf Fertig. Alle mit der Gerätekamera aufgenommenen Fotos werden als JPEG-Datei gespeichert.

  7. Tippen Sie auf Hochladen.

So fügen Sie Fotos aus dem Apple-Fotoalbum hinzu:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zum Zielordner, in dem Sie ein Foto hinzufügen möchten.

    Wenn der Zielordner obligatorische Kategorien oder Attribute enthält, können Sie diesem Ordner mithilfe der OpenText ECM Everywhere-App kein Foto hinzufügen. Wie Sie bestimmen können, ob der Ordner obligatorische Kategorien oder Attribute hat, erfahren Sie unter Objektdetails anzeigen.

  2. Tippen Sie auf .

  3. Tippen Sie auf Aus Album.

    Wenn Sie eine Fehlermeldung erhalten, stellen Sie sicher, dass in den Datenschutzeinstellungen für Fotos das Hochladen von Fotos mit OpenText ECM Everywhere zulässig ist. Tippen Sie auf der Gerätestartseite auf Einstellungen > Datenschutz > Fotos, und achten Sie darauf, dass der OpenText ECM Everywhere 10-Schalter auf EIN steht.

  4. Navigieren Sie durch die Alben auf Ihrem Gerät, und tippen Sie auf das Foto oder die Fotos, das bzw. die Sie hinzufügen möchten.

  5. Optional Wenn Sie nur eine Datei hochladen, tippen Sie auf das Feld Dateiname, und bearbeiten Sie den Namen des Fotos.

  6. Tippen Sie auf Fertig. Um die Aktion abzubrechen, tippen Sie auf das Symbol X.

So fügen Sie einen Ordner hinzu:

  1. Navigieren Sie zum Zielspeicherort für den neuen Ordner.

  2. Tippen Sie auf .

  3. Tippen Sie auf Neuer Ordner.

  4. Tippen Sie auf das Feld Ordnername, und geben Sie einen Namen für den neuen Ordner an.

  5. Tippen Sie auf Fertig.
