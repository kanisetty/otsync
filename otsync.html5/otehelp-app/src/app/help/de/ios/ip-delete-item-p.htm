Dateien oder Ordner löschen

So löschen Sie eine Datei oder einen Ordner:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zum Zielobjekt.

    Sie können eine Datei nicht löschen, während diese reserviert ist. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet. Um eine reservierte Datei zu löschen, müssen Sie erst die Reservierung aufheben. Weitere Informationen finden Sie unter Dateien reservieren und Reservierungen aufheben.

  2. Wischen Sie mit Ihrem Finger horizontal über den Namen der Datei oder des Ordners. In der Objektzeile wird die Schaltfläche Löschen angezeigt.

  3. Tippen Sie auf Löschen. Durch das Löschen einer Datei oder eines Ordners wird das Objekt aus dem Unternehmensarbeitsbereich oder Persönlichen Arbeitsbereich entfernt, selbst wenn das Objekt als Favorit markiert ist.

  4. Tippen Sie auf Dieses Objekt löschen, um diese Aktion zu bestätigen.

    Sie können eine Datei nicht löschen, während diese reserviert ist. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet. Um eine reservierte Datei zu löschen, müssen Sie erst die Reservierung aufheben. Weitere Informationen finden Sie unter Dateien reservieren und Reservierungen aufheben.

  2. Tippen Sie auf die Datei, die Sie entfernen möchten. Die Datei wird geöffnet.

  3. Tippen Sie auf . Durch das Löschen einer Datei wird diese aus dem Ordner Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich entfernt, selbst wenn die Datei als Favorit markiert ist.

  4. Tippen Sie auf Dieses Objekt löschen, um diese Aktion zu bestätigen.
