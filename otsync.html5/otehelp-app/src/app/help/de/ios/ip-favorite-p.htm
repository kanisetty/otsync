Favoriten hinzufügen, anzeigen und entfernen

 

So fügen Sie eine Favoritendatei oder einen Favoritenordner hinzu:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei oder zum Zielordner.

  2. Tippen Sie auf .

  3. Tippen Sie auf Zu Favoriten hinzufügen. Die Datei oder der Ordner ist mit der Überlagerung in der Ordneransicht markiert. So erhalten Sie in der Favoriten-App einfachen Zugriff auf die Datei oder den Ordner. Bei Ordnern werden die Dateien und Unterordner im Ordner nicht als Favoriten markiert.

    Ein Datensatz Ihrer Favoritendateien wird in Ihrem Profil gespeichert, das spezifisch für Ihr iPad oder iPhone ist. Sie müssen auf jedem Ihrer Smartphones und Tablets, auf denen die OpenText ECM Everywhere-App ausgeführt wird, Dateien als Favoriten markieren.

  4. Tippen Sie auf Zurück, um zu Ihren Dateien zurückzukehren.

So fügen Sie einen Favoriten beim Anzeigen hinzu:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei.

  2. Tippen Sie auf die Datei, die Sie als Favorit markieren möchten. Die Datei wird geöffnet.

  3. Tippen Sie auf . Das Symbol wechselt zu und die Datei wird mit der Überlagerung in der Ordneransicht markiert. Beispiel: . Hierdurch wird die Datei auf Ihr iPad oder iPhone heruntergeladen, sodass Sie sie offline anzeigen können.

  4. Tippen Sie auf Zurück.

So zeigen Sie einen Favoriten an:

  1. Tippen Sie auf Favoriten.

  2. Optional Tippen Sie auf , um sicherzustellen, dass der aktuelle Inhalt angezeigt wird.

  3. Tippen Sie auf die Datei, die Sie anzeigen möchten.

  4. Tippen Sie auf Zurück, um zu Ihren Dateien zurückzukehren.

So entfernen Sie einen Favoriten:

  1. Tippen Sie auf Favoriten bzw. auf Unternemensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei oder zum Zielordner.

  2. Tippen Sie auf für das Objekt und dann auf Favorit entfernen. Die Datei wird nicht mehr in der Liste der Favoriten angezeigt. Dadurch wird die Datei bzw. der Ordner nicht aus Ihrem Arbeitsbereich gelöscht.

  3. Tippen Sie auf Zurück, um zu Ihren Dateien zurückzukehren.
