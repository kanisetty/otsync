Dateien anzeigen, bearbeiten und verwenden

So zeigen Sie ein Objekt an:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei. Achten Sie darauf, dass Sie auf den Ordnernamen und nicht auf tippen, da im letzteren Fall Informationen zum Ordner angezeigt werden. In der Titelleiste wird der Name des aktuell geöffneten Ordners angezeigt.

  2. Optional Tippen Sie auf , um sicherzustellen, dass der aktuelle Inhalt angezeigt wird. Dadurch wird die Anzeige aktualisiert, aber es werden keine Dateien heruntergeladen. Um eine Datei herunterzuladen, öffnen Sie sie.

  3. Tippen Sie auf den Namen des Objekts, das Sie öffnen möchten. Achten Sie darauf, dass Sie auf den Ordnernamen und nicht auf tippen, da im letzteren Fall die Eigenschaften des Objekts angezeigt werden.

  4. Tippen Sie auf Zurück, um zu Ihren Dateien zurückzukehren.

So bearbeiten Sie ein Objekt:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zum Zielobjekt.

    Sie können keine Bearbeitungen an einer Datei speichern, wenn sie bereits durch einen anderen Kollegen im Unternehmensarbeitsbereich reserviert wurde. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet. Wer die Datei reserviert hat, können Sie anhand des Versionsverlaufs der Datei feststellen. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  2. Tippen Sie auf die Datei, die Sie entfernen möchten. Die Datei wird geöffnet.

  3. Tippen Sie auf , und wählen Sie einen Editor aus.

  4. Bearbeiten Sie die Datei, und speichern Sie Ihre Änderungen.

  5. Tippen Sie im Editor auf , und wählen Sie In Everywhere 10 öffnen aus.

  6. Tippen Sie in OpenText ECM Everywhere auf Neue Version hochladen. Die Änderungen werden zur Originaldatei hinzugefügt.

     

    Hinweis:
    Wenn Sie die Suchfunktion in der OpenText ECM Everywhere-App nutzen, um die zu bearbeitende Datei zu suchen, werden alle Änderungen an der Datei, die in einem Editor vorgenommen wurden, in einer neuen Datei gespeichert und auf der Stammebene abgelegt.

     

Informationen zur Verwendung Ihrer Dateien finden Sie in den anderen Themen unter Dateien anzeigen, bearbeiten und verwenden.
