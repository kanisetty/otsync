Anlagen anzeigen

So zeigen Sie Anlagen an:

  1. Tippen Sie auf Zuweisungen und dann auf eine Workflow-Aufgabe.

  2. Tippen Sie auf Anhänge anzeigen/hinzufügen.

  3. Tippen Sie auf den Namen des Objekts, das Sie öffnen möchten.

  4. Tippen Sie auf Zurück und dann auf Schließen, um zur Workflow-Aufgabe zurückzukehren.
