Dateien reservieren und Reservierungen aufheben

 

So reservieren Sie Dateien und heben Dateireservierungen auf:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei.

  2. Tippen Sie auf , um die Details der Datei, die Sie reservieren oder deren Reservierung Sie aufheben möchten, anzuzeigen.

  3. Führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf Reservieren, um die Datei zu reservieren. Die Datei ist mit der -Überlagerung gekennzeichnet.

      Wenn Sie versuchen, eine Datei in einem Unternehmensordner zu reservieren, für den Sie nur Lese- und Schreibzugriff haben und die Option Reservieren nicht aktiviert ist, besteht für Sie keine Berechtigung zum Reservieren für den Ordner in Content Server. Wenden Sie sich an Ihren Administrator, um diese Berechtigung anzufordern.

    • Tippen Sie auf Reservierung aufheben, um die Dateireservierung aufzuheben. Wenn Reservierung aufheben nicht verfügbar ist, ist die Datei von einem anderen Benutzer reserviert. Wer die Datei reserviert hat, können Sie anhand des Versionsverlaufs der Datei feststellen. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  4. Tippen Sie auf Zurück.
