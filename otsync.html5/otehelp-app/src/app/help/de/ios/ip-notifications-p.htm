Benachrichtigungen aktivieren

So aktivieren Sie Benachrichtigungen:

  1. Tippen Sie auf die Einstellungen für Ihr mobiles Gerät.

  2. Tippen Sie auf Mitteilungszentrale.

  3. Tippen Sie auf Everywhere 10.

  4. Aktivieren Sie den Benachrichtigungstyp, den Sie erhalten möchten.

  5. Tippen Sie bei In Mitteilungszentrale anzeigen auf den Schalter EIN.

 

Hinweis:
Benachrichtigungen für Feeds in der OpenText ECM Everywhere-App unterscheiden sich von denen für Pulse in Content Server, deshalb müssen die Benachrichtigungen für jedes System separat eingerichtet werden.
