Status mitteilen

So teilen Sie Ihren Status mit:

  1. Tippen Sie auf Feeds > Status.

  2. Geben Sie Ihren Kommentar in das Textfeld ein.

    Um Ihren Status einem bestimmten Kollegen mitzuteilen, verwenden Sie eine @Nennung. Weitere Informationen zu Nennungen finden Sie unter Benutzer erwähnen und Erwähnungen anzeigen.

    Informationen zum Hinzufügen einer Anlage zu einem Beitrag finden Sie unter Bild hinzufügen .

  3. Tippen Sie auf Posten.
