Objekte verschieben oder kopieren

So verschieben oder kopieren Sie ein Objekt:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zum Zielobjekt.

    Sie können ein Objekt nur kopieren, wenn es bereits durch einen anderen Kollegen reserviert wurde. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet. Wer die Datei reserviert hat, können Sie anhand des Versionsverlaufs der Datei feststellen. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  2. Tippen Sie auf die Datei, die Sie verschieben oder kopieren möchten. Die Datei wird geöffnet.

  3. Tippen Sie auf , um die Datei zu verschieben, oder auf , um die Datei zu kopieren.

  4. Tippen Sie auf Zurück, und navigieren Sie zum Zielordner, in dem Sie den Inhalt der Zwischenablage platzieren möchten.

  5. Tippen Sie auf , um die Datei einzufügen.

So verschieben oder kopieren Sie einen Ordner:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich, und navigieren Sie zum Zielordner.

  2. Tippen Sie auf , um die Details des Ordners, den Sie verschieben oder kopieren möchten, anzuzeigen.

  3. Tippen Sie auf , um den Ordner zu verschieben, oder auf , um den Ordner zu kopieren.

    Sie können unabhängig von Ihrer Zugriffsebene nur auf die oberste Ebene eines Unternehmensordners zugreifen. Sie können zudem nur Unterordner innerhalb eines Unternehmensordners kopieren, für den Sie Leserzugriff besitzen. Dabei wird eine private Kopie des Ordners an einem neuen Speicherort erstellt. Der ursprüngliche Unternehmensordner bleibt jedoch am ursprünglichen Speicherort.

  4. Tippen Sie auf Zurück, und navigieren Sie zum Zielordner, in dem Sie den Inhalt der Zwischenablage platzieren möchten.

  5. Tippen Sie auf , um den Ordner einzufügen.
