Dateien hinzufügen

 

So fügen Sie ein Objekt hinzu:

  1. Sie müssen das Objekt importieren, indem Sie es auch einer anderen App hochladen, falls die App diese Funktion unterstützt. Zeigen Sie in einer Drittanbieter-App die Datei an, die Sie importieren möchten, um führen Sie die notwendigen Schritte aus, um das Dokument in OpenText ECM Everywhere zu öffnen. Tippen Sie beispielsweise auf , Freigeben oder Öffnen in. Eine Liste von Apps wird angezeigt.

  2. Tippen Sie auf Everywhere 10.

  3. Optional Wenn Sie dazu aufgefordert werden, melden Sie sich bei OpenText ECM Everywhere an.

  4. Tippen Sie auf Zielspeicherort: Persönlicher Arbeitsbereich, um zum Speicherort zu navigieren, in dem Sie diese Datei hinzufügen möchten. Tippen Sie dann auf Auswählen.

  5. Optional Tippen Sie auf das Feld Dateiname, und bearbeiten Sie den Namen der Datei. Tippen Sie auf Fertig.

  6. Tippen Sie auf Hochladen.
