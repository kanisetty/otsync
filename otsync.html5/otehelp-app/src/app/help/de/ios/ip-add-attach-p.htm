Bild hinzufügen

So fügen Sie ein Bild hinzu:

  1. Tippen Sie beim Hinzufügen einer Statusaktualisierung oder Kommentieren eines Beitrags auf Anhängen, und führen Sie dann einen der folgenden Schritte aus:

    • Tippen Sie auf Von Kamera, um ein Bild mithilfe der Apple-Kamera anzuhängen.

      Nehmen Sie das Bild auf, und tippen Sie dann auf Foto verwenden, um das aktuelle Bild zu verwenden, oder tippen Sie auf Erneut aufnehmen, und nehmen Sie das Bild erneut auf.

    • Tippen Sie auf Aus Galerie, um ein Bild aus dem Apple-Fotoalbum anzuhängen.

      Navigieren Sie zu dem Bild, das Sie anhängen möchten, und tippen Sie dann darauf.

    Wenn Sie Ihre Meinung ändern, können Sie auf Entfernen tippen, um die Anlage zu entfernen.

  2. Geben Sie Ihren Kommentar in das Textfeld ein, und tippen Sie auf Posten.
