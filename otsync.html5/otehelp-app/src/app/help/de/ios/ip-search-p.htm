OpenText ECM Everywhere durchsuchen

 

So führen Sie eine Suche durch:

  1. Tippen Sie auf Unternehmensarbeitsbereich oder Persönlicher Arbeitsbereich.

  2. Navigieren Sie zu einem Zielordner, um die Suche zu starten. Wenn Sie den gesamten Arbeitsbereich durchsuchen möchten, bleiben Sie auf der Stammordnerebene.

  3. Geben Sie im Feld Diesen Ordner durchsuchen Ihren Suchbegriff ein.

  4. Tippen Sie auf Suchen. Eine Liste der Suchergebnisse wird angezeigt.

  5. Tippen Sie auf die Datei oder den Ordner, die bzw. den Sie anzeigen möchten.
