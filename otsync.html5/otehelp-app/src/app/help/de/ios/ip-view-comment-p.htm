Kommentare anzeigen

So zeigen Sie einen Beitrag mit einem langen Kommentar an:

  1. Tippen Sie auf Feeds.

  2. Tippen Sie auf den Beitrag, um die vollständigen Details eines langen Kommentars anzuzeigen.

So zeigen Sie alle Kommentare zu einem Beitrag an:

  1. Tippen Sie auf Feeds. Bei Beiträgen mit mehreren Kommentaren wird zusammen mit der Anzahl der hinzugefügten Kommentare angezeigt.

  2. Tippen Sie auf einen Beitrag.

  3. Tippen Sie auf die Verknüpfung Threads anzeigen, um eine Aktualisierung mit mehreren Kommentaren anzuzeigen.

  4. Tippen Sie auf , um zum Beitrag zurückzukehren.

 

Tipp:
Informationen zum Anzeigen von Kommentaren am Objektspeicherort finden Sie unter Kommentare zu Dateien oder Ordnern anzeigen.
