So kommentieren Sie eine Datei oder einen Ordner:

  1. Klicken Sie auf der Startseite auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zu dem Objekt, das Sie kommentieren möchten.

  2. Markieren Sie die Zieldatei oder den Zielordner.

  3. Drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Kommentare aus.

  4. Geben Sie Ihren Kommentar in das Textfeld ein.

    Um Ihren Beitrag einem bestimmten Kollegen mitzuteilen, verwenden Sie eine @Nennung. Weitere Informationen zu Nennungen finden Sie unter Benutzer erwähnen und Erwähnungen anzeigen.

  5. Drücken Sie die BlackBerry Eingabe-Taste.

  6. Drücken Sie die BlackBerry-Escape-Taste, um zu Enterprise Workspace oder Persönlicher Arbeitsbereich zurückzukehren.

 

Tipp:
Weitere Informationen zum Hinzufügen von Aktualisierungen finden Sie unter Beiträge in Feeds filtern, anzeigen, kommentieren und durchsuchen .
