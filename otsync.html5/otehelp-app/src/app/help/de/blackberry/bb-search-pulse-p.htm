So durchsuchen Sie alle Beiträge:

  1. Klicken Sie auf der Startseite auf Feeds.

  2. Drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Suchen aus.

    Geben Sie die Suchzeichenfolge in das Feld zum Durchsuchen von Feeds ein, und drücken Sie dann die BlackBerry-Enter-Taste.

  3. Drücken Sie die BlackBerry-Escape-Taste, um zum regulären Feed zurückzukehren.
