So fügen Sie ein Foto mit der BlackBerry-Kamera hinzu:

  1. Klicken Sie auf der Startseite auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zum Zielordner, in dem Sie ein Foto hinzufügen möchten. Ob Sie ein Foto einem Ordner im Enterprise Workspace hinzufügen können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Foto hinzufügen aus.

  3. Drücken Sie die BlackBerry Trackpad-Taste, um ein Bild aufzunehmen.

  4. Klicken Sie auf Zu OpenText Everywhere hinzufügen. Fotos werden als JPG-Dateien gespeichert. Sie können den Namen des Fotos vor dem Hochladen nicht bearbeiten.

So fügen Sie ein Foto aus einer BlackBerry-App hinzu:

  1. Starten Sie die BlackBerry-Kamera-App.

  2. Navigieren Sie zu dem Foto, das Sie importieren möchten.

  3. Drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Zu OpenText Everywhere hinzufügen aus.

  4. Optional Wenn Sie dazu aufgefordert werden, melden Sie sich bei ECM Everywhere an. Weitere Informationen finden Sie unter Anmelden und Profil verwalten.

  5. Klicken Sie auf Enterprise Workspace oder Persönlicher Arbeitsbereich.

  6. Navigieren Sie zu dem Speicherort, an dem Sie dieses Foto hinzufügen möchten. Ob Sie ein Foto einem Ordner im Enterprise Workspace hinzufügen können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  7. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hier hochladen aus.

So fügen Sie einen Ordner hinzu:

  1. Navigieren Sie in Enterprise Workspace oder Persönlicher Arbeitsbereich zum Zielspeicherort für den neuen Ordner. Ob Sie einen Ordner im Enterprise Workspace hinzufügen können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ordner hinzufügen aus.

  3. Geben Sie im Feld Ordnername einen Namen für diesen Ordner ein.

  4. Drücken Sie die BlackBerry Eingabe-Taste.
