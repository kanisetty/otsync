So zeigen Sie ein Objekt an:

  1. Klicken Sie auf der Startseite auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei, indem Sie einen Ordner markieren und die BlackBerry-Trackpad-Taste drücken. In der Titelleiste wird der Name des aktuell geöffneten Ordners angezeigt.

  2. Optional Um sicherzugehen, dass Sie den aktuellsten Inhalt anzeigen, sollten Sie den Ordnerinhalt regelmäßig aktualisieren. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Aktualisieren aus. Dadurch wird die Anzeige aktualisiert, aber die Dateien werden nicht heruntergeladen. Um eine Datei herunterzuladen, öffnen Sie sie.

  3. Markieren Sie die Datei, die Sie anzeigen möchten.

  4. Drücken Sie die BlackBerry Trackpad-Taste, um die Datei anzuzeigen.

  5. Drücken Sie die BlackBerry Esc-Taste, um zu Ihren Dateien zurückzukehren.

So zeigen Sie einen langen Dateinamen an:

  1. Navigieren Sie in Enterprise Workspace oder Persönlicher Arbeitsbereich zum Objektspeicherort, und markieren Sie das Objekt, das Sie anzeigen möchten.

  2. Drücken Sie die BlackBerry-Menü-Taste, und wählen Sie dann Objektdetails aus. Der vollständige Dateiname wird angezeigt.

So bearbeiten Sie eine Datei:

  1. Klicken Sie auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei.

  2. Laden Sie die Datei auf Ihr BlackBerry-Gerät herunter. Weitere Informationen finden Sie unter Dateien herunterladen.

  3. Öffnen Sie die Datei in der App eines Drittanbieters, die die Bearbeitung unterstützt. Bearbeiten Sie die Datei, und speichern Sie Ihre Änderungen.

  4. Die Änderungen, die Sie in der App eines Drittanbieters an der Datei vornehmen, werden nicht in Ihrem Arbeitsbereich gespeichert. Sie müssen die geänderte Datei aus der Drittanbieter-App oder dem BlackBerry-Cache-Ordner hochladen.

    Laden Sie in der ECM Everywhere-App die bearbeitete Datei durch Hinzufügen einer Version der Datei hoch. Weitere Informationen finden Sie unter Dateien hinzufügen.

Informationen zur Verwendung Ihrer Dateien finden Sie in den anderen Abschnitten unter Dateien anzeigen, bearbeiten und verwenden.
