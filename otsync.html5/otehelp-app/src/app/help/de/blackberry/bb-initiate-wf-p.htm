So initiieren Sie einen Workflow:

  1. Klicken Sie auf der Startseite auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zum Speicherort des Workflow-Diagramms.

    Ob Sie einen Workflow im Enterprise Workspace initiieren können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  2. Klicken Sie auf den Namen des Workflow-Diagramms, das Sie initiieren möchten. Eine Zusammenfassung des Workflow-Diagramms erscheint, in der Sie Attribute bearbeiten, Anhänge anzeigen oder Kommentare hinzufügen können, bevor Sie den Workflow initiieren.

  3. Optional Fügen Sie einen Anhang zum Workflow hinzu. Weitere Informationen finden Sie unter Anlage hinzufügen.

  4. Optional Bearbeiten Sie die Attribute, sofern verfügbar. Weitere Informationen finden Sie unter Zuweisungen anzeigen, bearbeiten und verwenden. Erforderliche Attribute sind mit einem Sternchen markiert.

  5. Optional Geben Sie in das Feld Kommentar die gewünschten Kommentare für den Empfänger des Schritts ein.

  6. Klicken Sie auf Initiieren.

  7. Klicken Sie auf OK.
