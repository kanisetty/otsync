So zeigen Sie Anlagen an:

  1. Klicken Sie auf der Startseite auf Zuweisungen und dann auf > für die Zuweisung, die Sie anzeigen möchten.

  2. Klicken Sie auf Anhänge anzeigen/hinzufügen.

  3. Wählen Sie einen Anhang aus.

  4. Drücken Sie die BlackBerry-Escape-Taste, um zu der Zuweisung zurückzukehren.
