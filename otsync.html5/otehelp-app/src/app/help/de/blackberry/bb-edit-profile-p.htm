So bearbeiten Sie Ihr Profil:

  1. Markieren Sie Ihr Profil auf dem Bildschirm Profil.

     

    Tipp:
    Wenn Sie angemeldet sind, drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Profil wechseln aus, um zum Bildschirm Profil zurückzukehren.

     

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Profil bearbeiten aus.

  3. Ändern Sie nach Bedarf Ihren Profilnamen, Ihren Benutzernamen oder den Server.

  4. Klicken Sie auf Profil speichern.

So aktualisieren Sie das Kennwort:

  1. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Abmelden aus. Sie können Ihr Kennwort nur ändern, wenn Sie online sind.

  2. Markieren Sie auf dem Bildschirm Profil Ihr Profil, und drücken Sie die Taste BlackBerry Trackpad.

  3. Geben Sie Ihr neues Kennwort ein. Beim Kennwort muss die Groß- und Kleinschreibung beachtet werden. Stellen Sie sicher, dass Sie das Kennwort korrekt eingeben.

  4. Drücken Sie die BlackBerry Eingabe-Taste.
