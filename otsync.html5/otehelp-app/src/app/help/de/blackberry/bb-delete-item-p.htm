So löschen Sie eine Datei oder einen Ordner:

  1. Klicken Sie auf der Startseite auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei oder zum Zielordner.

     

    Hinweis:
    Bevor Sie fortfahren, sehen Sie sich unten die Warn- Hinweise an.

     

  2. Markieren Sie die Zieldatei oder den Zielordner.

  3. Drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Objekt löschen oder Ordner löschen aus.

  4. Klicken Sie auf OK.

 

Achtung
Durch das Entfernen einer Datei oder eines Ordners wird das Objekt aus Enterprise Workspace oder Persönlicher Arbeitsbereich entfernt, selbst wenn die Datei als Favorit markiert ist. Das bedeutet, dass die Datei oder der Ordner aus Content Server gelöscht wird.
