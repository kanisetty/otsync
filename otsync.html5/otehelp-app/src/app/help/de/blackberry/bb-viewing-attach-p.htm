So fügen Sie einen Anhang zu einer Zuweisung hinzu:

  1. Klicken Sie auf der Startseite auf Zuweisungen und dann auf > für die Zuweisung, die Sie anzeigen möchten.

  2. Klicken Sie auf Anhänge anzeigen/hinzufügen.

  3. Drücken Sie die BlackBerry-Menü-Taste, und führen Sie eine der folgenden Aktionen aus:

    • Wählen Sie Neue Version hochladen aus, um eine neue Version der Anlage hinzuzufügen. Weitere Informationen finden Sie unter Dateien hinzufügen.

    • Wählen Sie Hier hochladen aus, um eine neue Anlage hinzuzufügen, die sich auf Ihrem BlackBerry befindet. Weitere Informationen finden Sie unter Dateien hinzufügen.

    • Wählen Sie Foto hinzufügen aus, um eine neue Anlage mit einem Bild auf Ihrem BlackBerry hochzuladen. Weitere Informationen finden Sie unter Fotos oder Ordner hinzufügen.

  4. Drücken Sie die BlackBerry-Escape-Taste, um zu der Zuweisung zurückzukehren.
