So melden Sie sich an:

  1. Sie müssen zunächst ein Profil erstellen, bevor Sie sich bei der App anmelden können. Wenn Sie noch kein Profil haben, gehen Sie wie folgt vor:

    1. Markieren Sie auf dem Bildschirm Profil die Option Neues Profil, und drücken Sie die BlackBerry-Trackpad-Taste. Sie müssen online sein, um ein neues Profil zu erstellen und sich anzumelden.

    2. Geben Sie im Feld Profilname einen eindeutigen Profilnamen ein.

    3. Geben Sie in das Feld Benutzername Ihren Content Server-Benutzernamen ein.

    4. Geben Sie Ihr Content Server-Kennwort in das Feld Kennwort ein. Beim Kennwort muss die Groß- und Kleinschreibung beachtet werden. Achten Sie darauf, dass Sie Ihr Kennwort korrekt eingeben.

    5. Geben Sie im Feld Server die URL des ECM Everywhere-Servers (OpenText AppWorks Gateway) ein.

      Die URL des ECM Everywhere-Servers erhalten Sie von Ihrem Administrator.

    6. Aktivieren Sie das Kontrollkästchen Angemeldet bleiben, um beim nächsten Start der App einen sofortigen Zugriff zuzulassen. Dies gilt nicht, wenn Sie sich von ECM Everywhere abmelden oder die Sitzung abläuft. Lesen Sie die Informationen unter Warn- unten, die sich auf diese Option beziehen.

    7. Klicken Sie auf Speichern und anmelden.

  2. Wenn Sie bereits ein Profil haben, gehen Sie wie folgt vor:

    1. Markieren Sie auf der Seite Profil Ihr Profil, und drücken Sie die BlackBerry-Trackpad-Taste.

    2. Geben Sie bei Aufforderung Ihr Kennwort ein. Beim Kennwort muss die Groß- und Kleinschreibung beachtet werden. Stellen Sie sicher, dass Sie das Kennwort korrekt eingeben.

    3. Aktivieren Sie das Kontrollkästchen Angemeldet bleiben, um beim nächsten Start der App einen sofortigen Zugriff zuzulassen. Dies gilt nicht, wenn Sie sich bei der App abmelden oder die Sitzung abläuft. Lesen Sie die Informationen unter Warn- unten, die sich auf diese Option beziehen.

    4. Drücken Sie die BlackBerry Eingabe-Taste.

 

Achtung
Wenn Sie zulassen, dass Sie bei ECM Everywhere angemeldet bleiben, kann dies die Sicherheit beeinträchtigen. Wenn Ihr BlackBerry-Gerät von einem anderen Benutzer verwendet wird und Sie sich nicht bei der ECM Everywhere-App abgemeldet haben, kann der andere Benutzer auf Ihr Content Server-Konto zugreifen. Um sich abzumelden, navigieren Sie zur Startseite, drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Abmelden aus.

 

Informationen zur Verwaltung Ihres Profils finden Sie in den anderen Abschnitten unter Anmelden und Profil verwalten .
