So kommentieren Sie einen Beitrag:

  1. Klicken Sie auf der Startseite auf Feeds.

  2. Klicken Sie auf den Beitrag, den Sie kommentieren möchten.

  3. Klicken Sie auf Antworten.

  4. Geben Sie Ihren Kommentar in das Textfeld ein.

  5. Optional Um Ihren Beitrag einem bestimmten Kollegen mitzuteilen, verwenden Sie eine @Nennung. Weitere Informationen zu Nennungen finden Sie unter Benutzer erwähnen und Erwähnungen anzeigen.

  6. Klicken Sie auf Posten.

  7. Drücken Sie die BlackBerry-Escape-Taste, um zum regulären Feed zurückzukehren.
