So zeigen Sie einen Favoriten an:

  1. Klicken Sie auf Favoriten, und wählen Sie dann die Datei oder den Ordner aus, die bzw. den Sie anzeigen möchten.

  2. Optional Um sicherzustellen, dass sämtliche Favoriten angezeigt werden, aktualisieren Sie die Anzeige regelmäßig. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Aktualisieren aus. Dabei werden die Dateien nicht zur Offline-Anzeige heruntergeladen. Um eine Datei herunterzuladen, synchronisieren Sie Ihre Favoriten, oder öffnen Sie die Datei. Weitere Informationen finden Sie unter Favoriten manuell synchronisieren.
