So verschieben oder kopieren Sie eine Datei oder einen Ordner:

  1. Klicken Sie auf der Startseite auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei oder zum Zielordner.

  2. Markieren Sie die Datei oder den Ordner, die bzw. den Sie verschieben oder kopieren möchten.

  3. Drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Ausschneiden aus, um das Objekt zu verschieben, oder wählen Sie Kopieren aus, um das Objekt zu kopieren.

  4. Navigieren Sie zum Zielordner, in dem Sie den Inhalt der Zwischenablage platzieren möchten.

    Ob Sie ein Objekt in einen Ordner im Enterprise Workspace verschieben oder kopieren können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  5. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Einfügen aus.
