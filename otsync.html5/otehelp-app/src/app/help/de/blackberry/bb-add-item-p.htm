So fügen Sie eine neue Version mithilfe einer Datei auf Ihrem BlackBerry hinzu:

  1. Klicken Sie auf der Startseite auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zur Zieldatei.

  2. Markieren Sie die Zieldatei, zu der Sie eine neue Version hinzufügen möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Neue Version hochladen aus.

  4. Navigieren Sie zur neuen Version der Datei auf Ihrem BlackBerry.

  5. Markieren Sie die Datei.

  6. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hochladen aus. Ob Sie einer Datei im Enterprise Workspace eine neue Version hinzufügen können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

So laden Sie eine auf Ihrem BlackBerry gespeicherte Datei hoch:

  1. Navigieren Sie in Enterprise Workspace oder Persönlicher Arbeitsbereich zum Zielordner, in den Sie eine Datei hochladen möchten.

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hier hochladen aus.

  3. Navigieren Sie auf Ihrem BlackBerry-Smartphone zur Zieldatei. Ob Sie eine Datei in einen Ordner im Enterprise Workspace hochladen können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  4. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hochladen aus.

So importieren Sie eine Datei aus einer BlackBerry-App:

  1. Starten Sie eine BlackBerry-App, und navigieren Sie zu der Zieldatei, die Sie importieren möchten.

  2. Drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Zu OpenText Everywhere hinzufügen aus.

  3. Optional Wenn Sie dazu aufgefordert werden, melden Sie sich bei ECM Everywhere an. Weitere Informationen finden Sie unter Anmelden und Profil verwalten.

  4. Klicken Sie auf Enterprise Workspace oder Persönlicher Arbeitsbereich.

  5. Navigieren Sie zu dem Speicherort, an dem Sie diese Datei hinzufügen möchten. Ob Sie eine Datei einem Ordner im Enterprise Workspace hinzufügen können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  6. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hier hochladen aus.
