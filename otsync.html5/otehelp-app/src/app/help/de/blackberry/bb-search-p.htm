So durchsuchen Sie ECM Everywhere:

  1. Klicken Sie auf der Startseite auf Enterprise Workspace oder Persönlicher Arbeitsbereich, und navigieren Sie zum Zielordner, in dem Sie Ihre Suche beginnen möchten.

  2. Verwenden Sie das BlackBerry-Trackpad zum Markieren, und wählen Sie .

  3. Geben Sie den Suchbegriff ein.

  4. Drücken Sie die BlackBerry Eingabe-Taste. Ihre Suchergebnisse werden angezeigt.

  5. Markieren Sie die Datei oder den Ordner, die bzw. den Sie anzeigen möchten.

  6. Drücken Sie die BlackBerry Menü-Taste, um die verfügbaren Optionen anzuzeigen.

So öffnen Sie den Speicherort eines Elements aus den Suchergebnissen:

  1. Durchsuchen Sie Enterprise Workspace oder Persönlicher Arbeitsbereich.

  2. Markieren Sie die Datei, die Sie öffnen möchten, am Dateispeicherort.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Speicherort öffnen aus.

  4. Drücken Sie die BlackBerry-Escape-Taste, um zu den Suchergebnissen zurückzukehren.
