So löschen Sie ein Profil:

  1. Markieren Sie auf dem Bildschirm Profil das Profil, das Sie entfernen möchten.

     

    Tipp:
    Wenn Sie angemeldet sind, drücken Sie die BlackBerry-Menü-Taste, und wählen Sie Profil wechseln aus, um zum Bildschirm Profil zurückzukehren.

     

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Profil löschen aus.

  3. Klicken Sie auf OK.
