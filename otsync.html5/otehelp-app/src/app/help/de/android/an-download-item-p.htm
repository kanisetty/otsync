Eine Datei auf Ihr Gerät herunterladen

So laden Sie eine Datei herunter:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, tippen Sie auf Favoriten.

  2. Berühren und halten Sie die Datei, die Sie herunterladen möchten.

  3. Tippen Sie auf Herunterladen.

  4. Navigieren Sie auf Ihrem Android-Gerät zum Zielordner.

  5. Tippen Sie auf die Android Menü-Taste und anschließend auf Herunterladen.
