Favoriten hinzufügen, anzeigen, entfernen und synchronisieren

So fügen Sie eine Favoritendatei oder einen Favoritenordner hinzu:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zur Zieldatei oder zum Zielordner.

  2. Berühren und halten Sie die Datei oder den Ordner, die bzw. den Sie als Favorit kennzeichnen möchten.

  3. Tippen Sie auf Zu Favoriten hinzufügen.

    Die Datei oder der Ordner ist mit der -Überlagerung gekennzeichnet. Beispiel: . So erhalten Sie in der Favoriten-App einfachen Zugriff auf die Datei oder den Ordner. Bei Ordnern wird hierbei eine Verknüpfung zum Ordner erstellt. Die Dateien und Unterordner im Ordner werden nicht als Favoriten markiert.

  4. Optional Wenn das Symbol angezeigt wird, wurde die markierte Datei nicht heruntergeladen. Beispiel: . Um die Datei herunterzuladen, synchronisieren Sie Ihre Favoriten, oder öffnen Sie die Datei. Weitere Informationen finden Sie unter Manuelles Synchronisieren Ihrer Favoriten.

    Informationen zum automatischen Synchronisieren Ihrer Favoritendateien finden Sie unter OpenText ECM Everywhere-Einstellungen ändern.

    Eine Aufzeichnung Ihrer Favoritendateien wird in Ihrem Profil gespeichert. Profile gelten speziell für Ihr Gerät. Sie müssen auf jedem Ihrer Smartphones und Tablets mit der OpenText ECM Everywhere-App ein Profil erstellen und Dateien als Favoriten markieren.

So zeigen Sie einen Favoriten an:

  1. Tippen Sie auf Favoriten.

  2. Optional Um sicherzugehen, dass Sie den aktuellsten Inhalt anzeigen, sollten Sie die Anzeige regelmäßig aktualisieren. Weitere Informationen finden Sie unter Anzeige aktualisieren.

    Dabei werden die Dateien nicht zur Offline-Anzeige heruntergeladen. Um die Datei herunterzuladen, synchronisieren Sie Ihre Favoriten, oder öffnen Sie die Datei. Weitere Informationen finden Sie unter Manuelles Synchronisieren Ihrer Favoriten unten.

So öffnen Sie den Speicherort einer Favoritendatei:

  1. Tippen Sie auf Favoriten.

  2. Berühren und halten Sie die Datei, die Sie an dem Dateispeicherort öffnen möchten.

  3. Tippen Sie auf Speicherort öffnen.

So entfernen Sie einen Favoriten:

  1. Tippen Sie auf Favoriten, bzw. tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zur Zieldatei.

  2. Berühren und halten Sie die Datei, die Sie aus der Liste Ihrer Favoriten entfernen möchten.

  3. Tippen Sie auf Aus Favoriten entfernen. Die Datei wird nicht mehr in der Liste der Favoriten angezeigt. Dadurch wird die Datei nicht aus Ihrem Arbeitsbereich gelöscht.

So synchronisieren Sie Favoriten manuell:

  1. Tippen Sie auf Favoriten.

  2. Tippen Sie auf Synchronisieren. Wenn die App gerade die Synchronisierung der Favoriten ausführt oder Sie offline sind, ist Synchronisieren nicht verfügbar.

     

    Hinweis:
    Durch Tippen auf Aktualisieren wird die Anzeige der Seite Favoriten aktualisiert. Ihre Favoritendateien werden dabei nicht zur Offline-Anzeige auf Ihr Gerät heruntergeladen. Um die Datei herunterzuladen, müssen Sie Ihre Favoritendateien synchronisieren oder die Datei öffnen.

     

    Informationen zum automatischen Synchronisieren Ihrer Favoritendateien finden Sie unter OpenText ECM Everywhere-Einstellungen ändern.
