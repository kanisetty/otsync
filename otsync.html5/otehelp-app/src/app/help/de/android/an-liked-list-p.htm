Anzeigen, wem ein Beitrag gefällt

So zeigen Sie an, wem ein Beitrag gefällt:

  1. Tippen Sie auf Feeds.

  2. Tippen Sie auf einen Beitrag mit dem Symbol Gefällt mir .

  3. Tippen Sie auf die Verknüpfung „Gefällt mir“-Bewertungen anzeigen.

  4. Tippen Sie auf die Android Zurück-Taste, um zum Beitrag zurückzukehren.
