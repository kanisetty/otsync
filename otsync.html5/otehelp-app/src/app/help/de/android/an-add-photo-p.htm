Fotos oder Ordner hinzufügen

So fügen Sie ein Foto mit der Android-Kamera hinzu:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Zielordner, wo Sie ein Foto hinzufügen möchten.

    Sie können einem Ordner, der obligatorische Kategorien und Attribute enthält, kein Foto mithilfe der OpenText ECM Everywhere-App hinzufügen. Zeigen Sie die Objektdetails an, um herauszufinden, ob der Ordner obligatorische Kategorien und Attribute enthält. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  2. Tippen Sie auf Hinzufügen und dann auf Von Kamera.

  3. Nehmen Sie ein Foto auf, und folgen Sie den Anweisungen auf dem Bildschirm. Fotos werden als JPG-Dateien gespeichert. Sie können den Namen des Fotos vor dem Hochladen nicht bearbeiten.

So laden Sie ein Foto aus der Galerie auf dem Gerät hoch:

  1. Starten Sie die Android-Galerie-App, und navigieren Sie zum Foto, das Sie importieren möchten.

  2. Tippen Sie auf Freigeben oder Öffnen in. Eine Liste von Apps wird angezeigt.

  3. Tippen Sie auf Everywhere 10.

  4. Optional Wenn Sie dazu aufgefordert werden, melden Sie sich bei OpenText ECM Everywhere an.

  5. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Speicherort, dem Sie dieses Foto.

    Sie können ein Foto nicht mithilfe der OpenText ECM Everywhere-App in einen Ordner hochladen, der obligatorische Kategorien und Attribute enthält. Zeigen Sie die Objektdetails an, um herauszufinden, ob der Ordner obligatorische Kategorien und Attribute enthält. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  6. Tippen Sie auf Einfügen.

So fügen Sie einen Ordner hinzu:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Zielspeicherort für den neuen Ordner.

  2. Tippen Sie auf Hinzufügen und dann auf Ordner .

  3. Tippen Sie auf das Feld Ordnername, und geben Sie einen Namen für diesen Ordner ein.

  4. Tippen Sie auf Ok.
