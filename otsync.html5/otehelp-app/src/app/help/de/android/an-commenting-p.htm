Dateien oder Ordner kommentieren

So kommentieren Sie eine Datei oder einen Ordner:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Objekt, das Sie kommentieren möchten.

  2. Berühren und halten Sie das Objekt.

  3. Tippen Sie auf Kommentare.

  4. Tippen Sie auf das Textfeld, und geben Sie Ihren Kommentar ein.

    Um Ihren Beitrag einem bestimmten Kollegen mitzuteilen, verwenden Sie eine @Nennung. Weitere Informationen zu Nennungen finden Sie unter Benutzer erwähnen und Erwähnungen anzeigen.

    Informationen zum Hinzufügen einer Anlage finden Sie unter Anlage hinzufügen.

  5. Tippen Sie auf Posten.

  6. Tippen Sie auf die Android Zurück-Taste, um zum Arbeitsbereich zurückzukehren.

 

Tipp:
Weitere Informationen zum Hinzufügen von Aktualisierungen finden Sie unter Beiträge in Feeds filtern, anzeigen, kommentieren und durchsuchen.
