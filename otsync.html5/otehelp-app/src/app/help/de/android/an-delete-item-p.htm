Dateien oder Ordner löschen

 

So löschen Sie ein Objekt:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zur Zieldatei oder zum Zielordner.

    Sie können eine Datei nicht löschen, während diese reserviert ist. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet. Um eine reservierte Datei zu löschen, müssen Sie erst die Reservierung aufheben. Weitere Informationen finden Sie unter Dateien reservieren und Reservierungen aufheben.

  2. Berühren Sie den Namen des zu löschenden Objekts, und halten Sie ihn fest.

  3. Tippen Sie auf Löschen. Durch das Löschen einer Datei oder eines Ordners wird das Objekt aus dem Enterprise Workspace oder Personal Workspace entfernt, selbst wenn das Objekt als Favorit markiert ist.

  4. Tippen Sie auf Ok, um diesen Vorgang zu bestätigen.
