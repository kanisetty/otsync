Initiieren eines Workflows

So initiieren Sie einen Workflow:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Speicherort des Workflow-Diagramms.

    Ob Sie einen Workflow im Enterprise Workspace initiieren können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  2. Tippen Sie auf den Namen des Workflow-Diagramms, das Sie initiieren möchten. Eine Zusammenfassung des Workflow-Diagramms erscheint, in der Sie Attribute bearbeiten, Anhänge anzeigen oder Kommentare hinzufügen können, bevor Sie den Workflow initiieren.

    In OpenText ECM Everywhere angezeigte Workflow-Zuweisungen unterstützen keine HTML-Tags in den Anweisungs- und Kommentarbereichen. Wenn die Anweisungen nicht korrekt dargestellt werden, entfernen Sie die HTML-Tags aus dem Workflow-Diagramm in Content Server.

  3. Optional Bearbeiten Sie die Attribute, sofern verfügbar. Erforderliche Attribute sind mit einem Sternchen markiert. Weitere Informationen finden Sie unter Zuweisungen anzeigen, bearbeiten und verwenden.

  4. Optional Geben Sie in das Feld Kommentar die gewünschten Kommentare für den Empfänger des Schritts ein.

  5. Tippen Sie auf Initiieren.

    Die Schaltfläche Initiieren steht erst zur Verfügung, wenn die erforderlichen Attribute ausgefüllt wurden.
