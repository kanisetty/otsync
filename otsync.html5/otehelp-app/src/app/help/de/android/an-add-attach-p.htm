Anlage hinzufügen

 

So fügen Sie eine Anlage hinzu:

  • Tippen Sie beim Hinzufügen einer Statusaktualisierung oder Kommentieren eines Beitrags auf Anhängen, und führen Sie dann einen der folgenden Schritte aus:

    • Tippen Sie auf Von Kamera, um ein Bild mithilfe der Android-Kamera anzuhängen. Nehmen Sie ein Foto auf, und folgen Sie den Anweisungen auf dem Bildschirm, um das Bild zu speichern.

    • Tippen Sie auf die Option zum Hinzufügen aus dem Dateisystem, um eine auf dem Android-Gerät gespeicherte Datei anzuhängen. Navigieren Sie zur Datei, und tippen Sie dann darauf.

    • Tippen Sie auf Aus Galerie, um ein Bild aus der Android-Fotogalerie anzuhängen. Navigieren Sie zum Bild, und tippen Sie dann darauf.
