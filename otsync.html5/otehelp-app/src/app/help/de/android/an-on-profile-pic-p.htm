Profilbild ändern

 

So ändern Sie Ihr Profilbild:

  1. Tippen Sie im OpenText ECM Everywhere Landing- Bildschirm auf Einstellungen.

  2. Tippen Sie auf die Zeile Profilfoto, und führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf Von Kamera, um das Profilbild mithilfe der Android-Kamera festzulegen. Nehmen Sie ein Foto auf, und folgen Sie den Anweisungen auf dem Bildschirm, um das Bild zu speichern.

    • Tippen Sie auf Aus Galerie, um Ihr Profilbild mit einem Foto aus der Android Fotogalerie festzulegen. Navigieren Sie zu dem Bild, das Sie verwenden möchten, und tippen Sie dann darauf.

    • Tippen Sie auf Als Standard festlegen, um als Profilbild das standardmäßige Profilbild zu verwenden.

  3. Tippen Sie auf die Android Zurück-Taste, um zum Bildschirm Einstellungen zurückzukehren.

  4. Tippen Sie auf die Android Zurück-Taste, um zum Landing-Bildschirm zurückzukehren.
