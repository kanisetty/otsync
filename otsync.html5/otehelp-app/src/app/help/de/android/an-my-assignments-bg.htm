Zuweisungen anzeigen, bearbeiten und verwenden

 

So zeigen Sie Ihre Zuweisungen an:

  1. Tippen Sie auf Zuweisungen und dann auf > für den Workflow, den Sie anzeigen möchten. In OpenText ECM Everywhere angezeigte Workflow-Zuweisungen unterstützen keine HTML-Tags in den Anweisungs- und Kommentarbereichen. Wenn die Anweisungen nicht korrekt dargestellt werden, entfernen Sie die HTML-Tags aus dem Workflow-Diagramm in Content Server.

  2. Bearbeiten Sie die Attribute.

  3. Tippen Sie auf Senden am.

So bearbeiten Sie Attribute:

  1. Tippen Sie auf Zuweisungen und dann auf > für den Workflow, den Sie anzeigen möchten.

  2. Tippen Sie auf die Attributzeile, und geben Sie die erforderlichen Informationen ein, oder tippen Sie auf die Liste der bereitgestellten Optionen, um eine Option zu wählen.

    Erforderliche Attribute sind mit einem Sternchen markiert.

  3. Tippen Sie auf Senden am.

So fügen Sie im Attribut Benutzerfeld einen Kollegen hinzu:

  1. Tippen Sie in Ihrer Zuweisungsliste unter Zuweisungen auf eine Workflow-Aufgabe.

  2. Tippen Sie auf das Benutzerfeld-Attribut.

  3. Tippen Sie auf das Textfeld.

  4. Geben Sie den Namen Ihres Kollegen ein.

  5. Wählen Sie Ihren Kollegen in der Ergebnisliste per Fingertipp aus.

Informationen zur Verwendung von Zuweisungen finden Sie unter Zuweisungen anzeigen, bearbeiten und verwenden .
