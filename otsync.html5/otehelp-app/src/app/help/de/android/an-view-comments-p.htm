Kommentare zu Dateien oder Ordnern anzeigen

 

So zeigen Sie Kommentare zu einer Datei oder einem Ordner an:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Zielordner.

  2. Berühren und halten Sie das Objekt.

  3. Führen Sie einen der folgenden Schritte aus:

    • Tippen Sie auf Kommentare, um alle Beiträge zum gewählten Objekt anzuzeigen.

    • Tippen Sie auf Kommentare von hier, um alle Kommentare in jeder Datei und jedem Ordner in einem Ordner anzuzeigen.

  4. Tippen Sie auf die Android Zurück-Taste, um zum Arbeitsbereich zurückzukehren.
