Kommentare anzeigen

 

So zeigen Sie alle Kommentare zu einem Beitrag in Feeds an:

  1. Tippen Sie auf Feeds. Bei Beiträgen mit mehreren Kommentaren wird zusammen mit der Anzahl der hinzugefügten Kommentare angezeigt.

  2. Tippen Sie auf einen Beitrag.

  3. Tippen Sie auf die Verknüpfung Thread anzeigen, um eine Aktualisierung mit mehreren Kommentaren anzuzeigen.

  4. Tippen Sie auf die Android Zurück-Taste, um zur Feeds-App zurückzukehren.

 

Tipp:
Informationen zum Anzeigen aller Kommentare in jeder Datei und jedem Ordner in einem Ordner am Objektspeicherort finden Sie unter Kommentare zu Dateien oder Ordnern anzeigen .

 

So zeigen Sie lange Beiträge an:

  1. Tippen Sie auf Feeds.

  2. Tippen Sie auf den Beitrag, um die vollständigen Details eines langen Kommentars anzuzeigen.
