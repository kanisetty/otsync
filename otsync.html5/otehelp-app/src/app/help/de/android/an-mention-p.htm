Benutzer erwähnen und Erwähnungen anzeigen

 

So erstellen Sie für einen Benutzer eine Nennung:

  • Geben Sie beim Hinzufügen einer Statusaktualisierung oder beim Kommentieren eines Beitrags @<Benutzername> ohne Leerzeichen dazwischen ein. Dadurch wird der Kommentar der Liste der Pulse-Erwähnungen in Ihrem Profil hinzugefügt.

So zeigen Sie die Beiträge an, in denen Sie erwähnt wurden:

  1. Tippen Sie auf Filter und dann auf Pulse-Erwähnungen.

  2. Um zu einem regulären Feed zurückzukehren, tippen Sie auf Filter und dann auf Pulse-Status, Pulse-Inhalt, Pulse-Benutzer, denen ich folge oder Alles.
