Dateien anzeigen, bearbeiten und verwenden

 

So zeigen Sie ein Objekt an:

  1. Navigieren Sie in Enterprise Workspace oder Personal Workspace zum Zielelement.

  2. Optional Um sicherzugehen, dass Sie den aktuellsten Inhalt anzeigen, sollten Sie die Anzeige regelmäßig aktualisieren. Tippen Sie auf Menü und dann auf Aktualisieren.

  3. Tippen Sie auf die Datei, die Sie anzeigen möchten.

     

    Tipp:
    Sie können die anzuzeigende Datei auch berühren und halten und dann auf Öffnen tippen. Eine Liste von Apps, die diese Datei öffnen können, wird angezeigt. Tippen Sie auf eine App.

     

  4. Tippen Sie auf die Android Zurück-Taste, um zu Ihren Dateien zurückzukehren.

So bearbeiten Sie ein Objekt:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Zielobjekt. Um eine Datei zu bearbeiten, müssen Sie diese in eine App eines Drittanbieters exportieren und die bearbeitete Datei dann wieder in OpenText ECM Everywhere importieren.

    Sie können keine Bearbeitungen an einer Datei speichern, wenn sie bereits durch einen anderen Kollegen im Enterprise Workspace reserviert wurde. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet. Wer die Datei reserviert hat, können Sie anhand des Versionsverlaufs der Datei feststellen. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  2. Berühren und halten Sie das Objekt.

  3. Tippen Sie auf E-Mail / Exportieren.

  4. Tippen Sie auf eine App eines Drittanbieters.

  5. Bearbeiten Sie die Datei in der App des Drittanbieters, und speichern Sie Ihre Änderungen auf dem Gerät.

  6. In OpenText ECM Everywhere können Sie der Originaldatei nun eine neue Version hinzufügen. Weitere Informationen finden Sie unter Dateien hinzufügen.

    Wenn Sie die Datei aus der App eines Drittanbieters importieren, wird die neue Datei in OpenText ECM Everywhere hinzugefügt.

Weitere Informationen zur Verwendung von Dateien finden Sie unter Dateien anzeigen und verwenden.
