Beitrag mit Gefällt mir markieren

 

So teilen Sie in Feeds mit, dass Ihnen ein Beitrag gefällt:

  1. Tippen Sie auf Feeds.

  2. Tippen Sie auf einen Beitrag.

  3. Tippen Sie auf Gefällt mir. Um die Aktion zu entfernen, klicken Sie auf Entfernen.

  4. Tippen Sie auf die Android Zurück-Taste, um zur Feeds-App zurückzukehren.

So teilen Sie am Objektspeicherort mit, dass Ihnen ein Beitrag gefällt:

  1. Navigieren Sie in Enterprise Workspace oder Personal Workspace zu einem Zielelement.

  2. Berühren und halten Sie das Element, und tippen Sie dann auf Kommentare.

  3. Tippen Sie auf einen Beitrag und anschließend auf Gefällt mir.

  4. Tippen Sie auf die Android Zurück-Taste, um zum Arbeitsbereich zurückzukehren.
