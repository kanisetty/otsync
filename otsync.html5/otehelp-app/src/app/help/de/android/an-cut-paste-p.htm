Dateien oder Ordner verschieben oder kopieren

So verschieben oder kopieren Sie eine Datei oder einen Ordner:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Zielobjekt.

    Sie können ein Objekt nur kopieren, wenn es bereits durch einen anderen Kollegen im Enterprise Workspace reserviert wurde. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet. Wer die Datei reserviert hat, können Sie anhand des Versionsverlaufs der Datei feststellen. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  2. Berühren und halten Sie das Objekt, das Sie verschieben oder kopieren möchten.

  3. Tippen Sie auf Ausschneiden, um das Objekt zu verschieben, oder auf Kopieren, um das Objekt zu kopieren.

  4. Navigieren Sie zum Zielordner, in dem Sie den Inhalt der Zwischenablage platzieren möchten.

    Sie können ein Objekt nicht mithilfe der OpenText ECM Everywhere-App in einen Ordner verschieben oder kopieren, der obligatorische Kategorien und Attribute enthält. Zeigen Sie die Objektdetails an, um herauszufinden, ob der Ordner obligatorische Kategorien und Attribute enthält. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  5. Tippen Sie auf Einfügen.
