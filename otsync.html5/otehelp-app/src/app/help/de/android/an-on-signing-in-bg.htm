Ihr Profil bearbeiten und verwalten

 

So bearbeiten Sie Ihr Profil:

  1. Berühren und halten Sie Ihr Profil im Bildschirm Profil, und tippen Sie dann auf Bearbeiten.

     

    Tipp:
    Wenn Sie angemeldet sind, tippen Sie auf die Android Zurück-Taste, um zum Bildschirm Profil zurückzukehren.

     

  2. Ändern Sie bei Bedarf die Informationen zu Benutzername oder Server.

  3. Tippen Sie auf Änderungen speichern.

So aktualisieren Sie das Kennwort:

  1. Tippen Sie im Bildschirm Profil auf Ihr Profil.

     

    Tipp:
    Wenn Sie angemeldet sind, tippen Sie auf die Android Zurück-Taste, um zum Bildschirm Profil zurückzukehren.

     

  2. Tippen Sie auf das Feld Kennwort, und geben Sie das neue Kennwort ein. Beim Kennwort muss die Groß- und Kleinschreibung beachtet werden. Achten Sie darauf, dass Sie Ihr Kennwort korrekt eingeben.

  3. Tippen Sie auf Anmelden.

Informationen zur Verwaltung Ihres Profils finden Sie in den anderen Themen unter Ihr Profil bearbeiten und verwalten .
