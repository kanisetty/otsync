Dateien hinzufügen

 

So fügen Sie eine neue Version mithilfe einer Datei auf Ihrem Gerät hinzu:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zur Zieldatei.

  2. Berühren und halten Sie die Datei, der Sie eine neue Version hinzufügen möchten.

    Sie können einer Datei keine neue Version hinzufügen, wenn sie bereits durch einen anderen Kollegen im Enterprise Workspace reserviert wurde. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet. Wer die Datei reserviert hat, können Sie anhand des Versionsverlaufs der Datei feststellen. Weitere Informationen finden Sie unter Objektdetails anzeigen.

    Ob Sie einer Datei im Enterprise Workspace eine neue Version hinzufügen können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  3. Tippen Sie auf Version hinzufügen und dann auf Von Gerät.

  4. Navigieren Sie zur neuen Version der Datei auf Ihrem Gerät.

  5. Tippen Sie auf die Datei.

So laden Sie eine auf Ihrem Gerät gespeicherte Datei hoch:

  1. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Zielordner, wo Sie eine Datei hochladen möchten.

  2. Berühren und halten Sie den Zielordner.

    Sie können eine Datei nicht mithilfe der OpenText ECM Everywhere-App in einen Ordner hochladen, der obligatorische Kategorien und Attribute enthält. Zeigen Sie die Objektdetails an, um herauszufinden, ob der Ordner obligatorische Kategorien und Attribute enthält. Weitere Informationen finden Sie unter Objektdetails anzeigen.

    Ob Sie eine Datei in einen Ordner im Enterprise Workspace hochladen können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

  3. Tippen Sie auf Hier hochladen, und führen Sie dann einen der folgenden Schritte aus:

    • Tippen Sie auf Von Kamera, um mit Ihrem Telefon ein Bild aufzunehmen und dieses in den Zielordner hochzuladen. Fotos werden als JPG-Dateien gespeichert. Sie können den Namen des Fotos vor dem Hochladen nicht bearbeiten.

    • Tippen Sie auf Von Gerät, um eine auf dem Gerät gespeicherte Datei hochzuladen. Navigieren Sie zur Zieldatei auf Ihrem Android-Gerät, und tippen Sie auf die Datei.

    • Tippen Sie auf Aus Galerie, um ein Bild aus der Fotogalerie des Geräts hochzuladen. Navigieren Sie zur Gerätegalerie, und tippen Sie auf das Bild.

So importieren Sie eine Datei aus einer anderen App:

  1. Zeigen Sie die hinzuzufügende Datei in der App eines Drittanbieters an.

  2. Tippen Sie auf Freigeben oder Öffnen in. Eine Liste von Apps wird angezeigt.

  3. Tippen Sie auf Everywhere 10.

  4. Optional Wenn Sie dazu aufgefordert werden, melden Sie sich bei OpenText ECM Everywhere an.

  5. Tippen Sie auf Enterprise Workspace oder Personal Workspace, und navigieren Sie zum Speicherort, dem Sie diese Datei hinzufügen möchten. Ob Sie eine Datei in einen Ordner im Enterprise Workspace importieren können, hängt von Ihren Berechtigungen für den ausgewählten Ordner ab.

    Sie können eine Datei nicht mithilfe der OpenText ECM Everywhere-App in einen Ordner importieren, der obligatorische Kategorien und Attribute enthält. Zeigen Sie die Objektdetails an, um herauszufinden, ob der Ordner obligatorische Kategorien und Attribute enthält. Weitere Informationen finden Sie unter Objektdetails anzeigen.

  6. Tippen Sie auf Einfügen.
