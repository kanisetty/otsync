Viewing Comments on a File or Folder

To view comments on a file or folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target folder.

  2. Touch and hold the item.

  3. Do one of the following:

    • Tap Comments to see all of the posts associated with the item selected.

    • Tap Comments From Here to see all of the comments added to every file and folder within a folder.

  4. Tap the Android Back key to return to the workspace.
