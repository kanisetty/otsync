Viewing and Adding Attachments

To view attachments:

  1. Tap Assignments and then tap for the workflow you want to view.

  2. Tap View/Add Attachments.

  3. To view an attachment, tap the name of the attachment. To view the attachment properties, touch and hold the attachment and then tap Object Details.

  4. Tap the Android Back key to return to the workflow task.

To add a version to an attachment:

  1. Tap Assignments and then tap for the workflow you want to view.

  2. Tap View/Add Attachments.

  3. Touch and hold the attachment and then tap Add Version. Tap From Device. Navigate to the new version of the file on your device and tap the file.

  4. Tap the Android Back key to return to the workflow task.

To add a new attachment:

  1. Tap Assignments and then tap for the workflow you want to view.

  2. Tap View/Add Attachments.

  3. Tap Add and then From Device. Navigate to the new version of the file on your device and tap the file.

  4. Tap the Android Back key to return to the workflow task.
