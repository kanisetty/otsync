Adding a File

 

To add a new version using a file on your device:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target file.

  2. Touch and hold the file to which you want to add a new version.

    You cannot add a new version to a file when it is reserved by another colleague in the Enterprise Workspace. Reserved files are marked with the overlay. To determine who reserved the file, review the audit history of the file. For details, see Viewing Object Details.

    Your ability to add a version to a file in the Enterprise Workspace is dependant on your permissions for the selected folder.

  3. Tap Add Version and then tap From Device.

  4. Navigate to the new version of the file on your device.

  5. Tap the file.

To upload a file stored on your device:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target folder where you want to upload a file.

  2. Touch and hold the target folder.

    You cannot upload a file to a folder that has mandatory categories and attributes using the OpenText ECM Everywhere app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.

    Your ability to upload a file to a folder in the Enterprise Workspace is dependant on your permissions for the selected folder.

  3. Tap Upload Here and do one of the following:

    • Tap From Camera to take a picture using your phone and upload it to the target folder. Photos are saved as .jpg files. You cannot edit the name of the photo before uploading it.

    • Tap From Device to upload a file that is stored on your device. Navigate to the target file on your Android device and tap the file.

    • Tap From Gallery to upload a picture from the device photo gallery. Navigate the device gallery and tap the picture.

To import a file from another app:

  1. From a third-party app, view the file you want to add.

  2. Tap Share or Open In. A list of apps displays.

  3. Tap Everywhere 10.

  4. Optional If prompted, sign in to OpenText ECM Everywhere.

  5. Tap Enterprise Workspace or Personal Workspace and navigate to the location where you want to add this file. Your ability to import a file to a folder in the Enterprise Workspace is dependant on your permissions for the selected folder.

    You cannot import a file to a folder that has mandatory categories and attributes using the OpenText ECM Everywhere app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.

  6. Tap Paste.
