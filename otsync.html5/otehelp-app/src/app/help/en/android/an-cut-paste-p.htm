Moving or Copying a File or Folder

To move or copy a file or folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target item.

    You can only copy an item when it is reserved by another colleague in the Enterprise Workspace. Reserved files are marked with the overlay. To determine who reserved the file, review the audit history of the file. For details, see Viewing Object Details.

  2. Touch and hold the item that you want to move or copy.

  3. Tap Cut to move the item or tap Copy to copy the item.

  4. Navigate to the target folder where you want to place the clipboard content.

    You cannot move or copy an item to a folder that has mandatory categories and attributes using the OpenText ECM Everywhere app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.

  5. Tap Paste.
