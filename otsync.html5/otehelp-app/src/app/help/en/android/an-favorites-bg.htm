Viewing Files When Offline

To view files when offline:

  1. You need to tag the file as a favorite and sync or download the file when you are online. Folders are not available when you are offline.

    For details, see Adding, Viewing, Removing, and Syncing Favorites.

    Files tagged as favorites are indicated by the overlay. For example, .

    If the overlay appears, the favorite file is not downloaded. For example, . To download the file, open the file or sync your favorites when you are online.

  2. Tap Favorites and tap the file you want to view.

    The files you open when you are offline may not be the latest versions because any edits made to files using a client that is online cannot sync to your device.

  3. Tap the Android Back key to return to your files.
