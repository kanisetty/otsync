Changing Your Profile Picture

 

To change your profile picture:

  1. On the OpenText ECM Everywhere Landing screen, tap Settings.

  2. Tap the Profile Photo row and do one of the following:

    • Tap From Camera to set your profile picture using the Android camera. Take the picture and then follow the on-screen instructions to save the picture.

    • Tap From Gallery to set your profile picture using a photo from the Android photo gallery. Navigate to the picture that you want to use and then tap it.

    • Tap Set To Default to set your profile picture using the default profile picture.

  3. Tap the Android Back key to return to the Settings screen.

  4. Tap the Android Back key to return to the Landing screen.
