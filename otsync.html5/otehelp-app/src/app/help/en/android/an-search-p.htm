Searching OpenText ECM Everywhere

To search your OpenText folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target folder from which you want to begin your search.

  2. Tap Menu and then tap Search.

  3. In the search box, type your search string.

  4. Tap Go. A list of search results displays.

  5. Do one of the following:

    • Tap the file or folder that you want to view.

    • Touch and hold the file or folder to display the available options.

  6. Tap the Android Back key to return to your search results.

To open the location of a file from your search results:

  1. Search the Enterprise Workspace or Personal Workspace folders.

  2. Touch and hold the file or folder that you want to open at the file location.

  3. Tap Open Location.
