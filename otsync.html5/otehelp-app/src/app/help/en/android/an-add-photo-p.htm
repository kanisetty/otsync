Adding a Photo or Folder

 

To add a photo using the Android camera:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target folder where you want to add a photo.

    You cannot add a photo to a folder that has mandatory categories and attributes using the OpenText ECM Everywhere app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.

  2. Tap Add and then tap From Camera.

  3. Take the picture and follow the on-screen instructions. Photos are saved as .jpg files. You cannot edit the name of the photo before uploading it.

To upload a photo from the device gallery:

  1. Launch the Android Gallery app and navigate to the photo that you want to import.

  2. Tap Share or Open In. A list of apps displays.

  3. Tap Everywhere 10.

  4. Optional If prompted, sign in to OpenText ECM Everywhere.

  5. Tap Enterprise Workspace or Personal Workspace and navigate to the location where you want to add this photo.

    You cannot upload a photo to a folder that has mandatory categories and attributes using the OpenText ECM Everywhere app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.

  6. Tap Paste.

To add a folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target location for the new folder.

  2. Tap Add and then tap Folder.

  3. Tap the Folder Name box and type a name for this folder.

  4. Tap Ok.
