Sending for Review

 

To send for review:

  1. Tap Assignments.

  2. Tap a workflow task in your list of assignments.

  3. Tap Send For Review.

  4. Tap the Assignee field.

     

    Note:
    The Assignee field is required. The Send for Review button is unavailable until this field is populated.

     

  5. Type the name of the Content Server user youâ€™re sending this workflow step to.

  6. Tap the name of your colleague in the list of results.

  7. Optional Tap the Instructions field and then type the instructions for the reviewer.

  8. Optional Tap the Duration (hours) field and then type the duration, in hours, which the reviewer has to complete this step.

  9. Tap Send For Review.
