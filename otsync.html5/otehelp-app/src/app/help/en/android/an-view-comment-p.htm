Viewing Comments

To view all of the comments on a post in Feeds:

  1. Tap Feeds. Posts with multiple comments display along with the number of comments added.

  2. Tap a post.

  3. Tap the View Thread link to view an update with multiple comments.

  4. Tap the Android Back key to return to the Feeds app.

 

Tip:
To view all of the comments added to every file and folder within a folder at the item location, see Viewing Comments on a File or Folder.

 

To view long posts:

  1. Tap Feeds.

  2. Tap the post to view the full details of a long comment.

  3. Tap the Android Back key to return to the Feeds app.
