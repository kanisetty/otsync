Changing OpenText ECM Everywhere Settings

To change OpenText ECM Everywhere settings:

  1. Tap Settings.

  2. Optional Tap the Default Sort Order row and tap Date or Name to sort your files by file name or edit date of the file.

  3. Optional Tap the Device Storage row and tap a new storage limit for your cached files. If you exceed the set storage limit, the oldest cached files are deleted until the cache size is below the specified limit. The default cache size is 1 GB.

     

    Note:
    The Space Used row shows the total size of your favorite files, not your cached files.

     

  4. Tap the Android Back key to return to the Landing screen.
