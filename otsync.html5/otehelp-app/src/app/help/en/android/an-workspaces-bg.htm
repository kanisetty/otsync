Viewing, Editing, and Working With Your Files

To view an item:

  1. In Enterprise Workspace or Personal Workspace, navigate to the target item.

  2. Optional To ensure you are viewing the most up-to-date content, you should periodically refresh the display. Tap Menu and then tap Refresh.

  3. Tap the file you want to view.

     

    Tip:
    You can also touch and hold the file you want to view and then tap Open. A list of apps that can open the file displays. Tap an app.

     

  4. Tap the Android Back key to return to your files.

To edit an item:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target item. To edit a file, you need to export the file to a third party app and then import the edited file into OpenText ECM Everywhere again.

    You cannot save edits to a file when it is reserved by another colleague in the Enterprise Workspace. Reserved files are marked with the overlay. To determine who reserved the file, review the audit history of the file. For details, see Viewing Object Details.

  2. Touch and hold the item.

  3. Tap Email/Export.

  4. Tap a third party app.

  5. In the third party app, edit the file and save your changes to the device.

  6. In OpenText ECM Everywhere, you can now add a new version to the original file. For details, see Adding a File.

    If you import the file from the third party app, this adds a new file in OpenText ECM Everywhere.

For information on working with your files, refer to Viewing and Working With Your Files.
