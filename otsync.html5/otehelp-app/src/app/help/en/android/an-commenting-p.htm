Commenting on a File or Folder

 

To comment on a file or folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the item you want to comment on.

  2. Touch and hold the item.

  3. Tap Comments.

  4. Tap the text box and type your comment.

    To bring your post to the attention of a specific colleague, use an @ mention. For more information on mentions, see Mentioning a User and Viewing Mentions.

    To add an attachment, see Adding an Attachment.

  5. Tap Post.

  6. Tap the Android Back key to return to the workspace.

 

Tip:
For more information on adding updates, see Filtering, Viewing, Commenting On, and Searching Posts in Feeds.
