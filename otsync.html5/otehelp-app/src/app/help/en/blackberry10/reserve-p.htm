Reserving and Unreserving a File

To reserve or unreserve a file:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target file.

  2. Touch and hold the file that you want to reserve or unreserve.

  3. Do one of the following:

    • Tap Reserve to reserve the file. The file is marked with the overlay.

      If you are trying to reserve a file in an Enterprise folder for which you have read/write access and the Reserve option is not available, you do not have Reserve permission to the folder in Content Server. Contact your administrator to request this permission.

    • Tap Unreserve to unreserve the file. If Unreserve is not available, the file is reserved by another user. To determine who reserved the file, review the audit history of the file. For details, see Viewing Object Details.
