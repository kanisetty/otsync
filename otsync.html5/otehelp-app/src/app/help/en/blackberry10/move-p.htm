Moving or Copying a File or Folder

 

To move or copy a file or folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target item.

    When an item is reserved by another colleague in the Enterprise Workspace, you can only copy it. Reserved files are marked with the overlay. To determine who reserved the file, review the audit history of the file. For details, see Viewing Object Details.

  2. Touch and hold the file or folder that you want to move or copy.

  3. Tap Cut to move the item or tap Copy to copy the item.

  4. Navigate to the target folder where you want to place the item.

    You cannot move or copy an item to a folder that has mandatory categories and attributes using the ECM Everywhere app. To determine if the folder has mandatory categories or attributes, view the object details. For details, see Viewing Object Details.
