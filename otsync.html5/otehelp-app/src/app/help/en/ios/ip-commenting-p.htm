Commenting on a File or Folder

 

To comment on a file or folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the item you want to comment on.

  2. Tap .

  3. Tap Comments.

  4. Tap the text box and type your comment.

    To bring your comment to the attention of a specific colleague, use an @ mention. For more information on mentions, see Mentioning a User and Viewing Mentions.

    To add an attachment to a post, see Adding a Picture.

  5. Tap Post.

  6. Tap to return to the file details screen.

  7. Tap Back to return to your files.

 

Tip:
For more information on adding updates, see Filtering, Viewing, Commenting On, and Searching Updates in Feeds.
