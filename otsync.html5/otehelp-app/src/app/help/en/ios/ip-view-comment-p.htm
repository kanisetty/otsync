Viewing Comments

 

To view a post with a long comment:

  1. Tap Feeds.

  2. Tap the post to view the full details of a long comment.

To view all of the comments on a post:

  1. Tap Feeds. Posts with multiple comments display along with the number of comments added.

  2. Tap a post.

  3. Tap the View threads link to view an update with multiple comments.

  4. Tap to return to the post.

 

Tip:
To view comments at the item location, see Viewing Comments on a File or Folder.
