Searching OpenText ECM Everywhere

To search:

  1. Tap Enterprise Workspace or Personal Workspace.

  2. Navigate to a target folder to begin your search. To search the entire workspace, remain at the root folder.

  3. In the Search this folder box, type your search string.

  4. Tap Search. A list of search results displays.

  5. Tap the file or folder that you want to view.
