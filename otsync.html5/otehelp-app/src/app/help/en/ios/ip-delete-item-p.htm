Deleting a File or Folder

To delete a file or folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target item.

    You cannot delete a file while it is reserved. Reserved files are marked with the overlay. To delete a reserved file, you must first unreserve the file. For details, see Reserving and Unreserving a File.

  2. Swipe your finger horizontally over the name of the file or folder. A Delete button appears in the item row.

  3. Tap Delete. Deleting a file or folder removes it from the Enterprise Workspace or Personal Workspace, even if the item was tagged as a favorite.

  4. Tap Delete this item to confirm this action.

To delete a file while viewing it:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target file.

    You cannot delete a file while it is reserved. Reserved files are marked with the overlay. To delete a reserved file, you must first unreserve the file. For details, see Reserving and Unreserving a File.

  2. Tap the file you want to delete. The file opens.

  3. Tap . Deleting a file removes it from the Enterprise Workspace or Personal Workspace folder, even if the file was tagged as a favorite.

  4. Tap Delete this item to confirm this action.
