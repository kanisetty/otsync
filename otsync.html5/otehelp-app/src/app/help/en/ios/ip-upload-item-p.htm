Adding Photos or Folders

 

To add a photo using the Apple camera:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target folder where you want to add a photo.

    If the target folder that has mandatory categories or attributes, you cannot add a photo to this folder using the OpenText ECM Everywhere app. To determine if the folder has mandatory categories or attribute, see Viewing Object Details.

  2. Tap .

  3. Tap From Camera.

  4. Take the photo.

  5. Tap Use Photo to add the current photo, or tap Retake and repeat step 4.

  6. Optional Tap the File Name box and edit the name of the photo. Tap Done. All photos taken with the device camera are saved as .jpeg files.

  7. Tap Upload.

To add one or more photos from the Apple photo album:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target folder where you want to add a photo.

    If the target folder that has mandatory categories or attributes, you cannot add a photo to this folder using the OpenText ECM Everywhere app. To determine if the folder has mandatory categories or attribute, see Viewing Object Details.

  2. Tap .

  3. Tap From Album.

    If you receive an error message, ensure the privacy setting for Photos allows OpenText ECM Everywhere to upload photos. On the device home page, locate and tap Settings > Privacy > Photos and make sure the OpenText ECM Everywhere 10 switch is ON.

  4. Navigate the albums on your device and tap the photo(s) you want to add.

  5. Optional If you are uploading a single file, tap the File Name box and edit the name of the photo.

  6. Tap Done. To cancel the action, tap the X icon.

To add a folder:

  1. Navigate to the target location for the new folder.

  2. Tap .

  3. Tap New Folder.

  4. Tap the Folder Name box and type a name for the new folder.

  5. Tap Done.
