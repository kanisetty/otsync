Moving or Copying an Item

To move or copy an item:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target item.

    You can only copy an item when it is reserved by another colleague. Reserved files are marked with the overlay. To determine who reserved the file, you can check the audit history of the file. For details, see Viewing Object Details.

  2. Tap the file that you want to move or copy. The file opens.

  3. Tap to move the file or to copy the file.

  4. Tap Back and navigate to the target folder where you want to place the clipboard content.

  5. Tap to paste the file.

To move or copy a folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target folder.

  2. Tap to view the details of the folder that you want to move or copy.

  3. Tap to move the folder or to copy the folder.

    You can only copy the root of an Enterprise folder, regardless of your access level. Also, you can only copy subfolders within an Enterprise folder for which you have read-only access. In doing this, a private copy of the folder is created at the new location, but the original Enterprise folder remains at the original location.

  4. Tap Back and navigate to the target folder where you want to place the clipboard content.

  5. Tap to paste the folder.
