Viewing Comments on a File or Folder

 

To view comments on a file:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target file.

  2. Tap .

  3. Tap Comments.

  4. Tap to return to the file details screen.

  5. Tap Back to return to your files.

To view comments on a folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target folder.

  2. Tap .

  3. Tap Comments From Here. This displays all of the comments made on all of the files and folders within the folder.

  4. Tap to return to the folder details screen.

  5. Tap Back to return to your files.
