Understanding the Enterprise and Personal Workspace Icons

 

The following main icons appear throughout the Enterprise and Personal Workspace apps.

Icon Description
Takes you to the Landing page from an app.
Lets you add a new folder or a photo.
Refreshes the screen display.
Syncs all of the files in the Favorites app, except files in a favorited folder.
Shows the full screen when the iPad is in landscape mode.
Displays the folder structure and file icons when the iPad is in portrait mode.
Lets you view folder details.
