Viewing, Editing, and Working With Your Files

To view an item:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target file. Be sure to tap the folder name and not , which displays details of the folder. The title bar displays the name of the folder you are currently viewing.

  2. Optional Tap to ensure you are viewing the most up-to-date content. This refreshes the display, but does not download any files. To download a file, open it.

  3. Tap the name of the item you want to open. Be sure to tap the filename and not , which displays the itemâ€™s properties.

  4. Tap Back to return to your files.

To edit an item:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target item.

    You cannot save edits to a file when it is reserved by another colleague in the Enterprise Workspace. Reserved files are marked with the overlay. To determine who reserved the file, you can check the audit history of the file. For details, see Viewing Object Details.

  2. Tap the file you want to edit. The file opens.

  3. Tap and select an editor.

  4. Edit the file and save your changes.

  5. In the editor, tap and select Open in Everywhere 10.

  6. In OpenText ECM Everywhere, tap Upload New Version. The changes are added to the original file.

     

    Note:
    If you use the search feature in the OpenText ECM Everywhere app to find the file you want to edit, any changes made to the file in an editor will be saved to a new file and placed at the root level.

     

For information on working with your files, refer to the other topics in Viewing, Editing, and Working With Your Files.
