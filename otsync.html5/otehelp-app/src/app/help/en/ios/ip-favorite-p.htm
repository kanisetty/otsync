Adding, Viewing, Removing, and Syncing Favorites

To add a favorite file or folder:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target file or folder.

  2. Tap .

  3. Tap Add to Favorites. The file or folder is marked with the overlay in the folder view. This gives you easy access to the file or folder from the Favorites app. For folders, it does not favorite the files and subfolders in the folder.

    A record of your favorite files is stored on your profile, which is specific to your iPad or iPhone. You must tag files as favorites on each of your smartphones and tablets that run the OpenText ECM Everywhere app.

  4. Optional To make sure that the file is available when offline, you must open the file or sync your favorites. For details, see Syncing Your Favorites.

  5. Tap Back to return to your files.

To add a favorite while viewing it:

  1. Tap Enterprise Workspace or Personal Workspace and navigate to the target file.

  2. Tap the file you want to tag as a favorite. The file opens.

  3. Tap . The icon changes to and the file is marked with the overlay in the folder view. For example, . This downloads the file to your iPad or iPhone so that you can view the file when offline.

  4. Tap Back.

To view a favorite:

  1. Tap Favorites.

  2. Optional Tap to ensure you are viewing the most up-to-date content.

    This does not download the files for offline viewing. To download the file, sync your favorites or open it. For details, see Syncing Your Favorites below.

  3. Tap the file you want to view.

  4. Tap Back to return to your files.

To remove a favorite:

  1. Tap Favorites, or tap Enterprise Workspace or Personal Workspace and the navigate to the target file or folder.

  2. Tap for the item and then tap Remove Favorite. The file no longer appears in your list of favorites. This does not delete the file or folder from your workspace.

  3. Tap Back to return to your files.

To sync favorites:

  1. Tap Favorites.

  2. Tap to download all of the documents in the Favorites app to your device. This does not download the content inside favorited folders. If you are offline, is not available.
