プロファイルの写真の変更

プロファイルの写真を編集するには:

  1. OpenText ECM Everywhere の [Landing] 画面で、[設定] をタップします。

  2. [プロファイルの写真] 行をタップして、次のいずれかの操作を実行します。

    • Android のカメラを使用してプロファイルの写真を設定するには、[カメラから] をタップします。写真を撮り、画面に表示される指示に従って写真を保存します。

    • Android の写真ギャラリーにある写真を使用してプロファイルの写真を設定するには、[ギャラリーから] をタップします。使用する写真に移動して、写真をタップします。

    • [既定値に設定] をタップして、既定のプロファイル写真を使用してプロファイルの写真を設定します。

  3. Android の [戻る] キーをタップして、[設定] 画面に戻ります。

  4. Android の [戻る] キーをタップして、[Landing] 画面に戻ります。
