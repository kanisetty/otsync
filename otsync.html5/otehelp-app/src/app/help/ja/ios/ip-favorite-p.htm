第4章. お気に入りの追加、表示、削除

 

お気に入りのファイルまたはフォルダーを追加するには:

  1. [エンタープライズ ワークスペース] または [パーソナル ワークスペース] をタップして、ターゲット ファイルまたはフォルダーに移動します。

  2. 次のアイコンをタップします。 .

  3. [お気に入りに追加] をタップします。ファイルまたはフォルダーに のオーバーレイが表示されます。これにより、[お気に入り] アプリケーションのファイルやフォルダーにアクセスしやすくなります。フォルダーでは、フォルダー内のファイルやサブフォルダーをお気に入りに登録できません。

    お気に入りのファイルの記録がプロファイルに保管されます。そのプロファイルは、iPad や iPhone ごとに異なります。OpenText ECM Everywhere アプリを実行するスマートフォンやタブレットごとに、ファイルにお気に入りのタグを付ける必要があります。

  4. ファイルに戻るには、[戻る] をタップします。

表示中にお気に入りを追加するには:

  1. [エンタープライズ ワークスペース] または [パーソナル ワークスペース] をタップして、ターゲット ファイルに移動します。

  2. お気に入りのタグを付けるファイルをタップします。ファイルが開きます。

  3. 次のアイコンをタップします。. アイコンが次のように変化します。 に変わり、フォルダー表示のファイルに のオーバーレイが表示されます。例:. これにより、ユーザーの iPad または iPhone にファイルがダウンロードされ、オフライン時でもファイルを表示することができるようになります。

  4. [戻る] をタップします。

お気に入りを表示するには:

  1. [お気に入り] をタップします。

  2. オプション 最新のコンテンツが表示されるようにするには、 をタップします。

  3. 表示するファイルをタップします。

  4. ファイルに戻るには、[戻る] をタップします。

お気に入りを削除するには:

  1. [お気に入り] をタップするか、または [エンタープライズ ワークスペース] もしくは [パーソナル ワークスペース] をタップしてから、ターゲット ファイルまたはフォルダーに移動します。

  2. アイテムの をタップしてから、[お気に入りの削除] をタップします。ファイルがお気に入りのリストに表示されなくなります。この操作により、ワークスペースからファイルやフォルダーが削除されることはありません。

  3. ファイルに戻るには、[戻る] をタップします。
