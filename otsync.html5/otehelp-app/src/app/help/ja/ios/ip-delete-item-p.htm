ファイルまたはフォルダーの削除

ファイルまたはフォルダーを削除するには:

  1. [エンタープライズ ワークスペース] または [パーソナル ワークスペース] をタップして、ターゲット アイテムに移動します。

    予約されているファイルを削除することはできません。予約されているファイルには、 のオーバーレイが表示されます。予約されているファイルを削除するには、まずそのファイルの予約を解除する必要があります。詳細については、次を参照してください。 ファイルの予約と予約解除.

  2. ファイルまたはフォルダーの名前を指で水平になぞります。アイテム行に [削除] ボタンが表示されます。

  3. [削除] をタップします。ファイルまたはフォルダーを削除すると、お気に入りとしてタグを付けられている場合でも、[エンタープライズ ワークスペース] または [パーソナル ワークスペース] から削除されます。

  4. [このアイテムを削除] をタップして、この操作を確認します。

表示中のファイルを削除するには:

  1. [エンタープライズ ワークスペース] または [パーソナル ワークスペース] をタップして、ターゲット ファイルに移動します。

    予約されているファイルを削除することはできません。予約されているファイルには、 のオーバーレイが表示されます。予約されているファイルを削除するには、まずそのファイルの予約を解除する必要があります。詳細については、次を参照してください。 ファイルの予約と予約解除.

  2. 削除するファイルをタップします。ファイルが開きます。

  3. 次をタップします。. ファイルを削除すると、お気に入りとしてタグを付けられている場合でも、[エンタープライズ ワークスペース] または [パーソナル ワークスペース] から削除されます。

  4. [このアイテムを削除] をタップして、この操作を確認します。
