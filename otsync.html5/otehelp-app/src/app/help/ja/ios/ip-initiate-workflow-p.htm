ワークフローの開始

 

ワークフローを開始するには

  1. [エンタープライズ ワークスペース] または [パーソナル ワークスペース] をタップして、ターゲット アイテムに移動します。

  2. 開始するワークフロー マップの名前をタップします。ワークフロー マップのサマリーが表示されます。ここでは、ワークフローを開始する前に、属性の編集、添付ファイルの表示、コメントの追加を行えます。

    必須属性は、ラベルにアスタリスクでマークされています。必須属性を指定するまで、[開始] ボタンは使用できるようになりません。

    OpenText ECM Everywhere に表示されるワークフロー割り当ての指示領域やコメント領域では、HTML タグはサポートされていません。テキストが適切にレンダリングされない場合は、Content Server のワークフロー マップから HTML タグを削除してください。

  3. オプション 属性があれば、これを編集します。詳細については、次を参照してください。 割り当ての表示、編集、作業.

  4. オプション [コメント] ボックスに、このステップを割り当てられた人に対するコメントを入力します。

  5. [開始] をタップします。
