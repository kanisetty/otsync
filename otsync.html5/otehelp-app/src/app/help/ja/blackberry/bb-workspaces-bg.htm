アイテムを表示するには、次の操作を行います。

  1. [ホーム] ページで [Enterprise Workspace] または [パーソナル ワークスペース] をクリックし、フォルダーをハイライト表示して BlackBerry の [トラックパッド] キーを押すことによって、ターゲットのファイルに移動します。タイトル バーに、現在表示されているフォルダー名が表示されます。

  2. オプション 最新のコンテンツが表示されるようにするには、フォルダー コンテンツを定期的に更新する必要があります。BlackBerry の [メニュー] キーを押し、[最新の情報に更新] を選択します。これによって、表示が更新されますが、ファイルはダウンロードされません。ファイルをダウンロードするには、ファイルを開きます。

  3. 表示するファイルをハイライト表示します。

  4. BlackBerry の [トラックパッド] キーを押し、ファイルを表示します。

  5. BlackBerry の [Escape] キーを押して、ファイルに戻ります。

長いファイル名を表示するには:

  1. [Enterprise Workspace] または [パーソナル ワークスペース] で、表示するアイテムのロケーションに移動し、アイテムをハイライト表示します。

  2. BlackBerry の [メニュー] キーを押し、[オブジェクトの詳細] を選択します。完全なファイル名が表示されます。

ファイルを編集するには:

  1. [Enterprise Workspace] または [パーソナル ワークスペース] をクリックして、ターゲット ファイルに移動します。

  2. ファイルを BlackBerry デバイスにダウンロードします。詳細については、次を参照してください。 ファイルのダウンロード.

  3. 編集をサポートするサードパーティのアプリケーションでファイルを開きます。ファイルを編集し、変更を保存します。

  4. サードパーティのアプリケーションでファイルに対して行った変更は、ワークスペースに保存されません。変更済みのファイルをサードパーティ アプリケーションまたは BlackBerry キャッシュ フォルダーからアップロードする必要があります。

    ECM Everywhere アプリで、ファイルのバージョンを追加して、編集したファイルをアップロードします。詳細については、次を参照してください。 ファイルの追加.

ファイルの操作については、以下の項目にある他のセクションを参照してください。ファイルの表示、編集、作業.
