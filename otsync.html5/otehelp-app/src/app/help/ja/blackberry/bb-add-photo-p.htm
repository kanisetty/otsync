BlackBerry カメラを使用して写真を追加するには:

  1. [ホーム] ページで [Enterprise Workspace] または [パーソナル ワークスペース] をクリックし、写真を追加するターゲット フォルダーに移動します。[Enterprise Workspace] 内のフォルダーに写真を追加できるかどうかは、選択したフォルダーに対する権限によって決まります。

  2. BlackBerry の [メニュー] キーを押し、[写真の追加] を選択します。

  3. BlackBerry の [トラックパッド] キーを押し、写真を撮ります。

  4. [OpenText Everywhere に追加] をクリックします。写真は .jpg ファイルとして保存されます。アップロードする前に写真の名前を編集することはできません。

BlackBerry アプリケーションから写真を追加するには、次の操作を行います。

  1. BlackBerry カメラ アプリケーションを起動します。

  2. インポートする写真がある場所へ移動します。

  3. BlackBerry の [メニュー] キーを押し、[OpenText Everywhere に追加] を選択します。

  4. オプション サインインが求められた場合は、ECM Everywhere にサインインします。詳細については、次を参照してください。 サインインとプロファイルの管理.

  5. [Enterprise Workspace] または [パーソナル ワークスペース] をクリックします。

  6. この写真の追加先とする場所に移動します。[Enterprise Workspace] 内のフォルダーに写真を追加できるかどうかは、選択したフォルダーに対する権限によって決まります。

  7. BlackBerry の [メニュー] キーを押し、[ここにアップロード] を選択します。

フォルダーを追加するには:

  1. [Enterprise Workspace] または [パーソナル ワークスペース] で、新しいフォルダーを追加するターゲットの場所に移動します。[Enterprise Workspace] 内にフォルダーを追加できるかどうかは、選択したフォルダーに対する権限によって決まります。

  2. BlackBerry の [メニュー] キーを押し、[フォルダーの追加] を選択します。

  3. [フォルダー名] ボックスに、このフォルダーの名前を入力します。

  4. BlackBerry の [Enter] キーを押します。
