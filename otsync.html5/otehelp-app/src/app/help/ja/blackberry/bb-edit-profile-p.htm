プロフィールを編集するには:

  1. [プロファイル] 画面で、プロファイルをハイライト表示します。

     

    ヒント:
    すでにサインインしている場合は、BlackBerry の [メニュー] キーを押し、[プロファイルの切り替え] を選択して [プロファイル] 画面に戻ります。

     

  2. BlackBerry の [メニュー] キーを押し、[プロフィールを編集] を選択します。

  3. 必要に応じて、プロファイル名、ユーザー名またはサーバーを変更します。

  4. [プロファイルの保存] をクリックします。

パスワードを更新するには:

  1. BlackBerry の [メニュー] キーを押し、[ログアウト] を選択します。パスワードを変更できるのは、オンラインのときだけです。

  2. [プロファイル] 画面で、プロファイルをハイライト表示し、BlackBerry の [トラックパッド] キーを押します。

  3. 新しいパスワードを入力します。パスワードでは大文字と小文字が区別されます。パスワードを正しく入力していることを確認してください。

  4. BlackBerry の [Enter] キーを押します。
