サインインするには:

  1. アプリケーションにサインインする前に、まず、プロファイルを作成する必要があります。プロファイルを持っていない場合は、以下の操作を行います。

    1. [プロファイル] ページで、[新しいプロファイル] をハイライト表示し、BlackBerry の [トラックパッド] キーを押します。新しいプロファイルを作成してサインインするには、オンラインでなければなりません。

    2. [プロファイル名] ボックスに、固有のプロファイル名を入力します。

    3. [ユーザー名] ボックスに、Content Server ユーザー名を入力します。

    4. [パスワード] ボックスに、Content Server パスワードを入力します。パスワードでは大文字と小文字が区別されます。パスワードを正しく入力していることを確認してください。

    5. [サーバー] ボックスに、ECM Everywhere (OpenText AppWorks Gateway) サーバーの URL を入力します。

      ECM Everywhere サーバーの URL については、管理者にお問い合わせください。

    6. [ログインの続行] チェック ボックスを選択すると、次回アプリケーションを起動したときにすばやくアクセスできます。唯一の例外は、ECM Everywhere からログアウトしている場合やセッションがタイムアウトになっている場合です。このオプションに関する下記の注意 を参照してください。

    7. [保存とログイン] をクリックします。

  2. すでにプロファイルを持っている場合は、以下の操作を行います。

    1. [プロファイル] ページで、プロファイルをハイライト表示し、BlackBerry の [トラックパッド] キーを押します。

    2. 求められた場合は、パスワードを入力します。パスワードでは大文字と小文字が区別されます。パスワードを正しく入力していることを確認してください。

    3. [ログインの続行] チェック ボックスを選択すると、次回アプリケーションを起動したときにすばやくアクセスできます。唯一の例外は、アプリケーションからログアウトしている場合やセッションがタイムアウトになっている場合です。このオプションに関する下記の注意 を参照してください。

    4. BlackBerry の [Enter] キーを押します。

 

注意
ECM Everywhere でログインしたままにすると、セキュリティに問題が生じる可能性があります。ECM Everywhere アプリからログアウトしていないときに、BlackBerry デバイスが別のユーザーに使用された場合、その別のユーザーが Content Server アカウントにアクセスできるようになります。ログアウトするには、[ホーム] ページに移動し、BlackBerry の [メニュー] キーを押して、[ログアウト] を選択します。

 

プロファイルの管理については、以下の項目にある他のセクションを参照してください。サインインとプロファイルの管理 .
