BlackBerry にあるファイルに新しいバージョンを追加するには、次の操作を行います。

  1. [ホーム] ページで、[Enterprise Workspace] または [パーソナル ワークスペース] をクリックして、ターゲット ファイルに移動します。

  2. 新しいバージョンを追加するターゲット ファイルをハイライト表示します。

  3. BlackBerry の [メニュー] キーを押し、[新しいバージョンをアップロード] を選択します。

  4. BlackBerry にあるファイルの新しいバージョンに移動します。

  5. ファイルをハイライト表示します。

  6. BlackBerry の [メニュー] キーを押し、[アップロード] を選択します。[Enterprise Workspace] 内のファイルにバージョンを追加できるかどうかは、選択したフォルダーに対する権限によって決まります。

BlackBerry に保存されているファイルをアップロードするには、次の操作を行います。

  1. [Enterprise Workspace] または [パーソナル ワークスペース] で、ファイルのアップロード先とするフォルダーに移動します。

  2. BlackBerry の [メニュー] キーを押し、[ここにアップロード] を選択します。

  3. BlackBerry スマートフォン上のターゲット ファイルに移動します。[Enterprise Workspace] 内のフォルダーにファイルをアップロードできるかどうかは、選択したフォルダーに対する権限によって決まります。

  4. BlackBerry の [メニュー] キーを押し、[アップロード] を選択します。

BlackBerry アプリケーションからファイルをインポートするには、次の操作を行います。

  1. BlackBerry アプリを起動して、インポートするターゲット ファイルに移動します。

  2. BlackBerry の [メニュー] キーを押し、[OpenText Everywhere に追加] を選択します。

  3. オプション サインインが求められた場合は、ECM Everywhere にサインインします。詳細については、次を参照してください。 サインインとプロファイルの管理.

  4. [Enterprise Workspace] または [パーソナル ワークスペース] をクリックします。

  5. このファイルの追加先とする場所に移動します。[Enterprise Workspace] 内のフォルダーにファイルを追加できるかどうかは、選択したフォルダーに対する権限によって決まります。

  6. BlackBerry の [メニュー] キーを押し、[ここにアップロード] を選択します。
