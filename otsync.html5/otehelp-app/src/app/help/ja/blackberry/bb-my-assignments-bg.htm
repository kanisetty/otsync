割り当てを表示するには:

  1. [ホーム] ページで [割り当て] をクリックし、表示する割り当ての [>] をクリックします。

    アプリに表示されるワークフロー割り当ての指示領域やコメント領域では、HTML タグはサポートされていません。指示が適切にレンダリングされない場合は、Content Server のワークフロー マップから HTML タグを削除してください。

  2. 属性を編集します。

  3. [送信] をクリックします。

属性を編集するには:

  1. [割り当て] をクリックし、表示するワークフローの [>] をクリックします。

  2. 属性行をクリックし、必要な情報を入力するか、表示されたオプションのリストでクリックして選択します。

     

    注意:
    必須属性は、ラベルにアスタリスクでマークされています。

     

  3. [送信] をクリックします。

ユーザー フィールド 属性に同僚を追加するには

  1. [割り当て] をクリックし、表示するワークフローの [>] をクリックします。

  2. ユーザー フィールドの属性をクリックします。

  3. テキスト ボックスをクリックします。

  4. 同僚の姓、名、またはユーザー名を入力します。

  5. 結果のリストから同僚を選択します。

割り当ての操作については、以下の項目にある他のセクションを参照してください。割り当ての表示、編集、作業.
