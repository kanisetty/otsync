ファイルまたはフォルダーを削除するには:

  1. [ホーム] ページで [Enterprise Workspace] または [パーソナル ワークスペース] をクリックし、ターゲット ファイルまたはフォルダーに移動します。

     

    注意:
    作業を進める前に、下記の注意 を参照してください。

     

  2. ターゲット ファイルまたはフォルダーをハイライト表示します。

  3. BlackBerry の [メニュー] キーを押し、[アイテムの削除] または [フォルダーの削除] を選択します。

  4. [OK] をクリックします。

 

注意
ファイルまたはフォルダーを削除すると、お気に入りとしてタグを付けられている場合でも、[Enterprise Workspace] または [パーソナル ワークスペース] から削除されます。つまり、ファイルまたはフォルダーは Content Server から削除されます。
