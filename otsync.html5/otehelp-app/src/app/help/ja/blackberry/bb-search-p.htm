ECM Everywhere を検索するには:

  1. [ホーム] ページで [Enterprise Workspace] または [パーソナル ワークスペース] をクリックし、検索を開始するターゲット フォルダーに移動します。

  2. BlackBerry の [トラックパッド] を使用して、次のアイコンをハイライト表示して選択します。 .

  3. 検索用語を入力します。

  4. BlackBerry の [Enter] キーを押します。検索結果が表示されます。

  5. 表示するファイルやフォルダーをハイライト表示します。

  6. BlackBerry の [メニュー] キーを押して、使用可能なオプションを表示します。

検索結果からアイテムの場所を開くには、次の操作を行います。

  1. [Enterprise Workspace] または [パーソナル ワークスペース] を検索します。

  2. ファイルの場所で、開くファイルをハイライト表示します。

  3. BlackBerry の [メニュー] キーを押し、[場所を開く] を選択します。

  4. BlackBerry の [Escape] キーを押して、検索結果に戻ります。
