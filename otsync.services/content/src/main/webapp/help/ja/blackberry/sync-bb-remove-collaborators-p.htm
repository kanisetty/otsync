共同作業者の削除

OpenText Tempo Web ユーザー インターフェイスを使用して、所有する共有フォルダーから、共同作業者を削除できます。これによって共有フォルダーへの接続が切断され、共同作業者の OpenText Tempo フォルダーからフォルダーが削除されます。

Web ユーザー インターフェイスにアクセスするには、Web ブラウザーのアドレスに OpenText Tempo URL を入力し、OpenText Tempo にサインインします。共同作業者の削除の詳細については、OpenText Tempo Web ユーザー インターフェイスのオンライン ヘルプを参照してください。
