Отправка файла по электронной почте

  1. Нажмите Файлы и перейдите к целевому файлу. Если файл помечен тегом как избранное, нажмите Избранное.

  2. Выделите необходимый для просмотра адрес электронной почты.

  3. Нажмите клавишу BlackBerry Меню и выберите Электронная почта.

     

    Замечание:
    Если последняя версия файла на смартфоне недоступна, загрузка файла начинается до отображения сообщения электронной почты с присоединенным файлом.

     

  4. В поле Кому введите адреса электронной почты получателей.

  5. По выбору В поле Тема введите тему сообщения электронной почты.

  6. По выбору В теле сообщения введите текст.

  7. Нажмите клавишу BlackBerry Меню и выберите Отправить.
