So ändern Sie die Synchronisierungseinstellungen:

  1. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Einstellungen aus.

  2. Markieren Sie die Liste Favoriten synchronisieren, drücken Sie die BlackBerry Trackpad-Taste, und wählen Sie Manuell oder Automatisch aus.

  3. Drücken Sie die BlackBerry Esc-Taste, und wählen Sie Speichern aus.
