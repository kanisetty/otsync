So konfigurieren Sie OpenText Tempo-Einstellungen:

  1. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Einstellungen aus.

  2. Markieren Sie die Speicher, drücken Sie die BlackBerry Trackpad-Taste, und stellen Sie den Speicherplatz ein.

  3. Markieren Sie die Liste Favoriten synchronisieren, drücken Sie die BlackBerry Trackpad-Taste, und wählen Sie Manuell oder Automatisch aus.

  4. Markieren Sie die Liste Standardsortierreihenfolge, drücken Sie die BlackBerry Trackpad-Taste, und wählen Sie Datum oder Name aus.

  5. Drücken Sie die BlackBerry Esc-Taste, und wählen Sie Speichern aus.
