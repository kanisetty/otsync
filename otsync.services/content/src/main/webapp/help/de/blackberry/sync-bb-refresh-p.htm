So aktualisieren Sie die Anzeige:

  1. Klicken Sie auf Dateien, Favoriten oder Freigeben.

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Aktualisieren aus.

    Dadurch wird die Anzeige aktualisiert, aber die Dokumente werden nicht heruntergeladen. Öffnen Sie ein Dokument, um die aktuelle Version herunterzuladen.
