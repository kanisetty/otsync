So löschen Sie eine Datei:

  1. Klicken Sie auf , und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, klicken Sie auf .

     

    Hinweis:
    Lesen Sie vor dem Fortfahren die Informationen unter Achtung weiter unten.

    Sie können eine Datei nicht aus einem freigegebenen Ordner löschen, für den Sie nur Leserechte besitzen (durch gekennzeichnet).

     

  2. Markieren Sie die Zieldatei.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Element löschen aus.

  4. Bestätigen Sie diesen Vorgang, indem Sie auf OK klicken.

So löschen Sie einen Ordner:

  1. Klicken Sie auf , und navigieren Sie zum Zielordner.

     

    Hinweis:
    Lesen Sie vor dem Fortfahren die Informationen unter Achtung weiter unten.

    Wenn es sich beim Zielordner um einen freigegebenen Ordner handelt, lesen Sie vor dem Fortfahren den Abschnitt Freigegebenen Ordner löschen. Freigegebene Ordner sind durch oder gekennzeichnet.

     

  2. Markieren Sie den Ordner.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ordner löschen aus.

  4. Bestätigen Sie diesen Vorgang, indem Sie auf OK klicken.

 

Achtung

  • Durch Löschen einer Datei oder eines Ordners wird diese(r) aus dem OpenText Tempo-Ordner entfernt, selbst wenn die Datei als Favorit markiert ist. Das heißt, die Datei oder der Ordner wird aus allen anderen installierten OpenText Tempo-Clients, z. B. dem Desktop-Client, oder von Smartphones oder Tablets sowie aus der OpenText Tempo-Web-Benutzerschnittstelle gelöscht.

  • Wenn eine Datei oder ein Ordner aus einem freigegebenen Ordner gelöscht wird, für den Sie Lese- und Schreibrechte besitzen (durch gekennzeichnet), wird die Datei oder der Ordner automatisch für alle anderen Personen gelöscht, die Zugriff auf den Ordner haben, einschließlich des Eigentümers des freigegebenen Ordners.
