So bearbeiten Sie Ihr Profil:

  1. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Profil wechseln aus.

  2. Markieren Sie Ihr Profil auf dem Bildschirm Profil.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Profil bearbeiten aus.

  4. Ändern Sie Benutzername oder Server bei Bedarf.

  5. Klicken Sie auf Profil speichern.

So aktualisieren Sie das Kennwort:

  1. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Abmelden aus.

  2. Markieren Sie auf dem Bildschirm Profil Ihr Profil, und drücken Sie die Taste BlackBerry Trackpad.

  3. Geben Sie Ihr neues Kennwort ein.

     

    Hinweis:
    Für das Kennwort muss Groß-/Kleinschreibung beachtet werden. Stellen Sie sicher, dass Sie das Kennwort korrekt eingeben.

     

  4. Drücken Sie die BlackBerry Eingabe-Taste.
