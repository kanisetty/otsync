Per aggiungere un preferito:

  1. Fare clic su e posizionarsi sul file desiderato.

  2. Evidenziare il file.

  3. Premere sul BlackBerry il tasto Menu e selezionare Aggiungi a Preferiti.

    Il file preferito è contrassegnato con l'overlay . Ad esempio, .

  4. Opzionale La visualizzazione dell'overlay indica che il file non è stato scaricato. Ad esempio, . Per scaricare il file, aprirlo o sincronizzare i preferiti. Per ulteriori dettagli, vedere la sezione Sincronizzazione manuale dei Preferiti in basso.

    Tramite questa operazione il file viene scaricato sul dispositivo BlackBerry, in modo da poterlo visualizzare quando si è offline. Per sincronizzare i file preferiti automaticamente, vedere Modifica delle impostazioni di OpenText Tempo.

    Nel proprio profilo è memorizzato l'elenco dei propri file preferiti. I profili sono specifici del dispositivo. È necessario creare un profilo e contrassegnare file come preferiti su ogni smartphone e tablet in cui viene eseguita l'applicazione OpenText Tempo. Per ulteriori informazioni, vedere Creazione di un profilo.

Per visualizzare un preferito:

  1. Fare clic su .

  2. Opzionale Per assicurarsi di visualizzare tutti i preferiti, è necessario aggiornare periodicamente la visualizzazione. Premere sul BlackBerry il tasto Menu e selezionare Aggiorna. Questa operazione non consente di scaricare i documenti per la visualizzazione offline. Per scaricare il documento, sincronizzare i preferiti o aprire i documenti. Per ulteriori dettagli, vedere la sezione Sincronizzazione manuale dei Preferiti in basso.

Per aprire il percorso di un file preferito:

  1. Fare clic su .

  2. Evidenziare il file di cui si desidera aprire il percorso.

  3. Premere sul BlackBerry il tasto Menu e selezionare Apri percorso.

Per rimuovere un preferito:

  1. Fare clic su , oppure fare clic su e posizionarsi sul file desiderato.

  2. Evidenziare il file che si desidera rimuovere dall'elenco dei preferiti.

  3. Premere sul BlackBerry il tasto Menu e selezionare Rimuovi dai Preferiti. Il file non viene più visualizzato nell'elenco dei preferiti. Questa operazione non elimina il file dalla cartella OpenText Tempo.

Per sincronizzare manualmente i preferiti:

  1. Fare clic su .

  2. Premere sul BlackBerry il tasto Menu e selezionare Sincronizza Preferiti. Se l'applicazione sta sincronizzando i preferiti, o ci si trova offline, l'opzione Sincronizza Preferiti non è disponibile.

     

    Nota:
    La selezione di Aggiorna consente di aggiornare la visualizzazione della schermata. Questa operazione non consente di scaricare documenti preferiti sul proprio dispositivo BlackBerry per la visualizzazione offline. Per scaricare il file, è necessario sincronizzare i file preferiti o aprire il file.

     

Per sincronizzare i file preferiti automaticamente, vedere Modifica delle impostazioni di OpenText Tempo.
