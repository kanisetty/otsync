To create a profile:

  1. If this is the first time you are launching the OpenText Tempo Box app, click New Profile.

    If you want to create an additional profile and you have already signed in, do the following:

    1. Press the BlackBerry Menu key.

    2. Select Switch Profile.

    3. Click New Profile.

  2. In the Profile Name box, type a unique profile name.

  3. In the User Name box, type your user name. If you are outside of the company, type your email address.

     

    Note:
    A user who is outside of the company must first create an OpenText Tempo Box account before creating a profile.

    To create an account, a user inside the company must share a folder with the external user. This sends the external user an email invitation outlining the steps required to create an account.

     

  4. In the Password box, type your password. Your password is case-sensitive. Ensure you are typing the password correctly.

    If you are outside of the company, type the password you chose when you created your OpenText Tempo Box account. If you canâ€™t remember your password, refer to the email that was sent to you when you created your OpenText Tempo Box account.

  5. In the Server box, type the URL of your OpenText Tempo Box server.

    Contact your administrator for the URL of the OpenText Tempo Box server.

    If you are outside of the company, the OpenText Tempo Box URL is available in the email you were sent when you created your OpenText Tempo Box account.

  6. Optional Select the Keep me logged in check box to allow for immediate access the next time you launch the app. The only exceptions are if you log out of OpenText Tempo Box or if the session times out. See the Caution below that relates to this option.

     

    Note:
    This check box is not available if your BlackBerry is offline.

     

  7. Click Save and Log In.

    Profile data includes your account credentials, the OpenText Tempo Box URL, and a list of your favorite files. Profiles are specific to your device. You must create a profile and tag files as favorites on each of your smartphones and tablets running the OpenText Tempo Box app.

 

Caution
Allowing OpenText Tempo Box to keep you logged in can compromise security. If your BlackBerry device is used by another user when you have not logged out of the OpenText Tempo Box app, the other user will be able to access your OpenText Tempo Box account. To log out, press the BlackBerry Menu key and select Log Out.
