To delete a file:

  1. Click and navigate to the target file. You cannot delete a file from a shared folder to which you have read-only access. Files in a read-only shared folder are marked with the overlay. For example, .

     

    Note:
    Before proceeding, refer to the Caution information below.

     

  2. Highlight the target file.

  3. Press the BlackBerry Menu key and select Delete Item.

  4. Click OK to confirm the action.

To delete a folder:

  1. Click and navigate to the target folder. If the target folder is a shared folder, see Deleting a Shared Folder. Shared folders are identified by the overlay (read/write access) or overlay (read-only access). For example, or .

     

    Note:
    Before proceeding, refer to the Caution information below.

     

  2. Highlight the folder.

  3. Press the BlackBerry Menu key and select Delete Folder.

  4. Click OK to confirm the action.

 

Caution

  • Deleting a file or folder removes it from your Tempo Box folder, even if the file was tagged as a favorite. This means the file or folder is deleted from all other OpenText Tempo Box clients you have installed such as the desktop client, other smartphones or tablets, as well as the OpenText Tempo Box Web user interface.

  • If a file or folder is deleted from a shared folder to which you have read/write access, the file or folder is automatically deleted for all users who have access to the shared folder, including the owner of the shared folder.
