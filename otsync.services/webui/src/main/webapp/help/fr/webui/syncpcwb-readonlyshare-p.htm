Modifier les autorisations d'un collaborateur

Pour modifier les autorisations d'un collaborateur :

  1. Cliquez sur un dossier partagé dont vous êtes le propriétaire.

  2. Dans le volet droit, sous COLLABORATEURS, cliquez sur Modifier les [ ] collaborateurs.

  3. En regard du nom du collaborateur, cliquez sur ou et sélectionnez la nouvelle autorisation.

  4. Cliquez sur Enregistrer.
