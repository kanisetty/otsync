Ajouter une version d'élément

Pour ajouter une version d'élément :

  1. Accéder à l'emplacement de l'élément.

    Vous ne pouvez pas ajouter de version à un élément dans un dossier pour lequel vous disposez d'un accès en lecture seule. Les dossiers en lecture seule sont identifiés par le cache de verrouillage. Par exemple, ou .

  2. Pointez sur le menu de fonction de l'élément , puis cliquez sur Ajouter une version.

    Si le fichier est déjà réservé par un autre utilisateur, vous ne pouvez pas ajouter une version au fichier. Les fichiers réservés sont marqués à l'aide du recouvrement. Par exemple, . Pour savoir qui a réservé le fichier, vous pouvez afficher l'historique de fichier.

     

    Important :
    Quand vous ajoutez une version d'élément à un fichier dans un dossier partagé pour lequel vous disposez d'un accès en lecture/écriture, le fichier n'est pas automatiquement réservé. Vous pouvez télécharger et modifier un fichier alors même qu'un de vos collaborateurs y apporte des modifications. Les dossiers en lecture/écriture sont identifiés par ou . Pour empêcher cela, vous pouvez réserver le fichier .

     
     

    Attention
    Si vous modifiez du contenu dans un dossier pour lequel vous disposez d'un accès en lecture/écriture, les modifications sont synchronisées avec tous les utilisateurs qui ont accès au dossier.

    Si vous modifiez le contenu d'un dossier Entreprise les modifications sont synchronisées avec le dossier dans l'Espace de travail Entreprise. Les dossiers Entreprise sont identifiés sous la forme .

     

  3. Procédez de l'une des façons suivantes :

    • Si le bouton Parcourir apparaît, cliquez sur Parcourir, sélectionnez un élément, puis cliquez sur Ajouter une version .

    • Si la boîte de dialogue Téléchargement du fichier s'ouvre, sélectionnez un élément, puis cliquez sur Ouvrir.
