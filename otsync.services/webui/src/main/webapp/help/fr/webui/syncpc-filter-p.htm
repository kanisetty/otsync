Filtrer l'affichage du dossier

Pour filtrer l'affichage du dossier :

  • Cliquez sur l'un des liens suivants dans le volet de gauche :

    • Tous les fichiers : affiche tout le contenu dans votre dossier Tempo Box.

    • Mes fichiers : affiche tous les fichiers et dossiers qui vous appartiennent.

    • Partagé avec moi : affiche uniquement les dossiers qui ont été partagés avec vous.

    • Contenu Entreprise : affiche les dossiers Entreprise auxquels vous êtes abonnés.

    • Mes communautés : affiche les bibliothèques de fichier dans les communautés Tempo Social que vous suivez.

    Les filtres Mes fichiers, Partagé avec moi , Contenu Entreprise et Mes communautés s'affichent uniquement si vous avez au moins un dossier qui correspond à la description du filtre.
