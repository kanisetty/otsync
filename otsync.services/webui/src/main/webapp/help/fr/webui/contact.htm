ii – Informations de contact

OpenText Online est un point d'accès unique pour les informations fournies par OpenText sur les produits. Vous pouvez accéder aux ressources de support suivantes via OpenText Online :

  • Communautés

  • Base de connaissances

Les communautés OpenText Online offrent les ressources suivantes :

  • Conseils d'utilisation, fichiers d'aide et meilleures pratiques destinés aux clients et aux partenaires

  • Informations sur les versions des produits

  • Groupes d'utilisateurs et forums permettant de poser des questions aux experts OpenText

L'OpenText Knowledge Center constitue le site extranet de l'entreprise OpenText, ainsi que le site principal pour obtenir une assistance technique. Le Knowledge Center est la source officielle des éléments suivants :

  • Téléchargements, correctifs et documentation sur les produits, y compris les notes de mise à jour

  • Forums de discussion, communautés en ligne et base de connaissances

  • OpenText Developer Network (OTDN), qui inclut la documentation pour le développeur et des exemples de programmation pour les produits OpenText

Si vous avez besoin d'une assistance supplémentaire, contactez le support OpenText à l'adresse http://support.opentext.com/.
