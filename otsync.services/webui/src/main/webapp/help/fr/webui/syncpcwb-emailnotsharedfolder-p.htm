Activer les notifications par e-mail pour un dossier partagé

Pour activer les notifications par e-mail pour un dossier partagé :

  1. Cliquez sur un dossier partagé dont vous n'êtes pas le propriétaire.

  2. Dans le volet de droite, cochez la case Recevoir des notifications de dossiers.

 

Notes :

  • Les notifications par e-mail sont envoyées à l'adresse spécifiée dans le champ E-mail de votre profil Content Server. Si vous ne recevez pas les notifications, contactez votre administrateur. Si vous êtes utilisateur externe, les e-mails sont envoyés à l'adresse que vous utilisez pour vous connecter à votre compte Tempo Box.

  • Cette option est disponible uniquement si votre administrateur a activé les notifications par e-mail et si vous avez désactivé l'option globale permettant de recevoir des notifications par e-mail. Pour plus d'informations sur l'option globale, voir Activer les notifications par e-mail.
