OpenText Tempo Box: Ayuda de usuario para el cliente web

   

Capítulo 2 – Gestión de archivos y carpetas

Puede utilizar el cliente web para añadir, copiar o mover elementos y carpetas a otras ubicaciones de la carpeta de Tempo Box. También puede consultar la versión y el historial de auditorías, cambiar el nombre o suprimir elementos y carpetas dentro de dicha carpeta. Todos los cambios que realice se sincronizarán automáticamente con la carpeta de Tempo Box de todos los clientes que estén en línea.

 

Notas:

  • No puede usar el cliente web para editar elementos en su carpeta Tempo Box. Para actualizar un archivo mediante el cliente web, deberá bajarlo al escritorio, editarlo allí y, a continuación, volver a añadirlo como una versión nueva.

  • Los resultados de mover, cambiar de nombre o suprimir una carpeta compartida varían en función de si es el propietario de la carpeta compartida. Para obtener más información, consulte Mover, cambiar de nombre y suprimir carpetas compartidas .

 
 

Importante:
Cuando visualice archivos en una carpeta compartida para la que tenga acceso de lectura/escritura, dichos archivos no se reservarán de forma automática. Puede bajar y editar un archivo mientras un colaborador realiza cambios en él. Para evitarlo, puede reservar el archivo. Las carpetas de lectura/escritura se identifican mediante o .

 

El administrador puede establecer un límite de tamaño para su carpeta de Tempo Box. Si el tamaño del conjunto de los elementos de su carpeta supera el límite, sin incluir los elementos de carpetas que comparte con otros usuarios, necesitará suprimir o sacar elementos de su carpeta Tempo Box hasta que el tamaño sea inferior al máximo. Para obtener más información, consulte Comprobación del tamaño de su carpeta de Tempo Box.

Las fotografías e imágenes de la carpeta de Tempo Box muestran una vista en miniatura de la imagen. Para verla a un tamaño mayor, puede pasar a vista en miniatura o seleccionar la imagen.

También puede publicar elementos y carpetas en Content Server. Esta opción no está disponible para usuarios externos. Asimismo, si está suscrito a una carpeta de empresa y posee acceso de lectura/escritura a dicha carpeta, puede interactuar directamente con los elementos de la carpeta. Para obtener más información acerca de la adición de una carpeta de empresa a la carpeta de Tempo Box, consulte OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).
