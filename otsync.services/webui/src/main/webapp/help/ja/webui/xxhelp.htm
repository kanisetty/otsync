以下のナビゲーション エレメントを使って必要な情報をすばやく得ることができます。

目次

目次 (TOC) が表示される見出しレベルを設定するには、TOC ページの先頭にある [Depth] で指定することができます。

  • では、見出しレベルの一番上に指定されている章だけが表示されます。

  • では、その下にあるセクションの見出しが表示されます。

  • では、3 つめのレベルまですべて表示されます。

  • 以降、これにならって表示されます。

目的のタブをクリックしてください。

ヘルプのレベルが 6 を超える場合、5 つめのタブ にすべてのレベルが表示されます。従って、タブの数は多くても 5 つです。

 

注意:

  • 現在表示されている目次のレベルよりも深いレベルのセクションにジャンプすると、すべてのレベルにわたる完全な目次が自動的に開かれます。現在表示されているセクションには不十分なレベルのタブをクリックした場合でも、同様の動作が起こります。

  • 通常、TOC では、別のページにブラウズまたはジャンプした後も、現在表示されているページがハイライトされます。例えば、インデックス ページ (下記参照) から TOC ページに切り替えた場合、このハイライトがうまく機能しないことがあります。この場合、TOC を再同期するには、ヘルプ ページのナビゲーション エリアの一番下にあるエントリを 1 ～ 2 回クリックします。

 

索引

レベルのタブの横にある タブをクリックすると、目次の代わりに [インデックス] が表示されます (文書にインデックスがある場合)。このキーワードをクリックと、このキーワードに関するページが表示されます。

キーボードを使用したナビゲーション

最近のバージョンのブラウザ (IE6、Mozilla 1.7、Firefox) では、キーボードを使用して、文書を参照することができます。参照のレベルには、文書レベルとページ レベルの 2 種類のレベルがあります。

  • ページ レベルを切り替えたり、現在表示されているページを上または下にスクロールしたりするには、上矢印/下矢印キーを使用します。

  • ESC キーを使用して、前のページ レベルに戻ることができます。また、PAGE UP/DOWN または 左矢印/右矢印 キーを使用して、前のページや次のページにジャンプすることができます。

    文書レベルでは、数字キー (1、2、3、...) を使用して、目次の別のレベルに変更することができます。また、I キーを押すと、インデックス ページが表示されます。

文書を開いた直後は文書レベルなので、ページ キーを使用して、ページを参照することができます。また、ページをクリックすると、通常はページ レベルになるので、文書のページを参照する必要がある場合は、最初に ESC キーを押してください。

特定のページの参照

特に、中央のファイル共有にこの HTML 文書を提供した場合、自分の文書内から、または電子メールでこの文書の特定のページを参照することができます。

このため、それぞれのページの URL を指定すれば十分です (このページは、目次から対応するエントリを右クリックし、“ショートカットのコピー”、“リンクの場所のコピー” またはこれに類似するコマンドを選択すれば、簡単にクリップボードへコピーできます)。JavaScript が Web ブラウザで有効化されている場合、このブラウザは自動的にそのページを開き、左側にマニュアル全体のフレームワーク、つまり目次を開きます。

 

注意:

  • このオンライン マニュアルの自動化機能は、使用している Web ブラウザで JavaScript が有効化されている場合にのみ使用可能です。それでも、一部のブラウザで機能のいくつかがサポートされていない可能性があります。特に、古いバージョンのブラウザ (Netscape 6 以前、IE5 以前) はこれに該当します。自動化機能には、例えば、現在のセクションを目次でハイライトする、前述の方法でキーボードを使用してナビゲーションを行う、ブラウザで特定のページを直接開いた場合に目次を自動的にアクティブ化するなどがあります。

    JavaScript がなくても、文書を読んだり、文書内を移動したりすることは可能です。JavaScript は、主に利便性を高めるために使用されます。
