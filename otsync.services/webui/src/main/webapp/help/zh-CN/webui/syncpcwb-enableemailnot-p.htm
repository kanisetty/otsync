启用电子邮件通知

启用电子邮件通知：

  1. 在顶部右侧窗格中，单击您的名称，单击 [ Tempo Box]，然后单击[设置]。

  2. 在[给我发送电子邮件]下，选择下列一个或两个复选框：

    [我收到共享请求时]

    他人邀请您与其共享文件夹时接收电子邮件。

    [有人对任何共享文件夹进行更改时]

    当其他用户在您参与的共享文件夹中添加、编辑或删除项目时接收电子邮件通知。电子邮件是根据管理员设置的时间表发送的定期摘要。此设置全局适用于所有共享文件夹。如果不启用此设置，仍可选择为单独的共享文件夹接收电子邮件通知。有关详细信息，请参阅 启用共享文件夹的电子邮件通知。

    管理员必须启用电子邮件通知以使该选项可用。

  3. 单击[保存设置]。电子邮件发送至 Content Server 配置文件的[电子邮件]字段中指定的地址。如果您没有收到通知，请联系您的管理员。对于外部用户，登录到 Tempo Box 时电子邮件会发送到您使用的电子邮件地址。
