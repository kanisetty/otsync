OpenText Tempo 简介

OpenText Tempo 为企业提供安全的文件共享和同步。无论您在何地，均可通过 OpenText Tempo 对您的 [OpenText Tempo] 文件夹中的文档进行管理和协作。[OpenText Tempo] 文件夹中的内容联机存储，可使用桌面客户端、智能手机和平板电脑或 Web 浏览器进行访问。其中任何一个客户端修改内容时，更改会立即自动与所有其他连接的 OpenText Tempo 客户端同步。

  • 桌面客户端

    通过在 Windows 桌面计算机上安装 OpenText Tempo 客户端，您可以访问 [OpenText Tempo] 文件夹。这会在您的本地文件系统上创建 [OpenText Tempo] 文件夹内容的文件夹。每次添加、删除或保存您桌面计算机上的 [OpenText Tempo] 文件夹中的内容时，这些更改均会自动与 Web 客户端和所有已运行 OpenText Tempo 应用程序的智能手机和平板电脑同步。

  • 移动客户端

    通过在 Apple iPhone、Apple iPad、Android 智能手机或平板电脑以及 BlackBerry 智能手机上安装 OpenText Tempo 应用程序，您可以访问您的 [OpenText Tempo] 文件夹。与桌面客户端类似，使用 OpenText Tempo 应用程序对内容所做的任何更改均会与所有其他客户端同步。

  • Web 客户端

    通过将 OpenText Tempo URL 输入浏览器地址区域，您可以从任何 Web 浏览器访问您的 [OpenText Tempo] 文件夹。使用 Web 客户端，您可以管理内容并与其他用户共享文件夹。与使用桌面和移动客户端一样，使用 Web 浏览器对 OpenText Tempo 内容所做的任何更改均会与所有其他客户端同步。

  • 共享内容

    使用 OpenText Tempo，您还可以与其他用户共享内容，轻松协作。对共享文件夹中的内容所做的任何更改均会自动同步到共享文件夹的所有成员，而不论是谁做的更改。
