使用帮助文件

欢迎使用联机文档！

在以下导航元素的帮助下，您可以快速找到所需的信息：

目录

通过选择位于 TOC 页顶部的[深度]，您可以定义目录显示的详尽程度：

  • 仅显示主要部分

  • 包括下级部分

  • 显示深入到第三级别的所有部分

  • 等

只需单击所需的选项卡。

如果深度级别超过五个，则第五个选项卡 会显示所有级别，因此最多为五个深度选项卡。

 

注意:

  • 如果您跳转至比当前显示的目录更深的部分，则会自动打开涵盖所有深度级别的完整目录。如果您单击某个选项卡，而它的深度对当前显示的部分来说不够深，则也会发生此情况。

  • 通常，TOC 会突出显示当前显示的页面，而这也是在您浏览或跳转至另一页之后才发生的。在某些情况下，例如，从索引页（请参阅以下内容）切换回到 TOC 页之后，此突出显示不起作用。在此情况下，为了重新同步 TOC，请单击一次或两次位于帮助页的导航区域中的最低的条目。

 

索引

如果您单击深度选项卡旁边的 选项卡，则只要文档具有索引，就会显示[索引]，而不显示目录。您将获得重要关键字列表以及链接，以便可以直接导航到出现该字词的文本中的段落。

导航区域

每个帮助页均包含两个导航区域：一个位于顶部，另一个位于底部。通过单击表示[下一个]或[上一个]的箭头（右上方区域）或链接（下方区域），您可以导航至文档的下一部分或上一部分。与 Internet 浏览器中的[后退]和[前进]等符号相反，这些链接并不引用已显示的页面的历史记录，而是会逐页浏览文档，如目录中最低级别所示。您可以像看书一样浏览各个部分。

除了用于简单浏览的箭头之外，上部的导航区域还提供额外的导航方法：

您在顶部会找到当前阅读的文档/手册的标题。位于标题下方的是各个分层级别，一直延伸到您当前显示的页面。因此，在文档标题下面，显示第一级别上的子部分，随后是第二级别上的子部分，依此类推。如果您单击此类标题，则可切换至相应的部分。

在标题的左侧，一共有四个小图标，它们允许您 – 在相应子部分级别上浏览 - 浏览到上一部分或下一部分（ 和 ）或立即移到该部分的目录 ( )。 图标与该部分标题本身相对应，因此是一个指向相应部分的开头的链接。

如果您将鼠标指针置于这些图标之一的上方，而不单击它，则图标链接至的部分的标题将会出现。

底部导航区域也包含额外的链接：[首页]指的是文档的开头，而[向上]将使您进入目录中的下一个更高级别。只需单击链接即可进入所需的页面。

使用键盘进行导航

在最新的浏览器版本（IE6、Mozilla 1.7、Firefox）中，您可以使用键盘来浏览文档。一共有两个不同的浏览级别，即，文档级别和页面级别：

  • 使用向上箭头/向下箭头，您可切换至页面级别，并且向上或向下滚动当前显示的页面。

  • 通过使用 ESC，您可切换回到页面级别，然后使用上页/下页或左箭头/右箭头，以跳转至上一页或下一页。

    在文档级别上，您可以使用数字键（1、2、3...）来切换至目录中的另一个深度，也可以使用 i 键来显示索引页面。

在打开文档后，您正好位于文档级别，因此可以使用页面键来浏览页面。但是，在单击页面时，您通常会位于该页面级别；如果想要浏览文档页面，则首先要按下 ESC 键。

引用特定页面

特别是，如果您在中央文件共享上提供此 HTML 文档，您可能要从自己的文档或电子邮件中引用此文档的特定页面。

出于这方面的目的，您只需要指定相应页面的 URL（右键单击导航区域最低标题（请参阅以上内容），然后选择“复制快捷键”、“复制链接位置”或类似功能，即可轻松将 URL 复制到剪贴板）。只要您在 Web 浏览器中启用 Javascript，浏览器就会在整个手册框架内自动打开该页面，即，也在左侧打开目录。

 

注意:

  • 如果您的桌面上空间不大，则导航系统甚至可让您在左侧没有目录的情况下轻松浏览文档。您只需将目录与内容区域之间的分隔线条尽可能地向左移动，一直到目录消失为止。

  • 仅在 Web 浏览器中的 JavaScript 处于活动状态时，此联机文档的自动功能才可用。即使这样，仍然可能会有一个或其它功能不受某些浏览器支持；特别是旧版本的浏览器（Netscape 6 和之前的版本以及 IE5 和之前的版本），这种情况尤为严重。自动功能有多种，例如，突出显示目录中的当前部分，如上所述使用键盘进行导航，以及在浏览器中直接打开特定页面的情况下自动激活目录。

    但是，您不需要 JavaScript 来阅读文档，也不需通过它来进行导航。JavaScript 专门用于提高舒适性。
