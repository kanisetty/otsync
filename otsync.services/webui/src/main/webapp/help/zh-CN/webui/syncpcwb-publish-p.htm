发布项目或文件夹

要发布项目或文件夹：

  1. 指向项目的功能菜单 ，然后单击[发布]。

    如果该选项不可用，请联系您的管理员。外部用户 不能将项目发布到 Content Server。

  2. 如果正在发布项目，请单击以下选项之一：

    [发布新版本]

    将项目作为 Content Server 中现有项目的新版本添加。

    [发布到文件夹]

    将项目作为新项目添加到 Content Server 位置，如个人工作区、文件夹或项目。

  3. 单击[浏览 Content Server]，然后选择以下选项之一：

    • Content Server 项目，如果您正在发布新版本项目。

    • Content Server 文件夹，如果您正在发布新项目或文件夹。

  4. 单击以下选项之一：

    [保留文件副本 ]

    让现有项目或文件夹保持在 [Tempo Box] 文件夹中的当前位置。

    [将文件移入 Content Server 并删除源文件]

    从 [Tempo Box] 文件夹删除项目或文件夹。

     

    注意:
    如果您正在发布由其他用户拥有的共享文件夹，则必须在 [Tempo Box] 文件夹中保留文件夹副本。

     

  5. 单击[发布]。
