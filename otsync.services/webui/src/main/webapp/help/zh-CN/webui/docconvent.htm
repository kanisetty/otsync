i – 文档约定

[用户界面]

该格式用于图形用户界面 (GUI) 中的元素，例如按钮、图标名称、菜单项和字段。

Filenames、commands 和示例数据

该格式用于文件名、路径、URL 和命令提示符中的命令。还用于示例数据、在文本框中输入的文本以及其它文字。

 

注意:
从 PDF 中复制命令行示例时，需注意 PDF 可能包含隐藏的字符。OpenText 建议如有 HTML 版本的文档，则从该版本的文档中复制。

 
键名

键名的字母全部大写，例如：
按 CTRL+V。

<变量名称>

尖括号 < > 用于表示变量或占位符。 用户用相应的值替换尖括号及其中的描述性内容。 例如，<server_name> 被替换成 serv01。

超链接或者 Weblink (http://www.opentext.com)

这些格式用于链接。 这些链接是对文档（超链接）中或互联网 (http://www.opentext.com) 上的其它位置的有效引用。

内部交叉引用

单击交叉引用，直接转至当前文档中的引用目标。

外部交叉引用

外部交叉引用一般是对其它文档的文本引用。 但是如果提供 HTML 格式的文档（例如，Knowledge Center中的文档），则外部引用可能是被引用文档中指定部分的有效链接。

警告、注释和提示
 

小心
警告可以帮助您避免一些不可逆性问题。请仔细阅读此信息并按照说明进行操作。

 
 

重要:
重要的注释可以帮助您避免一些重大问题。

 
 

注意:
注释提供关于任务的附加信息。

 
 

提示:
提示方便您轻松快捷地执行任务。
