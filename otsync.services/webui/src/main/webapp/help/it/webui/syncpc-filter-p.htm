Filtraggio della Vista cartelle

Per filtrare la Vista cartelle:

  • Fare clic su uno dei seguenti collegamenti nel riquadro sinistro:

    • Tutti i file – Mostra tutti i contenuti nella cartella Tempo Box.

    • I miei file – Mostra tutti i file e le cartelle personali.

    • Condiviso con me – Mostra solo le cartelle che sono state condivise con l'utente.

    • Enterprise Content – Mostra le cartelle Enterprise per le quali è stata effettuata una sottoscrizione.

    • Le mie comunità – Mostra le librerie di file nelle comunità Tempo Social seguite dall'utente.

    I filtri I miei file, Condiviso con me , Enterprise Content e Le mie comunità si visualizzano solamente se si dispone almeno di una cartella che corrisponde alla descrizione del filtro.
