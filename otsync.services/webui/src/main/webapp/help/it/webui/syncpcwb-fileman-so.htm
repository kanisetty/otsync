OpenText Tempo Box: Guida dell'utente del client Web

   

Gestione di elementi e cartelle

È possibile utilizzare il client Web per aggiungere, copiare o spostare elementi e cartelle in altre posizioni nella cartella Tempo Box. È inoltre possibile visualizzare la cronologia delle versioni e degli eventi e rinominare o eliminare elementi e cartelle all'interno di questa cartella. Qualsiasi modifica effettuata viene automaticamente sincronizzata con la cartella Tempo Box su qualsiasi altro client online.

 

Note:

  • Non è possibile utilizzare il client Web per modificare i file nella propria cartella Tempo Box. Per aggiornare un file utilizzando il client Web è necessario scaricarlo sul desktop, modificarlo in tale posizione, quindi aggiungerlo nuovamente come nuova versione .

  • I risultati di spostamento, ridenominazione o eliminazione di una cartella condivisa possono variare in base alla proprietà della cartella condivisa. Per ulteriori informazioni, vedere Spostamento, ridenominazione ed eliminazione delle cartelle condivise .

 
 

Importante:
Durante la visualizzazione di file in una cartella condivisa per cui si dispone di accesso in lettura/scrittura, i file non sono riservati automaticamente. È possibile scaricare e modificare un file anche mentre lo sta facendo un collaboratore. Per evitare tale situazione, è possibile riservare il file.. L'amministratore può impostare un limite per le dimensioni della cartella Tempo Box. Se le dimensioni di tutti gli elementi all'interno della propria cartella, esclusi quelli nelle cartelle in condivisione, superano il limite, sarà necessario eliminare o spostare gli elementi all'esterno della propria cartella Tempo Box finché le dimensioni non saranno al di sotto del limite massimo. Per ulteriori dettagli, vedere Controllo delle dimensioni della cartella Tempo Box.

Nella cartella Tempo Box le fotografie e le immagini sono visualizzate in anteprima. È possibile visualizzare una panoramica dei file passando alla visualizzazione in anteprima oppure selezionare l'immagine.

È inoltre possibile pubblicare elementi e cartelle su Content Server. Questa opzione non è disponibile per gli utenti esterni. In alternativa, se è stata effettuata la sottoscrizione a una cartella Enterprise e si dispone dell'accesso in lettura/scrittura a tale cartella, è possibile interagire direttamente con gli elementi nella cartella. Per ulteriori informazioni sull'aggiunta di cartelle Enterprise alla cartella Tempo Box, vedere OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).
