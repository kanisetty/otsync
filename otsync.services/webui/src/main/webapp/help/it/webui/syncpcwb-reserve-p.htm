Modalità per riservare e rilasciare un file

Per riservare un file:

  1. Selezionare il menu funzionale dell'elemento e fare clic su Riserva.

    Non è possibile riservare un file all'interno di una cartella per la quale si dispone dell'accesso in sola lettura. Le cartelle in sola lettura sono identificate dall'overlay di blocco. Ad esempio, o .

    Se si tenta di riservare un file in una cartella Enterprise per la quale si dispone dell'accesso in lettura/scrittura e l'opzione Riserva non è disponibile, ciò significa che non si dispone dell'autorizzazione Riserva per la cartella in Content Server. Contattare l'amministratore per richiedere l'autorizzazione.

  2. Fare clic su Riserva. Il file è contrassegnato con l'overlay . Ad esempio, .

Per rilasciare un file:

  1. Selezionare il menu funzionale del file e fare clic su Rilascia. Se l'opzione Rilascia non è disponibile, il file è riservato da un altro utente. Per stabilire chi ha riservato il file, è possibile verificare la cronologia eventi del file. Per ulteriori dettagli, vedere Visualizzazione della cronologia di elementi e cartelle.

  2. Fare clic su Rilascia.
