Pubblicazione di un elemento o di una cartella

Per pubblicare un elemento o una cartella:

  1. Selezionare il menu funzionale dell'elemento e fare clic su Pubblica.

    Contattare l'amministratore se questa opzione non è disponibile. Gli utenti esterni non possono pubblicare elementi su Content Server.

  2. Se si sta pubblicando un elemento, fare clic su una delle seguenti azioni:

    Pubblica una nuova versione

    Consente di aggiungere l'elemento come una nuova versione di un elemento esistente su Content Server.

    Pubblica in una cartella

    Consente di aggiungere l'elemento come nuovo elemento a un percorso Content Server come l'are di lavoro personale, una cartella o un progetto.

  3. Fare clic su Sfoglia Content Server e selezionare una delle opzioni seguenti:

    • Un elemento di Content Server, se si sta pubblicando la nuova versione di un elemento.

    • Una cartella di Content Server, se si sta pubblicando un nuovo elemento o una cartella.

  4. Fare clic su una delle seguenti azioni:

    Mantieni una copia del file

    Mantieni elemento o cartella esistente nel percorso attuale nella cartella Tempo Box.

    Sposta il file in Content Server ed elimina l'originale

    Elimina elemento o cartella dalla cartella Tempo Box.

     

    Nota:
    Se si sta pubblicando una cartella condivisa di cui è proprietario un altro utente, è necessario mantenerne una copia nella cartella Tempo Box.

     

  5. Fare clic su Pubblica.
