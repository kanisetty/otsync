Удаление пользователей с общим доступом из общей папки

Удаление пользователей с общим доступом из общей папки:

  1. Нажмите общую папку, владельцем которой вы являетесь.

  2. На панели справа на вкладке COLLABORATORS щелкните рядом с именем пользователя с общим доступом и щелкните Unshare.

    Если количество пользователей с общим доступом в папке превышает семь, щелкните Edit All [ ] Collaborators, щелкните рядом с именем пользователя с общим доступом и щелкните Сохранить.

     

    Подсказка:
    Чтобы отключить общий доступ для всех пользователей с общим доступом, нажмите Unshare All.

     
 

Замечание:
Если вам больше не нужно видеть папку, общий доступ к которой открыл вам другой пользователь, эту общую папку можно удалить. Удалять следует только саму общую папку. При удалении содержимого внутри общей папки оно удаляется для всех пользователей с общим доступом. Дополнительные сведения об удалении общей папки, общий доступ к которой открыл вам другой пользователь, см. в разделе Удаление общей папки.
