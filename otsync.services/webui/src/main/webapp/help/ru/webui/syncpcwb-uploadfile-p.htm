Добавление элемента

  1. Перейдите к местоположению, в которое необходимо добавить элемент.

    Нельзя добавить элемент в папку с разрешением только для чтения. Папки, допускающие только чтение, обозначаются изображением замка. Например, или .

    Нельзя добавить элемент в папку, которая получает обязательные категории и атрибуты через Tempo Box. Чтобы определить, имеет ли папка обязательные категории и атрибуты, см. раздел Просмотр журнала элементов или папок.

  2. На средней панели щелкните Добавить файл.

    Tempo Box поддерживает файлы размером не более 2 ГБ. Использование более крупных файлов не допускается.

     

    Предостережение
    Если вы добавите содержимое в папку, к которой имеется доступ для чтения-записи, изменения синхронизируются для всех пользователей, имеющих доступ к папке. Папки для чтения-записи обозначаются или .

    После редактирования содержимого в папке Enterprise изменения применяются к папке в рабочей области предприятия. Папки Enterprise можно опознать по значку .

     

  3. Выполните одно из следующих действий.

    • Если отображается кнопка Browse, нажмите Browse, выберите элемент, затем нажмите Upload.

    • Если отображается диалоговое окно File Upload, выберите элемент и щелкните Open.

 

Подсказка:
При использовании новейшей поддерживаемой версии Chrome файл также можно добавить путем его перетаскивания с рабочего стола в окно браузера с веб-клиентом. Перетаскивать можно только файлы, но не папки. При перетаскивании файла он всегда копируется в веб-клиент, даже если в браузере указано, что выполняется перемещение файла.
