Открытие общего доступа к папке

Открытие общего доступа к папке:

  1. Направьте курсор мыши на меню функций элемента и щелкните Предоставить общий доступ.

     

    Замечание:
    Если меню Share отсутствует, это указывает на отсутствие разрешения для открытия общего доступа к этой папке. Список папок, к которым нельзя открыть общий доступ, см. в разделе Общий доступ к папкам и ответ на приглашение к совместному использованию.

     

  2. По выбору При открытии общего доступа к папке пользователю можно предоставить разрешение либо на чтение и запись, либо только на чтение. По умолчанию пользователям с общим доступом предоставляется доступ для чтения-записи.

    Разрешение на чтение и запись дает всем пользователям с общим доступом, включая внешних пользователей, право на полный доступ к содержимому папки. Они могут добавлять, изменять, копировать, перемещать, переименовывать и удалять файлы и папки. Изменения, внесенные в общую папку, автоматически синхронизируются со всеми пользователями, имеющими общий доступ к папке, включая ее владельца.

    Разрешение только для чтения дает пользователям с общим доступом право только на чтение и копирование содержимого общей папки.

    Чтобы предоставить пользователю разрешение только на чтение, нажмите , а затем нажмите Только для чтения. Появляется значок .

  3. Нажмите поле Пользователи с общим доступом и введите имя пользователя, для которого необходимо открыть общий доступ к этой папке.

    Для приглашения внешнего пользователя введите адрес электронной почты этого пользователя. Под внешними понимаются пользователи, которые не являются сотрудниками вашей компании. После ввода действительного адреса электронной почты в поле автозаполнения отобразится адрес электронной почты внешнего пользователя. Если адрес электронной почты не будет заполнен автоматически, у вас нет разрешения на совместное использование папки с другими пользователями. Для получения этой привилегии свяжитесь с администратором. Дополнительные сведения о внешних пользователях см. в разделе Совместное использование с внешними пользователями.

  4. Щелкните имя пользователя или адрес электронной почты. Будет отправлено приглашение на открытие общего доступа. Чтобы получить доступ к содержимому папки, пользователь должен принять это приглашение.

  5. Для предоставления общего доступа к папке другому пользователю повторите шаги 2–4.

  6. Щелкните Сохранить.

    Теперь папка помечена как общая. Независимо от типа разрешения, предоставленного другим пользователям (либо чтение и запись, либо только чтение), общая папка будет отображаться у вас как .

 

Подсказка:
Также общий доступ к папке можно открыть с помощью нажатия имени папки и добавления другого пользователя. Дополнительные сведения см. в разделе Обновление пользователей с общим доступом к общей папке.
