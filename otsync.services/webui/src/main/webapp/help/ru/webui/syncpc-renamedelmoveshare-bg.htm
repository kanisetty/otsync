Перемещение, переименование и удаление общих папок

К принадлежащим вам общим папкам применимы следующие условия.

  • После перемещения общей папки в другое местоположение в папке Tempo Box к ней остается открытым общий доступ для всех других пользователей с общим доступом.

  • После перемещения общей папки в местоположение вне папки Tempo Box она больше не будет открыта для общего доступа, и другие пользователи с общим доступом не смогут ее видеть. Папка становится обычной, и содержимое удаляется из папки Tempo Box.

  • При переименовании общей папки она переименовывается для всех пользователей, имеющих к ней доступ.

  • При удалении общей папки она удаляется для всех пользователей, имеющих к ней доступ.

Следующие условия применимы к общим папкам, которые не принадлежат вам, но для которых у вас есть права доступа.

  • Нельзя переместить корневой каталог общей папки, папки Enterprise или библиотеки файлов сообщества.

  • Общая папка, перемещенная за пределы папки с вашим именем пользователя, становится для вас недоступной. Такая папка становится обычной папкой и больше не синхронизируется. Если вы случайно переместили папку за пределы папки с вашим именем пользователя, обратитесь к владельцу папки за новым приглашением к совместному использованию.

  • Нельзя переименовать корневой каталог общей папки.

  • При удалении общей папки она удаляется только из вашей папки Tempo Box. Для других пользователей сохраняются права общего доступа к этой папке. Если вы случайно удалили такую папку, обратитесь к владельцу папки за новым приглашением к совместному использованию.

  • При удалении папки Enterprise она удаляется только из вашей папки Tempo Box. Для других пользователей сохраняются права общего доступа к этой папке. Если вы случайно удалили такую папку, можете повторно ее добавить из рабочей области Enterprise Content Server. Дополнительные сведения о добавлении папок Enterprise в папку Tempo Box см. в разделе OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).

  • Корневой каталог папки Enterprise можно переименовать только при наличии разрешения на Редактирование разрешений для такой папки.

  • Нельзя переименовать или переместить корневой каталог библиотеки файлов в сообществе Tempo Social.
