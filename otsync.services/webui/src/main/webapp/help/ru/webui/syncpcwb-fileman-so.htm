Глава 2 – Управление файлами и папками

Веб-клиент можно использовать для добавления, копирования или перемещения элементов и папок в другое место в своей папке Tempo Box. Можно также просматривать версию и контрольный журнал, переименовывать или удалять элементы и папки в своей папке. Любые внесенные изменения автоматически синхронизируются с вашей папкой Tempo Box во всех других клиентах, которые находятся в оперативном режиме.

 

Примечания:

  • Веб-клиент невозможно использовать для изменения файлов в папке Tempo Box. Для обновлением файла с использованием веб-клиента необходимо загрузить его на свой настольный компьютер, отредактировать, затем добавить его опять как новую версию .

  • Результаты перемещения, переименования или удаления общей папки изменяются в зависимости от того, являетесь ли вы владельцем общей папки. Дополнительные сведения см. в разделе Перемещение, переименование и удаление общих папок .

 
 

Важно:
При просмотре файлов в общей папке с разрешением на чтение и запись файлы не резервируются автоматически. Может оказаться, что вы загружаете и редактируете файл в то же время, когда пользователь с общим доступом вносит в него изменения. Чтобы это предотвратить, можно зарезервировать файл. Папки для чтения-записи обозначаются или .

 

Администратор может определить лимит размера вашей папки Tempo Box. Если совокупный размер всех элементов в папке, исключая элементы в папках с общим для вас доступом, превышает ограничение, потребуется удаление или перемещение элементов из папки Tempo Box, пока объем не окажется меньше максимума. Подробнее см. в разделе Проверка размера папки Tempo Box.

Фотографии и изображения в папке Tempo Box отображаются с помощью эскизов изображений. Чтобы увидеть более крупное представление, можно переключиться на представление эскиза или выбрать изображение.

Можно также опубликовать элементы и папки в Content Server. Этот параметр недоступен для внешних пользователей. Иным образом, если вы подписались на папку Enterprise и имеете доступ к папке для чтения-записи, можно непосредственно взаимодействовать с элементами в папке. Дополнительные сведения о добавлении папки Enterprise к своей папке Tempo Box см. в разделе OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).
