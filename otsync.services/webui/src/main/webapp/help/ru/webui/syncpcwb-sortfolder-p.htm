Сортировка представления папки

Сортировка представления папки:

  1. Щелкните заголовок столбца, чтобы отсортировать содержимое по имени, времени последнего изменения или размеру.

  2. По выбору Чтобы отменить порядок сортировки, нажмите повторно заголовок столбца.

 

Замечание:
Если выбрано представление эскизов страниц, нажмите Sort By в верхнем правом углу для выбора метода сортировки. Дополнительные сведения о представлении эскизов см. в разделе Переключение на представление эскизов .
