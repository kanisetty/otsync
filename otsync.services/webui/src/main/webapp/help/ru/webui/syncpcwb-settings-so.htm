OpenText Tempo Box: Пользовательская справка по веб-клиентам

   

Глава 4 – Настройка параметров

На панели вверху справа щелкните свое имя, чтобы раскрыть меню, в котором можно выполнить следующее:

  • Обновление фотографии, которая отображается для других пользователей, просматривающих папки, для которых у них открыт общий доступ с вами.

  • Настройка OpenText Tempo Box для уведомления по эл. почте о приглашении вас любыми пользователями к совместному использованию папки или об обновлении общей папки. Ваш администратор должен включить уведомления по электронной почте, чтобы вы могли их получать.

  • Просмотр сведений о версии.

  • Выход из веб-клиента.
