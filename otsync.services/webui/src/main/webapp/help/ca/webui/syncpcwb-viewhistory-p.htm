Visualització de l'historial d'un element o una carpeta

Per visualitzar l'historial d'un element o una carpeta:

  1. Seleccioneu el menú de funcions de l'element i, a continuació, feu clic a Historial.

  2. Si visualitzeu l'historial d'un element, feu clic a una de les pestanyes següents:

    Historial d'auditoria

    Visualitza totes les accions que s'han dut a terme a l'element.

    Historial de versions

    Visualitza una llista amb totes les versions anteriors de l'element. Per visualitzar una versió anterior, feu clic al nom del document. Atès que no es poden crear versions de les carpetes, només es visualitza l'Historial d'auditoria.

     

    Nota:
    El contingut de la pestanya Historial d'auditoria depèn dels tipus d'esdeveniment que l'administrador hagi habilitat per auditar.
