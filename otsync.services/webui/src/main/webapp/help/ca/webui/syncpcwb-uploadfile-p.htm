Addició d'un element

Per afegir un element:

  1. Aneu a la ubicació a la qual vulgueu afegir un element.

    No podeu afegir elements a una carpeta per a la qual només teniu accés de lectura. Les carpetes de només lectura s'identifiquen mitjançant la superposició. Per exemple, o .

    No podeu afegir elements a una carpeta que tingui categories o atributs obligatoris a través de Tempo Box. Per determinar si una carpeta té categories i atributs obligatoris, consulteu Visualització de l'historial d'un element o una carpeta.

  2. Al panell central, feu clic a Afegeix fitxer.

    Tempo Box només admet fitxers de fins a 2 GB. Els fitxers més grans no són compatibles.

     

    Atenció
    Si afegiu contingut a una carpeta per a la qual teniu accés de lectura/escriptura, els canvis se sincronitzen amb tots els usuaris que hi tenen accés. Les carpetes amb permís de lectura/escriptura s'identifiquen mitjançant o .

    Si editeu contingut d'una carpeta d'empresa, els canvis se sincronitzen amb la carpeta de l'Espai de treball empresarial. Les carpetes d'empresa s'identifiquen mitjançant .

     

  3. Feu una de les accions següents:

    • Si es visualitza el botó Navega, feu-hi clic, seleccioneu un element i, tot seguit, feu clic a Puja.

    • Si es visualitza el quadre de diàleg Puja el fitxer, seleccioneu un element i feu clic a Obre.

 

Suggeriment:
Si utilitzeu l'última versió admesa del Chrome, també podreu afegir un fitxer arrossegant-lo des de l'escriptori a la finestra del navegador del client web. Només podeu arrossegar els fitxers, però no les carpetes. Quan s'arrossega un fitxer, el que es fa és desar-ne una còpia al client web, encara que el navegador indiqui que s'està desplaçant.
