1.2 – Para trocar para a página inicial padrão

Para trocar para a página inicial padrão:

  • No painel superior, clique em Minha página inicial padrão .

 

Notas:

  • Dependendo de como o OpenText Tempo estiver configurado, esta opção poderá estar indisponível. Usuários externos não podem acessar o Content Server.

  • Você pode definir a página inicial padrão na guia Geral da página Ferramentas > Configurações > Minhas configurações gerais no Content Server.
