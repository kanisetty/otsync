Interface do usuário

Este formato é usado para elementos da interface gráfica do usuário (GUI), como botões, nomes de ícones, itens de menu e campos.

Filenames, commands e dados de amostra

Esse formato é utilizado em nomes de arquivos, caminhos, URLs e comandos no prompt de comando. Ele também é usado para dados de exemplo, texto a ser inserido em caixas de texto e outros literais.

 

Nota:
Se você copiar exemplos de linha de comando de um PDF, esteja ciente de que os arquivos em formato PDF podem conter caracteres ocultos. A OpenText recomenda a cópia da versão HTML do documento, se estiver disponível.

 
NOMES DE TECLAS

Os nomes de teclas aparecem em MAIÚSCULAS, por exemplo:
Pressione CTRL+V.

<Nome de variável>

Os símbolos < > são usados ​​para denotar uma variável ou espaço reservado. O usuário deve substitui esses símbolos e o conteúdo descritivo pelo valor apropriado. Por exemplo, <server_name> torna-se serv01.

Referências cruzadas internas

Clique na referência cruzada para ir diretamente para o destino de referência no documento atual.

Referências cruzadas externas

Referências cruzadas externas são normalmente referências de texto a outros documentos. No entanto, se um documento estiver disponível em formato HTML, por exemplo, no Knowledge Center, referências externas podem ser links ativos para uma seção específica no documento referenciado.

Advertências, notas e dicas
 

Cautela
Cuidados ajudam a evitar problemas irreversíveis. Leia essas informações cuidadosamente e siga todas as instruções.

 
 

Importante:
Notas importantes o ajudam a evitar problemas maiores.

 
 

Nota:
Notas fornecem informações adicionais sobre uma tarefa.

 
 

Dica:
Dicas oferecem formas mais rápidas ou mais fáceis de executar uma tarefa.
