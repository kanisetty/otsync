Atualizando os colaboradores para uma pasta compartilhada

Para atualizar os colaboradores para uma pasta compartilhada:

  1. Clique em uma pasta compartilhada que seja sua.

  2. No painel direito, sob COLABORADORES, siga um destes procedimentos:

    • Para remover um colaborador, clique em ao lado do nome do colaborador e depois clique em Descompartilhar.

    • Para adicionar um colaborador, faça o seguinte:

      1. Clique em e depois clique em Somente leitura. para especificar um colaborador com acesso somente leitura.

      2. Na caixa, digite o nome da pessoa com quem deseja compartilhar essa pasta. Para um usuário externo digite o endereço de e-mail do usuário.

         

        Nota:
        Se o endereço de e-mail for ignorado e o recurso de preenchimento automático não exibir o endereço de e-mail do usuário externo, você não terá permissão para compartilhar pastas com usuários externos. Entre em contato com o administrador e solicite esse privilégio. Para obter mais informações sobre usuários externos, consulte Compartilhando com usuários externos.

         

      3. Clique no nome de usuário ou endereço de e-mail. Um convite de compartilhamento é enviado. Os colaboradores devem aceitar o convite de compartilhamento antes de acessar o conteúdo da pasta.

 

Nota:
Caso haja mais de sete colaboradores em uma pasta, clique em Editar todos os [ ] colaboradores para atualizar todos.
