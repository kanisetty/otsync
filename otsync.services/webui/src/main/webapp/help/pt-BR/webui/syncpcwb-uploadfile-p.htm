Adicionando um item

Para adicionar um item:

  1. Navegue até o local onde você deseja adicionar um item.

    Não é possível adicionar um item a uma pasta para a qual você tem acesso somente leitura. Pastas somente leitura são identificadas pela camada de sobreposição de bloqueio. Por exemplo, ou .

    Não é possível adicionar um item a uma pasta que possui categorias e atributos obrigatórios por meio do Tempo Box. Para determinar se uma pasta tem categorias e atributos obrigatórios, consulte Exibindo histórico de itens ou pastas.

  2. No painel central, clique em Adicionar arquivo.

    O Tempo Box lida apenas com arquivos de até 2 GB. Arquivos maiores que isso apresentarão erro.

     

    Cautela
    Se você adicionar conteúdo a uma pasta para a qual tem acesso de leitura/gravação, as alterações serão sincronizadas entre todos os usuários que têm acesso à pasta. As pastas de leitura/gravação são identificadas por ou .

    Se você editar o conteúdo de uma pasta corporativa, as alterações serão sincronizadas com a pasta no espaço de trabalho corporativo. As pastas corporativas são identificadas como .

     

  3. Siga um destes procedimentos:

    • Se um botão Procurar aparecer, clique em Procurar, selecione um item e clique em Carregar.

    • Se a caixa de diálogo Carregamento de arquivo aparecer, selecione um item e clique em Abrir.

 

Dica:
Se você estiver usando a última versão aceita do Chrome, poderá também adicionar um arquivo arrastando-o do seu desktop para a janela do navegador do cliente da Web. Você só pode arrastar arquivos, não pastas. Arrastar uma pasta sempre a copia para o cliente da Web, mesmo que o navegador indique que o arquivo está sendo movido.
