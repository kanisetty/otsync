Kapitel 1 – Erste Schritte mit dem Web-Client

Mithilfe des Web-Clients können Sie auf den Inhalt des OpenText Tempo-Online-Ordners auf dem OpenText Tempo-Server zugreifen. Mithilfe des Web-Clients können Sie die Elemente und Ordner im OpenText Tempo-Ordner verwalten sowie Ordner freigeben und auf Freigabeeinladungen anderer Benutzer antworten.

Für den Zugriff auf den Web-Client haben Sie folgende Möglichkeiten:

  • In einem Ordner im OpenText Tempo-Ordner des Desktop-Clients auf OpenText Tempo > Diesen Ordner freigeben klicken

  • Im Nachrichtenbereich mit der rechten Maustaste auf das Symbol für den OpenText Tempo-Desktop-Client klicken und dann auf Webschnittstelle starten klicken

Sie können die Befehle, die im Willkommen-Menü am oberen Rand des Web-Clients angezeigt werden, für folgende Aktionen verwenden:

  • Aktualisieren des Fotos, das andere Benutzer sehen, wenn Sie Ordner ansehen, die für Sie freigegeben sind.

  • Wechseln zur Content Server-Standardseite, wenn OpenText Tempo so konfiguriert ist, dass dies unterstützt wird. Diese Option ist nicht für externe Benutzer verfügbar.

  • Wenn Sie ein externer Benutzer sind, ändern Sie das Kennwort, das Sie bei der ersten Registrierung Ihres OpenText Tempo-Kontos eingegeben haben.

  • Anzeigen der Versionsinformationen.

  • Abmelden vom Web-Client.
