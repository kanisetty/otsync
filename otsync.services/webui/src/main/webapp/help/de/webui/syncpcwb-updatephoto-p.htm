OpenText Tempo Box: Benutzerhilfe für Web-Client

   

Ihr Foto aktualisieren

So aktualisieren Sie Ihr Foto:

  1. Klicken Sie oben rechts auf Ihren Namen, und klicken Sie dann auf Profil.

  2. Führen Sie einen der folgenden Schritte aus:

    • Klicken Sie auf Durchsuchen, und wählen Sie das zu verwendende Foto aus.

      Das Foto muss das Format .png, .gif oder .jpg aufweisen. Die Größe wird angepasst, und es wird eventuell beschnitten, um es an den verfügbaren Platz auf dem Display anzupassen.

    • Klicken Sie auf Standard verwenden, um das generische Profilbild zu verwenden.

  3. Klicken Sie auf Änderungen speichern.

 

Hinweis:
Wenn Sie OpenText Content Server Pulse verwenden, wird durch das Aktualisieren Ihres OpenText Tempo Box-Fotos auch Ihr Pulse Profile-Foto aktualisiert.
