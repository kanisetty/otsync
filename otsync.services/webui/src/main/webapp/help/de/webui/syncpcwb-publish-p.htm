Objekte oder Ordner veröffentlichen

So veröffentlichen Sie ein Objekt oder einen Ordner:

  1. Zeigen Sie auf das Funktionsmenü des Objekts und klicken Sie dann auf Veröffentlichen.

    Wenn diese Option nicht verfügbar ist, wenden Sie sich an den Administrator. Externe Benutzer können keine Objekte in Content Server veröffentlichen.

  2. Wenn Sie ein Objekt veröffentlichen möchten, klicken Sie auf eine der folgenden Optionen:

    Neue Version veröffentlichen

    Das Objekt wird als neue Version eines vorhandenen Objekts in Content Server hinzugefügt.

    In einem Ordner veröffentlichen

    Das Objekt wird als neues Objekt an einem Content Server-Speicherort wie Ihrem persönlichen Arbeitsbereich, einem Ordner oder einem Projekt hinzugefügt.

  3. Klicken Sie auf DurchsuchenContent Server, und wählen Sie eine der folgenden Optionen:

    • Ein Content Server-Objekt, wenn Sie eine neue Version eines Objekts veröffentlichen.

    • Einen Content Server-Ordner, wenn Sie ein neues Objekt oder einen neuen Ordner veröffentlichen.

  4. Klicken Sie auf eine der folgenden Optionen:

    Eine Kopie Ihrer Datei behalten

    Das bestehende Objekt bzw. der Ordner bleibt am derzeitigen Speicherort in Ihrem Tempo Box-Ordner erhalten.

    Datei in Content Server verschieben und Original löschen

    Das Objekt oder der Ordner wird aus Ihrem Tempo Box-Ordner gelöscht.

     

    Hinweis:
    Wenn Sie einen freigegebenen Ordner eines anderen Benutzers veröffentlichen, müssen Sie eine Kopie des Ordners in Ihrem Tempo Box-Ordner beibehalten.

     

  5. Klicken Sie auf Veröffentlichen.
