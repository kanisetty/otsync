Objekt- oder Ordnerverlauf anzeigen

So zeigen Sie den Objekt- oder Ordnerverlauf an:

  1. Zeigen Sie auf das Funktionsmenü des Objekts und klicken Sie dann auf Verlauf.

  2. Wenn Sie den Verlauf eines Objekts anzeigen, klicken Sie auf eine der folgenden Registerkarten:

    Protokollverlauf

    Zeigt jede an diesem Objekt durchgeführte Aktion an.

    Versionsverlauf

    Zeigt eine Liste aller früheren Versionen des Objekts an. Zum Anzeigen einer früheren Version klicken Sie auf Dokumentname. Ordner sind nicht versionierbar, sodass nur der Protokollverlauf angezeigt wird.

     

    Hinweis:
    Der Inhalt der Registerkarte Protokollverlauf hängt von den Ereignistypen ab, die Ihr Administrator für das Protokoll aktiviert hat.
