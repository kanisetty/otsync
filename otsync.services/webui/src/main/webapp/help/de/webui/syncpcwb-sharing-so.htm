Ordner freigeben und auf eine Freigabeeinladung antworten

Ihr Tempo Box-Ordner kann die folgenden Typen von Ordnern für die Zusammenarbeit enthalten: freigegebene Ordner, Unternehmensordner und Community-Dateibibliotheken.

Ein freigegebener Ordner ist ein Tempo Box-Ordner, den Sie freigegebenen haben oder der für Sie freigegeben wurde. Ist der Ordnername um einen Benutzernamen ergänzt, wurde er für Sie freigegeben. Zum Beispiel: Marketingmaterialien (jsmith).
Ein Unternehmensordner ist ein Ordner im Unternehmensarbeitsbereich, der mit Ihrem Tempo Box-Ordner synchronisiert wird. Ein Benutzer, der die Berechtigung Berechtigungen bearbeiten für den Ordner hat, kann den Ordner für Sie freigeben, oder Sie können zu dem Ordner im Unternehmensarbeitsbereich navigieren und den Ordner zu Ihrem Tempo Box-Ordner hinzufügen. Der Order muss Tempo Box-Anmeldungen zulassen, und Sie benötigen mindestens die Berechtigung Anzeigen zum Hinzufügen des Ordners. Weitere Informationen zum Hinzufügen von Unternehmensordnern zu Ihrem Tempo Box-Ordner finden Sie unter OpenText Tempo Box, Content Server Edition - User Online Help (SYNCLLES-H-UGD).
Eine Dateibibliothek aus einer Tempo Social-Community. Community-Dateibibliotheken werden nur angezeigt, wenn OpenText Tempo Social installiert ist. Kontaktieren Sie Ihren Administrator, um weitere Informationen zu erhalten.

Sie können mit anderen zusammenarbeiten, indem Sie Ordner innerhalb Ihres Tempo Box-Ordners freigeben. Wenn Sie einen Ordner freigeben, fügen Sie andere Benutzer als potenzielle Teilnehmer hinzu, die auch auf den Ordner zugreifen können, für den Sie Lese-/Schreibzugriff oder nur Lesezugriff gewährt haben.

Mit Schreib- und Leserechten erhalten Teilnehmer volle Rechte für den Inhalt des freigegebenen Ordners, einschließlich der Berechtigung zum Hinzufügen, Bearbeiten, Kopieren, Verschieben, Umbenennen und Löschen von Elementen und Ordnern im freigegebenen Ordner.

Mit reinen Leserechten können Teilnehmer die Elemente in einem freigegebenen Ordner anzeigen und kopieren. Teilnehmer, die nur Leserecht haben, können keine Elemente bearbeiten, verschieben oder löschen. Wenn ein Teilnehmer, der nur Leserecht hat, ein Element hinzufügt, wird es nicht mit anderen Teilnehmern synchronisiert. Wenn ein Teilnehmer nur Lesezugriff auf einen Ordner hat, wird die Schlossüberlagerung angezeigt. Beispiel: .

Sie können jeden Ordner in Ihrem Tempo Box-Ordner freigeben. Dabei gelten jedoch folgende Ausnahmen:

  • Der Tempo Box-Ordner auf der untersten Ebene der Struktur.

  • Ordner, die einem freigegebenen Ordner untergeordnet oder übergeordnet sind.

  • Ordner, die für Sie freigegeben wurden.

  • Unternehmensordner bei nur Lesezugriff, die durch gekennzeichnet sind. Wenn Sie Lese-/Schreibzugriff für einen Unternehmensordner haben, der durch gekennzeichnet ist, können Sie den Ordner freigeben, wenn Sie die Berechtigung Berechtigungen bearbeiten für diesen Ordner in Content Server haben.

  • Community-Dateibibliotheken unabhängig vom Zugriff (Lese-/Schreibzugriff oder nur Lesezugriff). Community-Dateibibliotheken sind durch und gekennzeichnet.

  • Externe Benutzer können keine Ordner freigeben.

Auf freigegebene Ordnerinhalte zugreifen

Als Teilnehmer müssen Sie eine Freigabeeinladung vom Eigentümer des Orders erhalten und angenommen haben, um den Ordner anzeigen zu können. Im oberen Bereich erscheint ein rote Benachrichtigung , wenn es ausstehende Einladungen an Sie gibt. Die Anzahl der ausstehenden Einladungen wird in der Benachrichtigung angezeigt. Nach der Annahme der Freigabe erscheint der freigegebene Ordner auf der untersten Ebene Ihres Tempo Box-Ordners, wobei dem Ordnernamen der Benutzername des Eigentümers angefügt wird. Beispiel: Marketingmaterialien (jsmith). Der Ordner wird als angezeigt, wenn Sie Lese-/Schreibzugriff haben, bzw. als , wenn Sie nur Lesezugriff haben.

Eine Benachrichtigung wird auch dann angezeigt, wenn Einladungen zu einem Unternehmensordner ausstehen. In diesem Fall wird der Ordner weiterhin im Stammverzeichnis Ihres Tempo Box-Ordners platziert und als angezeigt, wenn Sie Lese-/Schreibzugriff haben, bzw. als , wenn Sie nur Lesezugriff haben. Für den Unternehmensordner wird je nach Ihren Berechtigungen für den übergeordneten Ordner evtl. der Name des übergeordneten Ordners angehängt. Beispiel: Gehalt (Personal).

Die Ergebnisse beim Hinzufügen, Umbenennen oder Löschen des freigegebenen Ordners variieren je nachdem, ob Sie dessen Besitzer sind. Weitere Informationen finden Sie unter Freigegebene Ordner verschieben, umbenennen und löschen .

 

Wichtig:
Wenn Sie ein Objekt in einem freigegebenen Ordner, für den Sie Schreib-/Lesezugriff haben, zum Bearbeiten öffnen, wird es nicht automatisch reserviert. Mehrere Teilnehmer können gleichzeitig Änderungen an demselben Objekt vornehmen. Um dies zu verhindern, können Sie die Datei reservieren , bevor Sie sie bearbeiten. Jedes Mal, wenn Sie oder ein Teilnehmer Änderungen speichern, wird eine neue Version des Objekts hinzugefügt.

 

Unternehmensordner freigeben

Sie können einen Unternehmensordner freigeben, wenn Sie die Berechtigung Berechtigungen bearbeiten für diesen Ordner in Content Server haben. Bei der Freigabe eines Ordners können Sie Lese-/Schreibzugriff oder nur Lesezugriff auf den Ordner gewähren, wodurch folgende Ordnerberechtigungsebenen in Content Server gewährt werden.

Tabelle 3-1: Ordnerberechtigung und -zugriff in Tempo Box

Wenn Sie versuchen, einem Benutzer nur Lesezugriff zu gewähren, der umfassendere Berechtigungen als Anzeigen und Inhalt anzeigenhat, wird der Benutzer mit seiner aktuellen Zugriffsebene zur Freigabe hinzugefügt. Wenn Sie einen Teilnehmer aus einem Unternehmensordner entfernen oder wenn der Ordner aus dem Tempo Box-Ordner gelöscht wird, bleibt die gewährte Zugriffsebene in Content Server gleich.

Für externe Benutzer freigeben

Um einen Unternehmensordner für externe Benutzer freizugeben, müssen Sie die Berechtigung haben, Ordner für Benutzer freizugeben, die keine OpenText Tempo Box-Benutzerkonten haben. Wenden Sie sich an Ihren Administrator, um diese Berechtigung anzufordern. Sie müssen ferner die Berechtigung Berechtigungen bearbeiten für den Order in Content Server haben.

Wenn beide Bedingungen erfüllt sind, müssen Sie bei der Freigabe eines Ordners die E-Mail-Adresse des Benutzers angeben. Der externe Benutzer erhält dann eine Freigabeeinladung, in der er zur Registrierung für ein OpenText Tempo Box-Konto eingeladen wird. Nach der Registrierung kann der externe Benutzer Clients installieren und wie ein regulärer Benutzer auf OpenText Tempo Box zugreifen, wobei folgende Ausnahmen bestehen:

  • Externe Benutzer können Freigabeeinladungen von anderen Benutzern annehmen, aber selbst keine Ordner für andere freigeben.

  • Externe Benutzer können keine Elemente in Content Server veröffentlichen oder anderweitig auf das Programm zugreifen.
