Teilnehmer für einen freigegebenen Ordner aktualisieren

So aktualisieren Sie die Teilnehmer für einen freigegebenen Ordner:

  1. Klicken Sie auf einen Ihrer eigenen freigegebenen Ordner.

  2. Führen Sie im rechten Bereich unter TEILNEHMER einen der folgenden Schritte durch:

    • Um einen Teilnehmer zu entfernen, klicken Sie neben dem Namen des Teilnehmers auf und danach auf Freigabe aufheben.

    • Gehen Sie zum Hinzufügen eines Teilnehmers folgendermaßen vor:

      1. Klicken Sie auf und dann auf Schreibgeschützt, um einen Teilnehmer mit reinem Lesezugriff zu versehen.

      2. Geben Sie in das Feld den Namen der Person ein, für die Sie diesen Ordner freigeben möchten. Bei einem externen Benutzer geben Sie dessen vollständige E-Mail-Adresse ein.

         

        Hinweis:
        Wird die E-Mail-Adresse ignoriert und bleibt das Feld für die E-Mail-Adresse des externen Benutzers leer, verfügen Sie nicht über die Berechtigung, Ordner für externe Benutzer freizugeben. Wenden Sie sich an Ihren Administrator, um diese Berechtigung anzufordern. Weitere Informationen zu externen Benutzern finden Sie unter Für externe Benutzer freigeben.

         

      3. Klicken Sie auf den Namen oder die E-Mail-Adresse des Benutzers. Eine Freigabeeinladung wird gesendet. Teilnehmer müssen die Freigabeeinladung akzeptieren, um auf die Inhalte des Ordners zugreifen zu können.

 

Hinweis:
Wenn es mehr als sieben Teilnehmer für einen Ordner gibt, klicken Sie auf Alle [ ] Teilnehmer bearbeiten, um die Teilnehmeranzahl zu aktualisieren.
