Objektversion hinzufügen

So fügen Sie eine Objektversion hinzu:

  1. Navigieren Sie zum Ort des Objekts.

    Einem Objekt in einem Ordner, für den Sie nur Lesezugriff besitzen, können Sie keine Version hinzufügen. Ordner mit reinem Lesezugriff sind mit einer Schlossüberlagerung gekennzeichnet, zum Beispiel oder .

  2. Zeigen Sie auf das Funktionsmenü des Objekts und klicken Sie dann auf Version hinzufügen.

    Wenn die Datei bereits von einem anderen Benutzer reserviert wurde, können Sie keine Version zur Datei hinzufügen. Reservierte Dateien sind durch die -Überlagerung gekennzeichnet, zum Beispiel . Wer die Datei reserviert hat, können Sie anhand des Dateiverlaufs erkennen.

     

    Wichtig:
    Wenn Sie eine Objektversion zu einer Datei in einem freigegebenen Ordner hinzufügen, auf den Sie Schreib-/Lesezugriff haben, werden die Dateien nicht automatisch reserviert. Sie können eine Datei herunterladen und bearbeiten, während ein Teilnehmer Änderungen daran vornimmt. Ordner mit Lese- und Schreibzugriff sind gekennzeichnet durch oder . Um dies zu verhindern, können Sie die Datei reservieren .

     
     

    Achtung
    Wenn Sie Inhalte in einem Ordner bearbeiten, für den Sie Lese- und Schreibzugriff besitzen, werden die Änderungen mit allen Benutzern synchronisiert, die Zugriff auf den Ordner haben.

    Wenn Sie Inhalte in einem Unternehmensordner bearbeiten, werden die Inhalte mit dem Ordner im Unternehmensarbeitsbereich synchronisiert. Unternehmensordner sind gekennzeichnet durch .

     

  3. Führen Sie einen der folgenden Schritte aus:

    • Wenn die Schaltfläche Durchsuchen angezeigt wird, klicken Sie auf Durchsuchen, wählen ein Objekt aus und klicken auf Version hinzufügen.

    • Wenn das Dialogfeld Datei hochladen angezeigt wird, wählen Sie ein Objekt aus, und klicken Sie dann auf Öffnen.
