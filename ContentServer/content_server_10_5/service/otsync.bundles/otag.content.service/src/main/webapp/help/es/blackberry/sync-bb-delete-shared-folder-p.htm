Para suprimir una carpeta compartida de su propiedad:

  1. Haga clic en y desplácese hasta la carpeta destino, identificable mediante .

  2. Resalte la carpeta compartida.

  3. Pulse la tecla BlackBerry Menú y seleccione Suprimir carpeta.

     

    Atención
    La carpeta se eliminará para usted y para todos sus colaboradores. Como alternativa, puede suprimir colaboradores. Para más información, consulte Eliminación de un colaborador.

     
     

    Nota:
    Si Suprimir carpeta no está disponible es porque ha seleccionado una carpeta que se comparte con usted. Para obtener más información, consulte Para suprimir una carpeta que se comparte con usted más abajo.

     

  4. Haga clic en Aceptar.

Para suprimir una carpeta que se comparte con usted:

  1. Haga clic en .

  2. Resalte la carpeta compartida.

  3. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Eliminar elemento para compartir. Esta acción interrumpe la conexión con la carpeta compartida y la elimina de su carpeta OpenText Tempo. Esta acción no suprime la carpeta para el propietario ni el resto de colaboradores.

     

    Atención
    Si Eliminar elemento para compartir no está disponible, ha seleccionado una carpeta compartida de su propiedad. Para obtener más información, consulte Para suprimir una carpeta compartida de su propiedad al principio de esta página.

     

  4. Haga clic en Aceptar.
