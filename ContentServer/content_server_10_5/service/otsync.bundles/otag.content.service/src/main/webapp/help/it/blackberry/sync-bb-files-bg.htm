Per visualizzare un file:

  1. Fare clic su e posizionarsi sul file desiderato evidenziando una cartella e premendo il tasto BlackBerry Trackpad. Sulla barra del titolo di OpenText Tempo è visualizzato il nome della cartella visualizzata. Se il file è contrassegnato come preferito, fare clic su .

  2. Opzionale Per assicurarsi di consultare sempre i contenuti più recenti, aggiornare periodicamente i contenuti della cartella. Premere sul BlackBerry il tasto Menu e selezionare Aggiorna. Questa operazione consente di aggiornare la visualizzazione, ma non di scaricare i documenti. Per scaricare un documento è necessario aprirlo.

  3. Evidenziare il file che si desidera visualizzare.

  4. Premere sul BlackBerry il tasto Trackpad per visualizzare il file.

    Durante la visualizzazione di documenti in una cartella condivisa per cui si dispone di autorizzazioni di lettura/scrittura, i documenti non sono riservati. È possibile aprire e modificare un documento anche mentre lo sta facendo un collaboratore. Le cartelle condivise in lettura/scrittura sono identificate da .

    Durante la visualizzazione di documenti in una cartella condivisa per cui si dispone di un'autorizzazione di sola lettura, i documenti sono riservati. In queste cartelle è possibile visualizzare i documenti, ma non modificarli. Le cartelle condivise in sola lettura sono identificate da e durante l'esplorazione nella cartella sulla barra del titolo è visualizzato uno sfondo arancione.

  5. Premere sul BlackBerry il tasto Esc per tornare ai propri file.

Per modificare un file:

  1. Fare clic su e posizionarsi sul file desiderato. Se il file è contrassegnato come preferito, fare clic su .

  2. Evidenziare il file che si desidera modificare.

     

    Nota:
    Non è possibile modificare un file in una cartella condivisa per cui si dispone di un'autorizzazione di sola lettura, identificata da .

     

  3. Premere sul BlackBerry il tasto Trackpad per visualizzare il file.

  4. Il file viene aperto in un'applicazione di terze parti. Se l'applicazione supporta la modifica, modificare il file e salvare le modifiche.

    Le modifiche apportate al file non vengono salvate sul file nella cartella OpenText Tempo. È necessario caricare il file modificato dall'applicazione di terze parti o dalla cartella della cache del dispositivo BlackBerry.

  5. In OpenText Tempo, caricare il file modificato aggiungendo una versione del file. Per ulteriori dettagli, vedere la sezione Aggiunta di un file.

Per ulteriori informazioni sulla gestione dei file, fare riferimento alla sezione Visualizzazione, modifica e lavoro sui file.
