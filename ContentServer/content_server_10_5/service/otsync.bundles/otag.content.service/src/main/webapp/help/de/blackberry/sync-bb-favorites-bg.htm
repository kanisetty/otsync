So zeigen Sie einen Favoriten an:

  1. Klicken Sie auf .

  2. Optional Um sicherzustellen, dass sämtliche Favoriten angezeigt werden, aktualisieren Sie die Anzeige regelmäßig. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Aktualisieren aus. Hierdurch werden die Dokumente nicht zur Offline-Anzeige heruntergeladen. Um das Dokument herunterzuladen, öffnen Sie es oder synchronisieren Sie Ihre Favoriten. Weitere Informationen hierzu finden Sie unter So synchronisieren Sie Favoriten manuell weiter unten.

So öffnen Sie den Speicherort einer Favoritendatei:

  1. Klicken Sie auf .

  2. Markieren Sie die Datei, die Sie öffnen möchten, am Dateispeicherort.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Speicherort öffnen aus.

So entfernen Sie einen Favoriten:

  1. Klicken Sie auf oder auf , und navigieren Sie zur Zieldatei.

  2. Markieren Sie die Datei, die Sie aus der Favoritenliste entfernen möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Aus Favoriten entfernen aus. Die Datei wird nicht mehr in der Liste der Favoriten angezeigt. Dadurch wird die Datei nicht aus Ihrem OpenText Tempo-Ordner gelöscht.

So synchronisieren Sie Favoriten manuell:

  1. Klicken Sie auf .

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Favoriten synchronisieren aus. Wenn sich die Anwendung in der Synchronisierung befindet oder Sie offline sind, ist die Option Favoriten synchronisieren nicht verfügbar.

     

    Hinweis:
    Durch Auswählen von Aktualisieren wird die Bildschirmanzeige aktualisiert. Ihre Favoritendokumente werden dabei nicht zur Offline-Anzeige auf Ihr BlackBerry heruntergeladen. Um die Datei herunterzuladen, müssen Sie Ihre Favoritendateien synchronisieren oder die Datei öffnen.

     

Informationen zum automatischen Synchronisieren Ihrer Favoritendateien finden Sie unter OpenText Tempo-Einstellungen ändern.
