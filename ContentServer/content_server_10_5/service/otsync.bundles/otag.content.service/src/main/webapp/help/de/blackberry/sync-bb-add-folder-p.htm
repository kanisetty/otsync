So fügen Sie einen Ordner hinzu:

  1. Klicken Sie auf Dateien, und navigieren Sie zum Zielspeicherort für den neuen Ordner.

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ordner hinzufügen aus.

  3. Geben Sie im Feld Ordnername einen Namen für diesen Ordner ein.

  4. Drücken Sie die BlackBerry Eingabe-Taste.
