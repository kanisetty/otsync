Per creare un profilo:

  1. Se è la prima volta che si avvia l'applicazione OpenText Tempo, fare clic su Nuovo profilo.

    Se si desidera creare un profilo aggiuntivo ed è già stato effettuato l'accesso a OpenText Tempo, eseguire le seguenti operazioni:

    1. Premere sul BlackBerry il tasto Menu.

    2. Selezionare Scambia profilo.

    3. Fare clic su Nuovo profilo.

  2. Nella casella Nome profilo digitare un nome di profilo che risulti univoco sul dispositivo BlackBerry.

  3. Nella casella Nome utente, digitare il proprio nome utente. Se ci si trova fuori dall'azienda, digitare il proprio indirizzo e-mail.

     

    Nota:
    Un utente esterno all'azienda deve creare un account OpenText Tempo prima di poter creare un profilo.

    Per creare un account, un utente interno all'azienda deve condividere una cartella con l'utente esterno. Questa operazione genera un invito via e-mail per l'utente esterno, in cui vengono specificati i passaggi richiesti per creare un account.

     

  4. Toccare la casella Password e digitare la propria password.

    Se ci si trova fuori dall'azienda, digitare la password scelta quando è stato creato l'account OpenText Tempo. Se non si riesce a ricordare la password, fare riferimento al messaggio e-mail inviato dopo la creazione dell'account OpenText Tempo.

     

    Nota:
    La password è sensibile alle maiuscole/minuscole. Assicurarsi di digitare la password correttamente.

     

  5. Nella casella Server, digitare l'URL del proprio server OpenText Tempo.

    Contattare l'amministratore per conoscere l'URL del server OpenText Tempo.

    Se ci si trova fuori dall'azienda, l' URL di OpenText Tempo è contenuto nel messaggio e-mail ricevuto dopo la creazione dell'account OpenText Tempo.

  6. Opzionale Selezionare la casella di controllo Mantieni connessione per consentire l'accesso immediato all'avvio successivo dell'applicazione. Le uniche eccezioni a questa regola si presentano quando ci si scollega da OpenText Tempo o se la sessione scade. Vedere le informazioni contenute in Attenzione relative a questa opzione.

     

    Nota:
    Questa casella di controllo non è disponibile se il dispositivo BlackBerry è offline.

     

  7. Fare clic su Salva e accedi.

    I dati di profilo includono le credenziali dell'account, l'URL di OpenText Tempo e un elenco dei file preferiti. I profili sono specifici del dispositivo. È necessario creare un profilo e contrassegnare file come preferiti su ogni smartphone e tablet in cui viene eseguita l'applicazione OpenText Tempo.

 

Cautela

  • Consentire a OpenText Tempo di mantenere la connessione può rappresentare un rischio per la sicurezza, ad esempio nel caso in cui il proprio BlackBerry venga utilizzato da un'altra persona.

  • Se nel dispositivo BlackBerry sono configurati più profili, altri profili possono accedere all'account OpenText Tempo dell'utente se è selezionata la casella di controllo Mantieni connessione e l'utente non si è scollegato. Per uscire, premere sul BlackBerry il tasto Menu e selezionare Uscita.
