Para añadir una fotografía desde la cámara de BlackBerry:

  1. Haga clic en y desplácese hasta la carpeta destino en la que quiere añadir la fotografía.

     

    Nota:
    No puede añadir una fotografía a una carpeta compartida para la que tenga permiso de solo lectura, identificable mediante .

     

  2. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Añadir fotografía.

  3. Pulse el panel táctil del dispositivo BlackBerry para hacer la fotografía.

  4. Haga clic en Añadir aOpenText Tempo. Las fotos se guardan como archivos .jpg. No puede editar el nombre de la foto antes de cargarla.

Para añadir una fotografía desde una aplicación de BlackBerry:

  1. Abra la aplicación de cámara de BlackBerry.

  2. Desplácese hasta la fotografía que desea importar.

  3. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Añadir a OpenText Tempo.

  4. Opcional Si se le solicita, conéctese a OpenText Tempo. Para más información, consulte Conexión y administración de su perfil.

  5. Desplácese hasta la ubicación destino en la que desea colocar el archivo.

     

    Nota:
    No puede añadir una fotografía a una carpeta compartida para la que tenga permiso de solo lectura, identificable mediante .

     

  6. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Subir aquí.

Para añadir una carpeta:

  1. Haga clic en y desplácese hasta la ubicación destino para la nueva carpeta.

     

    Nota:
    No puede añadir una carpeta a otra carpeta compartida para la que tenga permiso de solo lectura, identificable mediante .

     

  2. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Añadir carpeta.

  3. En el recuadro Nombre de carpeta, escriba un nombre para esta carpeta.

  4. Pulse la tecla Intro del dispositivo BlackBerry.
