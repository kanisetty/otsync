Per inviare un file via e-mail:

  1. Fare clic su File e passare alla cartella del file di destinazione. Se il file è contrassegnato come preferito, fare clic su Preferiti.

  2. Evidenziare l'elemento che si desidera inviare via e-mail.

  3. Premere il tasto Menu BlackBerry e selezionare E-mail.

     

    Nota:
    Se la versione più recente del file non è presente sul proprio smartphone, il file viene scaricato prima che un messaggio e-mail venga visualizzato con il file allegato.

     

  4. Nella casella A, digitare gli indirizzi e-mail dei destinatari.

  5. Opzionale Nella casella Oggetto, digitare l'oggetto del messaggio e-mail.

  6. Opzionale Nel corpo del messaggio, digitare un messaggio.

  7. Premere il tasto Menu BlackBerry e selezionare Invia.
