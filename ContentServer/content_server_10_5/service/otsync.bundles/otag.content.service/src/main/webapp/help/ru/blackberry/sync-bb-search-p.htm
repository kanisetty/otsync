Поиск своей папки OpenText Tempo

  1. Щелкните .

  2. Введите термин поиска.

  3. Нажмите клавишу BlackBerry Ввод. Отобразятся результаты поиска.

  4. Выделите необходимый для просмотра файл или папку.

  5. Нажмите клавишу BlackBerry Меню для отображения доступных параметров.

Чтобы открыть местоположение элемента в результатах поиска, выполните следующие действия.

  1. Выполните поиск папки OpenText Tempo.

  2. Выделите файл, который необходимо открыть в местоположении файла.

  3. Нажмите клавишу BlackBerry Меню и выберите Открыть местоположение.
