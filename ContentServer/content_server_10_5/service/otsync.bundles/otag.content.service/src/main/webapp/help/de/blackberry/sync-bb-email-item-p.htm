So senden Sie eine Datei per E-Mail:

  1. Klicken Sie auf Dateien, und navigieren Sie zur Zieldatei. Wenn die Datei als Favorit markiert ist, klicken Sie auf Favoriten.

  2. Markieren Sie das Element, das Sie per E-Mail versenden möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie E-Mail aus.

     

    Hinweis:
    Wenn die neueste Version der Datei auf Ihrem Smartphone nicht verfügbar ist, wird die Datei heruntergeladen, bevor eine E-Mail-Nachricht mit der Datei im Anhang angezeigt wird.

     

  4. Geben Sie im Feld An die E-Mail-Adressen der Empfänger der E-Mail ein.

  5. Optional Geben Sie im Feld Betreff den Betreff der Nachricht ein.

  6. Optional Geben Sie im Textfeld eine Nachricht ein.

  7. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Senden aus.
