So löschen Sie Ihr Profil:

  1. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Profil wechseln aus.

  2. Markieren Sie das Profil, das Sie entfernen möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Profil löschen aus.

  4. Bestätigen Sie diesen Vorgang, indem Sie auf OK klicken.
