Para suprimir un archivo:

  1. Haga clic en y desplácese hasta el archivo destino. Si el archivo destino se ha marcado como favorito, haga clic en .

     

    Nota:
    Antes de continuar, consulte la información de precaución más abajo.

    No puede suprimir un archivo de una carpeta compartida para la que tenga permiso de solo lectura, identificable mediante .

     

  2. Resalte el archivo destino.

  3. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Suprimir elemento.

  4. Pulse Aceptar para confirmar la acción.

Para suprimir una carpeta:

  1. Haga clic en y desplácese hasta la carpeta destino.

     

    Nota:
    Antes de continuar, consulte la información de precaución más abajo.

    Si la carpeta destino es una carpeta compartida, consulte Eliminación de una carpeta compartida antes de continuar. Las carpetas compartidas se identifican mediante o .

     

  2. Resalte la carpeta.

  3. Pulse la tecla BlackBerry Menú y seleccione Suprimir carpeta.

  4. Pulse Aceptar para confirmar la acción.

 

Atención

  • Al suprimir un archivo o carpeta, se elimina de la carpeta OpenText Tempo, incluso si el archivo se marcó como favorito. Esto significa que el archivo o carpeta se elimina del resto de clientes de OpenText Tempo instalados, como el cliente de escritorio, otros smartphones o tabletas, así como la interfaz de usuario web de OpenText Tempo.

  • Si se suprime un archivo o carpeta de una carpeta compartida para la que tenga permisos de lectura/escritura, identificable mediante , dicho archivo o carpeta se suprime automáticamente para todos los colaboradores que tienen acceso compartido a la carpeta, incluido el propietario de dicha carpeta.
