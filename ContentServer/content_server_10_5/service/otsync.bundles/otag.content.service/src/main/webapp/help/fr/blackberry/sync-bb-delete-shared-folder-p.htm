Pour supprimer un dossier partagé qui vous appartient :

  1. Cliquez sur , puis accédez au dossier partagé cible, identifié par .

  2. Mettez en surbrillance le dossier partagé.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Supprimer un dossier.

     

    Attention
    Cette action supprime le dossier pour vous et pour l'ensemble des collaborateurs. Il vous est possible de supprimer des collaborateurs. Pour plus d'informations, voir Suppression d'un collaborateur.

     
     

    Note :
    Si Supprimer un dossier n'est pas disponible, vous avez sélectionné un dossier qu'un utilisateur partage avec vous. Pour plus d'informations, voir Pour supprimer un dossier qu'un utilisateur partage avec vous, ci-dessous.

     

  4. Cliquez sur OK.

Pour supprimer un dossier qu'un utilisateur partage avec vous :

  1. Cliquez sur .

  2. Mettez en surbrillance le dossier partagé.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Supprimer le partage. Cela interrompt l'accès au dossier partagé et supprime ce dernier de votre dossier OpenText Tempo. Cela ne supprime le dossier ni pour le propriétaire ni pour les autres collaborateurs.

     

    Attention
    Si Supprimer le partage n'est pas disponible, vous avez sélectionné un dossier partagé qui vous appartient. Pour plus d'informations, voir Pour supprimer un dossier partagé qui vous appartient, en haut de la page.

     

  4. Cliquez sur OK.
