To add a photo using the BlackBerry camera:

  1. Click and navigate to the target folder where you want to add a photo. You cannot add a photo to a shared folder to which you have read-only access. Files in a read-only shared folder are marked with the overlay. For example, .

  2. Press the BlackBerry Menu key and select Add Photo.

  3. Press the BlackBerry Trackpad key to take a picture.

  4. Click Add to Tempo Box. Photos are saved as .jpg files. You cannot edit the name of the photo before uploading it.

To add a photo from a BlackBerry app:

  1. Launch a BlackBerry camera app.

  2. Navigate to the photo that you want to import.

  3. Press the BlackBerry Menu key and select Add to Tempo Box.

  4. Optional If prompted, sign in to OpenText Tempo Box. For details, see Signing In and Managing Your Profile.

  5. Navigate to the target location where you want to place the file. You cannot add a photo to a shared folder to which you have read-only access. Read-only shared folders are identified by the overlay. For example, or .

  6. Press the BlackBerry Menu key and select Upload Here.

To add a folder:

  1. Click and navigate to the target location for the new folder. You cannot add a folder to a shared folder to which you have read-only access. Read-only shared folders are identified by the overlay. For example, or .

  2. Press the BlackBerry Menu key and select Add Folder.

  3. In the Folder Name box, type a name for this folder.

  4. Press the BlackBerry Enter key.
