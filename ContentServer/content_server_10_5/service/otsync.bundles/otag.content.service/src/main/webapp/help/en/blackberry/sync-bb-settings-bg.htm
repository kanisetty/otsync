To change OpenText Tempo Box settings:

  1. Press the BlackBerry Menu key and select Settings.

  2. Optional Highlight the Device Storage list, press the BlackBerry Trackpad key and adjust the storage space for your cached files. If you exceed the set storage limit, the oldest cached files are deleted until the cache size is below the specified limit. The default cache size is 10 MB.

     

    Notes:

    • The Space Used row shows the total size of your favorite files, not your cached files.

    • The Quota Usage row shows the quota and size of your Tempo Box folder. This excludes shared folders that you do not own.

     

  3. Optional Highlight the Sync Favorites list, press the BlackBerry Trackpad key and select Manual or Auto.

    With automatic syncing, the following occurs:

    • New versions of favorite files are downloaded in the background.

    • The file is downloaded for offline viewing when you tag the file as a favorite.

    With manual syncing, the following occurs:

    • New versions of favorite files are not downloaded automatically.

    • You need to open the file or manually sync your favorites to download the file for offline viewing.

      To sync your favorites manually, do the following:

      1. Click .

      2. Press the BlackBerry Menu key.

      3. Select Sync Favorites.

  4. Optional Highlight the Default Sort Order list, press the BlackBerry Trackpad key, and select Date or Name.

  5. Press the BlackBerry Escape key and select Save.
