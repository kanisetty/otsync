Pour ajouter un dossier :

  1. Cliquez sur Fichiers et accédez à l'emplacement cible du nouveau dossier.

  2. Appuyez sur la touche BlackBerry Menu et sélectionnez Ajouter un dossier.

  3. Dans le champ Nom du dossier, saisissez un nom pour ce dossier.

  4. Appuyez sur la touche BlackBerry Entrée.
