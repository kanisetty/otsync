要删除文件：

  1. 单击 ，然后导航到目标文件。如果目标文件已标记为收藏，则单击 。

     

    注意:
    继续操作之前，请参阅下方的注意信息。

    您无法在具有只读权限、由 标识的共享文件夹中删除文件。

     

  2. 突出显示目标文件。

  3. 按 BlackBerry [菜单]键，然后选择[删除项目]。

  4. 单击[确定]以确认操作。

要删除文件夹：

  1. 单击 ，然后导航到目标文件夹。

     

    注意:
    继续操作之前，请参阅下方的注意信息。

    如果目标文件夹是共享文件夹，请在继续操作之前参阅删除共享文件夹。共享文件夹由 或 标识。

     

  2. 突出显示该文件夹。

  3. 按 BlackBerry [菜单]键，然后选择[删除文件夹]。

  4. 单击[确定]以确认操作。

 

小心

  • 即使文件标记为收藏，删除该文件或文件夹也会将其从 OpenText Tempo 文件夹移除。这意味着该文件或文件夹会从您安装的所有其他 OpenText Tempo 客户端删除，例如桌面客户端、其他智能手机或平板电脑以及 OpenText Tempo Web 用户界面。

  • 如果从您具有读/写权限、由 标识的共享文件夹删除文件或文件夹，则自动为包括共享文件夹所有者在内的所有拥有文件夹共享访问权限的协作者删除该文件或文件夹。
