要删除您拥有的共享文件夹：

  1. 单击 [文件]，然后导航到目标文件夹。

  2. 突出显示该共享文件夹。

  3. 按 BlackBerry [菜单]键，然后选择[删除文件夹]。

     

    注意:
    这将为您和所有协作者删除该文件夹。或者，您可移除协作者。有关详细信息，请参阅移除协作者。

    如果[删除文件夹]不可用，则您选择了与您共享的文件夹。继续操作之前，请参阅删除共享文件夹。

     

  4. 单击[确定]。

要删除他人与您共享的文件夹：

  1. 单击 [文件]。

  2. 突出显示要切断连接的共享文件夹。

  3. 按 BlackBerry [菜单]键，然后选择[移除共享]。

     

    注意:
    如果[移除共享]不可用，则您选择了自己拥有的文件夹。继续操作之前，请参阅删除共享文件夹。

     

  4. 单击[确定]。

删除共享文件夹

共享文件夹由文件夹 右下角中的两个绿色头像表示。存在两种类型的共享文件夹：

  • 您拥有并与同事共享的文件夹。删除您拥有的共享文件夹将为您和拥有文件夹共享访问权限的所有协作者删除该文件夹。或者，您可移除一个或所有协作者。有关详细信息，请参阅移除协作者。

  • 您不拥有但共享的文件夹。删除与您共享的文件夹会切断与共享文件夹的连接并从您的 OpenText Tempo 文件夹移除该文件夹。此操作不会删除所有者或其他协作者的文件夹。
