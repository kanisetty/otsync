So geben Sie einen Ordner für einen externen Benutzer frei:

  1. Klicken Sie auf Dateien, und navigieren Sie zum Zielordner.

  2. Markieren Sie den Ordner.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ordner freigeben aus.

     

    Hinweis:
    Wenn die Option Ordner freigeben nicht verfügbar ist, verfügen Sie nicht über die Berechtigung zum Freigeben dieses Ordners. Eine Liste von Ordnern, die nicht freigegeben werden können, finden Sie unter Ausnahmen zur Freigabe für externe Benutzer.

     

  4. Geben Sie die E-Mail-Adresse des externen Benutzers, für den Sie diesen Ordner freigeben möchten, ein.

    Externe Benutzer sind Benutzer, die nicht Mitarbeiter des Unternehmens sind. Nachdem Sie eine gültige E-Mail-Adresse eingegeben haben, wird die E-Mail-Adresse des externen Benutzers automatisch angezeigt.

     

    Hinweis:
    Wird die E-Mail-Adresse ignoriert und „Keine Ergebnisse gefunden“ angezeigt, verfügen Sie nicht über die Berechtigung, Ordner für externe Benutzer freizugeben. Wenden Sie sich an Ihren Administrator, um diese Berechtigung anzufordern.

     

  5. Markieren Sie die E-Mail-Adresse, und drücken Sie die Taste BlackBerry Trackpad.

  6. Klicken Sie auf Ja, um zu bestätigen, dass Sie diese Einladung senden möchten. Der Ordner wird nun als freigegebener Ordner markiert: .

Ausnahmen zur Freigabe für externe Benutzer

Sie können jeden Ordner in Ihrem OpenText Tempo-Ordner freigeben. Es gelten jedoch die folgenden Ausnahmen:

  • Sie können keinen Ordner freigeben, der einem freigegebenen Ordner untergeordnet oder übergeordnet ist.

  • Sie können keinen Ordner freigeben, der für Sie freigegeben wurde.

  • Das Stammverzeichnis eines freigegebenen Ordners können Sie nur freigeben, wenn Sie der Eigentümer des Ordners sind.

  • Sie können Ihren OpenText Tempo-Ordner nicht freigeben.

  • Wenn Sie nicht Mitarbeiter des Unternehmens sind, können Sie keine Ordner freigeben.

 

Achtung
Durch Freigeben eines Ordners für einen externen Benutzer erhält der Benutzer die vollständige Kontrolle über die Inhalte des Ordners. Wenn am freigegebenen Ordner Änderungen vorgenommen wurden, werden diese Änderungen automatisch mit allen Teilnehmern, die Zugriff auf diesen Ordner haben, einschließlich des Eigentümers, synchronisiert.
