So zeigen Sie eine Datei an:

  1. Klicken Sie auf , und navigieren Sie zur Zieldatei, indem Sie einen Ordner markieren und die BlackBerry Trackpad-Taste drücken. In der OpenText Tempo-Titelleiste wird der Name des aktuell angezeigten Ordners angezeigt. Wenn die Zieldatei als Favorit markiert ist, klicken Sie auf .

  2. Optional Um sicherzugehen, dass Sie den aktuellsten Inhalt anzeigen, sollten Sie den Ordnerinhalt regelmäßig aktualisieren. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Aktualisieren aus. Dadurch wird die Anzeige aktualisiert, aber die Dokumente werden nicht heruntergeladen. Öffnen Sie das Dokument, um es herunterzuladen.

  3. Markieren Sie die Datei, die Sie anzeigen möchten.

  4. Drücken Sie die BlackBerry Trackpad-Taste, um die Datei anzuzeigen.

    Bei der Anzeige von Dokumenten in einem freigegebenen Ordner, für den Sie Lese- und Schreibrechte besitzen, werden diese Dokumente nicht reserviert. Sie können ein Dokument zur selben Zeit öffnen und bearbeiten, zu der ein anderer Teilnehmer Änderungen daran vornimmt. Freigegebene Ordner mit Lese- und Schreibrechten sind durch gekennzeichnet.

    Bei der Anzeige von Dokumenten in einem freigegebenen Ordner, für den Sie nur Leserechte besitzen, werden diese Dokumente reserviert. Sie können Dokumente in diesen Ordnern nur anzeigen, jedoch nicht bearbeiten. Freigegebene Ordner mit Leserechten sind durch gekennzeichnet, und die Titelleiste weist beim Navigieren im Ordner einen orangefarbenen Hintergrund auf.

  5. Drücken Sie die BlackBerry Esc-Taste, um zu Ihren Dateien zurückzukehren.

So bearbeiten Sie eine Datei:

  1. Klicken Sie auf , und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, klicken Sie auf .

  2. Markieren Sie die Datei, die Sie entfernen möchten.

     

    Hinweis:
    Eine Datei in einem freigegebenen Ordner, für den Sie nur Leserechte besitzen (durch gekennzeichnet), können Sie nicht bearbeiten.

     

  3. Drücken Sie die BlackBerry Trackpad-Taste, um die Datei anzuzeigen.

  4. Die Datei wird in einer Drittanbieteranwendung geöffnet. Wenn die Anwendung das Bearbeiten unterstützt, bearbeiten Sie die Datei, und speichern Sie Ihre Änderungen.

    Die Änderungen, die Sie an der Datei vorgenommen haben, werden nicht in der Datei im OpenText Tempo-Ordner gespeichert. Sie müssen die geänderte Datei aus der Drittanwendung oder dem BlackBerry-Cache-Ordner hochladen.

  5. Laden Sie in OpenText Tempo die bearbeitete Datei durch Hinzufügen einer Version der Datei hoch. Ausführliche Informationen finden Sie in Dateien hinzufügen.

Weitere Informationen zur Verwaltung von Dateien finden Sie unter Dateien anzeigen, bearbeiten und verwenden.
