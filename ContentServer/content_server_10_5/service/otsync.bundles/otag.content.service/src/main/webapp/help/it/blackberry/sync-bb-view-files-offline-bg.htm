Per visualizzare file offline:

  1. Per rendere un file disponibile per la visualizzazione offline, contrassegnarlo come preferito e sincronizzarlo o scaricarlo mentre ci si trova online. Per ulteriori dettagli, vedere la sezione Aggiunta, visualizzazione, rimozione e sincronizzazione dei preferiti.

    La visualizzazione di un overlay indica che il file preferito non è stato scaricato. Ad esempio, . Per scaricare il file, aprirlo o sincronizzare i preferiti. Per ulteriori dettagli, vedere la sezione Aggiunta, visualizzazione, rimozione e sincronizzazione dei preferiti.

  2. Fare clic su .

  3. Evidenziare il file che si desidera visualizzare.

  4. Premere sul BlackBerry il tasto Trackpad per visualizzare il file.

    I file aperti quando si è offline potrebbero non essere le versioni più recenti, poiché le modifiche apportate tramite un altro client non possono essere sincronizzate quando si è offline.

  5. Premere sul BlackBerry il tasto Esc per tornare ai propri file.
