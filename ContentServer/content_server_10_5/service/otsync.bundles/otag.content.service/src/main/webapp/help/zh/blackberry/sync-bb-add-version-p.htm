要使用您 BlackBerry 上的文件添加新版本：

  1. 单击 ，然后导航到目标文件。如果目标文件已标记为收藏，则单击 。

  2. 突出显示要向其添加新版本的目标文件。

     

    注意:
    您无法在具有只读权限、由 标识的共享文件夹中向文件添加版本。

     

  3. 按 BlackBerry [菜单]键，然后选择[上载新版本]。

  4. 将您的 BlackBerry 导航至文件的新版本。

  5. 突出显示该文件。

  6. 按 BlackBerry [菜单]键，然后选择[上载]。文件会自动上载。

要上载存储在 BlackBerry 中的文件：

  1. 单击 ，然后导航到要上载文件的目标文件夹。

  2. 按 BlackBerry [菜单]键，然后选择[在此上载]。

  3. 导航到 BlackBerry 智能手机上的目标文档。

     

    注意:
    您无法将文件上载到具有只读权限、由 标识的共享文件夹中。

     

  4. 按 BlackBerry [菜单]键，然后选择[上载]。

要从 BlackBerry 应用程序导入文件：

  1. 启动 BlackBerry 应用程序。

  2. 按 BlackBerry [菜单]键，然后选择[添加到 OpenText Tempo]。

  3. 可选 如果出现提示，登录到 OpenText Tempo。有关详细信息，请参阅登录和管理配置文件。

  4. 导航到要在其中放置文件的目标位置。

     

    注意:
    您无法将文件导入具有只读权限、由 标识的共享文件夹中。

     

  5. 按 BlackBerry [菜单]键，然后选择[在此上载]。
