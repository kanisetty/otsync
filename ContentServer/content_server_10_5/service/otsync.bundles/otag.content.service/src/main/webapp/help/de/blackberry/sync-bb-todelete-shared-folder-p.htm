So löschen Sie einen freigegebenen Ordner, dessen Eigentümer Sie sind:

  1. Klicken Sie auf Dateien, und navigieren Sie zum Zielordner.

  2. Markieren Sie den freigegebenen Ordner.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ordner löschen aus.

     

    Hinweis:
    Dadurch wird der Ordner für Sie und alle Teilnehmer gelöscht. Alternativ können Sie Teilnehmer entfernen. Ausführliche Informationen finden Sie in So entfernen Sie Teilnehmer.

    Wenn Ordner löschen nicht verfügbar ist, haben Sie einen Ordner ausgewählt, der für Sie freigegeben ist. Vor dem Fortfahren finden Sie weitere Informationen unter Freigegebenen Ordner löschen.

     

  4. Klicken Sie auf OK.

So löschen Sie einen Ordner, der für Sie freigegeben wurde:

  1. Klicken Sie auf Dateien.

  2. Markieren Sie den freigegebenen Ordner, in dem Sie die Verbindung trennen möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Freigabe entfernen aus.

     

    Hinweis:
    Wenn Freigabe entfernen nicht verfügbar ist, haben Sie einen Ordner, dessen Eigentümer Sie sind, ausgewählt. Vor dem Fortfahren finden Sie weitere Informationen unter Freigegebenen Ordner löschen.

     

  4. Klicken Sie auf OK.

Freigegebenen Ordner löschen

Die freigegebenen Ordner werden durch zwei grüne Avatare in der unteren rechten Ecke des Ordners markiert. Es gibt zwei Arten von freigegebenen Ordnern:

  • Ein Ordner, der Ihnen gehören und den Sie für Ihre Kollegen freigeben. Durch Löschen eines freigegebenen Ordners, der Ihnen gehört, wird dieser Ordner für Sie und alle anderen Teilnehmer, die Zugriff auf diesen Ordner haben, gelöscht. Als Alternative können Sie einen oder alle Teilnehmer entfernen. Weitere Informationen erhalten Sie unter So entfernen Sie Teilnehmer.

  • Ein Ordner, der Ihnen nicht gehört und der für Sie freigegeben wurde. Durch Löschen eines Ordners, der für Sie freigegeben ist, wird die Verbindung zum freigegebenen Ordner aufgehoben und der Ordner aus Ihrem OpenText Tempo-Ordner entfernt. Für den Eigentümer und andere Teilnehmer wird der Ordner nicht gelöscht.
