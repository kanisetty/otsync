要使用 BlackBerry 照相机添加照片：

  1. 单击 ，然后导航到要在其中添加照片的目标文件夹。

     

    注意:
    您无法在具有只读权限、由 标识的共享文件夹中添加照片。

     

  2. 按 BlackBerry [菜单]键，然后选择[添加照片]。

  3. 按 BlackBerry [触控板]键以拍摄照片。

  4. 单击[添加到 OpenText Tempo]。照片保存为 .jpg 文件。上载照片之前不能编辑照片名称。

要从 BlackBerry 应用程序添加照片：

  1. 启动 BlackBerry 照相机应用程序。

  2. 导航到您要导入的照片。

  3. 按 BlackBerry [菜单]键，然后选择[添加到 OpenText Tempo]。

  4. 可选 如果出现提示，登录到 OpenText Tempo。有关详细信息，请参阅登录和管理配置文件。

  5. 导航到要在其中放置文件的目标位置。

     

    注意:
    您无法在具有只读权限、由 标识的共享文件夹中添加照片。

     

  6. 按 BlackBerry [菜单]键，然后选择[在此上载]。

要添加文件夹：

  1. 单击 ，然后导航到新文件夹的目标位置。

     

    注意:
    您无法在具有只读权限、由 标识的共享文件夹中添加文件夹。

     

  2. 按 BlackBerry [菜单]键，然后选择[添加文件夹]。

  3. 在[文件夹名称]框中，键入此文件夹的名称。

  4. 按 BlackBerry [Enter] 键。
