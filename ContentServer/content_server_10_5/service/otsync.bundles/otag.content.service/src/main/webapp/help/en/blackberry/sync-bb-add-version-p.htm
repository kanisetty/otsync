To add a new version using a file on your BlackBerry:

  1. Click and navigate to the target file. You cannot add a version to a file in a shared folder to which you have read-only access. Files in a read-only shared folder are marked with the overlay. For example, .

  2. Highlight the target file to which you want to add a new version.

  3. Press the BlackBerry Menu key and select Upload New Version.

  4. Navigate to the new version of the file on your BlackBerry.

  5. Highlight the file.

  6. Press the BlackBerry Menu key and select Upload.

To upload a file stored on your BlackBerry:

  1. Click and navigate to the target folder where you want to upload a file. You cannot upload a file to a shared folder to which you have read-only access. Files in a read-only shared folder are marked with the overlay. For example, .

  2. Press the BlackBerry Menu key and select Upload Here.

  3. Navigate to the target file on your BlackBerry smartphone.

  4. Press the BlackBerry Menu key and select Upload.

To import a file from a BlackBerry app:

  1. Launch a BlackBerry app.

  2. Press the BlackBerry Menu key and select Add to OpenText Tempo Box.

  3. Optional If prompted, sign in to OpenText Tempo Box. For details, see Signing In and Managing Your Profile.

  5. Press the BlackBerry Menu key and select Upload Here.
