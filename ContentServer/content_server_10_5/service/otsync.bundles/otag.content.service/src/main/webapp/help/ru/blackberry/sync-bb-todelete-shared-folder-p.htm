Удаление общей папки, принадлежащей вам

  1. Нажмите Файлы и перейдите к целевой папке.

  2. Выделите общую папку.

  3. Нажмите клавишу BlackBerry Меню и выберите Удалить папку.

     

    Замечание:
    Это приведет к удалению папки для вас и всех пользователей, имеющих общий доступ. В качестве альтернативы можно удалить пользователей, имеющих общий доступ. Подробную информацию см. в Удаление пользователей с общим доступом.

    Если кнопка Удалить папку недоступна, была выбрана папка, к которой вам предоставлен общий доступ. До продолжения просмотрите Удаление общей папки.

     

  4. Нажмите OK.

Удаление общей папки, принадлежащей вам

  1. Нажмите Файлы.

  2. Выделите общую папку, для которой необходимо удалить соединение.

  3. Нажмите клавишу BlackBerry Меню и выберите Удалить общий ресурс.

     

    Замечание:
    Если пункт Удалить общий ресурс недоступен, выберите папку, владельцем которой вы являетесь. До продолжения просмотрите Удаление общей папки.

     

  4. Нажмите OK.

Удаление общей папки

Общие папки указываются с помощью двух зеленых аватаров в нижнем правом углу папки . Существует два типа общих папок:

  • Папки, которые совместно используются с коллегами. При удалении общей папки, которая принадлежит вам, папка будет удалена для вас и всех пользователей с общим доступом, для которых открыт общий доступ к этой папке. В качестве альтернативы можно удалить одного или всех пользователей с общим доступом. Дополнительную информацию см. в Удаление пользователей с общим доступом.

  • Папка, которая вам не принадлежит, и к которой у вас есть общий доступ. При удалении папки, к которой для вас предоставлен общий доступ, разрывается соединение с общей папкой и она удаляется из папки OpenText Tempo. При этом папка не удаляется для ее владельца или других пользователей с общим доступом.
