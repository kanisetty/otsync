Per eliminare un file:

  1. Fare clic su e posizionarsi sul file desiderato. Se il file è contrassegnato come preferito, fare clic su .

     

    Nota:
    Prima di continuare, fare riferimento alle informazioni contenute in Attenzione in basso.

    Non è possibile eliminare un file in una cartella condivisa per cui si dispone di un'autorizzazione di sola lettura, identificata da .

     

  2. Evidenziare il file.

  3. Premere sul BlackBerry il tasto Menu e selezionare Elimina elemento.

  4. Fare clic su OK per confermare l'azione.

Per eliminare una cartella:

  1. Fare clic su e posizionarsi sulla cartella desiderata.

     

    Nota:
    Prima di continuare, fare riferimento alle informazioni contenute in Attenzione in basso.

    Se la cartella di destinazione è una cartella condivisa, vedere Eliminazione di una cartella condivisa prima di continuare. Le cartelle condivise sono identificate da o .

     

  2. Evidenziare la cartella.

  3. Premere sul BlackBerry il tasto Menu e selezionare Elimina cartella.

  4. Fare clic su OK per confermare l'azione.

 

Cautela

  • Eliminando un file o una cartella, lo si rimuove dalla cartella OpenText Tempo, anche se il file era stato contrassegnato come preferito. Ciò significa che il file o la cartella è stato eliminato da tutti gli altri client OpenText Tempo installati come client desktop, dagli altri smartphone o tablet, nonché dall'interfaccia utente Web OpenText Tempo.

  • Se un file o una cartella viene eliminato da una cartella condivisa di cui si dispone un'autorizzazione di lettura/scrittura, identificata da , viene automaticamente eliminato per tutti i collaboratori che hanno accesso in condivisione alla cartella, incluso il proprietario della cartella condivisa.
