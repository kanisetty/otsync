Pour visualiser des éléments tout en étant hors ligne :

  1. Pour faire en sorte qu'un élément puisse être visualisé hors ligne, marquez le fichier souhaité comme favori lorsque vous êtes en ligne. Pour plus d'informations, voir Ajout d'un favori. Les fichiers marqués comme favoris sont indiqués par le recouvrement . Par exemple : .

  2. Cliquez sur Favoris.

  3. Mettez en surbrillance l'élément que vous souhaitez visualiser.

  4. Appuyez sur la touche BlackBerry Trackpad pour afficher l'élément. Il est possible que les fichiers que vous aviez ouverts lorsque vous étiez en ligne ne soient pas les versions les plus récentes des fichiers contenus dans votre dossier OpenText Tempo car les modifications effectuées dans les fichiers ne peuvent pas se synchroniser lorsque vous êtes hors ligne.

  5. Pour revenir à vos fichiers, appuyez sur la touche BlackBerry Echap.
