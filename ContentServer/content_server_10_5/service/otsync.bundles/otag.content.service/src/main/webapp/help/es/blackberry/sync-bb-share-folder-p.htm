Para compartir una carpeta:

  1. Pulse Archivos y desplácese hasta la carpeta destino.

  2. Resalte la carpeta.

  3. Pulse la tecla BlackBerry Menú del dispositivo y seleccione Compartir carpeta.

     

    Nota:
    Si Compartir carpeta no está disponible, es porque no tiene permiso para compartir esta carpeta. Para ver una lista de las carpetas que no pueden compartirse, consulte Excepciones para compartir carpetas.

     

  4. Escriba el nombre de la persona con la que desea compartir esta carpeta.

  5. Resalte el nombre de usuario y pulse el BlackBerry panel táctil del dispositivo. Se le enviará una invitación para compartir la carpeta.

    La carpeta aparecerá ahora marcada como carpeta compartida, .

Excepciones para compartir carpetas

Puede compartir cualquier carpeta de su carpeta OpenText Tempo, pero se aplican las excepciones siguientes:

  • No puede compartir una carpeta que sea hija o padre de una carpeta compartida.

  • No puede compartir una carpeta que se comparta con usted.

  • No puede compartir la raíz de una carpeta compartida, a menos que sea el propietario de la carpeta.

  • El usuario no puede compartir su carpeta OpenText Tempo.

  • Si es un usuario externo y no un empleado de la compañía, no puede compartir ninguna carpeta.

Uso compartido de carpetas con usuarios externos

El administrador puede concederle el privilegio de compartir carpetas con usuarios que no pertenezcan a la compañía. Póngase en contacto con el administrador para solicitar este privilegio. Para obtener información sobre cómo compartir carpetas con usuarios externos, consulte Para compartir una carpeta con un usuario externo.

 

Atención
Al compartir una carpeta, el colaborador obtiene el control completo del contenido de la carpeta. Tan pronto como se hagan revisiones a una carpeta compartida, los cambios se sincronizan automáticamente con todos los colaboradores que tienen acceso compartido a dicha carpeta, incluido el propietario de la misma.

 

Para obtener más información sobre carpetas compartidas, consulte Uso compartido.
