So fügen Sie ein Foto mit der BlackBerry-Kamera hinzu:

  1. Klicken Sie auf , und navigieren Sie zum Zielordner, wo Sie ein Foto hinzufügen möchten.

     

    Hinweis:
    Sie können ein Foto nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen (durch gekennzeichnet).

     

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Foto hinzufügen aus.

  3. Drücken Sie die BlackBerry Trackpad-Taste, um ein Bild aufzunehmen.

  4. Klicken Sie auf Hinzufügen zu OpenText Tempo. Fotos werden als JPG-Dateien gespeichert. Sie können den Namen des Fotos vor dem Hochladen nicht bearbeiten.

So fügen Sie ein Foto aus einer BlackBerry-Anwendung hinzu:

  1. Starten Sie eine BlackBerry-Kameraanwendung.

  2. Navigieren Sie zu dem Foto, das Sie importieren möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hinzufügen zu OpenText Tempo aus.

  4. Optional Wenn Sie dazu aufgefordert werden, melden Sie sich bei OpenText Tempo an. Ausführliche Informationen finden Sie in Anmelden und Profil verwalten.

  5. Navigieren Sie zum Zielspeicherort, an dem Sie die Datei platzieren möchten.

     

    Hinweis:
    Sie können ein Foto nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen (durch gekennzeichnet).

     

  6. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Hier hochladen aus.

So fügen Sie einen Ordner hinzu:

  1. Klicken Sie auf , und navigieren Sie zum Zielspeicherort für den neuen Ordner.

     

    Hinweis:
    Sie können einen Ordner nicht einem freigegebenen Ordner hinzufügen, für den Sie nur Leserechte besitzen (durch gekennzeichnet).

     

  2. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ordner hinzufügen aus.

  3. Geben Sie im Feld Ordnername einen Namen für diesen Ordner ein.

  4. Drücken Sie die BlackBerry Eingabe-Taste.
