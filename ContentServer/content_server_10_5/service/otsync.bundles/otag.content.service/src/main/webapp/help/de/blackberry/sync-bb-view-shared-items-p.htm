So zeigen Sie Dateien in freigegebenen Ordnern an:

  1. Klicken Sie auf . Eine Liste von Freigabeeinladungen wird angezeigt. Bevor Sie den Inhalt eines freigegebenen Ordners anzeigen können, muss Ihnen der Ordnereigentümer eine Freigabeeinladung senden, die Sie akzeptieren müssen. Sie haben eine Freigabeeinladung erhalten, wenn eine Überlagerung angezeigt wird. Beispiel: .

  2. Optional Um sicherzustellen, dass sämtliche Freigabeeinladungen angezeigt werden, aktualisieren Sie die Anzeige regelmäßig. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Aktualisieren aus.

  3. Markieren Sie eine Freigabeeinladung.

  4. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Annehmen aus.

    Der freigegebene Order wird im Stammverzeichnis Ihres OpenText Tempo-Ordners abgelegt. Der Name des freigegebenen Ordners wird mit dem Benutzernamen des Ordnereigentümers versehen. Beispiel: Marketingmaterialien (tschmidt).

    Sie haben entweder Lese- und Schreibrechte oder nur Leserechte für den Inhalt im freigegebenen Ordner.

    Mit Schreib- und Leserechten (durch gekennzeichnet) können Sie den Inhalt in einem freigegebenen Ordner aufrufen und bearbeiten, so als ob es Ihr eigener wäre. Sämtliche Änderungen am Inhalt eines freigegebenen Ordners werden mit allen anderen Teilnehmern synchronisiert, die Zugriff auf diesen Ordner haben, einschließlich des Eigentümers.

    Mit Leserecht (durch gekennzeichnet) können Sie den Inhalt im freigegebenen Ordner nur lesen. Die Titelleiste weist beim Navigieren im Ordner einen orangefarbenen Hintergrund auf.
