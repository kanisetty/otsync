要删除您拥有的共享文件夹：

  1. 单击 ，然后导航至由 标识的目标共享文件夹。

  2. 突出显示该共享文件夹。

  3. 按 BlackBerry [菜单]键，然后选择[删除文件夹]。

     

    小心
    这将为您和所有协作者删除该文件夹。您也可移除协作者。有关详细信息，请参阅移除协作者。

     
     

    注意:
    如果[删除文件夹]不可用，则您选择了与您共享的文件夹。有关详细信息，请参阅下方的删除他人与您共享的文件夹。

     

  4. 单击[确定]。

要删除他人与您共享的文件夹：

  1. 单击 。

  2. 突出显示该共享文件夹。

  3. 按 BlackBerry [菜单]键，然后选择[移除共享]。此操作将断开与共享文件夹的连接，并从您的 OpenText Tempo 文件夹中将其移除。此操作不会删除所有者或其他协作者的文件夹。

     

    小心
    如果[移除共享]不可用，则您选择了自己拥有的共享文件夹。有关详细信息，请参阅此页面开始处的删除您拥有的共享文件夹。

     

  4. 单击[确定]。
