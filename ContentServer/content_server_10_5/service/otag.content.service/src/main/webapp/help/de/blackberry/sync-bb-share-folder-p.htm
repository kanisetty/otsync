So geben Sie einen Ordner frei:

  1. Klicken Sie auf Dateien, und navigieren Sie zum Zielordner.

  2. Markieren Sie den Ordner.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ordner freigeben aus.

     

    Hinweis:
    Wenn die Schaltfläche Ordner freigeben nicht verfügbar ist, verfügen Sie nicht über die Berechtigung zum Freigeben dieses Ordners. Eine Liste von Ordnern, die nicht freigegeben werden können, finden Sie unter Ausnahmen für die Ordnerfreigabe.

     

  4. Geben Sie den Namen der Person ein, für die Sie diesen Ordner freigeben möchten.

  5. Markieren Sie den Namen des Benutzers, und drücken Sie die Taste BlackBerry Trackpad. Eine Freigabeeinladung wird gesendet.

    Der Ordner wird nun als freigegebener Ordner markiert: .

Ausnahmen für die Ordnerfreigabe

Sie können jeden Ordner in Ihrem OpenText Tempo-Ordner freigeben. Dabei gelten jedoch folgende Ausnahmen:

  • Sie können keinen Ordner freigeben, der einem freigegebenen Ordner untergeordnet oder übergeordnet ist.

  • Sie können keinen Ordner freigeben, der für Sie freigegeben wurde.

  • Das Stammverzeichnis eines freigegebenen Ordners können Sie nur freigeben, wenn Sie der Eigentümer des Ordners sind.

  • Sie können Ihren OpenText Tempo-Ordner nicht freigeben.

  • Wenn Sie ein externer Benutzer und nicht Mitarbeiter des Unternehmens sind, können Sie keine Ordner freigeben.

Einen Ordner für externe Benutzer freigeben

Ihr Administrator kann Ihnen die Berechtigung zum Freigeben von Ordnern für Personen außerhalb des Unternehmens erteilen. Wenden Sie sich an Ihren Administrator, um diese Berechtigung anzufordern. Informationen zum Freigeben von Ordnern für externe Benutzer finden Sie unter So geben Sie einen Ordner für einen externen Benutzer frei.

 

Achtung
Durch Freigeben eines Ordners erhalten Personen vollständige Kontrolle über die Inhalte des Ordners. Wenn am freigegebenen Ordner Änderungen vorgenommen wurden, werden diese Änderungen automatisch mit allen Personen, die Zugriff auf diesen Ordner haben, einschließlich des Eigentümers, synchronisiert.

 

Weitere Informationen zu freigegebenen Ordnern finden Sie unter Freigabe.
