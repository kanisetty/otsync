So verschieben oder kopieren Sie eine Datei:

  1. Klicken Sie auf , und navigieren Sie zur Zieldatei. Wenn die Zieldatei als Favorit markiert ist, klicken Sie auf .

  2. Markieren Sie die Datei, die Sie verschieben oder kopieren möchten.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ausschneiden oder Kopieren aus.

     

    Hinweis:
    Sie können eine Datei nur aus einem freigegebenen Ordner kopieren, für den Sie Leserechte besitzen (durch gekennzeichnet).

     

  4. Navigieren Sie zum Zielordner, in dem Sie den Inhalt der Zwischenablage platzieren möchten.

     

    Hinweis:
    Sie können eine Datei nicht in einen freigegebenen Ordner verschieben oder kopieren, für den Sie nur Leserechte besitzen (durch gekennzeichnet).

     

  5. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Einfügen aus.

So verschieben oder kopieren Sie einen Ordner:

  1. Klicken Sie auf , und navigieren Sie zum Zielordner.

  2. Markieren Sie die Datei, die Sie verschieben oder kopieren möchten.

     

    Hinweis:
    Sie können freigegebene Ordner, die Ihnen gehören, an eine beliebige Stelle im OpenText Tempo-Ordner verschieben, außer in einen anderen freigegebenen Ordner.

    Wenn Sie einen Ordner verschieben, der für Sie freigegeben wurde, wird eine private Kopie am neuen Speicherort erstellt, der freigegebene Ordner verbleibt jedoch auf der Stammebene des OpenText Tempo-Ordners.

     

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ausschneiden oder Kopieren aus.

  4. Navigieren Sie zum Zielordner, in dem Sie den Inhalt der Zwischenablage platzieren möchten.

     

    Hinweis:
    Ein Ordner kann nicht in einen freigegebenen Ordner verschoben oder kopiert werden, für den Sie nur Leserechte besitzen (durch gekennzeichnet).

     

  5. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Einfügen aus.
