Per aggiungere una nuova versione utilizzando un file sul proprio dispositivo BlackBerry:

  1. Fare clic su e posizionarsi sul file desiderato. Se il file è contrassegnato come preferito, fare clic su .

  2. Evidenziare il file a cui si desidera aggiungere una nuova versione.

     

    Nota:
    Non è possibile aggiungere una versione in una cartella condivisa per cui si dispone di un'autorizzazione di sola lettura, identificata da .

     

  3. Premere sul BlackBerry il tasto Menu e selezionare Carica nuova versione.

  4. Posizionarsi tramite il dispositivo BlackBerry sulla nuova versione del file.

  5. Evidenziare il file.

  6. Premere sul BlackBerry il tasto Menu e selezionare Carica. Il file viene caricato automaticamente.

Per caricare un file memorizzato sul dispositivo BlackBerry:

  1. Fare clic su e posizionarsi nella cartella di destinazione in cui si desidera caricare un file.

  2. Premere sul BlackBerry il tasto Menu e selezionare Carica qui.

  3. Spostarsi nella posizione del documento desiderato sul proprio smartphone BlackBerry.

     

    Nota:
    Non è possibile caricare un file in una cartella condivisa per cui si dispone di un'autorizzazione di sola lettura, identificata da .

     

  4. Premere sul BlackBerry il tasto Menu e selezionare Carica.

Per importare un file da un'applicazione BlackBerry:

  1. Avviare un'applicazione BlackBerry.

  2. Premere sul BlackBerry il tasto Menu e selezionare Aggiungi aOpenText Tempo.

  3. Opzionale Se richiesto, effettuare l'accesso a OpenText Tempo. Per ulteriori dettagli, vedere la sezione Accesso e gestione del proprio profilo.

  4. Spostarsi alla posizione dove si desidera posizionare file.

     

    Nota:
    Non è possibile importare un file in una cartella condivisa per cui si dispone di un'autorizzazione di sola lettura, identificata da .

     

  5. Premere sul BlackBerry il tasto Menu e selezionare Carica qui.
