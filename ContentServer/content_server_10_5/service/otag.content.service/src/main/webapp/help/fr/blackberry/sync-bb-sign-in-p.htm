Pour se connecter :

  1. Lancez l'application OpenText Tempo.

  2. Mettez votre profil en surbrillance, puis appuyez sur la touche BlackBerry Trackpad. Si vous n'avez pas créé de profil, voir Création d'un profil.

  3. Si vous y êtes invité, effectuez l'une des opérations suivantes :

    • Saisissez votre mot de passe. Si vous avez oublié votre mot de passe, contactez votre administrateur.

    • Si vous n'appartenez pas à l'entreprise, saisissez le mot de passe que vous avez choisi lorsque vous avez créé votre compte OpenText Tempo. Si vous avez oublié votre mot de passe, reportez-vous à l'e-mail que vous avez reçu lorsque vous avec créé votre compte OpenText Tempo.

     

    Note :
    Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

     

  4. Facultatif Activez la case Rester connecté, si elle est disponible.

     

    Note :
    Cette case à cocher n'est pas disponible si votre BlackBerry est hors ligne.

     

  5. Appuyez sur la touche BlackBerry Entrée.

 

Attention

  • En autorisant OpenText Tempo à maintenir votre connexion, vous risquez de compromettre la sécurité de votre BlackBerry s'il est utilisé par un tiers.

  • Lorsque plusieurs profils sont configurés sur votre BlackBerry, d'autres profils pourront accéder à votre compte OpenText Tempo si la case Rester connecté est sélectionnée et si vous n'êtes pas déconnecté. Pour savoir comment se déconnecter d'OpenText Tempo, voir Déconnexion.
