Pour synchroniser manuellement des favoris :

  1. Cliquez sur Favoris.

  2. Appuyez sur la touche BlackBerry Menu et sélectionnez Synchroniser les favoris. Si le processus de synchronisation de vos favoris est en cours ou si vous êtes hors ligne, l'option Synchroniser les favoris n'est pas disponible.

     

    Note :
    Si vous sélectionnez Actualiser, l'affichage de l'écran Favoris est mis à jour. Toutefois, vos documents favoris ne sont pas téléchargés sur votre BlackBerry pour une visualisation hors ligne. Pour télécharger le fichier, vous devez synchroniser vos fichiers favoris ou ouvrir le fichier.

     

Pour synchroniser vos fichiers favoris automatiquement, voir Modification des paramètres de synchronisation.
