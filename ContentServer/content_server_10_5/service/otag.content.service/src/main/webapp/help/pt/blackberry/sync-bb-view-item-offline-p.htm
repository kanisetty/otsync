Para exibir itens off-line:

  1. Para tornar um item disponível para exibição off-line, marque o arquivo como favorito quando estiver on-line. Para obter detalhes, consulte Para adicionar um favorito. Arquivos marcados como favoritos são indicados pela camada . Por exemplo, .

  2. Clique em Favoritos.

  3. Destaque o item que deseja exibir.

  4. Pressione a tecla Trackpad do BlackBerry para exibir o item. Os arquivos abertos quando você estava off-line talvez não sejam versões anteriores dos arquivos na pasta OpenText Tempo uma vez que as edições feitas nos arquivos não podem ser sincronizadas quando você está off-line.

  5. Pressione a tecla Escape do BlackBerry para retornar aos seus arquivos.
