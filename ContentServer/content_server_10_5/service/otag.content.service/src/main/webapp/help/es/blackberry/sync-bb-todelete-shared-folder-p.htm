Para suprimir una carpeta compartida de su propiedad:

  1. Pulse Archivos y desplácese hasta la carpeta destino.

  2. Resalte la carpeta compartida.

  3. Pulse la tecla BlackBerry Menú y seleccione Suprimir carpeta.

     

    Nota:
    La carpeta se eliminará para usted y para todos sus colaboradores. Como alternativa, puede suprimir colaboradores. Para obtener información detallada, consulte Para eliminar colaboradores.

    Si Suprimir carpeta no está disponible es porque ha seleccionado una carpeta que se comparte con usted. Antes de continuar, consulte Eliminación de una carpeta compartida.

     

  4. Haga clic en Aceptar.

Para suprimir una carpeta que se comparte con usted:

  1. Pulse Archivos.

  2. Resalte la carpeta compartida para la que desea interrumpir la conexión.

  3. Pulse la tecla BlackBerry Menú del dispositivo y seleccione Eliminar elemento para compartir.

     

    Nota:
    Si Eliminar elemento para compartir no está disponible, ha seleccionado una carpeta de su propiedad. Antes de continuar, consulte Eliminación de una carpeta compartida.

     

  4. Haga clic en Aceptar.

Eliminación de una carpeta compartida

Las carpetas compartidas incluyen dos iconos verdes en la esquina inferior derecha de la carpeta, . Existen dos tipos de carpetas compartidas:

  • Carpetas de su propiedad que comparte con sus compañeros. Si suprime una carpeta compartida de su propiedad, se suprimirá la carpeta para usted y para todos los colaboradores que comparten el acceso a dicha carpeta. Como alternativa, puede suprimir a uno o a todos los colaboradores. Para obtener más información, consulte Para eliminar colaboradores.

  • Carpetas que no son de su propiedad, pero que se comparten con usted. Al suprimir una carpeta que se comparte con usted, se interrumpe la conexión con la carpeta compartida y se elimina de su carpeta OpenText Tempo. Esta acción no suprime la carpeta para el propietario ni el resto de colaboradores.
