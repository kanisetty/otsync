So löschen Sie einen freigegebenen Ordner, dessen Eigentümer Sie sind:

  1. Klicken Sie auf , und navigieren Sie zum freigegebenen Zielordner (durch gekennzeichnet).

  2. Markieren Sie den freigegebenen Ordner.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Ordner löschen aus.

     

    Achtung
    Dadurch wird der Ordner für Sie und alle Teilnehmer gelöscht. Alternativ können Sie Teilnehmer entfernen. Ausführliche Informationen finden Sie in Einen Teilnehmer entfernen.

     
     

    Hinweis:
    Wenn Ordner löschen nicht verfügbar ist, haben Sie einen Ordner ausgewählt, der für Sie freigegeben ist. Weitere Informationen hierzu finden Sie unter So löschen Sie einen Ordner, der für Sie freigegeben wurde weiter unten.

     

  4. Klicken Sie auf OK.

So löschen Sie einen Ordner, der für Sie freigegeben wurde:

  1. Klicken Sie auf .

  2. Markieren Sie den freigegebenen Ordner.

  3. Drücken Sie die BlackBerry Menü-Taste, und wählen Sie Freigabe entfernen aus. Hierdurch wird die Verbindung zum freigegebenen Ordner getrennt und der Ordner aus Ihrem OpenText Tempo-Ordner entfernt. Für den Eigentümer und andere Teilnehmer wird der Ordner nicht gelöscht.

     

    Achtung
    Wenn Freigabe entfernen nicht verfügbar ist, haben Sie einen freigegebenen Ordner ausgewählt, dessen Eigentümer Sie sind. Weitere Informationen finden Sie oben auf dieser Seite unter So löschen Sie einen freigegebenen Ordner, dessen Eigentümer Sie sind.

     

  4. Klicken Sie auf OK.
