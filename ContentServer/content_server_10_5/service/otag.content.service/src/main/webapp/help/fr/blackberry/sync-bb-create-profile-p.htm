Pour créer un profil :

  1. Si vous lancez l'application OpenText Tempo pour la première fois, cliquez sur Nouveau profil.

    Si vous voulez créer un autre profil et que vous êtes déjà connecté à OpenText Tempo, procédez comme suit :

    1. Appuyez sur la touche BlackBerry Menu.

    2. Sélectionnez Changer de profil.

    3. Cliquez sur Nouveau profil.

  2. Dans le champ Nom du profil, saisissez un nom de profil unique sur votre BlackBerry.

  3. Dans le champ Nom de l'utilisateur, saisissez votre nom d'utilisateur. Si vous n'appartenez pas à l'entreprise, saisissez votre adresse e-mail.

     

    Note :
    Un utilisateur qui n'appartient pas à l'entreprise doit d'abord créer un compte OpenText Tempo pour pouvoir créer un profil.

    Pour créer un compte, un utilisateur au sein de l'entreprise doit partager un dossier avec l'utilisateur externe. Cela permet à l'utilisateur externe de recevoir un e-mail d'invitation détaillant la procédure à suivre pour créer un compte.

     

  4. Dans le champ Mot de passe, saisissez votre mot de passe.

    Si vous n'appartenez pas à l'entreprise, saisissez le mot de passe que vous avez choisi lorsque vous avez créé votre compte OpenText Tempo. Si vous avez oublié votre mot de passe, reportez-vous à l'e-mail que vous avez reçu lorsque vous avec créé votre compte OpenText Tempo.

     

    Note :
    Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

     

  5. Dans le champ Serveur, saisissez l'URL de votre serveur OpenText Tempo.

    Pour connaître l'URL du serveur OpenText Tempo, contactez votre administrateur.

    Si vous n'appartenez pas à l'entreprise, l'URL d'OpenText Tempo apparaît dans l'e-mail que vous avez reçu lors de la création de votre compte OpenText Tempo.

  6. Facultatif Cochez la case Rester connecté pour un accès immédiat lors du prochain lancement de l'application. Les exceptions sont les suivantes : si vous vous déconnectez d'OpenText Tempo ou si la session arrive à expiration. Reportez-vous à la section Attention relative à cette option.

     

    Note :
    Cette case à cocher n'est pas disponible si votre BlackBerry est hors ligne.

     

  7. Cliquez sur Enregistrer et se connecter.

    Les données du profil incluent les informations d'identification de votre compte, l'URL d'OpenText Tempo et la liste de vos fichiers favoris. Les profils sont spécifiques à votre périphérique. Vous devez créer un profil et marquer les fichiers comme favoris sur chacun de vos smartphones et tablettes qui utilisent l'application OpenText Tempo.

 

Attention

  • En autorisant OpenText Tempo à maintenir votre connexion, vous risquez de compromettre la sécurité de votre BlackBerry s'il est utilisé par un tiers.

  • Lorsque plusieurs profils sont configurés sur votre BlackBerry, d'autres profils pourront accéder à votre compte OpenText Tempo si la case Rester connecté est sélectionnée et si vous n'êtes pas déconnecté. Pour vous déconnecter, appuyez sur la touche Menu BlackBerry, puis sélectionnez Déconnexion.
