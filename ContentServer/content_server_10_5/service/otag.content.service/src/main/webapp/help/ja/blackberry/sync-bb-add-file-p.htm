BlackBerry に保存されているファイルをアップロードするには、次の操作を行います。

  1. [ファイル] をクリックし、ファイルをアップロードするターゲット フォルダーに移動します。

  2. BlackBerry [メニュー] キーを押し、[ここにアップロード] を選択します。

  3. BlackBerry スマートフォン上のターゲット ドキュメントに移動します。

  4. BlackBerry [メニュー] キーを押し、[アップロード] を選択します。

BlackBerry アプリケーションからファイルをインポートするには、次の操作を行います。

  1. BlackBerry アプリケーションを起動します。たとえば、[カメラ] アプリケーションを起動します。

  2. BlackBerry [メニュー] キーを押し、[OpenText Tempo への追加] を選択します。

  3. オプション 求められた場合は、OpenText Tempo にサインインします。詳細についてはサインインするにはを参照してください。

  4. ファイルを配置するターゲットの場所に移動します。

  5. BlackBerry [メニュー] キーを押し、[ここにアップロード] を選択します。
