Per sincronizzare manualmente i preferiti:

  1. Fare clic su Preferiti.

  2. Premere il tasto Menu BlackBerry e selezionare Sincronizza Preferiti. Se l'applicazione sta sincronizzando i preferiti, o ci si trova offline, l'opzione Sincronizza Preferiti non è disponibile.

     

    Nota:
    Selezionare Aggiorna provoca l'aggiornamento della visualizzazione della schermata Preferiti. Non provoca lo scaricamento dei documenti preferiti sul proprio dispositivo BlackBerry per la visualizzazione offline. Per scaricare il file, è necessario sincronizzare i file preferiti o aprire il file.

     

Per sincronizzare i file preferiti automaticamente, vedere Modifica delle impostazioni di sincronizzazione.
