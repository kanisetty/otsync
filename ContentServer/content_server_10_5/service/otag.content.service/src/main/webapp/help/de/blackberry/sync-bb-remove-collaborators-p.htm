Einen Teilnehmer entfernen

Sie können einen Teilnehmer von einem freigegebenen Ordner, dessen Eigentümer Sie sind, über die OpenText Tempo-Webbenutzerschnittstelle entfernen. Dadurch wird die Verbindung zum freigegebenen Ordner getrennt, und der Ordner wird aus dem OpenText Tempo-Ordner des betreffenden Teilnehmers gelöscht.

Sie können auf die Webbenutzerschnittstelle zugreifen, indem Sie den OpenText Tempo-URL in der Adressleiste eines Webbrowsers eingeben und sich bei OpenText Tempo anmelden. Weitere Informationen zum Entfernen von Teilnehmern finden Sie in der Online-Hilfe der OpenText Tempo-Webbenutzerschnittstelle.
