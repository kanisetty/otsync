Para ver un archivo:

  1. Haga clic en y desplácese hasta el archivo destino resaltando una carpeta y pulsando en el panel táctil del dispositivo BlackBerry. La barra de título de OpenText Tempo muestra el nombre de la carpeta que está viendo en ese momento. Si el archivo destino se ha marcado como favorito, haga clic en .

  2. Opcional Para asegurar que está viendo el contenido más actualizado, debe actualizar de forma periódica el contenido de la carpeta. Pulse la tecla Menú del dispositivo BlackBerry y seleccione Actualizar. Esto actualiza la pantalla, pero no baja los documentos. Para bajar un documento, ábralo.

  3. Resalte el archivo que desea ver.

  4. Pulse el panel táctil del dispositivo BlackBerry para ver el archivo.

    Cuando visualice documentos en una carpeta compartida para la que tenga permisos de lectura/escritura, dichos documentos no están reservados. Puede abrir y editar un documento mientras un colaborador realiza cambios en él. Las carpetas de lectura/escritura compartidas se identifican mediante .

    Cuando visualice documentos en una carpeta compartida para la que tenga permisos de solo lectura, dichos documentos están reservados. De modo que podrá ver, pero no editar los documentos que se encuentren en estas carpetas. Las carpetas compartidas de solo lectura se identifican mediante y la barra de título mostrará un fondo naranja cuando se desplace por ellas.

  5. Pulse la tecla Escape del dispositivo BlackBerry para volver a sus archivos.

Para editar un archivo:

  1. Haga clic en y desplácese hasta el archivo destino. Si el archivo destino se ha marcado como favorito, haga clic en .

  2. Resalte el archivo que desea editar.

     

    Nota:
    No puede editar un archivo en una carpeta compartida para la que tenga permiso de solo lectura, identificable mediante .

     

  3. Pulse el panel táctil del dispositivo BlackBerry para ver el archivo.

  4. El archivo se abre en una aplicación de terceros. Si la aplicación admite la edición, edite el archivo y guarde los cambios.

    Los cambios que haga en el documento no se guardan en el archivo de la carpeta OpenText Tempo. Debe subir el archivo modificado desde la aplicación de terceros o la carpeta de caché de BlackBerry.

  5. En OpenText Tempo, suba el archivo editado añadiendo una versión del archivo. Para más información, consulte Adición de un archivo.

Para obtener más información sobre la administración de archivos, consulte Visualización, edición y trabajo con archivos.
