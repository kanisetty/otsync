Pour supprimer votre profil :

  1. Appuyez sur la touche Menu BlackBerry et sélectionnez Changer de profil.

  2. Mettez en surbrillance le profil que vous souhaitez supprimer.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Supprimer le profil.

  4. Pour confirmer l'opération, appuyez sur OK.
