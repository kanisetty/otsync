Per cercare la propria cartella OpenText Tempo:

  1. Fare clic su .

  2. Digitare il termine da cercare.

  3. Premere sul BlackBerry il tasto Invio. Vengono visualizzati i risultati della ricerca.

  4. Evidenziare il file o la cartella che si desidera visualizzare.

  5. Premere sul BlackBerry il tasto Menu per visualizzare le opzioni disponibili.

Per aprire il percorso di un elemento dai risultati della ricerca:

  1. Cercare la propria cartella OpenText Tempo.

  2. Evidenziare il file di cui si desidera aprire il percorso.

  3. Premere sul BlackBerry il tasto Menu e selezionare Apri percorso.
