Удаление пользователя с общим доступом

Удалить пользователя с общим доступом из общей папки можно с помощью веб-интерфейса пользователя OpenText Tempo. Это действие приведет к удалению соединения с общей папкой и папки из папки OpenText Tempo пользователя с общим доступом.

Получить доступ к веб-интерфейсу пользователя можно с помощью ввода в веб-браузере URL-адреса OpenText Tempo и входа в систему OpenText Tempo. Подробные сведения по удалению пользователей с общим доступом см. в разделе интерактивной справки веб-интерфейса пользователя OpenText Tempo.
