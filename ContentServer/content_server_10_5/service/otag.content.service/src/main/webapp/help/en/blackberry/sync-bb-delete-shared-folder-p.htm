To delete a shared folder that you own:

  1. Click and navigate to the target shared folder. Shared folders that you own are identified by the overlay. For example, . If the folder name of the target folder is appended with a user name, the target folder is shared with you. See Deleting a Folder That is Shared With You below.

  2. Highlight the shared folder.

  3. Press the BlackBerry Menu key and select Delete Folder. If Delete Folder is not available, you selected a folder that is shared with you. For details, see Deleting a Folder That is Shared With You below.

     

    Caution
    This will delete the folder for you and all collaborators who have share access to the folder, including the owner of the shared folder. As an alternative, you can remove collaborators. For details, see Removing a Collaborator.

     

  4. Click OK.

To delete a folder that is shared with you:

  1. Click and locate the target folder. A folder that is shared with you shows the user name of the folder owner in the folder name. For example, Marketing Materials (tsmith).

  2. Highlight the shared folder.

  3. Press the BlackBerry Menu key and select Remove Share. This severs the connection to the shared folder and removes it from your Tempo Box folder. This does not delete the folder for the owner or other collaborators.

     

    Caution
    If Remove Share is not available, you selected a shared folder that you own. For details, see Deleting a Shared Folder That You Own at the start of this page.

     

  4. Click OK.
