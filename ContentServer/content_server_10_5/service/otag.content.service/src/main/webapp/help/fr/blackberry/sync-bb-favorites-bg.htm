Pour ajouter un favori :

  1. Cliquez sur , puis accédez au fichier cible.

  2. Mettez le fichier en surbrillance.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Ajouter aux favoris.

    Le fichier est marqué à l'aide du recouvrement . Par exemple : .

  4. Facultatif Si le recouvrement apparaît, le fichier n'est pas téléchargé. Par exemple : . Pour télécharger le fichier, ouvrez-le ou synchronisez vos favoris. Pour plus d'informations, voir Pour synchroniser manuellement des favoris, ci-dessous.

    Le fichier est alors téléchargé sur votre BlackBerry pour que vous puissiez le visualiser hors ligne. Pour synchroniser vos fichiers favoris automatiquement, voir Modification des paramètres d'OpenText Tempo.

    Un enregistrement de vos fichiers favoris est stocké dans votre profil. Les profils sont spécifiques à votre périphérique. Vous devez créer un profil et marquer les fichiers comme favoris sur chacun de vos smartphones et tablettes qui utilisent l'application OpenText Tempo. Pour plus d'informations, voir Création d'un profil.

Pour visualiser un favori :

  1. Cliquez sur .

  2. Facultatif Pour être sûr de visualiser tous vos favoris, actualisez régulièrement l'affichage. Appuyez sur la touche BlackBerry Menu et sélectionnez Actualiser. Les documents ne sont pas téléchargés pour être visualisés hors ligne. Pour télécharger le document, synchronisez vos favoris ou ouvrez le document. Pour plus d'informations, voir Pour synchroniser manuellement des favoris, ci-dessous.

Pour ouvrir l'emplacement d'un fichier favori :

  1. Cliquez sur .

  2. Mettez en surbrillance le fichier que vous souhaitez ouvrir.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Ouvrir l'emplacement.

Pour supprimer un favori :

  1. Cliquez sur ou sur , puis accédez au fichier cible.

  2. Mettez en surbrillance le fichier que vous souhaitez supprimer de votre liste de fichiers favoris.

  3. Appuyez sur la touche BlackBerry Menu et sélectionnez Supprimer des favoris. Le fichier n'apparaît plus dans votre liste de favoris. Il n'est toutefois pas supprimé de votre dossier OpenText Tempo.

Pour synchroniser manuellement des favoris :

  1. Cliquez sur .

  2. Appuyez sur la touche BlackBerry Menu et sélectionnez Synchroniser les favoris. Si le processus de synchronisation de vos favoris est en cours ou si vous êtes hors ligne, l'option Synchroniser les favoris n'est pas disponible.

     

    Note :
    Si vous sélectionnez Actualiser, l'affichage est mis à jour. Toutefois, vos documents favoris ne sont pas téléchargés sur votre BlackBerry pour une visualisation hors ligne. Pour télécharger le fichier, vous devez synchroniser vos fichiers favoris ou ouvrir le fichier.

     

Pour synchroniser vos fichiers favoris automatiquement, voir Modification des paramètres d'OpenText Tempo.
