Pour se connecter :

  1. Lancez l'application OpenText Tempo.

  2. Pour accéder à votre dossier OpenText Tempo, vous devez d'abord créer un profil avant de vous connecter. Pour plus d'informations, voir Création d'un profil.

  3. Mettez votre profil en surbrillance, puis appuyez sur la touche BlackBerry Trackpad.

  4. Si vous y êtes invité, saisissez votre mot de passe.

    Si vous n'appartenez pas à l'entreprise, saisissez le mot de passe que vous avez choisi lorsque vous avez créé votre compte OpenText Tempo. Si vous avez oublié votre mot de passe, reportez-vous à l'e-mail que vous avez reçu lorsque vous avec créé votre compte OpenText Tempo.

     

    Note :
    Votre mot de passe est sensible à la casse. Veillez à le saisir correctement.

     

  5. Facultatif Cochez la case Rester connecté pour un accès immédiat lors du prochain lancement de l'application. Les exceptions sont les suivantes : si vous vous déconnectez d'OpenText Tempo ou si la session arrive à expiration. Reportez-vous à la section Attention ci-dessous.

     

    Note :
    Cette case à cocher n'est pas disponible si votre BlackBerry est hors ligne.

     

  6. Appuyez sur la touche BlackBerry Entrée.

 

Attention

  • En autorisant OpenText Tempo à maintenir votre connexion, vous risquez de compromettre la sécurité de votre BlackBerry s'il est utilisé par un tiers.

  • Lorsque plusieurs profils sont configurés sur votre BlackBerry, d'autres profils pourront accéder à votre compte OpenText Tempo si la case Rester connecté est sélectionnée et si vous n'êtes pas déconnecté. Pour vous déconnecter, appuyez sur la touche Menu BlackBerry, puis sélectionnez Déconnexion.

 

Pour plus d'informations sur la gestion de votre profil, voir Connexion et gestion de votre profil.
